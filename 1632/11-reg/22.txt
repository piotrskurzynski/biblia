A nie było przez trzy látá wojny miedzy Syryjcżykámi y miedzy Izráelcżykámi.
Y ſtáło śię roku trzećiego / że przyjechał Jozáfát król Judzki / do królá Izráelſkiego.
Tedy rzekł król Izráelſki do ſług ſwojich : Nie wiećież / iż náƺe jeſt Rámot Gáláád? A my zániedbywámy odebráć go z ręki królá Syryjſkiego.
Przetoż rzekł do Jozáfátá : Poćiągnieƺże ze mną ná wojnę przećiwko Rámot Gáláád? Y rzekł Jozáfát do królá Izráelſkiego : Jákom ja / ták y ty ; jáko lud mój / ták lud twój ; jáko konie moje / ták konie twoje.
Nádto rzekł Jozáfát do królá Izráelſkiego : Spytáj śię proƺę dźiś ſłowá Páńſkiego.
A ták zebrał król Izráelſki proroków około cżterech ſet mężów / y rzekł do nich : Mámże ćiągnąć ná wojnę przećiwko Rámot Gáláád? cży zániecháć? Y odpowiedźieli mu : Ciągnij ; bo je Pán da w ręce królewſkie.
Ale Jozáfát rzekł : Niemáƺże tu którego proroká Páńſkiego / żebyſmy śię go pytáli?
Y rzekł król Izráelſki do Jozáfátá : Jeſt jeƺcże mąż jeden / przez któregobyſmy śię mogli rádźić Páná ; ále go ja nienáwidzę / bo mi nic dobrego nie prorokuje / jedno złe / Micheáƺ / ſyn Jemlá. Y rzekł Jozáfát : Niech ták nie mówi król.
A ták záwołał król Izráelſki komorniká niektórego / y rzekł : Przywiedź tu rychło Micheáƺá / ſyná Jemlowego.
Miedzy tym król Izráelſki / y Jozáfát / król Judzki / śiedźieli ná ſtolicách ſwojich / ubráni w ƺáty królewſkie / ná plácu u wrót bramy Sámáryjſkiej / á wƺyſcy prorocy prorokowáli przed nimi.
A Sedechyjáƺ / ſyn Chenááná / ſpráwił ſobie rogi żelázne / y rzekł : Ták mówi Pán : Temi będźieƺ bódł Syryjcżyki / áż je wyniƺcżyƺ.
Tákże wƺyſcy prorocy prorokowáli / mówiąc : Ciągnij do Rámot Gáláád / á będźieć śię ƺcżęśćiło ; ábowiem je podá Pán w ręce królewſkie.
Tedy poſeł / który chodźił / áby przyzwał Micheáƺá / rzekł do niego / mówiąc : Oto teraz ſłowá proroków jednemi uſty dobrze tuƺą królowi ; niechże będźie proƺę ſłowo twoje / jáko ſłowo jednego z nich / á mów dobre rzecży.
Y rzekł Micheáƺ : Jáko żywy Pán / że co mi kolwiek rzecże Pán / to mówić będę.
A gdy przyƺedł do królá / rzekł król do niego : Micheáƺu / mámyż ćiągnąć ná wojnę przećiw Rámot Gáláád / cżyli zániecháć? A on mu rzekł : Ciągnij / á będźieć śię ƺcżęśćiło ; ábowiem je podá Pán w ręce królewſkie.
Y rzekł do niego król : A wieleż ćię rázy mam przyśięgą obowiązáć / ábyś mi nie mówił jedno práwdę w imieniu Páńſkim?
Przetoż rzekł : Widźiałem wƺyſtek lud Izráelſki rozproƺony po górách jáko owce / które nie máją páſterzá ; bo rzekł Pán : Nie máją ći páná ; niech śię wróći káżdy do domu ſwego w pokoju.
Y rzekł król Izráelſki do Jozáfátá : Izáżem ći nie powiádał / że mi nie miał prorokowáć dobrego / ále złe?
A Micheáƺ rzekł : Słuchájże tedy ſłowá Páńſkiego : Widźiałem Páná śiedzącego ná ſtolicy ſwojey / y wƺyſtko wojſko niebieſkie ſtojące po práwicy jego / y po lewicy jego.
Y rzekł Pán : Kto zwiedźie Achábá / áby ƺedł á upádł w Rámot Gáláád? A gdy mówił jeden ták / á drugi inácżey :
Tedy wyſtąpił duch / y ſtánął przed Pánem / mówiąc : Ja go zwiodę. A Pán mu rzekł : Przez cóż?
Odpowiedźiał : Wynijdę / á będę duchem kłámliwym w uſtách wƺyſtkich proroków jego. Y rzekł mu Pán : Zwiedźieƺ / y pewnie przemożeƺ. Idźże / á ucżyń ták.
Przetoż teraz oto dał Pán duchá kłámliwego w uſtá tych wƺyſtkich proroków twojich / gdyż Pán wyrzekł przećiwko tobie złe.
Tedy przyſtąpiwƺy Sedechyjáƺ / ſyn Chenááná / uderzył Micheáƺá w policżek / mówiąc : Kiedyż odƺedł Duch Páńſki odemnie / áby z tobą mówił?
Y odpowiedźiał Micheáƺ : Oto ty ujrzyƺ dniá onego / kiedy wnijdźieƺ do nájſkrytƺej komory / ábyś śię ſkrył.
Y rzekł król Izráelſki : Weźmij Micheáƺá / á wiedź go do Amoná / ſtároſty miejſkiego / y do Joáſá / ſyná królewſkiego.
Y rzecżeƺ : Ták mówi król : Wſádźćie tego mężá do więźieniá ; á dáwájćie mu jeść chleb utrapienia y wodę ućiſku / áż śię wrócę w pokoju.
Ale odpowiedźiał Micheáƺ : Jeſliże śię wróćiƺ w pokoju / tedyć nie mówił Pán przez mię. Nádto rzekł : Słuchájćież wƺyſcy ludźie.
A ták ćiągnął król Izráelſki y Jozáfát / król Judzki / do Rámot Gáláád.
Y rzekł król Izráelſki do Jozáfátá : Odmienię śię / gdy pójdę do bitwy ; ále ty ubierz śię w ƺáty twoje. Y odmienił śię król Izráelſki / á ƺedł ku bitwie.
A król Syryjſki rozkázał był hetmánom / których było trzydźieśći y dwá nád wozámi jego / mówiąc : Nie potykájćie śię áni z máłym / áni z wielkim / tylko z ſámym królem Izráelſkim.
Y ſtáło śię / gdy ujrzeli Jozáfátá hetmáni / co byli nád wozámi / rzekli : Zápráwdę to król Izráelſki ; y obróćili śię przećiwko niemu / chcąc śię z nim potykáć ; ále Jozáfát záwołał.
Wtem obácżywƺy hetmáni / co byli nád wozámi / że nie ten był król Izráelſki / odwróćili śię od niego.
Lecż mąż niektóry ſtrzelił z łuku ná niepewne / y poſtrzelił królá Izráelſkiego miedzy nity y miedzy páncerz ; który rzekł woźnicy ſwemu : Náwróć / á wywieź mię z wojſká ; bom jeſt rániony.
Y wzmogłá śię bitwá dniá onego / á król ſtał ná woźie przećiw Syryjcżykom : potem umárł w wiecżór / á krew ćiekłá z rány jego ná wóz.
Tedy wołał woźny w wojſku / gdy już ſłońce záchodźiło / mówiąc : Wróć śię káżdy do miáſtá ſwego y káżdy do źiemi ſwojey.
A ták umárł król / á odwieźion jeſt do Sámáryi / y pochowano go w Sámáryi.
A gdy umywano wóz w ſádzáwce Sámáryjſkiej / lizáli pſy krew jego / tákże gdy umywano zbroję jego : według ſłowá Páńſkiego / które był powiedźiał.
A inne ſpráwy Achábowe y wƺyſtko / co cżynił / y dom z kośći ſłoniowych / który zbudował / wƺyſtkie też miáſtá / które pobudował / ázaż to nie jeſt ſpiſáne w kronikách o królách Izráelſkich?
Y záſnął Acháb z ojcámi ſwymi / á królował Ochozyjáƺ / ſyn jego / miáſto niego.
A Jozáfát / ſyn Azy / pocżął królowáć nád Judą cżwártego roku zá pánowánia Achábá / królá Izráelſkiego.
A Jozáfát miał trzydźieśći y pięć lat / gdy królowáć pocżął / á dwádźieśćiá y pięć lat królował w Jeruzálemie ; á imię mátki jego było Azubá / córká Sálájowá.
Y chodźił po wƺyſtkiej drodze Azy / ojcá ſwego / á nie odchylał śię od niej / cżyniąc to / co było dobrego przed ocżymá Páńſkimi.
Wƺákże iż wyżyn nie poburzyli / jeƺcże lud ofiárował y kádźił po wyżynách.
Ucżynił też pokój Jozáfát z królem Izráelſkim.
A inne ſpráwy Jozáfátowe / y moc jego / którey dokázywał / y jáko wálcżył / ázaż to nie jeſt nápiſáne w kronikách królów Judzkich?
Ten wyplenił z źiemi oſtátek Sodomcżyków / którzy byli pozoſtáli zá dni Azy / ojcá jego.
Ná ten cżás nie było królá w Edomſkiej źiemi ; tylko ſtároſtá był miáſto królá.
Y náſpráwiał Jozáfát okrętów ná morze / áby chodźiły do Ofir po złoto. Ale nie doƺły ; bo śię rozbiły one okręty w Aſyjongáber.
Rzekł tákże był Ochozyjáƺ / ſyn Achábá / do Jozáfátá : Niech jádą ſłudzy moi z ſługámi twymi w okrętách. Ale niechćiał Jozáfát.
Záſnął tedy Jozáfát z ojcámi ſwymi / y pochowány jeſt z ojcámi ſwymi w mieśćie Dawidá / ojcá ſwego ; á królował Jorám / ſyn jego / miáſto niego.
Ochozyjáƺ / ſyn Achábá / pocżął królowáć nád Izráelem w Sámáryi roku śiedmnáſtego Jozáfátá / królá Judzkiego / y królował nád Izráelem dwá látá.
Y cżynił złe przed ocżymá Páńſkimi / chodząc drogą ojcá ſwego / y drogą mátki ſwej / y drogą Jeroboámá / ſyná Nábátowego / który przywiódł do grzechów Izráelá.
Służył tákże Báálowi / á kłániał mu śię / y pobudzał do gniewu Páná / Bogá Izráelſkiego / według wƺyſtkiego / co cżynił oćiec jego.
