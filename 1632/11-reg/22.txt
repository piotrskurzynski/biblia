A nie było przez trzy látá wojny miedzy Syryjcżykámi / y miedzy Izráelcżykámi.
Y ſtáło śię roku trzećiego / <i>że</i> przyjáchał Jozáfát Król Judſki / do Królá Izráelſkiego.
Tedy rzekł Król Izráelſki do ſług ſwojich : Nie wiećieƺ / iż náƺe jeſt Rámot Gáláád? A my zániedbywamy odebráć go z ręku Królá Syryjſkiego?
Przetoż rzekł do Jozáfátá : Poćiągnieƺże ze mną ná wojnę / przećiwko Rámot Gáláád? Y rzekł Jozáfát do Królá Izráelſkiego : Jákom ja / ták y ty / jáko lud mój / ták lud twój : jáko konie moje / ták konie twoje.
Nád to rzekł Jozáfát do Królá Izráelſkiego : Spytaj śię / proƺę dźiś ſłowá PAńſkiego.
A ták zebrał Król Izráelſki proroków około cżterech ſet mężów / y rzekł do nich : Mamże ćiągnąć ná wojnę przećiwko Rámot Gáláád? cży zániecháć? Y odpowiedźieli mu : Ciągni / bo <i>je</i> PAN da w ręce królewſkie.
Ale Jozáfát rzekł : Niemáƺże tu którego Proroká PAńſkiego / żebyſmy śię go pytáli?
Y rzekł Król Izráelſki do Jozáfátá : Jeſt jeƺcże mąż jeden / przez któregobyſmy śię mogli rádźić PAná ; ále go ja nienawidzę / bo mi nic dobrego nie prorokuje / jedno złe : Micheaƺ / Syn Jemlów. Y rzekł Jozáfát : Niech ták nie mówi Król.
A ták záwołał Król Izráelſki komorniká niektórego / y rzekł ; Przywiedź tu rychło Micheáƺá Syná Jemlowego.
( Miedzy tym Król Izráelſki / y Jozáfát Król Judſki / śiedźieli ná ſtolicách ſwojich / ubráni w ƺáty królewſkie / ná plácu u wrót bramy Sámáryjſkiey. A wƺyſcy prorocy prorokowáli przed nimi.
A Sedechiaƺ Syn Chenáánów / ſpráwił ſobie rogi żelázne / y rzekł : Ták mówi PAN : Tymi będźieƺ bódł Syryjcżyki / áż je wyniƺcżyƺ.
Tákże wƺyſcy prorocy prorokowáli / mówiąc : Ciągni do Rámot Gáláád / á będźieć śię ƺcżęśćiło : ábowiem je poda PAN w ręce królewſkie ).
Tedy poſeł który chodźił áby przyzwał Micheaƺá / rzekł do niego / mówiąc : Oto teraz ſłowá proroków jednymi uſty dobrze tuƺą Królowi / niechże będźie proƺę ſłowo twoje / jáko ſłowo jednego z nich / á mów dobre rzecży.
Y rzekł Micheaƺ ; Żywie PAN / że co mi kolwiek rzecże PAN / to mówić będę.
A gdy przyƺedł do Królá / rzekł Król do niego : Micheaƺu / mamyż ćiągnąć ná wojnę przećiw Rámot Gáláád / cżyli zániecháć? A on mu rzekł : Ciągni / á będźieć śię ƺcżęśćiło ; ábowiem je poda PAN w ręce królewſkie.
Y rzekł do niego Król : A wieleż ćię rázy mam przyśięgą obowiązáć / ábyś mi nie mówił jedno prawdę w Imieniu PAńſkim?
Przetoż rzekł : Widźiałem wƺyſtek lud Izráelſki rozproƺony po górách jáko owce / które nie máją Páſterzá : Bo rzekł PAN ; Nie máją ći Páná / niech śię wróći káżdy do domu ſwego w pokoju.
Y rzekł Król Izráelſki do Jozáfátá : Izażem ći nie powiádał / że mi nie miał prorokowáć dobrego / ále złe?
A <i>Micheaƺ</i> rzekł : Słuchajże tedy ſłowá PAńſkiego : Widźiałem PAná śiedzącego ná ſtolicy ſwojey / y wƺyſtko wojſko niebieſkie ſtojące po práwicy jego / y po lewicy jego.
Y rzekł PAN : Kto zwiedźie Achábá / áby ƺedł á upadł w Rámot Gáláád? A gdy mówił jeden ták / á drugi inák :
Tedy wyſtąpił Duch y ſtánął przed PAnem / mówiąc ; Ja go zwiodę : A PAN mu rzekł / Przez cóż?
Odpowiedźiał : Wynidę / á będę Duchem kłamliwym w uściech wƺyſtkich proroków jego : y rzekł mu <i>PAN</i> : Zwiedźieƺ / y pewnie przemożeƺ. Idźże á ucżyń ták.
Przetoż teraz oto dał PAN Duchá kłamliwego w uſtá tych wƺyſtkich proroków twojich : gdyż PAN wyrzekł przećiwko tobie złe.
Tedy przyſtąpiwƺy Sedechiaƺ / Syn Chenáánów uderzył Micheaƺá w policżek / mówiąc : Kiedyż odƺedł Duch PAńſki ode mnie / áby z tobą mówił?
Y odpowiedźiał Micheaƺ : Oto ty ujrzyƺ dniá onego / kiedy wnidźieƺ do naſkrytƺey komory / ábyś śię ſkrył.
Y rzekł Król Izráelſki : Weźmi Micheaƺá / á wiedź go do Amoná / ſtároſty miejſkiego / y do Joáſá / Syná królewſkiego.
Y rzecżeƺ : Ták mówi Król : Wſadźćie tego <i>mężá</i> do więźienia / á dawajćie mu jeść chleb utrapienia / y wodę ućiſku / áż śię wrócę w pokoju.
Ale odpowiedźiał Micheaƺ : Jeſliże śię wróćiƺ w pokoju / tedyć nie mówił PAN przez mię. Nád to rzekł : Słuchajćież wƺyſcy ludźie.
A ták ćiągnął Król Izráelſki / y Jozáfát Król Judſki / do Rámot Gáláád.
Y rzekł Król Izráelſki do Jozáfátá : Odmienię śię / gdy pójdę do bitwy : ále ty ubierz śię w ƺáty twoje : Y odmienił śię Król Izráelſki / á ƺedł ku bitwie.
A Król Syryjſki rozkazał był Hetmánom / których było trzydźieśći y dwá nád wozámi jego / mówiąc : Nie potykajćie śię áni z máłym / áni z wielkim / tylko z ſámym Królem Izráelſkim.
Y ſtáło śię / gdy ujrzeli Jozáfátá Hetmáni / co byli nád wozámi / rzekli : Záprawdę <i>to</i> Król Izráelſki : y obróćili śię przećiwko niemu / chcąc śię z nim potykáć : Ale Jozáfát záwołał.
W tym obacżywƺy Hetmáni co byli nád wozámi że nie ten był Król Izráelſki / odwróćili śię od niego.
Lecż mąż niektóry ſtrzelił z łuku ná niepewną / y poſtrzelił Królá Izráelſkiego miedzy nity / y miedzy Páncerz : który rzekł woźnicy ſwemu ; Náwróć / á wywieź mię z wojſká / bom jeſt rániony.
Y wzmogłá śię bitwá dniá onego : á Król ſtał ná woźie przećiw Syryjcżykom : potym umárł wiecżór : á krew ćiekłá z rány <i>jego</i> ná wóz.
Tedy wołał woźny w wojſku / gdy już ſłońce záchodźiło / mówiąc : Wróć śię káżdy do miáſtá ſwego / y káżdy do źiemie ſwojey.
A ták umárł Król / á odwieźion jeſt do Sámáryjey / y pochowano go w Sámáryjey.
A gdy umywano wóz w ſádzawce Sámáryjſkiey lizáli pśi krew jego / tákże gdy umywano zbroję jego : według ſłowá PAńſkiego / które był powiedźiał.
A inne ſpráwy Achábowe / y wƺyſtko co cżynił / y dom z kośći Słoniowych który zbudował / wƺyſtkie też miáſtá które pobudował / ázaż to nie jeſt ſpiſano w Kronikách o Królách Izráelſkich?
Y záſnął Acháb z Ojcy ſwymi / á królował Ochoziaƺ Syn jego miáſto niego.
A Jozáfát Syn Azy pocżął Królowáć nád Judą cżwartego roku <i>zá pánowánia</i> Achábá Królá Izráelſkiego.
A Jozáfát miał trzydźieśći y pięć lat gdy królowáć pocżął : á dwádźieśćiá y pięć lat królował w Jeruzalem : á imię mátki jego <i>było</i> Azubá / Córká Sálájowá.
Y chodźił po wƺyſtkiey drodze Azy Ojcá ſwego / á nie odchylał śię od niey / cżyniąc to co było dobrego przed ocżymá PAńſkimi.
Wƺákże yż wyżyn nie poburzyli / jeƺcże lud ofiárował y kádźił po wyżynách.
Ucżynił też pokój Jozáfát z Królem Izráelſkim.
A inne ſpráwy Jozáfátowe / y moc jego / którey dokázował / y jáko walcżył / ázaż to nie jeſt nápiſano w Kronikách Królów Judſkich?
Ten wyplenił z źiemie oſtátek Sodomcżyków / którzy byli pozoſtáli zá dni Azy Ojcá jego.
Ná ten cżás nie <i>było</i> Królá w Edomſkiey źiemi : tylko Stároſtá był <i>miáſto</i> Królá.
Y náſpráwiał Jozáfát Okrętów ná morze / áby chodźiły do Ofir po złoto. Ale nie doƺły / bo śię rozbiły one Okręty w Aſyongáber.
Rzekł tákże był Ochoziaƺ / Syn Achábów do Jozáfátá : Niech jádą ſłudzy moji z ſługámi twymi w okręciech : Ale niechćiał Jozáfát.
Záſnął tedy Jozáfát z Ojcy ſwymi : y pochowány jeſt z Ojcy ſwymi w mieśćie Dawidá Ojcá ſwego / á królował Jorám Syn jego miáſto niego.
Ochoziaƺ Syn Achábów pocżął królowáć nád Izráelem w Sámáryjey roku śiedmnaſtego Jozáfátá / Królá Judſkiego : y królował nád Izráelem dwie lecie.
Y cżynił złe przed ocżymá PAńſkimi / chodząc drogą Ojcá ſwego / y drogą mátki ſwey / y drogą Jeroboámá / Syná Nábátowego / który przywiódł do grzechów Izráelá.
Służył tákże Báálowi / á kłániał mu śię : y pobudzał do gniewu PANA Bogá Izráelſkiego / według wƺyſtkiego co cżynił Oćiec jego.
