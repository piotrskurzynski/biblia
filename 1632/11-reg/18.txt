Potym po wielu dni / <i>miánowićie</i> po onym roku trzećim / ſtáło śię ſłowo PAńſkie do Eliaƺá / mówiąc : Idź ukaż śię Achábowi / bo ſpuƺcżę deƺcż ná źiemię.
Szedł tedy Eliaƺ / áby śię ukazał Achábowi : á był głód gwałtowny w Sámáryjey.
Y záwołał Acháb Abdiaƺá / który był ſpráwcą domu jego. ( A Abdiaƺ śię bárzo PANA bał.
Bo gdy mordowáłá Jezábel Proroki PAńſkie / tedy wźiął Abdiaƺ ſto Proroków / y ſkrył ich po pięćdźieśiąt do jáſkini / y żywił je chlebem y wodą )
Y rzekł Acháb do Abdiaƺá : Idź przez źiemię do wƺyſtkich źrzódeł wód / y do wƺyſtkich potoków / áza gdźie znajdźiemy trawę / żebyſmy żywo záchowáli konie y muły / y żebyſmy nie zgubili bydłá.
Y rozdźielili ſobie źiemię którą przejść mieli : Acháb ſam ƺedł jedną drogą / Abdiaƺ też ƺedł drugą drogą oſobno.
A gdy Abdiaƺ <i>był</i> w drodze / oto śię z nim Eliaƺ potkał / który gdy go poznał / upadł ná oblicże ſwoje / y rzekł : A tyżeś jeſt / Pán mój / Eliaƺ?
Y odpowiedźiał mu : Jam jeſt : Idź powiedz Pánu twemu : Oto Eliaƺ <i>tu jeſt</i>.
Do którego on rzekł : Cóżem zgrzeƺył / iż wydawaƺ ſługę twego w ręce Achábowe / áby mię zábił?
Żywie PAN Bóg twój / że niemáƺ narodu / y króleſtwá / gdźieby nie poſłał PAN mój áby ćię ƺukano : á <i>gdy</i> powiedźiano iż ćię nie máƺ / tedy obowiązał przyśięgą króleſtwá y narody / jáko ćię ználeść nie mogą.
A ty mi teraz mówiƺ : Idź á powiedz Pánu twemu ; Oto Eliaƺ.
Y ſtáłoby śię / gdybym ja odƺedł od ćiebie / żeby ćię Duch PAńſki zánióſł / gdźiebym nie wiedźiał ; á ja ƺedƺy opowiedźiałbym Achábowi : á gdyby ćię nie ználazł / zábiłby mię. A ſługá twój boji śię PANA od dźiećińſtwá ſwego.
Aza nie powiedźiano Pánu memu / com ucżynił gdy mordowáłá Jezábel Proroki PAńſkie / żem ſkrył z Proroków PAńſkich ſto mężów : po piąćidźieśiąt mężów w jáſkini / y żywiłem je chlebem y wodą?
A ty teraz mówiƺ : Idź powiedz Pánu twemu / oto Eliaƺ : y zábije mię.
Y odpowiedźiał Eliaƺ : Żywie PAN zaſtępów / przed którego oblicżnośćią ſtoję / że mu śię dźiś ukażę.
A ták ƺedł Abdiaƺ przećiw Achábowi / y oznájmił mu to : Przetoż ƺedł Acháb przećiw Eliaƺowi.
A ujrzawƺy Acháb Eliaƺá / rzekł Acháb do niego : Azaż nie ty jeſteś / który cżyniƺ zámięƺánie w Izráelu?
Ná co mu odpowiedźiał : Nie jać cżynię zámięƺánie w Izráelu / ále ty y dom Ojcá twego : gdyż opuśćiwƺy rozkazánia PAńſkie náśládujećie Báálów.
Przetoż teraz pośli <i>á</i> zbierz do mnie wƺyſtkiego Izráelá / ná górę Kármel ; y Proroków Báálowych cżtery ſtá y pięćdźieśiąt ; przytym proroków gájowych cżtery ſtá / którzy jadáją z ſtołu Jezábele.
Poſłał tedy Acháb do wƺyſtkich Synów Izráelſkich / y zebrał te proroki ná górę Kármel.
A przyſtąpiwƺy Eliaƺ do wƺyſtkiego ludu / rzekł ; Y długoż będźiećie chramáć ná obie ſtrony? Jesli PAN jeſt Bogiem / idźćież zá nim : á jesli Báál / idźćież zá nim. Y nie odpowiedźiał mu lud y ſłowá.
Tedy rzekł Eliaƺ do ludu ; Jam tylko ſam zoſtał Prorok PAńſki : á proroków Báálowych cżtery ſtá / y pięćdźieśiąt mężów.
Niech nam dádzą dwu ćielców / á niech ſobie obiorą ćielcá jednego / á porąbią go ná ƺtuki / y włożą ná drwá ; ále ogniá niech nie podkładáją : ja też przygotuję drugiego ćielcá / którego włożę ná drwá / á ogniá nie podłożę.
Potym wzywajćie imienia bogów wáƺych / á ja będę wzywał imienia PAńſkiego / á Bóg który śię ozwie przez ogień / ten niech będźie Bogiem : Ná co odpowiádájąc wƺyſtek lud rzekł ; Dobrześ powiedźiał.
Y rzekł Eliaƺ do proroków Báálowych : Obierzćie ſobie ćielcá jednego / á zgotujćie go pierwey / bo was jeſt więcey : Wzywajćież imienia bogów wáƺych / ále ogniá nie podkładajćie.
A ták wźięli ćielcá którego im dał / á zgotowawƺy / wzywáli imienia Báálowego / od poránku áż do południá / mówiąc : O Báálu wyſłuchaj nas! Ale nie było głoſu / áni ktoby odpowiedźiał. Y ſkakáli koło ołtarzá który byli ucżynili.
A gdy było południe / náśmiewał śię z nich Eliaƺ / mówiąc : Wołajćie więkƺym głoſem ; ponieważ jeſt Bóg : Tylko / że śię álbo zámyślił / álbo jeſt zábáwny / álbo też jeſt w drodze / álbo też ſpi ; áza ocući.
A ták wołáli głoſem wielkim / y rzezáli śię według zwycżáju ſwego nożámi / y włócżenkámi / áż śię krwią oblewáli.
Y ſtáło śię gdy minęło południe / że prorokowáli áż do <i>cżáſu</i> ofiárowánia ofiáry śniedney : ále nie było głoſu / áni ktoby odpowiedźiał / áni ktoby wyſłuchał.
Zátym rzekł Eliaƺ do wƺyſtkiego ludu ; Przyſtąpćie do mnie. Y przyſtąpił wƺyſtek lud do niego. Tedy nápráwił Ołtarz PAńſki który był rozwálony.
Abowiem wźiął Eliaƺ dwánaśćie kámieni ; ( według licżby pokolenia Synów Jákóbowych / do którego śię ſtáło ſłowo PAńſkie / mówiąc ; Izráel będźie imię twoje ).
Y zbudował z tego kámieniá Ołtarz w Imię PAńſkie / á ucżynił około Ołtarzá ƺeroki rów / coby mógł dwie miáry zbożá wyśiać.
Potym ułożył drwá / y ná ƺtuki porąbał ćielcá / y ſkładł <i>go</i> ná drwá.
Y rzekł : Nápełnićie cżtery wiádrá wodą / á wylyjćie ná cáłopalenie / y ná drwá. Rzekł nád to : Powtórzćie / y powtórzyli : rzekł jeƺcże : Ucżyńćie po trzećie ; y ucżynili po trzećie :
Ták że płynęły wody około Ołtarzá : áż y rów był nápełniony wodą.
Y ſtáło śię / gdy był cżás ſpráwowánia Ofiáry śniedney / przyſtąpił Eliaƺ Prorok / y rzekł ; PAnie Boże Abráhámów / Izááków / y Izráelów : dźiś niech poznáją / żeś ty jeſt Bogiem w Izráelu / á jam ſługá twój : á / żem według ſłowá twego ucżynił to wƺyſtko.
Wyſłuchaj mię PAnie / wyſłucháj mię / áby poznał ten lud / żeś ty PANIE <i>jeſt</i> BOGIEM / gdybyś záś náwróćił ſercá ich.
Tedy ſpadł ogień PAńſki / y pożárł cáłopalenie / y drwá y kámienie / y proch : á wodę która byłá w rowie / wyſuƺył.
Co gdy ujrzał wƺyſtek lud / upádli ná oblicże ſwe / y rzekli : PAN jeſt Bogiem / PAnći jeſt Bogiem.
Tedy rzekł Eliaƺ do nich : Pojimajćie proroki Báálowe / á żaden niech z nich nie uchodźi. Y pojimano je. A ták odwiódł je Eliaƺ do potoká Ciſon / y támże je pobił.
Potym rzekł Eliaƺ do Achábá : Idź / jedz / á pij ; ábowiem <i>oto</i> ƺum dżdżá wielkiego.
Tedy ƺedł Acháb / áby jadł y pił : á Eliaƺ wſtąpił ná wierzch Kármelu / y położył śię ná źiemię : á włożył twarz ſwoję miedzy koláná ſwoje.
Potym rzekł do ſługi ſwego : Idź teraz á pojrzy ku morzu : który poƺedł / á pojrzawƺy / rzekł ; Niemáƺ nic. Záśię rzekł : <i>Idź á</i> wracaj śię po śiedm kroć.
A zá śiódmym <i>razem</i> rzekł : Oto obłok máły jáko dłoń cżłowiecżá wyſtępuje z morza. Tedy <i>on</i> rzekł ; Idź á powiedz Achábowi : Záprzągaj / á ujeżdżaj áby ćię deƺcż nie záſtał.
Y ſtáło śię miedzy tym / że śię niebioſá obłokámi y wiátrem záćmiły / z kąd był deƺcż wielki. A ták wśiadƺy Acháb jáchał do Jezreelá.
A ręká PAńſka byłá nád Eliaƺem : y przepáſał biodrá ſwe / y bieżał przed Achábem / áż przyƺedł do Jezreelá.
