Tedy Benádád Król Syryjſki zebrał wƺyſtko wojſko ſwoje / májąc z ſobą trzydźieśći y dwu Królów / przytym jezdne y wozy : á przyćiągnąwƺy obległ Sámáryją y dobywał jey.
Y wypráwił poſły do Achábá / Królá Izráelſkiego / do onego miáſtá :
Y rzekł mu : Ták mówi Benádád / śrebro twoje / y złoto twoje / mojeć jeſt : tákże żony twoje y Synowie twoji nacudniejƺy / moji ſą.
Y odpowiedźiał Król Izráelſki / á rzekł : Według ſłowá twego Królu Pánie mój / twojem ja / y wƺyſtko co mam.
A wróćiwƺy śię poſłowie <i>do niego</i> rzekli : Ták powiedźiał Benádád / mówiąc : Poſłałem do ćiebie / ábyć rzecżono ; Śrebro twoje / y złoto twoje / y żony twoje / y Syny twoje daƺ mi.
Ale wiedz że jutro o tym cżáśie poślę ſługi moje do ćiebie / którzy wyƺperáją dom twój / y domy ſług twojich / y wƺyſtko w cżym śię kochaƺ / w ręce ſwe wezmą y rozbiorą.
A ták wezwał Król Izráelſki wƺyſtkich Stárƺych źiemie oney / y rzekł <i>im</i> : Uwáżćież proƺę / á obacżćie żeć ten złego ƺuka : ábowiem poſłał do mnie po żony moje / y po Syny moje / y po śrebro moje / y po złoto moje / á nie odmówiłem mu.
Y rzekli do niego oni wƺyſcy Stárƺy / y wƺyſtek lud : Nie ſłuchaj / áni przyzwalaj.
Przetoż odpowiedźiał poſłom Benádádowym : Powiedzćie Królowi Pánu memu : Wƺyſtko o coś poſłał do ſługi twego przed tym / ucżynię ; ále tey rzecży ucżynić nie mogę. A ták poſłowie odeƺli / y odnieśli mu odpowiedź.
Znowu poſłał do niego Benádád / y rzekł : Niech mi to ucżynią Bogowie / y to niech przycżynią / jeſli śię doſtánie prochu Sámáryjey po gárśći wƺyſtkiemu ludowi który zá mną idźie.
Y odpowiedźiał Król Izráelſki / á rzekł : Powiedzćie <i>mu</i> : Niech śię nie chlubi zbrojny / jáko ten / który złożył zbroję.
A gdy Benádád uſłyƺał to ſłowo / ( á on w ten cżás z Królmi pił w namieciech ) rzekł do ſług ſwych : Ruƺćie śię. Y ruƺyli śię ku miáſtu.
A oto niektóry Prorok przyƺedł do Achábá / Królá Izráelſkiego / y rzekł ; Ták powiáda PAN : Yzażeś nie wiedźiał tego wƺyſtkiego wielkiego mnóſtwá? Oto ja je dam w rękę twoję dźiśia / ábyś wiedźiał żem <i>JA PAN</i>.
Tedy rzekł Acháb : Przez kogoż? A on odpowiedźiał ; Ták mówi PAN / przez ſługi Kśiążąt powiátowych. Y rzekł : Któż pocżnie bitwę? Tedy mu on odpowiedźiał : Ty.
Oblicżył tedy ſługi Kśiążąt powiátowych / których <i>było</i> dwie śćie trzydźieśći y dwá : á po nich policżył wƺyſtek lud / wƺyſtkich Synów Izráelſkich śiedm tyśięcy.
Y wyƺli o południu. A Benádád pił y upił śię w namieciech ſam y trzydźieśći y dwá Królów pomocników jego.
A ták wyƺli ſłudzy Kśiążąt powiátowych naprzód. Tedy poſłał Benádád / ( gdy mu powiedźiano / mówiąc : Mężowie wyƺli z Sámáryjey )
Y rzekł : Choćiáżby o pokój ƺli <i>prośić,</i> pojimajćie je żywo. Choćiáżby też ku bitwie wyƺli / żywo je pojimajćie.
A gdy oni ſłudzy Kśiążąt powiátowych wyƺli z miáſtá / y <i>inne</i> wojſko zá nimi ;
Poráźił káżdy mężá ſwego / ták / że ućiekli Syryjcżycy : y gonił je Izráel : ućiekł też Benádád Król Syryjſki ná koniu / y z jezdnymi.
Potym wyćiągnął Król Izráelſki / y pobił konie y wozy : á poráźił Syryjcżyká porażką wielką.
Znowu przyƺedł Prorok do Królá Izráelſkiego / y rzekł mu : Idź zmacniaj śię / á wiedz y bacż co maƺ cżynić : ábowiem po roku Król Syryjſki wyćiągnie przećiwko tobie.
Tedy ſłudzy Królá Syryjſkiego rzekli do niego : Bogowie ich / ſą Bogowie górni ; przetoż nas przemogli : á wƺákże zwiedźmy z nimi bitwę w polu / <i>á ujrzyƺ</i> jeſli ich nie przemożemy.
Przetoż ták ucżyń : Odpráw Króle / káżdego z miejſcá ſwego / á poſtánów Hetmánów miáſto nich.
A ty nálicż ſobie wojſká z ſwojich / jáko <i>było</i> wojſko onych / którzy polegli / á koni jáko one konie / á wozów jáko one wozy / y ſtocżemy bitwę z nimi w polu / á ujrzyƺ jeſli ich nie przemożemy. Y uſłuchał głoſu ich / á ucżynił ták.
A gdy wyƺedł rok / oblicżył Benádád Syryjcżyki / á ćiągnął ku Afeku ná wojnę przećiw Izráelitom.
Synowie Izráelſcy tákże ſą oblicżeni / á nábrawƺy z ſobą żywnośći / ćiągnęli przećiwko nim : y położyli śię obozem Synowie Izráelſcy przećiwko nim / jákoby dwie máłe ſtadká kóz : á Syryjcżycy nápełnili źiemię.
Tedy przyƺedł mąż Boży / y mówił do Królá Izráelſkiego / á rzekł : Ták mówi PAN : Przeto iż mówili Syryjcżycy / Bogiem gór jeſt PAN / á nie jeſt Bogiem równin / podam to wƺyſtko mnóſtwo wielkie w ręce twoje / ábyśćie wiedźieli żem <i>JA PAN</i>.
A ták oni leżeli obozem przećiwko nim / przez śiedm dni. Y ſtáło śię dniá śiódmego / że ſtocżyli bitwę / y poráźili Synowie Izráelſcy Syryjcżyków ſto tyśięcy pieƺych jednegoż dniá :
A oſtátek ućiekli do Afeku miáſtá / y upadł mur ná dwádźieśćiá y śiedm tyśięcy mężów / co byli pozoſtáli. A Benádád ućiekƺy / przyƺedł do miáſtá / <i>y ſkrył śię</i> do naſkrytƺey komory.
Ale mu rzekli ſłudzy jego : Słycháliſmy zápewne że Królowie domu Izráelſkiego ſą Królowie miłośierni. Niech włożymy / proƺę / wory ná biodrá náƺe / y powrozy ná głowy náƺe / á wynidźiemy do Królá Izráelſkiego / ſnadź żywo zoſtáwi duƺę twoję.
Tedy opáſáli wormi biodrá ſwe / á <i>włożyli</i> powrozy ná głowy ſwoje / y przyƺli do Królá Izráelſkiego / y rzekli : Benádád ſługá twój mówi. Niech żywie proƺę duƺá mojá. A on rzekł : A żywże jeƺcże? Brát to mój.
A oni mężowie wźiąwƺy to zá dobry znák / y prętko uchwyćiwƺy <i>to ſłowo</i> od niego / rzekli / Brát ći twój Benádád : Y rzekł : Idźćież / przywiedźćie go : Przetoż wyƺedł do niego Benádád : y kazał mu wśieść ná wóz.
Y rzekł do niego <i>Benádád</i> : Miáſtá które wźiął Oćiec mój Ojcu twemu / powrócę / á ty ulice ſobie pocżyniƺ w Dámáƺku / jáko pocżynił Oćiec mój w Sámáryjey. <i>Y odpowiedźiał :</i> Ja według przymierza puƺcżę ćię wolno. A ták z nim ucżynił przymierze / y puśćił go wolno.
Tedy niektóry mąż z Synów Prorockich rzekł do bliźniego ſwego z rozkazánia PAńſkiego : Uderz mię proƺę : ále go nie chćiał on mąż uderzyć.
Y rzekł mu : Przeto yżeś nie uſłuchał głoſu PAńſkiego / oto ſkoro odejdźieƺ ode mnie zábije ćię lew. A gdy odƺedł od niego / ználázł go lew / y zábił go.
Potym ználázł drugiego mężá / y rzekł mu : Uderz mię proƺę : który mąż ták go uderzył że go ránił.
Szedł tedy on Prorok / á zábieżał Królowi ná drodze / y odmienił śię záſłoniwƺy ocży ſwoje.
A gdy Król mijał / záwołał ná Królá / y rzekł : Sługá twój wyƺedł w pośrodek bitwy / á oto mąż przyƺedƺy / przywiódł do mnie mężá / y rzekł ; Strzeż tego mężá / bo jeſlibyś go upuśćił / duƺá twojá będźie zá duƺę jego / álbo talent śrebrá odważyƺ.
W tym gdy śię ſługá twój zábáwił tym y owym / oto on zniknął : Tedy rzekł do niego Król Izráelſki : Táki <i>jeſt</i> ſąd twój / ſameś śię oſądźił.
A on záraz odjął zaſłonę od ocżu ſwych / y poznał go Król Izráelſki że był Prorokiem.
Zátym rzekł do niego : Ták mówi PAN : Ponieważeś wypuśćił z ręki ſwey mężá godnego śmierći / duƺá twojá będźie zá duƺę jego / y lud twój / zá lud jego.
Przetoż odƺedł Król Izráelſki / do domu ſwego ſmętny y zágniewány ; y przyƺedł do Sámáryjey.
