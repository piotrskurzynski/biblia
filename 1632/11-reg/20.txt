Tedy Benádád / król Syryjſki / zebrał wƺyſtko wojſko ſwoje / májąc z ſobą trzydźieśći y dwu królów / przytem jezdne y wozy ; á przyćiągnąwƺy obległ Sámáryję y dobywał jey.
Y wypráwił poſły do Achábá / królá Izráelſkiego / do onego miáſtá ;
Y rzekł mu : Ták mówi Benádád : Srebro twoje y złoto twoje mojeć jeſt ; tákże żony twoje y ſynowie twoi nájcudniejśi moi ſą.
Y odpowiedźiał król Izráelſki á rzekł : Według ſłowá twego królu / pánie mój / twojim ja / y wƺyſtko / co mam.
A wróćiwƺy śię poſłowie do niego / rzekli : Ták powiedźiał Benádád / mówiąc : Poſłáłem do ćiebie / ábyć rzecżono : Srebro twoje / y złoto twoje / y żony twoje / y ſyny twoje dáƺ mi.
Ale wiedz / że jutro o tym cżáśie poślę ſługi moje do ćiebie / którzy wyƺperáją dom twój / y domy ſług twojich / y wƺyſtko / w cżym śię kocháƺ / w ręce ſwe wezmą / y rozbiorą.
A ták wezwał król Izráelſki wƺyſtkich ſtárƺych źiemi oney / y rzekł im : Uwáżćież proƺę / á obácżćie / żeć ten złego ƺuká ; ábowiem poſłał do mnie po żony moje / y po ſyny moje / y po ſrebro moje / y po złoto moje / á nie odmówiłem mu.
Y rzekli do niego oni wƺyſcy ſtárśi / y wƺyſtek lud : Nie ſłucháj áni przyzwáláj.
Przetoż odpowiedźiał poſłom Benádádowym : Powiedzćie królowi / pánu memu : Wƺyſtko / o coś poſłał do ſługi twego przed tym / ucżynię : ále tey rzecży ucżynić nie mogę. A ták poſłowie odeƺli / y odnieśli mu odpowiedź.
Znowu poſłał do niego Benádád y rzekł : Niech mi to ucżynią bogowie / y to niech przycżynią / jeſli śię doſtánie prochu Sámáryi po gárśći wƺyſtkiemu ludowi / który zá mną idźie.
Y odpowiedźiał król Izráelſki / á rzekł : Powiedzćie mu : Niech śię nie chlubi zbrojny / jáko ten / który złożył zbroję.
A gdy Benádád uſłyƺał to ſłowo / ( á on w ten cżás z królmi pił w námiotách / ) rzekł do ſług ſwych : Ruƺćie śię. Y ruƺyli śię ku miáſtu.
A oto / niektóry prorok przyƺedł do Achábá / królá Izráelſkiego / y rzekł : Ták powiádá Pán : Izáżeś nie wiedźiał tego wƺyſtkiego wielkiego mnóſtwá? Oto Ja je dam w rękę twoję dźiśiáj / ábyś wiedźiał / żem Ja Pán.
Tedy rzekł Acháb : Przez kogoż? A on odpowiedźiał : Ták mówi Pán : przez ſługi kśiążąt powiátowych. Y rzekł : Któż pocżnie bitwę? Tedy mu on odpowiedźiał : Ty.
Oblicżył tedy ſługi kśiążąt powiátowych / których było dwieśćie trzydźieśći y dwá ; á po nich policżył wƺyſtek lud / wƺyſtkich ſynów Izráelſkich śiedm tyśięcy.
Y wyƺli o południu. A Benádád pił / y upił śię w námiotách / ſám y trzydźieśći y dwu królów / pomocników jego.
A ták wyƺli ſłudzy kśiążąt powiátowych náprzód. Tedy poſłał Benádád / ( gdy mu powiedźiáno / mówiąc : Mężowie wyƺli z Sámáryi / )
Y rzekł : Choćiáżby o pokój ƺli prośić / pojmájćie je żywo ; choćiáżby też ku bitwie wyƺli / żywo je pojmájćie.
A gdy oni ſłudzy kśiążąt powiátowych wyƺli z miáſtá / y inne wojſko zá nimi /
Poráźił káżdy mężá ſwego / ták / że ućiekli Syryjcżycy / y gonił je Izráel ; ućiekł też Benádád / król Syryjſki / ná koniu y z jezdnymi.
Potem wyćiągnął król Izráelſki / y pobił konie y wozy / á poráźił Syryjcżyká poráżką wielką.
Znowu przyƺedł prorok do królá Izráelſkiego / y rzekł mu : Idź / zmácniáj śię / á wiedz y bácż / co maƺ cżynić ; ábowiem po roku król Syryjſki wyćiągnie przećiwko tobie.
Tedy ſłudzy królá Syryjſkiego rzekli do niego : Bogowie ich ſą bogowie górni / przetoż nas przemogli ; á wƺákże zwiedźmy z nimi bitwę w polu á ujrzyƺ / jeſli ich nie przemożemy.
Przetoż ták ucżyń : Odpráw królów / káżdego z miejſcá ſwego / á poſtánów hetmánów miáſto nich.
A ty nálicż ſobie wojſká z ſwojich / jáko było wojſko onych / którzy polegli / á koni jáko one konie / á wozów / jáko one wozy / y ſtocżymy bitwę z nimi w polu / á ujrzyƺ / jeſli ich nie przemożemy. Y uſłuchał głoſu ich / á ucżynił ták.
A gdy wyƺedł rok / oblicżył Benádád Syryjcżyki / á ćiągnął ku Afeku ná wojnę przećiw Izráelitom.
Synowie Izráelſcy tákże ſą oblicżeni / á nábráwƺy z ſobą żywnośći / ćiągnęli przećiwko nim. Y położyli śię obozem ſynowie Izráelſcy przećiwko nim / jákoby dwá máłe ſtádká kóz ; á Syryjcżycy nápełnili źiemię.
Tedy przyƺedł mąż Boży / y mówił do królá Izráelſkiego / á rzekł : Ták mówi Pán : Przeto iż mówili Syryjcżycy : Bogiem gór jeſt Pán / á nie jeſt Bogiem równin / podám to wƺyſtko mnóſtwo wielkie w ręce twoje / ábyśćie wiedźieli / żem Ja Pán.
A ták oni leżeli obozem przećiwko nim przez śiedm dni. Y ſtáło śię dniá śiódmego / że ſtocżyli bitwę / y poráźili ſynowie Izráelſcy Syryjcżyków ſto tyśięcy pieƺych jednegoż dniá.
A oſtátek ućiekli do Afeku miáſtá / y upádł mur ná dwádźieśćiá y śiedm tyśięcy mężów / co byli pozoſtáli. A Benádád ućiekƺy przyƺedł do miáſtá / y ſkrył śię do nájſkrytƺej komory.
Ale mu rzekli ſłudzy jego : Słycháliśmy zá pewne / że królowie domu Izráelſkiego ſą królowie miłośierni. Niech włożymy proƺę wory ná biodrá náƺe / y powrozy ná głowy náƺe / á wynijdźiemy do królá Izráelſkiego / ſnadź żywo zoſtáwi duƺę twoję.
Tedy opáſáli wormi biodrá ſwe / á włożyli powrozy ná głowy ſwoje y przyƺli do królá Izráelſkiego / y rzekli : Benádád / ſługá twój / mówi : Niech żyje proƺę duƺá mojá! A on rzekł : A żywże jeƺcże? Brát to mój.
A oni mężowie wźiąwƺy to zá dobry znák / y prędko uchwyćiwƺy to ſłowo od niego / rzekli : Brátći twój Benádád. Y rzekł : Idźćież / przywiedźćie go. Przetoż wyƺedł do niego Benádád / y kázał mu wśiąść ná wóz.
Y rzekł do niego Benádád : Miáſtá / które wźiął oćiec mój ojcu twemu / powrócę / á ty ulice ſobie pocżyniƺ w Dámáƺku / jáko pocżynił oćiec mój w Sámáryi. Y odpowiedźiał : Ja według przymierza puƺcżę ćię wolno. A ták z nim ucżynił przymierze y puśćił go wolno.
Tedy niektóry mąż z ſynów prorockich rzekł do bliźniego ſwego z rozkázánia Páńſkiego : Uderz mię proƺę ; ále go nie chćiał on mąż uderzyć.
Y rzekł mu : Przeto iżeś nie uſłuchał głoſu Páńſkiego / oto ſkoro odejdźieƺ odemnie / zábije ćię lew. A gdy odƺedł od niego / ználázł go lew / y zábił go.
Potem ználázł drugiego mężá / y rzekł mu : Uderz mię proƺę ; który mąż ták go uderzył / że go ránił.
Szedł tedy on prorok / á zábieżał królowi ná drodze / y odmienił śię / záſłoniwƺy ocży ſwoje.
A gdy król mijał / záwołał ná królá / y rzekł : Sługá twój wyƺedł w pośrodek bitwy / á oto mąż przyƺedƺy przywiódł do mnie mężá / y rzekł : Strzeż tego mężá ; bo jeſlibyś go opuśćił / duƺá twojá będźie zá duƺę jego / álbo tálent ſrebrá odwáżyƺ.
Wtem gdy śię ſługá twój zábáwił tym y owem / oto on zniknął. Tedy rzekł do niego król Izráelſki : Táki jeſt ſąd twój / ſámeś śię oſądźił.
A on záráz odjął záſłonę od ocżu ſwych / y poznał go król Izráelſki / że był prorokiem.
Zátem rzekł do niego : Ták mówi Pán : Poniewáżeś wypuśćił z ręki ſwej mężá godnego śmierći / duƺá twojá będźie zá duƺę jego / y lud twój zá lud jego.
Przetoż odƺedł król Izráelſki do domu ſwego ſmutny y zágniewány / y przyƺedł do Sámáryi.
