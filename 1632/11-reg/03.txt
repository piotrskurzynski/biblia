Y zpowinowáćił śię Sálomon z Fáráonem Królem Egipſkim : bo pojął córkę Fáráonowę / y przyprowádźił ją do miáſtá Dawidowego / áżby dobudował domu ſwego / y domu PAńſkiego / y muru Jeruzalemſkiego w około.
Wƺákże lud ofiárował po górách / przeto / że nie był <i>jeƺcże</i> zbudowány dom Imieniowi PAńſkimu / áż do onych dni.
Y miłował Sálomon PAná / chodząc w przykazániách Dawidá Ojcá ſwego : Tylko że ná górách ofiárował y kádźił.
Szedł tedy Król do Gábáon / áby tám ofiárował ; bo <i>tám</i> byłá górá nawiękƺa : Tyśiąc ofiar cáłopalonych ofiárował Sálomon ná onym Ołtarzu.
Y ukazał śię PAN w Gábáon Sálomonowi przez ſen w nocy / y rzekł <i>Bóg</i> : Proś cżego chceƺ / <i>á</i> damći.
Tedy rzekł Sálomon. Tyś ucżynił z ſługą twojim Dawidem / Ojcem mojim / miłośierdźie wielkie / gdyż chodźił przed tobą w prawdźie y w ſpráwiedliwośći / y w proſtośći ſercá <i>ſtał</i> przy tobie : y záchowałeś mu to miłośierdźie wielkie / yżeś mu dał Syná któryby śiedźiał ná ſtolicy jego / jáko śię <i>to</i> dźiś <i>okázuje</i>.
A teraz o PAnie Boże mój / tyś poſtánowił Królem ſługę twego miáſto Dawidá Ojcá mego / á jam jeſt dźiećię máłe / y nie umiem wychodźić áni wchodźić.
A ſługá twój jeſt w pośrzodku ludu twego / któryś obrał : ludu wielkiego / który nie może zlicżony áni poráchowány być przez mnóſtwo.
Przetoż daj ſłudze twemu ſerce rozumne / áby ſądźił lud twój / y áby rozeznawał miedzy dobrym y złym : ábowiem któż może ſądźić ten lud twój ták wielki?
Y podobáło śię to PAnu / że żądał Sálomon tey rzecży.
Tedy rzekł Bóg do niego : Dla tego żeś oto prośił / á nie żądałeś ſobie długich dni / ániś żądał ſobie bogactw / ániś prośił o wytrácenie nieprzyjaćiół twojich : áleś ſobie prośił o rozum / dla rozeznánia ſądu :
Otożem ucżynił według ſłów twojich : Otomći dał ſerce mądre y rozumne : ták / yż żaden równy tobie nie był przed tobą / áni po tobie powſtánie równy tobie.
Do tego / y to cżegoś nie żądał dałemći : to jeſt bogáctwá y ſławę / ták áby nie było żadnego tobie równego miedzy Królmi po wƺyſtkie dni twoje.
A będźieƺli chodźił drogámi mojimi / ſtrzegąc wyroków mojich y rozkazánia mego / jáko chodźił Dawid Oćiec twój : tedy przedłużę dni twoje.
A gdy śię ocućił Sálomon / <i>zrozumiał że</i> to był ſen : y przyƺedł do Jeruzalem : á ſtánąwƺy przed ſkrzynią przymierza PAńſkiego / ſpráwował cáłopalenia / y ofiárował ofiáry ſpokojne : ſpráwił też ucżtę ná wƺyſtkie ſługi ſwoje.
Tedy przyƺły dwie niewieśćie wƺetecżnice do Królá / y ſtánęły przed nim.
Y rzekłá jedná z onych niewiaſt : Proƺę Pánie mój / ja y tá niewiáſtá mieƺkamy w jednym domu / y porodźiłám u niey w tymże domu :
Y ſtáło śię dniá trzećiego po porodzeniu mojim / że porodźiłá y tá niewiáſtá : y byłyſmy poſpołu / á nie było nikogo obcego z námi w domu / oprócż nas dwu w tymże domu.
Y umárł Syn tey niewiáſty w nocy / przeto yż go byłá przyległá.
A wſtawƺy o północy / wźięłá Syná mego ode mnie / gdy ſłużebnicá twojá ſpáłá / y położyłá go ná łonie ſwojim / á Syná ſwego umárłego położyłá ná łonie mojim.
A gdym wſtáłá ráno chcąc dáć ſáć Synowi memu / otom <i>ználázłá</i> umárłego : któremu gdym śię ráno przypátrzyłá / á oto nie był Syn mój któregom porodźiłá.
Y rzekłá oná drugá niewiáſtá : Nie ták : ále Syn mój <i>jeſt</i> ten żywy / á Syn twój ten umárły : Ale oná rzekłá : Nie / ále Syn twój <i>jeſt</i> ten umárły / á Syn mój ten żywy : Y ták śię ſpieráły przed Królem.
Y rzekł Król : Tá mówi / ten żywy jeſt Syn mój / á Syn twój ten umárły : á tá záśię mówi : Nie ták ; ále Syn twój ten umárły / á Syn mój ten żywy.
Przetoż rzekł Król : Przynieśćie mi miecż. Y przynieśiono miecż przed Królá.
Tedy rzekł Król : Rozetnićie to żywe dźiećię ná dwoje / á dajćie połowicę jedney / á połowicę drugiey.
Ale niewiáſtá którey był ten Syn żywy / mówiłá do Królá / ( bo śię były poruƺyły wnętrznośći jey / nád Synem jey ) y rzekłá : Proƺę Pánie mój / dajćie jey to dźiećię żywe / á żadnym ſpoſobem niezábijajćie go. Ale drugá rzekłá : Niech nie będźie áni mnie / áni tobie : Rozetnićie <i>je</i>.
Tedy odpowiedźiał Król / y rzekł : Dajćież tey dźiećię żywe / á żadną miárą nie zábijajćie go : Táć jeſt mátká jego.
A uſłyƺawƺy wƺyſtek lud Izráelſki ten ſąd który oſądźił Król / bali śię Królá : ábowiem wiedźieli / że mądrość Boża byłá w ſercu jego / ku cżynieniu ſądu.
