Y ſpowinowáćił śię Sálomon z Fáráonem / królem Egipſkim ; bo pojął córkę Fáráonową / y przyprowádźił ją do miáſtá Dawidowego / áżby dobudował domu ſwego / y domu Páńſkiego / y muru Jeruzálemſkiego w około.
Wƺákże lud ofiárował po górách / przeto / że nie był jeƺcże zbudowány dom imieniowi Páńſkimu áż do onych dni.
Y miłował Sálomon Páná / chodząc w przykázániách Dawidá / ojcá ſwego ; tylko że ná górách ofiárował y kádźił.
Szedł tedy król do Gábáon / áby tám ofiárował ; bo tám byłá górá nájwiękƺá ; tyśiąc ofiár cáłopálonych ofiárował Sálomon ná onym ołtárzu.
Y ukázał śię Pán w Gábáon Sálomonowi przez ſen w nocy / y rzekł Bóg : Proś cżego chceƺ / á dam ći.
Tedy rzekł Sálomon : Tyś ucżynił z ſługą twoim Dawidem / ojcem moim / miłośierdźie wielkie / gdyż chodźił przed tobą w práwdźie y w ſpráwiedliwośći / y w proſtośći ſercá ſtał przy tobie ; y záchowáłeś mu to miłośierdźie wielkie / iżeś mu dał ſyná / któryby śiedźiał ná ſtolicy jego / jáko śię to dźiś okázuje.
A teraz / o Pánie Boże mój / tyś poſtánowił królem ſługę twego miáſto Dawidá / ojcá mego / á jam jeſt dźiećię máłe / y nie umiem wychodźić áni wchodźić.
A ſługá twój jeſt w pośrzodku ludu twego / któryś obrał / ludu wielkiego / który nie może zlicżony áni poráchowány być przez mnóſtwo.
Przetoż dáj ſłudze twemu ſerce rozumne / áby ſądźił lud twój / y áby rozeznáwał miedzy dobrem y złem ; ábowiem któż może ſądźić ten lud twój ták wielki?
Y podobáło śię to Pánu / że żądał Sálomon tey rzecży.
Tedy rzekł Bóg do niego : Dla tego / żeś o to prośił / á nieżądáłeś ſobie długich dni / ániś żądał ſobie bogáctw / ániś prośił o wytrácenie nieprzyjáćiół twojich / áleś ſobie prośił o rozum dla rozeznánia ſądu :
Otożem ucżynił według ſłów twojich ; otom ći dał ſerce mądre y rozumne / ták / iż żáden równy tobie nie był przed tobą / áni po tobie powſtánie równy tobie.
Do tego y to / cżegoś nie żądał / dáłem ći / to jeſt bogáctwá y ſłáwę / ták áby nie było żádnego tobie równego miedzy królmi po wƺyſtkie dni twoje.
A będźieƺli chodźił drogámi mojemi / ſtrzegąc wyroków mojich / y rozkázánia mego / jáko chodźił Dawid / oćiec twój / tedy przedłużę dni twoje.
A gdy śię ocućił Sálomon / zrozumiał / że to był ſen. Y przyƺedł do Jeruzálemu / á ſtánąwƺy przed ſkrzynią przymierza Páńſkiego / ſpráwował cáłopalenia / y ofiárował ofiáry ſpokojne / ſpráwił też ucżtę ná wƺyſtkie ſługi ſwoje.
Tedy przyƺły dwie niewiáſty wƺetecżnice do królá / y ſtánęły przed nim.
Y rzekłá jedná z onych niewiáſt : Proƺę pánie mój / ja y tá niewiáſtá mieƺkámy w jednym domu / y porodźiłám u niej w tymże domu.
Y ſtáło śię dniá trzećiego po porodzeniu mojim / że porodźiłá y tá niewiáſtá ; y byłyśmy poſpołu / á nie było nikogo obcego z námi w domu / oprócż nas dwu w tymże domu.
Y umárł ſyn tey niewiáſty w nocy / przeto / iż go byłá przyległá.
A wſtáwƺy o północy / wźięłá ſyná mego odemnie / gdy ſłużebnicá twojá ſpáłá / y położyłá go ná łonie ſwojim / á ſyná ſwego umárłego położyłá ná łonie mojim.
A gdym wſtáłá ráno / chcąc dáć ſſáć ſynowi memu / otom ználázłá umárłego ; któremu gdym śię ráno przypátrzyłá / á oto nie był ſyn mój / któregom porodźiłá.
Y rzekłá oná drugá niewiáſtá : Nie ták ; ále ſyn mój jeſt ten żywy / á ſyn twój ten umárły. Ale oná rzekłá : Nie ; ále ſyn twój jeſt ten umárły / á ſyn mój ten żywy. Y ták śię ſpieráły przed królem.
Y rzekł król : Tá mówi : Ten żywy jeſt ſyn mój / á ſyn twój ten umárły ; á tá záśię mówi : Nie ták ; ále ſyn twój ten umárły / á ſyn mój ten żywy.
Przetoż rzekł król : Przynieśćie mi miecż. Y przynieśiono miecż przed królá.
Tedy rzekł król : Rozetnijćie to żywe dźiećię ná dwoje / á dájćie połowę jedney / á połowę drugiej.
Ale niewiáſtá / którey był ten ſyn żywy / mówiłá do królá / ( bo śię były poruƺyły wnętrznośći jey nád ſynem jey / ) y rzekłá : Proƺę / pánie mój / dájćie jey to dźiećię żywe / á żádnym ſpoſobem nie zábijájćie go. Ale drugá rzekłá : Niech nie będźie áni mnie / áni tobie / rozetnijćie je.
Tedy odpowiedźiał król / y rzekł : Dájćież tey dźiećię żywe / á żádną miárą nie zábijájćie go ; táć jeſt mátká jego.
A uſłyƺáwƺy wƺyſtek lud Izráelſki ten ſąd / który oſądźił król / báli śię królá : ábowiem wiedźieli / że mądrość Boża byłá w ſercu jego ku cżynieniu ſądu.
