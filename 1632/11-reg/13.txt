A oto mąż Boży przyƺedł z Judſtwá z ſłowem PAńſkim / do Bethel / gdy Jeroboám ſtał u Ołtarzá / áby kádźił.
Y záwołał przećiw Ołtarzowi ſłowem PAńſkim / y rzekł : Ołtarzu / Ołtarzu / ták mówi PAN : Oto Syn národźi śię domowi Dawidowemu / imieniem Jozyaƺ : który będźie ofiárował ná tobie Kápłany wyżyn / kádzące ná tobie / y kośći ludzkie popalą ná tobie.
Y dał <i>mu</i> znák dniá onegoż mówiąc : Tenći jeſt znák że <i>to</i> mówił PAN : Oto śię Ołtarz rozpádnie / á wyſypie śię popiół / który <i>jeſt</i> ná nim.
A gdy uſłyƺał Król Jeroboám ſłowo mężá Bożego / które obwoływał przećiw Ołtarzowi w Bethel / zćiągnął rękę ſwą od Ołtarzá / mówiąc : Pojimajćie go. Y uſchłá ręká jego / którą był wyćiągnął przećiw niemu / á niemógł jey przyćiągnąć do śiebie.
Ołtarz śię też rozpadł : á wyſypał śię popiół z Ołtarzá / według znáku który był dał mąż Boży ſłowem PAńſkim.
Przetoż odpowiádájąc Król / rzekł do mężá Bożego : Proƺę ćię / proś oblicża PANA Bogá twego / á módl śię zá mną / áby śię wróćiłá ręká mojá do mnie. Y modlił śię mąż Boży oblicżu PAńſkiemu : y wróćiłá śię ręká Królewſka do niego / y byłá jáko pierwey.
Tedy rzekł Król do mężá Bożego : Pódź ze mną do domu / ábyś śię pośilił / á damći upominek.
Ale rzekł mąż Boży do Królá : Byś mi dał połowicę domu twego / niepojádę z tobą : áni będę jadł chlebá / áni będę pił wody ná tym miejſcu.
Bo mi ták PAN rozkazał ſłowem ſwojim / mówiąc : Nie będźieƺ jadł chlebá / áni będźieƺ pił wody / áni śię wróćiƺ tą drogą / którąś przyƺedł.
Odƺedł tedy inƺą drogą : á nie wróćił śię tą drogą którą był przyƺedł do Bethel.
A prorok niejáki ſtáry mieƺkał w Bethel : którego Syn przyƺedƺy / opowiedźiał mu wƺyſtkę ſpráwę / którą był ucżynił onegoż dniá mąż Boży w Bethel : <i>y</i> ſłowá które mówił do Królá / opowiedźieli Ojcu ſwemu.
Y rzekł im Oćiec ich : Którąż drogą poƺedł? Y pokazáli Synowie jego drogę / którą poƺedł on mąż Boży / który był przyƺedł z Judſtwá.
Zátym rzekł Synom ſwym : Ośiodłajćie mi oſłá : y ośiodłáli mu oſłá / y wśiadł nań :
Y jáchał zá mężem Bożym : á ználazł go śiedzącego pod dębem / y rzekł mu : Tyżeś jeſt on mąż Boży / któryś przyƺedł z Judſtwá? á on rzekł : Jeſtem.
Y rzekł do niego : Pódź zemną do domu / żebyś jadł chleb.
Ale mu on rzekł : Nie mogę śię wróćić z tobą áni iść z tobą / áni będę jadł chlebá / áni będę pił wody z tobą / ná tym miejſcu.
Bo śię ſtáłá do mnie mowá ſłowem PAńſkim : Nie będźieƺ tám jadł chlebá / áni pił wody / áni śię wróćiƺ idąc tą drogą / którąś ƺedł.
Któremu <i>on</i> odpowiedźiał : Y jam Prorok jáko y ty : Anjoł też rzekł do mnie ſłowem PAńſkim / mówiąc : Wróć go z ſobą do domu twego / áby jadł chleb / y pił wodę : y <i>ták</i> ſkłamał przed nim.
Y wróćił śię z nim / á jadł chleb w domu jego / y pił wodę.
A gdy śiedźieli u ſtołu / ſtáło śię ſłowo PAńſkie do Proroká / który go był wróćił.
Y záwołał ná mężá Bożego / który był przyƺedł z Judſtwá / mówiąc : Ták mówi PAN : Przeto żeś był odpornym uſtom PAńſkim / á nie ſtrzegłeś rozkazánia któreć przykazał PAN Bóg twój :
Aleś śię wróćił / y jadłeś chleb / á piłeś wodę ná miejſcu / o którymemći był rzekł / Nie będźieƺ <i>tám</i> jadł chlebá / áni pił wody : Nie będźie pochowány trup twój w grobie Ojców twojich.
A ták gdy śię nájadł chlebá y nápił śię / ośiodłał oſłá Prorokowi onemu / którego był wróćił.
A gdy odjáchał / potkał go Lew w drodze / y zábił go : á trup jego był porzucony w drodze / á ośieł ſtał wedle niego / Lew tákże ſtał podle trupá.
A oto mężowie mimo idący / ujrzeli trupá porzuconego ná drodze / y Lwá ſtojącego podle niego : którzy przyƺedƺy / powiedźieli <i>to</i> w mieśćie / w którym on ſtáry Prorok mieƺkał.
Co gdy uſłyƺał on Prorok / który go był wróćił z drogi / rzekł : Mąż Boży jeſt / który był odpornym uſtam PAńſkim / przetoż podał go PAN Lwowi / który go potárł / y zábił go według ſłowá PAńſkiego / które mu był powiedźiał.
Nád to rzekł do Synów ſwojich / mówiąc : Ośiodłajćie mi Oſłá : y ośiodłáli.
A wyjáchawƺy ználazł trupá jego porzuconego ná drodze / á Oſłá y Lwá ſtojące przy trupie / <i>ále</i> nie jadł Lew onego trupá / áni obráźił oſłá.
Tedy wźiął Prorok trupá mężá Bożego / á włożywƺy go ná Oſłá / przywiózł go : Y przyƺedł do miáſtá ſwego áby płákał y pogrzebł go.
A położył trupá jego w grobie ſwojim / y płákáli go / mówiąc : Ach / bráćie mój.
A pochowawƺy go / rzekł do Synów ſwojich : Gdy ja umrę / pochowajćie mię w tym grobie / w którym jeſt mąż Boży pochowány : podle kośći jego / połóżćie kośći moje.
Bo zápewne śię to ſtánie / co obwołał ſłowem PAńſkim przećiw Ołtarzowi / który jeſt w Bethel : y przećiwko wƺyſtkim domom wyżyn / które <i>ſą</i> w mieśćiech Sámáryjſkich.
To gdy śię ſtáło / <i>przećię</i> śię nie odwróćił Jeroboám od drogi ſwey złey : ále znowu nacżynił z poſpolitego ludu Kápłanów wyżyn : kto <i>jedno</i> chćiał / poświącał rękę jego / áby był Kápłanem wyżyn.
Y byłá tá rzecż domowi Jeroboámowemu  <i>przycżyną</i> do grzechu / áby był wykorzeniony y wygłádzony z źiemie.
