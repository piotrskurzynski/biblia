A gdy śię przybliżał cżás śmierći Dawidowej / rozkázał Sálomonowi / ſynowi ſwemu / mówiąc :
Ja idę w drogę wƺyſtkiej źiemi / á ty zmácniáj śię / y bądź mężem.
Záchowywáj uſtáwy Páná / Bogá twego / ábyś chodźił drogámi jego / y przeſtrzegał wyroków jego / y przykázáń jego y ſądów jego / y świádectw jego / jáko nápiſano w zakonie Mojżeƺowym / ábyć śię ƺcżęśćiło wƺyſtko / co ſpráwowáć będźieƺ / y we wƺyſtkim / do cżego śię obróćiƺ :
Żeby utwierdźił Pán ſłowo ſwoje / które rzekł do mnie / mówiąc : Jeſli będą ſtrzedz ſynowie twoi drogi ſwej / chodząc przedemną w práwdźie / ze wƺyſtkiego ſercá ſwego / y ze wƺyſtkiej duƺy ſwojey / tedy nie będźie wytrácony po tobie mąż z ſtolicy Izráelſkiej.
A ty też wieƺ / co mi ucżynił Joáb ſyn Sárwii / co ucżynił dwom hetmánom wojſk Izráelſkich / Abnerowi / ſynowi Nerowemu / y Amáźie / ſynowi Jeterowemu / że je pozábijał / á wylał krew jáko ná wojnie cżáſu pokoju / y zmázał krwią jáko ná wojnie pás ſwój rycerſki / który miał ná biodrách ſwojich / y bóty ſwoje / które miał ná nogách ſwojich.
Ucżyniƺ tedy według mądrośći twojey / á nie dopuśćiƺ zejść ſędźiwośći jego w pokoju do grobu.
Ale nád ſynámi Bárſylái Gáláádcżyká użyjeƺ miłośierdźiá / á niech jádáją weſpół z inƺymi u ſtołu twego ; ábowiem oni tákimże ſpoſobem przyƺli do mnie / gdym ućiekał przed Abſálomem / brátem twoim.
Oto też jeſt u ćiebie Semej / ſyn Giery / ſyná Jemini / z Báhurym / który mi też złorzecżył złorzecżeniem wielkiem w on dźień / gdym ƺedł do Máhánáim ; á wƺákże záƺedł mi drogę u Jordánu / y przyśiągłem mu przez Páná / mówiąc : Nie zámorduję ćię miecżem.
Teraz jednák nie przepuƺcżáj mu tego / á iżeś jeſt mężem mądrym / będźieƺ wiedźiał / co mu maƺ ucżynić / ábyś wprowádźił ſędźiwość jego ze krwią do grobu.
Zátem záſnął Dawid z ojcy ſwoimi / á pogrzebiony jeſt w mieśćie Dawidowem.
A dni / których królował Dawid nád Izráelem / było cżterdźieśći lat. W Hebronie królował śiedm lat / á w Jeruzálemie królował trzydźieśći y trzy látá.
A ták Sálomon uśiádł ná ſtolicy Dawidá ojcá ſwego / y zmocniło śię bárzo króleſtwo jego.
Tedy przyƺedł Adonijáƺ / ſyn Hággity / do Betſáby / mátki Sálomonowej / któremu oná rzekłá : A ſpokojneż jeſt przyśćie twoje? A on odpowiedźiał : Spokojne.
Nádto rzekł : Mam nieco mówić z tobą. A oná rzekłá : Mów.
Tedy rzekł : Ty wieƺ / iż moje było króleſtwo / á ná mię obróćili byli wƺyſcy Izráelcżycy twárz ſwoję / ábym królował ; ále przenieśione jeſt króleſtwo / y doſtáło śię brátu memu ; bo mu od Páná náznácżone było.
Przetoż ćię teraz proƺę o jednę rzecż / á nie odmáwiáj mi tego. A oná mu rzekłá : Mów.
Zátem on rzekł : Mów proƺę do Sálomoná królá / ( bo wiem / żeć nie odmówi / ) áby mi dał Abiſág Sunámitkę zá żonę.
Y odpowiedźiáłá Betſábá : Dobrze ; będę mówiłá o ćię z królem.
A ták ƺłá Betſábá do królá Sálomoná / áby z nim mówiłá zá Adonijáƺem ; y wſtał król przećiwko niej / á pokłoniwƺy śię jey uśiádł ná ſtolicy ſwej ; kázał też poſtáwić ſtolicę mátce ſwej / która śiádłá po práwicy jego.
Y rzekłá : Proƺę ćię o jednę máłą rzecż / nie odmáwiáj mi. Y odpowiedźiał jey król : Proś mátko mojá ; ábowiem ći nie odmówię.
Tedy rzekłá : Niech będźie dáná Abiſág Sunámitká Adonijáƺowi / brátu twemu / zá żonę.
Lecż odpowiedźiał król Sálomon / y rzekł mátce ſwojey : Przecżże prośiƺ o Abiſág Sunámitkę Adonijáƺowi? uproś mu y króleſtwo / ábowiem on jeſt brátem moim ſtárƺym nád mię / á ma po ſobie Abijátárá kápłáná / y Joábá / ſyná Sárwii.
Y przyśiągł król Sálomon przez Páná / mówiąc : To mi niech ucżyni Bóg / y to niech przycżyni / że przećiwko duƺy ſwej mówił Adonijáƺ te ſłowá.
A teraz jáko żywy Pán / który mię utwierdźił / y poſádźił ná ſtolicy Dawidá ojcá mojego / y który mi zbudował dom / jáko obiecał / iż dźiś zábity będźie Adonijáƺ.
A ták poſłał król Sálomon Bánájáſá / ſyná Jojádowego / który śię nań tárgnął / y zábił go.
A do Abijátárá kápłáná rzekł król : Idź do Anátot / do ośiádłośći twojey / ábowiemeś mężem śmierći ; wƺákże ćię dźiś nie zábiję / gdyżeś nośił ſkrzynię Páńſką przed Dawidem / ojcem moim / á iżeś to wƺyſtko ćierpiał / cżym był trápiony oćiec mój.
Y wyrzućił Sálomon Abijátárá / áby nie był kápłánem Páńſkim / żeby śię wypełniło ſłowo Páńſkie / które był wyrzekł nád domem Heli w Sylo.
Tá wieść gdy przyƺłá do Joábá / ( ábowiem Joáb przeſtáwał z Adonijáƺem / choćiáż z Abſálomem nie przeſtáwał / ) tedy ućiekł Joáb do namiotu Páńſkiego / á uchwyćił śię rogów ołtárzá.
Y oznájmiono królowi Sálomonowi / że ućiekł Joáb do namiotu Páńſkiego / á że jeſt u ołtárzá : tedy poſłał Sálomon Bánájáſá / ſyná Jojádowego / mówiąc : Idź / zábij go.
A przyƺedƺy Bánájás do namiotu Páńſkiego / rzekł do niego : Ták mówi król : Wynijdź. Który odpowiedźiał : Nie wyjdę / ále tu umrę. Y odnióſł to Bánájás królowi mówiąc : Ták mówił Joáb / y ták mi odpowiedźiał.
Y rzekł mu król : Ucżyńże / jáko mówił / á zábij go / y pogrzeb go / á odejmieƺ krew niewinną / którą wylał Joáb / odemnie y od domu ojcá mego.
A obróći Pán krew jego ná głowę jego : ábowiem tárgnął śię ná dwu mężów ſpráwiedliwƺych y lepƺych niżli ſám / y zábił je miecżem / á oćiec mój Dawid nie wiedźiał o tym : Abnerá / ſyná Nerowego / hetmáná wojſká Izráelſkiego / y Amázę / ſyná Jeterowego / hetmáná wojſká Judzkiego.
A ták wróći śię krew ich ná głowę Joábowę / y ná głowę náśieniá jego ná wieki ; lecż Dawidowi y náśieniu jego / y domowi jego / y ſtolicy jego niech będźie pokój áż ná wieki od Páná.
Szedł tedy Bánájás / ſyn Jojádá / á rzućiwƺy śię nań zábił go / y pogrzebiony jeſt w domu ſwym ná puƺcży.
Y poſtánowił król Bánájáſá ſyná Jojádowego / miáſto niego nád wojſkiem / á Sádoká kápłáná poſtánowił król miáſto Abijátárá.
Potem poſłał król / y przyzwał Semejego / y rzekł mu : Zbuduj ſobie dom w Jeruzálemie / y mieƺkáj tám / á nie wychodź z támtąd nigdźie ;
Bo któregobyś dniá wyƺedł á przyƺedł zá potok Cedron / wiedz wiedząc / że pewnie umrzeƺ ; krew twojá będźie ná głowę twoję.
Tedy rzekł Semej do królá : Dobre jeſt to ſłowo ; jáko mówił król / pán mój / ták ucżyni ſługá twój / Y mieƺkał Semej w Jeruzálemie przez wiele dni.
Y ſtáło śię po trzech lat / że ućiekli dwáj ſłudzy Semejemu do Achiſá ſyná Mááchy / królá Gietſkiego / y opowiedźiáno Semejemu / mówiąc : Oto ſłudzy twoi ſą w Giet.
Przetoż wſtáwƺy Semej / y ośiodłáwƺy oſłá ſwego / jechał do Giet / do Achiſá / áby ƺukał ſług ſwojich ; y wróćił śię Semej y przywiódł ſługi ſwe z Giet.
Y oznájmiono Sálomonowi / że był wyjechał Semej z Jeruzálemu do Giet / y záśię śię wróćił.
Tedy poſłał król / y wezwał Semejego / y rzekł mu : Izálim ćię nie poprzyśiągł przez Páná / á nie oświádcżyłem śię przed tobą / mówiąc : Któregobyśkolwiek dniá gdźie wyƺedł / wiedz wiedząc / że zápewne umrzeƺ? Y mówiłeś do mnie : Dobre to ſłowo / którem ſłyƺał.
Przecżżeś tedy nie ſtrzegł przyśięgi Páńſkiej y przykázánia / którem ći był przykázał?
Nádto król rzekł do Semejego : Ty wieƺ wƺyſtko złe / którego świádome jeſt ſerce twoje / coś ucżynił Dawidowi / ojcu memu / y oddał Pán złość twoję ná głowę twoję.
Ale król Sálomon błogoſłáwiony / ſtolicá Dawidowá będźie utwierdzoná przed Pánem áż ná wieki.
A ták rozkázał król Bánájáſowi / ſynowi Jojádowemu / który wyƺedƺy tárgnął śię nań / y zábił go. A ták utwierdzone jeſt króleſtwo w ręce Sálomonowej.
