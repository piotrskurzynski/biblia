A gdy śię przybliżał cżás śmierći Dawidowey / rozkazał Sálomonowi Synowi ſwemu / mówiąc :
Ja idę w drogę wƺyſtkiey źiemie / á <i>ty</i> zmacniaj śię y bądź mężem.
Záchowywaj uſtáwy PANA Bogá twego / ábyś chodźił drogámi jego / y przeſtrzegał wyroków jego / y przykazań jego / y ſądów jego / y świádectw jego / jáko nápiſano w zakonie Mojzeƺowym : ábyć śię ƺcżęśćiło wƺyſtko / co ſpráwowáć będźieƺ / y we wƺyſtkim do cżego śię obróćiƺ :
Żeby utwirdźił PAN ſłowo ſwoje które rzekł do mnie / mówiąc : Jeſli będą ſtrzedz Synowie twoji drogi ſwey / chodząc przede mną w prawdźie / ze wƺyſtkiego ſercá ſwego / y ze wƺyſtkiey duƺe ſwojey : tedy nie będźie wytrácony <i>po</i> tobie mąż z ſtolice Izráelſkiey.
A ty też wieƺ / co mi ucżynił Joáb Syn Sárwijey / co ucżynił dwiemá Hetmánom wojſk Izráelſkich / Abnerowi Synowi Nerowemu / y Amáźie Synowi Jeterowemu : że je pozábijał / á wylał krew jáko ná wojnie cżáſu pokoju / y zmázał krwią jáko ná wojnie pás ſwój rycerſki / który miał ná biodrách ſwojich / y bóty ſwoje które <i>miał</i> ná nogách ſwojich.
Ucżyniƺ tedy według mądrośći twojey : á niedopuśćiƺ zejść ſędźiwośći jego w pokoju do grobu.
Ale nád Synámi Bárſyláje Gáláádcżyká / użyjeƺ miłośierdźia / á niech jadáją weſpół z inƺymi u ſtołu twego : ábowiem oni tákimże ſpoſobem przyƺli do mnie / gdym ućiekał przed Abſálomem brátem twojim.
Oto też jeſt u ćiebie Semej Syn Gery / Syná Jemini z Báhurym który mi też złorzecżył złorzecżeniem wielkiem w on dźień gdym ƺedł do Máhánáim : á wƺákże záƺedł mi drogę u Jordanu / y przyśiągłem mu przez PAná / mówiąc : Nie zámorduję ćię miecżem.
Teraz jednák nie przepuƺcżaj mu <i>tego</i> á yżeś jeſt mężem mądrym / będźieƺ wiedźiał / co mu maƺ ucżynić : ábyś wprowádźił ſędźiwość jego ze krwią do grobu.
Zátym záſnął Dawid z Ojcy ſwojimi / á pogrzebiony jeſt w mieśćie Dawidowym.
A dni których Królował Dawid nád Izráelem / <i>było</i> cżterdźieśći lat : w Hebronie Królował śiedm lat : á w Jeruzalem Królował trzydźieśći y trzy látá.
A ták Sálomon uśiadł ná ſtolicy Dawidá Ojcá ſwego : y zmocniło śię bárzo Króleſtwo jego.
Tedy przyƺedł Adoniaƺ Syn Hággity / do Betſáby mátki Sálomonowey : któremu <i>oná</i> rzekłá / A ſpokojneż jeſt przyśćie twoje? A <i>on</i> odpowiedźiał ; Spokojne.
Nád to rzekł : Mam nieco mówić z tobą : A <i>oná</i> rzekłá : Mów.
Tedy rzekł : Ty wieƺ yż moje było Króleſtwo / á ná mię obróćili byli wƺyſcy Izráelcżycy twarz ſwoję / ábym Królował : ále przenieśione jeſt Króleſtwo / y doſtáło śię brátu memu : bo mu od PAná náznácżone było :
Przetoż ćię teraz proƺę o jednę rzecż / á nie odmawiaj mi tego : A <i>oná</i> mu rzekłá : Mów.
Zátym on rzekł : Mów proƺę Sálomonowi Królowi / ( bo wiem żeć nie odmówi ) áby mi dał Abiſág Sunámitkę zá żonę.
Y odpowiedźiáłá Betſábe : Dobrze ; będę mówiłá o ćię z Królem.
A ták ƺłá Betſábe do Królá Sálomoná / áby z nim mówiłá zá Adoniaƺem : y wſtał Król przećiwko niey : á pokłoniwƺy śię jey uśiadł ná ſtolicy ſwey : Kazał też poſtáwić ſtolicę mátce ſwey / która śiádłá po práwicy jego.
Y rzekłá : Proƺę ćię o jednę máłą rzecż : nie odmawiaj mi : Y odpowiedźiał jey Król : Proś mátko mojá / ábowiem ći nie odmówię.
Tedy rzekłá : Niech będźie dáná Abiſág Sunámitká Adoniaƺowi brátu twemu zá żonę.
Lecż odpowiedźiał Król Sálomon / y rzekł mátce ſwojey : Przecżże prośiƺ o Abiſág Sunámitkę Adoniaƺowi? uproś mu y króleſtwo : ábowiem on jeſt brátem mojim ſtárƺym nád mię : á ma po ſobie Abiátárá Kápłaná / y Joábá Syná Sárwijey.
Y przyśiągł Król Sálomon przez PANA / mówiąc : To mi niech ucżyni Bóg / y to niech przycżyni / że przećiwko duƺy ſwey mówił Adoniaƺ te ſłowá.
A teraz żywie PAN / który mię utwirdźił / y poſádźił ná ſtolicy Dawidá Ojcá mojego / y który mi zbudował dom jáko obiecał : yż dźiś zábit będźie Adoniaƺ.
A ták poſłał Król Sálomon Bánájáſá Syná Jojádowego / który śię nań tárgnął y zábił go.
A do Abiátárá Kápłaná rzekł Król ; Idź do Anátot / do ośiádłośći twojey / ábowiemeś mężem śmierći : wƺákże ćię dźiś nie zábiję ; gdyżeś nośił ſkrzynię PAńſką przed Dawidem Ojcem mojim : á yżeś to wƺyſtko ćierpiał / cżym był trapiony Oćiec mój.
Y wyrzućił Sálomon Abiátárá / áby nie był Kápłanem PAńſkim : żeby śię wypełniło ſłowo PAńſkie / które był wyrzekł nád domem Heli w Sylo.
Tá wieść gdy przyƺłá do Joábá / ( ábowiem Joáb przeſtawał z Adoniaƺem / choćia z Abſálomem nie przeſtawał / ) tedy ućiekł Joáb do namiotu PAńſkiego / á uchwyćił śię rogów Ołtarzá.
Y oznájmiono Królowi Sálomonowi / że ućiekł Joáb do namiotu PAńſkiego / á że jeſt u Ołtarzá : tedy poſłał Sálomon Bánájáſá / Syná Jojádowego / mówiąc : Idź zábij go.
A przyƺedƺy Bánájás do namiotu PAńſkiego / rzekł do niego : Ták mówi Król : Wynidź : Który odpowiedźiał : Nie wynidę ; ále tu umrę. Y odnióſł to Bánájás Królowi / mówiąc : Ták mówił Joáb / y ták mi odpowiedźiał :
Y rzekł mu Król : Ucżyńże jáko mówił / á zábij go / y pogrzeb go : á odejmieƺ krew niewinną / którą wylał Joáb / ode mnie y od domu Ojcá mego :
A obróći PAN krew jego ná głowę jego / ábowiem tárgnął śię ná dwu mężów ſpráwiedliwƺych / y lepƺych niżli ſam / y zábił je miecżem : á Oćiec mój Dawid nie wiedźiał o tym : Abnerá / Syná Nerowego / Hetmáná Wojſká Izráelſkiego / y Amázę / Syná Jeterowego / Hetmáná wojſká Judſkiego.
A ták wróći śię krew ich ná głowę Joábowę / y ná głowę naśienia jego ná wieki : Lecż Dawidowi y naśieniu jego / y domowi jego / y ſtolicy jego / niech będźie pokój áż ná wieki od PANA.
Szedł tedy Bánájás Syn Jojádów / á rzućiwƺy śię nań / zábił go / y pogrzebiony jeſt w domu ſwym ná puƺcży.
Y poſtánowił Król Bánájáſá / Syná Jojádowego miáſto niego nád wojſkiem : á Sádoká Kápłáná poſtánowił Król miáſto Abiátárá.
Potym poſłał Król / y przyzwał Semejego / y rzekł mu : Zbuduj ſobie dom w Jeruzalem / y mieƺkaj tám / á niewychodź z támtąd nigdźie.
Bo któregobyś dniá wyƺedł / á przyƺedł zá potok Cedron / wiedz wiedząc że pewnie umrzeƺ : Krew twojá będźie ná głowę twoję.
Tedy rzekł Semej do Królá : Dobre jeſt to ſłowo : jáko mówił Król Pán mój / ták ucżyni ſługá twój. Y mieƺkał Semej w Jeruzalem przez wiele dni.
Y ſtáło śię po trzech lat / że ućiekli dwáj ſłudzy Semejemu / do Achiſá Syná Mááchy / Królá Getſkiego : y opowiedźiano Semejemu / mówiąc : Oto ſłudzy twoji <i>ſą</i> w Get.
Przetoż wſtawƺy Semej / y ośiodławƺy oſłá ſwego / jáchał do Get / do Achiſá / áby ƺukał ſług ſwojich : y wróćił śię Semej / y przywiódł ſługi ſwe z Get.
Y oznájmiono Sálomonowi / że był wyjáchał Semej z Jeruzalem do Get / y záśię śię wróćił.
Tedy poſłał Król / y wezwał Semejego / y rzekł mu : Izalim ćię nie poprzyśiągł przez PANA / á nie oświadcżyłem śię przed tobą / mówiąc : Któregobyśkolwiek dniá gdźie wyƺedł / wiedz wiedząc / że zápewne umrzeƺ : y mówiłeś do mnie : Dobre to ſłowo <i>którem</i> ſłyƺał.
Przecżżeś tedy nie ſtrzegł przyśięgi PAńſkiey / y przykazánia któremći był przykazał.
Nád to Król rzekł do Semejego : Ty wieƺ wƺyſtko złe / którego świádome jeſt ſerce twoje / coś ucżynił Dawidowi Ojcu memu : y oddał PAN złość twoję / ná głowę twoję :
Ale Król Sálomon błogoſłáwiony / y ſtolicá Dawidowá będźie utwirdzona przed PAnem áż ná wieki.
A ták rozkazał Król Bánájáſowi Synowi Jojádowemu / który wyƺedƺy tárgnął śię nań / y zábił <i>go</i> : A ták utwirdzone jeſt Króleſtwo w ręce Sálomonowey.
