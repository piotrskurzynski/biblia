Y ſtáło śię potem : Miał Nábot Jezreelitá winnicę / która byłá w Jezreelu podle páłácu Achábá / królá w Sámáryi.
Y rzekł Acháb do Nábotá / mówiąc : Dáj mi winnicę twoję / ábym miał z niej ogród dla járzyn / ábowiem bliſká jeſt domu mego ; á dam ći zá nię winnicę lepƺą / niżli tá jeſt ; álbo jeſlić śię zda / dam ći pieniędzy / ile ſtoi.
Y odpowiedźiał Nábot Achábowi : Nie dáj tego Pánie / ábym ći miał dáć dźiedźictwo ojców mojich.
Tedy przyƺedł Acháb do domu ſwego ſmutny y zágniewány dla ſłowá / które mu rzekł Nábot Jezreelitá / mówiąc : Nie dam ći dźiedźictwá ojców mojich ; y ukłádł śię ná łożu ſwem / á odwróćił twárz ſwoję / y nie jádł chlebá.
Wtem przyƺedƺy do niego Jezábelá / żoná jego / rzekłá mu : Przecżże duch twój ták ſmutny / że nie jeƺ chlebá?
Y odpowiedźiał jey : Przeto żem mówił z Nábotem Jezreelitą / y rzekłem mu : Dáj mi winnicę twoję zá pieniądze / álbo jeſli chceƺ / dam ći winnicę zá nię ; ále on odpowiedźiał : Nie dam ći winnicy mojey.
Tedy rzekłá do niego Jezábelá / żoná jego : Y tákże ty ſpráwujeƺ króleſtwo Izráelſkie? Wſtáń / jedz chleb / á bądź dobrej myśli ; ja tobie dam winnicę Nábotá Jezreelity.
A ták nápiſáłá liſt imieniem Achábowem / który zápiecżętowáłá piecżęćią jego / y poſłáłá on liſt do ſtárƺych y do przedniejƺych / którzy byli w mieśćie jego / y mieƺkáli z Nábotem.
A nápiſáłá on liſt w ten ſpoſób : Zápowiedźćie poſt / á poſádźćie Nábotá miedzy przedniejƺymi z ludu ;
Y poſtáwćie dwu mężów przewrotnych przećiw niemu / którzyby przećiwko niemu świádcżyli / mówiąc : Złorzecżyłeś Bogu y królowi ; potem wywiedźćie go / á ukámionujćie go / áby umárł.
Y ucżynili mężowie onego miáſtá ſtárśi y przedniejśi / którzy mieƺkáli w onem mieśćie jego / jáko byłá wſkázáłá do nich Jezábelá / według tego / jáko nápiſano było w liśćie / który poſłáłá do nich.
Zápowiedźieli poſt / y poſádźili Nábotá miedzy przedniejƺymi z ludu.
Potem przyƺli dwáj mężowie przewrotni / y uśiedli przećiw niemu / á świádcżyli przećiwko niemu oni mężowie przewrotni / to jeſt przećiw Nábotowi przed ludem / mówiąc : Złorzecżył Nábot Bogu y królowi. Y wywiedli go zá miáſto / y ukámionowáli go / y umárł.
Y poſłáli do Jezábeli / mówiąc : Ukámionowán jeſt Nábot / y umárł.
Y ſtáło śię / gdy uſłyƺáłá Jezábelá / że ukámionowány był Nábot / á iż umárł / rzekłá Jezábelá do Achábá : Wſtáń / pośiądź winnicę Nábotá Jezreelity / któryć jey nie chćiał dáć zá pieniądze ; ábowiem nie żyje Nábot / ále umárł.
A ták uſłyƺáwƺy Acháb / że umárł Nábot / wſtał / á ƺedł do winnicy Nábotá Jezreelity / áby ją pośiádł.
Tedy śię ſtáło ſłowo Páńſkie do Elijáƺá Teſbity / mówiąc :
Wſtáń / idź przećiw Achábowi / królowi Izráelſkiemu / który jeſt w Sámáryi ; oto jeſt ná winnicy Nábotowej / do którey ƺedł / áby ją pośiádł.
Y rzecżeƺ do niego / mówiąc : Ták mówi Pán : Azáś nie zábił y nie pośiádł? Powiedzże mu / mówiąc : Ták mówi Pán : Ták jáko pſy lizáli krew Nábotowę / ták też pewnie pſy będą lizáć krew twoję.
Y rzekł Acháb do Elijáƺá : A jużeś mię ználázł nieprzyjáćielu mój? A on odpowiedźiał : Ználázłem ; ábowiemeś śię záprzedał / ábyś cżynił złość przed ocżymá Páńſkimi.
Oto Ja przywiodę ná ćię złe / á odejmę potomki twe / y wytrácę z domu Achábowego / áż do nájmniejƺego ƺcżenięćiá / y więźniá / y opuƺcżonego w Izráelu.
A ucżynię z domem twojim / jáko z domem Jeroboámá / Syná Nábátowego / y jáko z domem Báázy Syná Achiaƺowego / dla rozdráźnieniá / któremeś mię do gniewu pobudźił / y przywiodłeś do grzechu Izráelá.
Tákże y o Jezábeli rzekł Pán / mówiąc : Pſy zjedzą Jezábelę miedzy murámi Jezreelſkimi.
Tego / który umrze z domu Achábowego w mieśćie / pſy zjedzą / á tego / który umrze ná polu / zjedzą ptáki powietrzne.
Abowiem nie był nikt jáko Acháb / któryby śiebie ſámego záprzedał / áby cżynił złe przed ocżymá Páńſkimi ; bo go poduƺcżáłá Jezábelá / żoná jego.
Abowiem śię dopuśćił rzecży bárzo obrzydłych / chodząc zá báłwánámi według wƺyſtkiego / jáko cżynili Amorejcżycy / których wygnał Pán przed oblicżem ſynów Izráelſkich.
A gdy uſłyƺał Acháb te ſłowá / rozdárł odźienie ſwoje / á włożywƺy wór ná ćiáło ſwoje / pośćił y leżał w worze / á chodźił po málucżku.
Y ſtáło śię ſłowo Páńſkie do Elijáƺá Teſbity / mówiąc :
Widźiáłżeś / jáko śię upokorzył Acháb przed twárzą moją? Ponieważ śię tedy upokorzył przed twárzą moją / nie przywiodę tego złego zá dni jego ; ále zá dni ſyná jego przywiodę to złe ná dom jego.
