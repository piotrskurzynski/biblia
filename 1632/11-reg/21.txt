Y ſtáło śię potym : Miał Nábot Jezreelitá winnicę / która byłá w Jezreelu / podle páłacu Achábá Królá w Sámáryjey.
Y rzekł Acháb do Nábotá / mówiąc : Daj mi winnicę twoję / ábym miał z niey ogród dla járzyn : ábowiem bliſka jeſt domu mego ; á damći zá nię winnicę lepƺą / niżli tá jeſt : álbo jeſlić śię zda / damći pieniędzy ile ſtoji.
Y odpowiedźiał Nábot Achábowi : Nie daj tego PAnie / ábymći miał dáć dźiedźictwo Ojców mojich.
Tedy przyƺedł Acháb do domu ſwego ſmętny y zágniewány / dla ſłowá które mu rzekł Nábot Jezreelitá / mówiąc : Nie damći dźiedźictwá Ojców mojich : y układł śię ná łożu ſwym / á odwróćił twarz ſwoję / y nie jadł chlebá.
W tym przyƺedƺy do niego Jezábelá żoná jego / rzekłá mu ; Przecżże Duch twój ták ſmętny / że nie jeƺ chlebá?
Y odpowiedźiał jey : Przeto żem mówił z Nábotem Jezreelitą / y rzekłem mu : Daj mi winnicę twoję zá pieniądze álbo jeſli chceƺ / damći winnicę zá nię : Ale <i>on</i> odpowiedźiał : Nie damći winnice mojey.
Tedy rzekłá do niego Jezábelá żoná jego : Y tákże ty ſpráwujeƺ króleſtwo Izráelſkie? Wſtań / jedz chleb / á bądź dobrey myśli ; Ja tobie dam winnicę Nábotá Jezreelity.
A ták nápiſáłá liſt imieniem Achábowym / który zápiecżętowáłá piecżęćią jego / y poſłáłá on liſt do ſtárƺych y do przedniejƺych / którzy <i>byli</i> w mieśćie jego / y mieƺkáli z Nábotem.
A nápiſáłá on liſt w ten ſpoſób : Zápowiedźćie poſt / á poſadźćie Nábotá miedzy przedniejƺymi z ludu.
Y poſtawćie dwu mężów przewrotnych przećiw niemu / którzyby przećiwko niemu świadcżyli / mówiąc : Złorzecżyłeś Bogu y Królowi : Potym wywiedźćie go á ukámionujćie go / áby umárł.
Y ucżynili mężowie onego miáſtá ſtárƺy y przedniejƺy którzy mieƺkáli w onym mieśćie jego / jáko byłá wſkazáłá do nich Jezábelá / według tego jáko nápiſano <i>było</i> w liśćie / który poſłáłá do nich.
Zápowiedźieli poſt / y poſádźili Nábotá miedzy przedniejƺymi z ludu.
Potym przyƺli dwá mężowie przewrotni / y uśiedli przećiw niemu / á świadcżyli przećiwko niemu oni mężowie przewrotni / <i>to jeſt,</i> przećiw Nábotowi przed ludem / mówiąc : Złorzecżył Nábot Bogu y Królowi. Y wywiedli go zá miáſto / y ukámionowáli go y umárł.
Y poſłáli do Jezábele / mówiąc ; Ukámionowan jeſt Nábot / y umárł.
Y ſtáło śię / gdy uſłyƺáłá Jezábelá / że ukámionowány był Nábot / á iż umárł / rzekłá Jezábelá do Achábá : Wſtań / pośiądź winnicę Nábotá Jezreelity / któryć jey nie chćiał dáć zá pieniądze : ábowiem nie żyw Nábot / ále umárł.
A ták uſłyƺawƺy Acháb / że umárł Nábot / wſtał / á ƺedł do winnice Nábotá Jezreelity / áby ją pośiadł.
Tedy śię ſtáło ſłowo PAńſkie do Eliaƺá Teſbity / mówiąc :
Wſtań idź przećiw Achábowi Królowi Izráelſkiemu / który jeſt w Sámáryjey : oto <i>jeſt</i> ná winnicy Nábotowey : do którey ƺedł / áby ją pośiadł.
Y rzecżeƺ do niego / mówiąc : Ták mówi PAN : Azaś nie zábił y nie pośiadł? Powiedzże mu / mówiąc ; Ták mówi PAN : Ták jáko pśi lizáli krew Nábotowę / ták też pewnie pśi będą lizáć krew twoję.
Y rzekł Acháb do Eliaƺá : A jużeś mię ználazł nieprzyjaćielu mój? A on odpowiedźiał : Ználazłem : ábowiemeś śię záprzedał / ábyś cżynił złość przed ocżymá PAńſkimi.
Oto ja przywiodę ná ćię złe / á odejmę potomki twe / y wytrácę z domu Achábowego áż do namniejƺego ƺcżenięćiá / y więźniá / y opuƺcżonego w Izráelu.
A ucżynię z domem twojim / jáko z domem Jeroboámá / Syná Nábátowego / y jáko z domem Báázy Syná Achiaƺowego : dla rozdraźnienia / którymeś mię do gniewu pobudźił / y przywiodłeś do grzechu Izráelá.
Tákże y o Jezábeli rzekł PAN / mówiąc : Pśi zjedzą Jezábelę miedzy murámi Jezreelſkimi.
Tego który umrze z domu Achábowego w mieśćie / pśi zjedzą / á tego który umrze ná polu / zjedzą ptacy powietrzni.
Abowiem nie był nikt jáko Acháb / któryby śiebie ſámego záprzedał / áby cżynił złe przed ocżymá PAńſkimi : bo go poduƺcżáłá Jezábelá żoná jego.
Abowiem śię dopuśćił rzecży bárzo obrzydłych / chodząc zá báłwany / według wƺyſtkiego / jáko cżynili Ammorejcżycy / które wygnał PAN przed oblicżem Synów Izráelſkich.
A gdy uſłyƺał Acháb te ſłowá / rozdárł odźienie ſwoje / á włożywƺy wór ná ćiáło ſwoje / pośćił / y leżał w worze / á chodźił po málucżku.
Y ſtáło śię ſłowo PAńſkie do Eliaƺá Tesbity / mówiąc :
Widźiałżeś jáko śię upokorzył Acháb przed twarzą moją? Ponieważ śię tedy upokorzył przed twarzą moją / nie przywiodę tego złego zá dni jego : <i>Ale</i> zá dni Syná jego przywiodę to złe ná dom jego.
