Tedy Król Sálomon rozmiłował śię niewiaſt obcych wiele : nie tylko córki Fáráonowey / <i>ále y</i> Moábitcżanek / Ammonitcżanek / Edomcżanek / Sydońcżanek / Hettejcżanek.
Z tych narodów o których powiedźiał PAN Synom Izráelſkim : Nie wchodźćie do nich / y one niech nie wchodzą do was : ábowiem nákłoniłyby ſerce wáƺe zá Bogi ſwymi : do tych przylgnął Sálomon miłośćią.
Ták yż miał żon Królowych śiedm ſet : á założnic trzy ſtá : y odwróćiły żony jego ſerce jego.
Y ſtáło śię gdy już był Sálomon <i>ſtáry,</i> że żony jego nákłoniły ſerce jego zá Bogi cudzymi / ták yż nie było ſerce jego zupełne z PAnem Bogiem jego / jáko ſerce Dawidá Ojcá jego.
Ale udał śię Sálomon zá Aſtárotą boginią Sydońſką / y zá Molochem obrzydliwośćią Ammonitów.
Y ucżynił Sálomon co śię nie podobáło PAnu / áni chodźił doſkonále zá PAnem / jáko Dawid Oćiec jego.
Tedy zbudował Sálomon Káplicę Chámoſowi obrzydliwośći Moábſkiey / ná górze przećiw Jeruzalem / y Molochowi obrzydliwośći Synów Ammonowych.
Y ták ucżynił wƺyſtkim żonom ſwym cudzoźiemkam / które kádźiły / y ofiárowáły Bogom ſwojim.
Y rozgniewał śię PAN ná Sálomoná / że śię odwróćiło ſerce jego od PANA Bogá Izráelſkiego / który śię mu był ukazał po dwá kroć.
Y zákazał mu tego / áby nie chodźił zá Bogi cudzymi : wƺákże nie ſtrzegł tego co PAN przykazał.
Przetoż rzekł PAN do Sálomoná : Ponieważeś śię tego dopuśćił / nie ſtrzegąc przymierza mego / áni wyroków mojich któremći przykazał / pewnie oderwę króleſtwo od ćiebie / á dam je ſłudze twojemu.
Wƺákże zá dni twojich nie ucżynię tego / dla Dawidá Ojcá twego. <i>Ale</i> z ręki Syná twego oderwę je.
Lecż wƺyſtkiego króleſtwá nie oderwę : Pokolenie jedno dam Synowi twemu / dla Dawidá ſługi mego / y dla Jeruzalem / którem obrał.
Przetoż wzbudźił PAN przećiwniká Sálomonowi / Adádá Edomcżyká / z naśienia królewſkiego który <i>był</i> w Edom.
Abowiem ſtáło śię / gdy Dawid był w Edom / á Joáb Hetmán wojſká wyjáchał / áby pochował pobite / y pobił wƺyſtkie mężcżyzny w Edom :
( Bo tám ƺeść mieśięcy mieƺkał Joáb ze wƺyſtkimi Izráelcżykámi / áż wytráćił wƺyſtkie mężcżyzny w Edom )
Tedy ućiekł Adád ſam / <i>y niektórzy</i> mężowie Edomſcy z ſług Ojcá jego z nim / áby ƺli do Egiptu : A Adád <i>był</i> chłopcem nie wielkim.
Którzy wyƺedƺy z Mádián przyƺli do Fárán : á wźiąwƺy z ſobą <i>niektóre</i> męże z Fárán / weƺli do Egiptu do Fáráoná Królá Egipſkiego : który mu dał dom / y żywność mu náznácżył / dał mu też y źiemię.
Y ználazł Adád wielką łáſkę w ocżách Fáráonowych / ták yż mu dał zá żonę śioſtrę żony ſwey / śioſtrę Królowey Táfnes.
Y porodźiłá mu śioſtrá Táfnes Genubátá Syná jego / którego odchowáłá Táfnes w domu Fáráonowym : Y był Genubát w domu Fáráonowym miedzy Syny Fáráonowymi.
A gdy uſłyƺał Adád w Egipćie iż záſnął Dawid z Ojcy ſwymi / á iż umárł Joáb Hetmán Wojſká : tedy rzekł Adád do Fáráoná : Puść mię że pójdę do źiemie mojey.
Któremu odpowiedźiał Fáráo : Cżegożći niedoſtawa u mnie / że chceƺ iść do źiemie twojey? A on rzekł : Nicżego : á wƺákże puść mię.
Wzbudźił tákże Bóg nań przećiwniká / Rezoná Syná Eliádowego / który był ućiekł od Adárezerá Królá Soby Páná ſwego.
A zebrawƺy do śiebie męże / był kśiążęćiem roty / gdy je Dawid mordował : Przetoż odƺedƺy do Dámáƺku / mieƺkáli w nim / á królowáli nád Dámáƺkiem.
Y był przećiwnikiem Izráelowi / po wƺyſtkie dni Sálomonowe / á to oprócż ƺkód które <i>mu cżynił</i> Adád : Bo śię brzydźił Izráelem / gdy królował w Syriey.
Jeroboám też Syn Nábátá Efrátejcżyká z Sáredy / ( á imię mátki jego Serwá <i>która byłá wdową</i> ) ſługá Sálomonów / podnióſł przećiwko Królowi rękę.
A táć byłá przycżyná / dla którey podnióſł rękę ſwą przećiwko Królowi : że Sálomon zbudowawƺy Mello / zápráwił dźiurę w mieśćie Dawidá Ojcá ſwego.
A Jeroboám był mąż mocny <i>y</i> możny : Przetoż widząc Sálomon młodźieńcá / że był ſpráwny / poſtánowił go nád podátkámi wƺyſtkimi domu Józefowego.
Y ſtáło śię tegoż cżáſu gdy Jeroboám wyƺedł z Jeruzalem / że go ználazł ná drodze Achiaƺ Silonitcżyk Prorok / będąc odźiany płaƺcżem nowym : á tylko ſámi dwáj byli ná polu.
Tedy wźiąwƺy Achiaƺ płaƺcż nowy / który miał ná ſobie / rozdárł go ná dwánaśćie ƺtuk.
Y rzekł do Jeroboámá : Weźmi ſobie dźieśięć ƺtuk : bo ták mówi PAN / Bóg Izráelſki : Oto ja oderwę Króleſtwo z rąk Sálomonowych / á dam tobie dźieśięcioro pokolenie.
Jedno tylko pokolenie zoſtánie mu / dla ſługi mego Dawidá / y dla miáſtá Jeruzalem / którem obrał ze wƺyſtkich pokoleń Izráelſkich.
Przeto że mię opuśćili / á kłaniáli śię Aſtároćie bogini Sydońſkiey / <i>y</i> Chámoſowi Bogu Moábſkiemu / y Molochowi Bogu Synów Ammonowych / á nie chodźili drogámi mojimi / áby cżynili to / co śię mnie podoba : áni też <i>ſtrzegli</i> wyroków mojich / y ſądów mojich / jáko Dawid Oćiec jego.
Wƺákże nie odejmę wƺyſtkiego Króleſtwá z ręki jego / owƺem zoſtáwię go Kśiążęćiem po wƺyſtkie dni żywotá jego / dla Dawidá ſługi mego / któregom obrał / który ſtrzegł rozkazánia mego / y wyroków mojich.
Ale wźiąwƺy Króleſtwo z ręki Syná jego / dam tobie z niego dźieśięć pokoleń.
A Synowi jego dam pokolenie jedno / áby zoſtáłá pochodnia Dawidowi ſłudze memu po wƺyſtkie dni przede mną / w mieśćie Jeruzalem którem ſobie obrał áby tám przebywáło imię moje :
A ćiebie wezmę ábyś królował nád wƺyſtkim / cżego żąda duƺá twojá : á będźieƺ Królem nád Izráelem.
Przetoż / jeſli będźieƺ ſłuchał wƺyſtkiego coć przykażę / á będźieƺ chodźił drogámi mojimi / cżyniąc to co dobrego jeſt w ocżách mojich / ſtrzegąc wyroków mojich y przykazań mojich / jáko cżynił Dawid ſługá mój ; tedy będę z tobą / á zbudujęć dom mocny / jákom zbudował Dawidowi / y podamći Izráelá :
Y trapić będę naśienie Dawidowe dla tego : á wƺákże nie po wƺyſtkie dni.
Przetoż Sálomon chćiał zábić Jeroboámá : ále wſtawƺy Jeroboám ućiekł do Egiptu / do Seſáká Królá Egipſkiego : y był w Egipćie áż do śmierći Sálomonowey.
A inne ſpráwy Sálomonowe / którekolwiek cżynił / y mądrość jego / izali nie ſą wypiſáne w kśięgách ſpraw Sálomonowych?
A dni których królował Sálomon w Jeruzalem nád wƺyſtkim Izráelem <i>było</i> cżterdźieśći lat.
Y záſnął Sálomon z Ojcy ſwymi / á pogrzebiony jeſt w mieśćie Dawidá Ojcá ſwego : y królował Roboám Syn jego miáſto niego.
