Tedy król Sálomon rozmiłował śię niewiáſt obcych wiele : nie tylko córki Fáráonowej / ále y Moábitcżánek / Ammonitcżánek / Edomcżánek / Sydońcżánek / Hetejcżánek.
Z tych narodów / o których powiedźiał Pán ſynom Izráelſkim : Nie wchodźćie do nich / y one niech nie wchodzą do was ; ábowiem nákłoniłyby ſerce wáƺe zá bogi ſwymi. Do tych przylgnął Sálomon miłośćią.
Ták iż miał żon królowych śiedm ſet / á záłożnic trzy ſtá ; y odwróćiły żony jego ſerce jego.
Y ſtáło śię / gdy już był Sálomon ſtáry / że żony jego nákłoniły ſerce jego zá bogi cudzymi / ták iż nie było ſerce jego zupełne z Pánem / Bogiem jego / jáko ſerce Dawidá / ojcá jego.
Ale udał śię Sálomon zá Aſtárotą / boginią Sydońſką / y zá Molochem / obrzydliwośćią Ammonitów.
Y ucżynił Sálomon / co śię nie podobáło Pánu / áni chodźił doſkonále zá Pánem / jáko Dawid / oćiec jego.
Tedy zbudował Sálomon káplicę Chámoſowi / obrzydliwośći Moábſkiej / ná górze przećiw Jeruzálemowi y Molochowi / obrzydliwośći ſynów Ammonowych.
Y ták ucżynił wƺyſtkim żonom ſwym cudzoźiemkom / które kádźiły / y ofiárowáły bogom ſwoim.
Y rozgniewał śię Pán ná Sálomoná / że śię odwróćiło ſerce jego od Páná / Bogá Izráelſkiego / który śię mu był ukázał po dwá kroć.
Y zákázał mu tego / áby nie chodźił zá bogi cudzymi ; wƺákże nie ſtrzegł tego / co Pán przykázał.
Przetoż rzekł Pán do Sálomoná : Poniewáżeś śię tego dopuśćił / nie ſtrzegąc przymierza mego / áni wyroków mojich / którem ći przykázał / pewnie oderwę króleſtwo od ćiebie / á dam je ſłudze twojimu /
Wƺákże zá dni twojich nie ucżynię tego dla Dawidá / ojcá twego ; ále z ręki ſyná twego oderwę je.
Lecż wƺyſtkiego króleſtwá nie oderwę ; pokolenie jedno dam ſynowi twemu dla Dawidá / ſługi mego / y dla Jeruzálemu / którem obrał.
Przetoż wzbudźił Pán przećiwniká Sálomonowi / Adádá Edomcżyká z náśieniá królewſkiego / który był w Edom.
Abowiem ſtáło śię / gdy Dawid był w Edom / á Joáb / hetmán wojſká / wyjechał / áby pochował pobite / y pobił wƺyſtkie mężcżyzny w Edom ;
( Bo tám ƺeść mieśięcy mieƺkał Joáb ze wƺyſtkimi Izráelcżykámi / áż wytráćił wƺyſtkie mężcżyzny w Edom. )
Tedy ućiekł Adád ſám / y niektórzy mężowie Edomſcy z ſług ojcá jego z nim / áby ƺli do Egiptu ; á Adád był chłopcem nie wielkim.
Którzy wyƺedƺy z Mádyján przyƺli do Fárán / á wźiąwƺy z ſobą niektóre męże z Fárán weƺli do Egiptu / do Fáráoná królá Egipſkiego / który mu dał dom / y żywność mu náznácżył / dał mu też y źiemię.
Y ználázł Adád wielką łáſkę w ocżách Fáráonowych / ták / iż mu dał zá żonę śioſtrę żony ſwej / śioſtrę królowej Táfnes.
Y porodźiłá mu śioſtrá Táfnes Gienubátá / ſyná jego / którego odchowáłá Táfnes w domu Fáráonowym. Y był Gienubát w domu Fáráonowym miedzy ſyny Fáráonowymi.
A gdy uſłyƺał Adád w Egipćie / iż záſnął Dawid z ojcy ſwymi / á iż umárł Joáb / hetmán wojſká / tedy rzekł Adád do Fáráoná : Puść mię / że pójdę do źiemi mojey.
Któremu odpowiedźiał Fáráo : Cżegoż ći niedoſtáwá u mnie / że chceƺ iść do źiemi twojey? A on rzekł : Nicżego ; á wƺákże puść mię.
Wzbudźił tákże Bóg nań przećiwniká / Rezoná / ſyná Elijádowego / który był ućiekł od Adárezerá / królá Soby / páná ſwego.
A zebráwƺy do śiebie męże / był kśiążęćiem roty / gdy je Dawid mordował ; przetoż odƺedƺy do Dámáƺku / mieƺkáli w nim / á królowáli nád Dámáƺkiem.
Y był przećiwnikiem Izráelowi po wƺyſtkie dni Sálomonowe / á to oprócż ƺkód / które mu cżynił Adád ; bo śię brzydźił Izráelem / gdy królował w Syryi.
Jeroboám też / ſyn Nábátá Efrátejcżyká z Sáredy / ( á imię mátki jego Serwá / która byłá wdową / )ſługá Sálomonowy / podnióſł przećiwko królowi rękę.
A táć byłá przycżyná / dla którey podnióſł rękę ſwą przećiwko królowi / że Sálomon zbudowáwƺy Mello / zápráwił dźiurę w mieśćie Dawidá / ojcá ſwego.
A Jeroboám był mąż mocny y możny. Przetoż widząc Sálomon młodźieńcá / że był ſpráwny / poſtánowił go nád podátkámi wƺyſtkiemi domu Józefowego.
Y ſtáło śię tegoż cżáſu / gdy Jeroboám wyƺedł z Jeruzálemu / że go ználázł ná drodze Achyjáƺ Sylonitcżyk / prorok / będąc odźiány płáƺcżem nowym ; á tylko ſámi dwáj byli ná polu.
Tedy wźiąwƺy Achyjáƺ płáƺcż nowy / który miał ná ſobie / rozdárł go ná dwánaśćie ƺtuk.
Y rzekł do Jeroboámá : Weźmij ſobie dźieśięć ƺtuk ; bo ták mówi Pán / Bóg Izráelſki : Oto Ja oderwę króleſtwo z rąk Sálomonowych / á dam tobie dźieśięć pokoleń.
Jedno tylko pokolenie zoſtánie mu dla ſługi mego Dawidá / y dla miáſtá Jeruzálem / którem obrał ze wƺyſtkich pokoleń Izráelſkich ;
Przeto / że mię opuśćili / á kłániáli śię Aſtároćie / bogini Sydońſkiej / y Chámoſowi / bogu Moábſkiemu / y Molochowi / bogu ſynów Ammonowych / á nie chodźili drogámi mojemi / áby cżynili to / co śię mnie podobá / áni też ſtrzegli wyroków mojich / y ſądów mojich / jáko Dawid / oćiec jego.
Wƺákże nie odejmę wƺyſtkiego króleſtwá z ręki jego / owƺem zoſtáwię go kśiążęćiem po wƺyſtkie dni żywotá jego dla Dawidá / ſługi mego / któregom obrał / który ſtrzegł rozkázánia mego / y wyroków mojich.
Ale wźiąwƺy króleſtwo z ręki ſyná jego / dam tobie z niego dźieśięć pokoleń :
A ſynowi jego dam pokolenie jedno / áby zoſtáłá pochodniá Dawidowi / ſłudze memu / po wƺyſtkie dni przedemną w mieśćie Jeruzálemie / którem ſobie obrał / áby tám przebywáło imię moje ;
A ćiebie wezmę / ábyś królował nád wƺyſtkim / cżego żądá duƺá twojá / á będźieƺ królem nád Izráelem.
Przetoż jeſli będźieƺ ſłuchał wƺyſtkiego / coć przykáżę / á będźieƺ chodźił drogámi mojemi / cżyniąc to / co dobrego jeſt w ocżách mojich / ſtrzegąc wyroków mojich / y przykázáń mojich / jáko cżynił Dawid / ſługá mój : tedy będę z tobą / á zbudujęć dom mocny / jákom zbudował Dawidowi / y podám ći Izráelá ;
Y trápić będę naśienie Dawidowe dla tego ; á wƺákże nie po wƺyſtkie dni.
Przetoż Sálomon chćiał zábić Jeroboámá ; ále wſtáwƺy Jeroboám ućiekł do Egiptu / do Seſáká / królá Egipſkiego / y był w Egipćie áż do śmierći Sálomonowej.
A inne ſpráwy Sálomonowe / którekolwiek cżynił / y mądrość jego / izali nie ſą wypiſáne w kśięgách ſpráw Sálomonowych?
A dni / których królował Sálomon w Jeruzálemie nád wƺyſtkim Izráelem / było cżterdźieśći lat.
Y záſnął Sálomon z ojcy ſwymi / á pogrzebiony jeſt w mieśćie Dawidá / ojcá ſwego ; y królował Roboám / ſyn jego / miáſto niego.
