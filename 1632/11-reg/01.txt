A gdy śię Król Dawid zſtárzał <i>y</i> záƺedł w látá : choć go odźiewano ƺátámi przećię śię zágrzać nie mógł.
Y rzekli mu ſłudzy jego : Niech poƺukáją Królowi Pánu náƺemu młodey pánienki / któraby ſtawáłá przed Królem / y opátrowáłá go : á ſypiájąc ná łonie jego / żeby zágrzewáłá Królá Páná náƺego.
Szukáli tedy pánienki piękney po wƺyſtkich gránicách Izráelſkich / y náleźli Abiſág Sunámitkę / á przywiedli ją do Królá.
A tá pánienká <i>byłá</i> bárzo piękná / y opátrowáłá Królá / y ſłużyłá mu ; ále jey Król nie uznał.
Lecż Adoniaƺ Syn Hággity wynośił śię / mówiąc ; Ja będę królował. Y náſpráwiał ſobie wozów / y jezdnych / y pięćdźieśiąt mężów którzy biegáli przed nim.
A nie gromił go nigdy Oćiec jego / mówiąc ; Przecżżeś to ucżynił? A był y ten bárzo piękney urody / którego byłá porodźiłá <i>Hággitá</i> po Abſálomie.
A miał zmowę z Joábem Synem Sárwijey / y z Abiátárem Kápłanem / którzy pomagáli zá Adoniaƺem.
Ale Sádok Kápłan / y Bánájás Syn Jojády / y Nátán Prorok / y Semej / y Rehi / y rycerſtwo Dawidowe / nie przeſtawáli z Adoniaƺem.
Tedy nábił Adoniaƺ owiec / y wołów / y bydłá tłuſtego / u kámieniá Zohelet / który był nád źrzódłem Rogel : y wezwał wƺyſtkiey bráći ſwey / Synów królewſkich / y wƺyſtkich mężów z Judá / ſług królewſkich.
Ale Nátáná Proroká / y Bánájáſá / y <i>innego</i> rycerſtwá / áni Sálomoná brátá ſwego / nie wezwał.
Tedy rzekł Nátán do Betſáby mátki Sálomonowey / mówiąc ; A nie ſłyƺáłáś yż króluje Adoniaƺ Syn Hággity / á Dawid Pán náƺ nie wie <i>o tym</i>?
Przetoż teraz pódź / proƺę / damći rádę / á záchowaƺ zdrowie twoje / y zdrowie Syná twego Sálomoná.
Idź / á wnidź do Królá Dawidá / y mów do niego : Zaliś ty Królu Pánie mój nieprzyśiągł ſłużebnicy twojey / mówiąc : Sálomon Syn twój będźie królował po mnie / á on będźie śiedźiał ná ſtolicy mojey? Przecżże tedy króluje Adoniaƺ?
A gdy ty jeƺcże tám będźieƺ mówiłá z Królem / ja przydę zá tobą / y dopełnię ſłów twojich.
A ták weƺłá Betſábee do Królá ná pokój : á Król śię już był bárzo zſtárzał : á Abiſág Sunámitká poſługowáłá Królowi :
Tedy náchyliwƺy śię Betſábee pokłoniłá śię Królowi : którey rzekł Król ; Cżego chceƺ?
A <i>oná</i> mu odpowiedźiáłá : Pánie mój / tyś przyśiągł przez PAná Bogá twego ſłużebnicy twojey / że Sálomon Syn twój będźie królował po mnie / á on będźie śiedźiał ná ſtolicy mojey.
A oto już Adoniaƺ króluje / á ty teraz Królu Pánie mój <i>o tym</i> niewieƺ :
Abowiem nábił wołów y bydłá tłuſtego / y owiec bárzo wiele / y wezwał wƺyſtkich Synów królewſkich / y Abiátárá kápłaná / y Joábá Hetmáná wojſká ; ále Sálomoná ſługi twego nie wezwał.
Lecż ty Królu Pánie mój <i>wieƺ,</i> yż śię ocży wƺyſtkiego Izráelá oglądáją ná ćię / ábyś im oznájmił / kto będźie śiedźiał ná ſtolicy Królá Páná mego po tobie :
Inácżey ſtánie śię / gdy záſnie Król Pán mój z Ojcy ſwymi / że będźiemy ja y Sálomon Syn mój jáko grzeƺnicy.
A oto gdy oná jeƺcże mówiłá z Królem / przyƺedł Nátán Prorok.
Y opowiedźiáno to Królowi / mówiąc : Oto Nátán Prorok : który wƺedƺy do Królá / pokłonił śię Królowi twarzą ſwą ku źiemi.
Rzekł zátym Nátán : Królu Pánie mój / zażeś ty rzekł : Adoniaƺ będźie królował po mnie / á on uśiędźie ná ſtolicy mojey?
Abowiem dźiś ƺedƺy nábił wołów y bydłá tłuſtego / y owiec bárzo wiele / y wezwał wƺyſtkich Synów królewſkich / y Hetmánów wojſk / y Abiátárá kápłaná / á oto oni jedzą z nim y piją / y mówią ; Niech żywie Król Adoniaƺ.
Ale mnie ſługi twego / y Sádoká kápłaná / y Bánájáſá Syná Jojádowego / y Sálomoná ſługi twego nie wezwał.
Yzali od Królá Páná mego ſtáłá śię tá rzecż? á nie oznájmiłeś ſłudze twemu / kto ma śiedźieć ná ſtolicy Królá Páná mego po nim?
Y odpowiedźiał Król Dawid / mówiąc : Záwołajćie do mnie Betſáby : która przyƺedƺy przed oblicżność królewſką / ſtánęłá przed Królem.
Tedy przyśiągł Król mówiąc : Żywie PAN który wybáwił duƺę moję z káżdego ućiſku /
Yż jákom ći przyśiągł przez PAná Bogá Izráelſkiego / mówiąc : Że Sálomon Syn twój królowáć będźie po mnie / á on uśiędźie ná ſtolicy mojey miáſto mnie / ták dźiś ucżynię.
Y náchyliwƺy śię Betſábee twarzą ku źiemi / ukłoniłá śię Królowi / y rzekłá : Niech żywie Dawid Król Pán mój ná wieki.
Zátym rzekł Król Dawid : Záwołajćie do mnie Sádoká Kápłaná / y Nátáná Proroká / y Bánájáſá Syná Jojádowego : Y weƺli do Królá.
Y rzekł im Król : Weźmićie z ſobą ſługi Páná wáƺego / á wſadźćie Sálomoná Syná mego ná mulicę moję / y prowadźćie go do Gihonu :
A tám go pomáże Sádok Kápłan / y Nátán Prorok / zá Królá nád Izráelem : potym zátrąbićie w trąbę / á rzecżećie : Niech żywie Król Sálomon!
Potym pójdźiećie zá nim : á on przyƺedƺy śiędźie ná ſtolicy mojey : Y będźie królował miáſto mnie : Bom mu ja rozkazał áby był wodzem nád Izráelem y nád Judą.
Y odpowiedźiał Bánájás Syn Jojádów Królowi / mówiąc / Amen : Niech to ztwirdźi PAN / Bóg Królá Páná mego.
Jáko był PAN z Królem Pánem mojim / ták niech będźie z Sálomonem / á niechaj wywyżƺy ſtolicę jego / nád ſtolicę Dawidá Królá Páná mego.
A ták ƺedł Sádok Kápłan / y Nátán Prorok / y Bánájás Syn Jojádów / przytym Cheretcżycy / y Feletcżycy : y wſádźili Sálomoná ná mulicę Królá Dawidá : á prowádźili go do Gihonu.
Tedy wźiął Sádok Kápłan róg Olejku z Namiotu / y pomázał Sálomoná : potym trąbili w trąbę : Y zákrzyknął wƺyſtek lud : Niech żyje Król Sálomon!
Y ƺedł wƺyſtek lud zá nim : Tenże lud grał ná piƺcżałkách / weſeląc śię weſelem wielkim : ták yż drżáłá źiemiá od głoſu ich.
Co gdy uſłyƺał Adoniaƺ / y wƺyſcy wezwáni którzy byli z nim / ( á już śię też byłá dokońcżyłá ucżtá ) ſłyƺąc też y Joáb głos trąby / rzekł ; Cóż to zá krzyk miáſtá hucżącego?
A gdy on tego domawiał / oto Jonátán / Syn Abiátárá Kápłaná / przyƺedł : Któremu rzekł Adoniaƺ ; Wnidź / boś ty mąż ſtátecżny / á powieƺ <i>nam</i> co dobrego.
Tedy odpowiedźiał Jonátán / y rzekł Adoniaƺowi ; Dawid Król Pán náƺ / poſtánowił zápewne Sálomoná Królem.
Abowiem poſłał z nim Król Sádoká Kápłaná / y Nátáná Proroká / y Bánájáſá Syná Jojádowego / do tego Cheretcżyki y Feletcżyki / którzy go wſádźili ná mulicę królewſką.
Y pomázáli go Sádok Kápłan / y Nátán Prorok zá Królá w Gihonie : y ƺli z támtąd weſeląc śię / ták że zádrżáło miáſto : tenći jeſt krzyk / któryśćie ſłyƺeli.
Y już uśiadł Sálomon ná ſtolicy króleſtwá.
Nád to y ſłudzy królewſcy przyƺli / áby błogoſłáwili Dawidowi Królowi Pánu náƺemu / mówiąc : Niech ſławniejƺe ucżyni Bóg imię Sálomonowe nád imię twoje / á niech wywyżƺy ſtolicę jego / nád ſtolicę twoję : Y pokłonił śię Król ná łożu <i>ſwym</i>.
Przytym ták rzekł Król : Błogoſłáwiony PAN Bóg Izráelſki / który dał dźiś śiedzącego ná ſtolicy mojey / ná co pátrzą ocży moje.
Zlękli śię tedy / y wſtáli wƺyſcy wezwáni / którzy byli z Adoniaƺem / y poƺli káżdy w drogę ſwą.
Adoniaƺ tákże bojąc śię Sálomoná / wſtał / y poƺedł / á uchwyćił śię rogów Ołtarzá.
Y oznájmiono Sálomonowi / mówiąc : Oto Adoniaƺ boji śię Królá Sálomoná / á oto uchwyćił śię rogów Ołtarzá / mówiąc : Niech mi dźiś przyśięże Król Sálomon / że niezábije ſługi ſwego miecżem.
Tedy rzekł Sálomon : Jeſli będźie mężem ſtátecżnym / nie ſpádnie y włos z niego ná źiemię : ále jeſli śię w nim znajdźie co złego / pewnie umrze.
A ták poſłał Król Sálomon : y przywiedźiono go od Ołtarzá : á gdy przyƺedł / ukłonił śię Królowi Sálomonowi : y rzekł mu Sálomon : Idź do domu twego.
