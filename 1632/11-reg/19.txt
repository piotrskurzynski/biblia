Tedy oznájmił Acháb Jezábeli wƺyſtko / co ucżynił Elijáƺ / á iż práwie wƺyſtkie proroki pomordował miecżem.
Przetoż poſłáłá Jezábelá poſłá do Elijáƺá / mówiąc : To mi niech ucżynią bogowie / y to mi niech przycżynią / jeſli o tym cżáśie jutro nie położę duƺy twojey / jáko duƺy którego z onych.
Co gdy wyrozumiał Elijáƺ / wſtał y odƺedł / áby duƺę ſwą záchował / á przyƺedł do Beerſeby / która byłá w Judztwie / y zoſtáwił tám ſługę ſwego.
A ſám poƺedł w puƺcżę ná jeden dźień drogi : á gdy przyƺedł / y uśiádł pod jednym jáłowcem / życżył ſobie umrzeć / y rzekł : Doſyć już / o Pánie ; weźmijże duƺę moję / bom nie jeſt lepƺym nád ojców mojich.
Y położył śię / á záſnął pod onym jáłowcem / á oto w tenże cżás tknął go Anjoł y rzekł mu : Wſtáń / á jedz.
A gdy śię obejrzał / oto w głowách jego był chleb ná węglu upiecżony y cżáƺá wody. A ták jádł y pił / y położył śię znowu.
Potem wróćił śię Anjoł Páńſki powtóre / y tknął go / á rzekł : Wſtáń / jedz / ábowiem dáleką maƺ drogę przed ſobą.
A ták wſtáwƺy jádł y pił / á ƺedł w mocy pokármu onego cżterdźieśći dni y cżterdźieśći nocy / áż do góry Bożey Horeb.
Y wƺedł tám do jáſkini / á przenocował tám. A oto ſłowo Páńſkie do niego / mówiąc : Cóż tu cżyniƺ Elijáƺu?
Który odpowiedźiał ; Gorliwiem śię záſtáwiał o Páná / Bogá zaſtępów ; ábowiem ſynowie Izráelſcy opuśćili przymierze twoje / ołtárze twoje zburzyli / y proroki twoje miecżem pomordowáli / á zoſtáłem ja ſám / y ƺukáją duƺy mojey / áby mi ją odjęli.
Tedy onże głos rzekł : Wynijdź / á ſtáń ná górze przed Pánem. A oto Pán przechodźił / y wiátr gwáłtowny y mocny podwrácájący góry / y łámiący ſkáły przed Pánem ; ále Pán nie był w onym wietrze. Zá wiátrem było trzęśienie źiemi ; ále nie był Pán y w onem trzęśieniu.
Zá trzęśieniem był ogień ; ále Pán nie był w ogniu ; zá ogniem był głos ćichy y wolny.
To gdy uſłyƺał Elijáƺ / zákrył oblicże ſwoje płáƺcżem ſwoim / á wyƺedƺy ſtánął we drzwiách jáſkini. A oto do niego głos mówiący : Co tu cżyniƺ Elijáƺu?
A on odpowiedźiał : Gorliwiem śię záſtáwiał o Páná / Bogá zaſtępów : ábowiem opuśćili przymierze twoje ſynowie Izráelſcy / ołtárze twoje poburzyli / á proroki twoje miecżem pomordowáli / y zoſtáłem ja ſám / á ƺukáją duƺy mojey / áby mi ją odjęli.
Ale Pán rzekł do niego : Idź / wróć śię drogą twą ná puƺcżę Dámáſką / á gdy tám przyjdźieƺ / pomáżeƺ Házáelá zá królá nád Syryją ;
A Jehu / ſyná Námſy / pomáżeƺ zá królá nád Izráelem / á Elizeuƺá / ſyná Sáfátowego / z Abelmecholá / pomáżeƺ zá proroká miáſto śiebie.
Y ſtánie śię / że ktokolwiek ujdźie miecżá Házáelowego / zábije go Jehu / á ktokolwiek ujdźie miecżá Jehu / zábije go Elizeuƺ.
Jednákiem ſobie záchował w Izráelu śiedm tyśięcy / których wƺyſtkich koláná nie kłániáły śię Báálowi / y których wƺyſtkich uſtá nie cáłowáły go.
A ták on odƺedƺy z támtąd / ználázł Elizeuƺá / ſyná Sáfátowego / á on orze / á dwánaśćie járzm wołów przed nim / á ſám był przy dwunáſtem járzmie / á idąc mimo niego Elijáƺ / wrzućił nań płáƺcż ſwój.
Który opuśćiwƺy woły bieżał zá Elijáƺem / y rzekł : Niech pocáłuję proƺę ojcá mego / y mátkę moję / á pójdę zá tobą ; któremu rzekł : Idź / á wróć śię záśię / ponieważ widźiƺ / com ći ucżynił.
A ták wróćiwƺy śię do niego wźiął párę wołów / y zábił je / á przy drwách z pługá náwárzył mięſá z nich / y dał ludowi / y jedli. A wſtáwƺy ƺedł zá Elijáƺem / y ſłużył mu.
