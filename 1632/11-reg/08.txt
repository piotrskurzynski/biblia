Tedy zebrał Sálomon ſtárƺe Izráelſkie / y wƺyſtkie celniejƺe z káżdego pokolenia / y przedniejƺe z ojców ſynów Izráelſkich / do śiebie do Jeruzálemu / áby przenieśioná byłá ſkrzyniá przymierza Páńſkiego z miáſtá Dawidowego / które jeſt Syon.
Y zeƺli śię do królá Sálomoná wƺyſcy mężowie Izráelſcy mięśiącá Wrześniá w urocżyſte święto ; á ten mieśiąc jeſt śiódmy.
A gdy śię zeƺli wƺyſcy ſtárśi Izráelſcy / wźięli kápłáni ſkrzynię.
Y przenieśli ſkrzynię Páńſką / y námiot zgromádzenia / y wƺyſtkie nacżynia święte / które były w námioćie / á przenieśli je kápłáni y Lewitowie.
Lecż król Sálomon / y wƺyſtko mnóſtwo Izráelſkie / które śię zeƺło do niego / z nim przed ſkrzynią ofiárowáli owce y woły / których nie licżono áni ráchowano dla mnóſtwá.
Wnieśli tedy kápłáni ſkrzynię przymierza Páńſkiego ná miejſce jey do wnętrznego domu / do świątnicy świętych / pod ſkrzydłá Cherubinów.
Abowiem Cherubinowie mieli rozćiągnione ſkrzydłá nád miejſcem ſkrzyni / á okrywáli Cherubinowie ſkrzynię y drążki jey z wierzchu.
Y powyćiągáli one drążki / ták / że widáć było końce ich w świątnicy ná przodku świątnicy świętych ; ále nie widáć ich było zewnątrz ; y támże były áż do dniá tego.
Nic nie było w ſkrzyni tylko dwie táblice kámienne / które tám był ſchował Mojzeƺ ná Horebie / gdy ſtánowił przymierze Pán z ſynámi Izráelſkimi / gdy ƺli z źiemi Egipſkiej.
Y ſtáło śię / gdy wychodźili kápłáni z świątnicy / że obłok nápełnił dom Páńſki.
Ták iż śię nie mogli kápłáni oſtáć y ſłużyć dla onego obłoku ; ábowiem nápełniłá byłá chwałá Páńſka dom Páńſki.
Tedy rzekł Sálomon : Pán powiedźiał / iż miał mieƺkáć we mgle.
Jużem zbudował dom ná mieƺkánie tobie / miejſce / ábyś tám przebywał ná wieki.
Y obróćił król oblicże ſwoje / y błogoſłáwił wƺyſtkiemu zgromádzeniu Izráelſkiemu ; á wƺyſtko zgromádzenie Izráelſkie ſtáło.
Y rzekł : Błogoſłáwiony Pán / Bóg Izráelſki / który mówił uſty ſwemi do Dawidá / ojcá mego / y ſkutecżnie to wypełnił / mówiąc :
Ode dniá / któregom wywiódł lud mój Izráelſki z Egiptu / nie obráłem miáſtá ze wƺyſtkich pokoleń Izráelſkich ku zbudowániu domu / gdźieby przebywáło imię moje ; álem obrał Dawidá / áby był nád ludem moim Izráelſkim.
Poſtánowiłći był wpráwdźie w ſercu ſwem Dawid / oćiec mój / zbudowáć dom imieniowi Páná / Bogá Izráelſkiego ;
Ale rzekł Pán do Dawidá / ojcá mego : Acżkolwiekeś poſtánowił w ſercu twem / zbudowáć dom imieniowi memu / y dobrześ ucżynił / żeś to umyślił w ſercu twojim :
Wƺákże ty nie będźieƺ budował tego domu ; ále ſyn twój / który wynijdźie z biódr twojich / ten zbuduje dom imieniowi memu.
A ták utwierdźił Pán ſłowo ſwoje / które był powiedźiał. Bom ja powſtał miáſto Dawidá / ojcá mego / y uśiádłem ná ſtolicy Izráelſkiej / jáko był powiedźiał Pán / y zbudowáłem dom imieniowi Páná / Bogá Izráelſkiego.
Y náznácżyłem tám miejſce ſkrzyni / w którey jeſt przymierze Páńſkie / które ucżynił z ojcy náƺymi / gdy je wywiódł z źiemi Egipſkiej.
Tedy ſtánął Sálomon przed ołtárzem Páńſkim / przed wƺyſtkim zgromádzeniem Izráelſkiem / y wyćiągnął ręce ſwoje ku niebu /
Y rzekł : Pánie / Boże Izráelſki / niemáƺ tobie podobnego Bogá ná niebie wzgórę / áni ná źiemi niſko ; który chowáƺ umowę y miłośierdźie ſługom twym / którzy chodzą przed tobą cáłem ſercem ſwojim ;
Któryś ſpełnił ſłudze twemu Dawidowi / ojcu memu / coś mu powiedźiał ; coś mówił uſty ſwemi / toś ſkutecżnie wypełnił / jáko śię dniá tego pokázuje.
Przetoż teraz o Pánie / Boże Izráelſki / źiść ſłudze twemu Dawidowi / ojcu memu / coś mu powiedźiał / mówiąc : Nie będźie odjęty przed twárzą moją z narodu twego mąż / któryby śiedźiał ná ſtolicy Izráelſkiej / jeſli tylko będą przeſtrzegáli ſynowie twoi drogi ſwej / chodząc przede mną / jákoś ty chodźił przed oblicżnośćią moją.
Przetoż teraz / o Boże Izráelſki / niech będźie utwierdzone proƺę ſłowo twoje / któreś mówił do ſługi twego Dawidá / ojcá mego.
( Acżći wpráwdźie / izali Bóg będźie mieƺkał ná źiemi? Oto niebioſá / y niebá niebios / nie mogą ćię ogárnąć ; jákoż dáleko mniej ten dom / którym zbudował. )
A wejrzyj ná modlitwę ſługi twego / y ná prośbę jego / o Pánie Boże mój / wyſłucháj wołánie y modlitwę / którą śię dźiś ſługá twój modli przed tobą.
A niech będą otworzone ocży twoje nád tym domem w nocy y we dnie / nád tym miejſcem / o któremeś powiedźiał : Tu będźie imię moje ; ábyś wyſłuchiwał modlitwę / którą śię będźie modlił ſługá twój ná miejſcu tym.
Wyſłuchájże prośby ſługi twego / y ludu twego Izráelſkiego / który śię modlić będźie ná tym miejſcu. Ty wyſłucháj z miejſcá mieƺkánia twego / z niebá / á wyſłucháwƺy bądź miłośćiw.
Gdyby cżłowiek zgrzeƺył przećiwko bliźniemu ſwemu / á przywiódłby go do przyśięgi / ták / żeby przyśięgáć muśiał / á przyƺłáby tá przyśięgá przed ołtárz twój w tym domu :
Ty wyſłucháj z niebá / á rozeznáj y rozſądź ſługi twoje / potępiájąc niezbożnego / y obrácájąc ſpráwy jego ná głowę jego / á uſpráwiedliwiájąc ſpráwiedliwego / oddáwájąc mu według ſpráwiedliwośći jego.
Gdyby był poráżony lud twój Izráelſki od nieprzyjáćielá / przeto iż zgrzeƺyli przećiw tobie / á náwróćiliby śię do ćiebie / wyznáwájąc imię twoje / á modląc śię / przepráƺáliby ćię w tym domu :
Ty wyſłucháj z niebá / y odpuść grzech ludowi twemu Izráelſkiemu / á przywróć je záśię do źiemi / którąś dał ojcom ich.
Gdyby záwárte było niebo / á nie byłoby dżdżu / przeto że zgrzeƺyli przećiwko tobie / á modliliby śię ná tym miejſcu / wyznáwájąc imię twoje / á od grzechów ſwojich odwróćiliby śię / gdybyś je utrápił :
Ty wyſłucháj z niebá / á odpuść grzech ſług twojich / y ludu twego Izráelſkiego / náucżywƺy ich drogi práwej / po którey chodźić máją / á dáj deƺcż ná źiemię twoję / którąś dał w dźiedźictwo ludowi twemu.
Byłliby głód ná źiemi / byłliby mor / ſuƺá / rdzá / ƺáráńcżá / jeſliby były chrząƺcże / jeſliby go śćiſnął nieprzyjáćiel jego w źiemi mieƺkánia jego / álbo jákákolwiek plágá / álbo jákákolwiek niemoc :
Wƺelką modlitwę / y wƺelką prośbę / którąby cżynił którykolwiek cżłowiek / álbo wƺyſtek lud twój Izráelſki / ktoby jedno uznał ránę ſercá ſwego / y wyćiągnąłby ręce ſwe w domu tym :
Ty wyſłucháj z niebá / z miejſcá mieƺkánia twego / á odpuść / y ucżyń / y oddáj káżdemu według wƺelkich dróg jego / które znáƺ w ſercu jego ; bo ty / ty ſám znáƺ ſercá wƺyſtkich ſynów ludzkich ;
Aby śię ćiebie báli po wƺyſtkie dni / które żyć będą ná źiemi / którąś dał ojcom náƺym.
Nádto y cudzoźiemiec / który nie jeſt z ludu twego Izráelſkiego / przyjdźieli z źiemi dálekiej dla imienia twego ;
( Bo uſłyƺą o imieniu twojim wielkiem / y o ręce twojey możney / y o rámieniu twojim wyćiągnionem / ) przyjdźieli tedy / á będźie śię modlił w tym domu :
Ty wyſłucháj z niebá / z miejſcá mieƺkánia twego / á ucżyń wƺyſtko / o co záwołá do ćiebie on cudzoźiemiec / áby poználi wƺyſcy národowie źiemſcy imię twoje / y báli śię ćiebie / jáko lud twój Izráelſki / á żeby wiedźieli / że imię twoje wzywáne jeſt nád tym domem / którym zbudował.
Gdyby wyƺedł lud twój ná wojnę przećiwko nieprzyjáćielowi ſwemu drogą / którą je pośleƺ / á modliliby śię Pánu / obróćiwƺy śię ku miáſtu / któreś obrał / y ku domowi / którym zbudował imieniowi twemu :
Wyſłuchájże z niebá modlitwę ich / y prośbę ich / á wykonáj ſąd ich.
Gdyby też zgrzeƺyli przećiwko tobie / ( bo niemáƺ cżłowieká / któryby nie grzeƺył / ) á rozgniewáwƺy śię ná nie / podáłbyś je pod moc nieprzyjáćielowi / któryby je pojmáwƺy / záwiódł je w niewolę / do źiemi nieprzyjáćielſkiej / dálekiej álbo bliſkiej :
A upámiętáliby śię w ſercu ſwojim w oney źiemi / do którey ſą záprowádzeni w niewolę / y náwróćiliby śię / á przepráƺáliby ćię w źiemi tych / którzy je pojmáli / mówiąc : Zgrzeƺyliśmy / y źleśmy ucżynili / niepobożnieśmy śię ſpráwowáli ;
A ták náwróćiliby śię do ćiebie z cáłego ſercá ſwego / y z cáłej duƺy ſwej / w źiemi nieprzyjáćiół ſwojich / którzy je pojmáli / á modliliby śię tobie / obróćiwƺy śię ku źiemi ſwej / którąś dał ojcom ich / ku miáſtu / któreś obrał / y ku domowi / którym zbudował imieniowi twemu :
Wyſłuchájże tedy z niebá / z miejſcá mieƺkánia twego / modlitwę ich y prośbę ich / á wykonáj ſąd ich /
A bądź miłośćiw ludowi twemu / który przećiw tobie zgrzeƺył / y wƺyſtkim niepráwośćiom ich / któremi wyſtąpili przećiw tobie / á nákłoń ku nim miłośierdźiá tych / którzy je pojmáli / áby śię zmiłowáli nád nimi ;
Ponieważ ſą ludem twoim / y dźiedźictwem twojim / któreś wywiódł z Egiptu / z pośrzodku piecá żeláznego.
Niech będą ocży twoje otwárte ná prośbę ſługi twego / y ná prośbę ludu twego Izráelſkiego / ábyś je wyſłuchał we wƺyſtkim / o co ćię wzywáć będą.
Abowiemeś je ty ſobie odłącżył zá dźiedźictwo ze wƺyſtkich narodów źiemi / jákoś powiedźiał przez Mojżeƺá / ſługę twego / gdyś wywiódł ojce náƺe z Egiptu / o Pánie Boże!
Y ſtáło śię / gdy Sálomon modląc śię Pánu dokonał wƺyſtkiej oney modlitwy y prośby / że wſtał od ołtárzá Páńſkiego / á przeſtał klęcżeć y podnośić rąk ſwojich ku niebu ;
A ſtojąc błogoſłáwił wƺyſtkiemu zgromádzeniu Izráelſkiemu wielkim głoſem / mówiąc :
Błogoſłáwiony Pán / który dał odpocżnienie ludowi ſwemu Izráelſkiemu / według wƺyſtkiego / co powiedźiał ; nie chybiło żádne ſłowo ze wƺyſtkich ſłów jego dobrych / które mówił przez Mojżeƺá / ſługę ſwego.
Niechże będźie Pán / Bóg náƺ / z námi / jáko był z ojcy náƺymi ; niech nas nie opuƺcżá / áni nas odrzucá ;
Ale niech nákłoni ſerce náƺe ku ſobie / żebyſmy chodźili po wƺyſtkich drogách jego / ſtrzegąc rozkázánia jego / y wyroków jego / y ſądów jego / które przykázał ojcom náƺym.
A niech będą te ſłowá moje / któremim śię modlił przed Pánem / bliſkie Páná Bogá náƺego we dnie y w nocy / áby wykonywał ſąd ſługi ſwego / y ſąd ludu ſwego Izráelſkiego / ſądząc káżdą ſpráwę dniá ſwojego ;
Żeby poználi wƺyſcy národowie źiemſcy / iż Pán ſám jeſt Bogiem / á nikt inƺy.
Niechże tedy będźie ſerce wáƺe doſkonáłe ku Pánu / Bogu náƺemu / ábyśćie chodźili w wyrokách jego / á ſtrzegli przykázáń jego / jáko y dniá dźiśiejƺego.
Tedy król / y wƺyſtek Izráel z nim / ſpráwowáli ofiáry przed Pánem.
Y ofiárował Sálomon ofiárę ſpokojną / którą ſpráwował Pánu / wołów dwádźieśćiá y dwá tyśiące / y owiec ſto y dwádźieśćiá tyśięcy. A ták poświęcáli dom Páńſki król y wƺyſcy ſynowie Izráelſcy.
Onegoż dniá poświęćił król pośrodek śieni / która byłá przed domem Páńſkim ; bo tám ofiárował cáłopálenie / y ofiárę śniedną / y tłuſtośći ofiár ſpokojnych / przeto że ołtárz miedźiány / który był przed Pánem / był máły / y nie mogły śię ná nim zmieśćić cáłopalenia / y ofiáry śniedne / y tłuſtośći ofiár ſpokojnych.
A ták obchodźił Sálomon ná on cżás święto zácne / y wƺyſtek Izráel z nim / zgromádzenie wielkie od wejśćiá do Emát áż do rzeki Egipſkiej / przed Pánem / Bogiem náƺym / przez śiedm dni y przez śiedm dni / to jeſt / przez cżternaśćie dni.
A dniá óſmego rozpuśćił lud ; którzy błogoſłáwiąc królowi / rozeƺli śię do przybytków ſwojich / weſeląc śię / y ćieƺąc śię w ſercu ſwem ze wƺyſtkiego dobrego / które ucżynił Pán Dawidowi / ſłudze ſwemu / y Izráelowi / ludowi ſwemu.
