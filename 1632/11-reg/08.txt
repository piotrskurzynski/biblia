Tedy zebrał Sálomon ſtárƺe Izráelſkie / y wƺyſtkie celniejƺe z káżdego Pokolenia / <i>y</i> przedniejƺe z Ojców Synów Izráelſkich / do śiebie do Jeruzalem / áby przenieśiona byłá ſkrzyniá przymierza PAńſkiego / z miáſtá Dawidowego które jeſt Syon.
Y zeƺli śię do Królá Sálomoná wƺyſcy mężowie Izráelſcy / mieśiącá Wrześniá w urocżyſte święto : <i>á</i> ten mieśiąc <i>jeſt</i> śiódmy.
A gdy śię zeƺli wƺyſcy ſtárƺy Izráelſcy / wźięli Kápłani ſkrzynię.
Y przenieśli ſkrzynię PAńſką / y namiot zgromádzenia / y wƺyſtkie nacżynia święte / które <i>były</i> w namiećie / á przenieśli je Kápłani y Lewitowie.
Lecż Król Sálomon / y wƺyſtko mnóſtwo Izráelſkie / które śię zeƺło do niego z nim przed ſkrzynią ofiárowáli Owce y woły / których nie licżono áni ráchowano dla mnóſtwá.
Wnieśli tedy Kápłani ſkrzynię przymierza PAńſkiego / ná miejſce jey / do wnętrznego domu / do Świątnice świętych / pod ſkrzydłá Cherubinów :
Abowiem Cherubinowie mieli rozćiągnione ſkrzydłá nád miejſcem ſkrzynie / á okrywáli Cherubinowie ſkrzynię y drążki jey z wierzchu.
Y powyćiągáli one drążki ; ták / że widáć było końce ich w świątnicey ná przodku świątnice świętych : ále nie widáć ich było ze wnątrz : y támże były áż do dniá tego.
Nic nie było w ſkrzyni tylko dwie tablice kámienne / które tám był zchował Mojzeƺ ná Horebie / gdy ſtánowił przymierze PAN z Synámi Izráelſkimi / gdy ƺli z źiemie Egipſkiey.
Y ſtáło śię gdy wychodźili Kápłani z świątnice / że obłok nápełnił dom PAńſki.
Ták yż śię nie mogli Kápłani oſtáć / y ſłużyć dla onego obłoku : ábowiem nápełniłá byłá chwałá PAńſka dom PAńſki.
Tedy rzekł Sálomon : PAN powiedźiał / Yż miał mieƺkáć we mgle.
Jużem zbudował dom ná mieƺkánie tobie / miejſce / ábyś tám przebywał ná wieki.
Y obróćił Król oblicże ſwoje / y błogoſłáwił wƺyſtkiemu zgromádzeniu Izráelſkiemu. A wƺyſtko zgromádzenie Izráelſkie ſtało.
Y rzekł : Błogoſłáwiony PAN Bóg Izráelſki / który mówił uſty ſwymi do Dawidá Ojcá mego / y ſkutecżnie <i>to</i> wypełnił / mówiąc :
Ode dniá / któregom wywiódł lud mój Izráelſki z Egiptu / nie obrałem miáſtá ze wƺyſtkich pokoleń Izráelſkich / ku zbudowániu domu / gdźieby przebywáło imię moje : álem obrał Dawidá áby był nád ludem mojim Izráelſkim :
Poſtánowiłći był w prawdźie w ſercu <i>ſwym</i> Dawid Oćiec mój zbudowáć dom imieniowi PANA Bogá Izráelſkiego :
Ale rzekł PAN do Dawidá / ojcá mego : Acżkolwiekeś poſtánowił w ſercu twem / zbudowáć dom imieniowi memu / y dobrześ ucżynił / żeś to umyślił w ſercu twojim :
Wƺákże ty nie będźieƺ budował tego domu : ále Syn twój / który wynidźie z biódr twojich / ten zbuduje dom Imieniowi memu.
A ták utwirdźił PAN ſłowo ſwoje / które był powiedźiał : Bom <i>ja</i> powſtał miáſto Dawidá Ojcá mego / y uśiadłem ná ſtolicy Izráelſkiey / jáko był powiedźiał PAN / y zbudowałem dom Imieniowi PANA Bogá Izráelſkiego.
Y náznácżyłem tám miejſce ſkrzyni / w którey jeſt przymierze PAńſkie / które ucżynił z Ojcy náƺymi / gdy je wywiódł z źiemie Egipſkiey.
Tedy ſtánął Sálomon przed Ołtarzem PAńſkim / przed wƺyſtkim zgromádzeniem Izráelſkim / y wyćiągnął ręce ſwoje ku niebu :
Y rzekł ; PAnie Boże Izráelſki / niemáƺ tobie podobnego BOgá / ná niebie wzgórę / áni ná źiemi niſko : który chowaƺ umowę / y miłośierdźie ſługom twym / którzy chodzą przed tobą cáłym ſercem ſwojim :
Któryś ſpełnił ſłudze twemu Dawidowi Ojcu memu / coś mu powiedźiał : Coś mówił uſty ſwymi / toś ſkutecżnie wypełnił / jáko śię dniá tego pokázuje.
Przetoż teraz o PANIE BOże Izráelſki / źiść ſłudze twemu Dawidowi Ojcu memu / coś mu powiedźiał / mówiąc ; Nie będźie odjęty przed twarzą moją z narodu twego mąż / któryby śiedźiał ná ſtolicy Izráelſkiey : jeſli tylko będą przeſtrzegáli Synowie twoji drogi ſwey / chodząc przede mną / jákoś ty chodźił przed oblicżnośćią moją.
Przetoż teraz o BOże Izráelſki / niech będźie utwirdzone proƺę ſłowo twoje / któreś mówił do ſługi twego Dawidá / Ojcá mego.
( Acżći w prawdźie / izali Bóg będźie mieƺkał ná źiemi? Oto niebioſá / y niebá niebios / nie mogą ćię ogárnąć ; jákoż dáleko mniey ten dom którym zbudował? )
A wejrzy ná modlitwę ſługi twego / y ná proźbę jego / o PAnie Boże mój / wyſłuchaj wołánie y modlitwę / którą śię dźiś ſługá twój modli przed tobą :
A niech będą otworzone ocży twoje nád tym domem / w nocy y we dnie : nád tym miejſcem / o którymeś powiedźiał ; Tu będźie Imię moje : ábyś wyſłuchiwał modlitwę / którą śię będźie modlił ſługá twój ná miejſcu tym.
Wyſłuchajże proźby ſługi twego / y ludu twego Izráelſkiego / który śię modlić będźie ná tym miejſcu. Ty wyſłuchaj z miejſcá mieƺkánia twego / z niebá : á wyſłuchawƺy / bądź miłośćiw.
Gdyby cżłowiek zgrzeƺył przećiwko bliźniemu ſwemu / á przywiódłby go do przyśięgi / ták żeby przyśięgáć muśiał / á przyƺłáby tá przyśięgá przed Ołtarz twój / w tym domu :
Ty wyſłuchaj z niebá / á rozeznaj y rozſądź ſługi twoje / potępiájąc niezbożnego / y obrácájąc ſpráwy jego ná głowę jego ; á uſpráwiedliwiájąc ſpráwiedliwego / oddawájąc mu według ſpráwiedliwośći jego.
Gdyby był poráżony lud twój Izráelſki od nieprzyjaćielá / przeto iż zgrzeƺyli przećiw tobie / á náwróćiliby śię do ćiebie / wyznawájąc Imię twoje / á modląc śię przepraƺáliby ćię w tym domu :
Ty wyſłuchaj z niebá / y odpuść grzech ludowi twemu Izráelſkiemu ; á przywróć je záśię do źiemie / którąś dał Ojcom ich.
Gdyby záwárte było niebo / á nie byłoby dżdżu / przeto że zgrzeƺyli przećiwko tobie / á modliliby śię ná tym miejſcu / wyznawájąc Imię twoje / á od grzechów ſwojich odwróćiliby śię / gdybyś je utrapił :
Ty wyſłuchaj z niebá / á odpuść grzech ſług twojich / y ludu twego Izráelſkiego / náucżywƺy ich drogi práwey / po którey chodźić máją ; á daj deƺcż ná źiemię twoję / którąś dał w dźiedźictwo ludowi twemu.
Byłliby głód ná źiemi / byłliby mor / ſuƺa / rdzá / ƺáráńcża / jeſliby były chrząƺcże / jeſliby go zćiſnął nieprzyjaćiel jego w źiemi mieƺkánia jego / <i>álbo</i> jákakolwiek plagá / <i>álbo</i> jákakolwiek niemoc :
Wƺelką modlitwę / <i>y</i> wƺelką proźbę / którąby cżynił którykolwiek cżłowiek / <i>álbo</i> wƺyſtek lud twój Izráelſki ; ktoby jedno uznał ránę ſercá ſwego / y wyćiągnąłby ręce ſwe w domu tym :
Ty wyſłuchaj z niebá / z miejſcá mieƺkánia twego / á odpuść / y ucżyń / y oddaj káżdemu według wƺelkich dróg jego / które znaƺ w ſercu jego / ( bo ty / ty ſam znaƺ ſercá wƺyſtkich ſynów ludzkich. )
Aby śię ćiebie bali po wƺyſtkie dni / które żyć będą ná źiemi / którąś dał Ojcom náƺym.
Nád to / y Cudzoźiemiec / który nie jeſt z ludu twego Izráelſkiego / przydźieli z źiemie dálekiey dla Imienia twego :
( Bo uſłyƺą o Imieniu twojim wielkiem / y o ręce twojey możney / y o rámieniu twojim wyćiągnionym. ) Przydźieli tedy / á będźie śię modlił w tym domu :
Ty wyſłuchaj z niebá / z miejſcá mieƺkánia twego ; á ucżyń wƺyſtko o co záwoła do ćiebie on cudzoźiemiec : áby poználi wƺyſcy narodowie źiemſcy Imię twoje / y bali śię ćiebie / jáko lud twój Izráelſki : á żeby wiedźieli że Imię twoje wzywáne jeſt nád tym domem / którym zbudował.
Gdyby wyƺedł lud twój ná wojnę / przećiwko nieprzyjaćielowi ſwemu / drogą którą je pośleƺ / á modliliby śię PANU / obróćiwƺy śię ku miáſtu któreś obrał / y ku domowi którym zbudował Imieniowi twemu :
Wyſłuchajże z niebá modlitwę ich / y proźbę ich / á wykonaj ſąd ich.
Gdyby też zgrzeƺyli przećiwko tobie / ( bo niemáƺ cżłowieká któryby nie grzeƺył / ) á rozgniewawƺy śię ná nie podałbyś je pod moc nieprzyjaćielowi / któryby je pojimawƺy / záwiódł je w niewolą / do źiemie nieprzyjaćielſkiey dálekiey álbo bliſkiey :
A upámiętáliby śię w ſercu ſwojim / w oney źiemi / do którey ſą záprowádzeni w niewolą / y náwróćiliby śię / á przepraƺáliby ćię w źiemi tych którzy je pojimáli / mówiąc ; Zgrzeƺyliſmy / y źleſmy ucżynili / niepobożnieſmy śię ſpráwowáli :
A ták náwróćiliby śię do ćiebie z cáłego ſercá ſwego / y z cáłey duƺe ſwey / w źiemi nieprzyjaćiół ſwojich którzy je pojimáli / á modliliby śię tobie / obróćiwƺy śię ku źiemi ſwey / którąś dał Ojcom ich / ku miáſtu / któreś obrał / y ku domowi którym zbudował Imieniowi twemu :
Wyſłuchajże tedy z niebá / z miejſcá mieƺkánia twego modlitwę ich / y proźbę ich / á wykonaj ſąd ich.
A bądź miłośćiw ludowi twemu / który przećiw tobie zgrzeƺył / y wƺyſtkim niepráwośćiam ich / którymi wyſtąpili przećiw tobie : á nákłoń ku nim miłośierdźia tych / którzy je pojimáli áby śię zmiłowáli nád nimi :
Ponieważ ſą ludem twojim / y dźiedźictwem twojim / któreś wywiódł z Egiptu / z pośrzodku piecá żeláznego.
Niech będą ocży twoje otwárte ná proźbę ſługi twego / y ná proźbę ludu twego Izráelſkiego : ábyś je wyſłuchał we wƺyſtkim / o co ćię wzywáć będą.
Abowiemeś je ty ſobie odłącżył zá dźiedźictwo ze wƺyſtkich narodów źiemie : jákoś powiedźiał przez Mojzeƺá ſługę twego / gdyś wywiódł Ojce náƺe z Egiptu / o PAnie Boże!
Y ſtáło śię gdy Sálomon modląc śię PAnu dokonał wƺyſtkiey oney modlitwy y proźby / że wſtał od Ołtarzá PAnſkiego / á <i>przeſtał</i> klęcżeć y podnośić rąk ſwojich ku niebu.
A ſtojąc / błogoſłáwił wƺyſtkiemu zgromádzeniu Izráelſkiemu / wielkim głoſem / mówiąc ;
Błogoſłáwiony PAN / który dał odpocżnienie ludowi ſwemu Izráelſkiemu / według wƺyſtkiego co powiedźiał : nie chybiło żadne ſłowo / ze wƺyſtkich ſłów jego dobrych / które mówił przez Mojzeƺá ſługę ſwego.
Niechże będźie PAN Bóg náƺ z námi / jáko był z Ojcy náƺymi / niech nas nie opuƺcża / áni nas odrzuca.
Ale niech nákłoni ſerce náƺe ku ſobie / żebyſmy chodźili po wƺyſtkich drogách jego / ſtrzegąc rozkazánia jego / y wyroków jego / y ſądów jego / które przykazał Ojcom náƺym.
A niech będą te ſłowá moje / którymim śię modlił przed PANEM / bliſkie PANA Bogá náƺego / we dnie y w nocy / áby wykonywał ſąd ſługi ſwego / y ſąd ludu ſwego Izráelſkiego ſądząc káżdą ſpráwę dniá ſwojego.
Żeby poználi wƺyſcy narodowie źiemſcy / iż PAN ſam jeſt Bogiem / á nikt inƺy.
Niechże tedy będźie ſerce wáƺe doſkonáłe ku PANU BOgu náƺemu / ábyśćie chodźili w wyrokách jego / á ſtrzegli przykazań jego / jáko y dniá dźiśiejƺego.
Tedy Król / y wƺyſtek Izráel z nim / ſpráwowáli Ofiáry przed PAnem.
Y Ofiárował Sálomon ofiárę ſpokojną / którą ſpráwował PAnu / Wołów dwádźieśćiá y dwá tyśiącá / y owiec ſto y dwádźieśćiá tyśięcy. A ták poświącáli Dom PAńſki / Król / y wƺyſcy Synowie Izráelſcy.
Onegoż dniá poświęćił Król pośrodek śieni / która byłá przed domem PAńſkim : bo tám Ofiárował cáłopalenie / y Ofiárę śniedną ; y tłuſtośći Ofiar ſpokojnych : przeto / że Ołtarz miedźiány / który był przed PAnem / był máły / y nie mogły śię ná nim zmieśćić cáłopalenia / y Ofiáry śniedne / y tłuſtośći Ofiar ſpokojnych.
A ták obchodźił Sálomon ná on cżás święto zacne / y wƺyſtek Izráel z nim : zgromádzenie wielkie / od weśćia do Emát / áż do rzeki Egipſkiey przed PAnem Bogiem náƺym przez śiedm dni / y przez śiedm dni / <i>to jeſt.</i> przez cżternaśćie dni.
A dniá óſmego rozpuśćił lud : Którzy błogoſłáwiąc Królowi / rozeƺli śię do przybytków ſwojich / weſeląc śię / y ćieƺąc śię w ſercu <i>ſwym,</i> ze wƺyſtkiego dobrego / które ucżynił PAN Dawidowi ſłudze ſwemu / y Izráelowi ludowi ſwemu.
