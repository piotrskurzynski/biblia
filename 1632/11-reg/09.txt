Y ſtáło śię / gdy dokońcżył Sálomon budowánia Domu PAńſkiego / y domu królewſkiego / y wƺyſtkiego co żądał Sálomon y chćiał ucżynić :
Że śię PAN ukazał Sálomonowi powtóre / jáko mu śię był ukazał w Gábáon.
Y rzekł PAN do niego : Wyſłuchałem modlitwę twoję / y proźbę twoję / którąś śię modlił przede mną / á poświęćiłem ten dom któryś zbudował / áby tám przebywáło imię moje áż ná wieki ; y będą tám ocży moje y ſerce moje po wƺyſtkie dni.
A jeſli ty będźieƺ chodźił przede mną / jáko chodźił Dawid Oćiec twój / w doſkonáłośći ſercá / y w proſtośći / á będźieƺ śię ſpráwował według wƺyſtkiego com ći przykazał / ſtrzegąc wyroków mojich y ſądów mojich :
Tedy utwirdzę ſtolicę króleſtwá twego nád Izráelem ná wieki / jákom powiedźiał Dawidowi Ojcu twemu / mówiąc : Nie będźie odjęty z narodu twego mąż / z ſtolice Izráelſkiey.
Ale jeſli śię ná zad odwróćićie wy / y Synowie wáƺy ode mnie / á nie będźiećie ſtrzegli przykazań mojich / <i>y</i> wyroków mojich / którem wam podał ; ále odƺedƺy będźiećie ſłużyli Bogom cudzym / y będźiećie śię im kłaniáli :
Tedy wytrácę Izráelá z źiemie / którąm im dał / á Dom którym poświęćił Imieniowi memu odrzucę od oblicża mego / á będźie Izráel przypowieśćią y baśnią miedzy wƺyſtkimi narodámi.
A ták y ten dom który był ſławny / káżdemu mimo idącemu będźie ná podźiw y ná poświſtánie / y rzecże : Przecżże ták ucżynił PAN tey źiemi y temu domowi?
Tedy odpowiedzą : Przeto / iż opuśćili PANA Bogá ſwego / który wywiódł Ojce ich z źiemie Egipſkiey / á chwyćili śię Bogów cudzych / y kłaniáli śię im / á ſłużyli im : Dla tegoż przywiódł PAN ná nie to wƺyſtko złe.
Y ſtáło śię po wyśćiu dwudźieſtu lat / których zbudował Sálomon owe obá domy / Dom PAńſki / y dom Królewſki :
<i>Do cżego</i> Hirám król Tyrſki nádał był Sálomonowi drzewá Cedrowego / y drzewá jodłowego / y złotá ile jedno chćiał ; tedy <i>też</i> Król Sálomon dał Hirámowi dwádźieśćiá miaſt w źiemi Gálilejſkiey.
Y wyjáchał Hirám z Tyru / áby oglądał miáſtá / które mu dał Sálomon : ále mu śię niepodobáły.
Y rzekł : Cóż to zá miáſtá któreś mi dał Bráćie mój? Y názwał je źiemią Chábul / áż do dniá tego.
Abowiem poſłał był Hirám Królowi / ſto y dwádźieśćiá talentów złotá.
A przycżyná poboru który był rozkazał wybieráć Król Sálomon <i>byłá,</i> áby zbudował Dom PAńſki / y dom ſwój / y Mello / y mury Jeruzálemſkie / y Háſor / y Mágeddo / y Gázer.
Fáráo <i>bowiem</i> Król Egipſki / wyćiągnął był / y wźiął Gázer / y popalił je ogniem / á Chánánejcżyká który mieƺkał w tym mieśćie wymordował / á dał je zá poſág córce ſwey / żenie Sálomonowey.
A ták zbudował Sálomon Gázer y Betoron niżƺe.
Przytym Báálát / y Tádmor ná puƺcży w teyże źiemi :
Y wƺyſtkie miáſtá w których miał ſkłády Sálomon / y miáſtá wozów / y miáſtá jezdnych : <i>y wƺyſtko</i> według żądośći Sálomonowey : cożolwiek chćiał budowáć w Jeruzálem y ná Libánie / y we wƺyſtkiey źiemi Páńſtwá ſwojego.
Wƺyſtek tákże lud / który był pozoſtał z Amorrejcżyków / Hettejcżyków / Ferezejcżyków / Hewejcżyków / y Jebuzejcżyków / którzy nie byli z Synów Izráelſkich :
<i>To jeſt,</i> Syny ich / którzy byli pozoſtáli po nich w źiemi / których nie mogli Synowie Izráelſcy wytráćić / ucżynił Sálomon hołdownikámi <i>y</i> niewolnikámi áż do dniá dźiśiejƺego.
Ale z Synów Izráelſkich / nie ucżynił Sálomon żadnego niewolnikiem / jedno byli ludźmi rycerſkimi / y ſługámi jego / y Kśiążęty / y Hetmány jego / y przełożonymi nád wozámi jego / y nád jezdnymi jego.
<i>Było</i> tych przedniejƺych z przełożonych / którzy byli nád robotą Sálomonową / pięć ſet y pięćdźieśiąt : co byli nád ludźmi wykonywájącymi robotę.
Lecż córká Fáráonowá przeprowádźiłá śię z miáſtá Dawidowego / do domu ſwego / który jey zbudował <i>Sálomon</i> : tedy zbudował y Mello.
Y Ofiárował Sálomon trzy kroć ná káżdy rok cáłopalenia / y ſpokojne Ofiáry / ná Ołtarzu / który był zbudował PAnu : ále kádźił ná onym <i>Ołtarzu</i> który był przed PAnem / gdy dokonał domu.
Okrętów też nábudował Król Sálomon w Aſiongáber / które jeſt podle Elotu / nád brzegiem morzá cżerwonego w źiemi Edomſkiey.
Y poſłał Hirám ná tychże okrętách ſługi ſwe / żeglarze / świádome morzá / z ſługámi Sálomonowymi.
Którzy przypłynąwƺy do Ofir / wźięli z támtąd złotá / cżtery ſtá y dwádźieśćiá talentów / y przywieſli <i>je</i> do Królá Sálomoná.
