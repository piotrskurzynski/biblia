A to wiedz / iż w oſtátecżne dni náſtáną cżáſy trudne.
Abowiem będą ludźie ſámi śiebie miłujący / łákomi / chlubni / pyƺni / bluźniercy / rodźicom niepoſłuƺni / niewdźięcżni / niepobożni :
Bez przyrodzoney miłośći / przymierza nietrzymájący / potwarcy / niepowśćiągliwi / nieſkromni / dobrych niemiłujący :
Zdrajcy / ſkwápliwi / nádęći / roſkoƺy rácżey miłujący niż miłujący Bogá :
Którzy máją kƺtałt pobożnośći / ále śię ſkutku jey zápárli : Y tych śię chroń.
Abowiem z tych ſą ći którzy śię wrywáją w domy / y pojimáne wiodą niewiaſtki / grzechámi obćiążone / które uwodzą rozmájite pożądliwośći :
Które śię záwżdy ucżą / á nigdy do znájomośći prawdy przyść nie mogą.
A jáko Jánnes y Jámbres zprzećiwiáli śię Mojzeƺowi : ták y ći zprzećiwiáją śię práwdźie : ludźie rozumu ſkáżonego / odrzuceni z ſtroney wiáry.
Ale nie poſtąpią dáley : ábowiem głupſtwo ich jáwne będźie wƺyſtkim / jáko y onych było.
Aleś ty doƺedł náuki mojey / ſpoſobu żywotá mego / przedśięwźięćia wiáry / nieſkwápliwośći / miłośći y ćierpliwośći :
Prześládowánia / ućierpienia ; które mię potkáły w Antyochiey / w Ikonium <i>y</i> w Liſtrzech / jákiem prześládowánia podejmował : á ze wƺyſtkich wyrwał mię PAn.
Aleć y wƺyſcy którzy chcą pobożnie żyć w CHryſtuśie JEzuśie / prześládowáni będą.
Lecż źli ludźie y zwodźićiele / poſtąpią w gorƺe / <i>jáko</i> zwodzący <i>ták</i> y zwiedźieni.
Ale ty trwaj w tym cżegoś śię náucżył / y cżegoć powierzono ; wiedząc od kogoś śię tego náucżył.
A iż z dźiećińſtwá umieƺ Piſmá święte / które ćię mogą ucżynić mądrym ku zbáwieniu / przez wiárę która jeſt w CHryſtuśie JEzuśie.
Wƺyſtko Piſmo od Bogá <i>jeſt</i> nátchnione / y pożytecżne ku náuce / ku ſtrofowániu / ku nápráwie / ku ćwicżeniu / które jeſt w ſpráwiedliwośći :
Aby cżłowiek Boży był doſkonáły / ku wƺelkiey ſpráwie dobrey doſtátecżnie wyćwicżony.
