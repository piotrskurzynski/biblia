A Noemi miáłá powinowátego po mężu ſwym / cżłowieká możnego / z domu Elimelechowego / którego zwano Booz.
Y rzekłá Ruthá Moábitká do Noemi : pójdę / proƺę / ná pole / á niech zbieram kłoſy zá tym / przed którego ocżymá łáſkę znajdę. A oná rzekłá : Idź córko mojá.
Szłá tedy / á przyƺedƺy zbieráłá ná polu zá żeńcámi : y tráfiło śię / że przyƺłá ná dźiał polá Boozowego / który był z domu Elimelechowego.
A w tym przyƺedł Booz z Betlehem / y rzekł do żeńców / PAN z wámi : A oni mu odpowiedźieli / Niechżeć PAN błogoſłáwi.
Rzekł tedy Booz do ſługi ſwego / który był przyſtáwem nád żeńcámi : Cżyjáż to dźiewecżká?
Y odpowiedźiał mu ſługá on / który był przyſtáwem nád żeńcámi / y rzekł : Tá dźiewecżká jeſt Moábitká / która przyƺłá z Noemi z źiemie Moábſkiey.
Y rzekłá <i>mi,</i> Niech proƺę zbieram y zgromadzam <i>kłoſy</i> miedzy ſnopámi zá żeńcámi : á przyƺedƺy báwi śię tu od ſámego poránku áż do tąd : á bárzo máło domá śiedźi.
Tedy rzekł Booz do Ruthy : Słuchaj mię córko mojá : Nie chodź zbieráć <i>kłoſów</i> ná inƺe pole : y nie odchodź z tąd / ále śię tu trzymaj dźiewek mojich.
Pilnuj tego polá / ná którym żąć będą / á chodź zá nimi : bom rozkazał ſługom mojim / żeby śię ćiebie <i>żaden</i> nie tykał : á jeſli uprágnieƺ / idź do nacżynia / y nápji śię z tego / co cżerpáją ſłudzy moji.
Tedy oná upadƺy ná oblicże ſwoje / á ukłoniwƺy śię áż do źiemie / rzekłá do niego : Z kądżem nálázłá łáſkę w ocżách twojich / iż mię znaƺ / gdyżem jeſt cudzoźiemká?
Y odpowiedźiał Booz / á rzekł jey : Powiedźiano mi zápewne wƺyſtko / coś ucżyniłá świekrze twojey / po śmierći mężá twego / á jákoś opuśćiwƺy Ojcá twego y mátkę twoję / y źiemię w któreyś śię urodźiłá / przyƺłá do ludu / któregoś nie znáłá przed tym.
Niechżeć odda PAN ucżynek twój / y niech będźie zapłátá twojá doſkonáła od PAná Bogá Izráelſkiego : gdyżeś przyƺłá / ábyś nádźieję miáłá pod ſkrzydłámi jego.
A oná rzekłá : ználázłám łáſkę w ocżách twojich Pánie mój / gdyżeś mię poćieƺył / á mówiłeś do ſercá ſłużebnice twojey : choćiam ja nie jeſt jáko jedná z ſłużebnic twojich.
Y rzekł jey Booz : gdy będźie cżás jedzenia / przychodź tu / á jedz chleb / omocżywƺy ſtucżkę twoję w ocćie. Y uśiádłá przy żeńcách : y podał jey prażmá / które jádłá áż do ſytośći / y jeƺcże jey zbyło.
Potym wſtáłá áby zbieráłá : á Booz rozkazał ſługom ſwojim / mówiąc : Niech y miedzy ſnopámi zbiera / á nie brońćie jey <i>tego :</i>
Owƺem umyślnie upuƺcżajćie jey z ſnopów / y zoſtawiajćie áby zbieráłá / á nie fukajćie ná nię.
A ták zbieráłá ná onym polu áż do wiecżorá. Y wymłóćiłá to co zebráłá : y miáłá jákoby z Efę jęcżmieniá :
A wźiąwƺy to / ƺłá do miáſtá / y oglądáłá świekrá jey <i>to</i> co názbieráłá : á wyjąwƺy dáłá jey y to co jey zoſtáło / gdy śię nájádłá.
Y rzekłá do niey świekrá jey : kędyżeś dźiś zbieráłá / á gdźieś robiłá? niechajże ten / który ná ćię miał bacżenie / błogoſłáwionym będźie. Y oznájmiłá świekrze ſwey u kogo robiłá / mówiąc : Imię mężá / u któregom dźiś robiłá / Booz.
Potym rzekłá Noemi do Synowy ſwojey : Niech będźie błogoſłáwionym od PANá / który nie záwśćiągnął miłośierdźia ſwego / od żywych y od umárłych. Nád to jeƺcże rzekłá Noemi : Ten mąż jeſt powinowátym náƺym y z pokrewnych náƺych.
Rzekłá jey też Ruth Moábitká : Nád to mi jeƺcże mówił <i>on mąż :</i> Trzymaj śię cżeládźi mojey / póki nie pożną wƺyſtkiego zbożá mego.
Tedy rzekłá Noemi do Ruthy Synowey ſwey : Dobrze córko mojá / iż będźieƺ chodźiłá z dźiewkámi jego / żebyć <i>kto</i> przećiwnym nie był ná polu innym.
Przetoż śię trzymáłá ſłużebnic Boozowych / y zbieráłá kłoſy / póki śię nie ſkońcżyło żniwo jęcżmienne / y żniwo pƺenicżne. Potym mieƺkáłá u świekry ſwojey.
