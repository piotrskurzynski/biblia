Potym zebrał Mojzeƺ wƺyſtko zgromádzenie Synów Izráelſkich / y mówił do nich : Te ſą rzecży / które rozkazał PAN / ábyśćie je cżynili.
Przez ƺeść dni odpráwowána będźie robotá / ále dźień śiódmy będźie wam święty / Sábbát odpoczynienia Páńſkiego : ktoby weń robił robotę / umrze.
Nie rozniećićie ogniá we wƺyſtkich mieƺkániách wáƺych / w dźień Sábbátu.
Rzekł też Mojzeƺ do wƺyſtkiego zgromádzenia Synów Izráelſkich / mówiąc : Táć jeſt rzecz którą przykazał PAN / mówiąc :
Złóżćie od śiebie podárek PANU : káżdy kto jeſt ochotnego ſercá / przynieśie ten podárek PANU / złoto / y śrebro y miedź /
Y Hyácynt / y ƺárłat y kármázyn dwá kroć fárbowány / y biały jedwab / y śierść koźią /
Skóry też báránie cżerwono fárbowáne / y ſkóry borſukowe / y drzewo Syttym /
Y oliwę do świecenia / y rzecży wonne ná olejek pomázowánia / y dla kádzenia wonnego.
Kámienie też Onychyny / y kámienie do oſadzánia Nárámienniká y nápierſniká.
A wƺyſcy dowćipnego ſercá miedzy wámi przyjdą / y robić będą cokolwiek rozkazał PAN /
Przybytek / Namiot jego / y przykryćie jego / hacżyki jego / y deſki jego / drągi jego / ſłupy jego / y podſtáwki jego :
Skrzynię y drążki jey / ubłagálnią / y oponę / <i>do</i> zaſłony :
Stół y drążki jego / ze wƺyſtkim naczyniem jego / y chleby pokłádne :
Y świecznik do świecenia / z naczyniem jego / y lámpy jego / y oliwę do świecenia.
Ołtarz tákże do kádzenia z drążkámi jego / y olejek pomázowánia / y kádzenia wonne / y zaſłonę drzwi / do drzwi Przybytku.
Ołtarz do cáłopalenia / y kratę jego miedźiáną / drążki jego / y wƺyſtkie naczynia jego / wánnę z ſtolcem jey :
Opony do śieni / ſłupy jey / y podſtáwki jey / y zaſłonę do drzwi u śieni /
Kołki do przybytku / y kołki do śieni z ƺnurámi jey :
Száty ſłużebne do uſługowánia w świątnicy / ƺáty święte Aáronowi kápłánowi / y ƺáty Synom jego / dla ſpráwowánia urzędu kápłáńſkiego.
Wyƺło tedy wƺyſtko zgromádzenie Synów Izráelſkich / od oblicżnośći Mojzeƺowey :
Y przyƺedł káżdy mąż którego pobudźiło ſerce jego / y káżdy w którym dobrowolny był duch jego / przynieśli podárek PANU / do robienia namiotu zgromádzenia / y do wƺelkiey potrzeby jego / y ná ƺáty święte.
Przychodźili tedy mężowie z niewiáſtámi / káżdy kto był ochotnego ſercá przynośili zapony / y náuƺnice / y pierścienie / y mánele / <i>y</i> wƺelákie nacżynia złote / y ktokolwiek przynośił ofiárę złotą PANU.
Káżdy też co miał Hyácynt / y ƺárłat / y kármázyn dwá kroć fárbowány / y biały jedwab / y śierść koźią / y ſkóry báránie czerwono fárbowáne / y ſkóry borſukowe / przynośili.
Ktokolwiek ofiárował podárek śrebrá / y miedźi / przynośili ná Ofiárę PANU : káżdy też co miał drzewo Syttym / ná wƺeláką potrzebę ku uſłudze przynośili.
Y wƺyſtkie niewiáſty dowćipnego ſercá / rękámi ſwemi przędły / á przynośiły co náprzędły / Hyácynt y ƺárłat / kármázyn dwá kroć fárbowány / y biały jedwab.
A wƺyſtkie niewiáſty / których pobudźiło ſerce ich umiejętne / przędły śierść koźią.
Przełożeni záśię / przynośili kámienie Onychyny / y kámienie do oſadzánia Nárámienniká y napierſniká.
Tákże rzecży wonne / y oliwę do świecenia / y ná olejek pomázowánia / y ná wonne kádzenia :
Káżdy mąż y niewiáſtá / w których ochotne ſerce było do ofiárowánia / ná káżdą robotę którą rozkazał PAN czynić przez Mojzeƺá / przynośili Synowie Izráelſcy / ofiárę dobrowolną PANU.
Zátym rzekł Mojzeƺ do Synów Izráelſkich : Oto wezwał PAN z imienia Beſáleelá Syná Urowego / Syná Hurowego / z pokolenia Judy.
Y nápełnił go Duchem Bożym / mądrośćią / rozumem / y umiejętnośćią wƺelákiego rzemioſłá.
Y ku dowćipnemu wymyślániu <i>cokolwiek</i> może być urobiono ze złotá / y śrebrá / y z miedźi /
Do rzezánia kámienia ku oſadzániu / y ná wyrobienie drzewá / do czynienia wƺelákiey roboty zmyślney :
Dał nád to do ſercá jego / áby uczyć mógł <i>inƺych,</i> on / y Aholiáb / Syn Achyſámechów / z pokolenia Dán :
Nápełnił je mądrośćią ſercá áby robili wƺelákie rzemioſło ćieśielſkie / y háftárſkie / y tkáckie z Hyácyntu / y z ƺárłatu / z kármázynu dwá kroć fárbowánego / y z białego jedwabiu / tkácką robotą / áby robili káżdą robotę / dowćipnie wymyślájąc.
