Potym tedy przyƺli Mojzeƺ y Aáron / y mówili do Fáráoná : Ták mówi PAN / Bóg Izráelſki : Puść lud mój / áby mi obchodźili święto ná puƺczy.
Y rzekł Fáráo : Któż <i>jeſt</i> PAN / żebym miał ſłucháć głoſu jego / y puśćić Izráelá? Nie znam PANA / á Izráelá też nie puƺczę.
Y odpowiedźieli : Bóg Hebrejcżyków zábieżał nam : pójdźiemy teraz drogą trzech dni ná puƺczą / ábyſmy ofiárowáli PANU Bogu náƺemu / by ſnadź nie przepuśćił ná nas moru / álbo miecża.
Y rzekł do nich Król Egipſki : Przecż ty Mojzeƺu y Aáronie odrywaćie lud od robót ich? Idźćież do robót wáƺych.
Rzekł też Fáráo : Oto wielki teraz jeſt ten lud w źiemi / á wy je odrywaćie od robót ich.
Rozkazał tedy Fáráo onegoż dniá Przyſtáwom nád ludem / y Urzędnikom jego mówiąc :
Już więcey nie będźiećie dawáć plew ludowi do czynienia cegły jáko przed tym / ſámi niech idą / y zbieráją ſobie plewy.
A <i>tęż</i> licżbę cegieł którą cżynili przed tym włożyćie im / nie umniejƺyćie z niey / bo próznują : dla tego oni wołáją mówiąc : Póydźiemy / y będźiemy ofiárować Bogu náƺemu.
Niech śię przycżyni roboty mężom tym / á niech ją wykonywáją / áby nie ufáli ſłowom kłamliwym.
Wyƺedƺy tedy Przyſtáwowie nád ludem / y Urzędnicy jego / rzekli do ludu mówiąc : Ták mówi Fáráo : ja nie będę wam dawał plew.
Sámi idźćie / zbierajćie ſobie plewy gdźie znajdźiećie / bo śię namniey nie umnieyƺy z roboty wáƺey.
Y rozbieżał śię lud po wƺyſtkiey źiemi Egipſkiey / áby zbierał śćierniſko miáſto plew.
A przyſtáwowie przynagláli mówiąc : Wykonywajćie roboty wáƺe / zámiár káżdodźienny / jáko gdy wam <i>dawano</i> plewy.
Tedy bito Przyſtáwy Synów Izráelſkich które poſtánowieli nád nimi urzędnicy Fáráonowi mówiąc : przecz nie wykonywaćie zámiáru ſwego w robieniu cegieł jáko pierwey / áni wcżoráj / áni dźiś.
Y przyƺli przełożeni Synów Izráelſkich / y wołáli do Fáráoná mówiąc : Czemuż ták cżyniƺ ſługom twojim?
Plew nie dáją ſługom twojim / á mówią / cegłę róbćie : Y oto ſługi twe biją / á lud twój grzeszy.
Który rzekł : próznujećie / próznujećie : dla tegoż mówićie : Pójdźiemy / ofiárowáć będźiemy PANU.
Przetoż teraz idźćie / róbćie : á plew wam nie dádzą / ále <i>wy</i> liczbę cegieł oddawáć będźiećie.
A widząc Przełożeni Synów Izráelſkich / że źle z nimi / <i>ponieważ</i> mówiono : Nie umniejƺyćie z cegieł wáƺych / zámiáru káżdodźiennego.
Tedy oni zábieżeli Mojzeƺowi y Aáronowi / którzy ſtali áby śię z nimi potkáli / gdy wychodźili od Fáráoná.
Y rzekli do nich : Niech wejrzy PAN ná was / á rozſądźi : żeśćie nas ohydźili w oczách Fáráonowych / y w oczách ſług jego / y dáliśćie miecz w rękę ich / áby nas zábili.
Y wróćił śię Mojzeƺ do PANA / á rzekł : PANIE / cżemuś to złe wprowádźił ná lud twój / cżemuś mię <i>tu</i> poſłał?
Bo od onego cżáſu jákom wƺedł do Fáráoná / ábym mówił imieniem twojim / gorzey śię obchodźi z ludem tym / á przećięś nie wybáwił ludu twego.
