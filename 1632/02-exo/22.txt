Jesliby kto ukradł wołu álbo owcę / y zábiłby je / álbo je przedał / pięć wołów wróći zá <i>jednego</i> wołu / á cztery owce zá <i>jednę</i> owcę
( Jesliby przy podkopywániu záſtány był złodźiy / á ubity będąc umárłby / <i>kto zábił,</i> nie będźie winien krwie.
Jesliby to po weśćiu ſłońcá <i>uczynił</i> krwie winien będźie ) koniecżnie wróći : A jeſli niema / przedány będźie zá złodźiejſtwo ſwoje.
Jesli ználeźiona będźie w ręku jego rzecż krádźiona / bądź wół / bądź ośieł / bądź owcá / jeƺcze żywe / we dwój naſób wróći.
Gdyby wypásł kto pole / álbo winnicę / y puśćiłby bydlę ſwoje áby śię páſło ná polu cudzym : z nalepƺego <i>urodzáju</i> polá ſwego / y z nalepƺego <i>urodzáju</i> winnice ſwey / nádgrodźi :
Jesliby wyƺedł ogień / á tráfiłby ná ćiernie / y ſpaliłby ſtóg / álbo ſtojące zboże / álbo pole ſámo : Koniecznie nádgrodźi ten co zápalił / co zgorzáło.
Gdyby kto dał bliźniemu ſwemu śrebro / álbo naczynie do zchowánia / átoby ukrádźiono było z domu onego cżłowieká : jesliby náleźiony był złodźiey / wróći dwojáko.
Jesliby nie był ználeźiony złodźiey / ſtáwi śię Pan domu onego / przed ſędźie / <i>y przyśięże</i> że nie zćiągnął ręki ſwojey ná rzecz bliźniego ſwego.
O káżdą rzecż o którąby był ſpór / o wołu / o oſłá / o owcę / o ƺátę / o káżdą rzecż zgubioną / gdyby kto rzekł że to jeſt <i>moje,</i> przed ſędźie przydźie ſpráwá obudwu : kogo winnym znajdą ſędźiowie / nádgrodźi w dwój naſób bliźniemu ſwemu.
Jesliby kto dał bliźniemu ſwemu osłá / álbo wołu / álbo owcę / álbo inne bydlę ná chowánie / á zdechłoby / álbo okálicżáło / álbo gwałtem zájęte było / gdźieby nikt nie widźiał /
Przyśięgá Páńſka będźie miedzy obiemá / że nie zćiągnął ręki ſwey ná rzecz bliźniego ſwego / y przyjmie Pan oney rzecży <i>przyśięgę,</i> á <i>on</i> nie będźie nádgradzał.
A jesliby mu to kradzieżą wźięto / nádgrodźi Pánu rzeczy oney.
Jesliby <i>od zwierzá</i> rozƺárpáne było / poſtáwi <i>rozƺárpáne zá</i> ſwiádká / á rozƺárpánego nie nádgrodźi.
Gdyby też kto pożyczył <i>bydlęćia</i> od bliźniego ſwego / á okáliczáłoby / álbo zdechło w niebytnośći Páná jego / koniecżnie nádgrodźi.
Jesliby Pán jego był przy nim / nie będźie nádgradzał : á jesliby nájęte było / nájem tylko zápłáći.
Jesliby kto zwiódł Pánnę / która nie jeſt poślubiona / y ſpałby z nią : da jey koniecznie wiáno / <i>y</i> weźmie ją ſobie zá żonę :
Jesliby żadną miárą Oćiec jey nie chćiał mu jey dáć / odważy śrebrá według <i>zwyczáju</i> wianá Pánieńſkiego.
Cżárownicy żyć nie dopuśćiƺ :
Káżdy ktoby śię złączał z bydlęćiem / śmierćią umrze.
Ktoby ofiárował Bogom oprócz ſámego PANA / wytrácony będźie.
Przychodniowi nie uczyniƺ krzywdy / áni go ućiśnieƺ : bośćie byli przychodniámi w źiemi Egipſkiey.
Żadney wdowy áni śieroty trapić nie będźiećie.
Jeslibyś je bez litośći trapił / á oneby wołáły do mnie / ſłyƺąc wyſłucham wołánie ich.
Y rozgniewa śię zápálczywość mojá / á pobiję was mieczem / y będą żony wáƺe wdowámi / á Synowie wáƺy śierotámi.
Jeslibyś pieniędzy pożyczył ludowi memu ubogiemu / który mieƺka z tobą / nie będźieƺ mu jáko lichwiarz / nie obćiążyćie go lichwą.
Jeſli w záſtáwie weźmieƺ ƺátę bliźniego twego / przed zachodem ſłońcá / wróćiƺ mu ją :
Bo to odźienie jego / tylko to jeſt nákryćie ćiáłá jego / ná którym ſypia : będźieli do mnie wołał / wyſłucham go / bom ja miłośierny.
Sędźiom nie będźieƺ złorzeczył / á Przełożonego ludu twego / nie będźieƺ przeklinał.
Z obfitośći <i>zboża</i> / y ćiekących rzecży twych nie omieƺkaƺ <i>pierwiaſtek ofiárowáć,</i> pierworodnego z Synów twojich oddaƺ mi.
Toż uczyniƺ z wołów twych / y z owiec twojich : śiedm dni będźie z mátką ſwoją á ósmego dniá oddaƺ mi je.
Ludem świętym będźiećie mi / á mięſa ná polu rozƺárpánego jeść nie będźiećie : pſom je wyrzućićie.
