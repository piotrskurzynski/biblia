Tákże z hyácyntu / y z ƺárłatu / z kármázynu dwá kroć fárbowánego / poczynili ƺáty do uſługi ku uſługowániu w świątnicy. Urobili też ƺáty święte Aáronowi / jáko był rozkazał PAN Mojzeƺowi.
Y uczynił nárámiennik ze złotá / z hyácyntu / y z ƺárłatu / y z kármázynu dwá kroć fárbowánego / y z białego jedwabiu kręconego.
Náklepáli też bláƺek złotych / y náſtrżygli z nich nići / do przetykánia hyácyntu / y do przetykánia ƺárłatu / y do przetykánia kármázynu dwá kroć fárbowánego / y do przetykánia białego jedwabiu / robotą háftárſką.
Nárámienniki przytym porobili ták / áby śię jeden z drugim ſpojić mógł : ná dwu krájách ich ſpajáły śię.
Pás też Nárámienniká / który był ná nim / z tegoż był / y tąż robotą : ze złotá / z hyácyntu / y z ƺárłatu / y z kármázynu dwá kroć fárbowánego / y z białego jedwabiu kręconego / jáko był PAN rozkazał Mojzeƺowi.
Do tego wygotowáli kámienie Onychyny / opráwione złotym oſádzeniem / rzezáne / jáko ryte bywáją pieczęći / z Imiony Synów Izráelſkich :
Y wpráwił je ná wierzchne kráje Nárámienniká : <i>áby były</i> kámieńmi ná pámiątkę Synom Izráelſkim / jáko był rozkazał PAN Mojzeƺowi.
Uczynił też Nápierſnik robotą háftárſką / według roboty Nárámienniká : ze złotá / z hyácyntu / y z ƺárłatu / y z kármázynu dwá kroć fárbowánego / y z białego jedwabiu kręconego.
Czworográniſty był / dwojiſty uczynili Nápierſnik : ná piędźi długość jego / y ná piędźi ƺerokość jego / dwojiſty <i>był</i>.
Y náſádźili weń cztery rzędy kámienia <i>tym</i> porządkiem : Sárdius / Topázius y Smárágd / w rzędźie pierwƺym.
A w drugim rzędźie / Kárbunkuł / Sáfir / y Jáſpis.
A w trzećim rzędźie / Linkurius / Achátes / y Ametiſt.
A w czwártym rzędźie / Chryſolit / Onychyn y Beril / wƺyſtkie oſadzone w złoto w rzędźiech ſwych.
A tych kámieni z imionámi ſynów Izráelſkich dwánaśćie / według imion ich było / ták jáko rzeżą pieczęći / káżdy według imienia ſwego / podług dwunaśći pokolenia.
Pocżynili też do Nápierſniká łáńcuƺki jednoſtájne robotą plećioną / ze złotá ƺczerego.
Spráwili też dwá haczyki złote / y dwá kolcá złote / y przypráwili one dwie kolcá / do obu krájów Nápierſniká.
A przewlekli one dwá łáńcuƺki złote / przez obie kolcá / u krájów Nápierſniká :
Drugie záś dwá kolcá / obu łáńcuƺków / záwlekli do onych dwu haczyków / y przypráwili do zwierzchnych krájów Nárámienniká / ná przodku.
Uczynili tákże dwie kolcá złote / które przypráwili do dwu końców Nápierſniká / ná kráju jego / który był po ſtronie Nárámienniká / ze ſpodku.
Uczynili jeƺcze dwie kolcá złote / które przypráwili / ná dwu ſtronách Nárámienniká / ze ſpodku / ná przodku / przećiwko ſpojeniu jego / które jeſt nád przepáſániem Nárámienniká.
Y przywiązáli Nápierſnik / od kolców jego / do kolców Nárámienniká / ƺnurem hyácyntowym / áby był nád przepáſániem Nárámienniká żeby nie odewſtawał Nápierſnik od Nárámienniká : jáko był rozkazał PAN Mojzeƺowi.
Urobił tákże Płaƺcz pod Nárámiennik / robotą tkáną / wƺyſtek hyácyntowy.
A rozpór Płaƺczá w pośrzód jego / jáko rozpór u páncerzá : y bramá około kráju jego / áby śię nie rozdźierał.
Tákże u podołká Płaƺczá onego / uczynili jábłká gránatowe / z hyácyntu / y z ƺárłatu / y z kármázynu dwá kroć fárbowánego / y <i>z białego jedwabiu</i> kręconego.
Pocżynili też dzwonki ze złotá ƺczerego / y pozáwieƺáli one dzwonki miedzy one jábłká gránatowe / u podołku Płaƺczá w około / w pośrzód jábłek gránatowych.
Dzwonek á jábłko gránatowe / <i>y záś</i> dzwonek y jábłko gránatowe / u podołká Płaƺczá w około / ku poſługowániu : jáko rozkazał PAN Mojzeƺowi.
Porobili też ƺáty z białego jedwabiu robotą tkácką Aáronowi / y ſynom jego :
Czapeczkę też z białego jedwabiu / y czapki ozdobne z białego jedwabiu / y ubiory ćienkie z białego jedwabiu kręconego :
Pás tákże z białego jedwabiu kręconego / y z hyácyntu / y z ƺárłatu / y z kármázynu dwá kroć fárbowánego / robotą háftárſką / jáko PAN rozkazał Mojzeƺowi.
Do tego uczynili bláchę korony świętobliwośći ze złotá ƺczerego / y wyryſowáli ná niey robotą rytą jáko pieczęći rżeżą : <i>ŚWIĄTOŚĆ PANU</i>.
A przypráwili do niey ƺnur hyácyntowy / áby przywiązána byłá do czapki ná wierzchu : jáko był rozkazał PAN Mojzeƺowi.
A ták ſkończyłá śię wƺyſtká robotá około Przybytku / y Namiotu zgromádzenia : Y uczynili Synowie Izráelſcy wƺyſtko / jáko był rozkazał PAN Mojzeƺowi / ták uczynili.
Y przynieśli <i>ten</i> Przybytek do Mojzeƺá / Namiot / y wƺyſtkie naczynia jego / haki jego / deſki jego / drągi jego / y ſłupy jego / y podſtáwki jego /
Przykryćie też z ſkór báránich czerwono fárbowánych / y przykryćie z ſkór borſukowych : y Oponę zaſłony :
Skrzynię świádectwá / y drążki jey / y Ubłagálnią /
Stół / wƺyſtkie Naczynia jego / y chleb pokłádny /
Święcznik ochędożny / lámpy jego / lámpy ſporządzone / y wƺyſtkie naczynia jego / y Oliwę ku świeceniu :
Ołtarz tákże złoty / y Olejek pomázowánia / y kádźidło wonne / y zaſłonę / do drzwi Namiotu /
Ołtarz miedźiány / y kratę jego miedźiáną / drążki jego / y wƺyſtkie naczynia jego / Wánnę y Stolec jey :
Opony do śieni / y ſłupy ich / z podſtáwkámi ich / y zaſłonę do bramy śienney / y ƺnury jey / y kołki jey / y wƺelákie naczynia ku ſłużbie Przybytku / <i>y</i> Namiotu zgromádzenia :
Száty ſłużebne do uſługowánia w świątnicy / ƺáty święte Aáronowi kápłánowi / y ƺáty ſynów jego / do odpráwowánia urzędu kápłáńſkiego.
Według wƺyſtkiego / jáko był rozkazał PAN Mojzeƺowi / ták uczynili ſynowie Izráelſcy wƺyſtkę tę robotę.
Y obejrzał Mojzeƺ tę wƺyſtką robotę / á oto uczynili ją jáko był rozkazał PAN / ták uczynili : y błogoſłáwił im Mojzeƺ.
