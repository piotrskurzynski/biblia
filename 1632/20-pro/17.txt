Lepƺy jeſt kęs ſuchego chlebá á w pokoju / niżeli pełen dom nábitego bydłá ze ſwárem.
Sługá roztropny będźie pánował nád ſynem / który jeſt ku háńbie ; á miedzy bráćmi będźie dźielił dźiedźictwo.
Tygiel ſrebrá á piec złotá doświádcżá ; ále Pán ſerc doświádcżá.
Zły pilnuje wárg złośliwych / á kłámcá ſłuchá języká przewrotnego.
Kto śię náśmiewá z ubogiego / uwłácżá ſtworzyćielowi jego ; á kto śię ráduje z upádku cżyjego / nie ujdźie pomſty.
Koroná ſtárców ſą ſynowie ſynów ich / á ozdobá ſynów ſą ojcowie ich.
Nie przyſtoi mowá powáżná głupiemu / dopieroż kśięćiu uſtá kłámliwe.
Jáko kámień drogi / ták bywa dar wdźięcżny temu / który go bierze ; do cżegokolwiek zmierzy / zdárzy mu śię.
Kto pokrywá przeſtępſtwo / ƺuká łáſki ; ále kto wznáwiá rzecży / rozłącżá przyjáćiół.
Więcey wáży gromienie u roztropnego / niżeli ſto plág u głupiego.
Uporny tylko złego ƺuká / dla tego poſeł okrutny będźie nań zeſłány.
Lepiej cżłowiekowi ſpotkáć śię z niedźwiedźicą ośieroćiáłą / niżeli z głupim w głupſtwie jego.
Kto oddáje złem zá dobre / nie wynijdźie złe z domu jego.
Kto zácżyná zwádę / jeſt jáko ten / co przekopuje wodę ; przetoż niż śię zwádá rozśili / zániecháj go.
Kto uſpráwiedliwiá niezbożnego / á winnym cżyni ſpráwiedliwego / obá jednáko ſą obrzydliwośćią Pánu.
Cóż po doſtátku w ręku głupiego / ponieważ do nábyćiá mądrośći rozumu nie ma?
Wƺelkiego cżáſu miłuje przyjáćiel / á w ućiſku ſtáwiá śię jáko brát.
Cżłowiek głupi dáje rękę / cżyniąc rękojemſtwo przed twárzą przyjáćielá ſwego.
Kto miłuje zwádę / miłuje grzech ; á kto wynośi uſtá ſwe / ƺuká upádku.
Przewrotny w ſercu nie znájduje dobrego ; á kto jeſt przewrotnego języká / wpádnie we złe.
Kto ſpłodźił głupiego / ná ſmutek ſwój ſpłodźił go / áni śię rozweſeli oćiec niemądrego.
Serce weſołe ocżerſtwiá jáko lekárſtwo ; ále duch ſfráſowány wyſuƺá kośći.
Niezbożny potájemnie dar bierze / áby podwróćił śćieƺki ſądu.
Ná twárzy roztropnego znáć mądrość ; ále ocży głupiego áż ná kráju źiemi.
Syn głupi żáłośćią jeſt ojcu ſwemu / á gorzkośćią rodźićielce ſwojey.
Zájiſte nie dobrá / winowáć ſpráwiedliwego / álbo żeby przełożeni kogo dla cnoty bić mieli.
Kto záwśćiągá mowy ſwe / jeſt umiejętnym ; drogiego duchá jeſt mąż rozumny.
Gdy głupi milcży / zá mądrego pocżytány bywa ; á który zátulá wárgi ſwoje / zá rozumnego.
