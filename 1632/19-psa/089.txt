Náucżájący <i>złożony</i> od Etáná Ezráhitcżyká.
O miłośierdźiách PAńſkich ná wieki śpiewáć będę : od narodu do narodu opowiádáć będę uſty ſwymi prawdę twoję.
Rzekłem bowiem : Miłośierdźie ná wieki budowáne będźie : ná niebieśiech utwirdźiłeś prawdę twoję / <i>o któreyś rzekł</i> :
Poſtánowiłem przymierze z wybránym mojim : przyśiągłem Dawidowi ſłudze ſwemu :
Że áż ná wieki utwirdzę naśienie twoje : á zbuduję od narodu do narodu ſtolicę twoję : Selá.
Przetoż PAnie wyſławiáją niebioſá cud twój : y prawdę twoję w zgromádzeniu świętych.
Abowiem / któż ná niebie przyrównány może być PANU? <i>kto</i> podobien jeſt PAnu miedzy Synámi mocárzów.
Y w zgromádzeniu świętych / bárzo jeſt Bóg ſtráƺliwy / á ſtráƺny náde wƺyſtkie / <i>którzy ſą</i> około niego.
PAnie / Boże zaſtępów / któż <i>jest</i> jákoś ty PAn mocny? bo prawdá twojá <i>jeſt</i> około ćiebie.
Ty pánujeƺ nád nádętośćią morſką : gdy śię podnoƺą náwáłnośći jego / ty je ſkracaƺ.
Tyś potárł Egipt jáko zránionego : mocą rámienia twego roſproƺyłeś nieprzyjaćioły twoje.
Twojeć <i>ſą</i> niebioſá / twojá też y źiemiá : okrąg świátá y pełność jego tyś ugruntował.
Tyś ſtworzył Północy y południe : Tábor y Hermon śpiewáją o Imieniu twojim.
Rámię twoje mocne jeſt : można jeſt ręká twojá / á wywyżƺona jeſt práwicá twojá.
Spráwiedliwość y ſąd / ſą gruntem ſtolice twojey : miłośierdźie y prawdá / uprzedzáją oblicże twoje.
Błogoſłáwiony lud / który zna dźwięk <i>twój</i>. PAnie w świátłośći oblicża twego chodźić będą.
W Imieniu twojim weſelić śię będą káżdego dniá : á w ſpráwiedliwośći twojey wywyżƺáć śię będą.
Boś ty jeſt chwałą mocy ich : á zá wolą twoją wywyżƺy śię róg náƺ.
Bo od PAná / jeſt tarcża náƺá : á <i>od</i> Świętego Izráelſkiego / Król náƺ.
W on cżás / mówiąc w widzeniu / do Świętego twego / rzekłeś : Położyłem rátunek w <i>ręku</i> mocárzá ; wywyżƺyłem wybránego z ludu.
Ználazłem Dawidá ſługę mego : olejkiem świętym mojim pomázałem go.
Przetoż ręká mojá będźie ſtałá przy nim : á rámię moje pośili go.
Nie ućiśnie go nieprzyjaćiel : á ſyn niepráwośći nie utrapi go.
Bo potrę przed twarzą jego przećiwniki jego : á te którzy go máją w nienawiśći / poráżę.
Nád to / prawdá mojá y miłośierdźie moje z nim będźie : á w Imieniu mojim wywyżƺony będźie róg jego.
Y położę ná morzu rękę jego : y ná rzekách práwicę jego.
On wołájąc rzecże : Tyś Oćiec mój / Bóg mój / y ſkáłá zbáwienia mego.
Ja go też zá pierworodnego wyſtáwię / <i>y</i> zá wyżƺe° nád Królmi źiemie.
Ná wieki mu záchowam miłośierdźie moje : á przymierze moje ſtałe będźie przy nim.
Y ucżynię / że ná wieki będźie trwáło naśienie jego : á ſtolicá jego jáko dni niebios.
<i>Ale,</i> jeſliby Synowie jego opuśćili Zakon mój / á w ſądźiech mojich niechodźili :
Jeſliby uſtáwy moje zplugáwili / á przykazań mojich nie przeſtrzegáli :
Tedy náwiedzę rózgą przeſtępſtwo ich ; á karániem niepráwość ich.
Ale miłośierdźia ſwego nie odejmę od niego : áni zkłamam przećiw prawdźie mojey.
Nie ſplugáwię przymierza mego : á tego co wyƺło z uſt mojich / nie odmienię.
Razem przyśiągł przez świętobliwość moję / że nie ſkłamam Dawidowi :
<i>A, że</i> naśienie jego zoſtánie ná wieki : á ſtolicá jego jáko ſłońce przedemną :
Jáko mieśiąc / będźie utwirdzone ná wieki / y <i>jáko</i> świádkowie ná niebie godnowierni : Selá.
Aleś <i>go</i> ty odrzućił y wzgárdźił : rozgniewałeś śię ná pomázáńcá twego.
Zrzućiłeś przymierze z ſługą twojim : ztrąćiłeś ná źiemię koronę jego.
Roztárgałeś wƺyſtkie płoty jego : y báƺtyś jego rozwálił.
Szárpáją go wƺyſcy / którzy drogą mimo idą : pośmiewiſkiem jeſt <i>y</i> ſąśiádom ſwojim.
Wywyżƺyłeś práwicę przećiwników jego / uweſeliłeś wƺyſtkie nieprzyjaćioły jego.
Y oſtrze miecżá jego ſtępiłeś : á nie rátowałeś go w bitwie.
Znioſłeś ochędóſtwo jego : á ſtolicę jego uderzyłeś o źiemię.
Ukróćiłeś dni młodośći jego : á przyodźiałeś go háńbą : Selá.
Dokądże PANie? ná wiekiż śię kryć będźieƺ? tákże będźie jáko ogień pałáć zápálcżywość twojá?
Wſpomniże ná mię / jáko krótki jeſt wiek <i>mój</i> : Azaś prózno ſtworzył wƺyſtkie Syny ludzkie?
Któż z ludźi ták żyć może áby nie oglądał śmierći? <i>któż</i> wyrwie duƺę ſwą z mocy grobu : Selá.
Gdźież ſą litośći twoje dawne o PAnie / <i>któreś</i> przyśiągł Dawidowi w prawdźie ſwey.
Wſpomni PAnie ná zelżywość ſług twojich : á jákom ponośił <i>wzgárdę</i> w zánádrzu ſwym / od wƺyſtkich narodów możnych.
PAnie jáko urągáli nieprzyjaćiele twoji : jáko urągáli śćieƺkam pomázáńcá twego.
<i>Niech będźie</i> błogoſłáwiony PAN áż ná wieki / Amen Amen.
