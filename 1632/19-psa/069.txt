Przedniejƺemu śpiewakowi ná Seſánnim / Pſálm Dawidów.
Wybaw mię o Boże / boć przyƺły wody áż do duƺe <i>mojey</i>.
Pogrążony jeſtem w głębokim błoćie / gdźie dná niemáƺ : przyƺedłem w głębokośći wód / á náwáłność <i>ich</i> porwáłá mię.
Sprácowałem śię wołájąc : wyſchło gárdło moje / uſtáły ocży moje gdym ocżekawał Bogá mojego.
Więcey jeſt tych / którzy mię máją w nienawiśći bez przycżyny / niż włoſów ná głowie mojey / zmocnili śię ći którzy mię wygubić uśiłują / <i>á ſą</i> nieprzyjaćiołmi mymi nieſłuƺnie : Cżegom nie wydárł / muśiałem nádgradzáć.
Boże / ty znaƺ głupſtwo moje / á wyſtępki moje nie ſą tájne przed tobą.
Niechajże nie będą záwſtydzeni dla mnie ći / którzy ná ćię ocżekawáją PAnie / PAnie Zaſtępów : Niech nie przychodzą dla mnie do háńby ći / którzy ćię ƺukáją / o Boże Izráelſki.
Bo dla ćiebie ponoƺę urągánie / á zelżywość okryłá oblicże moje.
Stałem śię obcym bráćij mojey : á cudzoźiemcem Synom mátki mojey.
Przeto że gorliwość domu twego zżárłá mię : á urągánie urągájących tobie przypádło ná mię.
Gdym płákał y trapił poſtem duƺę moję / ſtáło mi śię <i>to</i> poháńbieniem.
Gdym wźiął ná śię wór miáſto ƺáty / byłem u nich przypowieśćią.
Mówili o mnie ći / którzy śiedźieli w bramie : á byłem pioſnką u tych / którzy pili mocny napój.
Ale ja <i>obracam</i> modlitwę moję / do ćiebie PAnie : cżás jeſt upodobánia twego / o Boże według wielkośći miłośierdźia twego wyſłuchajże mię / dla prawdy zbáwienia twego.
Wyrwi mię z błotá / ábym nie był pogrążony : niech będę wyrwány od tych którzy mię nienawidzą / jáko z głębokośći wód.
Aby mię nie zátopiły ſtrumienie wód / y nie pożárłá głębia / y nie záwárłá náde mną ſtudnia wierzchu ſwego.
Wyſłuchajże mię PAnie / boć dobre jeſt miłośierdźie twoje : według wielkey litośći twojey wejrzy ná mię.
Nie zákrywajże oblicża twego od ſługi ſwego / bom jeſt w utrapieniu : poſpieƺże śię / wyſłuchaj mię.
Przybliƺśię do duƺe mojey / á wybaw ją : dla nieprzyjaćiół mojich odkup mię.
Ty znaƺ poháńbienie moje / y zelżywość moję / y wſtyd mój : przed tobąć <i>ſą</i> wƺyſcy nieprzyjaćiele moji.
Poháńbienie pokruƺyło ſerce moje / z cżegom był żáłoſny : Ocżekawałem żaliby śię mnie kto użalił / ále nikt nie był : żaliby mię kto poćieƺył / álem nie ználazł.
Owƺem miáſto pokármu podáli mi żółć : á w prágnieniu mojim nápojili mię octem.
Niechajże im będźie ſtół ich przed nimi śidłem / á ƺcżeśćie ich ná upadek.
Niech śię záćmią ocży ich / áby nie widźieli / á biodrá ich niech śię záwżdy chwieją.
Wylij ná nie rozgniewánie ſwoje : á popędliwość gniewu twego / niech je ogárnie.
Niech będźie mieƺkánie ich puſte : w namiotách ich niech nikt nie mieƺka.
Bo <i>tego</i> któregoś ty ubił / prześládują : á o boleśći poránionych twojich / rozmawiáją.
Przydajże niepráwość ku niepráwośći ich : á niech nieprzychodzą do ſpráwiedliwośći twojey.
Niech będą wymázáni z kśiąg żywiących : á z ſpráwiedliwymi niech nie będą zápiſáni.
Jámći utrapiony / y z boláły : <i>lecż</i> zbáwienie twoje Boże ná miejſcu beſpiecżnym poſtáwi mię.
Tedy będę chwalił Imię Boże pieśńią : á będę je wielbił z dźiękcżynieniem.
A będźie <i>to</i> przyjemniejƺa PAnu / niżeli wół álbo ćielec rogáty z rozdźielonymi kopytámi.
To widząc pokorni rozrádują śię / ƺukájąc Bogá : á ożyje ſerce ich.
Iż wyſłuchawa PAn ubogich / á więźniámi ſwymi nie gárdźi.
Niech go chwalą niebioſá y źiemiá : morze / y wƺyſtko co śię w nich rucha.
Bógći zájiſte záchowa Syon / y pobuduje miáſtá Judſkie : y będą tám mieƺkáć / á <i>źiemię</i> tę dźiedźicżnie otrzymáją.
Tákże y naśienie ſług jego dźiedźicżnie ją otrzyma / á którzy miłują Imię jego / będą w niey mieƺkáć.
