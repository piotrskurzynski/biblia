<i>Pſálm</i> Dawidów. Rozpieraj śię PAnie z tymi / którzy śię ze mną ſpieráją : á walcż przećiwko tym / którzy walcżą przećiwko mnie.
Porwij puklerz y tarcżą : á powſtań ná rátunek mój.
Dobądź włócżnie / á ſtaw śię ná drodze przećiwko tym / którzy mię prześládują. Rzecżże duƺy mojey : Jam <i>jeſt</i> zbáwieniem twojim.
Niech będą poháńbieni y záwſtydzeni / którzy ƺukáją duƺe mojey : niech tył podádzą / y niech będą záwſtydzeni / którzy mi źle myślą.
Niech będą jáko plewy przed wiátrem : á Anjoł PAńſki niechaj je rozproƺy.
Niech będźie drogá ich ćiemna y śliſka : á Anjoł PAńſki niech je goni.
Abowiem bez przycżyny záſtáwili ná mię w dole śieći ſwoje : y bez przycżyny ukopáli <i>dół</i> duƺy mojey.
Niechaj ná nie przydźie ſpuſtoƺenie / którego śię nie ſpodźiewáli : á śieć jch którą záſtáwili / niech je ułowi ná zginienie / á niechaj w nię wpádną.
Ale duƺá mojá / niech śię rozráduje w PAnu : niech śię rozweſeli w zbáwieniu jego.
<i>Tedy</i> wƺyſtkie kośći moje rzeką ; PAnie / któż podobny tobie / który wyrywaƺ utrapionego od mocniejƺego nádeń : á nędznego y ubogiego / od drapieżce jego.
Powſtawáją świádkowie fałƺywi : á o cżym nie wiem / pytáją mię.
Oddawáją mi złym zá dobre : <i>chcąc</i> mię pozbáwić duƺe mojey.
Choćiam śię ja w wór obłócżył / gdy oni chorowáli : trapiłem poſtem duƺę moję / y modliłem śię cżęſto ſam u śiebie zá nimi.
Jáko <i>do</i> przyjaćielá / jáko <i>do</i> brátá mego / uſtáwicżniem chadzał : poniżałem śię jáko ten / który śię ſmęći chodząc po mátce w żałobie.
Lecż oni / gdym ja chorował / weſelili śię / y zbieráli śię : Zbieráli śię przećiwko mnie / <i>jákoby byli dla mnie</i> utrapieni / cżegom ja niepoſtrzegł : ƺcżypáli mię / á nie milcżeli.
Z obłudnikámi / z naśmiewcámi / z pochlebcámi / zgrzytáli ná mię zębámi ſwymi.
PAnie / długoż <i>na to</i> pátrzáć będźieƺ? wyrwiże duƺę moję od zguby ich : od lwiąt jedynacżkę moję.
Będę ćię wyſławiał w zgromádzeniu wielkim : miedzy ludem wielkim będę ćię chwalił.
Niech śię nie weſelą ze mnie / którzy mi ſą nieprzyjaćiołmi bez przycżyny : którzy mię máją w nienawiśći nieſłuƺnie / niech nie mrugáją okiem.
Abowiem nie mówią o pokoju / ále przećiwko ſpokojnym ná źiemi zdrádliwe ſłowá zmyśláją.
Owƺem rozdźieráją ná mię gębę ſwą / mówiąc : Ehej / Ehej widźić <i>to</i> oko náƺe.
Widźiƺ <i>to</i> PANie / nie milcżże : PANIE / nie oddalaj śię ode mnie.
Obudźże śię / á ocuć dla ſądu mego : Boże mój y PANIE mój / dla ſpráwy mojey.
Sądź mię według ſpráwiedliwośći twojey / PANIE Boże mój : á niech śię nie weſelą náde mną.
Niech nie mówią w ſercu ſwojim ; Ehej duƺo náƺá : Niech nie mówią ; Pożárliſmy go.
Niechajże będą poháńbieni / y záwſtydzeni wƺyſcy weſelący śię ze złego mego : niech będą oblecżeni w háńbę / y w ſromotę / którzy śię chlubią przećiwko mnie.
Ale ći / którzy śię kocháją w ſpráwiedliwośći mojey / niech śpiewáją / y rádują śię / á niech mówią uſtáwicżnie : Niech będźie uwielbiony PAn / który życży pokoju ſłudze ſwemu.
A język mój będźie opowiádał ſpráwiedliwość two ję / <i>y</i> ná káżdy dźień chwałę twoję.
