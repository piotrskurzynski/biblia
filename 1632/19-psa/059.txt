Przedniejƺemu śpiewakowi / <i>jáko</i> ; Niezátracaj : Złota Pieśń Dawidowá : gdy poſłał Saul / áby ſtrzeżono domu <i>jego,</i> á zábito go.
Wyrwij mię od nieprzyjaćiół mojich o Boże mój : á od powſtawájących przećiwko mnie / ucżyń mię beſpiecżnym.
Wyrwij mię od tych / którzy broją niepráwość : á od mężów krwáwych wybaw mię.
Abowiem / oto cżyháją ná duƺę moję : zbieráją śię przećiwko mnie mocárze / bez przeſtępſtwá mego / y bez grzechu mego / o PAnie.
Bez wƺelkiey niepráwośći mojey zbiegáją śię / y gotują śię : powſtańże / zábież mi / á obacż.
Ty ſam PAnie / Boże Zaſtępów / Boże Izráelſki / ocuć śię / ábyś náwiedźił te wƺyſtkie narody : á nie mij litośći nád żadnym z onych przeſtępców złośliwych : Selá.
Náwracáją śię pod wiecżór / á wárcżą jáko pśi / y biegáją około miáſtá.
Oto blegocą uſty ſwymi / miecże ſą w wárgách ich : ábowiem <i>mówią</i> ; Któż ſłyƺy?
Ale ty PAnie náśmiewaƺ śię z nich : náśmiewaƺ śię ze wƺyſtkich narodów.
Gdy on moc przewodźi / ná ćiebie pozór mieć będę : boś <i>ty</i> Boże twirdzą moją.
Bóg mój miłośierny uprzedźi mię : Bóg mi da oglądáć <i>pomstę</i> nád nieprzyjaćioły mymi.
Nie zábijajże ich / áby nie zápomniał lud mój : <i>ále</i> je rozproƺ mocą twoją y zrzuć je / tarcżo náƺá / o PAnie.
Grzech uſt ſwych / ſłowá warg ſwych ( pojimáni będąc w hárdośći ſwey / dla złorzecżeńſtwá y kłamſtwá ) niech wyznawáją.
Wytráćże <i>je</i> w popędliwośći : wytráć je / áż ich nie ſtánie : Niech poznáją że Bóg pánuje w Jákóbie / y po krájách źiemie : Selá.
Y niech śię záś náwrócą pod wiecżór : niech wárcżą jáko pśi / á biegáją około miáſtá.
Niech oni ćiekáją / chcąc śię nájeść : wƺákże głodni będąc / ukłáść śię muƺą.
Ale ja będę śpiewał o mocy twojey : záraz z poránku wyſławiáć będę miłośierdźie twoje : boś ty był twirdzą moją / y ućiecżką w dźień ućiſku mego.
O mocy mojá! tobie będę śpiewał : Boś ty Boże twirdza mojá / Bóg mój miłośierny.
