Przedniejƺemu śpiewakowi dla Jedytuná / Pſálm Aſáfów.
Głos mój <i>podnoƺę</i> do Bogá kiedy wołam : głos mój <i>podnoƺę</i> do Bogá / áby mię wyſłuchał.
W dźień utrapienia mego ƺukałem PAná : wyćiągałem w nocy ręce moje bez przeſtánia / á nie dáłá śię ućieƺyć duƺá mojá.
Wſpominałem ná Bogá / á trwożyłem ſobą : rozmyślałem / á utrapieniem śći śniony był duch mój : Selá.
Zátrzymawałeś ocży moje / áby cżuły : potártym był / áżem nie mógł mówić.
Przychodźiły mi ná pámięć dni przeƺłe <i>y</i> látá dawne.
Wſpominałem ſobie ná śpiewánie moje / w nocym w ſercu ſwym rozmyślał / y wywiádował śię <i>o tym</i> duch mój / <i>mówiąc</i> :
Izali mię ná wieki odrzući PAn / á więcey <i>mi</i> już łáſki nie ukaże?
Izali do końcá uſtánie miłośierdźie jego? <i>y</i> koniec weźmie ſłowo od rodzáju áż do rodzáju? Izali zápomniał Bóg zmiłowáć śię?
Izali zátrzymał w gniewie litośći ſwoje? Selá.
Y rzekłem : Toć jeſt śmierć mojá : <i>Wƺakże</i> práwicá Nawyżƺego ucżyni odmiánę.
Wſpomináć ſobie będę ná ſpráwy PAńſkie : á przypomináć ſobie będę dźiwne ſprá wy twoje zdawná <i>ucżynione</i>.
Y będę rozmyślał o wƺelkim dźiele twojim : y o ucżynkách twojich będę mówił :
Boże / święta jeſt drogá twojá : Któryż Bóg jeſt ták wielki / jáko Bóg <i>náƺ</i>?
Tyś <i>jeſt</i> Bóg / który cżyniƺ cudá : podałeś do znájomośći miedzy narody moc twoję.
Odkupiłeś rámieniem <i>twojim</i> lud ſwój / Syny Jákóbowe y Józefowe : Selá.
Widźiáły ćię wody o Boże! widźiáły ćię wody / <i>y</i> ulękły śię : y wzruƺyły śię przepáśći.
Obłoki wydáły powodźi : niebioſá wydáły gromy : á ſtrzały twoje tám y ſám biegáły.
Hucżáło grzmienie twoje po obłokách / błyſkáwice oświećiły okrąg źiemie : źiemiá śię wzruƺyłá y zátrzęſłá.
Przez morze <i>byłá</i> drogá twojá / á śćieƺki twoje przez wody wielkie : wƺákże śládów twojich znáć nie było.
Prowádźiłeś lud twój jáko ſtádo owiec / przez rękę Mojżeƺá y Aároná.
