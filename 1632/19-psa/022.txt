Przedniejƺemu śpiewakowi ná cżás poranny Pſálm Dawidów.
Boże mój Boże mój cżemuś mię opuśćił? oddaliłeś śię od wybáwienia mego : od ſłów ryku mojego!
Boże mój wołam we dnie / á nie ozywaƺ mi śię! y w nocy / á nie mogę śię uſpokojić!
Aleś ty Święty / mieƺkájący w chwałách Izráelſkich.
W tobie nádźieję mieli Ojcowie náƺy : nádźieję mieli / á wybáwiłeś je.
Do ćiebie wołáli / á wybáwieni ſą : w tobie nádźieję mieli / á nie byli poháńbieni.
Alem ja robak / á nie cżłowiek : pośmiewiſko ludzkie / y wzgárdá poſpólſtwá.
Wƺyſcy którzy mię widzą ƺydzą ze mnie : wykrzywiáją gębę : chwieją głową / <i>mówiąc</i> :
Spuśćił śię ná PAná / niechże go wyrwie : Niech go wybáwi / ponieważ śię w nim kocha.
Aleś ty jeſt / któryś mię wywiódł z żywotá / cżyniąc mi dobrą nádźieję jeƺcże u pierśi mátki mojey.
Ná tobie ſpolegam od národzenia ſwego : z żywotá mátki mojey tyś Bogiem mojim.
Nie oddalajże śię odemnie : ábowiem utrapienie bliſkie jeſt : á niemáƺ ktoby rátował.
Obtocżyło mię mnóſtwo ćielców : bycy z Báſán oblegli mię.
Otworzyli ná mię gębę ſwą / jáko lew ƺárpájący y rycżący.
Rozpłynąłem śię jáko wodá / á rozſtąpiły śię wƺyſtkie kośći moje : ſtáło śię ſerce moje jáko woſk / zſtopniáło w pośrzód wnętrznośći mojich.
Wyſchłá jáko ſkorupá moc mojá : á język mój przyſechł do podniebienia mego : náwet w prochu śmierći położyłeś mię.
Abowiem pśi mię obſkocżyli : gromádá złośników obległá mię : przebodli ręce moje / y nogi moje.
Zlicżyłbym wƺyſtkie kośći moje : lecż oni ná mię pátrząc / przypátrują mi śię.
Rozdźielili odźienie moje miedzy śię / á o ƺáty moje los miotáli.
Ale ty PAnie / nie oddalaj śię : mocy mojá ná rátunek mój poſpieƺ.
Wyrwi od miecżá duƺę moję : z mocy pśiey jedynacżkę moję.
Wybaw mię z páƺcżeki lwey á od rogów jednorożcowych wyzwól mię.
<i>Tedy</i> opowiem Imię twoje bráćij mey : w pośrzód zgromádzenia chwalić ćię będę :
<i>Mówiąc</i> ; Którzy śię bojićie PAná / chwalćie go : wƺyſtko potomſtwo Jákóbowe wyſławiajćie go : á niech śię go boji wƺyſtko naśienie Izráelſkie.
Abowiem nie wzgárdźił / áni śię odwróćił od utrapienia ubogiego ; áni ſkrył od niego oblicża ſwego : owƺem gdy do niego wołał / wyſłuchał go.
O tobie chwałá mojá w zgromádzeniu wielkim : śluby moje oddam przed tymi / którzy śię ćiebie boją.
Będą jeść ubodzy / y náſycą śię : chwalić będą PAná / którzy go ƺukáją : ſerce wáƺe żyć będźie ná wieki.
Wſpomioną y náwrócą śię do PAná / wƺyſtkie gránice źiemie : y kłaniáć śię będą przed oblicżem twojim wƺyſtkie pokolenia narodów.
Abowiem PAńſkie <i>jeſt</i> króleſtwo / á on pánuje nád narody.
Wƺyſcy bogáći źiemie będą jeść / y upádáć <i>przed nim</i> : przed oblicżnośćią jego kłaniáć śię będą wƺyſcy zſtępujący w proch / y <i>którzy</i> duƺe ſwey żywo záchowáć niemogą.
Naśienie ich ſłużyć mu będźie / á będźie przywłaƺcżáne PAnu / w káżdym wieku.
Zbieżą śię / á będą opowiádáli ſpráwiedliwość jego ( narodowi który z nich wynidźie ) yż <i>ją on</i> wykonał.
