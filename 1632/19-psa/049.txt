Przedniejƺemu śpiewakowi z Synów Korego Pſálm.
Słuchajćie tego wƺyſcy narodowie : bierzćie <i>to</i> w uƺy wƺyſcy mieƺkájący ná okręgu źiemie.
Ták z ludu poſpolitego / jáko z ludźi zacnych : ták bogáty jáko ubogi.
Uſtá moje będą opowiádáły mądrość : á myśl ſercá mego roſtropność.
Nákłonię do przypowieśći uchá mego : á wyłożę przy hárfie gadkę moję.
Przecżże śię mam bać we złe dni / <i>aby</i> mię niepráwość tych / którzy mię depcą / miáłá ogárnąć.
Którzy ufáją bogáctwom ſwojim / á w mnóſtwie doſtátków ſwojich chlubią śię.
Gdyż brátá ſwego nikt żadnym ſpoſobem nie odkupi / áni może dáć Bogu okupu jego <i>zań</i>
( Abowiem drogi jeſt okup duƺe jch : y nie może śię oſtáć ná wieki )
Aby żył ná wieki / á nie oglądał grobu.
Bo widźiemy / iż <i>y</i> mądrzy umieráją : głupi y ƺalony zárówno giną : á zoſtawiáją obcym bogáctwá ſwoje.
Myślą <i>że</i> domy ich <i>ſą</i> wiecżne / á przybytki ich <i>trwáją</i> od narodu do narodu : <i>przetoż</i> je názywáją od imion ſwych ná źiemi.
Ale cżłowiek we ćći nie zoſtawa : podobnym będąc bydlętom / które giną.
Tákowa myśl ich / głupſtwem ich jeſt : á przećię potomkowie ich pochwaláją to uſty ſwymi : Selá.
Jáko owce w grobie złożeni będą ; śmierć je ſtrawi : ále ſpráwiedliwi pánowáć będą nád nimi z poránku : á kƺtałt ich zniƺcżony będźie w grobie / gdy uſtąpią z mieƺkánia ſwego.
Ale Bóg wykupi duƺę moję z mocy grobu / gdy mię przyjmie : Selá.
Niebójże śię gdy śię kto zbogáći / á gdy śię rozmnoży ſławá domu jego.
Bo umierájąc / nie weźmie nic <i>z ſobą</i> : áni zá nim zſtąpi ſławá jego.
A choć duƺy ſwey / zá żywotá ſwego pobłaża / y chwalono go / gdy ſobie dobrze cżynił :
Przećię muśi iść zá rodźiną Ojców ſwych : á ná wieki nie ogląda świátłośći.
Owa cżłowiek / który jeſt we ćći / á nie zrozumiewa <i>tego,</i> podobny jeſt bydlętom które giną.
