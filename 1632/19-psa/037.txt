<i>Pieśń</i> Dawidowá. Nie obruƺaj śię dla złośników / áni zájrzy cżyniącym niepráwość.
Bo jáko trawá prętko podćięći będą / á jáko liśćie źielone opádną.
Dufaj w PAnu / á cżyń dobrze : mieƺkajże ná źiemi / á żyw śię ſpráwiedliwie.
Kochaj śię w PAnu / á dać proźby ſercá twego.
Spuść ná PAná drogę twoję / á ufaj w nim / á on wƺyſtko ucżyni.
Y wywiedźie jáko świátłość ſpráwiedliwość twoję ; á ſąd twój jáko południe.
Poddaj śię PAnu / á ocżekawaj go : nie obruƺaj śię ná tego / któremu śię ƺcżeśći w ſpráwách jego / ná cżłowieká który dokázuje cokolwiek zámyśli.
Przeſtań gniewu / á zániechaj popędliwośći ; nie zápalaj śię gniewem / ábyś miał źle cżynić.
Abowiem złośnicy będą wykorzenieni : lecż którzy ocżekawáją PAná / ći odźiedźicżą źiemię.
Po máłey chwili álić nie máƺ niezbożniká : pojrzyƺli ná miejſce jego / álić go już niemáƺ.
Lecż pokorni odźiedźicżą źiemię / y rozkocháją śię w wielkośći pokoju.
Źle myśli niepobożny przećiwko ſpráwiedliwemu / y zgrzyta nań zębámi ſwymi.
Ale śię PAn śmieje z niego : bo widźi że przychodźi dźień jego.
Miecżá dobyli niezboźni / á náćiągnęli łuk ſwój / áby poráźili ubogiego / y niedoſtátecżnego : á / żeby pomordowáli tych / którzy chodzą proſtą drogą.
Aleć miecż ich przeniknie ſerce jch : á łuki ich będą połamáne.
Lepƺa jeſt trochá ſpráwiedliwego / niż wielkie bogáctwá wielu niepobożnych.
Abowiem rámioná niezbożników będą pokruƺone : ále ſpráwiedliwe PAn podpiera.
Znać PAn dni doſkonáłych : przetoż dźiedźictwo ich ná wieki zoſtánie.
Nie będą záwſtydzeni we zły cżás / á we dni głodu będą náſyceni.
Ale niezbożni poginą : á nieprzyjaćiele PAńſcy / jáko tłuſtość báránia z dymem niƺcżeje / ták oni zniƺcżeją.
Niezbożnik pożycżá / á nie ma cżym oddáć : ále ſpráwiedliwy pokázuje łáſkę y rozdawa.
Abowiem błogoſłáwieni od PAná odźiedźicżą źiemię : ále przeklęći od niego / będą wykorzenieni.
Od PAná bywáją ſpráwowáne drogi cżłowieká <i>dobrego,</i> á drogá jego podoba mu śię.
Gdy pádnie / nie ſtłucże śię : ábowiem PAn trzyma <i>go zá</i> rękę jego.
Byłem młodym / y ſtárzałem śię : á nie widźiałem ſpráwiedliwego opuƺcżonego / áni naśienia jego żebrzącego chlebá.
Ná káżdy dźień pokázuje miłośierdźie y pożycża : á <i>przećię</i> naśienie jego <i>jeſt</i> w błogoſłáwieńſtwie.
Odſtąp od złego / á cżyń dobrze : á będźieƺ mieƺkał ná wieki.
Abowiem PAn miłuje ſąd : á nie opuśći świętych ſwojich / ná wieki w ſtraży <i>jego</i> będą : ále naśienie niepobożnych będźie wykorzenione.
Spráwiedliwi odźiedźicą źiemię : y będą w niey mieƺkáli ná wieki.
Uſtá ſpráwiedliwego mówią mądrość / á język jego ſąd opowiáda.
Zakon Bogá je° <i>jeſt</i> w ſercu jego : przetoż nie záchwieją śię nogi jego.
Wypátruje niepobożny ſpráwiedliwego : y ƺuka jákoby go zábił.
<i>Ale</i> PAn niezoſtáwi go w ręku jego : y nie potępi go gdy będźie ſądzony.
Ocżekawaj PAná / y ſtrzeż drogi jego / á on ćię wywyżƺy / ábyś odźiedźicżył źiemię : á oglądaƺ / gdy niepobożni wytráceni będą.
Widźiałem niezbożniká náder wynioſłego / á rozłożone° jáko <i>drzewo</i> źielone ſámoroſłe :
Ale przeminął á oto go nie było : ƺukałem go / álem go ználeść niemógł.
Pojrzy ná niewinnego / á przypátrz śię ƺcżeremu / że oſtátnie rzecży <i>tákiego</i> cżłowieká <i>ſą</i> ſpokojne
Lecż przeſtępcy poſpołu poginą / á niezbożnicy ná oſtátek wykorzenieni będą.
Wƺákże zbáwienie ſpráwiedliwych <i>jeſt</i> od PAná / <i>który jest</i> mocą ich cżáſu ućiśnienia.
Wſpomaga ich PAn / y wyrywa je : wyrywa je od niepobożnych / y záchowywa je : bo w nim nádźieję máją.
