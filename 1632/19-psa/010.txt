PANIE przecżże ſtojiƺ z dáleká? <i>przecżże</i> śię ukrywaƺ cżáſu ućiſku?
Złośnik z hárdośći prześláduje ubogiego : niechajże będą uchwyceni w chytrych zamyſłách / które zámyśláją.
Bo śię chlubi niezbożnik w pożądliwośćiách duƺe ſwojey : á łákomy błogoſłáwi <i>ſobie, á</i> draźni PAná.
Niepobożny dla pychy którą po ſobie pokázuje nie pyta śię <i>o Bogu</i>. Wƺyſtká myśl jego / że niemáƺ Bogá.
Dárzą mu śię drogi jego ná káżdy cżás : dálekie ſą ſądy twoje od niego : Sapa przećiwko wƺyſtkim nieprzyjaćiołom ſwym.
Mówi w ſercu ſwym ; Nie będę wzruƺon od narodu do narodu / bo śię <i>nie boję</i> złego.
Uſtá jego pełne ſą złorzecżeńſtwá / y chytrośći / y zdrády : pod językiem jego uprzykrzenie y niepráwość.
Siedźi cżyhájąc we wśiách : w ſkrytośćiách zábija niewinnego : ocży jego upátrują ubogiego.
Cżyha w ſkrytym miejſcu / jáko lew w jámie ſwojey : dybie / jákoby uchwyćił ubogiego : ułápiwƺy go / ćiągnie do śieći ſwojey.
Przypada / przytula śię / y rzuca śię mocą ſwoją ná wiele ubogich.
Mówi w ſercu ſwym ; Zápomniałći <i>tego</i> Bóg : zákrył oblicże ſwoje / nie ujrży ná wieki.
Powſtańże PAnie Boże / podnieś rękę twoję : nie zápominajże ubogich.
Przecżże niezbożnik draźni Bogá / mówiąc w ſercu ſwym ; Nie będźieƺ śię <i>o tym</i> pytał.
<i>Ale</i> ty widźiƺ ućiſk / y krzywdę upátrujeƺ / ábyś <i>im</i> odpłáćił ręką twą : ná ćiebieć śię ſpuśćił ubogi : Tyś jeſt pomocnikiem śirotce.
Potrzyj rámię niepobożnego / y złośniká : dowiáduj śię o jego niezbożnośći / <i>aż</i> go nie ſtánie.
PAN <i>jeſt</i> Królem ná wieki wiecżne : ále narodowie zginą z źiemie jego.
Żądośći pokornych wyſłuchawaƺ PANIE : utwirdzaƺ ſercá ich : náchylaƺ ku <i>nim</i> uchá twojego.
Abyś ſąd ucżynił śiroćie y chudźinie : áby go więcey nie trapił cżłowiek śmiertelny ná źiemi.
