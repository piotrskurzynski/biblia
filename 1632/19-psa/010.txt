Pánie! przecżże ſtoiƺ z dáleká? przecżże śię ukrywáƺ cżáſu ućiſku?
Złośnik z hárdośći prześláduje ubogiego ; niechájże będą uchwyceni w chytrych zámyſłách / które zámyśláją.
Bo śię chlubi niezbożnik w pożądliwośćiách duƺy ſwojey / á łákomy błogoſłáwi ſobie á dráźni Páná.
Niepobożny dla pychy / którą po ſobie pokázuje / nie pytá śię o Bogá ; wƺyſtká myśl jego / że niemáƺ Bogá.
Dárzą mu śię drogi jego ná káżdy cżás ; dálekie ſą ſądy twoje od niego ; ſápá przećiwko wƺyſtkim nieprzyjáćiołom ſwym.
Mówi w ſercu ſwem : Nie będę wzruƺony od narodu do narodu ; bo śię nie boję złego.
Uſtá jego pełne ſą złorzecżeńſtwá / y chytrośći / y zdrády ; pod językiem jego uprzykrzenie y niepráwość.
Siedźi / cżyhájąc we wśiách / w ſkrytośćiách zábijá niewinnego ; ocży jego upátrują ubogiego.
Cżyhá w ſkrytem miejſcu / jáko lew w jámie ſwojey ; dybie jákoby uchwyćił ubogiego / ułápiwƺy go ćiągnie do śieći ſwojey.
Przypádá / przytulá śię / y rzucá śię mocą ſwoją ná wiele ubogich.
Mówi w ſercu ſwem : Zápomniałći tego Bóg ; zákrył oblicże ſwoje / nie ujrzy ná wieki.
Powſtáńże / Pánie Boże! podnieś rękę twoję ; nie zápominájże ubogich.
Przecżże niezbożnik dráźni Bogá / mówiąc w ſercu ſwem : Nie będźieƺ śię o tym pytał?
Ale ty widźiƺ ućiſk / y krzywdę upátrujeƺ / ábyś im odpłáćił ręką twą ; ná ćiebieć śię ſpuśćił ubogi / tyś jeſt pomocnikiem śieroćie.
Potrzyj rámię niepobożnego y złośniká / dowiáduj śię o jego niezbożnośći / áż go nie ſtánie.
Pán jeſt królem ná wieki wiecżne ; ále narody zginą z źiemi jego.
Żądośći pokornych wyſłuchiwáƺ / Pánie! utwierdzáƺ ſercá ich / náchyláƺ ku nim uchá twojego.
Abyś ſąd ucżynił śieroćie y chudźinie / áby go więcey nie trápił cżłowiek śmiertelny ná źiemi.
