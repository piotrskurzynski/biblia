Pieśń y Pſálm Aſáfów.
O Boże / nie milcżże / nie bądź <i>jáko</i> ten co nie ſłyƺy : y nie chćiey śię uſpokojć / o Boże!
Bo śię oto nieprzyjaćiele twoji burzą : á ći którzy ćię w nienawiśći máją / podnoƺą głowę.
Przećiwko ludowi twemu wymyślili chytrą rádę : á ſpiknęli śię przećiw tym / których ty ochraniaƺ.
Mówiąc ; Pódźćie / á wytráćmy je / niech nie będą narodem : ták / żeby y nie wſpominano więcey Imienia Izráelſkiego.
Abowiem ſpiknęli śię jednomyślnie / y przymierze przećiwko tobie ucżynili :
Namioty Edomcżyków / y Iſmáelcżyków / Moábcżyków / y Agáreńcżyków :
Gebálcżyków / y Ammonitcżyków / y Amálekitcżyków : <i>także</i> Filiſtyńcżyków z tymi którzy mieƺkáją w Tyrze :
Więc y Aſſyryjcżycy / złącżyli śię z nimi / będąc rámieniem Synom Lotowym : Selá.
Ucżyńże im <i>ták</i> jáko Mádiáńcżykom / jáko Siſárze / jáko Jábinowi u potoku Cyſon.
<i>Którzy</i> ſą wygłádzeni w Endor : ſtáli śię jáko gnój ná źiemi.
Obchodźże śię z nimi / y z ich hetmány / jáko z Orebem / y jáko z Zebą / y jáko z Zebeem / y jáko z Sálmánem / ze wƺyſtkimi Kśiążęty ich.
Bo rzekli : Pośiądźmy dźiedźicżnie przybytki Boże.
Boże mój / ucżyńże je jáko koło : y jáko źdźbło przed wiátrem.
Jáko ogień / który lás pali : á jáko płomień <i>który</i> zápala góry.
Ták ich ty wichrem twojim śćigaj / á burzą twą zátrwóż je.
Nápełni twarzy ich poháńbieniem / áby ƺukáli Imienia twego / PAnie.
Niech będą záwſtydzeni / y uſtráƺeni áż ná wieki : á będąc poháńbieni / niech záginą.
A ták / niech poznáją żeś ty / którego Imię jeſt PAn : Tyś ſam Nawyżƺym nád wƺyſtką źiemią.
