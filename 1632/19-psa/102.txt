Modlitwá utrapionego / gdy będąc w ućiſku przed PAnem wylewa żądość ſwoję.
PAnie wyſłuchaj modlitwę moję : á wołánie moje niechaj przydźie do ćiebie.
Nie ukrywaj oblicża twego przedemną : w dźień ućiſku mego nákłoń ku mnie uchá twego : w dźień którego ćię wzywam / prętko mię wyſłuchaj.
Abowiem niƺcżeją jáko dym dni moje : á kośći moje jáko ogniſko wypalone ſą.
Poráżone jeſt jáko trawá / y uwiędło ſerce moje : <i>ták,</i> żem zápomniał jeść chlebá ſwego.
Od głoſu wzdychánia mego / przylgnęły kośći moje do ćiáłá mego.
Stałem śię podobnym Pelikanowi ná puƺcży : jeſtem jáko Puhacż ná puſtyniách.
Cżuję : á jeſtem jáko wróbl ſámotny ná dáchu.
Przez cáły dźień urągáją mi nieprzyjaćiele moji : á naśmiewcy moji przeklináją mię.
Bo jadam popiół jáko chleb : á napój mój mięƺam ze łzámi :
Dla rozgniewánia twego / y dla zápálcżywośći gniewu twego : ábowiem podnióżƺy mię / porzućiłeś mię.
Dni moje / ſą jáko ćień náchylony : á jam jáko trawá uwiądł.
Ale ty PAnie trwaƺ ná wieki : á pámiątká twojá / od narodu do narodu.
Ty powſtawƺy zmiłujeƺ śię nád Syonem : boć cżás żebyś śię zlitował nád nim / gdyż przyƺedł cżás náznácżony.
Abowiem upodobáło śię ſługom twojim kámienie jego : y nád prochē jego zmiłują śię.
Aby śię bali Pogánie Imienia Páńſkiego : á wƺyſcy królowie źiemſcy chwały twojey :
Gdy pobuduje PAn Syon / y okaże śię w chwale ſwojey.
<i>Gdy</i> wejrzy ná modlitwę poniżonych / nie gárdząc modlitwą ich.
To zápiƺą dla narodu potomnego : á lud / który ma być ſtworzony / chwalić będźie PAná :
Że wejrzał z wyſokośći Świątnice ſwojey : Że z niebá ná źiemię pojrzał :
Aby wyſłuchał wzdychánia więźniów / y rozwiązał ná śmierć ſkazáne.
Aby opowiádáli ná Syonie Imię PAńſkie / á chwałę jego w Jeruzalemie :
Gdy śię poſpołu zgromádzą narodowie / y króleſtwá / áby ſłużyły PAnu.
Utrapił w drodze śiłę moję ; ukróćił dni mojich :
Ażem rzekł ; Boże mój / nie bierz mię w połowicy dni mojich : od narodu bowiem áż do narodu <i>trwáją</i> látá twoje.
Y pierwy niżeliś záłożył źiemię / y niebioſá dźieło rąk twojich.
One pominą / ále ty zoſtawaƺ : wƺyſtkie te rzecży jáko ƺátá zwiotƺeją : jáko odźienie odmieniƺ je / y odmienione będą.
Ale ty tenżeś <i>záwżdy jeſt</i> : á látá twoje nigdy nie uſtáną.
Synowie ſług twojich <i>u ćiebie</i> mieƺkáć będą : á naśienie ich zmocni śię przed tobą.
