Przedniejƺemu śpiewakowi z Synów Korego / ná Soſánnim <i>Pſálm</i> náucżájący / á Pieśń weſelna.
Wydáło ſerce moje ſłowo dobre : roſpráwiáć będą pieśni moje o Królu : język mój będźie jáko pióro prętkiego piſárzá.
Piękniejƺyś nád Syny ludzkie : rozlałá śię wdźięcżność po wárgách twojich ; przeto / że ćię pobłogoſłáwił Bóg / áż ná wieki.
Przypáƺ miecż twój ná biodrá o mocárzu! <i>pokáż</i> chwałę twoję / y zacnośći twoje.
A w doſtojnośći twojey / ƺcżeśliwie wyjedź z ſłowem prawdy / ćichośći / y ſpráwiedliwośći : á dokaże ſtráƺnych rzecży práwicá twojá.
Strzały twoje oſtre : <i>od nich</i> narodowie pod ćię upádną : á ſerce nieprzyjaćiół królewſkich <i>przenikną</i>.
Stolicá twojá / o Boże! ná wieki wieków : laſká ſpráwiedliwośći jeſt laſká króleſtwá twego.
Umiłowáłeś ſpráwiedliwość / á nienawidźiałeś niepráwośći : przetoż pomázał ćię o Boże! Bóg twój olejkiem weſela / nád ucżeſniki twoje.
Myrrą / Aloe / <i>y</i> Káſſią wƺyſtkie ƺáty twoje <i>pachną, gdy wychodźiƺ</i> z páłaców z kośći ſłoniowych <i>urobionych,</i> nád te którzy ćię uweſeláją.
Córki królewſkie / ſą miedzy twymi zacnymi <i>białymi głowámi</i> : ſtánęłá małżonká po práwicy twojey / w koƺtownym złoćie z Ofir.
Słuchajże córko / á obacż / y nákłoń uchá twego : á zápomni narodu twego y domu Ojcá twojego.
A zákocha śię Król w pięknośći twojey : ábowiem on jeſt Pánem twojim : przetoż kłaniaj śię przed nim.
Tyryjcżycy tákże z upominkámi / przed oblicżem twojim kłaniáć śię będą / nabogátƺy z narodów.
Wƺyſtká zacność córki królewſkiey / jeſt wewnątrz : á ƺáty jey brámowáne ſą złotem.
W odźieniu háftowánym przywiodą ją do Królá : tákże pánny zá nią / towárzyƺki jey / przywiodą do ćiebie.
Przywiodą <i>je</i> z weſelem y z rádośćią : á wnidą ná páłac królewſki.
Miáſto Ojców twych będźieƺ mieć Syny twe : które poſtánowiƺ Kśiążęty po wƺyſtkiey źiemi.
Wſpomináć będę imię twoje / od káżdego rodzáju do rodzáju : dla tego ćię narodowie wyſławiáć będą ná wieki wieków.
