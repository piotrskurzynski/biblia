Przedniejƺemu śpiewakowi Jedytunowi Pſálm Dawidów.
Rzekłem : Będę ſtrzegł dróg mojich / ábym nie zgrzeƺył językiem ſwym : włożę munƺtuk w uſtá moje : póki niepobożny <i>będźie</i> przede mną.
Zániemiałem milcżąc : zámilknąłem <i>y</i> w dobrey <i>sprawie</i> : ále boleść mojá bárźiey śię wzmagáłá.
Roſpaliło śię ſerce moje we wnętrznośćiách mojich : w rozmyślániu mojim rozżarżył śię ogień / <i>áżem</i> ták rżekł językiem ſwojim :
Dáj mi poznáć PANie dokońcżenie moje / y wymiar dni mojich jáki jeſt : ábym wiedźiał / jáko długo trwáć będę.
Otoś ná dłoni wymierzył dni moje : á wiek mój jeſt jáko nic przed tobą : Záprawdę ƺcżerą márnośćią jeſt wƺelki cżłowiek / choć nadużƺy : Selá.
Záprawdę pomija cżłowiek jáko ćień : záprawdę prózno śię kłopoce : zgromadza / á nie wie kto to pobierze.
A teraz ná cóż ocżekawam PAnie? Tyś jeſt ſam ocżekawániem mojim.
Przetoż od wƺyſtkich przeſtępſtw mojich wybaw mię : ná pośmiech głupiemu nie dawaj mię.
Zániemiałem / y nie otworzyłem uſt mojich : przeto / żeś <i>to</i> ty ucżynił.
Odejmi ode mnie karánie twoje : bom od ſmágánia ręki twojey uſtał.
Gdy <i>ty</i> gromiąc karzeƺ cżłowieká dla niepráwośći / wnet niƺcżyƺ jáko mól grzecżność jego : zájiſteć márnośćią jeſt wƺelki cżłowiek : Selá.
Wyſłuchajże modlitwę moję PAnie / á wołánie moje przyjmi w uƺy <i>ſwoje</i> : nie milcż ná łzy moje : bomći ja przychodniem u ćiebie / <i>y</i> komornikiem / jáko wƺyſcy Ojcowie moji.
Sfolguj mi / ábym śię pośilił / pierwey niżli odejdę / á nie będźie mię.
