Przedniejƺemu śpiewakowi Pſálm Dawidów. O Boże chwały mojey / nie milcż :
Bo śię uſtá niepobożnego / y uſtá kłamliwe / przećiwko mnie otworzyły : mówili przećiwko mnie językiem kłamliwym :
A ſłowy jádowitymi ogárnęli mię : walcżąc przećiwko mnie bez wƺelkiey przycżyny.
Przećiwili mi śię zá miłość moję : choćiam śię <i>zá nie</i> modlił.
Oddawáją mi złym zá dobre : á nienawiśćią zá miłość moję.
Poſtawże nád nim bezbożniká : á przećiwnik niech ſtoji po práwey ręce jego.
Gdy przed ſądem ſtánie / niech wynidźie potępionym : á modlitwá jego niech śię w grzech obróći.
Niech będą dni jego krótkie : á przełożeńſtwo jego niech inny weźmie.
Niech dźieći jego będą śirotámi : á żoná jego wdową.
Niech będą biegunámi y tułakámi Synowie jego / niech żebrzą : á niech <i>żebrzą wychodżąc</i> z puſtek ſwojich.
Niech lichwiarz záłápi wƺyſtko co jeſt jego : á niech obcy rozchwycą pracę jego.
Niech nie będźie ktoby mu miłośierdźie pokazał : niech nie będźie ktoby śię zmiłował nád śirotámi jego.
Potomkowie jego / niech z korzenia wyćięći będą : w drugim pokoleniu / niech będźie wygłádzone imię ich.
Niech przydźie ná pámięć niepráwość przodków jego przed PAnem : á grzech mátki jego niechaj nie będźie zgłádzony.
Niech będą przed PAnem uſtáwicżnie : áżby wygłádźił z źiemie pámiątkę ich.
Przeto / że nie pámiętał áby cżynił miłośierdźie / ále prześládował cżłowieká nędznego y ubogiego : á tego który był ſercá utrapionego / chćiał zámordowáć.
Ponieważ umiłował przeklęctwo / niechże przydźie ná niego : niechćiał błogoſłáwieńſtwá / niechże będźie oddalone od niego.
A ták niech będźie oblecżony w przeklęctwo / jáko w ƺátę ſwoję : á niech wnidźie jáko wodá we wnętrznośći jego / á jáko oley w kośći jego.
Niech mu <i>to</i> będźie jáko płaƺcż dla przyodźiania : á jáko pás dla uſtáwicżnego opáſowánia.
Tákowa zapłátá / niech będźie przećiwnikom mojim od PAná : y tym którzy źle mówią przećiwko duƺy mojey.
Ale ty PAnie / o PAnie! użyj nádemną litośći / dla Imienia twego : á yż dobre jeſt miłośierdźie twoje / wyrwijże mię.
Bomći ja <i>jeſt</i> ubogi y nędzny : á ſerce moje zránione jeſt / we wnętrznośćiách mojich.
Jáko ćień / który uſtępuje / uchodźić muƺę : zganiáją mię jáko ƺáráńcżą.
Koláná moje upadáją od poſtu : á ćiáło moje wychudło z tłuſtośći.
Nád to / ſtałem śię im pośmiewiſkiem : <i>gdy</i> mię widzą kiwáją głowámi ſwymi.
Wſpomóżże mię o PAnie Boże mój : záchowaj mię według miłośierdźia ſwego.
Ták / áby poznáć mogli / iż to ręká twojá / á / żeś ty PAnie to ucżynił.
Niechże oni przeklináją / ty błogoſław : <i>którzy</i> powſtáli / niech będą záwſtydzeni / áby śię weſelił ſługá twój.
Niech będą przećiwnicy moji w háńbę oblecżeni : á niech śię przyodźieją / jáko płaƺcżem / zelżywośćią ſwoją.
Będę PANA wielce wyſławiał uſty ſwymi : á w pośrzód wiela chwalić go będę :
Przeto / że ſtoji po práwey ſtronie nędznemu / áby go wybáwił od tych / którzy oſądzáją duƺę jego.
