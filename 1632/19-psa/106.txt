Hállelujah. Wyſławiajćie PAná / ábowiem dobry : ábowiem ná wieki miłośierdźie jego.
Któż wyſłowi niezmierną moc PAńſką / á wypowie wƺyſtkę chwałę jego.
Błogoſłáwieni / którzy ſtrzegą ſądu : á cżynią ſpráwiedliwość ná káżdy cżás.
Pámiętaj ná mię PANIE / dla miłośći ku ludowi ſwemu. Náwiedźże mię zbáwieniem ſwojim :
Abych używał dobrego z wybránymi twojimi / á weſelił śię w rádośći narodu twego : y chlubił śię weſpół z dźiedźictwem twojim.
Zgrzeƺyliſmy z Ojcy ſwymi : nieſpráwiedliwieſmy cżynili / y niepráwość popełniáli.
Ojcowie náƺy w Egipćie nie zrozumieli cudów twojich : áni pámiętáli ná wielkość miłośierdźia twego ; ále odpornymi byli przy morzu / przy morzu cżerwonym.
A wƺákże je wyſwobodźił dla Imienia ſwe° / áby oznájmił moc ſwoję.
Bo zgromił morze cżerwone / y wyſchło : á przewiódł je przez przepáśći / jáko <i>przez</i> puƺcżą.
A ták záchował je od ręki tego / który je miał w nienawiśći : á wykupił je z ręki nieprzyjaćielſkiey.
W tym okryły wody tych / którzy je ćiążyli : nie zoſtał áni jeden z nich.
A choć uwierzyli ſłowom jego / y wyſławiáli chwałę jego :
<i>Przećię</i> prętko zápomnieli ná ſpráwy jego / <i>y</i> nie cżekáli ná rády jego.
Ale zjęći będąc chćiwośćią ná puƺcży / kuśili Bogá ná puſtyniách.
Y dał im cżego żądáli : á wƺákże przepuśćił ſuchoty ná nie.
Zátym gdy śię wzruƺyli zawiśćią przećiw Mojżeƺowi w oboźie / y <i>przećiw</i> Aáronowi świętemu PAńſkiemu :
Otworzyłá śię źiemiá / y pożárłá Dátáná / y okryłá rotę Abironowę.
Y zápalił śię ogień ná zebránie ich : płomień ſpalił niepobożne.
Spráwili y ćielcá ná Horebie / y kłaniáli śię báłwanowi litemu.
Y odmienili chwałę ſwą w podobieńſtwo wołu / jedzącego trawę.
Zápomnieli ná Bogá wybáwićielá ſwego / który cżynił wielkie rzecży w Egipćie :
Rzecży dźiwne w źiemi Chámowey : Rzecży ſtráƺne przy morzu cżerwonym.
Przetoż rzekł / że je chćiał wytráćić : gdyby śię był Mojzeƺ / wybrány jego / nie ſtáwił w onym rozerwániu przed nim / a nie odwróćił popędliwośći jego / áby <i>ich</i> nie tráćił.
Wzgárdźili też źiemią pożądáną / niewierząc ſłowu jego.
Y ƺemrząc w namiećiech ſwojich / nie byli poſłuƺni głoſowi PAńſkiemu.
Przetoż podnióſł rękę ſwoję przećiwko nim / áby je pobił ná puƺcży :
A żeby rozrzućił naśienie ich miedzy pogány / y rozproƺył je po źiemiách.
Zprzęgli śię też byli z báłwanem Báálfegorem : á jedli ofiáry umárłych.
A ták draźnili <i>Bogá</i> ſpráwámi ſwymi / że śię ná nie oborzyłá plagá.
Aż śię záſtáwił Finees / á pomſtę ucżynił y rozerwána jeſt oná plagá.
Co mu pocżytano ku ſpráwiedliwośći / od narodu do narodu / áż ná wieki.
Znowu go byli wzruƺyli do gniewu / u wód Merybá : <i>ták, iż śię</i> źle dźiało y z Mojżeƺem dla nich.
Abowiem rozdraźnili duchá jego / że wyrzekł <i>co niesłuƺna</i> uſty ſwymi.
Nád to / nie wytráćili onych narodów / o których im był PAn powiedźiał :
Ale pomięƺawƺy śię z onymi narodámi / náucżyli śię ſpraw ich.
Y ſłużyli báłwanom ich / które im były śidłem.
Abowiem Diabłom ofiárowáli Syny ſwoje / y córki ſwoje.
Y wylewáli krew niewinną / krew ſynów ſwojich / y córek ſwojich ; które ofiárowáli báłwanom rytym Chánánejſkim : ták / że ſplugáwiona byłá źiemiá onym krwie rozlaniem.
Y zmázáli śię ſpráwámi ſwymi : á cudzołożyli wynalazkámi ſwymi.
Przetoż zápaliwƺy śię PAN w popędliwośći / przećiw ludowi ſwemu / obrzydźił ſobie dźiedźictwo ſwoje :
Y podał je w ręce pogánom : á pánowáli nád nimi / którzy je mieli w nienawiśći.
Y ućiſkáli je nieprzyjaćiele ich / ták / że poniżeni byli pod ręką ich.
Cżęſtokroć je wybáwiał / wƺákże go oni wzruƺáli do gniewu rádámi ſwymi : zácżym poniżeni byli dla niepráwośći ſwojich.
A wƺákże wejrżał ná ućiſk ich / y uſłyƺał wołánie ich.
Bo ſobie wſpomniał ná przymierze ſwoje z nimi : á żáłował tego / według wielkiey litośći ſwojey.
Ták / że im zjednał miłośierdźie przed ocżymá wƺyſtkich / którzy je byli pojimáli.
Wybawże nas PAnie Boże náƺ á zgromadź nas z tych pogánów : ábyſmy wyſławiáli Imię świętobliwośći twojey / á chlubili śię w chwale twojey.
Błogoſłáwiony PAn Bóg Izráelſki / od wieków áż ná wieki : ná co niech rzecże wƺyſtek lud / Amen / Hállelujah.
