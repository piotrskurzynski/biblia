Pſálm á pieśń ná dźień Sobotni.
Dobra rzecż jeſt wyſławiáć PAná / á śpiewáć Imieniowi twemu / o Nawyżƺy.
Opowiádáć z poránku miłośierdźie twoje : y prawdę twoję ná káżdą noc.
Ná inſtrumenćie o dźieśiąći ſtronách / ná lutni / y ná hárfie / z śpiewániem.
Abowiemeś mię rozweſelił PAnie ſpráwámi twymi : o ſpráwách rąk twojich śpiewáć będę.
O jáko wielmożne ſą ſpráwy twoje PAnie : bárzo głębokie ſą myśli twoje.
Cżłowiek bydlęcy nie zna : á głupi nie zrozumiewa tego.
Iż wyraſtáją niezbożnicy jáko źiele : á kwitną wƺyſcy / którzy cżynią niepráwość / áby byli wykorzenieni áż ná wieki.
Ale ty o Nawyżƺy / jeſteś PAnem ná wieki.
Abowiem / oto nieprzyjaćiele twoji PAnie : Abowiem oto nieprzyjaćiele twoji zginą : rozproƺeni będą wƺyſcy / którzy cżynią niepráwość.
Ale róg mój wywyżƺyƺ jáko jednorożców : pokropiony będę olejkiem świeżym.
Y ujrzy oko moje <i>nieƺcżeśćie</i> tych / co ná mię cżyháją : o złośnikách / którzy powſtawáją przećiwko mnie / uſłyƺą uƺy moje.
Spráwiedliwy / jáko Pálmá zákwitnie : jáko Cedr ná Libánie rozmnoży śię.
Wƺcżepieni w domu PAńſkim / w śieniách Bogá náƺego zákwitną.
Náwet y w ƺedźiwośći przynioſą owoc / cżyrſtwymi y źielonymi będą.
Aby to opowiádano / że uprzyjmym jeſt PAN / ſkáłá mojá : á / że w nim niemáƺ żadney niepráwośći.
