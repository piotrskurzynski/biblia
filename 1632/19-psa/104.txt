Błogoſław duƺo mojá PANU. PAnie Boże mój wielceś jeſt wielmożnym : chwałę y ozdobę przyoblokłeś.
Przyodźiałeś śię świátłośćią jáko ƺátą : rozćiągnąłeś niebioſá jáko oponę.
Któryś záſklepił ná wodách páłace ſwoje : który używaƺ obłoków miáſto wozów : który chodźiƺ ná ſkrzydłách wiátrowych :
Który cżyniƺ Duchy poſłámi ſwymi : <i>Ty cżyniƺ</i> ſługi ſwe ogniem pałájącym.
Ugruntowałeś źiemię ná ſłupách jey : ták / że śię nie poruƺy ná wieki wiecżne.
Przepáśćią / jáko ƺátą / przyodźiałeś ją był : <i>á</i> wody ſtały nád górámi.
Ná zgromienie twoje rozbiegły śię : á ná głos pogromu twego prętko zućiekáły.
Wyſtąpiły góry / zniżyły śię doliny / ná miejſce / któreś im záłożył.
Zámierzyłeś <i>im</i> kres / áby go nie przeſtępowáły : áni śię wracáły ná okryćie źiemie.
Który wypuƺcżaƺ źrzódłá po dolinách / <i>áby</i> płynęły miedzy górámi :
A napój dawáły wƺyſtkiemu zwierzowi polnemu : <i>á z nich</i> gáƺą leśni oſłowie prágnienie ſwoje.
Przy nich mieƺka ptáſtwo niebieſkie / á z pośrzód gáłązek głos wydawa.
Który pokrapiaƺ góry z páłaców ſwojich / áby śię z owoców ſpraw twojich náſycáłá źiemiá.
Zá twoją ſpráwą rośćie trawá dla bydłá / á źiołá ná pożytek cżłowiecży : ty wywodźiƺ chleb z źiemie :
Y wino / które uweſela ſerce cżłowiecże / od którego śię łſni twarz <i>jáko</i> od oleju : y chleb / który zátrzymawa żywot ludzki.
Náſycone bywáją y drzewá PAńſkie : <i>y</i> Cedry Libánu / których náſádźił.
Ná których ptacy gniazdá ſwe máją / y boćian ná jedlinách <i>ma</i> dom ſwój.
Góry wyſokie dźikim kozam : á ſkáły / <i>ſą</i> ućiecżką królikom.
Ucżynił mieśiąc dla pewnych cżáſów : <i>á</i> ſłońce zna zachód ſwój.
Przywodźiƺ ćiemność / y bywa noc / w którą wychodzą wƺyſtkie zwierzętá leśne.
Lwiętá rycżą do łupu / y ƺukáją od Bogá pokármu ſwego.
<i>Lecż gdy<i> ſłońce wznidźie / záś śię zgromadzáją / y w jámách ſwojich kłádą śię.
<i>Tedy</i> wychodźi cżłowiek do roboty ſwojey / y do prace ſwojey / áż do wiecżorá.
O jákoż wielkie ſą ſpráwy twoje PAnie! te wƺyſtkie mądrześ ucżynił : <i>á</i> nápełnioná jeſt źiemiá bogáctwem twojim.
W morzu záś wielkim y bárzo ƺerokim / tám ſą płázy / którym niemáƺ licżby : y zwierzętá máłe y wielkie.
Po nim okręty przechodzą / y wieloryb / któregoś ty ſtworzył / áby w nim igrał.
Wszyſtko to ná ćię ocżekawa / ábyś im dał pokarm cżáſu ſwego :
<i>Gdy</i> im dawaƺ / zbieráją : gdy otwieraƺ rękę twoję / náſycone bywáją dobrymi rzecżámi.
<i>Lecż gdy</i> ukrywaƺ oblicże twoje / trwożą ſobą : gdy odbieraƺ duchá ich / giną / y w proch śię ſwój obracáją.
<i>Gdy</i> wyſyłaƺ duchá twego / ſtworzone bywáją : y odnawiaƺ oblicże źiemie.
Niechajże będźie chwałá PAńſka ná wieki : niech śię rozweſeli PAN w ſpráwách ſwojich.
On gdy wejrzy ná źiemię / zádrży : dotknie śię gór / á zákurzą śię.
Będę śpiewał PANU zá żywotá mego : będę śpiewał Bogu memu / póki mię ſtawa.
O nim będźie wdźięcżna mowá mojá : á ja śię rozweſelę w PAnu.
O by byli wytráceni grzeƺnicy z źiemie / á niezbożnych áby już nie było. Błogoſław duƺo mojá PAnu / Hállelujah.
