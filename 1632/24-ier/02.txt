Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Idź / á wołáj w uƺy Jeruzálemſkie / mówiąc : Ták mówi Pán : Wſpomniáłem ná ćię dla miłośierdźiá młodośći twojey pokázánego / y dla miłośći ślubin twojich ; gdyś chodźiłá zá mną ná puƺcży / w źiemi / w którey nie ośiewáją.
Kiedy Izráel był świętobliwośćią Pánu / y pierwoćinámi urodzájów jego ; wƺyſcy / którzy go pożeráli / winni byli / złe rzecży przyƺły ná nich / mówi Pán.
Słuchájćie ſłowá Páńſkiego / domie Jákóbowy / y wƺyſtkie rodzáje domu Izráelowego!
Ták mówi Pán : Jákąż niepráwość ználeſli ojcowie wáśi przy mnie / iż śię oddálili odemnie / á chodząc zá márnośćią márnymi śię ſtáli?
Ták iż áni rzekli : Gdźież jeſt Pán / który nas wywiódł z źiemi Egipſkiej? który nas wodźił po puƺcży / po źiemi puſtey y ſtráƺney / po źiemi ſuchej y ćieniu śmierći / po źiemi / po którey nikt nie chodźił / á gdźie żáden cżłowiek nie mieƺkał?
Owƺem / gdym was wprowádźił do źiemi obfitey / ábyśćie pożywáli owoców jey / y dóbr jey / wƺedƺy tám ſplugáwiliśćie źiemię moję / á dźiedźictwo moje ucżyniliśćie obrzydliwośćią.
Kápłáni nie rzekli : Gdźież jeſt Pán? áni ći / którzy śię obieráją ucżeni w zakonie / poználi mię / y páſterze odſtąpili odemnie / y prorocy prorokowáli przez Báálá / y zá rzecżámi nie użytecżnemi chodźili.
Przecżże śię wżdy wádzę z wámi / mówi Pán / á z ſynámi ſynów wáƺych ſpieráć śię muƺę?
Przejdźćie przynájmniej wyſpy Cytym / á obácżćie ; y do Kedar poślijćie á uwáżájćie pilnie / y przypátrzćie śię / jeſli śię ſtáło co tákowego ;
Jeſli odmienił który naród bogów ſwojich / choćiáż oni nie ſą bogámi ; ále lud mój odmienił ſłáwę ſwoję w rzecż niepożytecżną.
Zdumiejćie śię niebioſá nád tym / á ulęknijćie śię / á zátrwożćie śię bárzo / mówi Pán ;
Bo dwojáką złość popełnił lud mój : mnie opuśćili / źródło wód żywych / á wykopáli ſobie cyſterny / cyſterny dźiuráwe / które wody zátrzymáć nie mogą.
Izali Izráel jeſt niewolnikiem álbo wychowáńcem w domu ſpłodzonym? Cżemuż jeſt podány ná łup?
Rycżą nań lwiętá / y wydáwáją głos ſwój / á obrácáją źiemię jego w puſtynię ; miáſtá jego ſpálone ſą / ták / że niemáƺ y jednego obywátelá.
Synowie też Nof y Táchpánes wierzch głowy twojey zetrą.
Záż tego ſám ſobie nie ſpráwujeƺ? opuƺcżájąc Páná / Bogá ſwego / wtencżás / kiedy ćię prowádźi drogą ſwą.
A teraz co zá ſpráwę maƺ ná drogách Egipſkich / iż pijeƺ wodę z Nilu? álbo co maƺ zá ſpráwę ná drogách Aſſyryjcżyków / iż pijeƺ wodę z rzeki ich?
Skárze ćię złość twojá á odwrócenie twoje ſfuká ćię. Wiedzże tedy y obácż / iż jeſt rzecż złá y gorzká / iżeś opuśćił Páná / Bogá twego / á niemáƺ bojáźni mojey w tobie / mówi Pán / Pán zaſtępów.
Gdym dáwno połámał járzmo twoje / y rozerwáłem związki twoje / mówiłáś : Nie będę ſłużyłá báłwánom ; á przećie ná káżdym págórku wyſokim / y pod káżdym drzewem gáłęźiſtem tułáƺ śię / o nierządnico!
A Jam ćię był náſádźił winną máćicą wyborną / którejby wƺyſtko naśienie było práwdźiwe ; jákożeś mi śię tedy odmieniłá w płonne gáłęźie obcey máćicy?
Bo choćbyś śię umywáłá y ſáletrą / y mydłem śię jáko nájbárdźiej tárłá / przećież znácżná zoſtánie niepráwość twojá przedemną / mówi pánujący Pán.
Jákoż mówiƺ : Nie jeſtem ſplugáwioná / zá Báálámi nie chodźiłám? Spojrzyj ná drogę twoję w tey dolinie / obácż / coś cżyniłá / o wielbłądźico prędká / która wikleƺ drogi ſwoje.
Oślicáś dźiká / przywykłáś ná puƺcży / która według żądzy duƺy ſwej wiátr łápie / gdy śię jey przycżyná da ; któż ją odwróći? Wƺyſcy / którzy jey ƺukáją / nie ſtrudzą śię / y w mieśiącu jey znájdą ją.
Rzekęlić : Záwśćiągnij nogi twojey / áby boſá nie byłá / y gárdło twe od prágnieniá / tedy mówiƺ : Już to prózno / nie ucżynię ; bom śię rozmiłowáłá w cudzych / y zá nimi pójdę.
Jáko wſtyd złodźiejá / kiedy go záſtáną / ták śię záwſtydźi dom Izráelſki / ſámi królowie ich / kśiążętá ich / y kápłáni ich / y prorocy ich /
Którzy mówią drewnu : Tyś jeſt oćiec mój / á kámieniowi : Tyś mię ſpłodźił. Bo śię do mnie obróćili tyłem / á nie twárzą ; ále cżáſu utrapienia ſwego máwiáją : Wſtáń á wybáw nas.
Y gdźież ſą bogowie twoi / którycheś ſobie nácżynił? Niech wſtáną / jeſli ćię mogą wybáwić cżáſu utrapienia twego / ponieważ ile maƺ miaſt ſwojich / tyle maƺ bogów ſwojich / o Judo!
Cżemuż śię zemną ſpieráć chcećie? Wyśćie wƺyſcy odſtąpili odemnie / mówi Pán.
Próżnom bił ſynów wáƺych / káránia nie przyjęli ; miecż wáƺ pożárł przećie proroków wáƺych / jáko lew trácący.
O národźie! wy rozſądźćie ſłowá Páńſkie. Izálim był puſtynią Izráelowi? Izali źiemią ćiemną? Przecżże mówi lud mój : Pánujemy / nie pójdźiemy więcey do ćiebie?
Izali zápominá pánná ubioru ſwego / y oblubienicá klejnotów ſwojich? Ale lud mój zápomniał mię przez dni niezlicżone.
Przecż dobrą być twierdźiƺ drogę twoję / ƺukájąc tego / w cżym śię kocháƺ? Przecż y innych nierządnic ucżyƺ złośliwych dróg twojich?
Nádto y ná podołkách twojich znájduje śię krew duƺ ubogich y niewinnych ; nie z prácą ználázłem to / bo to widźieć ná wƺyſtkich podołkách twojich.
A przećież mówiƺ : Poniewáżem niewinną / pewnie odwróconá jeſt zápálcżywość jego odemnie. Oto Ja w ſąd wnijdę z tobą / przeto / że mówiƺ : Nie zgrzeƺyłám.
Przecżże ták biegáƺ / odmieniájąc drogi ſwe? Ták będźieƺ poháńbioná od Egipcżánów / jákoś poháńbioná byłá od Aſſyryjcżyków.
Y z támtąd wyjdźieƺ / májąc ręce ſwe nád głową ſwą : bo Pán odrzucá ufnośći twoje / á nie poƺcżęśćić śię w nich.
