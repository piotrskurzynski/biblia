Jeſlibyś śię chćiał náwróćić / Izráelu! mówi Pán / do mnie śię náwróć. Bo jeſli odejmieƺ obrzydłośći twoje od oblicża mego / á nie będźieƺ śię tułał /
Y przyśiężeƺli w práwdźie / w ſądźie y w ſpráwiedliwośći / mówiąc : Jáko żyje Pán : tedy błogoſłáwić ſobie w nim będą narody / y w nim śię przechwáláć.
Abowiem ták mówi Pán mężom Judzkim y Jeruzálemſkim : Poorzćie ſobie nowinę / á nie śiejćie ná ćierniu ;
Obrzeżćie śię Pánu / á odejmijćie nieobrzezki ſercá wáƺego / mężowie Judzcy / á obywátele Jeruzálemſcy! by ſnadź nie wyƺłá jáko ogień popędliwość mojá / á nie zápáliłá śię / á nie byłby / ktoby ugáśił dla złośći przedśięwźięćiá wáƺego.
Opowiádájćie w Judźie / á w Jeruzálemie ogłáƺájćie / y mówćie : Zátrąbćie w trąbę w źiemi / zwołájćie / zgromádźćie lud / á mówćie : Zbierzćie śię / á wejdźmy do miaſt obronnych.
Podnieśćie chorągiew ná Syonie / bądźćie ſercá dobrego / nie záſtánáwiájćie śię ; bo Ja złe przywiodę od północy / y poráżkę wielką.
Wychodźi lew z jáſkini ſwojey / á ten / który niƺcży narody / wyƺedƺy z miejſcá ſwego / ćiągnie / áby obróćił źiemię twoję w puſtynię / á miáſtá twoje áby zburzone były / áby nie było żádnego obywátelá.
Przetoż przepáƺćie śię worámi / nárzekájćie á kwilćie / bo nie jeſt odwrócony gniew zápálcżywośći Páńſkiej od nas.
Y ſtánie śię dniá onego / mówi Pán / że zginie ſerce królewſkie / y ſerce kśiążąt / á zdumieją śię kápłáni / y prorocy dźiwowáć śię będą.
Y rzekłem : Ach pánujący Pánie! zápráwdęś bárzo ten lud y Jeruzálem omylił / mówiąc : Pokój mieć będźiećie! á wżdy miecż przeniknął áż do duƺy.
Cżáſu onego rzeką temu ludowi y Jeruzálemowi : Wiátr gwáłtowny z miejſc wyſokich ná puƺcży idźie proſto ná lud mój / nie żeby przewiewał / áni wycżyƺcżał.
Wiátr gwáłtowniejƺy niż oni przydźie mi ; teraz Ja też opowiem im ſądy.
Oto wyſtępuje jáko obłoki / á wozy jego jáko wicher / prędƺe ſą niż orły konie jego. Biádá nam! bośmy ſpuſtoƺeni.
Omyj od złośći ſerce twoje / Jeruzálem! ábyś wybáwione było. Dokądże trwáć będą w pośrzodku ćiebie myśli niepráwośći twojey.
Bo głos opowiádájącego idźie od Dánu / á tego / który ogłáƺá niepráwość / z góry Efráim.
Przypominájćież tym národom : Oto ogłáƺájćie Jeruzálemcżykom / że ſtróżowie przychodzą z źiemi dálekiej / á wydáwáją przećiwko miáſtom Judzkim głos ſwój.
Jáko ſtróżowie pól położą śię przećiwko niemu w około ; bo mię do gniewu wzruƺyło / mówi Pán.
Drogá twojá y poſtępki twoje to ucżyniły tobie ; toć złość twojá przynioſłá / że to jeſt gorzkie / á że przeniká áż do ſercá twego.
O wnętrznośći moje / wnętrznośći moje! boleść ćierpię. O ośierdźiá moje! trwoży śię we mnie ſerce moje / nie zámilcżę ; bo głos trąby ſłyƺyƺ / duƺo mojá! y okrzyk wojenny.
Poráżká zá poráżką ogłáƺá śię / ſpuſtoƺoná będźie zájiſte wƺyſtká źiemiá ; nágle ſpuſtoƺone będą námioty moje / y opony moje w okámgnieniu.
Dokądże widźieć będę chorągiew / y ſłyƺeć głos trąby?
Bo głupi lud mój nie zná mię / ſynowie niemądrzy y nierozumni ſą ; mądrzy ſą do cżynieniá złego / ále dobrze cżynić nie umieją.
Spojrzęli ná źiemię / á oto jeſt niepozorná y próżná ; jeżeli ná niebo / niemáƺ ná nim świátłośći.
Spojrzęli ná góry / á oto śię trzęſą / y wƺyſtkie págórki chwieją śię.
Spojrzęli / á oto niemáƺ cżłowieká / y wƺelkie ptáſtwo niebieſkie odlećiáło.
Spojrzęli / á oto pole urodzájne jeſt puſtynią / á wƺyſtkie miáſtá jego zburzone ſą od oblicża Páńſkiego / y od oblicża gniewu zápálcżywośći jego.
Bo ták mówi Pán : Spuſtoƺoná będźie wƺyſtká źiemiá / wƺákże końcá jeƺcże nie ucżynię.
Nád tym źiemiá kwilić będźie / á niebioſá w górze záćmią śię / przeto / żem mówił / com umyślił / á nie żáłuję / áni śię odwrócę od tego.
Przed grzmotem jeźdźców y ſtrzelájących z łuku ućiecże wƺyſtko miáſto ; wejdą do gęſtych obłoków / y ná ſkáły wſtąpią. Wƺyſtkie miáſtá opuƺcżone będą / á nie będźie / ktoby w nich mieƺkał.
A ty zburzoná będąc cóż ucżyniƺ? Choćbyś śię ubráłá w ƺárłát / choćbyś śię ozdobiłá ozdobą złotą / choćbyś też oblicże twe przypráwiłá bárwicżką / prózno śię ſtroiƺ ; wzgárdzą tobą zálotnicy twoi / á duƺy twojey ƺukáć będą.
Bom ſłyƺał głos jáko rodzącey / ućiſki jáko pierworodzącey / głos córki Syońſkiej nárzekájącey / á záłámującey ręce ſwe / mówiąc : Biádá mnie teraz! bo uſtáłá duƺá mojá dla morderców.
