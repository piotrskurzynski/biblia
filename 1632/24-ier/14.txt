Słowo Páńſkie / które śię ſtáło do Jeremijáƺá o ſuƺy.
Ziemiá Judzká płákáć będźie / á bramy jey zemdleją / żáłobę nośić będą ná źiemi / á nárzekánie Jeruzálemſkie wſtąpi w górę ;
Y zácniejśi z nich rozſyłáć będą nájpodlejƺych ſwojich po wodę ; á przyƺedƺy do cyſtern / y nie ználázƺy wody / náwrócą śię z nácżyniem ſwojim próżnem / zápłonąwƺy y záwſtydźiwƺy śię ; przetoż nákryją głowę ſwoję.
Dla źiemi uprágnioney / przeto / że deƺcżu nie będźie ná źiemi / y orácże wſtydząc śię nákryją głowy ſwoje.
Owƺem y łáni / co ná polu porodźiłá / opuśći ; bo ná polu tráwy nie będźie.
A oſły dźikie / ſtájąc ná wyſokich miejſcách / chwytáć będą wiátr jáko ſmoki ; uſtáną ocży ich / bo nie będźie tráwy.
O Pánie! ponieważ niepráwośći náƺe świádcżą przećiwko nam / zmiłuj śię dla imienia twego ; boć wielkie ſą odwróceniá náƺe / tobieśmy zgrzeƺyli.
O nádźiejo Izráelowá / wybáwićielu jego cżáſu utrapienia! cżemuż maƺ być jáko przychodźień w tey źiemi / á jáko podróżny wſtępujący ná nocleg?
Cżemuż śię pokázujeƺ jáko mąż ſtrudzony / álbo jáko mocárz / który nie może wybáwić? Wƺákeś ty jeſt w pośrzodku nas / Pánie! á imię twoje wzywáne jeſt nád námi ; nie opuƺcżájże nas.
Ták mówi Pán o tym ludu : Iż ták miłują tułánie / á nóg ſwych nie powśćiągáją / przetoż śię Pánu niepodobáją / y teraz wſpominá niepráwośći ich / á náwiedzá grzechy ich.
Potem rzekł Pán do mnie : Nie módl śię zá tym ludem.
Gdy pośćić będą / Ja nie wyſłuchám wołánia ich ; á gdy ofiárowáć będą cáłopálenie / y ofiárę śniedną / Ja tego nie przyjmę ; ále miecżem / y głodem / y morem wytrácę ich.
Y rzekłem : Ach / pánujący Pánie! oto im ći prorocy mówią : Nie oglądáćie miecżá / á głód nie przydźie ná was / ále pokój pewny dam wam ná tym miejſcu.
Y rzekł Pán do mnie : Fáłƺ prorokują ći prorocy w imieniu mojim ; nie poſłáłem ich / ánim im rozkázał / owƺem / ánim mówił do nich ; widzenie kłámliwe / y wieƺcżbę / y márność / y kłámſtwo ſercá ſwego oni wam prorokują.
Przetoż ták mówi Pán o prorokách / którzy prorokują w imieniu mojim / choćiáżem Ja ich nie poſłał / y którzy mówią : Miecżá áni głodu nie będźie w tey źiemi ; ći ſámi prorocy miecżem y głodem zginą.
A lud ten / któremu oni prorokują / rozrzucony będźie po ulicách Jeruzálemſkich od głodu y od miecżá / á nie będźie / ktoby ich pogrzebał / onych ſámych / żony ich / y ſynów ich / y córki ich ; ták wyleję ná nich złość ich.
Przetoż rzecżeƺ do nich to ſłowo : Ocży moje wylewáją łzy w nocy y we dnie bez przeſtánku ; bo ſkruƺeniem wielkiem ſkruƺoná będźie pánná / córká ludu mojego / y ráną bárzo boleſną.
Wynidęli ná pole / oto tám pomordowáni miecżem ; wnidęli do miáſtá / oto y tám zmorzeni głodem ; bo jáko prorok ták y kápłán obchodząc kupcżą źiemią / á ludźie tego nie bácżą.
Izali do końcá odrzucáƺ Judę? Izali Syon obrzydźiłá ſobie duƺá twojá? Przecż nas bijeƺ / ták ábyſmy już nie byli uzdrowieni? Ocżekujemyli ná pokój / álić oto náſtępuje nic dobrego ; á jeſli ná cżás ulecżeniá / á oto zátrwożenie.
Uznájemy / Pánie! niezbożność ſwoję / y niepráwość ojców náƺych / iżeśmy zgrzeƺyli przećiw tobie.
Nie odrzucájże nas dla imienia twego / nie podáwájże w lekkość ſtolicy chwały twojey ; wſpomnijże / nie tárgáj przymierza twego z námi.
Izali ſą miedzy márnośćiámi pogáńſkimi / coby ſpuƺcżáli deƺcż? álbo niebioſá mogąli ſáme przez śię dáwáć deƺcże? Izáliś nie ty ſám Pán / Bóg náƺ? Przetoż ocżekujemy ná ćię ; bo to wƺyſtko ty cżyniƺ.
