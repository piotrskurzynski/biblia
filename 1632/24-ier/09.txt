Kto mi to da / áby głowá mojá wodą byłá / á ocży moje źródłem łez / ábym we dnie y w nocy płákał pomordowánych córki ludu mego!
Któż mi da ná puƺcży goſpodę podróżnych / ábym opuśćił lud mój / y odƺedł od nich? bo wƺyſcy ſą cudzołożnicy / zgrájá przeſtępników ;
Y náćiągáją języká ſwego do kłámſtwá jáko łuku ſwego / zmocnili śię ná źiemi / ále nie ku práwdźie ; bo ze złego w złe poſtępują / á mnie nie znáją / mówi Pán.
Káżdy niech śię ſtrzeże bliźniego ſwego / á nie káżdemu brátu dowierzá ; bo káżdy brát jeſt ná tym jákoby podƺedł / á káżdy bliźni zdrádliwie poſtępuje.
Káżdy też bliźniego ſwego oƺukuje / á prawdy nie mówi ; náucżá języká ſwego mówić kłámſtwo / źle cżyniąc uſtáwáją.
Mieƺkánie twoje / o proroku! jeſt w pośrzodku ludu zdrádliwego ; dla zdrád nie chcą mię poznáć / mówi Pán.
A przetoż ták mówi Pán zaſtępów : Oto Ja płáwiąc ich próbowáłem ich ; jákoż śię tedy już mam obchodźić z córką ludu mego?
Strzáłą śmiertelną jeſt język ich / zdrádę mówi ; uſty ſwemi o pokoju z przyjáćielem ſwym mówi / ále w ſercu ſwem zákłádá nań śidłá ſwoje.
Izali dla tego nie náwiedzę ich? mówi Pán ; izali nád národem tákowym nie pomśći śię duƺá mojá?
Dla tych gór udám śię ná płácż y ná nárzekánie / y dla páſtwiſk / które ſą ná puƺcży / ná kwilenie ; bo ſpálone będą / ták / że nie będźie / ktoby je przechodźił / áni tám głoſu bydlęćiá ſłycháć będźie ; ptáſtwo niebieſkie y bydlętá rozbieżą śię y odejdą.
Y obrócę Jeruzálem w gromády rumu / w mieƺkánie ſmoków ; á miáſtá Judzkie obrócę w puſtynię / ták / iż nie będźie obywátelá.
Któż jeſt ták mądry / coby to wyrozumiał? á do kogo mówiły uſtá Páńſkie / coby to oznájmił / dlácżego zginąć ma tá źiemiá / y wypáloná być ma jáko puſtyniá / ták áby nie było / ktoby ją przeƺedł?
Bo Pán mówi : Iż opuśćili zakon mój / którym im przedłożył / á nie ſłucháli głoſu mojego / áni chodźili zá nim ;
Ale chodźili zá uporem ſercá ſwego y zá Báálem / cżego ich náucżyli ojcowie ich.
Dla tego ták mówi Pán zaſtępów / Bóg Izráelſki : Oto Ja nákármię ich / to jeſt lud ten / piołunem / á nápoję ich wodą żółći.
Abowiem rozproƺę ich miedzy narody / których nie ználi oni y ojcowie ich / y poślę zá nimi miecż / áż ich do końcá wygłádzę.
Ták mówi Pán zaſtępów : Uwáżćie to / á przyzówćie nárzekájących niewiáſt / niech przyjdą / á do tych / które ſą w tym wyćwicżone / poślijćie / áby przyƺły ;
Niech śię poſpieƺą / á niech ucżynią nád námi nárzekánie / áby ocży náƺe łzy wylewáły / á powieki náƺe opływáły wodą.
Głos zájiſte nárzekánia ſłyƺeć z Syonu : O jákośmy ſpuſtoƺeni! bárdzośmy zelżeni ; bośmy ſtráćili źiemię / bo rozrzucone ſą przybytki náƺe.
Owƺem / ſłuchájćie niewiáſty! ſłowá Páńſkiego / á niech przyjmie ucho wáƺe wyrok uſt jego / ábyśćie ucżyły córek ſwojich lámentu / á káżda z was towárzyƺkę ſwoję nárzekánia ;
Bo wlázłá śmierć oknámi náƺemi / weƺłá ná páłáce náƺe / áby wytráćiłá dźieći z rynku / á młodźieńce z ulic.
( Mów y to : Ták mówi Pán : ) Y pádły trupy ludzkie jáko gnój po polu / á jáko ſnopy zá żeńcámi / á niemáƺ ktoby pochował.
Ták mówi Pán : Niech śię nie chlubi mądry z mądrośći ſwojey / y niech śię nie chlubi mocárz z mocy ſwojey / y niech śię nie chlubi bogáty z bogáctw ſwojich ;
Ale w tym niecháj śię chlubi / kto śię chlubi / że rozumie á zná mię / żem Ja jeſt Pán / który cżynię miłośierdźie / ſąd y ſpráwiedliwość ná źiemi ; bo mi śię to podobá / mówi Pán.
Oto dni idą / mówi Pán / w których náwiedzę káżdego obrzezáńcá y nieobrzezáńcá :
Egipcżánów / y Judę / y Edomcżyków / y Amonitcżyków / y Moábcżyków / y wƺyſtkich / którzy w oſtátnim kąćie mieƺkáją ná puƺcży ; bo te wƺyſtkie narody nieobrzezáne ſą / á wƺyſtek dom Izráelſki jeſt nieobrzezány ſercem.
