Słowo / które mówił Pán przećiwko Bábilonowi / y przećiwko źiemi Cháldejſkiej przez Jeremijáƺá proroká ;
Opowiádájćie miedzy národámi / á rozgłáƺájćie / podnieśćie chorągiew / rozgłáƺájćie / nie tájćie / mówćie : Wźięty będźie Bábilon / poháńbiony będźie Bel / potárty będźie Merodách / poháńbione będą báłwány jego / pokruƺeni będą plugáwi bogowie jego.
Bo przyćiągnie przećiwko niemu naród z północy / który źiemię jego obróći w puſtynię / ták / że nie będźie / coby mieƺkał w niej ; ták ludźie jáko y bydlętá ruƺą śię y odejdą.
W onych dniách / y w onych cżáſách / mówi Pán / przyjdą ſynowie Izráelſcy / oni y ſynowie Judzcy ; płácżąc ſpołem ochotnie pójdą / á Páná / Bogá ſwego / ƺukáć będą.
O drodze do Syonu pytáć śię będą / á obróćiwƺy śię tám twárzą / rzekną : Pójdźćie / á przyłącżćie śię do Páná przymierzem wiecżnem / które nie przydźie w zápámiętánie.
Lud mój jeſt jáko trzodá owiec ſtráconych / páſterze ich záwiedli ich w błąd / po górách rozegnáli ich : z góry ná págórek chodźiły / zápomniáwƺy legowiſká ſwego.
Wƺyſcy / którzy ich znájdują / pożeráją ich / á nieprzyjáćiele ich mówią : Nie będźiemy nic winni / przeto / że zgrzeƺyli Pánu w przybytku ſpráwiedliwośći / Pánu / który jeſt nádźieją ojców ich.
Uchodźćie z pośrzodku Bábilonu / á z źiemi Cháldejſkiej wychodźćie / y bądźćie jáko kozły przed trzodą.
Bo oto Ja wzbudzę y przywiodę ná Bábilon zgromádzenie narodów wielkich z źiemi północney ; którzy śię uƺykują przećiwko niemu / á z támtąd go wezmą ; których ſtrzáły ſą jáko mocárzá ośierácájącego / z których żádná śię nie wróći prózno.
Y będźie Cháldejſká źiemiá ná łup / á wƺyſcy / którzy ją złupią / náſyceni będą / mówi Pán.
Przeto / że śię weſelićie / przeto / że śię rádujećie / rozchwytájąc dźiedźictwo moje / przeto / żeśćie nábráli ćiáłá jáko jáłowicá utucżoná / á wyſkákujećie jáko mocárze.
Záwſtydzoná będźie mátká wáƺá bárzo ; zápłonie śię rodźićielká wáƺá ; oto nájpośledniejƺą będźie z narodów / puſtą źiemią / ſuchą y bezdrożną.
Dla gniewu Páńſkiego nie będą w niej mieƺkáć / ále oná wƺyſtká obróći śię w puſtynię ; ktokolwiek pójdźie mimo Bábilonu / zdumieje śię / y záświśnie nád wƺyſtkiemi plágámi jego.
Szykujćie wojſká przećiw Bábilonowi zewƺąd / wƺyſcy / co ćiągniećie łuk / ſtrzelájćie do niego / nie żáłujćie ſtrzał ; bo przećiwko Pánu zgrzeƺył.
Wołájćie przećiwko niemu zewƺąd ; poddał śię / upádły grunty jego / ſkáżone ſą mury jego ; bo pomſtá Páńſka jeſt. Pomśćijćież śię nád nim ; jáko on cżynił innym / ták mu też ucżyńćie.
Wytráććie śiejącego z Bábilonu / y tego / który trzymá śierp cżáſu żniwá ; przed oſtrzem miecżá burzącego káżdy niech śię do ludu ſwego obróći / y káżdy do źiemi ſwojey niech ućiecże.
Izráel jeſt jáko bydlątko zágnáne / które lwy zápłoƺyły. Król Aſſyryjſki nájpierwƺy był / który go żreć pocżął / á ten oſtátecżny / Nábuchodonozor / król Bábilońſki / kośći jego pokruƺył.
Przetoż ták mówi Pán zaſtępów / Bóg Izráelſki : Oto Ja náwiedzę królá Bábilońſkiego y źiemię jego / jákom náwiedźił królá Aſſyryſkiego ;
Y przywrócę záś Izráelá do mieƺkánia jego / á páść śię będźie ná Kármelu / y ná Báſánie / y ná górze Efráimowej / á w Gáláádźie náſyconá będźie duƺá jego.
W onych dniách y w onych cżáſách / mówi Pán / będą ƺukáć niepráwośći Izráelowej / ále żádney nie będźie ; y grzechów Judzkich / ále nie będą ználeźione ; bo odpuƺcżę tym / których pozoſtáwię.
Wyćiągnij przećiwko tey źiemi odpornych / przećiwko niej / mówię / á obywáteli jey náwiedź ; ſpuſtoƺ á wygłádź / goniąc ich / mówi Pán ; ucżyńże według wƺyſtkiego / jákoć rozkázuję.
Niech będźie głos wojenny w tey źiemi / y ſpuſtoƺenie wielkie.
Jákożby pośiekány y połámány być miał młot wƺyſtkiej źiemi? Jákożby śię Bábilon ſtał zdumienie miedzy národámi?
Sidłom záſtáwił ná ćię / y będźieƺ pojmány / o Bábilonie! nim śię obácżyƺ ; ználeźiony náwet y pojmány będźieƺ / poniewáżeś z Pánem zwádę zácżął.
Otworzył Pán ſkárb ſwój / á wynióſł nacżynia gniewu ſwego ; bo to jeſt ſpráwá Páná / Páná zaſtępów w źiemi Cháldejſkiej.
Pójdźćież przećiwko niej od końcżyn źiemi / otwórzćież ƺpichlerze jey / podepcżćie ją jáko ſtogi / á wygłádźćie ją ták / áby jey nic nie zoſtáło ;
Pozábijájćie wƺyſtkie ćielce jey / niech zſtępują ná zábićie. Biádá im! bo przyƺedł dźień ich / cżás náwiedzeniá ich.
Głos ućiekájących / y tych / co uchodzą z źiemi Bábilońſkiej / áby oznájmili ná Syonie pomſtę Páná / Bogá náƺego / pomſtę kośćiołá jego.
Zgromádźćie przećiwko Bábilonowi wƺyſtkich ſtrzelców / którzy łuk ćiągną ; połóżćie śię obozem przećiw niemu w około / áby nikt nie uƺedł ; oddájćie mu według ſpráw jego / według wƺyſtkiego / co innym cżynił / ucżyńćie mu ; bo śię przećiwko Pánu wynośił / przećiwko Świętemu Izráelſkiemu.
Przetoż polegną młodźieńcy jego ná ulicách jego / y wƺyſcy mężowie wálecżni jego wygłádzeni będą dniá onego / mówi Pán.
Otom ja przećiwko tobie / o hárdy! mówi Pán / Pán zaſtępów ; bo już przyƺedł dźień twój y cżás / ábym ćię náwiedźił.
Potknie śię zájiſte hárdy / y upádnie / á nie będźie / ktoby go podnióſł ; y rozniecę ogień w miáſtách jego / który pożre wƺyſtko około niego.
Ták mówi Pán zaſtępów : Gwáłt śię dźieje ſynom Izráelſkim y ſynom Judzkim ſpołem / á wƺyſcy / którzy ich pojmáli / trzymáją ich / nie chcą ich puśćić.
Ale odkupićiel ich możny / Pán zaſtępów imię jego / pewnie że śię ujmie o krzywdę ich / áby ſpráwił pokój tey źiemi / y poruƺył obywáteli Bábilońſkich.
Miecż przydźie ná Cháldejcżyków / mówi Pán / y ná obywáteli Bábilońſkich / y ná kśiążąt jego / y ná mędrców jego ;
Miecż ná kłámców / áby zgłupieli / miecż ná mocárzy jego / áby ſkruƺeni byli ;
Miecż ná konie jego / y ná wozy jego / y ná wƺyſtko poſpólſtwo / które jeſt w pośrzodku jego / áby byli jáko niewiáſty ; miecż ná ſkárby jego / áby były rozchwycone ;
Suƺá ná wody jego / áby wyſchły ; bo źiemiá jeſt pełna obrázów rytych / á przy báłwánách ſwojich ƺáleją.
Przeto tám mieƺkáć będą beſtyje y ſtráƺne zwierzętá / mieƺkáć w nim będą młode ſowy ; á nie będą w nim mieƺkáć więcey ná wieki / y nie będą w nim mieƺkáć od narodu do narodu.
Jáko Pán podwróćił Sodomę y Gomorę / y przyległośći ich / mówi Pán / ták śię tám nikt nie oſádźi / áni będźie mieƺkał w niej ſyn cżłowiecży.
Oto lud przyćiągnie od północy / y naród wielki / y królowie wielcy wzbudzeni będą ze ſtron źiemi.
Łuk y włócżnię pochwycą / okrutnymi będą / á nie zmiłują śię ; głos ich jáko morze záƺumi / á ná koniách pojádą / uƺykowáni jáko mąż do bitwy przećiwko tobie / o córko Bábilońſká!
Jáko uſłyƺy król Bábilońſki wieść o nich / oſłábieją ręce jego / á ućiſk ogárnie go / y boleść jáko rodzącą.
Oto ácżkolwiek jáko lew wyſtępuje / y bárdźiej niż nádętość Jordánu śię podnośi przećiwko przybytkowi mocnego / wƺákże go w okámgnieniu wypędzę z niej / á tego / który jeſt wybrány / przełożę nád nią ; bo któż jeſt mnie podobnym ; y kto mi da rok? á kto jeſt tym páſterzem / któryby śię poſtáwił przećiwko mnie?
Przetoż ſłuchájćie rády Páńſkiej / którą urádźił przećiwko Bábilonowi ; y zámyſłów jego / które umyślił przećiwko źiemi Cháldejſkiej ; zájiſte żeć ich wywloką nájmniejśi z tey trzody / zájiſte poburzą ich y przybytek ich.
Od huku przy dobywániu Bábilonu poruƺy śię tá źiemiá / á krzyk miedzy národámi ſłyƺány będźie.
