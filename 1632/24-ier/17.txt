Grzech Judzki nápiſány jeſt piórem żeláznem / á oſtrym dyjámentem wyryty jeſt ná táblicy ſercá ich / y ná rogách ołtárzów wáƺych ;
Gdy wſpomináją ſynowie ich ná ołtárze ich / y ná gáje ich pod drzewem źielonem / ná págórkách wyſokich.
O góro / y pole moje! májętność twoję y wƺyſtkie ſkárby twoje podám ná rozƺárpánie dla grzechu wyżyn twojich / które maƺ po wƺyſtkich gránicách twojich.
A ty muśiƺ zániecháć zá przewinieniem twojim dźiedźictwá twego / którem ći dał. Y dam ćię w niewolę nieprzyjáćiołom twoim / y źiemi / którey nie znáƺ ; bośćie ogień rozniećili w popędliwośći mojey / który áż ná wieki gorzeć będźie.
Ták mówi Pán : Przeklęty mąż / który ufá w cżłowieku / y który pokłádá ćiáło rámieniem ſwojim / á od Páná odſtępuje ſerce jego.
Abowiem ſtánie śię jáko wrzos ná puƺcży / który nie cżuje / gdy co dobrego przychodźi / ále bywa ná ſuchych miejſcách ná puƺcży w źiemi ſłoney / y w którey nikt nie mieƺká.
Błogoſłáwiony mąż / który ufá w Pánu / á Pán jeſt nádźieją jego.
Bo będźie jáko drzewo wƺcżepione nád wodámi / á nád ſtrumieniem zápuƺcżájące korzenie ſwoje / które nie cżuje / gdy przychodźi gorącość / ále liść jego zoſtáje źielony / á roku ſuchego nie fráſuje śię / y nie przeſtáje przynośić owocu.
Nájzdrádliwƺe jeſt ſerce nádewƺyſtko y nájprzewrotniejƺe ; któż je pozná?
Ja Pán / który śię bádám ſercá / y doświádcżám nerek / ták ábym káżdemu oddał według drogi jego / y według owocu ſpráw jego.
Jáko kuropátwá zgromádzá jájká / ále ich nie wylęgá : ták / kto zbierá bogáctwá / á nieſpráwiedliwie / w połowie dni ſwojich opuśći je / á ná oſtátek będźie głupim ;
Ale miejſce świątnicy náƺej / to jeſt ſtolicá chwały Nájwyżƺego / wiecżnie trwá.
O nádźiejo Izráelſká / Pánie! wƺyſcy / którzy ćię opuƺcżáją / niech będą záwſtydzeni ; którzy odſtępują odemnie / niech ná źiemi zápiſáni będą ; ábowiem opuśćili źródło wód żywych / Páná.
Uzdrów mię / Pánie! á będę uzdrowiony : zbáw mię / á będę zbáwiony ; ábowiemeś ty chwałá mojá.
Oto oni do mnie mówią : Gdźież jeſt to ſłowo Páńſkie? Niechże już przydźie ;
Choćiáżem Ja tego nie zábiegał / ábym był páſterzem twoim / ánim dniá boleśći prágnął / ty wieƺ / cokolwiek wyƺło z uſt mojich / przed oblicżem twojim jeſt.
Nie bądźże mi ná poſtrách ; tyś nádźiejá mojá w dźień utrapienia.
Niech będą poháńbieni / którzy mię prześládują / á ja niech nie będę záwſtydzony ; niech śię oni lękáją / á ja niech śię nie lękám ; przywiedź ná nich dźień utrapienia / á dwojákiem ſkruƺeniem ſkruƺ ich.
Ták Pán rzekł do mnie : Idź / á ſtáń w bramie ſynów ludu tego / którą wchodzą królowie Judzcy / y którą wychodzą / y we wƺyſtkich bramách Jeruzálemſkich /
Y rzecż do nich : Słuchájćie ſłowá Páńſkiego / królowie Judzcy / y wƺyſtek Judo / y wƺyſcy obywátele Jeruzálemſcy / którzy chádzáćie temi bramámi!
Ták mówi Pán : Strzeżćie pilnie duƺ wáƺych / á nie nośćie brzemion żádnych w dźień ſábátu / áni ich wnośćie bramámi Jeruzálemſkiemi ;
Ani wynáƺájćie brzemion z domów wáƺych w dźień ſábátu / áni żádney roboty odpráwujćie / ále święććie dźień ſábátu / jákom rozkázał ojcom wáƺym.
Wƺákże nie uſłucháli / áni nákłonili uchá ſwego / owƺem / zátwárdźili kárk ſwój / nie ſłuchájąc áni przyjmując náuki.
A jeſli mię pilnie ſłucháć będźiećie / mówi Pán / ták / żebyśćie nie wnośili brzemion bramámi miáſtá tego w dźień ſábátu / ále święćili dźień ſábátu / nie odpráwując weń żádney roboty ;
Tedy wchodźić będą bramámi miáſtá tego królowie y kśiążętá śiedzący ná ſtolicy Dawidowej / jeżdżąc ná wozách y ná koniách / oni y kśiążętá ich / mężowie Judzcy y obywátele Jeruzálemſcy / y ſtáć będźie to miáſto áż ná wieki.
Y zbieżą śię z miaſt Judzkich y z okolicżnych miejſc Jeruzálemſkich / y z źiemi Benjáminowej / y z równin / y z tey góry / y od południá / przynoƺąc cáłopálenie / y ofiárę / y dar / y kádźidło / tákże y dźiękcżynienie nioſąc do domu Páńſkiego.
Ale jeſli mię nie uſłucháćie / ábyśćie święćili dźień ſábátu / á nie nośili brzemion / wchodząc bramámi Jeruzálemſkiemi w dźień ſábátu / tedy rozniecę ogień w bramách jego / który pożre páłáce Jeruzálemſkie / á nie będźie ugáƺony.
