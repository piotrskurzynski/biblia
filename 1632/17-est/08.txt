Onegoż dniá dał król Aſwerus Eſterze królowej dom Hámáná / nieprzyjáćielá żydowſkiego ; á Márdocheuƺ przyƺedł przed królá ; bo mu byłá oznájmiłá Eſter / że był jey pokrewnym.
Tedy zdjął król pierśćień ſwój / który był wźiął od Hámáná / y dał go Márdocheuƺowi / á Eſter poſtánowiłá Márdocheuƺá nád domem Hámánowym.
Potem jeƺcże Eſter mówiłá do królá / upádƺy u nóg jego / y płákáłá / y prośiłá go / áby wniwecż obróćił złość Hámáná Agágiejcżyká / y zámyſł jego / który był wymyślił przećiwko Żydom.
Tedy wyćiągnął król ná Eſterę ſceptr złoty / á Eſterá wſtáwƺy ſtánęłá przed królem /
Y rzekłá : Jeſli śię królowi podobá / á jeſlim ználázłá łáſkę przed oblicżem jego / y jeſli śię to zá ſłuƺne zda być królowi / y jeſlim ja przyjemná w ocżách jego / niech nápiƺą / áby były odwołáne liſty zámyſłów Hámáná / ſyná Hámedátá Agágiejcżyká / które rozpiſał ná wytrácenie Żydów / którzy ſą po wƺyſtkich kráinách królewſkich.
Abowiem jákożbym mogłá pátrzeć ná to złe / któreby przyƺło ná lud mój? álbo jákobym mogłá widźieć zginienie rodźiny mojey?
Y rzekł król Aſwerus do Eſtery królowej / y do Márdocheuƺá Żydá : Otom dom Hámánowy dał Eſterze / á onego powieƺono ná ƺubienicy / przeto / iż śćiągnął rękę ſwoję ná Żydów.
Wy tedy piƺćie do Żydów / jáko śię wam podobá / imieniem królewſkiem / y zápiecżętujćie pierśćieniem królewſkim ; ábowiem to / co śię piƺe imieniem królewſkiem / y piecżętuje śię pierśćieniem królewſkim / nie może być odwołáne.
A ták zwołáno piſárzy królewſkich onego cżáſu / mieśiącá trzećiego / ( ten jeſt mieśiąc Sywán ) dwudźieſtego y trzećiego dniá tegoż mieśiącá / á piſáno wƺyſtko / jáko rozkázał Márdocheuƺ / do Żydów y do kśiążąt / y do ſtároſtów / y do przełożonych nád kráinámi / którzy ſą od Indyi áż do Murzyńſkiej źiemi nád ſtem dwudźieſtą y śiedmią kráin / do káżdej kráiny piſmem jey / y do káżdego narodu językiem jego / y do Żydów piſmem ich y językiem ich.
A gdy nápiſał imieniem królá Aſweruſá / y zápiecżętował pierśćieniem królewſkim / rozeſłał liſty przez poſłów / którzy jeżdżáli ná koniách prędkich / y ná mułách młodych :
Iż król dał wolność Żydom / którzy byli we wƺyſtkich miáſtách / áby śię zgromádźili / á záſtáwiáli śię o duƺę ſwoję / á żeby wytráćili / wymordowáli / y wygubili wƺyſtkie wojſká ludu onego / y kráin tych / którzyby im gwáłt cżynili / dźiátkom ich / y żonom ich / á łupy ich żeby rozchwyćili ;
A to jednego dniá po wƺyſtkich kráinách królá Aſweruſá / to jeſt trzynáſtego dniá / mieśiącá dwunáſtego / ten jeſt mieśiąc Adar.
Sumá tych liſtów byłá : Żeby wydáno wyrok w káżdej krájinie / y oznájmiono wƺyſtkim národom / áby byli Żydźi gotowi ná on dźień ku pomśćie nád nieprzyjáćiołmi ſwymi.
Tedy poſłowie / którzy jeżdżáli ná koniách prędkich y ná mułách / bieżeli ják nájprędzej z rozkázániem królewſkiem / á przybity był ten wyrok w Suſán ná páłácu królewſkim.
A Márdocheuƺ wyƺedł od królá w ƺáćie królewſkiej hijácyntowej y biáłej / y w wielkiej koronie złotey / y w płáƺcżu biśiorowem / y ƺárłátnym ; á miáſto Suſán weſeliło y rádowáło śię.
A Żydom weƺłá świátłość y weſele / y rádość y cżeść.
Tákże w káżdej krájinie / y w káżdym mieśćie / y ná wƺelkiem miejſcu / gdźiekolwiek rozkáz królewſki / y wyrok jego doƺedł / mieli Żydźi weſele / rádość / ucżty / y dźień ućieƺny ; á wiele z narodów onych kráin zoſtáwáli Żydámi ; ábowiem ſtrách był przypádł od Żydów ná nie.
