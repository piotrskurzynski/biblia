Powſtáń / objáśnij śię! ponieważ przyƺłá świátłość twojá / á chwałá Páńſka weƺłá nád tobą.
Bo oto ćiemnośći okryją źiemię / á záćmienie narody ; ále nád tobą wejdźie Pán / á chwałá jego nád tobą widźiáná będźie.
Y będą chodźić narody w świátłośći twojey / á królowie w jáſnośći / która wejdźie nád tobą.
Podnieś w około ocży twe / á ſpojrzyj ; ći wƺyſcy / którzy śię zgromádźili / pójdą do ćiebie ; ſynowie twoi z dáleká przyjdą / á córki twoje przy boku twoim chowáne będą.
Tedy oglądáƺ to / á rozweſeliƺ śię ; tedy śię zdumieje y rozƺerzy ſerce twoje / gdy śię obróći ku tobie zgrájá morſká / á moc narodów przydźie do ćiebie.
Obfitość wielbłądów okryje ćię / tákże dromedárze z Mádyján y z Efy. Wƺyſcy ći przyjdą z Sáby / złoto y kádźidło przynioſą / á chwały Páńſkie opowiádáć będą.
Wƺyſtkie ſtádá z Kedar zgromádzą śię do ćiebie ; bárány z Nebájotu ſłużyć ći będą / á ofiárowáne będąc ná ołtárzu moim / przyjemne będą ; á ták dom májeſtátu mego ozdobię.
Y rzecżeƺ : Którzyż to ſą / co śię jáko obłoki zlátują / y jáko gołębie do okien ſwojich?
Ná mięć zájiſte wyſpy ocżekują / y okręty morſkie zdáwná / áby przywiedli ſynów twojich z dáleká / tákże ſrebro ſwoje z ſobą / y złoto ſwoje imieniowi Páná / Bogá twego / y Świętego Izráelſkiego ; bo ćię uwielbi.
Y pobudują cudzoźiemcy mury twoje / á królowie ich ſłużyć ći będą / gdyż w rozgniewániu mojim uderzę ćię / á w upodobániu mojim zlituję śię nád tobą.
Y będą otworzone bramy twoje uſtáwicżnie ; we dnie y w nocy nie będą zátkáne / áby przywiedźiono do ćiebie moc pogán / y królowie ich áby byli przywiedźieni.
Náród ten y króleſtwo / którećby nie ſłużyło / zginie ; narody tákie / mówię / do ƺcżętu ſpuſtoƺone będą.
Słáwá Libánu do ćiebie przydźie / jedliná / ſoſná / tákże bukƺpán / dla ozdoby miejſcá świątnicy mojey / ábym miejſce nóg mojich uwielbił.
Przyjdą tákże do ćiebie w pokorze ſynowie tych / którzy ćię trápili / y będą śię kłániáć ſtopom nóg twojich / którzykolwiek pogárdźili tobą / y názwią ćię miáſtem Páńſkim / Syonem Świętego Izráelſkiego.
Miáſto tego / coś opuƺcżoná y w nienáwiśći byłá / ták / że nie było / ktoby przez ćię chodźił / wyſtáwię ćię zá doſtojność wiecżną / y weſele od narodu do narodu.
Bo ſſáć będźieƺ mleko narodów / y pierśiámi królów kármioná będźieƺ ; y poznáƺ / iżem Ja Pán / zbáwićiel twój y odkupićiel twój / mocárz Jákóbowy ;
Miáſto miedźi nánioſę złotá / á miáſto żelázá nánioſę ſrebrá / á miáſto drew miedźi / á miáſto kámieniá żelázá ; y poſtáwię nád tobą dozorców ſpokojnych / y urzędników ſpráwiedliwych.
Nie będźie więcey ſłycháć o drápieſtwie w źiemi twojey / o zburzeniu y ſpuſtoƺeniu ná gránicách twojich ; ále ogłáƺáć będźieƺ zbáwienie ná murách twojich / á chwałę w bramách twojich.
Nie będźieƺ miał więcey ſłońcá zá świátłość dźienną / á jáſność mieśiącá nie oświeći ćię / ále Pán będźie świátłośćią twoją wiecżną / á Bóg twój ſłáwą twoją.
Nie zájdźie więcey ſłońce twoje / á mieśiąc twój nie ſkryje śię ; bo Pán będźie wiecżną świátłośćią twoją ; á ták dokonáją śię dni ſmutku twego.
Lud tákże twój / którzybykolwiek byli ſpráwiedliwi / ná wieki odźiedźicżą źiemię ; będą látoroślą ƺcżepieniá mego / dźiełem rąk mojich / ábym w nim był uwielbiony.
Nájmniejƺy rozmnoży śię ná tyśiące / á málucżki pocżet w naród niezlicżony. Ja Pán cżáſu ſwego prędko to ucżynię.
