Któż uwierzył kázániu náƺemu / á rámię Páńſkie komu objáwione jeſt?
Bo wyróſł jáko látoroſłká przed nim / á jáko korzeń z źiemi ſuchej / nie májąc kƺtáłtu áni pięknośći ; y widźieliśmy go ; ále nic nie było widźieć / cżemubyſmy go żądáć mieli.
Nájwzgárdzeńƺy był / y nájpodlejƺy z ludźi / mąż boleśći / á świádomy niemocy / y jáko zákrywájący twárz ſwoję ; nájwzgárdzeńƺy mówię / ſkądeśmy go zá nic nie mieli.
Zájiſte on niemocy náƺe wźiął ná śię / á boleśći náƺe włáſne nośił ; á myśmy mniemáli / że jeſt zrániony / ubity od Bogá y utrápiony.
Lecż on zrániony jeſt dla wyſtępków náƺych / ſtárty jeſt dla niepráwośći náƺych ; káźń pokoju náƺego jeſt ná nim / á śinośćią jego jeſteśmy uzdrowieni.
Wƺyſcyśmy jáko owce zbłądźili / káżdy ná drogę ſwą obróćiliśmy śię / á Pán włożył nań niepráwość wƺyſtkich nas.
Ućiśniony jeſt y utrápiony / á nie otworzył uſt ſwojich ; jáko báránek ná zábićie wiedźiony był / y jáko owcá przed tymi / którzy ją ſtrzygą / oniemiał / y nie otworzył uſt ſwojich.
Z więźieniá y z ſądu wyjęty jeſt ; przetoż rodzáj jego któż wypowie? Abowiem wyćięty jeſt z źiemi żyjących / á zrániony dla przeſtępſtwá ludu mojego ;
Który to lud podał niezbożnym grób jego / á bogátemu śmierć jego / choć jednák niepráwośći nie ucżynił / áni zdrádá ználeźioná jeſt w uſtách jego.
Tákći śię Pánu upodobáło zetrzeć go / y niemocą utrápić / áby położywƺy ofiárą zá grzech duƺę ſwą / ujrzał naśienie ſwoje / przedłużył dni ſwojich ; á to / co śię podobá Pánu / przez rękę jego áby śię ƺcżęśliwie wykonáło.
Z prácy duƺy ſwej ujrzy owoc / którym náſycon będźie. Znájomośćią ſwoją wielu uſpráwiedliwi ſpráwiedliwy ſługá mój ; bo niepráwośći ich on ſám ponieśie.
Przetoż mu dam dźiał dla wielu / áby śię dźielił korzyśćią z mocárzámi / ponieważ wylał ná śmierć duƺę ſwoję / á z przeſtępcámi policżon będąc / on ſám grzech wielu odnióſł / y zá przeſtępców śię modlił.
