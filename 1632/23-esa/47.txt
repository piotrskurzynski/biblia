Zſtąp / á uśiądź w prochu / pánno / córko Bábilońſká! śiądź ná źiemi / á nie ná ſtolicy / córko Cháldejſká! bo ćię nie będą więcey názywáć kochánką y rozkoƺnicą.
Weźmij żárná / á miel mąkę ; odkryj wárkocże ſwoje / obnáż nogi / odkryj golenie / brnij przez rzekę.
Odkrytá będźie nágość twojá / á háńbá twojá widźiáná będźie ; wezmę pomſtę z ćiebie / á nie dam śię nikomu záhámowáć.
To mówi odkupićiel náƺ / imię jego Pán zaſtępów / Święty Izráelſki.
Siedź milcżąc / á wnijdź do ćiemnośći / córko Cháldejſká! bo ćię więcey nie będą názywáć pánią króleſtw.
Rozgniewáłem śię był ná lud mój / ſplugáwiłem dźiedźictwo moje / á dáłem je w ręce twoje ; áleś im ty nie okázáłá miłośierdźiá / y ſtárców obćiążáłáś járzmem twojim bárzo /
Y rzekłáś : Ná wieki pánią będę ; y ták nie przypuśćiłáś tego do ſercá ſwego / ániś ſobie przywodźiłá ná pámięć dokońcżeniá tego.
Przetoż ſłucháj tego teraz / rozkoƺnico! ( która mieƺkáƺ bezpiecżnie / á mówiƺ w ſercu ſwem : Jam jeſt / á niemáƺ oprócż mnie inney / nie będę wdową / áni uznam śieroctwá ; )
Że to oboje przydźie ná ćię nágle dniá jednego / śieroctwo y wdowſtwo / á doſkonále przypádnie ná ćię dla mnóſtwá guſeł twojich / y dla wielkośći cżárów twojich.
Bo ufáƺ w złośći twojey / á mówiƺ : Nie widźi mię nikt. Mądrość twojá y umiejętność twojá / tá ćię przewrotną ucżyniłá / ábyś mówiłá w ſercu ſwem : Jam jeſt / á niemáƺ oprócż mnie inney.
Dla tego przydźie ná ćię złe / którego wyśćia nie wieƺ / y przypádnie ná ćię biedá / którey nie będźieƺ mogłá zbyć ; á przydźie ná ćię nagłe ſpuſtoƺenie / nim wzwieƺ.
Stáńże teraz z cżárámi ſwemi / y z mnóſtwem guſeł twojich / któremiś śię páráłá od młodośći twojey / ázáżbyś co ſobie mogłá pomóc / álbo śię ſnadź cżym zmocnić.
Uſtáwáƺ z mnóſtwem rád twojich ; niechájże teraz ſtáną práktykárze / którzy śię przypátrują gwiázdom / którzy dáwáją znáć / co ma być káżdego mieśiącá / á niech ćię wybáwią z tego / co ma przyść ná ćię.
Oto ſą jáko plewá ; ogień popáli ich / nie wybáwią áni duƺy ſwej z mocy płomieniá ; nie zoſtánie węglá do ogrzánia śię / áni ogniá / coby pośiedźieć przy nim.
Tákći śię ſtánie kupcom twoim / z którymiś śię zábáwiáłá od młodośći twojey : káżdy śię z nich w ſwą ſtronę udá / nie będźie / ktoby ćię wybáwił.
