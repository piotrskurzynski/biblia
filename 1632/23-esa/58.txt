Wołáj wƺyſtkim gárdłem / nie záwśćiągáj ; wynoś głos ſwój jáko trąbá / á opowiedz ludowi mojemu przeſtępſtwá ich / á domowi Jákóbowemu grzechy ich ;
Choćiáż mię káżdego dniá ƺukáją / á znáć chcą drogi moje / jáko naród / który ſpráwiedliwość cżyni / á ſądu Bogá ſwego nie opuƺcżá ; pytáją mię o ſądách ſpráwiedliwośći á prágną śię przybliżyć do Bogá / mówiąc :
Przecżże pośćimy / gdyż ná to nie pátrzyƺ? trápimy duƺę náƺę / á nie widźiƺ? Oto w dźień poſtu wáƺego przewodźićie wolę ſwoję / á wƺyſtkie práce ſwoje wyćiągáćie.
Oto pośćićie ná ſwáry / y ná zwády / y bijećie pięśćią niemiłośćiwie ; nie pośćićie / ják śię godźi tych dni / áby był ſłyƺány ná wyſokośći głos wáƺ.
Izali to jeſt tákowy poſt / jákim obrał / á dźień / w któryby trápił cżłowiek duƺę ſwoję? żeby zwieśił jáko śitowie głowę ſwoję / á wór y popiół ſobie podśćielał? Toż to názwieƺ poſtem / y dniem przyjemnym Pánu?
Ale to jeſt poſt / którym obrał : Rozwiąż związki niepobożnośći / rozwiąż brzemioná ćiężkie / y wolno puść ſkruƺonych / á ták wƺelákie járzmo rozerwij ;
Ułámuj łáknącemu chlebá twego / á ubogich wygnáńców wprowádź do domu twego ; ujrzyƺli nágiego / przyodźiej go / á przed ćiáłem ſwojim nie ukrywáj śię.
Tedy wyniknie jáko zorzá ránná świátłość twojá / á zdrowie twoje prędko zákwitnie! y pójdźie przed tobą ſpráwiedliwość twojá / á chwałá Páńſka zbierze ćię.
Tedy wzywáć będźieƺ / á Pán wyſłuchá ; záwołáƺ / á odpowieć : Owom Ja. Jeſli odejmieƺ z pośrzodku śiebie y járzmo / á przeſtánieƺ pálcá wyćiągáć / y mówić niepráwośći ;
Jeſli wylejeƺ łáknącemu duƺę ſwoję / á duƺę utrápioną náſyćiƺ : tedy wejdźie w ćiemnośći świátłość twojá / á zmierzk twój będźie jáko południe.
Bo ćię Pán uſtáwicżnie poprowádźi / y náſyći pod nájwiękƺą ſuƺą duƺę twoję / á kośći twoje utucży / y będźieƺ jáko ogród wilgotny / á jáko zdrój wód / którego wody nie uſtáwáją.
Y pobudują ſpłodzeni od ćiebie puſtynie ſtárodáwne / grunty od narodu do narodu wywiedźieƺ ; y názwą ćię nápráwcą obálin / y przepráwcą śćieżek ku mieƺkániu.
Jeſliże odwróćiƺ od ſábátu nogę ſwoję / ábyś nie przewodźił woli ſwojey w dźień mój święty ; y jeżeli názwieƺ ſábát rozkoƺą / dniem świętym á Pánu ſłáwnym / y będźieƺli go miał w ucżćiwośći / ták / ábyś weń nie cżynił dróg ſwojich / y nie przewodźił woli ſwej / y nie mówił ſłowá próżnego :
Tedy będźieƺ rozkoƺował w Pánu ; y wprowádzę ćię ná wyſokie miejſcá źiemi / y ſpráwię to / ábyś pożywał dźiedźictwá Jákóbá / ojcá twego ; bo uſtá Páńſkie mówiły.
