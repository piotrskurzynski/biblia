Brzemię Dámáƺku. Oto Dámáƺek przeſtánie być miáſtem / á ſtánie śię kupą rumu.
Miáſtá Aroer opuƺcżone będą ; dla trzód będą / które tám odpocżywáć będą / á nie będźie / ktoby je ſtráƺył.
Y uſtánie obroná od Efráimá / y króleſtwo od Dámáƺku / y od oſtátká Syryjcżyków / y jáko ſłáwá ſynów Izráelſkich zniƺcżeją / mówi Pán zaſtępów.
Y ſtánie śię dniá onego / że umniejƺoná będźie ſłáwá Jákóbowá / á tłuſtość ćiáłá jego ſchudnie.
Abowiem Azur będźie jáko ten / który w żniwá zboże zbierá / á rámię jego żnie kłoſy ; y będźie jáko ten / co zbierá kłoſy w dolinie Refáim.
Wƺákże zoſtáną ná nim pominione groná / jáko ná otrzęśnioney oliwie dwie álbo trzy oliwki zoſtáną ná wierzchu drzewá / á cżtery álbo pięć ná rodzájnych gáłęźiách jego / mówi Pán / Bóg Izráelſki.
Dniá onego obejrzy śię cżłowiek ná ſtworzyćielá ſwego / á ocży jego ná Świętego Izráelſkiego poglądáć będą ;
A nie obejrzy śię ná ołtárze / ſpráwę rąk ſwojich / áni ná to / co ucżyniły pálce jego / pátrzyć będźie / áni ná gáje poświęcone / áni ná obrázy ſłonecżne.
Dniá onego miáſtá mocy jego będą opuƺcżone / jáko chrośćinká y róƺcżká / które opuƺcżone będą od ſynów Izráelſkich / y będźieƺ źiemią ſpuſtoƺoną.
Boś zápomniáłá ná Bogá zbáwieniá ſwego / y ná ſkáłę mocy twojey nie wſpomináłáś. Przetoż choć ƺcżepy rozkoƺne ƺcżepiƺ / y látorośli winne obce ſádźiƺ ;
Cżáſu ƺcżepieniá twego ƺcżepy áby roſły / opátrujeƺ ; náwet tegoż poránku / co śiejeƺ / áby śię puśćiło / ſtáráƺ śię : wƺákże w dźień pożytku gromádno boleść y rozpácż żąć będźieƺ.
Biádá zgrái ludu wielkiego / którzy hucżą jáko ƺum morſki / y zgiełkowi narodów / które ƺumią jáko ƺum wód gwáłtownych ;
Národom / które ƺumią jáko ƺum wód wielkich ; bo je Pán ſfuká / y ućiekną dáleko / y gonione będą od wiátru jáko plewy po wierzchách gór / á jáko wiecheć od wichru.
Bo cżáſu wiecżornego náſtąpi trwogá / á niż poránek przydźie / álić go niemáƺ. Tenći jeſt dźiał tych / którzy nas puſtoƺą / y los tych / którzy nas plundrują.
