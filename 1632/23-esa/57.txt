Spráwiedliwy ginie / á nikt tego do ſercá nieprzypuƺcżá ; y mężowie pobożni ſchodzą / á nikt tego nie uwáżá / że przed przyjśćiem złego ſpráwiedliwy zebrány bywa ;
Że wchodźi do pokoju / á odpocżywá ná łożu ſwojim / ktokolwiek chodźi w uprzejmośći.
Ale wy ſámi przyſtąpćie / ſynowie cżárownicy / naśienie cudzołożniká y wƺetecżnicy!
Nád kimże śię ćieƺyćie? przećiwko komuż rozdźieráćie gębę / y wywieƺáćie język? Izali nie jeſteśćie ſynowie nierządu / naśienie kłámliwe?
Którzy nierząd płodźićie w gájách pod káżdym drzewem źielonem / zábijájąc ſynów ſwych przy potokách / pod wyſokiemi ſkáłámi.
Miedzy głádkim kámieniem potokowym jeſt dźiał twój. Cić ſą / ći loſem twoim / ná które też wylewáƺ ofiárę mokrą / á ofiárujeƺ ofiárę śniedną / y w temże bym śię Ja kochał?
Ná górze wyſokiej y wynioſłej poſtáwiłáś łoże twoje / á tám wſtępujeƺ ku ſpráwowániu ofiár.
A zá drzwiámi y zá podwojim położyłáś pámiątkę twoję / gdyż odemnie odchodząc odkrywáƺ śię / á wſtąpiwƺy rozƺerzáƺ łoże ſwe / cżyniąc je przeſtworniejƺe / niżeli pogánie ; umiłowáłáś łoże ich / gdźiekolwiek miejſce upátrzyƺ.
Chodźiƺ y do królá / z olejkiem y z rozmáitemi wonnemi máśćiámi twemi ; poſyłáƺ bowiem poſłów ſwych dáleko / á poniżáƺ śię áż do grobu.
Mnóſtwem dróg ſwojich ſprácowáłáś śię / á nie mówiƺ : Dáremnáć to. Ználázłáś pomoc ręce ſwojey / dlátegoś nie zemdláłá.
Kogożeś śię obáwiáłá y lękáłá / iżeś kłámáłá? Ná mięś nie pomniáłá / ániś tego przypuśćiłá do ſercá ſwego : dlátegoż to / żem Ja milcżał / á to z dáwná / nie boiƺ śię mnie?
Ja opowiem ſpráwiedliwość twoję y ſpráwy twoje / któreć nic nie pomogą.
Gdy záwołáƺ / niech ćię wybáwi zgrájá twojá ; ále wƺyſtkie one roznieśie wiátr / y pochwyći márność. Lecż ten / co we mnie ufá / odźiedźicży źiemię / á pośiądźie górę świętą moję.
Bo rzeką : Wyrównájćie / wyrównájćie / zgotujćie drogę / uprzątnijćie záwády z drogi ludu mojego.
Bo ták mówi on nájdoſtojniejƺy y nájwyżƺy / który mieƺká w wiecżnośći / á święte jeſt imię jego : Ja / który mieƺkám ná wyſokośći ná miejſcu świętym / mieƺkám y z tym / który jeſt ſkruƺonego y uniżonego duchá / ożywiájąc duchá pokornych / ożywiájąc ſerce ſkruƺonych.
Nie będę śię zájiſte ná wieki wádźił / áni śię wiecżnie gniewał ; boćby duch przed oblicżem mojim zemdlał / y duƺe / którem Ja ucżynił.
Dla niepráwośći łákomſtwá jego rozgniewáłem śię / á uderzyłem go ; ukryłem śię / á rozgniewáłem śię / przeto / że odpornym będąc / poƺedł drogą ſercá ſwego.
Widzę drogi jego / wƺákże uzdrowię go ; doprowádzę go / y przywrócę mu poćiechy / y tym / którzy z nim płácżą.
Stworzę owoc wárg / pokój dálekiemu y bliſkiemu / mówi Pán ; á ták uzdrowię go.
Lecż niepobożni będą jáko morze wzburzone / gdy śię uſpokoić nie może / á którego wody wymiátáją kał y błoto.
Niemáƺ pokoju niepobożnym / mówi Bóg mój.
