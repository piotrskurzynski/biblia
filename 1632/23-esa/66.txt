Ták mówi Pán : Niebo jeſt ſtolicą moją / á źiemiá podnóżkiem nóg mojich. Gdźież tedy będźie ten dom / który mi zbudujećie? álbo gdźie będźie miejſce odpocżnieniá mego?
Bo to wƺyſtko ręká mojá ucżyniłá / y nią ſtánęło to wƺyſtko / mówi Pán. Wƺákże Ja ná tego pátrzę / który jeſt utrápionego y ſkruƺonego duchá / á który drży ná ſłowo moje.
Inácżey ten / kto zábijá wołu ná ofiárę / jákoby zábił cżłowieká ; kto zábijá ná ofiárę bydlątko / jákoby pſá śćiął ; kto ofiáruje ofiárę ſuchą / jákoby krew świnią ofiárował ; kto kádźi kádźidłem / jákoby báłwánowi błogoſłáwił. A jáko oni ſobie obráli drogi ſwoje / y w obrzydliwośćiách ſwojich kocháłá śię duƺá ich ;
Ták y Ja obiorę zá wynálázki ich / á to / cżego śię boją / przywiodę ná nich / przeto / że gdym wołał / żáden śię nie ozwał / gdym mówił / nie ſłucháli / ále cżynili to / co złego jeſt przed ocżymá mojemi / á to / cżegom nie chćiał / obieráli.
Słuchájćie ſłowá Páńſkiego / wy którzy drżyćie ná ſłowo jego. Bráćia wáśi nienáwidzący was / á wygániájący was dla imienia mego / mówią : Niech śię okaże ſłáwá Páńſka. Okáżeć śię zájiſte ku poćieƺe wáƺej ; ále oni poháńbieni będą.
Głos grzmotu z miáſtá ſłyƺány będźie / głos z kośćiołá / głos Páná oddáwájącego zápłátę nieprzyjáćiołom ſwoim.
Pierwej niż prácowáłá ku porodzeniu / porodźiłá / pierwej niż ją ogárnęłá boleść / porodźiłá mężcżyznę.
Któż ſłyƺał co tákowego? Kto widźiał co podobnego? Możeż to być / áby źiemiá národźiłá ludu zá jeden dźień? Izali naród ſpłodzony bywa jednym rázem? Ale Syon ledwie pocżął prácowáć ku porodzeniu / álić porodźił ſynów ſwych.
Cóżbym Ja / który otwierám żywot / rodźić nie miał? mówi Pán. Cóżbym Ja / który to cżynię / że rodzą / záwártym był? mówi Bóg twój.
Weſelćie śię z Jeruzálemem á rádujćie śię w nim wƺyſcy / którzy go miłujećie. Weſelćie śię z nim wielce / wƺyſcy którzyśćiekolwiek płákáli nád nim.
Przeto / że ſſáć będźiećie / y ſyćić śię pierśiámi poćiech jego / ſſáć będźiećie / y rozkoƺámi opływáć w jáſnośći chwały jego.
Bo ták mówi Pán : Oto Ja obrócę ná nich pokój jáko rzekę / á ſłáwę narodów jáko ſtrumień zálewájący / y będźiećie ſſáć ; ná ręku noƺeni / y ná kolánách rozkoƺnie piáſtowáni będźiećie.
Jáko ten / którego ćieƺy mátká jego / ták Ja was ćieƺyć będę ; á ták w Jeruzálemie ućiechy miewáć będźiećie.
Ujrzyćie zájiſte / á rádowáć śię będźie ſerce wáƺe / á kośći wáƺe jáko tráwá zákwitną. Y poznáná będźie ręká Páńſka przy ſługách jego ; ále śię gniewem zápáli przećiwko nieprzyjáćiołom ſwoim.
Bo oto Pán w ogniu przydźie / á pocżwórne jego jáko wicher / áby wylał gniew ſwój w popędliwośći / á łájánie ſwoje w płomieniu ogniá.
Pán / mówię / przez ogień ſądźić będźie / y przez miecż ſwój wƺelkie ćiáło / á pobitych od Páná wiele będźie.
Y ći / którzy śię poświęcáją y ocżyƺcżáją w ogrodách / jeden zá drugim jáwnie ; którzy jedzą mięſo świnie / y inną obrzydłość / y myƺy / koniec tákże wezmą / mówi Pán.
Abowiem Ja znam ſpráwy ich / y myśli ich ; y przydźie ten cżás / że zgromádzę wƺyſtkie narody / y języki / y przyjdą á oglądáją chwałę moję.
Y położę ná nich znák / á poślę z tych / którzy záchowáni będą / do narodów przy morzu do Pul y Lud / którzy ćiągną łuk do Tubálá / y do Jáwánu / ná wyſpy dálekie / które nic o mnie nie ſłyƺáły / y nie widźiáły chwały mojey ; y będą opowiádáły chwałę moję miedzy národámi.
Y przywiodę wƺyſtkich bráći wáƺych ze wƺyſtkich narodów Pánu w dáry / ná koniách y ná wozách / y ná lektykách / y ná mułách / y ná záwodnikách / ná górę świętobliwośći mojey do Jeruzálemu / mówi Pán / ták jáko przynoƺą ſynowie Izráelſcy dar w nácżyniu cżyſtem do domu Páńſkiego.
Y z tych też nábiorę kápłanów y Lewitów / mówi Pán.
Bo jáko te niebioſá nowe / y tá źiemiá nowá / którą Ja ucżynię / ſtánie przedemną / mówi Pán / ták ſtánie naśienie wáƺe y imię wáƺe.
Y ſtánie śię / że od nowiu mieśiącá do nowiu mieśiącá / y od ſábátu do ſábátu przychodźić będźie wƺelkie ćiáło / áby śię kłániáło przed oblicżnośćią moją / mówi Pán.
Y wynijdą á oglądáją trupy ludźi tych / którzy wyſtąpili przećiwko mnie ; ábowiem robák ich nie zdechnie / á ogień ich nie zágáśnie / á będą obrzydliwośćią wƺelkiemu ćiáłu.
