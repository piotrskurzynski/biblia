Widzenie Izájaƺá Syná Amoſowego / które widźiał nád Judą y <i>nád</i> Jeruzalem : zá dni Ozeaƺá / Joátámá / Acházá / <i>y</i> Ezechiaƺá królów Judſkich.
Słuchajćie niebioſá / á ty źiemio przyjmi w uƺy <i>ſwe</i> : ábowiem PAn mówi : Synym wychował y wywyżƺył / ále oni odſtąpili odemnie.
Zna wół goſpodarzá ſwego / y ośieł żłób PAná ſwego : <i>Ale</i> Izráel <i>mię</i> nie zna / lud mój nie zrozumiewa.
Biádá narodowi grzeƺnemu / ludowi obćiążonemu niepráwośćią / náśieniu złośliwych / Synom ſkáżonym! Opuśćili PAná / do gniewu pobudźili świętego Izráelſkiego / odwróćili śię ná zad.
Przecżże tym więcey przycżyniaćie przeſtępſtwá / im więcey was biją? Wƺyſtká głowá chora / y wƺyſtko ſerce mdłe.
Od ſtopy nogi áż do <i>wierzchu</i> głowy / niemáƺ ná nim nic cáłego : ráná y śiność / y rázy zágniłe / nie <i>ſą</i> wyćiśnione / áni záwiązáne / áni olejkiem odmiękcżone.
Ziemiá wáƺá zpuſtoƺona : miáſtá wáƺe popalone ogniem. Ziemię wáƺę cudzoźiemcy przed wámi pożyráją y puſtoƺą : jáko zwykli cudzoźiemcy.
Y zoſtáłá córká Syońſka / jáko chłodnik ná winnicy : jáko budká w ogrodźie ogórcżánym / <i>y</i> jáko miáſto zburzone.
By nam był PAn Zaſtępów nie zoſtáwił trochy oſtátków : bylibyſmy jáko Sodomá / ſtálibyſmy śię byli Gomorze podobnymi.
Słuchajćie ſłowá PAńſkiego Kśiążętá Sodomſkie : przyjmujćie w uƺy Zakon Bogá náƺego / ludźie Gomorſcy.
Cóż mi po mnóſtwie ofiar wáƺych? mówi PAn. Jużem ſyty cáłopalenia báránów / y łoju tłuſtego bydłá : á krwie ćielców / y báránków / y kozłów nieprágnę.
Gdy przychodźićie ábyśćie śię okázowáli przed twarzą moją : któż tego żądał z rąk wáƺych / ábyśćie deptáli śieni moje?
Nie ofiárujćież więcey ofiáry dáremney. Kádzenie jeſt mi obrzydłośćią : Nowiu mieśiącá y Sábbátu / gdy zwoływaćie zgromádzenia / niemogę <i>zćierpieć,</i> ( Bo niepráwośćią jeſt ) áni dniá zápowiedźiánego.
Nowych mieśiąców wáƺych / y urocżyſtych świąt wáƺych nienawidźi duƺá mojá : ſtáły mi śię ćiężarem : uprácowałem śię noƺąc je.
Przetoż gdy wyćiągniećie ręce wáƺe / ſkryję ocży moje przed wámi : á gdy rozmnożyćie modlitwę / nie wyſłucham : <i>bo</i> ręce wáƺe krwie ſą pełne.
Omyjćie śię / cżyſtymi bądźćie / odejmićie złość ucżynków wáƺych od ocżu mojich : przeſtańćie źle cżynić.
Ucżćie śię dobrze cżynić : Szukajćie ſądu / podźwignićie ućiśnionego / Sąd cżyńćie śieroćie / ujmujćie śię <i>o krzywdę</i> wdowy.
Przydźćież teraz / á rozpierajmy śię z ſobą / mówi PAN : Choćby były grzechy wáƺe jáko ƺárłat / jáko śnieg zbieleją : Choćby były cżerwone jáko kármázyn / jáko wełná <i>białe</i> będą.
Będźiećieli powolni / á poſłuchaćie <i>mię</i> ; dóbr źiemie pożywáć będźiećie.
Lecż jeſli nie będźiećie poſłuƺnymi ále odpornymi ; od miecżá pożárći będźiećie : bo uſtá PAńſkie mówiły.
Jákoć śię ſtáło nierządnicą to miáſto wierne / pełne ſądu : ſpráwiedliwość mieƺkáłá w nim / lecż teraz mężobójcy.
Srebro twoje obróćiło śię w zużel : wino twoje pomięƺáło śię z wodą.
Kśiążętá twoje <i>ſą</i> uporne / y towárzyƺe złodźiejów : káżdy z nich miłuje dáry á idą zá nadgrodą : śieroćie nie cżynią ſpráwiedliwośći / á ſpráwá wdowy nieprzychodźi przed nie.
Przetoż mówi PAN / PAn Zaſtępów / możny Izráelſki : Oto / ućieƺę śię nád nieprzyjaćioły mojimi / á pomƺcżę śię nád przećiwnikámi ſwymi.
Y obrócę rękę moję ná ćię / á wypalę áż do cżyſtá zużelicę twoję / y odpędzę wƺyſtkę cenę twoję.
A przywrócę ſędźie twoje / jáko przed tym <i>byli</i> : y rajce twoje jáko ná pocżątku. Potym ćię názywáć będą / miáſtem ſpráwiedliwośći / miáſtem wiernym.
Syon w ſądźie odkupione będźie : á ći co śię do niego náwrócą / w ſpráwiedliwośći.
Ale przewrotnicy y grzeƺnicy / weſpół ſtárći będą : á ći co opuśćili PAná / zniƺcżeją.
Abowiem záwſtydzeni będźiećie dla gájów / którycheśćie pożądáli : y poháńbieni dla ogrodów / któreśćie <i>ſobie</i> obráli.
Gdy śię ſtániećie jáko dąb / z którego liśćie opádło : á jáko ogród / w którym wody niemáƺ.
Y będźie mocarz jáko zgrzebi : á ten który go ucżynił / jáko iſkrá : y zápali śię oboje poſpołu / á nie będźie ktoby zágáśił.
