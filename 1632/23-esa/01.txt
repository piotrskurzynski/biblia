Widzenie Izájáƺá / ſyná Amoſowego / które widźiał nád Judą y nád Jeruzálemem / zá dni Ozeáƺá / Joátámá / Acházá / y Ezechijáƺá / królów Judzkich.
Słuchájćie niebioſá / á ty źiemio przyjmij w uƺy ſwe! Abowiem Pán mówi : Synówem wychował y wywyżƺył ; ále oni odſtąpili odemnie.
Zná wół goſpodárzá ſwego / y ośieł żłób páná ſwego ; ále Izráel mię nie zná / lud mój nie zrozumiewá.
Biádá národowi grzeƺnemu / ludowi obćiążonemu niepráwośćią / náśieniu złośliwych ; ſynom ſkáżonym! Opuśćili Páná / do gniewu pobudźili świętego Izráelſkiego / odwróćili śię názád.
Przecżże tym więcey przycżyniáćie przeſtępſtwá / im więcey was biją? Wƺyſtká głowá chorá / y wƺyſtko ſerce mdłe.
Od ſtopy nogi áż do wierzchu głowy niemáƺ ná nim nic cáłego ; ráná y śiność / y rány zágniłe nie ſą wyćiśnione / áni záwiązáne / áni olejkiem odmiękcżone.
Ziemiá wáƺá ſpuſtoƺoná / miáſtá wáƺe popálone ogniem. Ziemię wáƺę cudzoźiemcy przed wámi pożeráją y puſtoƺą / jáko zwykli cudzoźiemcy.
Y zoſtáłá córká Syońſká jáko chłodnik ná winnicy / jáko budká w ogrodźie ogórcżánym / y jáko miáſto zburzone.
By nam był Pán zaſtępów nie zoſtáwił trochy oſtátków / bylibyſmy jáko Sodomá / ſtálibyſmy śię byli Gomorze podobnymi.
Słuchájćie ſłowá Páńſkiego / kśiążętá Sodemſcy! przyjmujćie w uƺy zakon Bogá náƺego / ludźie Gomorſcy!
Cóż mi po mnóſtwie ofiár wáƺych? mówi Pán. Jużem ſyty cáłopalenia báránów / y łoju tłuſtego bydłá ; á krwi ćielców / y báránków / y kozłów nie prágnę.
Gdy przychodźićie / ábyśćie śię okázywáli przed twárzą moją / któż tego żądał z rąk wáƺych / ábyśćie deptáli śieni moje?
Nie ofiárujćież więcey ofiáry dáremney. Kádzenie jeſt mi obrzydłośćią ; nowiu mieśiącá y ſábátu / gdy zwoływáćie zgromádzenia / nie mogę śćierpieć ( bo niepráwośćią jeſt ) áni dniá zápowiedźiánego.
Nowych mieśięcy wáƺych / y urocżyſtych świąt wáƺych nienáwidźi duƺá mojá ; ſtáły mi śię ćiężárem ; uprácowáłem śię noƺąc je.
Przetoż gdy wyćiągniećie ręce wáƺe / ſkryję ocży moje przed wámi ; á gdy rozmnożyćie modlitwę / nie wyſłuchám ; bo ręce wáƺe krwi ſą pełne.
Omyjćie śię / cżyſtymi bądźćie / odejmijćie złość ucżynków wáƺych od ocżów mojich ; przeſtáńćie źle cżynić.
Ucżćie śię dobrze cżynić ; ƺukájćie ſądu / podźwignijćie ućiśnionego / ſąd cżyńćie śieroćie / ujmujćie śię o krzywdę wdowy.
Przyjdźćież teraz / á rozpierájmy śię z ſobą / mówi Pán : Choćby były grzechy wáƺe jáko ƺárłát / jáko śnieg zbieleją ; choćby były cżerwone jáko kármázyn / jáko wełná biáłe będą.
Będźiećieli powolni / á poſłucháćie mię / dóbr źiemi pożywáć będźiećie.
Lecż jeſli nie będźiećie poſłuƺnymi / ále odpornymi / od miecżá pożárći będźiećie ; bo uſtá Páńſkie mówiły.
Jákoć śię ſtáło nierządnicą to miáſto wierne / pełne ſądu? Spráwiedliwość mieƺkáłá w nim ; lecż teraz mężobójcy.
Srebro twoje obróćiło śię w żużeł ; wino twoje pomięƺáło śię z wodą.
Kśiążętá twoi ſą uporni / y towárzyƺe złodźiei ; káżdy z nich miłuje dáry / á jádą zá nágrodą ; śieroćie nie cżynią ſpráwiedliwośći / á ſpráwá wdowy nie przychodźi przed nich.
Przetoż mówi Pán / Pán zaſtępów / możny Izráelſki : Oto ućieƺę śię nád nieprzyjáćiółmi moimi / á pomƺcżę śię nád przećiwnikámi ſwymi.
Y obrócę rękę moję ná ćię / á wypálę áż do cżyſtá zużelicę twoję / y odpędzę wƺyſtkę cenę twoję.
A przywrócę ſędźiów twojich / jáko przed tym byli / y rádców twojich / jáko ná pocżątku. Potem ćię názywáć będą miáſtem ſpráwiedliwośći / miáſtem wiernem.
Syon w ſądźie okupione będźie / á ći / co śię do niego náwrócą / w ſpráwiedliwośći.
Ale przewrotnicy y grzeƺnicy weſpół ſtárći będą / á ći / co opuśćili Páná / zniƺcżeją.
Abowiem záwſtydzeni będźiećie dla gájów / którycheśćie pożądáli ; y poháńbieni dla ogrodów / któreśćie ſobie obráli.
Gdy śię ſtániećie jáko dąb / z którego liśćie opádły / á jáko ogród / w którym wody niemáƺ.
Y będźie mocárz jáko zgrzebiá / á ten / który go ucżynił / jáko iſkrá ; y zápálą śię oboje poſpołu / á nie będźie / ktoby zágáśił.
