Abowiem oto pánujący PAN Zaſtępów / odejmie od Jeruzalemu y od Judy laſkę / y podporę : wƺeláką podporę chlebá / y wƺeláką podporę wody.
Mocarzá y mężá walecżnego ; y Sędźiego / y Proroká / y mędrcá / y ſtárcá.
Rotmiſtrzá nád piąćiądźieśiąt / á mężá poważnego : y rajcę / y mądrego rzemieśniká / y kráſomówcę.
Y dam jm dźieći zá Kśiążętá : dźieći mówię pánowáć będą nád nimi.
Y będźie ućiſkał miedzy ludem jeden drugiego / y bliźni bliźniego ſwego : powſtánie dźiećię przećiwko ſtárcowi / á podły przećiwko zacnemu.
A gdy śię uchwyći káżdy brátá ſwego z domu Ojcá ſwego / y <i>rzecże</i> : Maƺ odźienie / bądźże Kśiążęćiem náƺym / á upadek ten zátrzymaj ręką ſwą :
<i>Tedy</i> on przyśięże dniá onego / mówiąc : Nie będę záwięzował <i>tych ran</i> : ábowiem w domu mojim niemáƺ chlebá / áni odźienia. Nie ſtánowćież mię Kśiążęćiem nád ludem.
Bo Jeruzalem upada / á Judá śię wáli ; dla tego / że język ich / y ſpráwy ich ſą przećiwko PAnu : pobudzájąc do gniewu ocży Májeſtatu jego.
Poſtáwá oblicża ich świadcży przećiwko nim / grzech ſwój / jáko Sodomcżycy / opowiádáją / á nie táją <i>go</i>. Biádá duƺy ich / ábowiem ſámi ná śię złe przywodzą.
Powiedzćie ſpráwiedliwemu / że mu dobrze będźie : bo owocu ucżynków ſwojich pożywáć będźie.
<i>Ale</i> biádá niepobożnemu / źle <i>mu</i> będźie : ábowiem odpłátá rąk jego dáná mu będźie.
Kśiążętá ludu mego ſą dźiećmi / á niewiáſty pánują nád nim. O ludu mój! ći którzy ćię wodzą / zwodzą ćię / á drogę śćieżek twojich ukrywáją.
Powſtał PAn áby ſądźił : ſtoji / áby ſądźił lud.
PAn przydźie ná ſąd przećiwko ſtárƺym ludu ſwego / y przećiwko Kśiążętom ich. A <i>rzecże</i> ; Wyśćie ſpuſtoƺyli winnicę <i>moję</i> : źdźierſtwo z ubogiego jeſt w domách wáƺych.
Przecżże trzećie lud mój / á oblicża ubogich bijećie? mówi PAn / PAn Zaſtępów.
Y rzekł PAN : Yż śię wynoƺą córki Syońſkie / á chodzą ƺyje wyćiągnąwƺy / y mrugájąc ocżymá przechodzą śię : á drobno poſtępując / nogámi ſwymi ƺeleſt cżynią :
Przetoż obłyśi PAn wierzch głowy Córek Syońſkich : á PAn ſromotę jich obnáży.
Dniá onego odejmie PAn ochędóſtwo podwiązek : tákże cżepce y záwieƺenia.
Piżmowe jábłká / y mánelle / y zatycżki.
Bieretki / y zapony / y bindy / y przednicżki / y nauƺnice.
Pierśćionki / y nacżelniki.
Odmienne ƺáty / y płaƺcżyki / y podwiki / y wácki.
Zwierćiádłá / y rántuƺki / y tkanki / y letniki.
Y będźie miáſto wonnych rzecży ſmród : á miáſto páſá / rozpaſánie : á miáſto utráfionych włoſów / łyśiná : á miáſto ƺerokiey ƺáty / opaſánie worem : á miáſto pięknośći / ogorzeliná.
Mężowie twoji od miecżá upádną / á mocarze twoji w bitwie.
Y záſmucą śię / á płákáć będą bramy jego : á ſpuſtoƺony / ná źiemi śiedźieć będźie.
