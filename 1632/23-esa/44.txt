A teraz ſłucháj Jákóbie ſługo mój! y ty / Izráelu! któregom wybrał.
Ták mówi Pán / który ćię ucżynił / y który ćię ukƺtáłtował záráz z żywotá mátki / y który ćię wſpomágá : Nie bój śię Jákóbie / ſługo mój! y uprzejmy / któregom wybrał.
Bo wyleję wody ná prágnącego / á potoki ná ſuchą źiemię ; wyleję Duchá mego ná naśienie twoje / y błogoſłáwieńſtwo moje ná potomki twoje.
Y rozkrzewią śię jáko miedzy tráwą / y jáko wierzby przy ćiekących wodách.
Ten rzecże : Jam jeſt Páńſki / á ów śię ozowie do imienia Jákóbowego / á inny śię zápiƺe ręką ſwą Pánu / y imieniem Izráelſkiem będźie śię názywał.
Ták mówi Pán / król Izráelſki / y odkupićiel jego / Pán zaſtępów : Jam pierwƺy / y jam oſtátni ; á oprócż mnie niemáƺ Bogá.
Bo któż jáko Ja ogłáƺá y opowiádá to / y ſporządzá mi to / záráz od onego cżáſu / jákom rozſądźił lud ná świećie? á kto przyƺłe rzecży / y to / co ma być / oznájmi im?
Nie bójćież śię / áni ſobą trwożćie. Izálim wam tego z dáwná nie oznájmił / y nie opowiedźiał? Tegośćie wy mnie ſámi świádkámi. Izali jeſt Bóg oprócż mnie? Niemáƺ zájiſte ſkáły ; Ja o żádney nie wiem.
Tworzyćiele báłwánów wƺyſcy nic nie ſą / y te nájmilƺe rzecży ich nic im nie pomogą ; cżego oni ſobie ſámi świádkámi będąc / nic nie widzą / áni rozumieją / żeby śię wſtydźić mogli.
Kto tworzy bogá / y báłwáná leje / do nicżego śię to nie przygodźi.
Oto wƺyſcy / y ucżeſnicy ich będą poháńbieni ; owƺem / rzemieślnicy ich / ći nád innych ludźi / choćby śię wƺyſcy zebráli y ſtánęli / lękáć śię muƺą / y ſpołem poháńbieni będą.
Kowál kleƺcżámi robi przy węglu / á młotámi kƺtáłtuje báłwáná ; gdy go robi mocą rámieniá ſwego / áż od głodu w nim y śiły uſtáją / áni pije wody / áż y omdlewá.
Cieślá záś rozćiąga ƺnur / znácży ƺnurem fárbowánym / y oćioſuje toporem / y cyrklem rozmierzá go / y cżyni go ná podobieńſtwo mężá / y ná podobieńſtwo pięknego cżłowieká / áby mieƺkał w domu.
Nárąbie ſobie cedrów / y bierze cyprys y dąb / álbo to / co jeſt nájmocniejƺego miedzy drzewem leśnem / álbo wſádźi jáwór / który zá deƺcżem odráſtá ;
Y używá tego cżłowiek do páleniá / álbo wźiąwƺy z niego / ogrzewá śię przy nim / tákże rozniecá ogień / áby nápiekł chlebá / nádto z tegoż drzewá robi ſobie bogá / y kłánia mu śię ; cżyni z niego báłwáná / y klęká przed nim.
Cżęść jego páli ogniem / przy drugiej cżęśći jego mięſo je / piecże piecżeń y náſycony bywa ; tákże rozgrzewá śię / y mówi : Ehej! rozgrzáłem śię / widźiałem ogień.
A z oſtátku jego cżyni bogá / báłwáná ſwego ; klęká przed nim / kłánia śię / y modli mu śię / mówiąc : Wybáw mię / boś ty bóg mój.
Nie wiedzą / áni rozumieją / przeto / że Bóg záślepił ocży ich / áby nie widźieli / y ſercá ich / áby nie rozumieli.
Y nie uwáżáją tego w ſercu ſwojim / nie májąż to umiejętnośći áni bácżeniá / áby rzekli : Cżęść z niego ſpáliłem ogniem / á przy węglu jego nápiekłem chlebá / upiekłem mięſo / y nájádłem śię ; y mámże ja z oſtátku jego obrzydliwość ucżynić / á przed klocem drewniánym klękáć?
Táki śię kármi popiołem / ſerce jego zwiedźione unośi go / áby nie mógł wybáwić duƺy ſwojey / áni rzec : Izali to nie jeſt oƺukánie / co jeſt w práwicy mojey?
Pomnijże ná to / Jákóbie y Izráelu! przeto / żeś ty jeſt ſługą moim. Stworzyłem ćię / ſługáś ty mój ; o Izráelu! nie zápomnę ná ćię.
Głádzę niepráwośći twoje jáko obłok / á grzechy twoje jáko mgłę ; náwróć śię do mnie / bom ćię odkupił.
Śpiewájćie niebioſá / bo to Pán ucżynił ; wykrzykájćie niſkośći źiemi / zábrzmijćie chwałę góry / lás / y wƺyſtkie drzewá w nim ; ábowiem Pán odkupił Jákóbá / á w Izráelu ſłáwnym śię ucżynił.
Ták mówi Pán / odkupićiel twój / y który ćię utworzył wnet z żywotá mátki : Ja Pán wƺyſtko cżynię / ſám rozćiągám niebioſá / rozpośćierám źiemię mocą ſwoją.
Wniwecż obrácám známioná práktykárzów / y wieƺcżków do ƺáleńſtwá przywodzę ; y mędrców ná wſtecż obrácám / á umiejętność ich głupią cżynię.
Potwierdzám ſłowá ſługi ſwego / á rádę poſłów ſwych wykonywám. Który mówię o Jeruzálemie : Mieƺkáć w nim będę ; á o miáſtách Judzkich : Pobudowáne będą ; bo ſpuſtoƺeniá ich pobuduję ;
Który mówię głębinie : Wyſchnij / Ja potoki twe wyſuƺę ;
Który mówię o Cyruśie : On páſterz mój / bo wƺyſtkę wolę moję wykoná ; y rzecże Jeruzálemowi : Będźieƺ zbudowáne ; á kośćiołowi : Będźieƺ záłożony.
