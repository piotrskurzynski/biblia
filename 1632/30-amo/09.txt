Widźiáłem Páná ſtojącego ná ołtárzu / który rzekł : Uderz w gáłkę / áż zádrżą podwoje / á rozetnij je wƺyſtkie od wierzchu ich / á oſtátek miecżem pobiję ; żáden z nich nie ućiecże / y nie będźie z nich nikt / coby tego uƺedł.
Choćby śię zákopáli w źiemię / y ſtámtądby ich ręká mojá wźięłá ; choćby wſtąpili áż do niebá / y ſtámtądbym ich ſtárgnął.
A choćby śię ſkryli ná wierzchu Kármelu / wyƺpieguję y wezmę ich ſtámtąd ; á choćby śię ſkryli przed ocżymá mojemi ná dnie morſkiem / przykáżę wężowi / áby ich y ſtámtąd wykąſał ;
A choćby poƺli w niewolę przed nieprzyjáćiółmi ſwymi / y tám przykáżę miecżowi / áby ich pomordował ; obrócę záiſte przećiwko nim oko ſwe ná złe / á nie ná dobre.
Bo pánujący Pán zaſtępów / gdy śię dotknie źiemi / rozpływá śię / á płácżą wƺyſcy mieƺkájący ná niej / y wzbierá wƺyſtká jáko rzeká / á zátopioná bywá jáko rzeką Egipſką.
Który ná niebieśiech zbudował páłáce ſwoje / á záſtęp ſwój ná źiemi uƺykował ; który może záwołáć wody morſkie / á wyláć je ná oblicże źiemi ; Pán jeſt imię jego.
Izali nie jeſteśćie podobni ſynom Murzyńſkim przedemną / o ſynowie Izráelſcy? mówi Pán ; izálim Izráelá nie wywiódł z źiemi Egipſkiej jáko Filiſtyńcżyków z Káftor / y Syryjcżyków z Kir?
Oto ocży pánującego Páná przećiwko temu króleſtwu grzeƺącemu / ábym je wygłádźił z oblicżá źiemi ; wƺákże nie wygłádzę do ƺcżętu domu Jákóbowego / mówi Pán.
Bom oto Ja rozkázał / á rozmiecę miedzy wƺyſtkie národy dom Izráelſki jáko miotáná bywá pƺenicá ná przetáku / ták / iż nieprzepádnie y kámyk ná źiemię.
Wƺyſcy grzeƺnicy z ludu mojego od miecżá pomrą / którzy mówią : Nie przybliży śię do nas / áni nas záchwyći to złe.
Dniá onego wyſtáwię upádły przybytek Dawidowy / á zágrodzę rozerwánie jego / y obáliny jego nápráwię / á pobuduję go / jáko zá dni dáwnych ;
Aby pośiedli oſtátki Edomcżyków y wƺyſtkie národy / nád którymi wzywano imieniá mojego / mówi Pán / który to cżyni.
Oto dni idą / mówi Pán / że orácż żeńcę zájmie / á ten / co tłocży winne jágody / rozśiewájącego náśienie ; á góry moƺcżem kropić będą / á wƺyſtkie págórki śię rozpłyną.
Y náwrócę záś z więźieniá lud mój Izráelſki / y pobudują miáſtá ſpuſtoƺone / á mieƺkáć w nich będą ; ſądźić też będą winnice / y wino z nich pić będą ; ſádów też náƺcżepią / y owoc ich jeść będą.
A ták ich wƺcżepię w źiemi ich / że nie będą więcey wykorzenieni z źiemi ſwojey / którąm im dał / mówi Pán / Bóg twój.
