Słuchájćie ſłowá tego / o krowy Báſáńſkie! któreśćie ná górách Sámáryi / które ućiſkáćie nędzników á niƺcżyćie ubogich / które mówićie pánom ich : Przynieśćie / ábyſmy piły.
Przyśiągł pánujący Pán przez świętobliwość ſwoję / iż oto dni idą ná was / których nieprzyjáćiel weźmie was ná háki / á potomki wáƺe ná wędy rybáckie / y wyjdźiećie przerwámi / káżda ták jáko ſtoi ;
Y będźiećie rozrzucáć / cokolwiek było w páłácách wáƺych / mówi Pán.
Idźćież do Betel á bądźćie tułácżámi w Gálgál ; rozmnóżćie przeſtępſtwá / á przynośćie ná káżdy poránek ofiáry wáƺe / y trzećiego roku dźieśięćiny wáƺe ;
A páląc ofiárę chwały z kwáƺonych rzecży / obwołájćie ofiáry dobrowolne / y rozgłośćie / ponieważ śię wam ták podobá / o ſynowie Izráelſcy! mówi pánujący Pán.
A choćiáżem Ja wam dał cżyſtość zębów we wƺyſtkich miáſtách wáƺych / to jeſt / niedoſtátek chlebá po wƺyſtkich miejſcách wáƺych / wƺákżeśćie śię nie náwróćili do mnie / mówi Pán.
Jam też záhámował od was deƺcż / gdy jeƺcże były trzy mieśiące do żniwá / á ſpuśćiłem deƺcż ná jedne miáſto / á ná drugiem miáſto nie ſpuśćił ; jedná dźiedźiná byłá deƺcżem odwilżoná / á drugá dźiedźiná / ná którą deƺcż nie pádał / uſchłá.
Y chodźiły dwá y trzy miáſtá do jednego miáſtá / áby piły wodę / á nie mogły śię nápić ; á wƺákżeśćie śię nie náwróćili do mnie / mówi Pán.
Uderzyłem was ſuƺą y rdzą ; obfitość / którą przynośiły ogrody wáƺe / y winnice wáƺe / y figowe ſády wáƺe / y oliwnice wáƺe / gąśienice pożárły / á wƺákżeśćie śię nie náwróćili do mnie / mówi Pán.
Poſłáłem ná was mór / ták jáko ná Egipt / pobiłem miecżem młodźieńców wáƺych / w pojmániem podał konie wáƺe / y ſpráwiłem / że ſmród wojſk wáƺych wyſtępował w nozdrzá wáƺe ; á wƺákżeśćie śię nie náwróćili do mnie / mówi Pán.
Wywróćiłem was / jáko Bóg wywróćił Sodomę y Gomorę / ták / żeśćie byli jáko głowniá wyrwáná z ogniá ; á wƺákżeśćie śię nie náwróćili do mnie / mówi Pán.
Przetoż ták ći ucżynię / o Izráelu! á iż ći ták ucżynić chcę / bądźże gotowym ná zábieżenie Bogu ſwemu / o Izráelu!
Abowiem oto on jeſt / który kƺtáłtuje góry / á tworzy wiátry / y który oznájmuje cżłowiekowi / jáká jeſt myśl jego ; on z ránney zorzy ćiemność cżyni / á depcże wyſokośći źiemi ; Pán Bóg zaſtępów jeſt imię jego.
