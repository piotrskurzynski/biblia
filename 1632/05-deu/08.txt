Wƺyſtkie przykazánia / które ja dźiś przykázuję tobie / chowáć y cżynić będźiećie / ábyśćie żyli / rozmnożeni byli / y weƺli / á ośiedli tę źiemię / o którą przyśiągł PAN Ojcom wáƺym.
Miejże w pámięći / wƺyſtkę drogę / którą ćię prowádźił PAN Bóg twój / przez <i>te</i> cżterdźieśći lat po puƺcży / áby ćię trapił / y doświadcżył / áby wiádomo było co <i>jeſt</i> w ſercu twojim / jeslibyś przeſtrzegał przykazań jego / cżyli nie.
Przetoż dręcżył ćię / y głodemći dokucżał / potym karmił ćię Mánną / któreyś nie znał / áni jey <i>też</i> ználi Ojcowie twoji / áby ćię náucżył / iż nie ſámym chlebem żywie cżłowiek / ále tym wƺyſtkim co pochodźi z uſt PAńſkich / żyć będźie cżłowiek.
Szátá twojá nie wiotƺáłá ná tobie / y nogá twojá nie nápuchłá / <i>przez</i> te cżterdźieśći lat.
Poznajże tedy w ſercu ſwym / że jáko ćwicży cżłowiek Syná ſwego / <i>ták</i> PAN Bóg twój ćwicży ćiebie.
A przeſtrzegaj przykazań PAná Bogá twego / ábyś chodźił drogámi jego / y bał śię go.
Abowiem PAN Bóg twój / prowádźi ćię do źiemie tey dobrey / <i>do</i> źiemie / gdźie ſą potoki wód / źrżódłá y przepáśći wynikájące po dolinách y po górách :
<i>Do</i> źiemie <i>obfitey</i> w pƺenicę y w jęcżmień / y w Wino / y w figi y w jábłká gránatowe / <i>do</i> źiemie hojney w Oliwę y w miód :
<i>Do</i> źiemie / w którey bez niedoſtátku jeść będźieƺ chleb / á ná żadney rzecży zchodźićći nie będźie : do źiemie / którey kámienie <i>ſą</i> żelázo / á z jey gór miedź kopáć będźieƺ.
Gdy będźieƺ jadł / á najeƺ śię / będźieƺ chwalił PAná Bogá twego / zá źiemię wyborną / którą dał tobie.
Strzeżże śię / byś ſnadź nie zápomniał PAná Bogá twego / á nie zániedbał przykazań jego / y ſądów jego / y uſtaw jego / które ja dźiś rozkázuję tobie :
By ſnadź / gdy jeść będźieƺ do ſytośći / y piękne domy pobudujeƺ / á mieƺkáć w nich będźieƺ :
Y gdyć śię wołów y owiec námnoży / śrebrá tákże / y złotá będźieƺ miał doſyć / y wƺyſtko co będźieƺ miał rozmnoży śię :
Nie podnioſło śię ſerce twoje / y zápomniałbyś PAná Bogá twego / który ćię wywiódł z źiemie Egipſkiey / z domu niewoli :
Który ćię przeprowádźił przez tę puƺcżą wielką y ſtráƺną / <i>gdźie byli</i> wężowie jádowići / y niedźwiadkowie / y <i>przez źiemię</i> ſuchą / gdźie niemáƺ wody / y wywiódłći wodę z ſkáły twárdey.
Który ćię karmił Mánną ná puƺcży / którey nie ználi Ojcowie twoji : áby ćię trapił / y doświadcżał ćię / ábyć ná oſtátek dobrze ucżynił.
Ani mów w ſercu ſwym : Moc mojá y śiłá ręki mojey / nábáwiłá mię tych dóbr :
Ale pámiętaj ná PAná Bogá twego : bo on dodawa tobie mocy ku nábywániu bogactw / áby potwirdźił przymierza ſwego / które poprzyśiągł Ojcom twojim / jáko to okázuje dźień dźiśiejƺy.
Ale / jeſliż cále zápomniƺ PAná Bogá twego / á pójdźieƺ zá Bogámi obcymi / y będźieƺ im ſłużył / y kłaniał śię im : oświadcżam śię przećiwko wam dźiś / że koniecżnie zginiećie.
Jáko narody które PAN wytraca przed wámi / ták záginiećie / dla tego iżeśćie poſłuƺnymi nie byli / głoſu PAná Bogá wáƺego.
