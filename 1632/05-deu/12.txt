Te ſą uſtáwy y ſądy / których ſtrzedz będźiećie / ábyśćie je cżynili / w źiemi którą dawa PAN Bóg Ojców twojich tobie / żebyśćie ją dźiedźicżnie trzymáli / po wƺyſtkie dni w których żyć będźiećie ná źiemi.
Zburzyćie do ƺcżętu wƺyſtkie miejſcá / ná których ſłużyli Narodowie / które wy pośiądźiećie / Bogom ſwojim / ná górách wyſokich / y ná págórkách / y pod káżdym drzewem źielonym.
Y porozwalaćie Ołtarze ich / y połamiećie ſłupy ich / gáje też ich <i>poświęcone</i> ſpalićie ogniem / y báłwany Bogów ich porąbiećie / á wygłádźićie imię ich z miejſcá onego.
Nie ucżynićie ták PAnu Bogu wáƺemu :
Ale miejſcá / które obierze PAN Bóg wáƺ / ze wƺyſtkich pokoleń wáƺych / áby tám wyſtáwił imię ſwoje / y mieƺkał ná nim / będźiećie ƺukáć / y do niego śię zchadzáć.
Támże będźiećie przynośić cáłopalenia wáƺe / y Ofiáry wáƺe / y dźieśięćiny wáƺe / y ofiáry rąk wáƺych / y śluby wáƺe / y dobrowolne <i>dáry</i> wáƺe / tákże pierworodztwá krów wáƺych / y owiec wáƺych.
Y tám będźiećie jeść przed PAnem Bogiem wáƺym : y będźiećie śię weſelić / we wƺyſtkim do cżego zćiągniećie ręce wáƺe / wy / y domy wáƺe / w którychći pobłogoſłáwi PAN Bóg twój.
Nie będźiećie cżynić według tego wƺyſtkiego / jáko my tu dźiś cżyniemy / káżdy co mu śię zda dobrego w ocżách jego.
Abowiemeśćie jeƺcże nie przyƺli do odpocżynienia / y do dźiedźictwá / które PAN Bóg twój dawa tobie.
Ale / gdy przeƺedƺy zá Jordan / mieƺkáć będźiećie w źiemi / którą PAN Bóg wáƺ dawa wam dźiedźicżnie ośieść / y da wam odpocżynek / od wƺyſtkich nieprzyjaćiół wáƺych w około / á mieƺkáć będźiećie beſpiecżnie.
Tedy ná miejſce / które obierze ſobie PAN Bóg wáƺ / áby tám mieƺkáło imię jego / tám znośić będźiećie wƺyſtko co ja wam rozkázuję : Cáłopalenia wáƺe / y ofiáry wáƺe / dźieśięćiny wáƺe y ofiáry rąk wáƺych / y wƺyſtko co przedniego jeſt w ślubiech wáƺych / które ślubowáć będźiećie PAnu.
Y weſelić śię będźiećie przed PAnem Bogiem wáƺym / wy / y Synowie wáƺy / y córki wáƺe / y ſłudzy wáƺy / y ſłużebnice wáƺe / y Lewitá który jeſt w bramách wáƺych / ponieważ nie ma dźiału áni dźiedźictwá z wámi.
Strzeżże śię / ábyś nie ofiárował cáłopalenia twego / ná káżdym miejſcu / gdźiebyć śię zdáło.
Ale tylko ná miejſcu / któreby obrał PAN / w którymkolwiek pokoleniu twojim : tám ofiárowáć będźieƺ cáłopalenia twoje / y tám cżynić będźieƺ wƺyſtko co ja rozkázuję tobie.
A wƺákże / jeſli śię upodoba duƺy twojey / zábijeƺ <i>ſobie</i> y będźieƺ jadł mięſo / według błogoſłáwieńſtwá PAná Bogá twego / które da tobie / we wƺyſtkich bramách twojich niecżyſty y cżyſty jeść je będźie / jáko ſárnę álbo jeleniá :
Krwie tylko jeść nie będźiećie / ná źiemię wylejećie ją / jáko wodę.
Nie będźieƺ mógł jeść w bramách twojich dźieśięćiny zboża twego / y Winá twego / y Oliwy twojey / y pierworodztw krów twojich / y owiec twojich / y wƺyſtkich ślubów twych / którebyś ślubował / y dobrowolnych <i>dárów</i> twojich / tákże y Ofiáry ręki twey.
Ale przed PAnem Bogiem twojim / jeść je będźieƺ / ná miejſcu / które obierze PAN Bóg twój / ty y Syn twój / y córká twojá y ſługá twój / y ſłużebnicá twojá / y Lewitá który jeſt w bramách twojich : y będźieƺ śię weſelił przed PAnem Bogiem twojim / we wƺyſtkich rzecżách / do których zćiągnieƺ ręce twoje.
A ſtrzeż śię / ábyś ſnadź nie opuƺcżał Lewity / po wƺyſtkie dni twoje / w źiemi twojey.
Gdy rozƺerzy PAN Bóg twój gránicę twoję / jákoć powiedźiał / y rzekłbyś : Będę jadł mięſo / przeto / że pożąda duƺá twojá jeść mięſá : według wƺyſtkiey żądośći duƺe twojey / będźieƺ jadł mięſo.
A jeſliby dálekie było od ćiebie miejſce / które obierze PAN Bóg twój / áby tám przebywáło imię jego : tedy zábijeƺ z wołów twojich y z owiec twojich / któreć da PAN / jákomći rozkazał / y będźieƺ jadł w bramách twojich / według wƺyſtkiey żądośći duƺe twojey.
Ale jáko jedzą ſárnę y jeleniá / ták je jeść będźieƺ / niecżyſty y cżyſty / zá równo jeść je będą.
Tylko bądź ſtátecżnym / ábyś krwie nie jadał : bo krew jeſt duƺá / przetoż nie będźieƺ jadł duƺe z mięſem jey.
Nie jedzże jey / ná źiemię ją wyley / jáko wodę.
Nie jedz jey / áby śię dobrze dźiało tobie / y Synom twojim po tobie / gdybyś cżynił co dobrego jeſt / przed ocżymá PAńſkimi.
Ale poświęcone rzecży twoje / które będźieƺ miał / y śluby twoje / weźmieƺ y przynieśieƺ ná miejſce / które obierze PAN :
Y będźieƺ ofiárował cáłopalenia twoje / mięſo y krew ná Ołtarzu PAná Bogá twego : ále krew <i>inƺych</i> Ofiar twojich wylána będźie / ná Ołtarzu PAná Bogá twego : mięſo jednák jeść będźieƺ.
Przeſtrzegajże / á ſłuchaj tych wƺyſtkich ſłów / które ja przykázuję tobie / áby dobrze było tobie / y Synom twojim po tobie / áż ná wieki / gdy cżynić będźieƺ / to co dobrego / y práwego jeſt przed ocżymá PAná Bogá twego.
Gdy wytráći PAN Bóg twój / przed oblicżem twojim te Narody / do których ty wnidźieƺ / ábyś je pośiadł / y opánował je / y mieƺkał w źiemi ich.
Strzeżże śię ábyś śię nie uśidlił <i>idąc</i> zá nimi / gdy wytráceni będą przed twarzą twoją : áni śię też pytaj ná Bogi ich / mówiąc : jáko ći Narodowie ſłużyli bogom ſwojim / ták y ja też ucżynię.
Nie ucżyniƺ ták PAnu Bogu twemu : bo wƺyſtko cżym śię brzydźi PAN / y cżego nienawidźi / cżynili Bogom ſwojim : <i>ták,</i> że też y Syny ſwoje / y córki ſwoje palili ogniem / Bogom ſwojim.
Cokolwiek ja wam rozkázuję / tego ſtrzedz będźiećie / ábyśćie cżynili : Nie przydaƺ nic do tego / áni też ujmieƺ z tego.
