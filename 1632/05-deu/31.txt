Przyƺedƺy tedy Mojzeƺ / opowiedźiał te ſłowá wƺyſtkiemu Izráelowi :
Y rzekł do nich : Sto / y dwádźieśćiá lat <i>jeſt</i> mi dźiś / nie mogę więcey wychodźić y wchodźić á też mi PAN powiedźiał : Nie przejdźieƺ zá ten Jordan :
PAN Bóg twój ſam pójdźie przed tobą : On wytráći narody te przed tobą / y pośiędźieƺ je : Jozue / ten pójdźie przed tobą / jáko powiedźiał PAN.
Y ucżyni im PAN / jáko ucżynił Sehonowi / y Ogowi Królom Amorrejſkim / y źiemi ich / które wygłádźił.
A <i>gdy</i> je wam PAN poda / tedy im ucżynićie według káżdego przykazánia / jákom wam przykazał.
Zmacniajćież śię / á mężnie ſobie pocżynajćie / nie bójćie śię / áni śię lękajćie twarzy ich : ábowiem PAN Bóg twój / on idźie z tobą / nie odſtąpi ćię áni ćię opuśći.
Tedy przyzwawƺy Mojzeƺ Jozuego / rzekł do niego przed ocżymá wƺyſtkiego Izráelá : Zmocni śię / á mężnie ſobie pocżynaj / bo ty wnidźieƺ z ludem tym do źiemie / o którą przyśiągł PAN Ojcom ich / że im ją da / á ty im ją w dźiedźictwo podźieliƺ.
A PAN on pójdźie przed tobą / on będźie z tobą : nie odſtąpi ćię / áni ćię opuśći. Nie bójże śię áni śię lękaj.
Tedy nápiſał Mojzeƺ ten zakon y oddał go Kápłanom Synom Lewiego / którzy nośili ſkrzynię Przymierza PAńſkiego / y wƺyſtkim Stárƺym Izráelſkim.
Przytym rozkazał im Mojzeƺ mówiąc : Ná początku <i>káżdego</i> śiódmego roku / w pewny cżás / roku odpuƺczánia / w Święto Kuczek :
Gdy śię zeydźie wƺyſtek Izráel / áby śię pokazał przed PAnem Bogiem twojim ná miejſcu / które ſobie obierze : obwołaƺ ten Zakon przed wƺyſtkim Izráelem w uƺách ich.
A zgromádźiƺ lud / męże y niewiáſty / y dźiatki / y przychodnie twoje / którzy ſą w bramách twojich / áby ſłucháli / y áby śię uczyli y bali PAná Bogá wáƺego : y przeſtrzegáli / áby cżynili wƺyſtkie ſłowá zakonu tego.
Synowie też ich / którzy <i>jeƺcże</i> nie wiedzą / niechaj ſłucháją / y niech śię ucżą bać PAná Bogá wáƺego / po wƺyſtkie dni których żyć będźiećie ná źiemi / do którey idźiećie / przepráwiwƺy śię przez Jordan wnidźiećie / ábyśćie ją pośiedli.
Zátym rzekł PAN do Mojzeƺá : Oto przybliżyły śię dni śmierći twojey : Przyzówże Jozuego / á ſtáńćie w Namiećie zgromádzenia / á przykażę mu : Szedł tedy Mojzeƺ / y Jozue / á ſtánęli w Namiećie zgromádzenia.
Tedy śię ukazał PAN w Namiećie / w Słupie obłokowym : y ſtánął on ſłup obłokowy náde drzwiámi Namiotu.
Y rzekł PAN do Mojzeƺá : Oto ty záśnieƺ z Ojcy twojimi : á powſtawƺy lud ten / będźie cudzołożył náśládując Bogów cudzoźiemców tey źiemie / do którey wchodźi / <i>áby mieƺkał</i> w pośrzodku niey / y opuśći mię / y złámie przymierze moje / którem poſtánowił z nim.
Przetoż zápali śię gniew mój przećiwko jemu onego dniá / á opuƺcżę je / y ſkryję oblicze moje przed nimi / á będźie ná pożárćie : y znajdą go wiele złych rzecży y utrapienia : y rzecże dniá onego : Azaż nie dla tego / iż Bogá mego niemáƺ w pośrzodku mnie / ználázły mię te złe <i>rzeczy</i>?
A ja kryjąc ſkryję oblicze moje w on dźień / dla wƺyſtkiego złego / które cżynili / udawƺy śię zá Bogi cudzymi.
Teraz tedy nápiƺćie ſobie tę pieśń / á náucż jey Synów Izráelſkich : włóż ją w uſtá ich / áby mi byłá tá pieśń świádectwem przećiwko Synom Izráelowym :
Abowiem wprowádzę go do źiemie / o którąm przyśiągł Ojcom jego / opływájącey mlekiem y miodem : y będźie jadł / á náje śię y utyje / á obróći śię do Bogów cudzych / y ſłużyć im będźie / y rozdrażnią mię / y zrzucą przymierze moje.
Y ſtánie śię / gdy go znajdźie wiele złego y utrapienia / że będźie tá pieśń przed nim ná świádectwo / ábowiem nie przydźie w zápámiętánie w uśćiech naśienia ich : gdyż wiem myśli jego <i>y</i> co on <i>jeƺcże</i> dźiś cżynić będźie pierwey niż go wprowádzę do źiemie / o którąm przyśiągł.
Y nápiſał Mojzeƺ tę pieśń onegoż dniá / y náucżył jey Synów Izráelſkich.
Potym przykazał Jozuemu Synowi Nunowemu / y rzekł : zmacniaj śię / y mężnie ſobie pocżynaj / ábowiem ty wprowádźiƺ Syny Izráelſkie do źiemie o którąm im przyśiągł : á ja będę z tobą.
Y ſtáło śię / gdy nápiſał Mojzeƺ ſłowá Zakonu tego w kśięgi / y dokońcżył ich :
Rozkazał Mojzeƺ Lewitom / którzy nośili ſkrzynię Przymierza PAńſkiego / mówiąc :
Weźmićie kśięgi zakonu tego y włóżćie je przy ſtronie / do ſkrzynie Przymierza PAná Bogá wáƺego / áby tám były przećiwko tobie ná świádectwo :
Abowiem ja znam upór twój / y kárk twój twárdy / oto póki ja jeƺcże dźiś żyję z wámi / jeſteśćie odpornymi PAnu / jákoż dáleko więcey po śmierći mojey?
Zgromadźćież do mnie wƺyſtkie Stárƺe pokoleń wáƺych / y Przełożone wáƺe / żebym mówił w uƺách ich ſłowá te / y będę wzywał ná świádectwo przećiwko nim / Niebá y źiemię :
Ponieważ wiem / że po śmierći mojey / popſujećie śię bárzo / á uſtąpićie z drogi / którąm wam rozkazał : przetoż przydźie ná was <i>to</i> złe w oſtátnie dni / gdy będźiećie cżynić złe przed Ocżymá PAńſkimi / drażniąc go ſpráwámi rąk wáƺych.
Y mówił Mojzeƺ w uƺách wƺyſtkiego zgromádzenia Izráelſkiego / ſłowá tey pieśni / áż jey dokończył.
