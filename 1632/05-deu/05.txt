Tedy zwoławƺy Mojzeƺ wƺyſtkiego Izráelá mówił do nich : Słuchaj Izráelu uſtaw y ſądów / które ja dźiś mówię w uƺy wáƺe / náucżćie śię ich / á przeſtrzegajćie tego / ábyśćie je cżynili.
PAN Bóg náƺ / ucżynił z námi Przymierze ná <i>górze</i> Horeb.
Nie z Ojcy náƺymi ucżynił PAN to Przymierze / ále z námi / którzyſmy tu dźiś wƺyſcy żywi.
Twarzą w twarz / mówił PAN z wámi ná górze / z pośrzodku ogniá :
( <i>A</i> jam ſtał miedzy PAnem / y miedzy wámi ná on cżás / ábym wam odnośił ſłowo PAńſkie : bośćie śię bali ogniá / á nie wſtąpiliśćie ná górę : ) y rzekł.
Jam jeſt PAN Bóg twój / którym ćię wywiódł z źiemie Egipſkiey / z domu niewoli.
Nie będźieƺ miał Bogów innych przedemną.
Nie cżyń ſobie Obrázu rytego / áni żadnego podobieńſtwá <i>tych rzecży,</i> które ſą ná Niebie w zgórę / y które ná źiemi niſko / y które w wodách pod źiemią :
Nie będźieƺ śię im kłaniał / áni ich chwalił : Bom ja PAN Bóg twój / Bóg zawiſny <i>w miłośći</i> / náwiedzájący niepráwość Ojców nád Syny / do trzećiego y do cżwartego <i>pokolenia,</i> tych którzy mię nienawidzą.
A cżyniący miłośierdźie nád tyśiącmi tych / którzy mię miłują y ſtrzegą przykazań mojich.
Nie bierz Imienia PAná Bogá twego ná dáremno : Bo śię będźie mśćił PAN nád tym / który imię jego ná dáremno bierze.
Przeſtrzegaj dniá ſobotniego / ábyś go święćił / jákoć rozkazał PAN Bóg twój.
Przez ƺeść dni będźieƺ robił / y wykonaƺ wƺeláką robotę twoję :
Ale dniá śiódmego jeſt odpocżynienie PAná Bogá twego / nie cżyń żadney roboty / ty y Syn twój / y córká twojá / y ſługá twój / y ſłużebnicá twojá / y wół twój / y Ośieł twój / y káżde bydlę twoje / y gość twój / który jeſt w bramách twojich / áby odpocżynął ſługá twój / y ſłużebnicá twojá / jáko y ty.
A pámiętaj żeś był niewolnikiem w źiemi Egipſkiey : y wywiódł ćię PAN Bóg twój z támtąd ręką możną / y rámieniem wyćiągnionym : przetożći przykazał PAN Bóg twój ábyś obchodźił dźień Sobotni.
Czći Ojcá twego y mátkę twoję / jákoć przykazał PAN Bóg twój : áby przedłużone były dni twoje / y żebyć śię dobrze dźiało ná źiemi / którą PAN Bóg twój da tobie.
Nie będźieƺ zábijał.
Nie będźieƺ cudzołożył.
Nie będźieƺ kradł.
Nie będźieƺ mówił przećiw bliźniemu twemu / świádectwá fałƺywego.
Nie będźieƺ pożądał żony bliźniego twego / áni będźieƺ pożądał domu bliźniego twego / roli jego / y ſługi jego / y ſłużebnice jego / wołu jego / y Oſłá jego / y wƺyſtkich rzecży / które <i>ſą</i> bliźniego twego.
Teć ſłowá mówił PAN / do wƺyſtkiego zgromádzenia wáƺego ná górze / z pośrzodku ogniá / obłoku / y mgły / głoſem wielkim / á <i>nic więcey</i> nie przydał : y nápiſał je ná dwu tablicách kámiennych które mnie oddał.
Y ſtáło śię / gdyśćie uſłyƺeli głos z pośrzodku ćiemnośći gdy górá ogniem pałáłá / żeśćie przyſtąpili do mnie / wƺyſtkie Kśiążętá pokoleń wáƺych / y Stárƺy wáƺy :
Y mówiliśćie : Oto nam ukazał PAN Bóg náƺ chwałę ſwoję / y wielmożność ſwoję / á głos jego ſłyƺeliſmy z pośrzodku ogniá : Dźiś widźieliſmy że Bóg mówił z cżłowiekiem / á <i>cżłowiek</i> żyw zoſtał :
A ták teraz przecżże mamy pomrzeć / ábowiem nas ten ogień wielki pożrze : Jeſli jeƺcże ſłyƺeć będźiemy głos PAná Bogá náƺego / pomrzemy.
Abowiem / cóż jeſt wƺelkie ćiáło / áby ſłyƺáło głos Bogá żywiącego / mówiącego z pośrzodku ogniá / jáko my / á żywo zoſtáło?
Idźże ty / á wyſłuchaj wƺyſtkiego / co będźie mówił PAN Bóg náƺ : Ty záś powieƺ nam wƺyſtko / co do ćiebie mówić będźie PAN Bóg náƺ / á my ſłucháć y cżynić <i>to</i> będźiemy.
A uſłyƺawƺy PAN głos ſłów wáƺych / gdyśćie mówili do mnie : rzekł mi PAN : ſłyƺałem głos ſłów ludu tego / które mówili do ćiebie / dobrze wƺyſtko mówili / <i>co mówili.</i>
Ktoby im to dał / żeby ſerce ich było tákie / áby śię mnie bali y ſtrzegli wƺyſtkich przykazań mojich / po wƺyſtkie dni / áby śię im dobrze dźiało y Synom ich ná wieki!
Idźże / <i>á</i> rzecż im / Wróććie śię do namiotów wáƺych.
A ty tu zoſtań przy mnie / y opowiem tobie wƺyſtkie przykazánia / y uſtáwy / y ſądy / których ich náucżáć będźieƺ / áby je cżynili w źiemi / którą ja im dawam / áby ją pośiedli.
Przetoż ſtrzeżćie ábyśćie cżynili / jáko wam rozkazał PAN Bóg wáƺ / nie uchylájąc śię ná práwo áni ná lewo.
Wƺelką też drogą / którą wam przykazał PAN Bóg wáƺ / chodźić będźiećie : ábyśćie żyli / y dobrze śię wam dźiało / y żebyśćie przedłużyli dni <i>ſwoje</i> ná źiemi / którą pośiądźiećie.
