Jesliby náleźiony był zábity w źiemi / którą PAN Bóg twój dawa tobie / ábyś ją pośiadł / leżący ná polu / á nie wiedźianoby kto go zábił :
Tedy wynidą Stárƺy twoji / y Sędźiowie twoji / y rozmierzą <i>źiemię</i> áż do miaſt które <i>ſą</i> w około onego zábitego.
A któreby miáſto bliżƺe było onego zábitego / tedy wezmą Stárƺy miáſtá onego / jáłowicę z ſtádá / którą jeƺcze nie robiono / á która nie ćiągnęłá w járzmie :
Y záwiodą Stárƺy tego miáſtá / onę jáłowicę w dolinę kámieniſtą / gdźie nigdy nie orano / áni śiano / y utną tám ƺyję jáłowicy w oney dolinie.
Zátym przyjdą Kápłani / Synowie Lewiego : bo je obrał PAN Bóg twój / áby mu ſłużyli / y żeby błogoſłáwili w imieniu PAńſkim / á według uznánia ich / kończyć śię ma káżdy ſpór y káżda ráná :
Tákże wƺyſcy Stárƺy miáſtá onego / które jeſt nabliżƺe zábitego / umyją ręce ſwoje / nád jáłowicą śćiętą w dolinie.
Y oświadczą śię / mówiąc ; Ręce náƺe nie wylały tey krwie / áni ocży náƺe <i>ná to</i> pátrzáły.
Oczyść lud twój Izráelſki / któryś odkupił PANIE / á nie pocżytaj krwie niewinney / w pośrzodku ludu twego Izráelſkiego : Tedy oczyśćieni będą od oney krwie.
A ty odejmieƺ krew niewinną z pośrzodku śiebie / gdy uczyniƺ co práwego jeſt / przed oczymá PAńſkimi.
Gdy <i>też</i> wynidźieƺ ná wojnę / przećiw nieprzyjaćiołom twojim / á podać je PAN Bóg twój w ręce twoje / y nábierzeƺ z nich więźniów.
A obacżyƺ miedzy więźniámi Niewiáſtę nádobną / którey rozmiłowawƺy śię / wźiąłbyś ją ſobie zá żonę :
Tedy ją wprowádźiƺ do domu ſwego / y ogoli głowę ſwoję / y obrzeże páznogćie ſwoje :
A złożywƺy z śiebie ƺátę / w którey jeſt pojimána / zoſtánie w domu twym / á płákáć będźie Ojcá ſwego / y Mátki ſwojey / przez cáły mieśiąc : á potym / wnidźieƺ do niey / y będźieƺ mężem jey / á oná będźie tobie zá żonę.
A jeslibyć śię <i>potym</i> nie podobáłá / tedy ją wolno puśćiƺ / á żadną miárą nie przedaƺ jey zá pieniądze / áni nią kupczyć będźieƺ / ponieważeś ją zelżył.
Gdyby też kto miał dwie żenie / jednę miłą / á drugą omierzłą / y národźiłyby mu Synów / miła y omierzła : á byłby Syn pierworodny omierzłey :
Tedy gdy ſtánowić będźie zá dźiedźice Syny ſwoje dóbr ſwojich : nie będźie mógł dáć pierworodztwá Synowi miłey / przed Synem pierworodnym oney omierzłey :
Ale pierworodztwo Synowi omierzłey przyzna / dawƺy mu dwojáką część wƺyſtkiego co ma : ponieważ on <i>jeſt</i> początkiem śiły jego : jego jeſt práwo pierworodztwá.
Jeſliby kto miał Syná ſwowolnego / y krnąbrnego / któryby nie ſłuchał głoſu Ojcá ſwego / y głoſu Mátki ſwojey / á będąc ſtrofowány / nie uſłuchałby ich :
Tedy go wezmą / Oćiec jego y Mátká jego / y przywiodą go do Stárƺych miáſtá ſwego / y do bramy <i>ſądowey</i> miejſcá onego :
Y rzeką do Stárƺych miáſtá onego : Ten náƺ Syn / jeſt ſwowolny / y krnąbrny / niepoſłuƺny głoſu náƺemu : żárłok <i>jeſt</i> y pijánicá.
Tedy ukámionują go wƺyſcy ludźie miáſtá onego / y umrze : á ták odejmieƺ złe z pośrzodku śiebie : co gdy wƺyſtek Izráel uſłyƺy / bać śię będą.
A gdyby kto popełnił grzech godny śmierći / á byłby zkazány ná śmierć / á obieśiłbyś go ná drzewie :
Nie zoſtánie przez noc trup jego ná drzewie / ále koniecznie pochowaƺ go tegoż dniá : bo przeklęctwem Bożym <i>jeſt</i> / ten co wiśi. Przetoż nie będźieƺ plugáwił źiemie twey / którą PAN Bóg twój dawa tobie w dźiedźictwo.
