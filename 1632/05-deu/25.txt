Jeſliby śię ſpór wƺcżął miedzy ſtronámi / á przyƺliby do ſądu / áby je ſądzono : tedy uſpráwiedliwią ſpráwiedliwego / á niezbożnego oſądzą zá niezbożnego.
A jeſli godźien będźie bićia niepobożny / tedy każe go położyć ſędźia / y káże go bić przed ſobą / według miáry niepráwośći jego / pod liczbą.
Czterdźieśći kroć uderzy go / <i>więcey</i> nie przyda / by ſnadź / jesliby mu nád to zádano więcey plag / nie znikczemniał brát twój / przed oczymá twymi.
Nie záwiążeƺ gęby wołowi młócącemu.
Gdyby mieƺkáli bráćia poſpołu / á zƺedłby jeden z nich nie májąc Syná / nie pójdźie żoná onego zmárłego / zá mężá obcego : brát jego wnidźie do niey / á pojmie ją ſobie zá żonę / á práwem powinowáctwá przyłączy ją ſobie.
A pierworodny <i>Syn</i> któregoby porodźiłá / názwány będźie imieniem brátá jego zmárłego / áby nie było wygłádzone imię jego z Izráelá.
A jeſliby nie chćiał on mąż pojąć brátowy ſwojey / tedy pójdźie brátowa jego do bramy przed Stárƺe / y rzecże : nie chce brát mężá mego wzbudźić brátu ſwemu imienia w Izráelu / y nie chce mię práwem powinowáctwá ſobie przyłącżyć.
Tedy go przyzowią Stárƺy miáſtá onego / y będą mówić z nim : A ſtánąwƺy / jeſli rzecże / nie chcę jey pojąć :
Przyſtąpi brátowá jego do niego przed ocżymá Stárƺych á z zuje trzewik jego / z nogi jego / y plunie ná twarz jego / á odpowiádájąc / rzecże : Ták śię ſtánie mężowi / któryby nie budował domu brátá ſwego.
Y názywáć będą imię jego w Izráelá : Dom wyzutego.
Gdyby śię powádźili z ſobą dwáj jácy mężowie / á przybieżáłáby żoná jednego / áby wyrwáłá mężá ſwego / z ręki onego co go bije / á zćiągnąwƺy rękę ſwą / uchwićiłáby go zá łono jego :
Utnieƺ jey rękę : y nie zlituje śię <i>nád nią</i> oko twoje.
Nie będźieƺ miał w worku twojim różnych gwichtów / więkƺego y mniejƺego.
Nie będźieƺ też miał w domu twojim dwojákiego korcá / więkƺego y mniejƺego :
Wagę ſłuƺną y ſpráwiedliwą mieć będźieƺ / <i>tákże</i> korzec ſłuƺny y ſpráwiedliwy mieć będźieƺ / áby śię przedłużyły dni twoje ná źiemi / którą PAN Bóg twój dawa tobie.
Abowiem obrzydliwośćią jeſt PAnu Bogu twemu káżdy / który to czyni / wƺelki cżyniący niepráwość.
Pomni ná to coć ucżynił Amálek w drodze / kiedyśćie ƺli z Egiptu /
Jákoć zábieżał drogę / á pobił oſtátnie wojſká twego / wƺyſtkie mdłe <i>idące</i> zá tobą / gdyś ty był utrudzony y ſprácowány : á nie bał śię Bogá :
Przetoż gdyć da odpocżynienie PAN Bóg twój / od wƺyſtkich nieprzyjaćiół twojich w około / w źiemi którą PAN Bóg twój dawa tobie w dźiedźictwo / ábyś ją pośiadł : wygłádźiƺ pámiątkę Amálekowę pod niebem. Nie zápominajże <i>tego</i>.
