Tedy wſtąpił Mojzeƺ <i>z onych</i> równin Moábſkich / ná górę Nebo ná wierzch págórku / który <i>jeſt</i> ná przećiwko Jerichu : á ukazał mu PAN wƺyſtkę źiemię od Gáláád áż do Dán /
Y wƺyſtkę źiemię Neftálimowę / y źiemię Efrájimowę / y Mánáſſeſſowę / y wƺyſtkę źiemię Judowę / áż do morzá oſtátniego.
Y <i>ſtronę</i> Południową / y równinę doliny Jerichá / miáſtá <i>oſadzonego</i> Pálmámi / áż do Segor.
Tedy mu rzekł PAN : Táć jeſt źiemiá / o którąm przyśiągł Abráhámowi / Izáákowi y Jákóbowi / mówiąc : naśieniu twemu dam ją : pokazałem ją ocżom twojim / ále do niey nie wnidźieƺ.
Y umárł tám Mojzeƺ / ſługá PAńſki w źiemi Moábſkiey / według ſłowá PAńſkiego.
Y pogrzebł go <i>PAN</i> w dolinie / w źiemi Moábſkiey / przećiw Betfegorowi / á nie dowiedźiał śię nikt o grobie jego / áż do dniá tego.
A Mojzeƺ miał ſto y dwádźieśćiá lat / gdy umárł : nie záćmiło śię oko jego / áni śię náruƺyłá cżerſtwość jego.
Y płákáli Synowie Izráelſcy Mojzeƺá / ná polách Moábſkich / przez trzydźieśći dni. A gdy śię ſkońcżyły dni płácżu / <i>y</i> żałoby Mojzeƺowey :
Tedy Jozue Syn Nunów / nápełniony jeſt Duchem mądrośći : bo był włożył Mojzeƺ ręce ſwoje nań : y poſłuƺni mu byli Synowie Izráelſcy : á cżynili jáko przykazał PAN Mojzeƺowi.
Y nie powſtał Prorok więcey w Izráelu / podobny Mojzeƺowi / któregoby ták znał PAN / twarzą w twarz :
We wƺyſtkich znákách / y cudách / <i>dla</i> których go był poſłał PAN / áby je cżynił w źiemi Egipſkiey / <i>przed</i> Fáráonem / y <i>przed</i> wƺyſtkimi ſługámi jego / y <i>przed</i> wƺyſtką źiemią jego :
Tákże we wƺyſtkich <i>ſpráwách</i> ręki możney / y we wƺyſtkich poſtráchách wielkich / które cżynił Mojzeƺ / przed ocżymá wƺyſtkiego Izráelá.
