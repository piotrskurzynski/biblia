Gdy ćię wprowádźi PAN Bóg twój do źiemie / do którey wchodźiƺ / ábyś ją pośiadł / y wytráći Narodów wiele przed twarzą twoją / Hettejcżyká / y Gergezeycżyká / y Amorrejcżyká / y Chánánejcżyká / y Ferezejcżyká / y Hewejcżyká / y Jebuzejcżyká / śiedm Narodów / więkƺych y możniejƺych niżliś ty :
A poda je PAN Bóg twój tobie / iż je poráźiƺ : <i>tedy</i> wytráćiƺ je do ƺcżątku / nie będźieƺ brał z nimi przymierza / áni śię zlitujeƺ nád nimi :
Ani śię ſpowinowáćiƺ z nimi : córki ſwey nie daƺ Synowi jego / y córki jego nie weźmieƺ Synowi ſwemu /
Boby zwiodłá Syná twego / żeby nie ƺedł zá mną / á ſłużyliby Bogom cudzym : zácżym zápaliłby śię gniew PAńſki przećiwko wam / á wytráćiłby was prętko.
Ale owƺem to im ucżyńćie : Ołtarze ich poobalaćie / á ſłupy ich pokruƺyćie / y gáje ich <i>poświęcone</i> wyrąbićie / á ryte ich báłwany ogniem popalićie :
Abowiemeś ty lud święty PANU Bogu twemu / ćiebie obrał PAN Bóg twój / ábyś mu był osobliwym ludem / ze wƺyſtkich narodów / które <i>ſą</i> ná źiemi.
Nie przeto / że was więcey nád inne narody / przyłącżył śię PAN do was / y obrał was : gdyż was było namniey ze wƺyſtkich narodów :
Ale iż was umiłował PAN / y ſtrzedz chćiał oney przyśięgi / którą przyśiągł Ojcom wáƺym : wywiódł was PAN ręką możną / y wykupił was z domu niewoli / z ręki Fáráoná królá Egipſkiego.
A ták wiedz / że PAN Bóg twój / jeſt Bóg / Bóg wierny / chowájący przymierze / y miłośierdźie / tym którzy go miłują y ſtrzegą przykazań jego / do tyśiącnego pokolenia :
A oddájący tym którzy go máją w nienawiśći <i>káżdemu</i> w twarz jego / áby go wytráćił : nie omieƺka temu który go ma w nienawiśći : w twarz jego odda mu.
Przetoż przeſtrzegaj przykazánia / y uſtaw / y ſądów które ja dźiś rozkázuję tobie / ábyś je cżynił.
Y ſtánie śię / że jeſli ſłucháć ſądów tych / á przeſtrzegáć / y cżynić je będźiećie / tedy też dotrzyma PAN Bóg twój tobie przymierza / y miłośierdźia / które poprzyśiągł Ojcom twojim.
Y umiłuje ćię / y ubłogoſłáwi ćię / y rozmnoży ćię : bo pobłogoſłáwi owocowi żywotá twego / y owocowi źiemie twojey / zbożu twojemu / y winu twojemu / y Oliwie twojey / płodowi krów twojich / y trzodom owiec twojich / w źiemi o którą przyśiągł Ojcom twojim / że ją da tobie.
Błogoſłáwionym będźieƺ nád wƺyſtkie narody : nie będźie u ćiebie niepłodny y niepłodna / áni miedzy bydłem twojim.
Oddali też PAN od ćiebie káżdą niemoc / y wƺelkie choroby Egipſkie złe / o których wieƺ : nie dopuśći ich ná ćię / ále je przepuśći ná wƺyſtkie którzy ćię nienawidzą.
Y wytráćiƺ wƺyſtkie narody / które PAN Bóg twój poda tobie : nie zfolguje im oko twoje / áni będźieƺ chwalił Bogów ich / boćby to było śidłem.
Jeſlibyś rzekł w ſercu ſwym : Więkƺe ſą te narody niżli ja : jákoż je będę mógł wygnáć?
Nie bój śię ich / <i>ále</i> pilnie pámiętaj ná to / co ucżynił PAN Bóg twój Fáráonowi / y wƺyſtkim Egipcżánom :
<i>Ná</i> one kuƺenia wielkie / które widźiáły ocży twoje / y ná znáki / ná cudá / y ná rękę możną / y ná rámię wyćiągnione / którym ćię wywiódł PAN Bóg twój : tákći ucżyni PAN Bóg twój / wƺyſtkim narodom / których śię ty twarzy bojiƺ.
Nád to / pośle PAN Bóg twój ná nie śierƺenie / áż wygubi oſtátki ich / y <i>te</i> którzyby śię pokryli przed tobą.
Nie lękajże śię twarzy ich : ábowiem PAN Bóg twój <i>jeſt</i> w pośrzodku ćiebie : Bóg wielki y ſtráƺny.
Y wyniƺcży PAN Bóg twój narody one przed tobą po lekku / y po troƺe :
Nie będźieƺ ich mógł wytráćić prętko / by śię ſnadź nie námnożyło przećiw tobie beſtyi polnych.
Y poda je tobie PAN Bóg twój / y zetrze je ſtárćiem wielkiem / áż będą wyniƺcżeni. A poda króle ich w ręce twoje / y wygubiƺ imię ich pod Niebem : nie oſtoji śię żaden przed tobą / áż je wytráćiƺ.
Obrázy ryte Bogów ich popaliƺ ogniem / nie będźieƺ pożądał śrebrá / áni złotá <i>które jeſt</i> ná nich / ábyś <i>je</i> miał ſobie bráć / byś śię ſnadź nie uśidlił w nim : ponieważ to obrzydliwość jeſt PAnu Bogu twemu.
Nie wnośże obrzydliwośći w dom twój / ábyś śię nie ſtał przeklęctwem jáko y oná / wielce śię nią brzydźić będźieƺ / y bárzo ją ſobie obmierźyƺ / ponieważ przeklęctwem jeſt.
