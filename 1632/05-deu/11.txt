Miłujże tedy PAná Bogá twego / y przeſtrzegaj obrzędów jego / y uſtaw jego / y ſądów jego / y przykazań jego / po wƺyſtkie dni.
A poznajćie dźiś ( bo nie <i>mówię</i> do Synów wáƺych / którzy nie ználi / áni widźieli karánia PAná Bogá wáƺego ) wielmożność jego / rękę jego mocną / y rámię jego wyćiągnione :
Y cudá jego / y ſpráwy jego / które ucżynił w pośród Egiptu Fáráonowi królowi Egipſkiemu / y wƺyſtkiey źiemi jego.
Y co uczynił wojſku Egipſkiemu / koniom jego / y wozom jego : który ſpráwił / że je okryły wody / morzá cżerwonego / gdy was gonili / y wytráćił je PAN áż do dniá tego.
Tákże co wam uczynił ná puƺczy / áżeśćie przyƺli ná to miejſce.
Y co ucżynił Dáthánowi / y Abironowi / Synom Eliábá Syná Rubenowego : jáko otworzywƺy źiemiá uſtá ſwe / pożárłá je / y domy ich / y namioty ich / y wƺyſtkę májętność / która byłá przy nich / w pośrzodku wƺyſtkiego Izráelá.
A ocży wáƺe widźiáły wƺyſtkie ſpráwy PAńſkie wielkie / które cżynił.
Strzeżćieƺ tedy káżdego przykazánia / które ja przykázuję wam dźiś ábyśćie śię zmocnili / á wƺedƺy pośiedli źiemię / do którey idźiećie / ábyśćie ją odźiedźicżyli.
A żebyśćie dni ſwojich przedłużyli ná źiemi / o którą przyśiągł PAN Ojcom wáƺym / że im ją dam / y naśieniu ich / źiemię opływájącą mlekiem y miodem.
Abowiem źiemiá / do którey ty idźieƺ / ábyś ją pośiadł / nie jeſt jáko źiemiá Egipſka / z któreyśćie wyƺli / w którey pośiawƺy naśienie twoje / pokrapiáćieś muśiał <i>z prácą</i> nóg twojich / jáko Ogród járzynny.
Ale źiemiá do którey idźiećie / ábyśćie ją pośiedli / źiemiá <i>jeſt</i> májącá góry y doliny / która dzdżem Niebieſkim odwilżána bywa :
Ziemiá / o którey PAN Bóg twój piecżą ma / y ná którą záwżdy oczy PAná Bogá twego <i>pátrzą,</i> od początku roku / y áż do ſkończenia jego.
A ták będźiećieli pilnie ſłucháli przykazań mojich / które ja dźiś przykázuję wam / żebyśćie miłowáli PAná Bogá wáƺego / y ſłużyli mu ze wƺyſtkiego ſercá wáƺego / y ze wƺyſtkiey duƺe wáƺey :
Tedy dam deƺcz źiemi wáƺey czáſu ſwego / ránny y pózdny / y będźieƺ zbierał zbożá twoje / y wino twoje / y Oliwę twoję.
Dam też trawę ná polách twojich / dla bydłá twojego / y będźieƺ jadł / á nájeƺ śię.
Strzeżćieƺ śię / by ſnadź nie było zwiedźione ſerce wáƺe / ábyśćie odſtąpiwƺy / nie ſłużyli Bogom cudzym / y nie kłaniáli śię im.
Zkądby śię zápalił gniew PAńſki przećiwko wam / y zámknąłby Niebá / y nie byłoby dżdżá / ániby źiemiá wydawáłá urodzáju ſwego / y zginęlibyśćie prętko z źiemie <i>tey</i> wyborney / którą PAN dawa wam.
Przetoż / złózćie te ſłowá moje do ſercá wáƺego / y do umyſłu wáƺego / á uwiążćie je ná znák ná ręku wáƺych / y niech będą <i>jáko</i> nácżelniki / miedzy ocżymá wáƺymi.
A náucżajćie ich Synów wáƺych / rozmawiájąc o nich / gdy uśiędźieƺ w domu twym / y gdy będźieƺ w drodze / y gdy śię ukłádźieƺ / y gdy wſtánieƺ.
Nápiƺeƺ je też ná podwojách domu twego / y ná bramách twojich :
Aby śię rozmnożyły dni wáƺe / y dni Synów wáƺych ná źiemi / o którą przyśiągł PAN Ojcom wáƺym / że ją im da : póki Niebo nád źiemią <i>trwáć będźie</i>.
Bo jeſliż z pilnośćią ſtrzedz będźiećie káżdego przykazania tego / które ja wam rozkázuję / ábyśćie je cżynili : żebyśćie miłowáli PAná Bogá wáƺego / y chodźili we wƺyſtkich drogách jego / trwájąc przy nim :
Tedyć wypędźi PAN / one wƺyſtkie Narody przed wámi / y pośiądźiećie Narody więkƺe / y możniejƺe niżeśćie wy ſámi.
Wƺelkie miejſce / po którym deptáć będźie ſtopá nogi wáƺey / wáƺe będźie : od puƺcży y Libánu / od rzeki / rżeki Eufrátes / áż do morzá oſtátniego / będźie gránicá wáƺá.
Nie zoſtoji śię żaden przed wámi : lękánie wáƺe / y ſtrách wáƺ / puśći PAN Bóg wáƺ / ná oblicze wƺyſtkiey źiemie / którą będźiećie deptáć / jáko wam powiedźiał.
Oto ja dźiś wam przedkładam błogoſłáwieńſtwo / y przeklęctwo :
Błogoſłáwieńſtwo / będźiećieli poſłuƺni przykazániu PAná Bogá wáƺego / które ja przykázuję wam dźiś.
A przeklęctwo / jeſli poſłuƺni nie będźiećie przykazániam PAná Bogá wáƺego / y uſtąpićie z drogi / którą ja wam dźiś rozkázuję / udawájąc śię zá Bogámi obcymi / których nie znaćie.
A gdy ćię wprowádźi PAN Bóg twój / do źiemie / do którey idźieƺ / ábyś ją pośiadł / tedy daƺ błogoſłáwieńſtwo <i>to</i> ná Górze Gárizym / á przeklęctwo ná Górze Hebál.
Azaż nie ſą zá Jordanem / zá drogą ná zachód ſłońcá / w źiemi Chánánejczyków / którzy w polách mieƺkáją / przećiwko Gálgál / przy równinnie Morech.
Abowiem wy przejdźiećie zá Jordan / ábyśćie wƺedƺy odźiedźicżyli <i>tę</i> źiemię / którą wam dawa PAN Bóg wáƺ / á pośiądźiećie ją / y mieƺkáć w niey będźiećie.
Pilnujćieƺ tedy / ábyśćie cżynili wƺyſtkie uſtáwy y ſądy które ja wam dźiś przedkładam.
