Ná pocżątku <i>káżdego</i> śiódmego roku cżynić będźieƺ odpuƺcżenie.
A toć jeſt ſpoſób odpuƺcżánia : żeby odpuśćił káżdy / który pożycżył z ręki ſwey / to cżego pożycżył bliźniemu ſwemu : nie będźie wyćiągał <i>długu</i> od bliźniego ſwego / y od brátá ſwego : ponieważ obwołáne jeſt odpuƺcżenie PAńſkie.
Od obcego wyćiągáć <i>dług</i> możeƺ : ále cobyś miał u brátá twego / odpuśći <i>mu</i> ręká twojá :
Dla tego / żeby śię nie ſtał miedzy wámi <i>kto</i> przez ćię ubogim / ponieważ hojnie błogoſłáwić tobie będźie PAN w źiemi / którą PAN Bóg twój dawa tobie w dźiedźictwo / ábyś ją pośiadł.
Tylko żebyś ſłuchájąc / poſłuƺny był głoſowi PANA Bogá twego / ábyś ſtrzegł / y cżynił káżde przykazánie to / które ja przykázuję tobie dźiśiaj.
Abowiem PAN Bóg twój / błogoſłáwić ćię będźie / jákoć obiecał : y będźieƺ pożycżał wielu narodom / á ſam <i>u nikogo</i> nie będźieƺ pożycżał : y będźieƺ pánował nád wielą narodów / á <i>one</i> nád tobą pánowáć nie będą.
Gdyby był u ćiebie ubogi ktokolwiek z bráćij twojey / w któremkolwiek mieśćie twojim / w źiemi twojey / którą PAN Bóg twój dawa tobie : nie zátwardzaj ſercá ſwego / áni záwieraj ręki twey / przed brátem twojim ubogim :
Ale ƺcżodrze otworzyƺ mu rękę twoję / y ochotnie pożycżyƺ mu / ile będźie potrzebował / y cżegoby mu niedoſtawáło.
Strzeż śię áby nie byłá jáká niepobożność w ſercu twojim / żebyś miał rzec : Nádchodźi rok on śiódmy / rok odpuƺcżenia : y ſurowobyś śię ſtáwił brátu twemu ubogiemu / ták żebyś mu nie użycżył / á <i>onby</i> wołał przećiwko tobie do PAná / y miałbyś grzech.
<i>Ale</i> ochotnie dawáć mu będźieƺ / y nie będźie niechętne ſerce twoje / gdy mu dawáć będźieƺ / ábowiem dla tegoć pobłogoſłáwi PAN Bóg twój / we wƺyſtkich ſpráwách twojich / y do cżegokolwiek zćiągnieƺ rękę twoję.
Boć śię nie przebierze ubogich / w źiemi <i>wáƺey</i> / dla tegoć rozkázuję / mówiąc : ábyś ƺcżodrze otwierał rękę twą brátu twemu / y nędznemu twemu / y ubogiemu twemu / w źiemi twojey.
Jesliby śię záprzedał tobie brát twój / Żydowin álbo żydówká / á ſłużyłćiby przez ƺeść lat : tedy śiódmego roku wypuśćiƺ go wolno od śiebie.
A gdy go wolno puśćiƺ od śiebie / nie puśćiƺ go próznego :
Szcżodrze go udárujeƺ z bydłá twego / y z gumná twego / y z práſy twojey : <i>z tego</i> w cżymći pobłogoſłáwił PAN Bóg twój / daƺ mu.
Y ſpomniƺ ſobie / żeś był niewolnikiem w źiemi Egipſkiey / z kąd ćię wykupił PAN Bóg twój : dla tego ja to dźiś tobie przykázuję.
Jesliby też rzekł do ćiebie : nie pójdę od ćiebie / przeto iż ćię umiłował / y dom twój / á iż śię ma dobrze u ćiebie :
Tedy wźiąwƺy ƺydło / przekoleƺ ucho jego ná drzwiách / y będźie ſługą twojim ná wieki : toż y ſłużebnicy ſwey ucżyniƺ.
Niech to nie będźie przykro w ocżách twojich / że go puśćiƺ wolno od śiebie : bo dwojáką zapłátę najemniká záſłużył u ćiebie / przez ƺeść lat : y błogoſłáwić będźie tobie PAN Bóg twój / we wƺyſtkim co będźieƺ cżynił.
Wƺyſtkiego pierworodztwá / które śię urodźi z bydłá twego / álbo z trzody twojey / ſámce / poświęćiƺ PAnu Bogu twemu : nie będźieƺ robił pierworodnym krowy twojey / áni będźieƺ ſtrzygł pierworodnych owiec twojich :
Przed PAnem Bogiem twojim będźieƺ je jadł / káżdego roku ná miejſcu które obierze PAN ty y dom twój.
A jesliby ná nim byłá wádá / <i>żeby</i> chrome / álbo ślepe / <i>álbo</i> z jákąkolwiek wádą złą <i>było,</i> nie będźieƺ go ofiárował PAnu Bogu twemu :
W bramách twych jeść je będźieƺ / niecżyſty y cżyſty zá równo / jáko ſárnę / y jáko Jeleniá.
Tylko krwie jego nie będźieƺ jadł / ná źiemię wylejeƺ ją jáko wodę.
