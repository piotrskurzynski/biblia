Synámi jeſteśćie PAná Bogá wáƺego : nie będźiećie śię rzezáć / áni cżynić łyśiny miedzy ocżymá wáƺymi nád umárłym :
Abowiemeś ludem świętym PAnu Bogu twemu / y obrał ćię PAN ábyś mu był zá lud oſobliwy / ze wƺyſtkich narodów / którzy ſą ná oblicżu źiemie.
Nie będźieƺ jadał żadney obrzydliwośći.
Teć ſą zwierzętá które jeść będźiećie : Woły / Owce / y kozy.
Jeleniá / y Sárnę / y Báwołu / y dźikiego kozłá / y jednorożcá / y Łośiá / y Kózkę ſkálną.
Y wƺelkie zwierzę które ma rozdźielone kopyto / ták że śię ná dwie kopytá dźieli ſtopá jego / y które przeżuwa / miedzy zwierzęty / jeść je będźiećie.
A wƺákże z tych które przeżuwáją / y które rozdźielone kopytá máją jeść nie będźiećie / Wielbłądá / y zájącá / y króliká : ábowiem choćiaż te przeżuwáją / ále kopyt rozdwojonych nie máją : niecżyſte wam będą.
Tákże Świniá / choć ma kopyto rozdwojone / ále iż nie przeżuwa / niecżyſta wam będźie : mięſá ich jeść nie będźiećie y śćierwu śię ich nie dotkniećie.
To záś jeść będźiećie / ze wƺyſtkich <i>rzecży</i> które <i>ſą</i> w wodách / cokolwiek ma łuſkę y ſkrzele / jeść będźiećie.
Ale wƺyſtkiego co nie ma ſkrzeli / áni łuſki / jeść nie będźiećie / niecżyſte wam będźie.
Wƺyſtko ptáſtwo cżyſte jeść będźiećie.
Te záśię ſą których jeść nie będźiećie : Orłá / y Gryfá / y morſkiego Orłá.
Y Sokołá / y Sępá / y Kánie wedle rodzáju jey.
Ani żadnego kruká wedle rodzáju jego.
Ani Struśiá / áni Sowy / áni wodney Kánie / áni Krogulcá / według rodzáju jego.
Y Rarogá / y Lelká / y Łábęćiá.
Y Pelikaná / y Porfirioná / y Nurká.
Ani Boćianá / áni Sójki / według rodzáju ich / áni Dudká / áni Nietopyrzá.
Wƺelki też płáz ſkrzydláſty / niecżyſty wam będźie / jeść go nie będźiećie.
Káżdego ptaká cżyſtego jeść będźiećie.
Nie będźiećie jeść żadney zdechliny : przychodniowi który jeſt w bramách twojich daƺ to / á jeść to będźie : álbo przedaƺ cudzoźiemcowi / boś ty lud święty PAnu Bogu twemu / Nie będźieƺ wárzył Koźlęćiá w mleku mátki jego.
Ochotnie będźieƺ dawał dźieśięćiny / ze wƺyſtkiego urodzáju naśienia twego / co śię urodźi ná polu / ná káżdy rok.
A będźieƺ pożywał przed PAnem Bogiem twojim ná miejſcu / które obierze / żeby tám mieƺkáło imię jego : dźieśięćin zbożá twego / z winá twego / y z Oliwy twojey / y z pierworodztwá wołów twojich / y z Owiec twojich / ábyś śię ucżył bać PAná Bogá twego po wƺyſtkie dni.
A jeſliby dáleka byłá drogá ná ćię / żebyś tego tám donieść nie mógł / przeto / że odległe jeſt tobie ono miejſce / któreby obrał PAN Bóg twój / ku mieƺkániu tám imieniowi ſwemu / gdyć błogoſłáwić będźie PAN Bóg twój :
Tedy to zpieniężyƺ / á májąc záwiązáne pieniądze w ręku twojich / pójdźieƺ ná miejſce / które ſobie obierze PAN Bóg twój.
Y zá one pieniądze nákupiƺ wƺyſtkiego co śię upodoba duƺy twojey wołów y Owiec / y Winá / y innego napoju mocnego y wƺyſtkiego cżego by pożądáłá duƺá twojá / á będźieƺ tám jadł przed PAnem Bogiem twojim / y będźieƺ śię weſelił / ty y dom twój.
A Lewity / który mieƺka w bramách twojich / nie opuśćiƺ go / ponieważ niema dźiału áni dźiedźictwá z tobą.
<i>A</i> po wyśćiu <i>káżdego</i> trzećiego roku / odłącżyƺ wƺyſtkie dźieśięćiny urodzáju twego / onegoż roku / y złożyƺ je w bramách twojich.
Tedy przydźie Lewitá / który niema dźiału y dźiedźictwá z tobą / y przychodźień / y śierotá / y wdowá / którzy ſą w bramách twojich / y będą jeść / y nájedzą śię / ábyć błogoſłáwił PAN Bóg twój w káżdey ſpráwie rąk twojich / którą cżynić będźieƺ.
