Słuchaj Izráelu : ty przejdźieƺ dźiś Jordan / ábyś wƺedƺy pośiadł narody więkƺe / y możniejƺe niżeś ty / miáſtá wielkie y wymurowáne / áż pod niebo.
Lud wielki / y wyſokiego wzroſtu / Syny Enákowe / które ty znaƺ / y o którycheś ſłychał / <i>gdy mówiono :</i> któż śię oſtoji przed Syny Enákowemi.
Przetoż wiedz dźiśiaj / że PAN Bóg twój / który idźie przed tobą / <i>jeſt</i> ogień trawiący : on je wytráći / y on je poniży przed tobą / że je wypędźiƺ / á wygłádźiƺ je prętko / jákoć to obiecał PAN.
Nie mówże w ſercu ſwym / gdy je wypędźi PAN Bóg twój przed tobą / powiádájąc : dla ſpráwiedliwośći mojey / wprowádźił mię PAN / ábym pośiadł tę źiemię / ále dla niezbożnośći Narodów onych / PAN wygnał je przed twarzą twoją.
Nie dla ſpráwiedliwośći twojey / áni dla proſtośći ſercá twego / ty wnidźieƺ / ábyś pośiadł źiemię ich / ále dla niezbożnośći Narodów tych / PAN Bóg twój wypędza ich przed twarzą twoją / y áby utwirdźił PAN ſłowo / które przyśiągł Ojcom twojim / Abráhámowi / Izáákowi y Jákóbowi.
Wiedzże tedy / że nie dla ſpráwiedliwośći twojey / PAN Bóg twój dáje tobie tę wyborną źiemię / ábyś ją pośiadł / gdyżeś ty lud twárdego kárku.
Pámiętaj / á nie zápominaj / żeś do gniewu pobudzał PAná Bogá twego ná puƺcży : od onego dniá / jákoś wyƺedł z źiemie Egipſkiey / áżeśćie przyƺli ná to miejſce / odpornymiśćie byli PAnu.
Tákże y przy górze Horeb pobudźiliśćie do gniewu PAná / y rozgniewał śię PAN ná was / áby was wygłádźił.
Gdym wſtąpił ná górę ábym wźiął tablice kámienne / tablice przymierza / które ucżynił PAN z wámi / y trwałem ná górze cżterdźieśći dni / y cżterdźieśći nocy / chlebá nie jedząc / áni wody pijąc.
Tedy mi dał PAN dwie tablice kámienne / nápiſáne pálcem Bożym / ná których te wƺyſtkie ſłowá były / które mówił PAN do was ná górze / z pośrzodku ogniá / w dźień zgromádzenia wáƺego.
A gdy wyƺło cżterdźieśći dni / y cżterdźieśći nocy / dał mi PAN one dwie tablice kámienne / tablice przymierza.
Y rzekł PAN do mnie : wſtań znidź z tąd rychło / ábowiem śię popſował lud twój / któryś wywiódł z Egiptu : Odſtąpili prętko z drogi / którąm im rozkazał / y ucżynili ſobie litego báłwaná.
Y rzekł PAN do mnie mówiąc : Widźiałem ten lud / á oto lud twárdego kárku jeſt.
Puść mię / á wytrácę je / y wygłádzę imię ich pod niebem / á ćiebie ucżynię w naród możnieyƺy / y więkƺy niżli ten.
Tedym śię obróćił / y zſtąpiłem z góry / ( á górá oná pałáłá ogniem ) dwie tablice przymierza nioſąc w obu rękách mojich.
A gdym ujrzał / iżeśćie zgrzeƺyli przećiw PAnu Bogu wáƺemu / ucżyniwƺy ſobie ćielcá litego / y odſtąpiliśćie byli prętko z drogi / którą wam rozkazał PAN :
Tedy wźiąwƺy ja one dwie tablice / porzućiłem je z obu rąk mojich / á zſtłukłem je przed ocżymá wáƺymi.
Potym upadłem przed PAnem / jáko y pierwey / przez cżterdźieśći dni / y cżterdźieśći nocy / chlebám nie jadł y wodym nie pił / dla wáƺych wƺyſtkich grzechów / któremiśćie zgrzeƺyli / cżyniąc złość przed ocżymá PAńſkiemi / y draźniąc go.
Bom śię bał oney popędliwośći y gniewu / którym śię był zápalił PAN przećiwko wam / áby was wytráćił : y wyſłuchał mię PAN / y tym rázem.
Ná Aároná też rozgniewał śię był PAN bárzo / chcąc go zátráćić : tedym śię też zá Aáronem modlił / tegoż cżáſu.
A grzech wáƺ któryśćie byli ucżynili / <i>to jeſt</i> ćielcá / porwałem / y ſpaliłem go ogniem / y ſkruƺyłem go / tłukąc dobrze / áż śię w proch obróćił / y wrzućiłem proch jego w potok / który płynął z oney góry.
Potym w Táberáh / y w Máſſá / w Kybrot Hátááwá / pobudźiliśćie do gniewu PAná.
A gdy was PAN wyſłał z Kádes Bárne / mówiąc : Idźćie / á pośiądźćie tę źiemię / którąm wam dał : odporniśćie byli ſłowu PAná Bogá wáƺego / ániśćie mu wierzyli / ániśćie uſłucháli głoſu jego.
<i>Owa</i> odporniśćie byli PAnu / od dniá <i>tego</i> jákom was poznał.
Dla tego / upadƺy przed PAnem cżterdźieśći dni / y cżterdźieśći nocy leżałem / bo już był PAN rzekł / że was wytráćić miał.
Y modliłem śię PAnu / y rzekłem : PAnie Boże / nie zátracajże ludu twego / y dźiedźictwá twego / któreś odkupił wielmożnośćią twoją / któreś wywiódł z Egiptu ręką możną :
Wſpomni ná ſłużebniki twoje / Abráhámá / Izááká y Jákóbá : nie pátrz ná zátwárdźiáłość ludu tego / y ná niezbożność jego / áni ná grzech jego.
By ſnadź nie rzekli obywátele źiemie tey / z któreyś nas wywiódł : przeto że nie mógł PAN wprowádźić ich do źiemie / którą im obiecał : álbo iż je miał w nienawiśći / wywiódł je / áby je pobił ná puƺcży.
Abowiem oni ſą ludem twojim / y dźiedźictwem twojim / któreś wywiódł w mocy twojey wielkiey / y w rámieniu twojim wyćiągnionym.
