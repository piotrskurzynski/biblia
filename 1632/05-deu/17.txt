Nie będźieƺ ofiárował PANU Bogu twemu wołu / áni bydlęćiá ná którymby byłá ſkázá / álbo jákakolwiek wádá / gdyż to <i>jeſt</i> obrzydliwośćią PAnu Bogu twemu.
Jesliby śię ználazł miedzy wámi / w którymkolwiek mieśćie twym / które PAN Bóg twój dawa tobie / mąż álbo niewiáſtá / któryby śię cżego złego dopuśćił / przed ocżymá PAná Bogá twego / przeſtępując przymierze jego /
A poƺedƺy / ſłużyłby obcym Bogom / y kłaniałby śię im / álbo ſłońcu / álbo mieśiącowi / álbo wƺyſtkim zaſtępom niebieſkim / cżegom nie przykazał :
Y oznájmionoćby to / á uſłyƺawƺy / wywiádowałbyś śię pilno / á oto byłáby prawdá / y rzecż pewna / że śię ſtáłá tá obrzydliwość w Izráelu :
Tedy wywiedźieƺ onego mężá / álbo onę niewiáſtę / którzy ucżynili tę złą rzecż / do bram twojich / mężá onego / álbo niewiáſtę / y ukámionujeƺ je / á pomrą.
W uśćiech dwu świádków álbo trzech świádków zábit będźie / kto umrzeć ma : niech nie umiera ná ſłowo świádká jednego.
Ręká świádków będźie ná nim napierwƺa / ná zábićie jego : á ręká wƺyſtkiego ludu potym : á ták odejmieƺ złe z pośrzodku śiebie.
Byłoliby też co trudnego przy ſądźie miedzy krwią á krwią / miedzy ſpráwą á ſpráwą / y miedzy ráną á ráną / <i>y</i> około poſwarków / w bramách twojich / tedy wſtánieƺ / á pójdźieƺ ná miejſce które obierze PAN Bóg twój /
Y przyjdźieƺ do Kápłanów Lewitów / y do ſędźiego / który będźie ná on cżás / y będźieƺ ich pytał / á oznájmiąć jáko to oſądźić maƺ.
Y ucżyniƺ według tego jákoć powiedzą / z miejſcá tego / które obrał PAN / y poſtąpiƺ ſobie według wƺyſtkiego jáko ćię náucżą.
Według ſłów Zakonu / którego ćię náucżą / y według ſądu / któryć powiedzą / ſpráwowáć śię będźieƺ : nie uſtąpiƺ od ſłowá / któreć powiedzą / áni ná práwo / áni ná lewo.
A jesliby kto ucżynił <i>to</i> z hárdośći / żeby nie chćiał uſłucháć Kápłaná tám poſtánowionego / ku ſłużbie przed PAnem Bogiem twojim / álbo ſędźiego / niech umrze on mąż : y odejmieƺ to złe z Izráelá :
Aby wƺyſtek lud uſłyƺawƺy bał śię / á więcey hárdźie ſobie nie pocżynał.
Gdy wnidźieƺ do źiemie / którą PAN Bóg twój dawa tobie / á opánujeƺ ją / y będźieƺ mieƺkał w niey / y rzecżeƺ : Poſtánowię nád ſobą Królá / jáko <i>y</i> wƺyſtkie Narody okolicżne.
Tego tylko poſtánowiƺ nád ſobą Królá / którego obierze PAN Bóg twój : z pośrzodku bráći twey / poſtánowiƺ nád ſobą Królá : Nie będźieƺ mógł przełożyć nád ſobą cudzoźiemcá / któryby nie był brátem twojim :
Tylko niech nie chowa wiele koni / áni náwraca ludu do Egiptu / áby wiele koni nábył : ponieważ wam PAN rzekł / nie wracajćie śię záś tą drogą więcey.
Nie będźie też miał wiele żon / áby śię nie odwróćiło ſerce jego : śrebrá też y złotá / niech názbyt wiele nie nábywa.
A gdy uśiędźie ná ſtolicy Króleſtwá ſwego / przepiƺe ſobie ten powtórzony Zakon w kśięgi / od Kápłanów Lewitów :
Y będźie go miał przy ſobie / á będźie go cżytał po wƺyſtkie dni żywotá ſwego / áby śię ucżył bać PAná Bogá ſwego / y przeſtrzegał / wƺyſtkich ſłów Zakonu tego / y uſtaw tych / á cżynił je.
Aby śię nie wynośiło ſerce jego / nád bráćią jego / y żeby śię nie unośiło od tego przykazánia ná práwo / áni ná lewo : áby długo żył ná króleſtwie ſwojim / on y ſynowie jego w pośrzodku Izráelá.
