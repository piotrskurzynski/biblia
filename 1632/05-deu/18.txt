Nie będą mieli Kápłani Lewitowie / y wƺyſtko pokolenie Lewi / dźiału / áni dźiedźictwá / z <i>innym</i> Izráelem : Ogniſte Ofiáry PAńſkie / y dźiedźictwo jego jeść będą.
A dźiedźictwá nie będą mieli miedzy bráćią ſwoją / PAN jeſt dźiedźictwem ich / jáko im powiedźiał.
A toć będźie práwo <i>należące</i> Kápłanom od ludu / od ofiárujących ofiárę / bądź wołu / bądź owcę : tedy oddádzą Kápłanowi łopátkę y cżeluśći / y káłdun :
Pierwoćiny zbożá twego / winá twego / y oliwy twojey / tákże pierwoćiny wełny z Owiec twojich / oddaƺ mu.
Abowiem obrał go PAN Bóg twój / ze wƺyſtkich pokoleń twojich / áby ſtał ku uſłudze w imieniu PAńſkim / on / y Synowie jego po wƺyſtkie dni.
A gdyby przyƺedł Lewitá / z któregokolwiek miáſtá twego / z cáłego Izráelá / gdźie przemieƺkiwa / á przyƺedłby z cáłey chęći duƺe ſwey / ná miejſce które ſobie obrał PAN :
Tedy ſłużyć będźie w imieniu PAná Bogá ſwego / jáko wƺyſcy bráćia jego Lewitowie / którzy tám ſtoją przed oblicżnośćią PAńſką :
Cżęść równą <i>z drugimi</i> jeść będą / oprócż tego co im należáło z dóbr Ojców ich.
Gdy tedy wnidźieƺ do źiemie / którą dawa PAN Bóg twój tobie / nie ucż śię cżynić / według obrzydliwośći tych narodów.
Niech śię miedzy wámi nie znájduje / któryby przewodźił Syná ſwego / álbo córkę ſwoję przez Ogień : <i>tákże</i> wieƺcżek / guſlarz y wróżek / y cżárownik /
Y Cżárnokśiężnik / y ten który ma ſpráwę z duchy złymi / y práktikarz / y wywiádujący śię <i>cżego</i> od umárłych.
Abowiem jeſt obrzydliwośćią PAnu káżdy / któryby to cżynił : bo dla tych obrzydliwośći / PAN Bóg twój / wyrzuca te <i>Narody</i> przed tobą.
Doſkonáłym będźieƺ przed PAnem Bogiem twojim.
Abowiem Narodowie ći / które ty opánujeƺ / wieƺcżków y guſlarzów ſłucháją : ále tobie nie dopuƺcża tego PAN Bóg twój.
Proroká z pośrżodku ćiebie / z bráćij twey / jákom ja <i>jeſt,</i> wzbudźi tobie PAN Bóg twój : onego ſłucháć będźiećie :
Według tego wƺyſtkiego / jákoś żądał od PAná Bogá twego / ná <i>górze</i> Horeb / w dźień zgromádzenia / mówiąc : niech więcey nie ſłucham głoſu PAná Bogá mego / y ná Ogień ten wielki niech nie pátrzę więcey / bym nie umárł.
Przetoż rzekł PAN do mnie : Dobrze mówili / Co mówili :
Proroká im wzbudzę z pośrzodku bráći ich / jákoś ty <i>jeſt,</i> y włożę ſłowá moje w uſtá jego / y opowiádáć im będźie wƺyſtko cokolwiek mu rozkażę.
Y ſtánie śię / że ktobykolwiek nie był poſłuƺny ſłowom mojim / które <i>on</i> mówić będźie w imię moje / ja <i>tego</i> ƺukáć będę ná nim.
Wƺákże Prorok któryby <i>ſobie</i> hárdźie pocżął / mówiąc ſłowo / w imieniu mojim / któregom mu mówić nie rozkazał / y któryby mówił w imię Bogów obcych / niech umrze Prorok tákowy.
A jeſlibyś rzekł w ſercu ſwym : Jákoż rozeznamy to ſłowo / którego nie mówił PAN?
Jeſliby co mówił on Prorok w imię PAńſkie / á nie ſtáłoby śię to áni wypełniło / onoć to jeſt ſłowo / którego nie mówił PAN / z hárdośći to mówił Prorok on : nie bójże śię go.
