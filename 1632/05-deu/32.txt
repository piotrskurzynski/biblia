Słuchajćie Niebioſá / á mówić będę : niech ſłucha y źiemiá / wymowy uſt mojich.
Kropić będźie jáko deƺcz náuká mojá / popłynie jáko roſá wymowá mojá / jáko drobny deƺcż ná źiołá / á jáko krople ná trawę.
Abowiem imienia PAńſkiego będę wzywał : dajćież wielmożność Bogu náƺemu :
Skále / którey ſpráwy ſą doſkonáłe : bo wƺyſtkie drogi jego ſą ſpráwiedliwe : Boć <i>jeſt</i> Bóg prawdźiwy / á niemáƺ w nim niepráwośći / ſpráwiedliwy y práwy jeſt.
Pokáźili mu śię / jákoby nie byli Synámi jego / ſproſnośćią ſwoją : Naród zły á przewrotny.
Y tákże to oddawaćie PAnu / ludu głupi y ƺalony? izali nie on jeſt Ojcem twojim / który ćię ſobie nábył / on ćię ucżynił y ſtworzył ćię.
Wſpomni ná dni dawne / uważajćie látá káżdego wieku : Spytaj Ojcá twego / á oznájmi tobie : ſtárƺych twojich / á powiedząć.
Gdy rozdźielił dźiedźictwá Nawyżƺy Narodom / á rozſádźił Syny Adámowe / położył gránice narodom / według licżby Synów Izráelſkich :
Abowiem dźiałem PAńſkim <i>jeſt</i> lud jego / Jákób ƺnurem dźiedźictwá jego :
Ználazł go w źiemi puſtey / y ná puƺcży wielkiey y ſtráƺney : obwodźił go / ucżył go / ſtrzegł go jáko źrzenice oká ſwego.
Jáko Orzeł wybawia Orlętá ſwoje / nád Orlętámi ſwojimi latájąc rozƺerza ſkrzydłá ſwoje / bierze je / <i>y</i> nośi je ná ſkrzydłách ſwojich.
<i>Ták</i> PAN ſam prowádźił go / á żaden obcy Bóg nie był z nim.
Wynióſł go ná wyſokie miejſcá źiemie / áby używał urodzájów polnych / y ucżynił áby ſał miód z ſkáły / y oliwę z opoki twárdey :
Aby jadł máſło od krów / y mleko od owiec z tłuſtośćią jágniąt / y bárány <i>odchowáne w</i> Báſán / y kozły z tłuſtośćią źiarn pƺenicżnych / á pił cżerwone wino / wyborne.
Y roztył Izráel y wierzgał : ( otyłes / zſtłuśćiałeś / zgrubiałeś / y opuśćiłeś Bogá / który ćię ucżynił : ) y lekce poważył <i>Bogá</i> opokę zbáwienia ſwego.
Ku zápálcżywośći pobudźili go przez cudze <i>Bogi,</i> y przez obrzydliwośći rozdraźnili go.
Ofiárowáli Diabłom nie Bogu / Bogom których nie ználi nowym / <i>którzy</i> z bliſká przyƺli / których śię nie bali Ojcowie wáƺy.
Opoki która ćię ſpłodźiłá zápomniałeś / zápomniałeś Bogá ſtworzyćielá twego.
Co widząc PAN / poruƺył śię gniewem ná Syny ſwoje / y ná Córki ſwoje.
Y rzekł : ſkryję oblicze moje przed nimi / obacżę jáki będźie koniec ich : ábowiem ſą Narodem przewrotnym : Synmi w których wiáry niemáƺ.
Oni mię wzruƺyli ku zápálczywośći / przez <i>to, co</i> nie jeſt Bogiem / pobudźili mię próznośćiámi ſwymi : ja też do zazdrośći pobudzę je / przez te <i>którzy</i> nie ſą ludem : przez naród głupi rozdraźnię je.
Abowiem ogień roſpalił śię w popędliwośći mojey : y będźie gorzał / áż do nagłębƺego piekłá / y pożrze źiemię / y urodzaj jey / y wypali grunty gór.
Zgromádzę ná nie złe <i>rzecży,</i> ſtrzały moje ná nie wyſtrzelam.
Zniƺcżeją od głodu / y <i>będą</i> ſtrawieni gorączką / y śmierćią gorzką : zęby też beſtyi poślę ná nie / z jádem gádźin źiemſkich.
Zewnątrz ośieroći je miecż / á w pokojách <i>będźie</i> ſtrách / ták ná młodźieńcá / jáko ná pánnę : ná ſącego pierśi / y ná mężá ſędźiwego.
Rzekłem : Roſproƺę je po kąciech / y wygubię miedzy ludźmi pámiątkę ich.
Gdybym śię ná pychę / nieprzyjaćiół nie oglądał by śię ſnadź nie podnieśli nieprzyjaćiele ich / á nie rzekli : Ręká náƺá wynioſła / á nie PAN / ſpráwiłá to wƺyſtko.
Abowiem ten Naród ginie przez rády <i>ſwe</i> y niema rozumu.
O by mądrymi byli! wyrozumieliby to / y oglądáliby śię ná oſtátnie rzecży ſwoje.
Jákoby uganiał jeden z nich tyśiąc! á dwáj goniliby dźieśięć tyśięcy! gdyby ich <i>Bóg</i> opoká ich nie záprzedał / y PAN ich nie wydał :
Abowiem nie jáko ſkáłá náƺá / ſkáłá ich / co nieprzyjaćiele náƺy <i>ſámi</i> oſądzą.
Abowiem z máćice Sodomſki / máćicá ich / y z látorośli Gomorſkich / jágody ich / jágody jádowite : groná ich gorzkie.
Jad ſmoczy wino ich / y trućizną żmijów okrutną.
Izali to nie jeſt ſkryto u mnie / zápieczętowano w ſkárbie mojim?
Mojáć jeſt pomſtá y nagrodá : cżáſu <i>ſwego</i> powinie śię nogá ich : ponieważ bliſki jeſt dźień zginienia ich / á ſpieƺą śię przyƺłe rzecży do nich.
Abowiem ſądźić będźie PAN lud ſwój / á ſług ſwojich użali śię : gdy ujrzy / że uſtáłá śiłá / á <i>iż ták</i> pojmány / jáko zoſtáwiony nic nie mogą :
Y rzecże : gdźież Bogowie ich / <i>oná</i> opoká w którey ufáli?
Którzy tłuſtość Ofiar ich jadáli / <i>y</i> pijáli wino Ofiar ich mokrych : Niechże wſtáną / y rátują was / y niechaj będą ućiecżką wáƺą.
Obaczćież teraz żem ja / jam <i>jeſt</i> ſam / á że niemáƺ Bogów oprócz nnie : ja zábijam y ożywiam / zránię / y ja uleczę / á niemáƺ ktoby z ręki mojey wyrwał :
Abowiem podnioſę ku Niebu rękę moję / y rżekę : Żyję ja ná wieki.
Jeſli záoſtrzę błyƺcżący śię miecz ſwój / y porwie ſąd ręká mojá / oddam pomſtę nieprzyjaćiołom mojim / y tym którzy mię nienawidzą nádgrodzę.
Opoję ſtrzały moje krwią / á miecz mój náje śię mięſá / ze krwie pobitych y pojimánych / ſkoro zácznę mśćić śię nád nieprzyjaćioły.
Weſelćie śię Narodowie z ludem jego : ábowiem krwie ſług ſwojich pomśći śię / y pomſtę odda nieprzyjaćiołom ſwojim / á miłośćiw będźie źiemi ſwojey / <i>y</i> ludowi ſwemu.
Tedy przyƺedł Mojzeƺ / y mówił wƺyſtkie ſłowá pieśni tey / w uƺách ludu tego / <i>on</i> y Jozue Syn Nunów.
A <i>gdy</i> dokonał Mojzeƺ mówić tych wƺyſtkich ſłów do wƺyſtkiego Izráelá :
Rzekł do nich : Przyłóżćie ſercá wáƺe do wƺyſtkich ſłów / którymi śię ja oświadcżam przećiwko wam dźiś / ábyśćie je przykázowáli Synom wáƺym / żeby ſtrzegli y pełnili wƺyſtkie ſłowá zakonu tego :
Abowiem <i>to</i> ſłowo nie ma bydź prózne u was / gdyż jeſt żywotem wáƺym / y w tym ſłowie przedłużyćie dni ná źiemi / do którey śię przeprawiáćie przez Jordan / ábyśćie ją ośiedli.
Y rzekł PAN do Mojzeƺá / tegoż dniá / mówiąc :
Wſtąp ná tę górę Abárim / ná górę Nebo / która jeſt w źiemi Moábſkiey / przećiw Jerichu : á oglądaj źiemię Chánánejſką / którą ja dawam Synom Izráelſkim w ośiádłość.
Y umrżeƺ ná tey górze ná którą wſtąpiƺ : y będźieƺ przyłącżony do ludu twego / jáko umárł Aáron brát twój / ná górze Hor / y przyłącżon jeſt do ludu ſwego :
Dla tego / żeśćie wyſtąpili przećiwko mnie / w pośrzód Synów Izráelſkich / przy wodách w poſwarku / w Kádes / ná puƺcży Syn : Y żeśćie mię nie poświęćili w pośrzód Synów Izráelſkich.
Bo przed ſobą oglądaƺ źiemię : ále tám nie wnidźieƺ / do tey źiemie / którą ja dáję Synom Izráelſkim.
