Tedy przykazał Mojzeƺ y Stárƺy Izráelſcy ludowi / mówiąc ; Strzeżćie wƺelkiego przykazánia które ja wam dźiś przykázuję.
Y ſtánie śię / że którego dniá przejdźiećie przez Jordan do źiemie / którąć PAN Bóg twóy dawa / á wyſtáwiƺ ſobie kámienie wielkie / y potynkujeƺ je wapnem /
Y nápiƺeƺ ná nich wƺyſtkie ſłowá Zakonu tego / ſkoro przejdźieƺ / ábyś wƺedł do źiemie / którą PAN Bóg twój dawa tobie : do źiemie opływájącey mlekiem y miodem / jákoć obiecał PAN Bóg Ojców twojich.
Gdy tedy przejdźiećie przez Jordan / wyſtáwićie te kámienie / o których ja wam dźiś rozkázuję / ná górze Hebál / y potynkujeƺ je wapnem.
Támże zbudujeƺ Ołtarz PAnu Bogu twemu / Ołtarz z kámienia : nie będźieƺ ich ćioſał żelazem /
Z kámienia cáłego zbudujeƺ Ołtarz PAná Bogá twego : y ofiárowáć będźieƺ ná nim cáłopalenia PAnu Bogu twemu.
Tákże ofiárowáć będźieƺ ofiáry ſpokojne / y będźieƺ tám jadł / á będźieƺ śię weſelił przed oblicżem PAná Bogá twego :
Y nápiƺeƺ ná onych kámieniách / wƺyſtkie ſłowá Zakonu tego / znácżnie y jáśnie.
Y rzekł Mojzeƺ / y Kápłani Lewitowie / do wƺyſtkiego Izráelá / mówiąc : pilnuj / á ſłuchaj Izráelu : dźiś ſtałeś śię ludem PAná Bogá twego /
Przetoż będźieƺ poſłuƺnym głoſowi PANA Bogá twego / á będźieƺ cżynił przykazánia jego / y uſtáwy jego które ja dźiś przykázuję tobie.
Tedy przykazał Mojzeƺ ludowi dniá onego / mówiąc :
Ci ſtáną áby błogoſłáwili ludowi ná górze Gáryzim / gdy przejdźiećie przez Jordan / Symeon y Lewi / y Judá / y Iſáſchár / y Józef / y Benjámin.
Ci záś ſtáną ku przeklinániu ná górze Hebál : Ruben / Gád / y Aſer / y Zábulon / Dán / y Neftáli.
Y oświadcżą śię Lewitowie / á mówić będą do wƺyſtkich mężów Izráelſkich głoſem wynioſłym :
Przeklęty cżłowiek / któryby cżynił obraz ryty / álbo odlewány / obrzydliwość PAńſką / robotę rąk rzemieśnicżych / choćby to poſtáwił ná miejſcu ſkrytym : Tedy odpowie wƺyſtek lud / y rzecże : Amen.
Przeklęty kto lekce waży Ojcá ſwego / y mátkę ſwoję : y rzecże wƺyſtek lud / Amen.
Przeklęty który przenośi gránicę bliźniego ſwego : á rzecże wƺyſtek lud / Amen.
Przeklęty który zwodźi ślepego z drogi áby błądźił : y rzecże wƺyſtek lud / Amen.
Przeklęty który wywraca ſpráwiedliwość przychodniowi / śieroćie y wdowie : á rzecże wƺyſtek lud / Amen.
Przeklęty ktoby obcował z żoną Ojcá ſwego / bo odkrył podołek Ojcá ſwego : y rzecże wƺyſtek lud / Amen.
Przeklęty któryby obcował z jákimkolwiek bydlęćiem : y rzecże wƺyſtek lud / Amen.
Przeklęty któryby obcował z śioſtrą ſwoją / córką Ojcá ſwego / álbo córką mátki ſwojey : y rzecże wƺyſtek lud / Amen.
Przeklęty któryby obcował z świekrą ſwoją : y rzecże wƺyſtek lud / Amen.
Przeklęty któryby zábił tájemnie bliźniego ſwego : y rzecże wƺyſtek lud / Amen.
Przeklęty któryby brał dáry / áby zábił cżłowieká / krew niewinną : y rzecże wƺyſtek lud / Amen.
Przeklęty ktoby niezoſtał przy ſłowiech Zakonu tego / áby je cżynił : y rzecże wƺyſtek lud / Amen.
