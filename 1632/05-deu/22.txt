Jeslibyś ujrzał wołu brátá twego álbo Owcę jego obłąkáną / nie mijayże ich / <i>ále</i> je koniecżnie odprowadź do brátá twego.
Choććiby nie był bliſki brát twój / ánibyś go znał / przećię zápędźiƺ je do domu ſwego / y będźie przy tobie / áżby tego ƺukał brát twój / y wróćiƺ mu je.
Toż ucżyniƺ Oſłowi jego : toż też ucżyniƺ ƺáćie jego : toż też ucżyniƺ wƺelákiey rzecży zgubioney brátá twego / któraby mu zginęłá / jeslibyś ją nálazł / nie mijajże jey.
Jeslibyś ujrzał Oſłá brátá twego / álbo wołu jego / że upadł ná drodze / nie mijajże ich / ále go záraz z nim podnieś.
Niech niebierze niewiáſtá ná śię ƺat męſkich : áni niech śię nie obłócży mąż w ƺátę niewieśćią : ábowiem jeſt obrzydliwośćią PAnu Bogu twemu ktoby to cżynił.
Gdybyś tráfił gniazdo ptáƺe przed ſobą w drodze / ná jákimkolwiek drzewie / álbo ná źiemi / <i>á w nim</i> ptaƺętá álbo jáicá : á mátkáby śiedźiáłá ná ptaƺętách / álbo ná jáicách : nie bierzże máćierze z dźiećmi :
Ale wolno puśćiwƺy mátkę / dźieći weźmieƺ ſobie / ábyć śię dobrze dźiało / y żebyś przedłużył dni <i>twojich</i>.
Gdy też zbudujeƺ dom nowy / tedy ucżyniƺ blánki po kráju dáchu twego / ábyś nie przywiódł krwie ná dom twój / gdyby kto ſpadł z niego.
Nie pośiewaj winnice twojey różnym naśieniem / byś ſnadź nie ſplugáwił pożytku naśienia któreś śiał / y urodzáju winnice.
Nie będźieƺ Orał wołem y Oſłem poſpołu.
Nie oblecżeƺ ƺáty utkáney z wełny y ze lnu poſpołu.
Pocżyniƺ ſobie ƺnurki ná cżterech rogách ƺáty twojey / którą śię odźiewáć będźieƺ.
Gdyby kto pojął żonę / á wƺedƺy do niey / miałby ją w nienawiśći /
A dałby przycżynę / áby mówiono o niey / y wprowádźiłby ná nię złą ſławę / mówiąc : Pojąłem tę żonę / á wƺedƺy do niey nie ználazłem jey Pánną :
Tedy weźmie Oćiec dźiewecżki / y Mátká jey / y przynioſą <i>znáki</i> pánieńſtwá dźiewecżki oney / do ſtárƺych miáſtá onego do bramy /
Y rzecże Oćiec oney dźiewecżki do ſtárƺych : córkę ſwoję dałem mężowi temu zá żonę / á on ją ma w nienawiśći.
A oto ſam dał przycżynę / áby mówiono o niey / powiádájąc : nie ználazłem przy córce twojey Pánieńſtwá : ále oto <i>ſą</i> znáki Pánieńſtwá / córki mojey : Tedy rozwiną ono prześćierádło przed ſtárƺymi miáſtá onego :
A wźiąwƺy ſtárƺy miáſtá / onego mężá / ſkarzą go.
Y wezmą z niego winę / Sto śrebrników / które oddádzą Ojcu oney dźiewecżki / przeto / że puśćił złą ſławę o Pánnie Izráelſkiey / y będźie ją miał zá żonę / y nie będźie jey mógł opuśćić / po wƺyſtkie dni ſwoje.
Ale byłoliby prawdźiwe to obwinienie jey / <i>á</i> nie ználázłyby śię <i>znáki</i> Pánieńſtwá przy oney dźiewecżce :
Tedy wywiodą onę dźiewecżkę przed drzwi domu Ojcá jey / y ukámionują ją ludźie miáſtá onego / y umrze : bo ucżyniłá háńbę w Izráelu / popełniwƺy nierząd w domu Ojcá ſwego : ták odejmieƺ złe z pośrzód śiebie.
Jeſliby kto ználeźion był obcujący z Niewiáſtą májącą mężá / tedy niech umrą oboje / mąż który obcował z <i>cudzą</i> żoną / y oná niewiáſtá : ták odejmieƺ złe z Izráelá.
Byłáliby dźiewecżká / Pánná poślubiona mężowi / á tráfiłby ją kto w mieśćie / y obcowałby z nią :
Tedy wywiódƺy ono oboje przed bramę miáſtá / ukámionujećie je / y umrą : dźiewecżkę przeto / iż nie wołáłá w mieśćie : á mężá przeto / iż zelżył żonę bliźniego ſwego : ták odejmieƺ złe z pośrzodku śiebie.
A jeſliby ná polu tráfił mąż dźiewecżkę poślubioną / á porwawƺy ją / zgwałćiłby ją / tedy umrze mąż / który obcował z nią / ſam tylko.
Ale dźiewecżce nic nie ucżyniƺ : dźiewecżká nie jeſt winná śmierći / bo jáko gdyby kto powſtawƺy przećiwko bliźniemu ſwemu zámordował go : táka to ſpráwá.
Ponieważ ją ná polu tráfił / <i>á gdy</i> wołáłá dźiewecżká poślubiona / nie był ktoby ją rátował.
Gdyby też kto / tráfiwƺy dźiewecżkę Pánnę / która nie jeſt zá mąż zmówiona porwał ją / y leżałby z nią / á záſtanoby je :
Tedy da on mąż który obcował z nią / Ojcu dźiewecżki pięćdźieśiąt śrebrników / y będźie mu zá żonę / przeto że ją zelżył / áni jey będźie mógł opuśćić / po wƺyſtkie dni ſwoje :
Nie pojmie nikt żony Ojcá ſwego / y nie odkryje podołká Ojcá ſwego.
