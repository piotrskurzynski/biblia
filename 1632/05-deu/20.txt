Gdy śię ruƺyƺ ná wojnę / przećiw nieprzyjaćiołom twojim / á obacżyƺ konie y wozy / <i>y</i> lud więkƺy nád ćię / nie bój śię ich : bo PAN Bóg twój z tobą <i>jeſt</i> / który ćię wywiódł z źiemie Egipſkiey.
A gdy śię przybliżáć będźiećie ku potykániu / wyſtąpi Kápłan / y przemówi do ludu :
A rzecże do nich : ſłuchaj Izráelu : wy dźiś ſtacżaćie bitwę z nieprzyjaćioły wáƺemi / niechajże nie ſłábieje ſerce wáƺe / nie bójćie śię / y nie trwożćie ſobą / áni śię ich lękajćie :
Abowiem PAN Bóg wáƺ idźie z wámi / áby walcżył zá was z nieprzyjaćioły wáƺymi / á yżby was wybáwił.
Tákże mówić będą Hetmáni do ludu / y rzeką : Jeſliż kto jeſt / co zbudował dom nowy / á w nim nie pocżął mieƺkáć / niech idźie / á wróći śię do domu ſwego : by ſnadź nie zginął w bitwie / á kto inny pocżąłby w nim mieƺkáć.
Albo / jesli kto jeſt / co náſádźił winnicę / á nie używał owocu z niey / niech idźie / á wróći śię do domu ſwego / by ſnadź nie zginął w bitwie / á kto inny używałby z niey.
Albo jesli kto jeſt / co ma poślubioną żonę / á jeƺcże jey nie pojął / niech idźie / á wróći śię do domu ſwego / by ſnadź nie zginął w bitwie / á kto inny nie pojął jey.
Nád to Hetmáni mówić będą do ludu / y rzeką : Jesli kto jeſt bojáźliwy / á lękliwego ſercá / niech idźie / á wróći śię do domu ſwego / áby nie pſował ſercá bráći ſwey / jáko <i>jeſt zepſowáne</i> ſerce jego.
A gdy przeſtáną Hetmáni mówić do ludu / tedy poſtáwią Rotmiſtrze wojſk przed ludem.
Gdy przyćiągnieƺ do <i>jákiego</i> miáſtá / ábyś go dobywał / pocżęſtujeƺ je pokojem.
A jesliże pokój Ofiárowány przyjmie / y otworzyć <i>bramy,</i> tedy wƺyſtek lud náleźiony w nim / będźie hołdował y ſłużył.
Ale jesli nie ucżyni pokoju z tobą / lecż podnieśie przećiw tobie wojnę / oblężeƺ je.
A gdy je da PAN Bóg twój w rękę twoję / tedy zábijeƺ w nim káżdego mężcżyznę oſtrzem miecżá.
Tylko niewiáſty / y dźiatki / y bydłá / y wƺyſtko co będźie w mieśćie / wƺyſtek łup jego obróćiƺ ſobie w korzyść / y będźieƺ jadł łupy nieprzyjaćiół twojich / które da PAN Bóg twój tobie.
Ták ucżyniƺ wƺyſtkim miáſtom / dáleko odległym od ćiebie / które nie ſą z miaſt tych narodów.
Ale z miaſt narodów tych / które PAN Bóg twój podawa tobie w dźiedźictwo / żadney duƺe żywić nie będźieƺ :
Lecż do ƺcżętu wytráćiƺ je / Hettejcżyká / Amorrejcżyká / Chánánejcżyká / y Ferezejcżyká / Hewejcżyká / y Jebuzejcżyká / jákoć rozkazał PAN Bóg twój.
Przeto / żeby nie ucżyli was cżynić / według wƺyſtkich obrzydliwośći ſwojich / które cżynili Bogom ſwym / y zgrzeƺylibyśćie przećiw PAnu Bogu wáƺemu.
Gdy oblężeƺ jákie miáſto / á przez wiele dni dobywáć go będźieƺ / ábyś je wźiął / nie pſuj drzewá jego / wyćinájąc je śiekierą / bo z niego jeść będźieƺ : przetoż go nie wyćinaj / ( bo / ázaƺ ſą ludźmi drzewá polne? ) ábyś ich używał do oblężenia.
Tylko drzewá / które obacżyƺ / że owocu nie rodzą / te pſowáć będźieƺ y wyćináć : y budowáć będźieƺ báƺty / przećiwko onemu miáſtu / które z tobą walcży / póki go nie opánujeƺ.
