Teć <i>ſą</i> ſłowá które mówił Mojzeƺ do wƺyſtkiego Izráelá / przed Jordanem ná puƺcży / ná równinie przećiw Morzu cżerwonemu / miedzy Fárán / y miedzy Tofel / y Lábán / y Háſeroth / y Dizáháb.
A jeſt jedenaśćie dni drogi od Horebu / <i>przez</i> górę Seir / áż do Kádes Bárne.
Y ſtáło śię cżterdźieſtego roku / jedenaſtego mieśiącá / pierwƺego dniá <i>tegoż</i> mieśiącá / że opowiedźiał Mojzeƺ Synom Izráelſkim to wƺyſtko / co mu był rozkazał PAN do nich :
Gdy poráźił Sehoná królá Amorrejſkiego / który mieƺkał w Heſebon / y Ogá królá Báſáńſkiego / który mieƺkał w Aſtárot w Edrey.
Przed Jordanem w źiemi Moábſkiey pocżął Mojzeƺ wykłádáć ten Zakon / mówiąc :
PAN / Bóg náƺ mówił do nas ná <i>górze</i> Horeb / y rzekł : Doſyćieśćie mieƺkáli ná tey górze /
Obróććież śię / á ćiągnąc idźćie do góry Amorrejſkiey / y ná wƺyſtkie miejſcá pogránicżne jey / ná równiny / ná góry / y ná doliny / y ku południowi / y ku brzegu morſkiemu / ku źiemi Chánánejſkiey / y do Libánu / áż do wielkiey rzeki / rzeki Eufrátes :
Oto dałem wam tę źiemię / wnidźćieƺ / á pośiądźćie tę źiemię / o którą przyśiągł PAN Ojcom wáƺym / Abráhámowi / Izáákowi y Jákóbowi / iż im ją dáć miał / y naśieniu ich po nich.
Y rzekłem do was ná on cżás / mówiąc : nie mogę ſam nośić was :
PAN Bóg wáƺ rozmnożył was : á otośćie dźiś jáko gwiázdy niebieſkie w mnóſtwie <i>ſwojim</i>.
PAN Bóg Ojców wáƺych / niech rozmnoży was nád to / jákośćie <i>teraz,</i> tyśiąc kroć więcey : y niech was błogoſłáwi jáko wam obiecał.
Jákoż ſam znośić mam ćięƺką pracę około was / y brzemię wáƺe / y ſwary wáƺe?
Obierzćie z śiebie męże mądre y umiejętne / á doświadcżone w pokoleniách wáƺych / á poſtánowię je przełożonymi nád wámi.
Tedyśćie mi odpowiedźieli / y rzekliśćie : Dobra rzecż / którąś rozkazał ucżynić.
Y obrałem przedniejƺe z pokoleń wáƺych / męże mądre / y doświadcżone / y poſtánowiłem je przełożonymi nád wámi : Tyśiącżniki / y Setniki / y pięćdźieśiątniki / y dźieśiątniki / y rządce w pokoleniách wáƺych.
Y rozkazałem ſędźiom wáƺym ná on cżás / mówiąc : wyſłuchywajćie <i>ſpraw</i> miedzy bráćią wáƺą / y ſądźćie ſpráwiedliwie / miedzy mężem y miedzy brátem jego / y miedzy przychodniem jego :
Nie miejćie względu ná Oſoby przy ſądźie : ták máłego jáko wielkiego / wyſłuchywajćie : nie bójćie śię nikogo / ábowiem ſąd Boży jeſt : á jeſliby co przytrudniejƺym było ná was / odnieśiećie do mnie / á <i>ja</i> tego przeſłucham.
Y przykazałem wam ná on cżás wƺyſtko cośćie cżynić mieli.
Potym ruƺywƺy śię z Horeb / przeƺliſmy wƺyſtkę onę puſtynią wielką y ſtráƺną : Którąśćie widźieli idąc drogą ku górze Amorrejſkiey / jáko nam był rozkazał PAN Bóg náƺ : y przyƺliſmy áż do Kádes Bárne.
Y rzekłem do was : Przyƺliśćie áż do góry Amorrejcżyká / którego PAN Bóg náƺ dawa nam.
Oto podawa PAN Bóg twój tobie tę źiemię : idźże / á pośiądź <i>ją,</i> jákoć powiedźiał PAN Bóg Ojców twojich / nie bój śię áni śię lękaj.
Y przyƺliśćie do mnie wƺyſcy / á mówiliśćie : Poślimy męże / przed ſobą którzyby nam przeƺpiegowáli źiemię / y dáli nam pewną ſpráwę / którąbyſmy drogą do niey wniść mieli / y miáſtá / do którychbyſmy weƺli.
Co gdy mi śię podobáło / obrałem z was dwánaśćie mężów / po jednym mężu / z káżdego pokolenia.
Którzy odƺedƺy weƺli ná górę / á przyƺli áż do doliny Eſchol / y przeƺpiegowáli źiemię.
Nábráli też z ſobą owocu oney źiemie / y przynieśli do nas / y dáli nam o tym ſpráwę / á powiedźieli : Dobra jeſt źiemiá / którą nam dawa PAN Bóg náƺ.
Aleśćie nie chćieli iść / lecżeśćie byli odpornymi ſłowu PAná Bogá wáƺego.
Y ƺemráliśćie w Namiećiech wáƺych / mówiąc : Yż nas PAN <i>miał</i> w nienawiśći / wywiódł nas z źiemie Egipſkiey / áby nas podał w ręce Amorrejcżyká / y wygubił nas.
Dokądże iść mamy? Bráćia náƺy zepſowáli nam ſerce / mówiąc : Lud ten więkƺy y roſlejƺy jeſt / niżli my : miáſtá wielkie / y wymurowáne áż ku niebu : á nád to / Syny Obrzymów / widźieliſmy tám.
Alem <i>ja</i> mówił do was : Nie lękájćie śię áni śię ich bójćie :
PAN Bóg wáƺ / który idźie przed wámi / walcżyć będźie zá was / ták jáko wam ucżynił w Egipćie przed ocżymá wáƺymi.
Tákże y ná puƺcży / gdźieś widźiał / jáko ćię nośił PAN Bóg twój / jáko nośi Oćiec Syná ſwego / w oney wƺyſtkiey drodze którąśćie ƺli / áżeśćie przyƺli ná to miejſce.
Lecż áni ták uwierzyliśćie PAnu Bogu wáƺemu /
Który chodźił przed wámi / w drodze / przepátrując wam miejſce / do ſtánowienia obozu / w nocy w Ogniu / áby wam ukazał drogę / którąbyśćie iść mieli : we dnie záś w Obłoku.
Y uſłyƺał PAN głos ſłów wáƺych / á rozgniewał śię bárzo / y przyśiągł mówiąc /
Zájſte / żaden z ludu tego narodu złego / nie ogląda oney źiemie dobrey / którąm przyśiągł dáć Ojcom wáƺym :
Oprócż Kálebá Syná Jefonowego / ten ją ogląda / y temu dam źiemię którą deptał / y Synom jego / przeto że cále ƺedł zá PAnem.
Tákże y ná mię / rozgniewał śię PAN dla was / mówiąc : y ty tám nie wnidźieƺ.
Jozue Syn Nunów któryć ſłuży / ten tám wnidźie / tegoż utwirdzaj : bo ją on w dźiedźictwo podá Izráelowi.
Dźiatki też wáƺe / o którycheśćie mówili / że będą <i>podáne</i> ná łup : tákże Synowie wáƺy / którzy dźiś nie wiedzą dobrego áni złego / ći tám wnidą / y onym ją dam / á oni ją dźiedźicżnie pośiedą :
Ale wy obróćiwƺy śię / idźćie ná puƺcżą / drogą ku morzu cżerwonemu.
A odpowiádájąc mówiliśćie do mnie : Zgrzeƺyliſmy PAnu : pójdźiemy / y walcżyć będźiemy / według tego wƺyſtkiego / jáko nam rozkazał PAN Bóg náƺ : y ubrał śię káżdy z was we zbroje ſwoję / á chćieliśćie wniść ná górę.
Lecż PAN rzekł do mnie : Powiedz im : Nie wſtępujćie / áni walcżćie / bom nie jeſt w pośrzodku was / ábyśćie nie byli poráżeni od nieprzyjaćiół wáƺych.
Co gdym wam opowiedźiał / nie ſłucháliśćie / áleśćie odporni byli ſłowu PAńſkiemu / y hárdźieśćie poſtąpili <i>ſobie</i> / á weƺliśćie ná górę.
Tedy wyćiągnął Amorrejcżyk / który mieƺkał ná oney górze / przećiwko wam : y gonili was / jáko zwykły pƺcżoły / y poráźili was w Seir áż do Hormá.
Y wróćiwƺy śię / płákáliśćie przed PAnem : lecż nie wyſłuchał PAN głoſu wáƺego / á nie nákłonił uƺu <i>ſwojich</i> do was.
Mieƺkáliśćie tedy w Kádes przez wiele dni / według <i>licżby</i> dni / którycheśćie tám mieƺkáli.
