Gdyby powſtał miedzy wámi Prorok / álbo ſny miewájący / y ukazałćiby znák / álbo cud :
Y ſtałby śię on znák / álbo cud o którym ći powiedźiał : <i>á</i> rzekłby : pódźmy zá Bogámi obcymi / których ty nie znaƺ / á ſłużmy im.
Nie uſłuchaƺ ſłów Proroká tego / áni tego co ſny miewa gdyż was doświadcżá PAN Bóg wáƺ / áby wiedźiał jeſli wy miłujećie PAná Bogá wáƺego / ze wƺyſtkiego ſercá wáƺego / y ze wƺyſtkiey duƺe wáƺey.
PAná Bogá wáƺego náśládujćie / y onego śię bójćie / á przykazań jego ſtrzegąc / y głoſu jego ſłuchájąc / ſłużćie mu y przy nim trwajćie.
Ale prorok on álbo miewájący ſny / zábity będźie : bo to mówił / cżymby was odwiódł od PAná Bogá wáƺego / ( który was wywiódł z źiemie Egipſkiey / y odkupił ćię z domu niewoli / ) áby ćię zráźił z drogi którą przykazał tobie PAN Bóg twój / żebyś nią chodźił : á ták wykorzeniƺ to złe z pośrzodku śiebie.
Jesliby ćię też zwodźił brát twój / Syn Mátki twojey / álbo Syn twój / álbo córká twojá / álbo żoná łoná twego / álbo przyjaćiel twój / którybyć był <i>miły,</i> jáko duƺá twojá potájemnie mówiąc : Pódźmy á ſłużmy bogom obcym którycheś nie znał ty / áni Ojcowie twoji.
Z bogów tych Narodów / które około was ſą / którzy bliſko ſą ćiebie / álbo dáleko od ćiebie / od końcá źiemie y áż do końcá źiemie.
Nie pozwoliƺ mu / áni go uſłuchaƺ / áni mu folgowáć będźie oko twoje / áni śię zmiłujeƺ / nád nim / áni go utájiƺ /
Ale koniecżnie zábijeƺ go : ręká twojá napierwƺá nád nim będźie / ná zábićie jego / á ręká wƺyſtkiego ludu potym.
Y ukámionujeƺ go áż ná śmierć / ponieważ chćiał ćię odwieść / od PAná Bogá twego / który ćię wywiódł z źiemie Egipſkiey / z domu niewoli :
Aby wƺyſtek Izráel uſłyƺawƺy bał śię / á nie cżynił więcey nic podobnego rzecży tey / bárzo złey miedzy wámi.
A jeſlibyś uſłyƺał / żeby w którymkolwiek mieśćie twojim / które PAN Bóg twój dawa tobie ku mieƺkániu / ktokolwiek rzekł :
Wyƺli <i>niektórzy</i> ludźie przewrotni z pośrzodku ćiebie / á podwiedli obywátele miáſtá ſwego / mówiąc : Pódźmy / á ſłużmy Bogom obcym których nie znaćie :
Tedy będźieƺ ƺukał y dochodźił / á pilnie śię wywiádował : á jeſli to prawdá á rzecż pewna / <i>że</i> śię ſtáłá obrzydliwość táká miedzy wámi /
Koniecżnie wytráćiƺ / obywátele miáſtá onego / oſtrzem miecżá : zgłádźiƺ je / y wƺyſtko co w nim / y bydłá jego <i>pobijeƺ</i> oſtrzem miecżá.
A wƺyſtkie łupy jego zbierzeƺ w pośród ulice jego / y ſpaliƺ ogniem miáſto y z onym wƺyſtkim łupem do ƺcżętu PAnu Bogu twemu : y będźie mogiłą wiecżną / á nie będźie budowáne więcey :
Y nie zoſtánie w rękách twojich nic z onych rzecży przeklętych / áby śię odwróćił PAN od gniewu popędliwośći ſwojey / á ucżynił z tobą miłośierdźie / y zmiłował śię nád tobą / á rozmnożył ćię jáko przyśiągł Ojcom twojim.
Gdy ſłucháć będźieƺ głoſu PAná Bogá twego / záchowywájąc wƺyſtkie przykazánia jego / które ja dźiś przykázuję tobie / żebyś cżynił co práwego jeſt / przed ocżymá PAná Bogá twego.
