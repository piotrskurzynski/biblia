Gdy wytráći PAN Bóg twój Narody / których źiemię PAN Bóg twój dawa tobie / á pośiędźieƺ ją / y mieƺkáć będźieƺ w miáſtách ich / y w domách ich.
Trzy miáſtá odłącżyƺ ſobie / w pośród źiemie twojey / którą PAN Bóg twój dawa tobie w ośiádłość.
Wyproſtujeƺ ſobie drogę / y rozdźieliƺ ná trzy cżęśći gránice źiemie twojey / którąć w dźiedźictwo poda PAN Bóg twóy / żeby tám ućiekał káżdy mężobójcá.
A toć będźie práwo mężobójce / który tám ućiecże áby żyw zoſtał / gdyby zábił bliźniego ſwego z nieobacżenia / á niemiałby go przed tym w nienawiśći :
Jáko / gdyby kto ƺedł z bliźnim ſwym do láſá drwá rąbáć / á zánióſłby śię ręką ſwą z śiekierą / áby ućiął drzewo / á ſpádłáby śiekierá z toporzyſká / y tráfiłáby bliźniego jego / żeby umárł / ten ućiecże do jednego z tych miaſt / áby żyw zoſtał :
By ſnadź nie gonił powinny zábitego / mężobójce onego / gdyby śię rozpaliło ſerce jego / á dogoniwƺy go ná dálekiey drodze / nie zábił go / choćby nie był winien śmierći / ponieważ go nie miał w nienawiśći przed tym.
Przetoż ja przykázuję tobie / mówiąc : trzy miáſtá odłączyƺ ſobie :
A gdy rozƺerzy PAN Bóg twój gránice twoje / jáko przyśiągł Ojcom twojim / y dać wƺyſtkę źiemię / którą záślubił dać Ojcom twojim :
Jeſliże będźieƺ ſtrzegł wƺyſtkich tych przykazań / y czynił je / które ja dźiś rozkázuję tobie żebyś miłował PAná Bogá twego / y chodźił drogámi jego / po wƺyſtkie dni / tedy przydaƺ ſobie jeƺcze trzy miáſtá / do onych trzech miaſt.
Aby nie byłá wylewána krew niewinna / w pośrzodku źiemie twojey / którą PAN Bóg twój dawa tobie w dźiedźictwo / áby nie zoſtáłá ná tobie krew.
Ale jeſliby kto májąc w nienawiśći bliźniego ſwego / á záſádźiwƺy śię nań / powſtałby przećiw jemu / y uderzyłby go śmiertelnie / áżby umárł / á ućiekł do jednego z tych miaſt :
Tedy poślą ſtárƺy miáſtá onego / y porwą go z támtąd / y wydádzą go w ręce powinnemu zábitego / y umrze.
Nie zfolguje mu oko twoje / ále odejmieƺ krew niewinną z Izráelá / ábyć śię dobrze dźiało.
Nie będźieƺ przenośił gránice bliźniego twego / którą záłożyli przodkowie w dźiedźictwie twojim / które ośiędźieƺ w źiemi / którą PAN Bóg twój dawa tobie w ośiádłość.
Nie powſtánie świádek jeden przećiwko cżłowiekowi / w jákieykolwiek niepráwośći / álbo w jákimkolwiek grzechu / ze wƺyſtkich grzechów któremiby kto zgrzeƺył : w uśćiech dwu świádków / álbo w uśćiech trzech świádków / ſtánie <i>káżde</i> ſłowo.
Jeſliby powſtał świádek kłamliwy przećiw komu / świádcżąc przećiw niemu / że odſtąpił <i>Bogá</i> :
Tedy ſtáną oni dwáj mężowie / którzy máją ſpór / przed PAnem : przed Kápłany / y przed Sędźiámi / którzy będą ná ten cżás :
Y będą śię pilnie wywiádowáli Sędźiowie / á obacząli że świádek on / jeſt świádkiem fałƺywym / kłamſtwo powiádájąc ná brátá ſwego.
Uczyńćie mu / jáko on myślił ucżynić brátu ſwemu : y odejmieƺ złe z pośrzodku śiebie :
Aby drudzy uſłyƺawƺy to / ulękli śię / y nie dopuƺcżáli śię więcey tákowey złośći cżynić w pośród ćiebie.
A nie zfolguje oko twoje : gárdło zá gárdło / oko zá oko / ząb zá ząb / ręká zá rękę / nogá zá nogę <i>będźie</i>.
