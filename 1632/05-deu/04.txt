Teraz tedy o Izráelu / ſłuchaj uſtaw / y ſądów / których ja was ucżę / ábyśćie je cżyniąc żyli / á wƺedƺy pośiedli źiemię którą PAN Bóg Ojców wáƺych dawa wam.
Nie przydaćie do ſłowá / które ja wam rozkázuję / áni ujmiećie z niego / ábyśćie ſtrzegli przykazań PAná Bogá wáƺego / które ja wam rozkázuję.
Ocży wáƺe widźiáły / co ucżynił PAN dla Báál fegorá : jáko káżdego mężá który śię udawał zá Báál fegorem / wytráćił PAN Bóg twój z pośrzodku was.
A wy którzyśćie trwáli przy PAnu Bogu wáƺym / żyjećie wƺyſcy / áż do dniá dźiśieyƺego.
Pátrzćieƺ żem was náucżał uſtaw y ſądów / jáko mi rozkazał PAN Bóg mój / ábyśćie ták cżynili w źiemi tey / do którey wchodźićie / ábyśćie ją dźiedźicżnie pośiedli.
Przeſtrzegajćież <i>ich</i> tedy / á cżyńćie je : to bowiem jeſt mądrość wáƺá / y rozum wáƺ / przed ocżymá Narodów : którzy uſłyƺawƺy o tych wƺyſtkich uſtáwách / rzeką : tylko ten lud mądry / y rozumny / y Naród wielki <i>jeſt</i>.
Abowiem któryż Naród <i>ták</i> wielki jeſt / coby mu byli Bogowie ták blizcy / jáko PAN Bóg náƺ / we wƺyſtkim / ilekroć go wzywamy?
Albo któryż Naród <i>ták</i> wielki / któryby miał uſtáwy y ſądy <i>ták</i> ſpráwiedliwe / jáko wƺyſtek ten Zakon / który ja przedkładam wam dźiś?
A wƺákże miey śię ná piecży / á ſtrzeż duƺe twey pilnie / byś ſnadź nie zápomniał ná te rzecży / które widźiáły ocży twoje / y áby ſnadź nie odſtąpiły od ſercá twego / po wƺyſtkie dni żywotá twego : y oznájmiƺ je Synom twym / y Synom Synów twojich.
<i>Nie zápominaj</i> żeś onego dniá ſtał przed oblicżnośćią PAná Bogá twego ná Horeb / gdy mówił PAN do mnie : Zgromadź mi lud / ábym im powiedźiał ſłowá moje / których śię ucżyć będą : áby śię mnie bali / po wƺyſtkie dni / pokąd żyć będą ná źiemi / y Synów ſwojich náucżájąc <i>tego</i>.
Tedyśćie przyſtąpili / á ſtánęliśćie pod górą / á oná górá pałáłá ogniem áż do ſámego niebá / á <i>byłá ná niey</i> ćiemność / Obłok / y mgłá.
Y mówił PAN do was z pośrzód ogniá : głos ſłów ſłyƺeliśćie / áleśćie podobieńſtwá <i>żadnego</i> nie widźieli / oprócż głoſu.
Y oznájmił wam Przymierze ſwoje / które rozkazał wam / ábyśćie je cżynili : <i>to jeſt,</i> dźieśięć ſłów / które nápiſał ná dwu tablicách kámiennych.
Mnie też rozkazał PAN ná on cżás / ábym was náucżał uſtaw y ſądów / ábyśćie je cżynili / w źiemi do którey idźiećie / ábyśćie ją dźiedźicżnie pośiedli.
A ták ſtrzeżćie pilnie duƺ wáƺych / gdyżeśćie nie widźieli żadnego podobieńſtwá / dniá którego mówił PAN do was ná <i>Horebie</i> z pośrzodku ogniá.
Abyśćie śię ſnadź nie popſowáli / y nie cżynili ſobie rytego Báłwáná / <i>álbo</i> obrázu wƺelkiego podobieńſtwá / kƺtałtu mężcżyzny álbo niewiáſty /
<i>Albo</i> kƺtałtu jákiego zwierzęćiá / które jeſt ná źiemi / kƺtałtu wƺelkiego ptaká ſkrzydláſtego / który lata po powietrzu.
Kƺtałtu wƺyſtkiego tego / co śię płaza po źiemi / kƺtałtu wƺelkiey ryby / która jeſt w wodźie pod źiemią.
Ani podnoś ocżu ſwych ku Niebu / żebyś widząc ſłońce y mieśiąc / y gwiazdy / ze wƺyſtkimi zaſtępy Niebieſkimi / nie dał śię uwieść / ábyś śię im kłaniał / y ſłużyłbyś im / ponieważ je oddał PAN Bóg twój <i>ná poſługę</i> wƺyſtkim ludźiom pod wƺyſtkim Niebem :
Aleć was wźiął PAN / y wywiódł was <i>jáko</i> z żeláznego piecá / z Egiptu : ábyśćie byli jego ludem dźiedźicżnym / jáko dźiś <i>jeſteśćie</i> :
A PAN rozgniewał śię ná mię dla was / á przyśiągł / że nie przejdę zá Jordan / áni wnidę do oney wyborney źiemie / którą PAN Bóg twój dawa tobie w dźiedźictwo.
Bo ja umrę w tey źiemi / ja nie przeydę zá Jordan : ále wy przeydźiećie / y dźiedźicżnie ośiądźiećie tę dobrą źiemię.
Strzeżćież śię / byśćie ſnadź nie zápomnieli przymierza PAná Bogá wáƺego / które ucżynił z wámi / á nie cżynili ſobie rytego obrázu / ná kƺtałt jákieykolwiek rzecży / jákoć rozkazał PAN Bóg twój.
Abowiem PAN Bóg twój / jeſt Ogień trawiący / Bóg zawiſny.
Gdy zpłodźiƺ Syny y wnuki / á ſtárzejećie śię w źiemi oney / jeſlibyśćie śię popſowáli / á cżynilibyśćie <i>ſobie</i> ryte báłwany / ná kƺtałt jákieykolwiek rzecży / y ucżynilibyśćie co złego przed ocżymá PAná Bogá wáƺego / drażniąc go :
Biorę ná świádectwo przećiwko wam dźiś / Niebo y źiemię / iż pewnie w rychle wyginiećie z źiemie / do którey pójdźiećie przez Jordan / ábyśćie ją dźiedźicżnie pośiedli : Nie przedłużyćie dni wáƺych w niey : bo pewnie wytráceni będźiećie.
Y rozproƺy was PAN miedzy Narody / á zoſtánie was máło miedzy Narody / do których záprowádźi was PAN.
Támże będźiećie ſłużyli Bogom / roboćie rąk ludzkich / drewnu y kámieniowi / które áni widzą / áni ſłyƺą / áni wącháją.
A wƺákże y tám jesli ƺukáć będźieƺ Paná Bogá twego / tedy znajdźieƺ : będźieƺli go ƺukał cáłem ſercem twojim / y cáłą duƺą twoją.
Gdy ućiſk przydźie ná ćię / á najdą ćię te wƺyſtkie rzecży w oſtátnie dni / tedy / náwróćiƺli śię do PAná Bogá twego / á poſłuƺny będźieƺ głoſowi jego :
( Abowiem Bóg miłośierny PAN Bóg twój jeſt ) nie opuśći ćię / áni ćię ſkáźi : áni zápomni ná Przymierze Ojców twojich / które im przyśiągł.
Pytaj śię teraz ná dni ſtárodawne które były przed tobą / ode dniá którego ſtworzył Bóg cżłowieká ná źiemi : á od jednego końcá Niebá / áż do drugiego końcá Niebá / ſtáłáli śię kiedy rzecż podobna tey ták wielkiey? Albo ſłuchanoli kiedy co tákowego?
Słyƺałli <i>kiedy który</i> naród głos Bogá mówiącego z pośrzodku ogniá / jákoś ty ſłyƺał / á żyw zoſtał?
Albo kuśiłli śię <i>który</i> Bóg przyść á wźiąć ſobie naród zpośrzodku <i>innego</i> narodu / przez pokuſy / przez znáki / y przez cudá / y przez wojny y przez mocną rękę / y przez rámię wyćiągnione / y w ſtráchách wielkich : według wƺyſtkiego / jáko ucżynił dla was PAN Bóg wáƺ w Egipćie / przed ocżymá twymi?
Tobie to ukazano / ábyś wiedźiał iż PAN jeſt Bóg / á nie máƺ innego oprócż niego.
Dałći z niebá ſłyƺeć głos ſwój / áby ćię wyćwicżył / á ná źiemi ukazałći ogień ſwój wielki / á ſłowá jego ſłyƺałeś / z pośrzodku ogniá.
A iż umiłował Ojce twoje / obrał naśienie ich po nich / y wywiódł ćię przed ſobą / mocą ſwoją wielką z Egiptu /
Aby wygnał Narody wielkie y możniejƺe nád ćię / przed twarzą twoją : y wprowádźił ćię / á dałći źiemię ich w dźiedźictwo / jáko <i>to</i> dźiś <i>widźiƺ</i>.
Wiedzże tedy dźiś / á wſpomni ná to w ſercu twojim / iż PAN jeſt Bogiem ná Niebie wyſoko / y ná źiemi niſko nie máƺ innego.
A ták przeſtrzegaj uſtaw jego / y przykazań jego / które ja dźiś rozkázuję tobie : ábyć śię dobrze wodźiło y Synom twojim po tobie : ábyś też przedłużył dni ná źiemi / którą PAN Bóg twój da tobie / po wƺyſtkie dni.
Tedy oddźielił Mojzeƺ trzy miáſtá / z tę ſtronę Jordanu / ná wſchód ſłońcá :
Aby tám ućiekał mężobójcá / któryby zábił bliźniego ſwego z nieobacżenia / nie májąc go w nienawiśći przed tym / áby ućiekƺy do jednego z tych miaſt / żyw zoſtał :
Beſer / ná puƺcży / w równinie / w źiemi Rubenitów : y Rámot w Gáláád / <i>miedzy</i> Gádyty : y Golám w Bázán / <i>miedzy</i> Mánáſſitámi.
Tenći jeſt Zakon / który przedłożył Mojzeƺ Synom Izráelſkim.
Te ſą świádectwá / y uſtáwy / y ſądy / które mówił Mojzeƺ do Synów Izráelſkich / gdy wyƺli z Egiptu /
Z tę ſtronę Jordanu / w dolinie przećiw Betfegor / w źiemi Sehoná królá Amorrejſkiego / który mieƺkał w Heſebon : Którego poráźił Mojzeƺ / y Synowie Izráelſcy / gdy ƺli z Egiptu :
Y ośiedli źiemię jego / y źiemię Ogá królá Báſáńſkiego / dwu królów Amorrejſkich / którzy <i>byli</i> z tę ſtronę Jordanu / ná wſchód ſłońcá /
Od Aroer / które jeſt nád brzegiem potoku Arnon : y áż do góry Syon / która jeſt Hermon :
Y wƺyſtkie polá nád Jordanem ná wſchód ſłońcá y áż do Morzá Puſtego / pod górą Fásgá.
