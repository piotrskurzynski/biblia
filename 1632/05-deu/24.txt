Gdyby pojął kto żonę / á ſtałby śię jey małżonkiem / á przydáłoby śię / żeby nie nálázłá łáſki w ocżách jego / przeto że ználazł przy niey co ſproſnego ; tedy jey nápiƺe liſt rozwodny / y da w rękę jey / á puśći ją z domu ſwego.
A gdyby wyƺłá z domu jego / á odƺedƺy ƺłáby zá drugiego mężá /
A májąc ją w nienawiśći on mąż drugi / nápiſałby jey liſt rozwodny / y dałby w rękę jey / y puśćiłby ją z domu ſwego : álbo jeſliby też umárł on mąż drugi / który ją był pojął ſobie zá żonę :
Nie będźie mógł mąż jey pierwƺy / który ją był opuśćił / znowu jey pojąć / áby mu byłá żoną / gdyż jeſt ſplugáwiona : ábowiem obrzydliwośćią to jeſt przed obliczem PAńſkim : przetoż nie dopuƺcżaj grzeƺyć źiemi / którą PAN Bóg twój dawa tobie w dźiedźictwo.
Gdyby śię kto świeżo ożenił / nie wynidźie ná Wojnę / áni nań włożona będźie jáka pracá / wolen będźie w domu ſwym / przez cáły rok / weſeląc śię z żoną ſwoją którą pojął.
Nikt nie weźmie w zaſtáwie zwierzchniego y ſpodniego kámieniá młyńſkiego : bo tákowy <i>jákoby</i> duƺę brał w zaſtáwie.
Jeſliby kto był ználeźion / coby ukradł cżłowieká z bráćij ſwey Synów Izráelſkich / á hándlowałby nim / y przedał go / tedy umrze on złodźij / y odejmieƺ złe z pośrzodku śiebie.
Strzeż śię zárázy trądu / á przeſtrzegaj pilnie / żebyś cżynił wƺyſtko / cżego was náuczą Kápłani Lewitowie : jákom im rozkazał / przeſtrzegáć tego / <i>y</i> cżynić to będźiećie.
Pámiętaj co ucżynił PAN Bóg twój Máriey / w drodze / gdyśćie wyƺli z Egiptu.
Gdy pożycżyƺ cżego bliźniemu twemu / nie wchodźże do domu jego / ábyś wźiął co w zaſtáwie od niego /
<i>Ale</i> ná dworze zoſtánieƺ / á cżłowiek któremuś pożycżył / wynieśie do ćiebie zaſtáwę przed dom.
A jeſliby <i>on</i> cżłowiek był ubogi / nie ukłádźieƺ śię z zaſtáwą jego /
Bez omieƺkánia wróćiƺ mu on zaſtáwę / gdy ſłońce zajdźie / żeby leżał ná odźieniu ſwym / y błogoſłáwiłći : á będźieć to ſpráwiedliwośćią / przed PANEM Bogiem twojim.
Nie ucżyniƺ krzywdy najemnikowi ubogiemu / y potrzebnemu z bráći twojey / álbo z cudzoźiemców twojich / którzy ſą w źiemi twey w bramách twojich /
Tegoż dniá daƺ mu zapłátę jego / przed zaśćiem ſłońcá / ábowiem ubogi jeſt / á z tego żywi duƺę ſwoję / żeby nie wołał przećiwko tobie do PAná / á zoſtałby ná tobie grzech.
Nie umrą Ojcowie zá Syny / á Synowie nie umrą zá Ojce / káżdy zá grzech ſwój umrze.
Nie wywróćiƺ ſądu przychodniowi / <i>áni</i> śieroćie / áni weźmieƺ w zaſtáwie ƺáty wdowy :
Ale pámiętaj / żeś niewolnikiem był w Egipćie / á <i>iż</i> ćię wybáwił PAN Bóg twój z támtąd : dla tegoż ja przykázuję tobie / ábyś to cżynił.
Gdybyś żął zboże twoje ná roli twojey / á zápámiętałbyś ſnopá ná polu / nie wracaj śię / ábyś go wźiął / przychodniowi / śieroćie / y wdowie / <i>to</i> będźie : ábyć błogoſłáwił PAN Bóg twój / w káżdey ſpráwie rąk twojich.
Gdy będźieƺ obierał Oliwki twoje / nie oglądajże śię ná káżdą gáłąſkę zá ſobą / przychodniowi / śieroćie / y wdowie to będźie.
Gdy będźieƺ obierał winnicę twoję / nie zbierajże gron pozoſtáłych zá tobą / przychodniowi / śieroćie / y wdowie to będźie.
A pámiętaj / iżeś był niewolnikiem w źiemi Egipſkiey. Przetoż ja tobie przykázuję / ábyś to cżynił.
