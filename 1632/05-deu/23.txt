Nie wnidźie wypukły / áni trzebieniec do zgromádzenia PAńſkiego.
Ani wnidźie niepocżćiwego łożá Syn / do zgromádzenia PAńſkiego / y dźieśiąte pokolenie jego / nie wnidźie do zgromádzenia PAńſkiego.
Nie wnidźie też Ammonitá / y Moábcżyk / do zgromádzenia PAńſkiego / áni dźieśiąte pokolenie <i>ich,</i> nie wnidźie do zgromádzenia PAńſkiego áż ná wieki :
Dla tego / że wam nie zábieżeli z chlebem / áni z wodą w drodze / gdyśćie ƺli z Egiptu / á że nájęli zá zapłátę przećiw tobie Báláámá / Syná Beorowego z Pethor Meſopotámiey Syryjſkiey / áby ćię przeklinał.
Acż nie chćiał PAN Bóg twój wyſłucháć Báláámá / ále obróćił PAN Bóg twój tobie <i>jego</i> przeklęctwo / w błogoſłáwieńſtwo / bo ćię umiłował PAN Bóg twój.
Nie ƺukaj pokoju ich / áni dobrego ich / po wƺyſtkie dni twoje ná wieki.
Nie będźieƺ śię brzydźił Idumejcżykiem / bo brátem twojim jeſt : Nie będźieƺ śię brzydźił Egipcżykiem / boś był przychodniem w źiemi jego.
Synowie którzy śię im zrodzą / w trzećiem pokoleniu / wnidą do zgromádzenia PAńſkiego.
Gdy śię ruƺyƺ z wojſkiem / ná przećiwko nieprzyjaćiołom twojim / ſtrzeƺ śię od wƺelkiey złey rzecży.
Jeſliby był miedzy wámi kto / coby był niecżyſtym z przygody nocney / wynidźie zá Obóz / á nie wróći śię do niego :
A ku wiecżorowi omyje śię wodą / á po zaśćiu ſłońcá / wnidźie do Obozu.
Będźiećie też mieli miejſce zá Obozem / gdźie będźiećie wychodźili ná <i>potrzebę przyrodzoną</i> :
Y będźieƺ miał rydlik miedzy nacżyniem ſwojim / á gdybyś chćiał uśieść dla potrzeby / wykopieƺ nim dołek / á obróćiwƺy śię / zágrzebieƺ plugáſtwo twoje :
Abowiem PAN Bóg twój / chodźi w pośrzodku Obozu twego / áby ćię wyrwał / y podałći nieprzyjaćioły twoje : przetoż niech będźie Obóz twój święty / áby nie widźiał przy tobie ſproſnośći jákiey / dla któreyby śię odwróćił od ćiebie.
Nie wydaƺ ſługi Pánu jego / któryby śię ućiekł do ćiebie / od Páná ſwego.
Z tobą będźie mieƺkał w pośrzodku ćiebie / ná miejſcu które ſobie obierze / w jednym z miaſt twojich / gdźieby mu śię podobáło : nie ucżyniƺ mu gwałtu.
Nie będźie nierządnicá z córek Izráelſkich / áni będźie nierządnik z Synów Izráelſkich.
Nie wnoś zapłáty nierządnice / áni zapłáty pſá / w dom PAná Bogá twego / zá jákikolwiek ślub : bo obrzydliwośćią u PAná Bogá twego jeſt to oboje.
Nie daƺ ná lichwę brátu twemu / áni pieniędzy / áni żywnośći / áni jákieykolwiek rzeczy / którą dawáją ná lichwę.
Cudzoźiemcowi ná lichwę dawáć możeƺ / ále brátu twemu ná lichwę nie daƺ : ábyć błogoſłáwił PAN Bóg twój / w káżdey ſpráwie / do którey zćiągnieƺ rękę twoję / w źiemi / do którey wnidźieƺ ábyś ją pośiadł.
Jeſlibyś ślubował ślub PAnu Bogu twemu / nie omieƺkywajże oddawáć go : bo koniecżnie upomni śię go PAN Bóg twój od ćiebie / á będźie ná tobie grzech.
A jeſli nie będźieƺ ślubował / nie będźie ná tobie grzechu.
<i>To</i> co wynidźie z uſt twojich / wypełniƺ / y uczyniƺ / jákoś ślubował PAnu Bogu twemu dobrowolnie / coś wymówił uſty twymi.
Gdy wnidźieƺ do Winnice bliźniego twego / jedz jágody ileć śię podoba / áż do ſytośći : ále do naczynia twego nie bierz.
<i>Tákże</i> gdy wnidźieƺ miedzy zboże bliźniego twego / tedy nárwieƺ kłoſów ręką twą / ále śierpá nie zápuƺczaj w zboże bliźniego twego.
