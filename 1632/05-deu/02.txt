Potym obróćiwƺy śię / poƺliſmy ná puƺczą / drogą ku morzu cżerwonemu / jáko mówił PAN do mnie : y krążyliſmy około góry Seir przez wiele dni.
Tedy rzekł PAN do mnie / mówiąc :
Doſyćieśćie już krążyli około tey góry / Obróććież śię ku pułnocy :
A ludowi rozkaż / mówiąc : Wy wnet pójdźiećie przez gránice bráći wáƺey / Synów Ezawowych / którzy mieƺkáją w Seir : y będą śię was bali : ále śię y wy pilnie ſtrzeżćie.
Nie drażnićież ich / ábowiem nie dam wam źiemie ich / áni ná ſtopę nogi : gdyżem w ośiádłość dał Ezawowi górę Seir.
Żywność będźiećie kupowáli u nich zá pieniądze / y jeść będźiećie : tákże y wodę od nich kupowáć będźiećie zá pieniądze / y pić będźiećie.
Abowiemći PAN Bóg twój błogoſłáwił we wƺelkiey ſpráwie rąk twojich : <i>y</i> wiedźiał drogę twoję ná tey puƺczy wielkiey : już przez cżterdźieśći lat PAN Bóg twój był z tobą / nie zchodźiłoć ná niczym.
Poƺliſmy tedy od bráći náƺey / Synów Ezáwowych / mieƺkájących w Seir / drogą równą od Eláth / y od Aſiongáber : á náwróćiwƺy śię událiſmy śię drogą / <i>ku</i> puƺczy Moábſkiey.
Y rzekł PAN do mnie : nie náćieraj ná Moábczyki / áni podnoś wojny przećiwko nim : boć nie dam źiemie ich w ośiádłość : gdyżem Synom Lotowym dał Ar / w dźiedźictwo.
( Emitowie pierwey mieƺkáli w niey / lud potężny / y wielki y wyſokiego wzroſtu / jáko Enákimowie :
Które też zá obrzymy miano / jáko Enákimy : á Moábczykowie zwáli je Emim.
Tákże w Seir mieƺkáli Horejcżycy przed tym / które Synowie Ezawowi wygnáli / y wygłádźili je przed ſobą : á mieƺkáli ná miejſcu ich jáko ucżynił Izráel w źiemi ośiádłośći ſwojey / którą im dał PAN. )
Wſtańćież tedy / á przeprawćie śię przez potok Záred : y przepráwiliſmy śię przez potok Záred.
A czáſu któregoſmy chodźili od Kádes Bárne / áżeſmy śię przepráwili przez potok Záred / było trzydźieśći y ośm lat : áż wyginął wƺyſtek on naród mężów walecznych z Obozu : jáko im <i>był</i> przyśiągł PAN.
Abowiem ręká PAńſka byłá przećiwko nim / ná wytrácenie ich z Obozu / áż ich wygubiłá.
Y ſtáło śię / gdy wyginęli wƺyſcy mężowie waleczni / y pomárli z pośrzodku ludu /
Że rzekł PAN do mnie / mówiąc :
Ty dźiś przejdźieƺ gránicę Moábſką Ar /
Y przyjdźieƺ bliſko ku Synom Ammonowym : nie náćierajże ná nie / áni podnoś wojny przećiwko nim : boć nie dam źiemie Synów Ammonowych w ośiádłość : ponieważ Synom Lotowym dałem ją w dźiedźictwo.
( Tę też źiemię <i>zá źiemię</i> Obrzymów miano / <i>ábowiem</i> Obrzymowie mieƺkáli w niey / przed tym : które Ammonitowie názywáli Zomzomim.
Lud potężny / y wielki / y wyſokiego wzroſtu / jáko Enákimowie / ále wygubił je PAN przed nimi / á <i>Ammonitowie</i> opánowáli je / y mieƺkáli ná miejſcu ich.
Jáko ucżynił Synom Ezawowym mieƺkájącym w Seir / dla których wytráćił Horejczyki przed obliczem ich : y wygnáli je / y mieƺkáli ná miejſcách ich / áż po dźiś dźień.
Heweycżyki tákże / którzy mieƺkáli w Aſerym / áż do Gázy / Káftorytowie / którzy wyƺli z Káftor / wytráćili je / y mieƺkáli ná miejſcu ich. )
Wſtawƺy tedy / idźćie / á przeprawćie śię przez potok Arnon : oto / ja dawam w ręce twoje / Sehoná Królá Heſebon Amorrejczyká / y źiemię jego : poczniże ją pośiadáć / á podnieś przećiwko niemu wojnę.
Dźiśia pocżnę puƺcżáć ſtrách y bojaźń twoję ná ludźie / którzy ſą pod wƺyſtkim Niebem : że gdy uſłyƺą wieść o tobie / drżeć y lękáć śię będą przed tobą.
Tedym poſłał poſły z puƺczy Kádemot do Sehoná królá Heſebonſkiego z poſelſtwem ſpokojnym / mówiąc :
Niech przejdę przez źiemię twoję : drogą poſpolitą pójdę / nie uchylę śię / áni ná práwo áni ná lewo.
Żywnośći zá pieniądze przedaƺ mi / ábym jadł : wody tákże zá pieniądze daƺ mi / ábym pił : tylko przejdę nogámi mymi :
Jáko mi ucżynili Synowie Ezawowi / którzy mieƺkáją w Seir / y Moábcżycy którzy mieƺkáją w Ar / áżbyſmy śię przepráwili zá Jordan / do źiemie którą PAN Bóg náƺ dawa nam.
Ale nie chćiał Sehon Król Heſebońſki pozwolić nam prześćia / przez <i>źiemię</i> ſwoję : bo był zátwárdźił PAN Bóg twój duchá jego / y ztwirdźił ſerce jego áby go podał w ręce twoje / jáko to dźiś widźiƺ.
Tedy rzekł PAN do mnie : Otomći już począł podawáć w moc Sehoná y źiemię jego : pocżniże <i>ją</i> pośiadáć / ábyś odźiedźiczył źiemię jego.
Ruƺył śię tedy Sehon przećiwko nam / ſam y wƺyſtek lud jego / chcąc z námi zwieść bitwę w Jázá.
Ale go nam podał PAN Bóg náƺ w moc y poráźiliſmy go / y Syny jego / y wƺyſtek lud jego.
Y wźięliſmy wƺyſtkie miáſtá jego ná on czás / y wniweczeſmy obróćili wƺyſtkie miáſtá / mężczyzny y niewiáſty / y dźieći / nie zoſtáwiwƺy <i>z nich</i> nikogo.
Tylkoſmy bydło pobráli ſobie / y korzyśći z miaſt / którychesmy dobyli.
Od Aroer które <i>leży</i> nád brzegiem potoku Arnon / y od miáſtá które jeſt w dolinie / áż do Gáláád / nie było miáſtá któreby śię nam nie podáło : wƺyſtkie podał nam PAN Bóg náƺ.
Tylkoś do źiemie Synów Ammonowych nie dochodźił / <i>áni do</i> żadnego miejſcá <i>przyległego</i> potokowi Jábok / áni do miaſt ná górách / áni do żadnych miejſc / których zákazał PAN Bóg náƺ.
