Y rzekł Báláám do Báláká : Zbuduj mi tu śiedm Ołtarzów / á nágotuj mi tu śiedm ćielców / y śiedm báránów.
Ucżynił tedy Bálák / jáko mówił Báláám / y ofiárował Bálák z Báláámem ćielcá / y báráná ná <i>káżdym</i> Ołtarzu.
Potym rzekł Báláám do Báláká : ſtań przy cáłopaleniu twojim / á <i>ja</i> odeydę : owa śię ſnadź potka PAN ze mną / á cokolwiek mi objáwi / opowiem ći / y odƺedł ſam.
Y potkał śię PAN z Báláámem / y rzekł mu <i>Báláám,</i> poſtáwiłem śiedm Ołtarzów / y ofiárowałem ćielcá y báráná ná <i>káżdym</i> Ołtarzu.
Tedy PAN włożył ſłowá w uſtá Báláámowe / y rzekł : wróć śię do Báláká / á mów ták.
Y wróćił śię do niego / á oto ſtał u ofiáry ſwojey paloney / on y wƺyſtkie Kśiążętá Moábſkie.
A ták zácżął przypowieść ſwoję / y rzekł : z Arám przywiódł mię Bálák / król Moábſki / z gór wſchodnich / <i>mówiąc :</i> Przydź przeklinaj mi Jákóbá : á przydź złorzecż Izráelowi :
Jákoż ja przeklináć mam / kogo Bóg nie przeklina? álbo jáko złorzeczyć mam / komu PAN nie złorzecży?
Bo z wierzchu ſkał oglądam go / á z págórków będę nań pátrzał : Oto lud <i>ten</i> ſam mieƺkáć będźie / á miedzy Narody mięƺáć śię nie będźie.
Któż policży proch Jákóbów? y liczbę cżwartey częśći Izráelá? Niech umrze duƺá mojá śmierćią ſpráwiedliwych / y niech będźie dokonánie moje / jáko ich.
Tedy rzekł Bálák do Báláámá : Cóżeś mi ucżynił? Ná przeklinánie nieprzyjaćiół mojich przyzwałem ćię / á oto błogoſłáwiąc / błogoſłáwiłeś <i>im</i>.
A <i>on</i> odpowiedźiał / y rzekł : ázaż nie mam tego pilnowáć y mówić / co PAN włożył w uſtá moje?
Y rzekł do niego Bálák : Pódź / proƺę / ze mną ná miejſce inne / żebyś go z támtąd widźiał : ( ále <i>tylko</i> część jego ujrzyƺ / á wƺyſtkiego widźieć nie będźieƺ ) przeklinajże mi go z támtąd.
Y záwiódł go ná pole Sofim / ná wierzch <i>góry</i> Fásgá : y zbudował śiedm Ołtarzów / y ofiárował ćielcá y báráná / <i>ná káżdym</i> Ołtarzu.
Rzekł tedy <i>Báláám</i> do Báláká / zoſtań tu przy cáłopaleniu twojim / á ja zábieżę tám <i>Panu</i>.
Y záƺedł PAN Báláámowi / który włożył ſłowá w uſtá jego / y rzekł :
Wróć śię do Báláká / á ták mów : Przyƺedł tedy do niego / á oto on ſtał przy cáłopaleniu ſwojim / y Kśiążętá Moábſkie z nim : Y rzekł mu Bálák : Cóż ći powiedźiał PAN?
Y záczął rzecż ſwą tymi ſłowy : Wſtań Báláku á ſłuchaj / przyjmi w uƺy ſwe ſłowá moje / Synu Seforów.
Nie jeſtći Bóg <i>jáko</i> cżłowiek / áby kłamał : áni <i>jáko</i> Syn cżłowiecży / żeby żáłował : ázaż on rzecże / á nie ucżyni? wymówi / á nie wypełni?
Otom <i>wźiął</i> rozkazánie / ábym błogoſłáwił : on błogoſłáwieńſtwo dał / á <i>ja</i> go nie odwrócę :
Nie bacży niepráwośći w Jákóbie / áni widźi przeſtępſtwá w Izráelu : PAN Bóg jego jeſt z nim / á trąbienie <i>zwyćięſtwá</i> królewſkiego przy nim :
Bóg wywiódł je z Egiptu / mocą jednorożcową był mu.
Abowiem niemáƺ wieƺcżby przećiw Jákóbowi / áni wróżki przećiw Izráelowi : Od tego cżáſu mówiono będźie o Jákóbie y o Izráelu / co <i>z nimi</i> uczynił Bóg.
Oto lud ten / jáko lew śilny powſtánie / jáko lwię młode podnieśie śię : nie ukłádźie śię / áż pożrże łupy / y krew pobitych wypije.
Tedy rzekł Bálák do Báláámá : áni ich przeklinaj więcey / áni im też błogoſław więcey.
Y odpowiedźiał Báláám / á rzekł do Báláká : Azażem ći nie powiádał mówiąc / że cokolwiek mówić będźie PAN / to ucżynię?
Y rzekł Bálák do Báláámá : pódź / proƺę záwiodę ćię ná inƺe miejſce / jesli ſnadź podoba śię Bogu / żebyś je z támtąd przeklinał.
Tedy <i>wiódł</i> Bálák Báláámá ná wierzch góry Fegor / która pátrza ku puƺczy.
Y rzekł Báláám do Báláká. Zbuduj mi tu śiedm Ołtarzów / á nágotuj mi tu śiedm ćielców / y śiedm báránów.
Y ucżynił Bálák / jáko <i>mu</i> rozkazał Báláám : y ofiárował ćielcá y báráná <i>ná káżdym</i> Ołtarzu.
