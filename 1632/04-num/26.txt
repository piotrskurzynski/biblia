Y ſtáło śię po oney pladze / że mówił PAN do Mojzeƺá / y do Eleázárá Syná Aároná Kápłaná / mówiąc :
Policzćie poczet wƺyſtkiego zgromádzenia Synów Izráelſkich / od tych którzy máją dwádźieśćiá lat / y wyżey / według domów Ojców ich / káżdego któryby mógł wyniść ná wojnę z Izráelá.
Tedy rzekł Mojzeƺ y Eleázár Kápłan do nich / ná polách Moábſkich / nád Jordanem / <i>przećiw</i> Jerichu / mówiąc :
<i>Liczćie lud</i> od tych którzy máją dwádźieśćiá lat y wyżey jáko był rozkazał PAN Mojzeƺowi / y Synom Izráelſkim / gdy wyƺli z źiemie Egipſkiey.
Ruben / pierworodny Izráelów : Synowie Rubenowi / Henoch <i>od którego</i> poƺedł dom Henochitów. Fállu / <i>od którego</i> dom Fálluitów.
Y Heſron / <i>od którego</i> dom Heſronitów. Chármi / <i>od którego</i> dom Chármitów.
Teć <i>ſą</i> domy Rubenitów : á było ich policzonych / czterdźieśći y trzy tyśiące / śiedm ſet / y trzydźieśći.
A Syn Fállów Eliáb.
Synowie záśię Eliábowi <i>byli</i> Námuel / y Dátán / y Abiron. A ći / Dátán y Abiron / zacniejƺy byli miedzy zgromádzeniem : Którzy śię ſwárzyli z Mojzeƺem / y z Aáronem w ſpiknieniu Korego / gdy śię byli zbuntowáli przećiwko PAnu.
Y otworzyłá źiemiá uſtá ſwoje / á pożárłá onych y Korego : gdy zginęłá oná rotá / á pożárł ich ogień dwieśćie y pięćdźieśiąt mężów / którzy śię ſtáli ná przykład <i>innym</i>.
Ale Synowie Korego / nie pomárli.
Synowie Symeonowi / wedle domów ſwych / <i>ći ſą,</i> Námuel : od <i>którego poƺedł</i> dom Námuelitów. Jámin / <i>od którego</i> dom Jáminitów. Jáchin / <i>od którego</i> dom Jáchinitów. Záre / od <i>którego</i> dom Záreytów.
Saul / od <i>którego</i> dom Saulitów.
Teć były domy Symeonitów / <i>których było</i> dwádźieśćiá y dwá tyśiące / y dwieśćie.
Synowie Gádowi według domów ſwych Sefon / od <i>którego poƺedł</i> dom Sefonitów. Aggi / od <i>którego</i> dom Aggitów. Suni / od <i>którego</i> dom Sunitów.
Ozni / od <i>którego</i> dom Oznitów. Heri / od <i>którego</i> dom Heritów.
Arod / od <i>którego</i> dom Aroditów. Ariel / od <i>którego</i> dom Arielitów.
Teć ſą domy Synów Gádowych / według pocztów ich czterdźieśći tyśięcy / y pięć ſet.
Synowie Judowi / Her / y Onán : ále pomárli Her y Onán / w źiemi Chánánejſkiey.
Y byli Synowie Judowi / wedle domów ſwych. Selá / od <i>którego poƺedł</i> dom Seláitów. Fáres / od <i>którego</i> dom Fáresitów. Záre / od <i>którego</i> dom Zárejczyków.
Byli też Synowie Fáreſowi. Hesron / od <i>którego</i> dom Hesronitów. Hámuel / od <i>którego</i> dom Hámuelitów.
Teć ſą domy Judá / według pocztów ich : śiedmdźieśiąt tyśięcy / y ƺeść / y pięć ſet.
Synowie Iſáſchárowi / według domów ſwych : Tolá / od <i>którego</i> dom Toláitów. Fuá / od <i>którego</i> dom Fuáitów.
Jáſub / od <i>którego</i> dom Jásubitów. Semrám / od <i>którego</i> dom Semrámitów.
Teć ſą domy Iſáſchárowe / wedle pocztów ich / ƺeśćdźieśiąt tyśięcy / y cztery / y trzy ſtá.
Synowie Zábulonowi / według domów ſwych : Záred / od <i>którego</i> dom Záredczyków. Elon / od <i>którego</i> dom Elonitów. Jáleel / od <i>którego</i> dom Jáleelitów.
A teć ſą domy Zábulonitów / według pocztów ich : ƺeśćdźieśiąt tyśięcy / y pięć ſet.
Synowie Józefowi / według domów ſwych : Mánáſſes y Efráim.
Synowie Mánáſſeſſowi / Máchir / od <i>którego</i> dom Mácháritów : á Máchir zpłodźił Gáláádá : od Gáláádá / dom Gálááditów.
Ci <i>ſą</i> Synowie Gáláádowi : Jezer / od <i>którego</i> dom Jezeritów. Chelek / od <i>którego</i> dom Chelekitów.
Y Aſriel / od <i>którego</i> dom Aſrielitów. Y Sechem / od <i>którego</i> dom Sechemitów.
Y Semidá / od <i>którego</i> dom Semidáitów : y Chefer / od <i>którego</i> dom Cheferitów.
A Sálfáád / Syn Cheferów / nie miał Synów / tylko córki : á imioná córek / Sálfáádowych / Máchlá / y Noá / Heglá / Melchá / y Terſá.
Teć ſą domy Mánáſſeſſowe / á poczet ich / pięćdźieśiąt / y dwá tyśiące / y śiedm ſet.
Synowie <i>záś</i> Efráimowi / według domów ſwych. Sutálá / od <i>którego</i> dom Sutálitów. Becher / od <i>którego</i> dom Becheritów. Techen / od <i>którego</i> dom Techenitów.
A ći <i>ſą</i> Synowie Sutálego. Herán / od <i>którego</i> dom Heránitów.
Teć <i>ſą</i> domy Synów Efráimowych / według pocztów ich / trzydźieśći tyśięcy y dwá / y pięć ſet. Ci ſą Synowie Józefowi / według domów ſwych.
A Synowie Beniáminowi / według domów ſwych / <i>ći ſą :</i> Belá / od <i>którego</i> dom Belitów. Aſbel / od <i>którego</i> dom Aſbelitów. Achirám / od <i>którego</i> dom Achirámitów.
Sufám / od <i>którego</i> dom Sufámitów. Hufám / od <i>którego</i> dom Hufámitów.
Byli też Synowie Bele : Heredá y Noemán. Z Heredá / dom Hereditów : á z Noemáná / dom Noemánitów.
Cić <i>ſą</i> Synowie Beniáminowi / według domów ich / á poczet ich / czterdźieśći y pięć tyśięcy / y ƺeść ſet.
Synowie záś Dánowi / według domów ſwych : Suchám / od <i>którego</i> dom Suchámitów : teć były domy Dánowe według fámilii ich :
Wƺyſtkie domy Suchámitów / według pocztów ich ƺeśćdźieśiąt y cztery tyśiące / y cztery ſtá.
Synowie Aſerowi / według domów ſwych <i>byli :</i> Jemná / od <i>którego</i> dom Jemnitów. Iswi / od <i>którego</i> dom Iſwitów. Beryá / od <i>którego</i> dom Berytów.
Synowie Beryjego / Heber : od <i>którego</i> dom Heberitów. Melchiel / od <i>którego</i> dom Melchielitów.
A imię córki Aſerowey / <i>było</i> Sará.
Te ſą domy Synów Aſerowych / według pocztów ich / pięćdźieśiąt y trzy tyśiące / y cztery ſtá.
Synowie Neftálimowi / według domów ſwych : Jáchſel / od <i>którego</i> dom Jáchſelitów. Guni / od <i>którego</i> dom Gunitów. Jeſſer / od <i>którego</i> dom Jeſſeritów.
Selem / od <i>którego dom Selemitów.
Teć ſą domy Neftálimowe / według fámilii ich / á poczet ich / czterdźieśći y pięć tyśięcy / y cztery ſtá.
Táć jeſt liczbá Synów Izráelſkich / ƺeść kroć ſto tyśięcy / y tyśiąc / śiedm ſet y trzydźieśći.
Zátym PAN rzekł do Mojzeƺá / mówiąc :
Między te podźielićie tę źiemię w dźiedźictwo / według liczby imion.
Więkƺey liczbie / więcey dźiedźictwá daƺ : á mniejƺey / mnieyƺe dźiedźictwo daƺ : káżdemu / według pocztu policzonych jego / będźie dano dźiedźictwo jego.
Wƺákże loſem niech będźie rozdźielona źiemiá : według imion pokoleń Ojców ſwych / dźiedźictwo bráć będą.
Loſem rozdźielone będźie dźiedźictwo jey : bądź ich wiele / bądź máło <i>będźie</i>.
Ci záśię <i>ſą</i> policzeni z Lewitów według domów ſwych. Gerſon / od <i>którego</i> dom Gerſonitów. Káát / od <i>którego</i> dom Káátitów. Merári / od <i>którego</i> dom Meráritów.
Teć ſą domy Lewi : dom Libnitów : dom Hebronitów : dom Moholitów : dom Muſitów : dom Koritów : A Káát zpłodźił Amrámá.
A imię żony Amrámowey <i>było</i> Jochábod / córká Lewiego / która mu śię urodźiłá w Egipćie : tá Amrámowi urodźiłá Aároná y Mojzeƺá / y Máryą śioſtrę ich.
Aáronowi też urodźili śię. Nádáb y Abiu / Eleázár / y Itámár.
Ale pomárli Nádáb y Abiu / gdy ofiárowáli ogień obcy przed PAnem.
A byłá liczbá ich / dwádźieśćiá y trzy tyśiące / wƺyſtkich mężczyzn <i>urodzonych</i> od mieśiącá y wyżey : jednák nie byli policzeni miedzy Syny Izráelſkie / bo im nie dano dźiedźictwá miedzy Syny Izráelſkimi.
Ci policzeni byli od Mojzeƺá y Eleázárá Kápłaná / którzy policzyli Syny Izráelſkie / ná polách Moábſkich / nád Jordanem / przećiw Jerychu.
A miedzy temi / nie był żaden <i>z onych</i> policzonych od Mojzeƺá y Aároná Kápłaná / gdy liczyli Syny Izráelſkie / ná puƺczy Synáj.
Bo rzekł był PAN o nich / śmierćią pomrą ná puƺczy : á nie zoſtał żaden z nich / oprócz Kálebá Syná Jefonowego / á Jozuego Syná Nunowego.
