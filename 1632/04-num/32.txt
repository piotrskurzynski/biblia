Y mieli Synowie Rubenowi / y Synowie Gádowi / bydłá bárzo wiele : á obacżywƺy źiemię Jázer / y źiemię Gáláád / że miejſce / ono <i>było ſpoſobne</i> dla bydłá /
Przyƺli <i>ćiż</i> Synowie Gádowi / y Synowie Rubenowi / y mówili do Mojzeƺá / y do Eleázárá Kápłaná / y do Kśiążąt zgromádzenia / y rzekli :
<i>Ziemiá</i> Atároth y Dybon / y Jázer / y Nemrá / y Heſebon / y Eleále / y Sebán / y Nebo / y Beon :
Ziemiá / którą zwojował PAN / przed zgromádzeniem Izráelſkiem / jeſt źiemiá <i>ſpoſobna</i> dla bydłá / á my ſłudzy twoji mamy bydłá <i>wiele</i>.
Przetoż rzekli : Jesliſmy ználeźli łáſkę / przed ocżymá twymi / niechże będźie dáná tá źiemiá ſługom twym ná ośiádłość : á niech nie chodźiemy zá Jordan.
Tedy odpowiedźiał Mojzeƺ Synom Gádowym / y Synom Rubenowym : Więc bráćia wáƺy pójdą ná wojnę / á wy tu śiedźieć będźiećie?
Czemuż pſujećie ſerce Synom Izráelſkim / żeby nie ƺli do źiemie / którą im dał PAN.
Tákći ucżynili Ojcowie wáƺy / gdym je był poſłał z Kádes Bárne / ku przeƺpiegowániu tey źiemie :
Bo <i>gdy</i> przyƺli / áż do doliny Eſchol / obejrzawƺy onę źiemię / popſowáli ſerce Synom Izráelſkim / áby nie ƺli do źiemie / którą im dał PAN :
Zkąd / zápaliwƺy śię gniewem PAN / dniá onego przyśiągł / mówiąc :
Zájiſte nie oglądáją ludźie ći / którzy wyƺli z Egiptu / od dwudźieſtu lat y wyżey / tey źiemie / o którąm przyśiągł Abráhámowi / Izáákowi / y Jákóbowi / przeto iż mię cále nie náśládowáli :
Oprócż Kálebá Syná Jefunowego / Kenezeycżyká / y Jozuego / Syná Nunowego : ponieważ <i>ći</i> cále náśládowáli PAná.
Y zápalił śię gniewem PAN ná Izráelá / y ſpráwił że śię tułáli po puƺcży przez cżterdźieśći lat / áż poginął wƺyſtek on naród / który cżynił źle przed ocżymá PAńſkimi.
A oto <i>wy</i> powſtáliśćie / miáſto Ojców wáƺych / plemię ludźi grzeƺnych / ábyśćie jeƺcże przycżynili gniewu zápálcżywośći PAńſkiey / przećiwko Izráelowi :
Bo jeſli śię odwróćićie od náśládowánia jego / tedy on też zániecha go jeƺcże ná tey puƺcży : á ták wy zgubićie ten wƺyſtek lud.
Tedy przyſtąpiwƺy do niego rzekli : Obory bydłu y dobytkowi náƺemu / y miáſtá dźiatkom náƺym / tu pobudujemy :
Ale ſámi zbrojno / ochotnie <i>pójdźiemy</i> przed Syny Izráelſkimi : áż je záprowádźimy ná miejſce ich / á dźiatki náƺe będą mieƺkáły w mieśćiech obronnych / dla obywátelów tey źiemie :
Nie wróćimy śię do domów náƺych / áż pośiądą Synowie Izráelſcy káżdy dźiedźictwo ſwoje :
Ani weźmiemy dźiedźictwá z nimi zá Jordanem / y dáley / ponieważ przychodźi dźiedźictwo náƺe ná nas / z tey ſtrony Jordanu / ná wſchód ſłońcá.
Y rzekł im Mojzeƺ : Jeſliż ucżynićie cośćie rzekli / á pójdźiećie zbrojno przed oblicżem PAńſkim ná wojnę :
Y pójdźie káżdy z was zbrojno zá Jordan / przed oblicżnośćią PAńſką / áżby wypędźił nieprzyjaćioły ſwoje od oblicża ſwego :
Y áż będźie poddána źiemiá oná PAnu : á potym śię wróćićie / y będźiećie bez winy przed PAnem / y przed Izráelem : tedy wam będźie tá źiemiá zá ośiádłość przed oblicżē PAńſkim.
Ale jeſli tego nie ucżynićie / oto zgrzeƺyćie PAnu : á wiedzćie że grzech wáƺ znajdźie was.
Budujćież tedy miáſtá dźiatkom wáƺym y obory bydłu wáƺemu : á co wyƺło z uſt wáƺych ucżyńćie.
Tedy rzekli Synowie Gádowi / y Synowie Rubenowi do Mojzeƺá / mówiąc : Słudzy twoji ucżynią / jáko PAN náƺ rozkázuje :
Dźiatki náƺe / y żony náƺe / ſtáda náƺe / y wƺyſtkie bydłá náƺe / zoſtáną tu w mieśćiech Gáláádſkich :
Ale ſłudzy twoji pójdą wƺyſcy zbrojno przed PAnem ná wojnę / jáko PAN náƺ mówi.
Y przykazał o nich Mojzeƺ Eleázárowi Kápłanowi / y Jozuemu Synowi Nunowemu / y Kśiążętom Ojców / pokoleń ſynów Izráelſkich /
Y rzekł im / jesli przejdą Synowie Gádowi / y Synowie Rubenowi z wámi zá Jordan / wƺyſcy zbrojno ná wojnę przed PAnem : á będźie pohołdowána źiemiá przed wámi : Tedy im daćie źiemię Gáláád w dźiedźictwo :
Ale jeſli nie zbrojno z wámi przeydą / tedy niech máją dźiedźictwo miedzy wámi / w źiemi Chánánejſkiey.
Y odpowiedźieli Synowie Gádowi / y Synowie Rubenowi / mówiąc / co wyrzekł PAN do ſług ſwojich / to ucżynimy.
Pójdźiemy zbrojno przed PAnem / do źiemie Chánánejſkiey / á zoſtánie przy nas ośiádłość dźiedźictwá náƺego / z tę ſtronę Jordanu.
Dał tedy Mojzeƺ Synom Gádowym / y Synom Rubenowym y połowicy pokolenia Mánáſſeſſá / Syná Józefowego / króleſtwo Sehoná królá Amorrejſkiego / y króleſtwo Ogá królá Báſáńſkiego / źiemię z miáſty jey / z gránicámi / <i>y</i> miáſtá źiemie oney w około.
Y pobudowáli Synowie Gádowi : Dybon / y Atárot / y Aroer.
Y Atrot / Sofán / y Jázer / y Jegbá /
Y Betnimerá / y Betárán / miáſtá obronne : y obory dla bydłá :
Synowie też Rubenowi pobudowáli / Heſebon / y Eleále / y Káriátháim.
Y Nebo y Báálmeon / odmieniwƺy im imioná : tákże Sábámá : y dáli imioná <i>inƺe</i> onym miáſtom / które pobudowáli.
Wtárgnęli też Synowie Máchyrá / Syná Mánáſſeſſowego / do Gáláád : á wźiąwƺy je wygnáli Amorrejcżyká / który tám mieƺkał.
Y dał Mojzeƺ Gáláád Máchirowi / Synowi Mánáſſeſſowemu / y mieƺkał w nim.
Potym Jáir / Syn Mánáſſeſſów / wtárgnął y pobrał wśi ich / które przezwał Cháwot Jáir.
Tákże Nobe wtárgnął / y wźiął Kánát / z jego wśiámi / y nazwał je Nobe / od imienia ſwego.
