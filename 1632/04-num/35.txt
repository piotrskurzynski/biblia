Y rzekł PAN do Mojzeƺá / ná polách Moábſkich / nád Jordanem <i>przećiw</i> Jerichowi / mówiąc :
Rozkáż Synom Izráelſkim / áby dáli Lewitom z dźiedźictwá ośiádłośći ſwojey / Miáſtá do mieƺkánia : y przedmieśćia / około miaſt ich oddaćie Lewitom :
Y będą mieli miáſtá ſobie do mieƺkánia / á przedmieśćia ich będą im dla bydłá ich / y dla májętnośći ich / y dla wƺyſtkiego dobytku ich.
A przedmieśćiá miaſt które daćie Lewitom / od muru miejſkiego pójdą ná tyśiąc łokiet wƺędy w około.
Przetoż wymierzyćie / <i>zá káżdym</i> miáſtem dwá tyśiącá łokiet ná wſchód ſłońcá / ná południe też dwá tyśiącá łokiet : tákże ná zachód dwá tyśiącá łokiet / y ná pułnocy dwá tyśiącá łokiet : á miáſto w pośrzodku będźie : tákoweć będą przedmieśćia miaſt ich.
A miedzy tymi miáſty które daćie Lewitom / ƺeść miaſt będą dla ućiecżki / które daćie áby tám ućiekał mężobójcá : á nád te daćie <i>im</i> cżterdźieśći miaſt y dwie.
Ták iż wƺyſtkich miaſt / które Lewitom daćie / będźie cżterdźieśći / y ośm miaſt / y z przedmieśćiámi ich.
A miaſt / które daćie z dźierzáw Synów Izráelſkich / od tych którzy więcey máją / więcey daćie / á od tych którzy mniey máją daćie mniey : káżdy według miáry dźiedźictwá ſwego / które pośiędźie / udźieli z miaſt ſwojich Lewitom.
Zátym rzekł PAN do Mojzeƺá / mówiąc :
Mów do Synów Izráelſkich / y rzecż im : Gdy przeydźiećie przez Jordan do źiemie Chánánejſkiey :
Poſtánowćież ſobie miáſtá : miáſtá dla ućiecżki mieć będźiećie / áby tám ućiekał mężobójcá / któryby zábił kogo z nieobacżenia :
A będą wam te miáſtá dla ućiecżki / przed powinowátym <i>zábitego</i> / áby nie dał gárdłá ten co zábił / pókiby nie ſtánął przed zgromádzeniem ná ſąd.
A miaſt które odłącżyćie / ƺeść miaſt dla ućiecżki mieć będźiećie.
Trzy miáſtá daćie z tę ſtronę Jordanu / á trzy miáſtá daćie w źiemi Chánánejſkiey / <i>te</i> miáſtá dla ućiecżki będą.
Synom Izráelſkim / y przychodniowi / y mieƺkájącemu miedzy nimi / będą te ƺeść miaſt do ućiecżki : áby tám ućiekł káżdy ktoby zábił cżłowieká z nieobacżenia.
Wƺákże / jesliby go żelázną bronią uderzył / ták żeby umárł / mężobójcą jeſt : śmierćią umrze on mężobójcá.
Albo / jeſliby májąc kámień w ręku / którymby mógł zábić / uderzył go / ták żeby umárł / mężobójcą jeſt : śmierćią umrze on mężobójcá.
Tákże jeſliby májąc w ręku drewno / którymby mógł zábić / uderzył go / y umárłby / mężobójcą jeſt : śmierćią umrze on mężobójcá :
Powinowáty zábitego / zábije tego mężobójcę / gdźiekolwiek śię z nim potka / on ſam zábije go.
A jeſliby kogo z nienawiśći popchnął / álbo nań cżym ćiſnął z záſadzki / á umárłby :
Albo jeſliby go z waśni uderzył ręką ſwoją / á umárłby : śmierćią umrze / ten który uderzył / mężobójcą jeſt : powinny zábitego / zábije mężobójcę / gdźiekolwiek go tráfi.
Ale jesliby z przygody / bez waśni / kogo popchnął : álboby nań ćiſnął cżymkolwiek nie umyślnie :
Albo jeſliby jákim kámieniem / od któregoby mógł umrzeć / rzućił nań z nieobacżenia / á umárłby / nie będąc mu nieprzyjaćielem / áni ƺukájąc jego złego :
Tedy rozſądek ucżyni zgromádzenie miedzy tym który zábił / á miedzy powinnym zábitego / według tego práwá.
Y wybáwi zgromádzenie mężobójcę tego / z rąk powinnego / onego zábitego : y każe mu śię wróćić zgromádzenie do miáſtá ućiecżki jego / gdźie był ućiekł : y támże będźie mieƺkał áż do śmierći Kápłaná nawyżƺego / który jeſt pomázány Olejkiem świętym.
A jeſliby wyƺedł mężobójcá zá gránice miáſtá ućiecżki ſwojey / do którego ućiekł :
Y tráfiłby go powinny zábitego / zá gránicą miáſtá ućiecżki jego / choćiażby zábił powinny zábitego / mężobójcę onego / nie będźie winien krwie /
Abowiem w mieśćie ućiecżki ſwojey mieƺkáć ma / áż do śmierći Kápłaná nawyżƺego : á po śmierći Kápłaná nawyżƺego / wróći śię on mężobójcá do źiemie ośiádłośći ſwojey.
A będźiećie to mieli zá uſtáwę práwną / w narodźiech wáƺych / we wƺyſtkich mieƺkániách wáƺych.
Ktobykolwiek chćiał zábić cżłowieká / zá świádectwem świádków / zábije mężobójcę : ále świádek jeden nie będźie mógł świádcżyć ná <i>ſkazánie</i> kogo ná śmierć.
Nie weźmiećie też okupu zá żywot mężobójce / który záſłużył śmierć / niech śmierćią umrze.
Nie weźmiećie też zapłáty od onego który ućiekł do miáſtá ućiecżki ſwojey / áby śię náwróćił ná mieƺkánie do źiemie ſwojey / pierwey niżby Kápłan umárł /
Byśćie nie ſplugáwili źiemie / w którey będźiećie : bo krew tákowa / ſplugáwiłáby źiemię : á źiemiá nie może być ocżyśćioná od krwie / która jeſt wylána ná niey / jedno krwią tego / który ją przelał.
Przetoż nie plugawćie źiemie w którey mieƺkaćie / w którey też ja mieƺkam / bom ja PAN / który mieƺkam miedzy Synámi Izráelſkimi.
