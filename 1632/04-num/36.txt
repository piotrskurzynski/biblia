Tedy przyſtąpili mężowie przedniejƺy z Ojców / Pokolenia Synów Gáláádá / Syná Máchyrowego / Syná Mánáſſeſſowego / z domów Synów Józefowych : y mówili przed Mojzeƺem / y przed Kśiążęty przedniejƺymi / Ojców Synów Izráelſkich / 
Y rzekli : <i>Tobie</i> pánu memu rozkazał PAN / ábyś podźielił źiemię w dźiedźictwo loſem / Synom Izráelſkim : nád to pánu memu rozkazano od PAná / ábyś dał dźiedźictwo Sálfáádá / brátá náƺego Córkam jego :
Które jeſliby kto <i>z inƺego</i> pokolenia Synów Izráelſkich <i>wźiął</i> zá żony / odjęte będźie ich dźiedźictwo / od dźiedźictwá Ojców náƺych / á przyłącży śię do dźiedźictwá <i>onego</i> pokolenia / do któregoby je wźięto <i>zá żony,</i> á ták z loſu dźiedźictwá náƺego ubędźie.
A gdy przydźie miłośćiwe láto Synom Izráelſkim / tedy przyłącżone będźie dźiedźictwo ich do dźiedźictwá <i>onego</i> pokolenia / do któregoby poƺły <i>zá mąż</i> / á ták od dźiedźictwá pokolenia Ojców náƺych odjęte będźie dźiedźictwo ich.
Tedy powiedźiał Mojzeƺ Synom Izráelſkim / według ſłowá PAńſkiego / mówiąc : Dobrze mówi pokolenie Synów Józefowych :
Toć <i>to</i> jeſt / co rozkazał PAN o Córkách Sálfáádowych / mówiąc : jáko śię im upodoba / niech idą zá mąż : tylko w domu pokolenia Ojców ſwojich niech idą zá mąż.
Aby nie było przenoƺone dźiedźictwo Synów Izráelſkich / z pokolenia ná pokolenie : bo káżdy z Synów Izráelſkich / zoſtawáć ma przy dźiedźictwie pokolenia Ojców ſwych :
Y káżda Córká któraby miáłá dźiedźictwo / z pokoleń Synów Izráelſkich / zá kogokolwiek z domu pokolenia Ojcá ſwego pójdźie / żeby otrzymáli dźiedźicżnie Synowie Izráelſcy / káżdy dźiedźictwo Ojców ſwych.
Bo nie ma być przenoƺone dźiedźictwo / z pokolenia ná pokolenie inƺe : ále káżdy przy dźiedźictwie ſwojim zoſtáć ma / z pokolenia Synów Izráelſkich.
Jáko <i>tedy</i> rozkzzał PAN Mojzeƺowi / ták ucżyniły Córki Sálfáádowe.
Bo / Máhálá / Therſá / y Heglá / y Melchá / y Noá / córki Sálfáádowe / ƺły zá mąż / zá Syny Stryjów ſwojich.
W domy Synów Mánáſſeſſá / Syná Józefowego / poƺły zá mąż : y ták zoſtáło dźiedźictwo ich / przy pokoleniu domu Ojcá ich.
Teć ſą przykazánia / y práwá / które rozkazał PAN przez Mojzeƺá Synom Izráelſkim / ná polách Moábſkich / nád Jordanem <i>przećiw</i> Jerichowi.
