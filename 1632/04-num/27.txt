Tedy przyƺły córki Sálfáádá Syná Heferowego / Syná Gáláádowego / Syná Máchirowego / Syná Mánáſſeſſowego / z pokolenia Mánáſſeſſá / Syná Józefowego : á te <i>ſą</i> imioná Córek jego. Máchlá / Noá / y Heglá / y Melchá / y Terſá.
Y ſtánęły przed Mojzeƺem / y przed Eleázárem Kápłanem / y przed Kśiążęty / y wƺyſtkim zgromádzeniem / u drzwi Namiotu zgromádzenia / y rzekły.
Oćiec náƺ umárł ná puƺcży / á on nie był w poczćie tych / którzy śię byli przećiw PAnu zbuntowáli / w ſpiknieniu Korego : ále dla grzechu ſwego umárł / nie májąc Synów :
Czemużby zginąć miáło imię Ojcá náƺego z domu jego / przeto że nie miał Syná? dajćie nam dźiedźictwo miedzy bráćią Ojcá náƺego.
Tedy odnióſł Mojzeƺ ſpráwę ich do PAná.
Y rzekł PAN do Mojzeƺá / mówiąc :
Dobrze mówią córki Sálfáádowe : Daj im koniecznie ośiádłość dźiedźictwá miedzy bráćią Ojcá ich : á przenieś dźiedźictwo Ojcá ich ná nie.
Synom tákże Izráelſkim powiedz mówiąc : Gdyby kto umárł nie májąc Syná / tedy przenieśiećie dźiedźictwo jego ná córkę jego.
A jesliby nie miał y córki / tedy dajćie dźiedźictwo jego / bráćij jego :
A jesliby y bráćij nie miał / tedy daćie dźiedźictwo jego / bráćij Ojcá jego :
A jesliby nie było bráćij Ojcá jego / tedy daćie dźiedźictwo jego / pokrewnemu jego / nabliżƺemu jego / z domu jego / áby je odźiedźicżył. A będźie to Synom Izráelſkim zá uſtáwę práwną / jáko rozkazał PAN Mojzeƺowi.
Potym rzekł PAN do Mojzeƺá : wſtąp ná tę górę Abárim / á oglądaj źiemię którąm dał Synom Izráelſkim /
A gdy ją oglądaƺ / przyłącżon będźieƺ do ludu twego y ty / jáko jeſt przyłączony Aáron brát twój :
Przeto żeśćie byli odpornymi ſłowu mojemu ná puƺcży Syn / przy poſwarku zgromádzenia : y nie poświęćiliśćie mię przy wodách / przed oczymá ich. Oneć <i>to ſą</i> wody poſwarku / w Kádes / ná puƺczy Syn.
Tedy rzekł Mojzeƺ do PAná / mówiąc :
Niech opátrzy PAN / Bóg duchów wƺelkiego ćiáłá / mężem <i>godnym</i> to zgromádzenie /
Któryby wychodźił przed nimi / y któryby wchodźił przed nimi / y któryby je wywodźił / y któryby je przywodźił : áby nie był lud PAńſki jáko owce nie májące Páſterzá :
Tedy rzekł PAN do Mojzeƺá : Weźmi do śiebie Jozuego / Syná Nunowego / mężá / w którym jeſt Duch <i>mój</i> / á włóż nań rękę ſwoję /
Y poſtaw go przed Eleázárem Kápłanem / y przed wƺyſtkim zgromádzeniem : á daƺ mu náukę przed ocżymá ich /
A udźieliƺ mu zacnośći ſwey / áby <i>go</i> ſłucháło wƺyſtko zgromádzenie Synów Izráelſkich.
Który przed twarzą Eleázárá Kápłaná ſtawáć będźie : áby śię zań rádźił ſądu Urim przed PAnem. Ná rozkazánie jego wychodźić będą / y ná rozkazánie jego wchodźić będą / on / y wƺyſcy Synowie Izráelſcy z nim / y wƺyſtko zgromádzenie.
Uczynił tedy Mojzeƺ jáko mu był rozkazał PAN : á wźiąwƺy Jozuego / poſtáwił go przed Eleázárem Kápłanem / y przed wƺyſtkim zgromádzeniem /
Y włożywƺy nań ręce ſwe / dał mu náukę / jáko mówił PAN przez Mojzeƺá.
