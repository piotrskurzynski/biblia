Y rzekł PAN do Mojzeƺá / mówiąc :
Rozkaż Synom Izráelſkim / á powiedz im : Ofiáry mojey chlebá mego / w ofiárách mojich ogniſtych / ná wdźięcżność wonnośći mojey / przeſtrzegáć będźiećie / ábyśćie mi je ofiárowáli czáſu ſwego.
Y rzecżeƺ do nich : Táć jeſt Ofiárá ogniſta / którą ofiárowáć będźiećie PAnu : Báránki rocżne / zupełne / dwá / ná káżdy dźień / ná cáłopalenie uſtáwicżne :
Báránká jednego ofiárowáć będźieƺ poránu : á báránká drugiego ofiárowáć będźieƺ miedzy dwiemá wieczorámi :
Do tego / dźieśiątą cżęść Efy / mąki pƺenney / ná Ofiárę śniedną / nágniatáney z Oliwą cżyſtą / z cżwartą cżęśćią Hinu.
<i>Toć jeſt</i> cáłopalenie uſtáwicżne / jákie było ſpráwowáne ná górze Synáj / ná wdźięcżną wonność : ogniſta <i>to</i> Ofiárá PAnu.
A Ofiárá jey mokra będźie / cżwarta część Hinu do káżdego báránká / w świątnicy ſpráwowáć będźieƺ Ofiárę mokrą z mocnego napoju PAnu.
A drugiego báránká ofiárowáć będźieƺ miedzy dwiemá wiecżorámi / jáko Ofiárę śniedną poránną / y jáko Ofiárę mokrą jey / ofiárowáć będźieƺ ná Ofiárę ogniſtą / ku wdźięcżney wonnośći PAnu.
Ale w dźień Sábbátu / <i>ofiárowáć będźieƺ</i> dwá báránki rocżne / zupełne / y dwie dźieśiąte częśći <i>Efy</i> mąki pƺenney / z Oliwą nágniećioney / ná Ofiárę śniedną / y z mokrą jey ofiárą.
<i>Toć jeſt</i> cáłopalenie Sobotnie / w káżdy Sábbát / oprócz cáłopalenia uſtáwicżnego / y mokrej Ofiáry jego.
A ná nowiu mieśiąców wáƺych / ofiárowáć będźiećie cáłopalenie PAnu : ćielców młodych dwu / y báráná jednego : báránków rocżnych zupełnych śiedm /
Y trzy dźieśiąte cżęśći <i>Efy</i> mąki pƺenney ná Ofiárę śniedną / nágniećioney z Oliwą / do káżdego ćielcá : y dwie dźieśiąte częśći pƺenney mąki / ná Ofiárę śniedną / zágniećioney z Oliwą / do káżdego báráná :
A jednę dźieśiątą cżęść mąki pƺenney / zágniećioney z Oliwą ná Ofiárę śniedną / do káżdego báránká <i>ná</i> cáłopalenie ku wdźięcżney wonnośći / ná Ofiárę ogniſtą PAnu.
Tákże mokre ich Ofiáry / z winá / puł Hinu będźie do káżdego ćielcá / á trzećia część Hinu / do báráná / cżwarta záś część Hinu / do káżdego báránká. Toć <i>jeſt</i> cáłopalenie ná nowiu mieśiącá / káżdego mieśiącá / przez rok.
Kozłá też jednego / z ſtádá / zá grzech / ofiárowáć będźiećie PAnu / oprócz uſtáwicżnego cáłopalenia / y mokrej Ofiáry jego.
Ale mieśiącá pierwƺego / w dźień cżternaſty <i>tegoż</i> mieśiącá / święto Prześćiá <i>będźie</i> PAnu.
A w piętnaſty dźień tegoż mieśiącá / uroczyſte święto <i>będźie</i> : przez śiedm dni <i>chleby</i> przáſne jeść będźiećie.
Pierwƺego dniá / zgromádzenie święte : żadney roboty ſłużebnicżey nie będźiećie czynić <i>weń</i> :
Ale ofiárowáć będźiećie Ofiárę ogniſtą ná cáłopalenie PAnu : dwu ćielców młodych / y báráná jednego / y śiedm báránków rocżnych : zupełni niech wam będą.
A ná Ofiárę ich śniedną / pƺenney mąki / nágniećioney z Oliwą / trzy dźieśiąte cżęśći <i>Efy</i> do káżdego ćielcá / á dwie dźieśiąte cżęśći / do káżdego báráná / ofiárowáć będźiećie.
Jednę dźieśiątą cżęść / ofiárowáć będźieƺ / przy káżdym báránku / z onych śiedmi báránków.
Kozłá też jednego / ná Ofiárę zá grzech / ku oczyśćieniu was.
Nád cáłopalenie poránne / które ma być cáłopalenie uſtáwicżne / ofiárowáć to będźiećie.
Ták ofiárowáć będźiećie káżdego dniá / przez onych śiedm dni / pokarm Ofiáry ogniſtey / ná wdźięcżną wonność PAnu : oprócż cáłopalenia uſtáwicznego / y mokrey Ofiáry jego.
A dniá śiódmego / święte zgromádzenie mieć będźiećie. Żadney roboty ſłużebniczey nie będźiećie cżynić.
W dźień záś pierwoćin / gdy będźiećie ofiárowáli nową śniedną Ofiárę PAnu / gdy śię wypełnią tygodnie wáƺe / zgromádzenie święte mieć będźiećie : żadney roboty ſłużebniczey cżynić nie będźiećie.
Y ofiárowáć będźiećie cáłopalenie ku wdźięcżney wonnośći PAnu / dwu ćielców młodych / báráná jednego / śiedm báránków rocżnych.
A ná Ofiárę śniedną ich / mąki pƺenney nágniećioney z Oliwą / trzy dźieśiąte częśći <i>Efy</i> do káżdego ćielcá / dwie dźieśiąte częśći do káżdego báráná.
Jednę dźieśiątą część do káżdego báránká / z onych śiedmi báránków.
Kozłá jednego z kóz / ná oczyśćienie was.
Oprócż cáłopalenia uſtáwicznego / y Ofiáry śniedney jego ofiárowáć <i>to</i> będźiećie. <i>Te rzecży</i> zupełne niech wam będą / y z mokrymi Ofiárámi ich.
