Potym rzekł PAN do Aároná : Ty y Synowie twoji / y dom Ojcá twego z tobą / ponieśiećie niepráwość świątnice. Y ty / y Synowie twoi z tobą / ponieśiećie niepráwość kápłáńſtwá wáƺego.
Bráćią tákże twoję pokolenie Lewiego / ród Ojcá twego przypuśćiƺ do śiebie / áby byli przy tobie / y poſługowáli tobie : á ty / y Synowie twoji z tobą / <i>ſłużyć będźiećie</i> przed Namiotem świádectwá :
Oni będą przeſtrzegáli rozkazánia twego / y pilnowáli wƺyſtkiego Namiotu : wƺákże do nacżynia świątnice / y do Ołtarzá / przyſtępowáć nie będą / áby nie pomárli / y oni / y wy.
Y przyłączą śię do ćiebie pilnie ſtrzegąc Namiotu zgromádzenia / w káżdey uſłudze Namiotu : á <i>nikt</i> obcy niechaj śię nie mięƺa miedzy was.
<i>Wy</i> tedy pilnie ſtrzeżćie świątnice / y uſługi Ołtarzowey / by śię nápotym nie wzruƺył gniew / przećiwko Synom Izráelſkim.
Bom / oto / ja obrał bráćią wáƺę Lewity / z pośrzodku Synów Izráelſkich / wam zá dar / oddáne PAnu / áby odpráwowáli uſługę w Namiećie zgromádzenia.
Ale ty / y Synowie twoji z tobą przeſtrzegáć będźiećie Kápłáńſtwá wáƺego / przy káżdey uſłudze Ołtarzowey / y zá zaſłoną ſłużyć będźiećie : urząd Kápłáńſtwá wáƺego dałem <i>wam</i> dármo : gdyby kto obcy przyſtąpił umrze.
Nád to mówił PAN do Aároná. Otom ja dał tobie pod ſtraż Ofiáry moje / podnoƺone : wƺyſtkie rzecży poświęcone / od Synów Izráelſkich / tobiem je dał / dla pomázánia / y Synom twojim práwem wiecżnym.
To będźie twoje z rzecży poświęconych / które nie bywáją palone. Káżda ofiárá ich / bądź Ofiárá śniedná ich / bądź Ofiárá zá grzech ich / álbo Ofiárá zá wyſtępek ich / cokolwiek mi oddawáć będą / to będźie rzeczą poświęconą / tobie y Synom twojim.
Ná miejſcu naświętƺym jadáć to będźieƺ / wƺelki mężcżyzná będźie jadł z tego / rzecżą poświęconą to będźie tobie.
To też twoje <i>będźie,</i> Ofiárá podnoƺenia dárów ich / ze wƺyſtkiemi Ofiárámi tám y ſám obracánia Synów Izráelſkich / tobiem je dał y Synom twojim / y córkam twojim z tobą / práwem wiecżnym : Káżdy czyſty w domu twojim będźie je jadł.
Káżdą co naprzednieyƺą Oliwę / y káżde co nalepƺe wino / y zboże / pierwiaſtki ich / które ofiárują PANU / tobiem je dał.
Pierwoćiny ze wƺyſtkich rzecży które <i>będą</i> w źiemi ich / które przynioſą Pánu / twoje będą : Káżdy czyſty w domu twojim / jadáć je będźie.
Wƺyſtko <i>cokolwiek</i> poślubione w Izráelu / twoje będźie.
Cokolwiek otwiera żywot wƺelkiego ćiáłá / á bywa ofiárowáne Pánu / ták z ludźi jáko z bydłá / twoje będźie : ále pierworodne z ludźi okupiƺ / tákże pierworodne niecżyſtego bydłá okupiƺ.
A okup jego gdy mu mieśiąc minie / daƺ według ƺácunku twego / pięć Syklów śrebrá / według Syklá świątnice : dwádźieśćiá pieniędzy <i>waży</i> Sykiel.
Ale pierworodnego od krowy / álbo pierworodnego od owcy / álbo pierworodnego od kozy / nie daƺ ná okup : <i>bo</i> poświęcone ſą : krew ich wylejeƺ ná Ołtarz / á tłuſtość ich zápaliƺ ná Ofiárę ogniſtą dla wdźięczney wonnośći Pánu.
Ale mięſo ich twoje będźie : jáko moſtek podnoƺenia / y jáko łopátká práwa / twoje będą.
Wƺyſtkie Ofiáry podnoƺenia / z rzecży poświęconych / które przynośić będą Synowie Izráelſcy Pánu / dałem tobie y Synom twym / y córkom twojim z tobą / práwem wiecżnym : uſtáwá to nie náruƺona <i>y</i> wiecżna przed Pánem / tobie y naśieniu twemu z tobą.
Potym mówił PAN do Aároná : W źiemi ich / dźiedźictwá mieć nie będźieƺ / y dźiału nie będźieƺ miał miedzy nimi : jam dźiał twój / y dźiedźictwo twoje / w pośrzód Synów Izráelſkich.
Synom záś Lewiego otom dał wƺyſtkę dźieśięćinę w Izráelu dźiedźicznie / zá ſłużbę ich / którą wykonywáją / ſłużąc około Namiotu zgromádzenia.
A niechaj nieprzyſtępują więcey Synowie Izráelſcy do Namiotu zgromádzenia / áby nie ponieśli grzechu / <i>y</i> nie pomárli :
Ale ſámi Lewitowie / odpráwowáć będą uſługę około Namiotu zgromádzenia / y ſámi ponioſą niepráwość ſwoję. Uſtáwá to wiecżná <i>będźie</i> w narodźiech wáƺych / áby w pośrzód Synów Izráelſkich / dźiedźictwá nie mieli.
Abowiem dźieśięćiny Synów Izráelſkich / które przynośić będą ná Ofiárę podnoƺenia Pánu / dałem Lewitom zá dźiedźictwo : przetoż rzekłem o nich / w pośrzód Synów Izráelſkich nie będą mieli dźiedźictwá.
Potym PAN rzekł do Mojzeƺá / mówiąc :
Mów też do Lewitów / á powiedz im : Gdy weźmiećie od Synów Izráelſkich dźieśięćiny / którem ja wam dał od nich / zá dźiedźictwo wáƺe / tedy ofiárowáć będźiećie <i>Ofiárę</i> podnoƺeniá Pánu / dźieśiątą cżęść dźieśięćin.
A poczytána wam będźie tá Ofiárá wáƺá / jáko zboże z bojewiſká / y jáko obfitość <i>winá</i> z práſy.
Ták wy też ofiárowáć będźiećie Ofiárę podnoƺenia PAnu / ze wƺyſtkich dźieśięćin wáƺych / które weźmiećie od Synów Izráelſkich / á oddaćie z nich Ofiárę podnoƺenia Pánu / Aáronowi Kápłanowi :
Ze wƺyſtkich dochodów wáƺych ofiárowáć będźiećie wƺelką Ofiárę podnoƺenia Pánu : ze wƺyſtkiego co nalepƺego jeſt / ofiárujćie cząſtkę poświęconą.
Powieƺ im też : Gdy oddawáć będźiećie z tego co nalepƺego jeſt / tedy pocżytáno będźie Lewitom jáko urodzáje z bojewiſká / y jáko urodzaj z práſy.
Y będźiećie to jeść ná káżdym miejſcu / wy y czeladź wáƺá : ábowiem <i>to</i> jeſt zapłátá wáƺá zá ſłużbę wáƺę przy Namiećie zgromádzenia :
Y nie ponieśiećie zá to grzechu / gdy ofiárowáć będźiećie co nalepƺego jeſt z tego / y nie ſplugáwićie rzecży poświęconych od Synów Izráelſkich / y nie pomrzećie.
