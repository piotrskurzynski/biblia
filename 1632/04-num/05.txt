Y rzekł PAN do Mojzeƺá / mówiąc :
Rozkaż Synom Izráelſkim / áby wyrzućili z Obozu káżdego trędowátego / y káżdego który ćierpi płynienie naśienia / y káżdego który śię ſplugáwił nád umárłym :
Ták mężcżyznę jáko y niewiáſtę wyrzućićie : precz zá Obóz wyrzućićie je / áby nie ſplugáwili Obozu tych / miedzy którymi ja mieƺkam.
Y uczynili ták Synowie Izráelſcy / á wygnáli je precz zá Obóz : jáko rozkazał PAN Mojzeƺowi / ták uczynili Synowie Izráelſcy.
Nád to rzekł PAN do Mojzeƺá / mówiąc :
Powiedz Synom Izráelſkim : Mąż álbo niewiáſtá / gdyby popełnili jákikolwiek grzech ludzki : dopuśćiwƺy śię wyſtępku przećiwko PAnu / á byłáby winna oná duƺá :
Tedy wyznáją grzech ſwój którego śię dopuśćili : y wrócą to wczymby winni byli cále / á przydawƺy jeƺcze nád to piątą część / oddádzą onemu / przećiw któremu zgrzeƺyli.
A jesliby nie było tego / komuby ƺkodę trzebá nádgrodźić / oná ƺkodá oddána będźie PAnu : <i>y zoſtánie</i> Kápłanowi / oprócz báráná oczyśćienia / przez którego ma być oczyśćiony.
Káżda też Ofiárá podnoƺenia / ze wƺech rzeczy poświęconych od Synow Izráelſkich / którą przynioſą do Kápłaná / jemu śię doſtánie.
<i>Owa</i> rzeczy poświęcone od kogożkolwiek / jego będą / y ktoby co oddał Kápłanowi / jemu zoſtánie.
Zátym rzekł PAN do Mojzeƺá / mówiąc :
Mów do Synow Izráelſkich / á powiedz im : Káżdy mąż któregoby żoná wyſtąpiłá / y dopuśćiłáby śię grzechu przećiwko niemu /
A złączyłby śię inƺy z nią / złączeniem naśienia / á byłoby to ſkryte przed oczymá mężá jey / y tájiłáby śię / będąc ſplugáwioną : á świádkáby nie było przećiwko niey / ániby jey záſtano :
Jednák przypadłby nań duch zápálczywośći / y miałby w podejrzeniu żonę ſwą / któraby ſplugáwiona byłá : álbo żeby przypadł nań duch zápálczywośći / y miałby w podejrzeniu żonę ſwą / któraby ſplugáwiona nie byłá :
Tedy przywiedźie on mąż żonę ſwoję do Kápłaná / y przynieśie z nią Ofiárę jey dźieśiątą część Efy / mąki jęczmienney : nie lejąc ná nię Oliwy / áni kłádąc ná nię kádźidłá : ábowiem jeſt Ofiárá podejrzenia / Ofiárá śniedna / pámiętna / przywodząca ná pámięć grzech.
A ták będźie ją Ofiárował Kápłan / y ſtáwi ją przed oblicżnośćią PAńſką :
Y weźmie Kápłan wody świętey w naczynie gliniáne : y prochu / który będźie ná tle Przybytku / weźmie Kápłan / á wſypie do wody.
Potym poſtáwi Kápłan Niewiáſtę przed PAnem / y odkryje głowę niewiáſty / á da w ręce jey Ofiárę śniedną / pámiętną : Ofiárá to śniedna podejrzenia : á Kápłan będźie miał w ręce wodę gorzką przeklęctwá :
Y poprzyśięże ją Kápłan / y rzecze do niewiáſty : Jesli nie ſpał kto <i>inƺy</i> z tobą / á jesliś śię nie unioſłá w grzech nieczyſty przy mężu ſwym / bądź nie náruƺona od tey wody gorzkiey przeklęctwá :
Ale jeſliżeś uſtąpiłá od mężá twego / y jeſteś ſplugáwiona / á kto inny ſpał z tobą oprócz mężá twego /
Tedy poprzyśięże Kápłan niewiáſtę onę przyśięgą przeklęctwá / y rzecze do niey : Niechaj ćię poda PAN ná złorzeczenie / y ná przeklinánie miedzy ludem twoim / przepuśćiwƺy <i>áby</i> łono twoje wypádło / y żywot twój opuchł :
Niechże przenikną te wody przeklęte / wnętrznośći twoje / áby opuchł żywot twój / y wypádło łono <i>twoje :</i> y odpowie niewiáſtá / Amen / Amen.
Tedy nápiƺe te przeklęctwá Kápłan ná kśięgách / á omyje je oną wodą gorzką.
Y da śię nápić Niewieśćie wody gorzkiey przeklęctwá / y przenikną ją wody przeklęctwá / y obrócą śię w gorzkość.
Potym weźmie Kápłan z rąk Niewiáſty oney Ofiárę śniedną podejrzenia / á będźie ją podnośił przed PAnem / Ofiárując ją ná Ołtarzu :
Weźmie też Kápłan garść pámiętnego z Ofiáry śniedney / y ſpali to ná Ołtarzu / potym da wypić wodę niewieśćie.
A gdy śię jey da nápić oney wody / ſtánie śię / jesliby ſplugáwiona byłá / y wyſtąpiłá grzechem przećiwko mężowi ſwemu / że ją przenikną wody przeklęctwá / y obrócą śię w gorzkość / y opuchnie żywot jey / y wypádnie łono jey / y ſtánie śię Niewiáſtá oná przeklęctwem miedzy ludem ſwoim.
A jesliby nie byłá ſplugáwiona Niewiáſtá / áleby czyſta była : niewinna zoſtánie / y dźiatki rodźić będźie.
Táć jeſt uſtáwá Podejrzenia / gdyby uſtąpiłá żoná od mężá ſwego / y byłáby ſplugáwiona.
Albo żeby ná mężá przypadł duch zápálczywy / á miałby w podejrzeniu żonę ſwoję / y poſtáwiłby ją przed PAnem / á uczyniłby z nią Kápłan wƺyſtko według tey uſtáwy :
Tedy nie będźie mąż on winien grzechu / ále niewiáſtá oná ponieśie niepráwość ſwoję.
