A gdy uſłyƺał Chánánejczyk / Król Hárád / który mieƺkał ná południe / że Izráelczycy ćiągnęli oną drogą / którą byli ƺpiegowie <i>przeƺli,</i> tedy zwiódł bitwę z Izráelem / y pojimał ich wiele.
Tám uczynił Izráel ślub PAnu / mówiąc : Jeſliże podaƺ lud ten w ręce moje / do gruntu wywrócę miáſtá ich.
Y wyſłuchał PAN głos Izráelá / á podał mu Chánánejczyki / y wytráćił je z gruntu / y miáſtá ich : á názwał imię onego miejſcá / Chormá.
Potym śię ruƺyli od góry Hor / drogą ku morzu czerwonemu / áby obeƺli źiemię Edomſką : y utrudźił śię lud bárzo <i>w oney</i> drodze :
Przetoż mówił lud przećiw Bogu / y przećiw Mojzeƺowi : Przeczżeśćie nas wywiedli z Egiptu / ábyſmy pomárli ná tey puƺczy? bo niemáƺ chlebá áni wody / á duƺá náƺá obrzydźiłá ſobie ten chleb nikczemny.
Przetoż przepuśćił PAN ná lud węże ogniſte / którzy kąſáli lud : y pomárło wiele ludu z Izráelá.
Y przyƺedƺy lud do Mojzeƺá / rzekli : Zgrzeƺyliſmy / żeſmy mówili przećiw PAnu / y przećiw tobie : Módl śię PAnu / áby oddalił od nas te węże. Y modlił śię Mojzeƺ zá ludem.
Y rzekł PAN do Mojzeƺá : Uczyń ſobie wężá miedźiánego / á wyſtaw go ná drzewcu : y ſtánie śię / ktokolwiek ukąƺony będąc wejrzy nań / że żyw zoſtánie.
Spráwił tedy Mojzeƺ wężá miedźiánego / y wyſtáwił go ná drzewcu : y było to / gdy kogo wąż ukąśił / á pojrzał ná wężá miedźiánego / że żyw zoſtał.
Zátym ruƺyli śię Synowie Izráelſcy / á ſtánęli Obozem w Obot.
A z Obot ruƺywƺy śię / położyli śię Obozem ná págórkách <i>gór</i> Hábárim / ná puƺczy / która jeſt przećiw Moábczykom od wſchodu słońcá.
<i>A</i> odƺedƺy z támtąd / położyli śię Obozem nád potokiem Zered.
Z támtąd odćiągnąwƺy / położyli śię Obozem u brodu Arnon / który jeſt ná puƺczy / á wychodźi z gránice Amorrejſkiey. Abowiem Arnon jeſt gránicá Moábſka miedzy Moábczykiem y Amorreyczykiem.
Przetoż mówi śię w kśięgách wojen PAńſkich / Przećiwko Wáhebowi w wichrze walczył / y przy potokách Arnon :
Bo śćiekánie tych potoków / które śię náchyliło ku położeniu Hár / to śię śćiąga / ku gránicy Moábſkiey.
Z támtąd potym przyƺli do Beer : á táć jeſt oná ſtudnia / o którey mówił PAN do Mojzeƺá : zgromadź lud / á dam im wody.
Tedy śpiewał Izráel tę pioſnkę : Wyſtąp ſtudnio. Śpiewajćież o niey :
Studnia / którą wykopáli Kśiążętá : wykopáli ją Hetmáni ludu / z uſtawcą zakonu / laſkámi ſwojemi. A z tey puƺczy <i>ruƺyli śię</i> do Mátáná :
A z Mátáná do Náhálielu : á z Náhálielu / do Bámothu :
A z Bámothu / ku Hágáy / które <i>jeſt</i> w polách Moábſkich / ná wierzchu págórku / który leży ku puƺczy.
Y poſłał Izráel poſły do Sehoná Królá Amorreyſkiego / mówiąc :
Niech przejdźiemy przez źiemię twoję : nie pójdźiemy áni przez polá / áni przez winnice : nie będźiemy pić wód z ſtudźien <i>twojich</i> : gośćińcem pójdźiemy / áż przejdźiemy gránice twoje.
Ale nie pozwolił Sehon Izráelowi iść przez gránice ſwoje : Y zebráwƺy Sehon wƺyſtek lud ſwój / wyćiągnął przećiw Izráelowi / ná puƺczą. A gdy przyƺedł do Jáházy zwiódł bitwę z Izráelem.
Y poráźił go Izráel oſtrzem mieczá / y odźiedźiczył źiemię jego / od Arnonu / áż do Jáboku / <i>y</i> áż do <i>źiemie</i> Synów Ammonowych : ábowiem opátrzone były gránice Ammonitów.
Tedy pobrał Izráel wƺyſtkie miáſtá one / y mieƺkał we wƺyſtkich miáſtách Amorrejſkich / w Heſebon / y we wƺyſtkich wśiách jego.
Bo Heſebon było miáſto Sehoná Królá Amorrejſkiego / który gdy pierwey walczył z Królem Moábſkim / wźiął mu był wƺyſtkę źiemię jego / z rąk jego / áż po Arnon :
Dla tegoż mówią w przypowieśći : Pódźćie do Heſebon / á niech zbudują y nápráwią miáſto Sehonowe.
Abowiem wyƺedł ogień z Heſebon / á płomień z miáſtá Sehonowego / y popalił Ar Moábſkie / y obywátele wyſokich miejſc Arnon.
Biádá tobie Moáb / zginąłeś o ludu Chámos podał Syny ſwoje ná ućiekánie / y córki ſwoje do więźienia / Królowi Amorrejſkiemu / Sehonowi.
A záginęło pánowánie ich od Heſeboná áż do Diboná / á poburzyliſmy <i>je</i> áż do Nofe / które <i>idźie</i> áż do Medády.
Y mieƺkał Izráel w źiemi Amorrejſkiey.
Tedy poſłał Mojzeƺ ná ƺpiegi do Jázer / którego wśi pobráli wypędźiwƺy Amorreyczyki / którzy tám <i>byli</i>.
Potym obróćiwƺy śię / ƺli ku Báſán : gdźie wyćiągnął Og / Król Báſáńſki przećiwko nim / ſam y wƺyſtek lud jego / áby z nimi ſtoczył bitwę w Edrej.
Tedy rzekł PAN do Mojzeƺá : Nie bój śię go / bo w ręce twoje podałem go / y wƺyſtek lud jego / y źiemię jego : y uczyniƺ mu / jákoś uczynił Sehonowi Królowi Amoreejſkiemu / który mieƺkał w Heſebon.
Y poráźili go y Syny jego / ze wƺyſtkim ludem jego / ták yż nikogo z niego nie zoſtáwili / y posiedli dźiedźicznie źiemię jego.
