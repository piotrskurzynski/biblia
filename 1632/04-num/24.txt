A <i>gdy</i> baczył Báláám że śię podobáło PAnu / áby błogoſłáwił Izráelowi / już nie chodźił / jáko przed tym / raz y drugi dla wieƺczby / ále obróćił ku puƺczy twarz ſwoję.
A podniózƺy Báláám oczy ſwe / obaczył Izráelá mieƺkájącego według pokoleń ſwojich / y był nád nim Duch Boży.
Y záczął przypowieść ſwoję / á mówił :
Rzekł Báláám Syn Beorów / rzekł mąż którego oczy ſą otworzone : Rzekł ſłyƺący wymowy Boże / á który widzenie wƺechmocnego widźiał / <i>który</i> kiedy pádnie / otworzone ma oczy :
Jáko piękne ſą namioty twoje o Jákóbie! przybytki twoje o Izráelu!
Jáko potoki rozćiągnęły śię / jáko Ogrody przy rzece / jáko drzewá wonne / które PAN náſádźił / jáko Cedry nád wodámi :
Popłynie wodá z wiádrá jego / á naśienie jego <i>będźie</i> nád wodámi obfitymi / á będźie wywyƺon nád Agágá Król jego / á wynieśie śię króleſtwo jego.
Bóg wywiódł go z Egiptu / mocą jednorożcową był mu : pożrze Narody przećiwne ſobie / á kośći ich pokruƺy / y ſtrzałámi ſwemi / przeráźi <i>je</i>.
Położył śię / leży jáko lwię y jáko lew śilny / któż go obudźi? ktobyć błogoſłáwił / błogoſłáwiony : á ktoby ćię przeklinał / przeklęty <i>będźie</i>.
Tedy śię zápalił gniew Báláków ná Báláámá / á kláſnąwƺy rękámi ſwymi / rzekł Bálák do Báláámá : Dla złorzeczenia nieprzyjaćiołom mojim przyzwałem ćię / á otoś im błogoſłáwił już po trzy kroć :
Przetoż teraz uchodź ná miejſce ſwoje : rzekłemći był / zacnie ćię uczczę / ále oto pozbáwił ćię PAN <i>tey</i> czći.
Y rzekł Báláám do Báláká : zażem y poſłom twoim / któreś ſłał do mnie / nie powiedźiał / mówiąc :
Choćby mi dał Bálák pełen dom ſwój śrebrá y złotá nie będę mógł przeſtąpić ſłowá PAńſkiego / ábym czynił <i>co</i> dobrego álbo złego ſam z śiebie : co mi opowie PAN / to będę mówił :
A teraz oto ja odchodzę do ludu mego / jednák oznájmięć / co uczyni lud ten / ludowi twemu / ná potym.
Y zácżął przypowieść ſwoję / y rzekł : Mówił Báláám Syn Beorów : mówił mąż / którego oczy ſą otworzone :
Mówił ten / który ſłyƺał wyroki Boże / á który ma umiejętność Nawyżƺego : <i>który</i> widźiał widzenie wƺechmocnego : który kiedy pádnie / otworzone ma oczy.
Ujrzę go / ále nie teraz : oglądam go / ále nie z bliſká : wynidźie gwiazdá z Jákóbá / y powſtánie laſká z Izráelá / y pobije Kśiążętá Moábſkie / y wytráći wƺyſtkie Syny Setowe :
Y przydźie Edom w opánowánie / á Seijr będźie pod władzą nieprzyjaćiół ſwojich / á Izráel będźie ſobie poczynał mężnie.
Y będźie pánował / <i>który wynidźie</i> z Jákóbá / á wytráći oſtátki z miaſt.
A gdy pojrzał ná Amáleká / záczął przypowieść ſwoję / y rzekł : początek Narodów <i>jeſt</i> Amálek / ále oſtátek jego do ƺczętu záginie.
Potym wejrzawƺy ná Kenejczyká / záczął przypowieść ſwoję / y rzekł : Mocneć jeſt mieƺkánie twoje / á záłożyłeś ná ſkále gniazdo twoje.
Wƺákże ſpuſtoƺony będźie Kenejczyk / áż ćię Aſſur záprowádźi do więźienia.
Znowu záczął przypowieść ſwoję / y rzekł : Ach któż będźie żyw / gdy to uczyni Bóg?
Bo okręty <i>przypłyną</i> od brzegów Chittimſkich / y utrapią Aſſyriány / utrapią y Hebrejczyki : ále też ſáme do ƺczętu záginą.
Wſtał tedy Báláám y odƺedł / á wróćił śię ná miejſce ſwoje : tákże y Bálák poƺedł w drogę ſwą.
