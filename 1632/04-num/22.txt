Z támtąd ruƺyli śię Synowie Izráelſcy / y położyli śię obozem ná polách Moábſkich / z tey ſtrony Jordanu / <i>przećiw</i> Jerichowi.
A widząc Bálák / Syn Seforów / wƺyſtko / co uczynił Izráel Amorrejczykowi :
Ulękł śię Moáb tego ludu wielce / bo go było wiele / y zátrwożył ſobą Moáb dla Synów Izráelſkich.
Przetoż rzekł Moáb do ſtárƺych Mádiáńſkich : Teraz pożre <i>to</i> mnóſtwo wƺyſtko co jeſt około nas / jáko wół pożyra trawę polną. A Bálák Syn Seforów / był Królem Moábſkim / ná on cżás.
Y poſłał posły do Báláámá Syná Beorowego / do Pethoru <i>miáſtá</i> które jeſt nád rzeką / źiemie Synów ludu jego / áby go wezwano / mówiąc : Oto lud wyƺedł z Egiptu / oto okrył wierzch źiemie / y oſadza śię przećiwko mnie :
Przetoż teraz przydź proƺę / á przeklinaj mnie <i>kwoli,</i> lud ten / bo możniejƺy jeſt nád mię : owa / ſnadź go będę mógł poráźić / y wygnáć go z źiemie : bo ja wiem / <i>że</i> komu błogoſłáwiƺ / błogoſłáwiony <i>będźie</i> : á kogo przeklinaƺ / przeklęty będźie.
Poƺli tedy ſtárƺy Moábſcy / y ſtárƺy Mádiáńſcy / májąc zapłátę zá wieƺczbę w ręku ſwych.
A przyƺedƺy do Báláámá / powiedźieli mu ſłowá Bálákowe. Y rzekł do nich : zoſtańćie tu przez noc / á dam wam odpowiedź jáko mi oznájmi PAN.
Y zoſtáły Kśiążętá Moábſkie z Báláámem. Tedy przyƺedł BÓG do Báláámá / y rzekł : Cóż <i>to</i> zá mężowie u ćiebie?
Y odpowiedźiał Báláám Bogu : Bálák / ſyn Seforów / Król Moábſki / poſłał do mnie / <i>mówiąc :</i>
Oto lud / który wyƺedł z Egiptu / y okrył wierzch źiemie / terazże pódź / przeklinaj mi go : ſnadź będę mógł walcżyć z nim / y wypędzę go.
Tedy rzekł Bóg do Báláámá. Nie chodź z nimi / áni przeklinaj ludu tego : bo jeſt błogoſłáwiony.
A wſtawƺy ráno Báláám / rzekł do Kśiążąt Bálákowych : wróććie śię do źiemie wáƺey / bo mi nie pozwala PAN / puśćić śię w drogę z wámi.
Wſtawƺy tedy Kśiążętá Moábſkie / wróćili śię do Báláká y powiedźieli : Nie chćiał Báláám iść z námi.
Tedy powtóre poſłał Bálák więcey Kśiążąt / y zacniejƺych nád pierwƺe.
Którzy przyƺedƺy do Báláámá / mówili mu : Ták mówi Bálák / Syn Seforów : Nie oćiągaj śię proƺę / przyść do mnie :
Abowiemći wielką uczćiwość wyrządzę / y wƺyſtko cobyś mi rzekł / ucżynię : tylko / przydź / proƺę : <i>á</i> przeklinaj mi ten lud.
Tedy odpowiedźiał Báláám / y rzekł ſługom Bálákowym : choćby mi dał Bálák pełen dom ſwój śrebrá y złotá / nie mógłbym przeſtąpić słowá PAná Bogá mego / y ucżynić <i>przećiwko niemu</i> bądź máło bądź wiele :
Ale proƺę / zoſtańćie tu y wy / przez noc / á dowiem śię / co jeƺcze PAN będźie mówił ze mną.
Tedy przyƺedł Bóg do Báláámá w nocy / y rzekł do niego : Jesliż / áby ćię wezwáli / przyƺli mężowie <i>ći,</i> wſtańże / idź z nimi : á wƺákże coć rozkażę / to uczyniƺ.
Tedy wſtawƺy Báláám ráno / ośiodłał Oślicę ſwoję / y jáchał z Kśiążęty Moábſkimi :
Y rozpalił śię gniew Boży / że on jáchał : y ſtánął Anjoł PAńſki ná drodze / áby mu záſtąpił / á on jáchał ná Oślicy ſwojey / y dwoje pácholąt jego z nim.
A <i>gdy</i> ujrzáłá Oślicá Anjołá PAńſkiego / ſtojącego ná drodze / á miecz jego dobyty w ręce jego / tedy uſtąpiłá Oślicá z drogi / á ƺłá ná rolą / lecż bił Báláám Oślicę / áby ją náwiódł ná drogę.
Tedy ſtánął Anjoł PAńſki ná śćieƺce u winnice / miedzy dwiemá płoty.
A widząc Oślicá Anjołá PAńſkiego / przyćiſkáłá śię do płotá / ták yż przypárłá nogę Báláámowę do śćiány : á on znowu ją bił :
Potym Anjoł PAńſki ƺedł dáley / y ſtánął ná miejſcu ćiáſnym / gdźie nie było drogi do uſtąpienia / ná práwo áni ná lewo :
A widząc Oślicá Anjołá PAńſkiego pádłá pod Báláámem : y rozgniewał śię Báláám wielce / á bił Oślicę kijem.
Zátym otworzył PAN uſtá oney Oślice / y rzekłá do Báláámá : Cóżemći uczyniłá żeś mię bił już po trzy kroć?
Y rzekł Báláám do Oślice : Yż ze mnie ƺydźiƺ : bym był miał miecz w ręku ſwych / byłbym ćię teraz zábił.
Tedy Oślicá rzekłá Báláámowi : Azażem ja nie Oślicá twojá / ná któreyś jeżdzał jákoś mię doſtał / áż do tego cżáſu / Azażem zwykłá ták tobie czynić? A <i>on</i> rzekł : nigdy.
Wtym otworzył PAN ocży Báláámowe / że obaczył Anjołá PAńſkiego / ſtojącego ná drodze / y miecz jego dobyty w ręce jego : tedy ſkłoniwƺy śię pokłonił śię twarzą ſwoją.
Y rzekł do niego Anjoł PAńſki : Przecćżeś bił Oślicę ſwoję już po trzy kroć? Otom ja wyƺedł ábym śię tobie zprzećiwił : bo przewrotna jeſt drogá twojá przede mną :
A widząc mię Oślicá / uſtąpiłá przede mną po trzy kroć : <i>á</i> gdyby byłá nie uſtąpiłá przede mną / jużbym ćię był teraz zábił / á onębym był żywą zoſtáwił.
Zátym rzekł Báláám do Anjołá PAńſkiego : zgrzeƺyłem / ábowiem nie wiedźiałem żeś ty ſtánął przećiwko mnie ná drodze / przetoż teraz jeſlić śię <i>to</i> nie podoba / wrócę śię.
Lecć Anjoł PAńſki rzekł do Báláámá : Jedź z ludźmi <i>tymi,</i> wƺákże tylko co ja tobie powiem / mówić będźieƺ. Y jáchał Báláám z Kśiążęty Bálákowemi.
A <i>gdy</i> uſłyƺał Bálák yż przyjeżdza Báláám / wyjáchał przećiwko niemu / do <i>niektórego</i> miáſtá Moábſkiego / które jeſt ná gránicy Arnonu / które jeſt przy końcu gránice.
Tedy rzekł Bálák do Báláámá : Azażem z pilnośćią nie poſyłał do ćiebie wzywájąc ćię? cżemużeś nie przyjáchał do mnie? ázaż ćię zacnie uczćić nie mogę?
Y rzekł Báláám do Báláká : Otom przyjáchał do ćiebie : yzali teraz choćbym chćiał będę mógł co mówić? ſłowo / które włoży Bóg w uſtá moje / mówić będę.
Tedy jáchał Báláám z Bálákiem / á przyjácháli do miáſtá Huſot.
A ták Bálák dał nábić wołów y Owiec / y poſłał do Báláámá / y do Kśiążąt / którzy z nim <i>byli</i>.
Y ſtáło śię názájutrz / że wźiął Bálák Báláámá / y wprowádźił go ná wyżyny Báálowe / z kąd widźiał <i>y</i> nadálƺą część ludu.
