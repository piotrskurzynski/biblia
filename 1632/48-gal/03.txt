O głupi Gálátowie! Któż was omamił / ábyśćie prawdźie nie byli poſłuƺni / którym przed ocżymá JEzus CHryſtus przed tym był wymálowány / y miedzy wámi ukrzyżowány?
Tego tylko radbym śię náucżył od was : Z ucżynkówli Zakonu wźięliśćie Duchá / cżyli z ſłuchánia wiáry?
Tákeśćie głupi ; pocżąwƺy Duchem / teraz ćiáłem dokonywaćie?
Tákeśćie wiele ćierpieli dármo / jeſli tylko y dármo?
Ten tedy / który wam dodawa Duchá / y cżyni cudá miedzy wámi ; z ucżynkówże zakonu <i>to cżyni,</i> cżyli z ſłuchánia wiáry?
Ták / Jáko Abráhám uwierzył Bogu y przycżytano mu <i>to</i> ku ſpráwiedliwośći.
Widźićie tedy / że ći którzy ſą z wiáry / ći ſą ſynámi Abráhámowymi.
A upátrzywƺy to piſmo / iż z wiáry Bóg uſpráwiedliwia pogány / przed tym opowiedźiáło Abráhámowi / iż w tobie będą błogoſłáwione wƺyſtkie narody.
A ták / ći którzy <i>ſą</i> z wiáry / doſtępują błogoſłáwieńſtwá z wiernym Abráhámem.
Abowiem ile ich jeſt z ucżynków zakonu / pod przeklęctwem ſą : Bo nápiſano ; Przeklęty káżdy / któryby nie zoſtał we wƺyſtkim co nápiſano w Kśięgách zakonu / áby to cżynił.
A iż przez zakon nikt nie bywa uſpráwiedliwiony przed Bogiem / jáwna jeſt <i>z tąd</i> ; bo Spráwiedliwy z wiáry żyć będźie”.
Ale zakon nie jeſtći z wiáry : lecż cżłowiek któryby je cżynił / żyw będźie przez nie.
<i>Ale</i> CHryſtus odkupił nas z przeklęctwá zakonu / ſtawƺy śię zá nas przeklęctwem : ( Abowiem nápiſano ; Przeklęty káżdy który wiśi ná drzewie. )
Aby ná pogány błogoſłáwieńſtwo Abráhámowe przyƺło w CHryſtuśie JEzuśie / <i>y</i> ábyſmy obietnicę Duchá wźięli przez wiárę.
Bráćia! po ludzku mówię : A wƺák y cżłowiecżego teſtámentu utwirdzonego nikt nie łamie / áni <i>do niego co</i> przydawa.
Lecż Abráhámowi ucżynione ſą obietnice y naśieniu jego. Nie mówi : Y naśieniom jego / jáko o wielu ; ále jáko o jednem : Y naśieniu twemu / który jeſt CHryſtus.
To tedy mówię ; iż przymierza przed tym od Bogá utwirdzonego / względem CHryſtuſá / zakon który po cżterech ſet y po trzydźieſtu lat náſtał / nie znośi áby miał zepſowáć obietnicę <i>Bożą</i>.
Abowiem jeſliż z Zakonu <i>jeſt</i> dziedzictwo / jużći nie z obietnice. Lecż Abráhámowi przez obietnicę dárował <i>je</i> Bóg.
Cóż tedy Zakon? Dla przeſtępſtwá przydány jeſt / áżby przyƺło ono naśienie któremu śię ſtáłá obietnicá / zporządzony przez Anjoły <i>y</i> przez rękę Pośrzedniká.
Lecż Pośrzednik nie jeſt jednego / ále Bóg jeden jeſt.
Zakon tedy jeſtże przećiwko obietnicąm Bożym? Nie daj tego Boże! Abowiem gdyby był dány zakon / któryby mógł ożywiáć / prawdźiwiećby z Zakonu byłá ſpráwiedliwość.
Ale piſmo zámknęło wƺyſtko pod grzech / áby obietnicá z wiáry JEzuſá CHryſtuſá byłá dána wierzącym.
A przed tym niż przyƺłá wiárá / byliſmy pod Zakonem ſtrzeżeni / weſpół zámknieni będąc w tę wiárę / która potym miáłá być objáwiona.
A przetoż zakon Pedágogiem náƺym był do CHryſtuſá / ábyſmy z wiáry byli uſpráwiedliwieni.
Ale gdy przyƺłá wiárá / już nie jeſteſmy pod Pedágogiem.
Abowiem wƺyſcy ſynámi Bożymi jeſteśćie przez wiárę w CHryſtuśie JEzuśie.
Bo którzykolwiek jeſteśćie w CHryſtuſá ochrzcżeni / w CHryſtuſáśćie śię oblekli.
Niemáƺ Żydá / áni Greká : niemáƺ niewolniká / áni wolnego : niemáƺ mężcżyzny / y niewiáſty. Abowiem wƺyſcy wy jednym jeſteśćie w CHryſtuśie JEzuśie.
A jeſliśćie wy CHryſtuſowi / tedyśćie naśieniem Abráhámowym / á według obietnice dźiedźicámi.
