Mówię tedy ; Pokąd dźiedźic jeſt dźiećięćiem / nic nie jeſt różny od ſługi / pánem będąc wƺyſtkiego.
Ale jeſt pod opiekunámi y dozorcámi / áż do cżáſu zámierzenia Ojcowſkiego.
Tákże y my gdyſmy byli dźiećmi / pod żywioły świátá byliſmy zniewoleni.
Lecż gdy przyƺło wypełnienie cżáſu / poſłał Bóg onego Syná ſwego / który śię urodźił z niewiáſty / który śię ſtał pod Zakonem.
Aby te którzy pod Zakonem <i>byli</i> wykupił / żebyſmy práwá przyſpoſobienia zá ſyny doſtąpili.
A iżeśćie ſynowie : <i>Przetoż</i> poſłał Bóg Duchá Syná ſwego w ſercá wáƺe wołájącego Abbá / <i>to jeſt,</i> Ojcże.
A ták już więcey nie jeſteś niewolnikiem / ále ſynem : á ponieważ ſynem / <i>tedy</i> y dźiedźicem Bożym przez CHryſtuſá.
Aleć ná on cżás nie znájąc Bogá / ſłużyliśćie tym którzy z przyrodzenia nie ſą bogowie.
A teraz poznawƺy Bogá / owƺem y poznáni będąc od Bogá ; jákoż śię záś ná zad wracáćie ku żywiołom mdłym y mizernym / którym záśię znowu ſłużyć chcećie?
Przeſtrzegaćie dni / y mieśiąców / y cżáſów y lat.
Boję śię o was / bym ſnadź dármo nie prácował około was.
Bądźćie jáko ja : gdyżem y ja <i>jeſt</i> jáko wy bráćia / proƺę was. W nicżymeśćie mię nie ukrżywdźili.
Bo wiećie żem w ſłábośći ćiáłá wam z przodku Ewángelią opowiádał.
A pokuƺenia mego które było w ćiele mojim / nie lekceśćie ſobie ważyli / ániśćie <i>nim</i> gárdźili / áleśćie mię jáko Anjołá Bożego przyjęli / <i>y</i> jáko CHryſtuſá JEzuſá.
Jákież tedy było błogoſłáwieńſtwo wáƺe? Abowiem wam dawam świádectwo / yż / by byłá rzecż można / dálibyśćie mi byli wyłupiwƺy ocży ſwoje.
Tákżem śię ſtał nieprzyjaćielem wáƺym práwdę wam mówiąc?
Pałáją ku wam miłośćią nie dobrze ; owƺem chcą nas odſtrychnąć ábyśćie je miłowáli.
A dobrać rzecż pałáć miłośćią w dobrym záwżdy / á nie tylko gdym jeſt obecnym u was.
Dźiatki moje / które znowu z boleśćią rodzę áżby CHryſtus był wykƺtałtowány w was :
Chćiałbym teraz być u was / y odmienić głos mój ; ponieważ wątpię o was.
Powiedzćie mi / którzy pod Zakonem chcećie być / nie ſłuchaćież Zakonu?
Abowiem nápiſano / Iż Abráhám miał dwu ſynów : jednego z niewolnice / á drugiego z wolney.
Lecż ten który był z niewolnice / według ćiáłá śię urodźił : á który z wolney / według obietnice.
Przez co znácżą śię inƺe rzecży. Abowiem te ſą one dwá Teſtámenty. Jeden z góry Synájſkiey / który rodźi w niewolą ; á ten jeſt <i>jáko</i> Agár.
Abowiem Agár jeſt górá Synáj w Arábiey : á ſtoſuje śię do niey terázniejƺe Jeruzalem ; bo jeſt w niewoli z dźiatkámi ſwojimi.
Lecż ono górne Jeruzalem / wolne jeſt ; które jeſt mátką wƺyſtkich nas.
Abowiem nápiſano : Rozweſel śię niepłodna / która nie rodźiƺ ; porwi śię á záwołaj / która nie prácujeƺ w porodzeniu : bo tá opuƺcżona wiele ma dźiatek / więcey niż tá która ma mężá.
My tedy bráćia ták jáko Izáák jeſteſmy dźiátkámi obietnice.
Ale jáko ná on cżás ten który śię był urodźił według ćiáłá / prześládował tego który śię był urodźił według duchá ; ták <i>śię dźieje</i> y teraz.
Ale co mówi piſmo? Wyrzuć niewolnicę y ſyná jey : Abowiem nie będźie dźiedźicżył ſyn niewolnice z ſynem wolney.
A ták bráćia / nie jeſteſmy dźiećmi niewolnice / ále wolney.
