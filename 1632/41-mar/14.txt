A po dwu dniách byłá Wielkanoc święto Przáſników : y ƺukáli Przedniejƺy Kápłani y Náucżeni w piśmie / jákoby go zdrádą pojimawƺy / zábili.
Lecż mówili : Nie w Święto / áby ſnadź nie był rozruch miedzy ludem.
A gdy on był w Betániey w domu Symoná trędowátego / gdy śiedźiał u ſtołu / przyƺłá niewiáſtá májąc ſłojek álábáſtrowy máśći Szpikánárdowey płynącey / bárzo koƺtowney ; á ztłukƺy ſłojek álábáſtrowy / wylałá ją ná głowę jego.
Y gniewáli śię niektórzy ſámi w ſobie / á mówili : Ná cóż śię ſtáłá utrátá tey máśći?
Abowiem śię to mogło przedáć drożey niż zá trzy ſtá groƺy / y rozdáć ubogim ; y ƺemráli przećiwko niey.
Ale JEzus rzekł : Zániechajćie jey / przecżże śię jey przykrzyćie? Dobryć ucżynek ucżyniłá przećiwko mnie.
Záwżdy bowiem ubogie maćie z ſobą / y kiedykolwiek chcećie / możećie im dobrze cżynić ; ále mnie nie záwżdy mieć będźiećie.
Oná co mogłá to ucżyniłá : poprzedźiłá / áby ćiáło moje pomázáłá ku pogrzebowi.
Záprawdę powiádam wam : Gdźiekolwiek kazána będźie tá Ewángelia po wƺyſtkim świećie / y to co oná ucżyniłá / powiádano będźie ná pámiątkę jey.
Tedy Judaƺ Iƺkáryot jeden ze dwunaśći / odƺedł do przedniejƺych Kápłanów / áby im go wydał.
Co oni uſłyƺawƺy / urádowáli śię / y obiecáli mu dáć pieniądze. Y ƺukał ſpoſobnego cżáſu jákoby go wydał.
Pierwƺego tedy dniá Przáſników / gdy Báránká Wielkonocnego zábijano / rzekli mu ucżniowie jego : Gdźie chceƺ ábyſmy ƺedƺy nágotowáli / żebyś jadł Báránká.
Y poſłał dwu z ucżniów ſwych / y rzekł im : Idźćie do miáſtá / á potka śię z wámi cżłowiek dzban wody nioſący ; idźćież zá nim.
A dokądkolwiek wnidźie / rzecżćie goſpodarzowi : Náucżyćiel mówi ; Gdźież jeſt goſpodá / kędybym jadł Báránká z ucżniámi mojimi?
A on wam ukaże ſalę wielką uſłáną / y gotową : támże nam nágotujećie.
Y odeƺli ucżniowie jego / y przyƺli do miáſtá : y ználeźli ták jáko im był powiedźiał / y nágotowáli Báránká.
A gdy był wiecżór / przyƺedł ze dwiemánaśćie.
A gdy zá ſtołem śiedźieli y jedli / rzekł JEzus : Záprawdę wam powiádam / iż jeden z was wyda mię / który je ze mną.
Tedy oni pocżęli śię ſmęćić y do niego mówić każdy z oſobná : Azażem ja <i>jeſt</i>? A drugi / Aza ja?
Lecż on odpowiádájąc rzekł im ; Jeden ze dwunaśći który ze mną macża w miśie.
Synći cżłowiecży idźie / jáko o nim nápiſano : Ale biádá cżłowiekowi temu / przez którego Syn cżłowiecży będźie wydány! dobrze <i>by</i> mu było / by śię był ten cżłowiek nie národźił.
A gdy oni jedli / wźiął JEzus chleb / á pobłogoſłáwiwƺy / łamał y dał im / mówiąc ; Bierzćie / jedzćie / to jeſt ćiáło moje.
A wźiąwƺy kielich / y podźiękowawƺy / dał im : y pili z niego wƺyſcy.
Y rzekł im ; To jeſt krew mojá Nowego Teſtámentu / która śię zá wielu wylewa.
Záprawdę powiádam wam ; Iż nie będę więcey pił z rodzáju winney máćice / áż do dniá onego / gdy go pić będę nowy w króleſtwie Bożym.
A záśpiewawƺy pieśń / wyƺli ná górę Oliwną.
Potym im rzekł JEzus ; Wƺyſcy wy zgorƺyćie śię ze mnie tey nocy : bo nápiſano ; Uderzę páſterzá / y będą roſproƺone owce.
Lecż gdy <i>zmartwych</i>wſtánę / poprzedzę was do Gálilejey.
Ale mu Piotr powiedźiał ; Choćby śię wƺyſcy zgorƺyli / ále ja nie.
Y rzekł mu JEzus ; Záprawdę powiádam tobie / Iż dźiś tey nocy / pierwey niż dwá kroć kur zápieje / trzy kroć śię mnie záprzyƺ.
Ale on tym więcey mówił ; Bym z tobą miał y umrzeć / nie záprzę śię ćiebie. Toż też y wƺyſcy mówili.
Y przyƺli ná miejſce które zwano Gietſemáne / tedy rzekł do ucżniów ſwych ; Siedźćie tu áż śię pomodlę.
Y wźiąwƺy z ſobą Piotrá y Jákubá y Janá / pocżął śię lękáć / y bárzo tęſknić.
Y rzekł im ; Bárzo jeſt ſmętna duƺá mojá áż do śmierći : Zoſtańćie tu / á cżujćie.
A poſtąpiwƺy trochę / padł ná źiemię ; y modlił śię / áby jeſli można / odeƺłá od niego tá godźiná.
Y rzekł ; Abbá Ojcże! wƺyſtko tobie jeſt możno / przenieś ode mnie ten kielich : wƺákże nie co Ja chcę / ále co ty.
Tedy przyƺedł / y ználazł je ſpiące : y rzekł Piotrowi ; Symonie / ſpiƺ? niemogłeś cżuć jedney godźiny?
Cżujćie á módlćie śię / ábyśćie nie weƺli w pokuƺenie : duchći jeſt ochotny / ále ćiáło mdłe.
Y odƺedƺy znowu / modlił śię też ſłowá mówiąc.
A wróćiwƺy śię / ználazł je záśię ſpiące / ( bo ocży ich były obćiążone ) á nie wiedźieli coby mu odpowiedźieć mieli.
Y przyƺedł potrzećie / á rzekł im ; Śpićież już y odpocżywajćie / Doſyćći! przyƺłáć tá godźiná : oto wydány bywa Syn cżłowiecży w ręce grzeƺników.
Wſtańćie / pódźmy : oto który mię wydawa bliſko jeſt.
A wnetże gdy on jeƺcże mówił / przyƺedł Judaƺ który był jeden ze dwunaśći / á z nim wielka zgrájá z miecżmi y z kijmi / od Przedniejƺych kápłanów y od Náucżonych w piśmie / y od Stárƺych.
A ten który go wydawał / dał im był znák / mówiąc ; Któregokolwiek pocáłuję / tenći jeſt ; jimajćieƺ go / á wiedźćie oſtrożnie.
A przyƺedƺy / zárázem przyſtąpił do niego / y rzekł ; Miſtrzu / Miſtrzu! y pocáłował go.
Tedy śię oni nań rękomá rzućili / y pojimáli go.
A jeden z tych co tám ſtali / dobywƺy miecżá / uderzył ſługę Nawyżƺego Kápłaná / y ućiął mu ucho.
A JEzus odpowiádájąc rzekł im ; Jáko ná zbójcę wyƺliśćie z miecżmi y z kijmi / ábyśćie mię pojimáli.
Ná káżdy dźień bywałem u was w Kośćiele ucżąc / á nie pojimáliśćie mię : ále <i>trzebá,</i> áby śię wypełniły piſmá.
A ták opuśćiwƺy go / wƺyſcy ućiekli.
A jeden jákiś młodźieniec ƺedł zá nim przyodźiany prześćierádłem ná nágie <i>ćiáło</i> ; y uchwyćili go młodźieńcy.
Ale on opuśćiwƺy prześćierádło / nágo ućiekł od nich.
Tedy przywiedli JEzuſá do Nawyżƺego Kápłaná : á zeƺli śię do niego wƺyſcy Przedniejƺy Kápłani / y Stárƺy y Náucżeni w piśmie.
A Piotr ƺedł zá nim z dáleká áż do dworu Nawyżƺego Kápłaná ; y śiedźiał z ſługámi grzejąc śię u ogniá.
Ale Przedniejƺy Kápłani / y wƺyſtká Rádá / ƺukáli przećiwko JEzuſowi świádectwá / áby go ná śmierć wydáli ; wƺákże nie ználeźli.
Abowiem ich wiele fałƺywie świadcżyli przećiwko niemu : ále świádectwá ich nie były zgodne.
Tedy niektórzy powſtawƺy / fałƺywie świadcżyli przećiwko niemu / mówiąc :
Myſmy to ſłyƺeli że mówił ; Ja rozwálę ten Kośćiół ręką ucżyniony / á we trzech dniách inny nie ręką ucżyniony zbuduję.
Lecż y ták nie było zgodne świádectwo ich.
Tedy ſtánąwƺy w pośrzodku Nawyżƺy Kápłan / pytał JEzuſá / mówiąc ; Nie odpowiádaƺ nic? <i>Cóż to jeſt,</i> co ći przećiwko tobie świadcżą?
Ale on milcżał / á nic nie odpowiedźiał. Znowu go pytał Nawyżƺy Kápłan / y rzekł mu ; Tyżeś jeſt on CHryſtus / Syn onego błogoſłáwionego?
A JEzus rzekł : Jam jeſt : y ujrzyćie Syná cżłowiecżego śiedzącego ná práwicy mocy <i>Bożey,</i> y przychodzącego z obłokámi niebieſkimi.
Tedy Nawyżƺy Kápłan rozdárƺy ƺáty ſwoje / rzekł ; Cóż jeƺcże potrzebujemy świádków?
Słyƺeliśćie bluźnierſtwo : Cóż śię wam zda? A oni wƺyſcy oſądźili go winnym być śmierći.
Y pocżęli niektórzy nań plwáć / y zákrywáć oblicże jego / y policżkowáć go / y mówić mu ; Prorokuj! A ſłudzy go laſkámi bili.
A gdy Piotr był we dworze ná dole / przyƺłá jedná z dźiewek Nawyżƺego Kápłaná :
A ujrzawƺy Piotrá grzejącego śię / wejrzáłá nań / y rzekłá ; Y tyś był z JEzuſem Názáreńſkim.
Ale śię on záprzał / mówiąc ; Nie znam go / á nie wiem co ty mówiƺ. Y wyƺedł ná dwór do przyśionká / á kur zápiał.
Tedy dźiewká ujrzawƺy go záśię / pocżęłá mówić tym którzy tám ſtali ; Ten jeſt <i>jeden</i> z nich.
A on záśię záprzał śię. A znowu po máłey <i>chwilce</i> ći co tám ſtali / rzekli Piotrowi : Prawdźiwie z nich jeſteś : boś y Gálilejcżyk / y mowá twa podobna jeſt.
A on śię pocżął przeklináć y przyśięgáć / mówiąc ; Nie znam cżłowieká tego / o którym mówićie.
Tedy powtóre kur zápiał. Y wſpomniał Piotr ná ſłowá / które mu był powiedźiał JEzus ; Że pierwey niż kur dwá kroć zápieje / trzy kroć śię mnie záprzyƺ. A wyƺedƺy płákał.
