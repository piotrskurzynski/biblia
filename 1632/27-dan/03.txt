Nábuchodonozor król ucżynił obraz złoty / którego wyſokość byłá ná ƺeśćdźieśiąt łokći / á ƺerokość jego ná ƺeść łokći / y poſtáwił go ná polu Durá w krájinie Bábilońſkiej.
Tedy król Nábuchodonozor poſłał / áby zebráno kśiążąt / ſtároſtów y hetmánów / ſtárƺych / poborców / w práwách biegłych / urzędników / y wƺyſtkich przełożonych nád kráinámi / áby przyƺli ná poświącánie obrázu / który był wyſtáwił król Nábuchodonozor.
Tedy śię zgromádźili kśiążętá / ſtároſtowie y hetmáni / ſtárśi / poborcy / w práwách biegli / urzędnicy / y wƺyſcy przełożeni nád kráinámi ku poświęcániu obrázu / który był wyſtáwił Nábuchodonozor król / y ſtánęli przed obrázem / który był wyſtáwił Nábuchodonozor.
A woźny wołał wielkim głoſem : Wam śię opowiádá / ludźiom / národom / y językom ;
Skoro uſłyƺyćie głos trąby / piƺcżáłki / lutni / ſkrzypiec / hárfy / ſymfonáłu / y wƺelákiego inſtrumentu muzyki / upádnijćie á kłániájćie śię obrázowi złotemu / który wyſtáwił król Nábuchodonozor ;
A ktoby nie upádł y nie pokłonił śię / tejże godźiny wrzucony będźie w pośród piecá ogniem páłájącego.
Záráz tedy / ſkoro uſłyƺeli wƺyſcy ludźie głos trąby / piƺcżáłki / lutni / ſkrzypiec / hárfy y wƺelákiego inſtrumentu muzyki / upádli wƺyſcy ludźie / narody y języki / kłániájąc śię obrázowi złotemu / który wyſtáwił król Nábuchodonozor.
Przetoż tego cżáſu przyſtąpiwƺy mężowie Cháldejſcy ſkárgę ucżynili przećiwko Żydom ;
A mówiąc rzekli do królá Nábuchodonozorá : Królu / żyj ná wieki!
Ty / królu! ucżyniłeś dekret / żeby káżdy cżłowiek / któryby uſłyƺał głos trąby / piƺcżáłki / lutni / ſkrzypiec / hárfy / y ſymfonáłu y wƺelákiego inſtrumentu muzyki / upádł y pokłonił śię obrázowi złotemu ;
A ktobykolwiek nie upádł y nie pokłonił śię / áby był wrzucony w pośrodek piecá ogniem páłájącego.
Wƺákże śię ználeſli niektórzy Żydowie / którycheś przełożył nád ſpráwámi kráiny Bábilońſkiej / Sádrách / Meſách y Abednego ; ći mężowie lekce powáżyli / o królu! dekret twój / bogów twojich nie chwálą / y obrázowi śię złotemu / któryś wyſtáwił / nie kłániáją.
Tedy Nábuchodonozor w popędliwośći y w gniewie rozkázał przyprowádźić Sádráchá / Meſáchá y Abednegá / których wnet przywiedźiono przed królá.
Y mówił Nábuchodonozor á rzekł im : Umyślnież wy / Sádráchu / Meſáchu / y Abednego / bogów mojich nie cżćićie / y obrázowi śię złotemu / którym wyſtáwił / nie kłániáćie?
Teraz tedy wy bądźćie gotowi / ábyśćie záráz / ſkoro uſłyƺyćie głos trąby / piƺcżáłki / lutni / ſkrzypiec / hárfy y ſymfonáłu y wƺelákiego inſtrumentu muzyki / upádli / y pokłonili śię temu obrázowi / którym ucżynił ; á oto jeſli śię nie pokłonićie / tejże godźiny będźiećie wrzuceni w pośród piecá ogniem páłájącego ; á któryż jeſt ten Bóg / coby was wyrwał z ręki mojey?
Odpowiedźieli Sádrách / Meſách y Abednego / y rzekli do królá : O Nábuchodonozorze! my śię nie fráſujemy o to / cobyſmy mieli odpowiedźieć ;
Bo oto lubo Bóg náƺ / którego my chwálimy / ( który mocen jeſt wyrwáć nas z piecá ogniem páłájącego / y z ręki twojey / o królu! ) wyrwie nas /
Lubo nie wyrwie / niech ći będźie wiádomo / o królu! że bogów twojich chwálić / y obrázowi złotemu / któryś wyſtáwił / kłániáć śię nie będźiemy.
Tedy Nábuchodonozor pełen będąc popędliwośći / ták / że śię kƺtáłt twárzy jego odmienił przećiw Sádráchowi / Meſáchowi y Abednegowi / odpowiádájąc rozkázał piec rozpálić śiedm kroć bárdźiej / niżeli był zwycżáj rozpáláć go /
A mężom co mocniejƺym / którzy byli w wojſku jego / rozkázał / áby związáwƺy Sádráchá / Meſáchá y Abednegá / wrzućili do piecá ogniem páłájącego.
Tedy onych mężów związáno w płáƺcżách ich / w ubrániách ich y w cżápkách ich y w ƺátách ich / á wrzucono ich w pośrodek piecá ogniem páłájącego.
A iż rozkázánie królewſkie przynágláło / á piec bárzo był rozpálony / dla tego onych mężów / którzy wrzućili Sádráchá / Meſáchá y Abednegá / záduśił płomień ogniowy.
Ale ći trzej mężowie / Sádrách / Meſách y Abednego / wpádli w pośród piecá ogniem páłájącego związáni.
Tedy król Nábuchodonozor zdumiał śię y powſtał prędko / á mówiąc rzekł hetmánom ſwoim : Izáliśmy nie trzech mężów związánych wrzućili w pośród ogniá? którzy odpowiádájąc rzekli królowi : Práwdá / królu!
A on odpowiádájąc rzekł : Oto Ja widzę cżterech mężów rozwiązánych przechodzących śię w pośrzodku ogniá / á niemáƺ żádnego náruƺeniá przy nich / á oſobá cżwártego podobná jeſt Synowi Bożemu.
Tedy przyſtąpiwƺy Nábuchodonozor do cżeluśći piecá ogniem páłájącego / rzekł mówiąc : Sádráchu / Meſáchu y Abednegu / ſłudzy Bogá nájwyżƺego! wynijdźćie / á przyjdźćie ſám ; y wyƺli Sádrách / Meſách y Abednego z pośrzodku ogniá.
A zgromádźiwƺy śię kśiążętá / ſtároſtowie urzędnicy y hetmáni królewſcy oglądáli onych mężów / że nie pánował ogień nád ćiáłámi ich / y włos głowy ich nie opálił śię / y płáƺcże ich nie náruƺyły śię / áni zápách ogniá nie przeƺedł przez nich.
Tedy rzekł Nábuchodonozor / mówiąc : Błogoſłáwiony Bóg ich / to jeſt / Sádráchá / Meſáchá y Abednegá / który poſłał Aniołá ſwego / á wyrwał ſług ſwojich / którzy ufáli w nim / którzy ſłowá królewſkiego nie uſłucháli / ále ćiáłá ſwe wydáli / áby nie ſłużyli / á nie kłániáli śię żádnemu bogu / oprócż Bogá ſwego.
Przetoż ja dáję táki dekret / áby káżdy ze wƺelkiego ludu / narodu / y języká / ktobykolwiek bluźnierſtwo wyrzekł przećiwko Bogu Sádráchowemu / Meſáchowemu y Abednegowemu / był ná ƺtuki rozśiekány / á dom jego w gnojowiſko obrócony / gdyż niemáƺ Bogá innego / któryby mógł wyrwáć / jáko ten.
Tedy król zácnie wywyżƺył Sádráchá / Meſáchá y Abednegá w krájinie Bábilońſkiej.
