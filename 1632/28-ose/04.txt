Słuchájćie ſłowá Páńſkiego / o ſynowie Izráelſcy! bo śię Pán rozpierá z obywátelámi tey źiemi / gdyż niemáƺ prawdy / áni żádnego miłośierdźiá / áni znájomośći Bożey w tey źiemi.
Krzywoprzyśięſtwem / y kłámſtwem / y mężobójſtwem / y złodźiejſtwem / y cudzołóſtwem wylewáli z brzegów / á mężobójſtwo mężobójſtwá śćigáło.
Dla tego źiemiá kwilić będźie / á zemdleje wƺyſtko / co ná niej mieƺká ; zwierz polny y ptáſtwo niebieſkie / náwet y ryby morſkie zginą.
A wƺákże niech śię nikt z nimi nie ſpierá / áni ich kto ſtrofuje ; bo lud twój jeſt jáko ći / którzy śię z kápłánem wádzą.
Przetoż upádnieƺ we dnie / upádnie też y prorok z tobą w nocy ; zágłádzę y mátkę twoję.
Lud mój wygłádzony będźie dla nieumiejętnośći ; á poniewáżeś ty odrzućił umiejętność / y Ja też ćiebie odrzucę / ábyś mi kápłáńſkiego urzędu nie odpráwował ; á iżeś zápomniał zakonu Bogá twego / Ja też zápomnę ná ſynów twojich.
Cżym śię więcey rozmnożyli / tym więcey grzeƺyli przećiwko mnie ; dla tego ſłáwę ich odmienię w háńbę.
Ofiáry zá grzech ludu mego jedzą : przetoż do niepráwośći ich podnoƺą duƺę ſwoję.
Przetoż ſtánie śię jáko ludowie / ták y kápłánowi ; bo náwiedzę ná nim drogi jego / á ucżynki jego oddam mu.
Y będą jeść / wƺákże śię nie náſycą / nierząd płodźić będą / ále śię nie rozrodzą ; bo nie chcą mieć względu ná Páná.
Wƺetecżeńſtwo / y wino / y moƺcż odejmuje ſerce.
Lud mój drewná ſwego śię pytá / á kij jego odpowiádá mu ; bo ich duch wƺetecżeńſtwá w błąd prowádźi / áby nierząd płodźili / odſtępując od Bogá ſwego.
Ná wierzchách gór ofiárują / á ná págórkách kádzą pod dębiną y pod topoliną / y pod więźiną / bo dobry jeſt ćień ich ; dla tego wƺetecżeńſtwo płodzą córki wáƺe / á niewiáſty wáƺe cudzołożą.
Nie miáłżebym náwiedźić córek wáƺych / gdy nierząd płodzą? y niewiáſt wáƺych / gdy cudzołożą? Albo gdy śię y oni ſámi z nierządnicámi odłącżáją / y z wƺetecżnicámi ofiárują? Owƺem / lud / który ſobie tego w rozum nie bierze / upádnie.
Jeſli nierząd płodźiƺ / ty Izráelu! niechże wżdy nie wyſtępuje Judá ; przetoż nie chodźćie do Gálgál / áni wſtępujćie do Bet Awen / áni przyśięgájćie mówiąc : Jáko żyje Pán!
Bo Izráel jeſt nieokrócony jáko jáłowicá nieokróconá : zápráwdę Pán ich páść będźie / jáko báránki ná przeſtrzeńſtwie.
Stowárzyƺył śię Efráim z báłwánámi ; zániecháj go.
Odpornymi ich cżyni nápój ich / nierząd uſtáwicżnie płodzą ; obrońcy jego z háńbą miłują dáry.
Záchwyći ich wiátr ſkrzydłámi ſwemi / y będą záwſtydzeni dla ofiár ſwojich.
