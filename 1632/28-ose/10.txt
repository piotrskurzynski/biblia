Izráel jeſt máćicą próżną / owoc przynośi ſobie ; cżym więcey miewał owocu ſwego / tym więcey nábudował ołtárzów ; á cżym obfitƺá jeſt źiemiá jego / tym więcey náſtáwiał obrázów.
Rozdźielá śię ſerce ich / przetoż winnymi ſą ; on pokruƺy ołtárze ich / y obrázy ich popſuje.
Ponieważ mówią : Nie mamy królá / náwet nie boimy śię Páná / á król nam co ucżyni?
Mówią ſłowá / przyśięgájąc kłámliwie / gdy cżynią przymierze ; y wzrośnie ſąd jáko jád ná zágonách polnych.
Dla jáłowic Bet Aweńſkich zátrwożą śię obywátele Sámáryjſcy / gdy kwilić będą nád nim lud jego / y Popi jego / ( którzy śię więc rádowáli z niego ) przeto / że ſławá jego odchodźi od niego.
Náwet y ſám do Aſſyryi záprowádzony będźie zá dar królowi Járeb ; záwſtydźi śię Efráim / y záſromá śię Izráel zá rádę ſwoję.
Wyćięty będźie król Sámáryjſki / jáko piáná ná wierzchu wody.
Y będą wyżyny Awenu wytrácone / grzech Izráelſki ; ćiernie y oſet wyrośnie ná ołtárzách ich ; á rzekną górom : Przykryjćie nas ; á págórkom : Upádnijćie ná nas.
Ode dni Gábáá grzeƺyłeś / Izráelu! tám śię oſtáli / nie záchwyćiłá ich w Gábáá bitwá przećiwko ſynom niepráwośći podnieśioná.
Przetoż ich pokárzę według upodobánia mego ; bo śię zbiorą ná nich narody / áby byli káráni dla dwojákich niepráwośći ſwojich.
Bo Efráim jeſt jáko jáłowicá wyucżoná / kocha śię w młóceniu / choćiáżem Ja náſtępował ná cudny kárk jey / ábym do jázdy używał Efráimá / Judá áby orał / á Jákób áby włócżył.
Y mówiłem : Siejćie ſobie ku ſpráwiedliwośći ; żnijćie ku miłośierdźiu / orzćie ſobie nowinę ; bo cżás jeſt ƺukáć Páná / áż przydźie / y ſpuśći ná was ſpráwiedliwość jáko deƺcż.
Aleśćie oráli niepobożność / żęliśćie niepráwość / jedliśćie owoc kłámſtwá ; ábowiem ufáƺ w drodze ſwej y w mnóſtwie mocárzów twojich.
Dlátegoż powſtánie rozruch miedzy ludem twoim / á wƺyſtkie zámki twoje poburzone będą / ták jáko poburzył Sálmán Bet Arbel w dźień bitwy ; mátki z ſynámi roztrącone będą.
Oto ták wam ucżyni Betel dla wielkiej złośći wáƺej ; król Izráelſki ná świtániu do ƺcżętu zgłádzony będźie.
