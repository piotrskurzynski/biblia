Ale zgrzeƺyli Synowie Izráelſcy przeſtępſtwem przy rzeczách przeklętych : ábowiem Achán Syn Chármiego / Syná Zábdy / Syná Záre z pokolenia Judá / wźiął nieco z rzeczy przeklętych / zácżym zápálił śię gniew PAńſki przećiw Synom Izráelſkim.
Bo <i>gdy</i> poſłał Jozue <i>kilka</i> mężów z Jerychá do Háj / które było bliſko Betáwen / ná wſchód <i>ſłońcá</i> od Bethel / y rzekł do nich / mówiąc : Idźćie á wyƺpiegujćie źiemię : tedy ƺedƺy oni mężowie / wyƺpiegowáli Háj.
A wróćiwƺy śię do Jozuego / rzekli mu : niech nie ćiągnie wƺyſtek lud : około dwu tyśięcy mężów / álbo około trzech tyśięcy mężów niech idą / á zburzą Háj : nie trudź tám wƺyſtkiego ludu / bo ich <i>tám</i> trochá.
Poƺło tedy około trzech tyśięcy mężów z ludu / y ućiekli przed mężámi z Háj.
A poráźili z nich mężowie z Háj / około trzydźieſtu y ƺeśći mężów / goniąc je od bramy / áż do Sábárym / á poráźili je / gdy <i>ućiekáli</i> z góry / y dla tego rozpłynęło śię ſerce ludu / y było jáko wodá.
Tedy rozdárƺy Jozue odźienie ſwoje / upadł twarzą ſwoją ná źiemię / przed ſkrzynią PAńſką / <i>á leżał</i> áż do wiecżorá / on / y ſtárƺy Izráelſcy / poſypawƺy prochem głowy ſwoje.
Zátym rzekł Jozue : Ach <i>PANIE</i> Pánujący / przecżżeś przeprowádźił lud ten zá Jordan / ábyś nas podał w rękę Amorrejcżyká / ná wytrácenie? O byſmy byli rácżey mieƺkáli zá Jordanem?
O PAnie / cóz rzekę / ponieważ podawa Izráel tył nieprzyjaćiołom ſwojim?
Bo uſłyƺawƺy Chánánejcżycy / y wƺyſcy obywátele tey źiemie / obtocżą nas ze wƺąd / á wytrácą imię náƺe z źiemie / y cóż <i>to</i> uczyniƺ imieniowi twemu wielkiemu?
Tedy rzekł PAN do Jozuego : Wſtań / przecżżeś upadł ná oblicże twoje?
Zgrzeƺył Izráel / y przeſtąpili przymierze moje / którem im przykazał / ábowiem wźięli z rzecży przeklętych / á ukrádli je / y zkłąmáli / y zchowáli je miedzy naczynie ſwoje :
A dla tegoć Synowie Izráelſcy / nie będą śię mogli oſtać przed nieprzyjaćioły ſwymi / tył będą podawáli nieprzyjaćiołom ſwym / bo śię zmázáli rzeczą przeklętą : nie będę więcey z wámi / jeſli nie wykorzenićie przeklęctwá tego z pośrzodku was.
Wſtań / poświęć lud / y rzecż : Poświęććie śię ná jutro / bo ták mówi PAN Bóg Izráelſki : przeklęctwo <i>jeſt</i> w pośrzodku ćiebie Izráelu / nie oſtojiƺ śię przed nieprzyjaćioły twymi áż odejmiećie przeklęctwo z pośrzodku śiebie.
A ták przyſtąpćie ráno / według pokoleń wáƺych : á pokolenie które okaże PAN / przyſtąpi według fámilji : á fámilia którą okaże PAN / przyſtąpi według domów : á dom / który okaże PAN / przyſtąpi według oſób.
A kto będźie náleźiony w przeklęctwie / będźie ſpalony ogniem / on / y wƺyſtko co jego jeſt / dla tego że przeſtąpił przymierze PAńſkie / á dopuśćił śię niegodney rzeczy w Izráelu.
Przetoż wſtawƺy Jozue ráno / rozkazał przyſtępowáć Izráelowi / według pokoleń ich : Y ználázło śię pokolenie Judá.
Y kazał przyſtąpić fámilji Judá / y ználázłá śię fámilia Záre / y kazał przyſtąpić fámilji Zárego według Oſób / y ználazł śię <i>dom</i> Zábdy.
Y kazał przyſtąpić domowi jego / według Oſób / y ználazł śię Achán / Syn Chármiego / Syná Zábdy / Syná Záre / z pokolenia Judá :
Y rzekł Jozue do Acháná : Synu mój / daj proƺę chwałę PAnu Bogu Izráelſkiemu / y wyznaj przed nim / á oznájmi mi proƺę / coś uczynił / nie taj przede mną.
Tedy odpowiedźiał Achán Jozuemu / mówiąc : Záprawdę / jam zgrzeƺył PAnu Bogu Izráelſkiemu / ták á ták uczyniłem.
Widźiałem miedzy łupy / płaƺcż Bábylońſki jeden / piękny / y dwie śćie Syklów Śrebrá / y pręt złoty jeden / pięćdźieśiąt Syklów ważący / y pożądałem tego / y wźiąłem to / á oto te rzecży ſą zákopáne w źiemi / w pośrzód namiotu mego / á śrebro pod nimi.
Tedy poſłał Jozue poſły / którzy bieżeli do namiotu / á oto <i>te rzecży</i> były ſkryte w namiećie jego / á śrebro pod nimi.
A wźiąwƺy je z namiotu / przynieśli je do Jozuego / y do wƺyſtkich Synów Izráelſkich / á położyli je przed oblicżem PAńſkim.
A ták wźiąwƺy Jozue / y wƺyſtek Izráel z nim Acháná Syná Zárowego / y śrebro / y płaƺcż / y pręt złoty / y Syny jego / y córki jego / y woły jego / y oſły jego / y Owce jego / y namiot jego / y wƺyſtko co miał : wywiedli je ná dolinę Achor.
Y rzekł Jozue : przecżżeś nas potrwożył? niechże ćię <i>też</i> PAN zátrwoży dniá tego / y ukámionował go wƺyſtek Izráel / y ſpalili je ogniem / ukámionowawƺy je kámieńmi.
Potym wyſtáwili ná nim wielką kupę kámienia <i>która trwa</i> áż do dniá tego / y odwróćił śię PAN od gniewu zápálcżywośći ſwojey : przetoż názwáne jeſt imię miejſcá onego / Doliná Achor / áż do dniá dźiśiejƺego.
