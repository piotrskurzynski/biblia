Pádł też los pokoleniu Mánáſſeſſowemu / ( bo on jeſt pierworodny Józefów / ) Máchirowi pierworodnemu Mánáſſeſſowemu / Ojcu Gáláádá / przeto że był mężem walecżnym / y doſtał mu śię Gáláád y Báſán.
Doſtáło śię też innym Synom Mánáſſeſſowym / według domów ich : Synom Abiezer / y Synom Helek / y Synom Eſriel / y Synom Sychem / y Synom Hefer / y Synom Semidá. Cić <i>ſą</i> Synowie Mánáſſeſſowi / Syná Józefowego / mężcżyzny / według domów ich.
Ale Sálfáád Syn Heferów / Syná Gáláádowego / Syná Máchirowego / Syná Mánáſſeſſowego / nie miał Synów / jedno Córki / á te imioná córek jego Máchlá / y Noá / Heglá / Melchá / y Terſá.
Te przyƺedƺy przed Eleázárá Kápłaná / y przed Jozuego Syná Nunowego / y przed Kśiążętá / rzekły : PAN rozkazał Mojzeƺowi / áby nam dał dźiedźictwo / w pośrzód Bráćji náƺey / y dał im <i>Jozue</i> według rozkazánia PAńſkiego dźiedźictwo / w pośrzodku Bráćji Ojcá ich.
<i>Y</i> przypádło ƺnurów ná Mánáſſeſſá dźieśięć / oprócz źiemie Gáláád y Báſán / które były zá Jordanem.
Abowiem córki Mánáſſeſſowe / otrzymáły dźiedźictwo miedzy Syny jego : á źiemiá Gáláád doſtáłá śię drugim Synom Mánáſſeſſowym.
Y byłá gránicá Mánáſſeſſowá / od Aſer / do Máchmátát / które jeſt przećiwko Sychem : á idźie gránicá <i>tá</i> po práwey ſtronie / do mieƺkájących w En Táfuá.
Mánáſſeſſowá byłá źiemiá Táfuá / ( ále Táfuá przy gránicy Mánáſſeſſowey / byłá Synów Efrájimowych. )
Y bieży tá gránicá do potoku Káná / ná południe tegoż potoku : á miáſtá Efrájimitów <i>ſą</i> miedzy miáſty Mánáſſeſſowemi / ále gránicá Mánáſſeſſowá / <i>idźie</i> od pułnocy onego potoku / á końcży śię u morzá.
Ná południe <i>był dźiał</i> Efrájimów / á ná pułnocy Mánáſſeſſów / á morze jeſt gránicá jego / á <i>w pokoleniu</i> Aſer zchodzą śię ná pułnocy / á w Iſáſchár ná wſchódu ſłońcá.
Y doſtáło śię Mánáſſeſſowi <i>w pokoleniu</i> Iſáſchár y w Aſer / Betſán / y miáſtecżká jego / y Jebláám y miáſtecżká jego / przytym mieƺkájący w Dor / y miáſtecżká ich : tákże mieƺkájący w Endor / y miáſtecżká ich : mieƺkájący też w Tenách / y miáſtecżká ich : y mieƺkájący w Mágedá / y miáſtecżká ich / trzy powiáty.
Ale nie mogli Synowie Mánáſſeſſowi wypędźić z onych miaſt <i>obywátelów</i> : przetoż pocżął Chánánejcżyk mieƺkáć w oney źiemi :
A gdy śię zmocnili Synowie Izráelſcy / ucżynili Chánánejcżyká hołdownikiem / ále go nie wygnáli do ƺcżętu.
Tedy rzekli Synowie Józefowi do Jozuego / mówiąc : przecżżeś nam dał w dźiedźictwo los jeden / y ƺnur jeden : á myſmy lud wielki / y do tąd błogoſłáwił nam PAN.
Y rzekł do nich Jozue : Jeſliś jeſt ludem wielkim / idźże do láſá / á wyśiecż ſobie tám <i>miejſcá</i> w źiemi Ferezejſkiey / y Refáimſkiey / jeſlić ćiáſna górá Efrájimowá.
Któremu odpowiedźieli Synowie Józefowi : Nie doſyć nam ná tey górze : do tego wozy żelázne <i>ſą</i> u wƺyſtkich Chánánejcżyków / którzy mieƺkáją w źiemi nadolney / y u tych którzy mieƺkáją w Betſán / y w miáſtecżkách jego / tákże u tych <i>którzy mieƺkáją</i> w dolinie Jezreel.
Rzekł tedy Jozue do domu Józefowego / do Efrájimá y do Mánáſſeſſá mówiąc : ludeś ty wielki / y moc twojá wielka / nie będźieƺ miał <i>tylko</i> loſu jednego.
Ale górę będźieƺ miał : á iż <i>tám jeſt</i> lás / tedy go wyrąbiƺ / y będźieƺ miał gránice jego : bo wypędźiƺ Chánánejcżyká / choć ma wozy żelázne / <i>y</i> choć jeſt potężny.
