Y ſtáło śię po śmierći Mojzeƺá ſługi PAńſkiego / że mówił PAN do Jozuego / Syná Nunowego / ſługi Mojzeƺowego : Y rzekł :
Mojzeƺ ſługá mój umárł : przetoż teraz wſtań / przepraw śię przez ten Jordan / ty / y wƺyſtek lud ten do źiemie / którą ja im / Synom Izráelſkim / dawam.
Káżde miejſce / po którym deptáć będźie ſtopá nogi wáƺey / dałem wam / jákom obiecał Mojzeƺowi.
Od puƺcży y od Libánu tego / y áż do rzeki wielkiey / rzeki Eufrátes : wƺyſtká źiemiá Hettejcżyków / y áż do morzá wielkiego / ná záchód ſłońcá / będźie gránicá wáƺá.
Nie oſtoji śię nikt przed tobą / po wƺyſtkie dni żywotá twego : jákom był z Mojzeƺem / <i>ták</i> będę z tobą / nie odſtąpię ćię / áni ćię opuƺcżę.
Zmacniajże śię y mężnie ſobie pocżynaj / ábowiem ty podaƺ w dźiedźictwo ludowi temu źiemię / o którąm przyśiągł Ojcom ich / że im ją dam.
Tylko śię zmacniáj / y bárzo mężnie ſobie pocżynaj / ábyś ſtrzegł y cżynił wƺyſtko według Zakonu / któryć rozkazał Mojzeƺ ſługá mój : nie uchylaj śię od niego / ná práwo áni ná lewo / żebyś śię roſtropnie ſpráwował / we wƺyſtkim do cżego śię udaƺ.
Niech nie odſtępują kśięgi Zakonu tego od uſt twojich / ále rozmyślaj w nich we dnie y w nocy / ábyś ſtrzegł y cżynił wƺyſtko co nápiſano w nich / ábowiem ná ten cżás poƺcześćią śię drogi twoje : Y ná ten cżás roſtropnym będźieƺ.
Azażemći nie rozkazał / zmocni śię y mężnie ſobie poczynaj / nie lękáj śię / á nie trwóż ſobą / ábowiem z tobą jeſt PAN Bóg twój / we wƺyſtkim do cżegokolwiek śię obróćiƺ.
A przetoż rozkazał Jozue przełożonym nád ludem / mówiąc :
Przejdźćie przez pośrzodek Obozu / á rozkażćie ludowi / mówiąc : gotujćie ſobie żywność : ábowiem po trzech dniách przejdźiećie przez ten Jordan / ábyśćie weƺli / á pośiedli źiemię / którą PAN Bóg wáƺ dawa wam w ośiádłość.
Rubenitom też / y Gádytom / y połowicy pokolenia Mánáſſeſſowego rzekł Jozue / mówiąc :
Pámiętajćie ná ſłowo / które wam rozkazał Mojzeƺ ſługá PAńſki / mówiąc : PAN Bóg wáƺ ſpráwił wam odpocżynienie / y dał wam tę źiemię :
Żony wáƺe / dźiatki wáƺe / y bydłá wáƺe / niech zoſtáną w źiemi / którą wam dał Mojzeƺ / z tey ſtrony Jordanu : ále wy pójdźiećie zbrojni przed bráćią wáƺą / wƺyſcy duży w śile / á będźiećie ich pośiłkowáć /
Aż odpocżynienie ſpráwi PAN bráći wáƺey / jáko <i>y</i> wam / á oni pośiędą źiemię / którą im dawa PAN Bóg wáƺ / potym śię wróćićie do źiemie ośiádłośći wáƺey / á będźiećie ją trzymáć / którą wam dał Mojzeƺ ſługá PAńſki / z tey ſtrony Jordanu / ná wſchód ſłońcá.
Y odpowiedźieli Jozuemu / mówiąc : wƺyſtko coś nam rozkazał ucżyniemy / á gdźiekolwiek nas pośleƺ / pójdźiemy.
Jákoſmy byli posłuƺni Mojzeƺowi / ták poſłuƺni będźiemy y tobie : tylko niech będźie PAN Bóg twój z tobą / jáko był z Mojzeƺem.
Ktobykolwiek zprzećiwił śię uſtam twojim / á nie byłby poſłuƺnym ſłowom twym / we wƺyſtkim co mu rozkażeƺ / niechaj umrze : tylko śię zmacniaj / á mężnie ſobie poczynaj.
