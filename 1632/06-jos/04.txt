A gdy śię wƺyſtek lud przepráwił zá Jordan ( Bo rzekł był PAN do Jozuego mówiąc :
Obierzćie ſobie z ludu dwánaśćie mężów / po jednym mężu z káżdego pokolenia /
Y rozkażćie im mówiąc : Weźmićie ſobie z tąd / z pośrzodku Jordanu / z tego miejſcá gdźie ſtały nogi Kápłanów porządnie / dwánaśćie kámieni / które z ſobą wyniózƺy poſtáwićie ná ſtanowiſku / gdźie będźiećie leżeć przez tę noc. )
Tedy wezwał Jozue dwánaśćie mężów / które był wybrał z Synów Izráelſkich / po jednemu mężu z káżdego pokolenia.
Y rzekł do nich Jozue : Idźćie przed ſkrzynią PAná Bogá wáƺego / w pośrzodek Jordanu : á weźmi káżdy kámień jeden ná rámię ſwoje / według licżby pokolenia Synów Izráelſkich :
Aby to było ná znák miedzy wámi / gdyby potym pytáli Synowie wáƺy / mówiąc : co wam <i>znáczy</i> to kámienie?
Tedy im powiećie / iż śię rozſtąpiły wody w Jordanie / przed Skrzynią przymierza PAńſkiego : <i>ábowiem</i> gdy ƺłá przez Jordan / rozſtąpiły śię wody Jordańſkie : y będźie to kámienie ná pámiątkę Synom Izráelſkim / áż ná wieki.
Y ucżynili ták Synowie Izráelſcy / jáko rozkazał Jozue / y wźięli dwánaśćie kámieni / z pośrzód Jordanu / jáko mówił PAN do Jozuego / według licżby pokolenia Synów Izráelſkich / á zánieśli je z ſobą / áż do ſtanowiſká : y tám je złożyli.
Jozue też wyſtáwił dwánaśćie kámieni / w pośrzód Jordanu / <i>ná miejſcu</i> kędy ſtały nogi Kápłanów / nioſących ſkrzynię przymierza / które tám zoſtáły / áż po dźiś dźień.
A ták Kápłani nioſący ſkrzynię / ſtali w pośrzód Jordanu / áż śię wypełniło to wƺyſtko / co był rozkazał PAN Jozuemu mówić do ludu / według wƺyſtkiego co był przykazał Mojzeƺ Jozuemu : Spieƺył śię tedy lud y przeƺedł <i>Jordan</i>.
Y ſtáło śię / gdy wƺyſtek lud przeƺedł / że <i>też</i> przeƺłá y Skrzyniá PAńſka / y Kápłani / przed oblicżnośćią ludu.
Przeƺli też Synowie Rubenowi / y Synowie Gádowi / z połowicą pokolenia Mánáſſeſſowego / zbrojno / przed Syny Izráelſkiemi : jáko im był powiedźiał Mojzeƺ.
Około czterdźieſtu tyśięcy ludu zbrojnego przeƺło przed PAnem / do boju / ná polá Jerichá.
Dniá onego / wywyƺył PAN Jozuego / przed ocżymá wƺyſtkiego Izráelá / y bali śię go / jáko śię bali Mojzeƺá / po wƺyſtkie dni żywotá jego.
Potym rzekł PAN do Jozuego / mówiąc :
Rozkaż Kápłanom / nioſącym Skrzynię świádectwá / áby wyſtąpili z Jordanu.
Y rozkazał Jozue Kápłanom / mówiąc : wyſtąpćie z Jordanu.
Y ſtáło śię / gdy wyſtąpili Kápłani / nioſący Skrzynię przymierza PAńſkiego / z pośrzodku Jordanu / á ſtánęły ſtopy nóg Kápłanów ná ſuƺy / wróćiły śię wody Jordańſkie / ná miejſce ſwoje / á płynęły jáko przed tym / we wƺyſtkich brżegách ſwojich.
A lud wyƺedƺy z Jordanu / dźieśiątego <i>dniá,</i> mieśiącá pierwƺego / położyli śię obozem w Gálgál / ku ſtronie wſchodniey Jerychá.
A dwánaśćie onych kámieni / które wynieśli z Jordanu / poſtáwił Jozue w Gálgál.
Y rzekł do Synów Izráelſkich / mówiąc : Gdy ſpytáją Synowie wáƺy potym Ojców ſwojich / mówiąc : Co <i>znácży</i> to kámienie?
Tedy oznájmićie Synom wáƺym / mówiąc : po ſuƺy przeƺedł Izráel ten Jordan :
Abowiem oſuƺył PAN Bóg wody Jordańſkie przed wámi / áżeśćie przeƺli : jáko ucżynił PAN Bóg wáƺ morzu cżerwonemu / które wyſuƺył przed námi / áżeſmy przeƺli :
Aby poználi wƺyſcy narodowie źiemie rękę PAńſką / że można jeſt / żebyśćie śię bali PAná Bogá wáƺego / po wƺyſtkie dni.
