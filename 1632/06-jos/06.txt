A Jerycho było zámknione y opátrzone przed Synámi Izráelſkimi / <i>y</i> nikt z niego nie wychodźił / áni do niego wchodźił.
Tedy rzekł PAN do Jozuego : Otom dał w ręce twoje Jerycho / y Królá jego / <i>y</i> możne wojſká <i>jego</i>.
A ták obchodźić będźiećie miáſto / wƺyſcy mężowie walecżni / około miáſtá chodząc raz <i>ná dźień</i> : ták ucżynićie po ƺeść dni.
Przytym śiedm Kápłanów / ponioſą śiedm trąb <i>z rogów</i> báránich / przed ſkrzynią : á dniá śiódmego / obejdźiećie miáſto śiedm kroć / á Kápłani trąbić będą w trąby.
A gdy przewłocżnie <i>trąbić</i> będą w trąby / z rogów báránich / ſkoro uſłyƺyćie głos trąby / wƺyſtek lud uczyni okrzyk bárzo wielki / y upádnie mur miáſtá ná miejſcu ſwym / y wnidźie lud <i>do miáſtá</i> káżdy przećiw <i>miejſcu</i> gdźie ſtał.
Tedy wezwawƺy Jozue / Syn Nunów / Kápłanów / rzekł do nich : weźmićie Skrzynię przymierza / á śiedm Kápłanów / niech nioſą śiedm trąb z báránich rogów / przed ſkrzynią PAńſką.
Potym rzekł do ludu : Idźćie / á obejdźćie miáſto / á zbrojni niech idą przed ſkrzynią PAńſką.
A gdy <i>to</i> Jozue ludowi powiedźiał / śiedm Kápłanów wźiąwƺy śiedm trąb / <i>z rogów</i> báránich / ƺli przed <i>ſkrzynią</i> PAńſką / y trąbili w trąby / á Skrzyniá przymierza PAńſkiego ƺłá zá nimi.
A zbrojni ƺli przed Kápłany trąbiącymi w trąby / oſtátek też ludu poſpolitego / ƺedł zá Skrzynią / gdy idąc trąbiono w trąby.
A ludowi przykazał Jozue / mówiąc : Nie będźiećie wołáć / áni będźie ſłyƺan głos wáƺ / áni wynidźie z uſt wáƺych ſłowo / áż do dniá / którego wam rżekę : wołajćie / y ucżynićie okrzyk.
Tedy obeƺłá Skrzyniá PAńſka miáſto w około raz : y wróćili śię do obozu : y zoſtáli w oboźie przez noc.
Wſtał záśię Jozue ráno / á Kápłani wźięli Skrzynię PAńſką.
A śiedm Kápłanów wźiąwƺy śiedm trąb z báránich <i>rogów</i> przed Skrzynią PAńſką ƺli / idąc y trąbiąc w trąby / á zbrojni ƺli przed nimi / oſtátek też ludu poſpolitego / ƺedł zá Skrzynią PAńſką / gdy idąc trąbiono w trąby.
A ták obeƺli miáſto drugi raz dniá wtórego / y wróćili śię do obozu / <i>y</i> ták cżynili po ƺeść dni.
Ale dniá śiódmego / wſtáli ráno ná świtániu / y obeƺli miáſto / tymże ſpoſobem śiedm kroć : ( tylko dniá tego obeƺli miáſto śiedm kroć. )
Y ſtáło śię / gdy śiódmy raz <i>obchodźili,</i> á Kápłani trąbili w trąby / rzekł Jozue do ludu : krzycżćież <i>teraz,</i> ábowiem PAN podał wam miáſto :
Y niech będźie to miáſto przeklęctwem PANU / ono / y wƺyſtko co w nim jeſt / tylko Rácháb wƺetecżnicá żywo zoſtánie / oná / y wƺyſcy którzy z nią ſą w domu : gdyż utájiłá poſłów / któreſmy byli poſłáli.
A wƺákże śię wy ſtrzeżćie od <i>rzecży</i> przeklętych / ábyśćie śię nie ſtali przeklęctwem / biorąc co z rzecży przeklętych / ábyśćie nie wpráwili Obozu Izráelſkiego w przeklęctwo / y nie zámięƺáli go.
Ale wƺyſtko śrebro y złoto / y nacżynia miedźiáne / y żelázne / święte będą PAnu / do ſkárbu PAńſkiego złożone będą.
Tedy krzycżał lud gdy zátrąbiono w trąby / ábowiem gdy uſłyƺał lud głos trąb / krzycżał y lud wielkim głoſem / y upadł mur ná miejſcu ſwym / y wƺedł lud do miáſtá káżdy przećiw <i>miejſcu</i> gdźie ſtał / y wźięli miáſto.
Y wytráćili wƺyſtko co było w mieśćie / męże y niewiáſty / dźieći / y ſtárce / woły też y owce / y oſły / oſtrzem miecżá <i>pobili</i>.
Ale dwiemá mężom / którzy ƺpiegowáli onę źiemię / rzekł Jozue : wnidźćie do domu niewiáſty wƺetecżney / á wywiedźćie z támtąd niewiáſtę / y wƺyſtko co jeſt jey / jákośćie jey przyśięgli.
Tedy wƺedƺy młodźieńcy oni / co byli wyƺpiegowáli <i>źiemię,</i> Ráchábę / y Ojcá jey / mátkę jey / y bráćią jey / y wƺyſtko co <i>było</i> jey / y wƺyſtkę rodźinę jey wywiedli / y zoſtáwili je zá Obozem Izráelſkim.
Ale miáſto ſpalili Ogniem / y wƺyſtko co w nim było : tylko śrebro y złoto / y naczynie miedźiáne / y żelázne / złożyli do ſkárbu domu PAńſkiego.
Ráchábę tákże wƺetecżnicę / y dom Ojcá jey / y wƺyſtko co było jey / Jozue żywo zoſtáwił / y mieƺkáłá w pośrzodku Izráelá / áż do terázniejƺego dniá / dla tego / iż utájiłá poſłów / które był poſłał Jozue / ku przeƺpiegowániu Jerychá.
Y wydał klątwę Jozue onego cżáſu / mówiąc : przeklęty mąż przed PAnem / któryby powſtał / á budował to miáſto Jerycho : ná pierworodnym ſwojim záłoży je / á ná namniejƺym poſtáwi bramy jego.
Y był PAN z Jozuem / á rozchodźiłá śię ſławá jego / po wƺyſtkiey źiemi.
