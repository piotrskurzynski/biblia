Tedy zebrał Jozue wƺyſtkie pokolenia Izráelſkie do Sychem / y zwołał Stárƺych z Izráelá / y Przedniejƺych z nich / y Sędźiów ich / y Przełożonych ich : y ſtánęli przed obliczem Bożym /
Y rzekł Jozue do wƺyſtkiego ludu : Ták mówi PAN Bóg Izráelſki : Zá rzeką mieƺkáli Ojcowie wáƺy / od dawnych cżáſów / Táre Oćiec Abráhámów / y Oćiec Náchorów / y ſłużyli Bogom obcym.
Y wźiąłem Ojcá wáƺego Abráhámá z miejſcá / które jeſt zá rzeką / y prowádźiłem go przez wƺyſtkę źiemię Chánánejſką / y rozmnożyłem naśienie jego / dawƺy mu Izááká.
Dałem też Izáákowi Jákóbá / y Ezawá : á podałem Ezawowi górę Sejr / áby ją pośiadł : ále Jákób y Synowie jego záƺli do Egiptu.
Y poſłałem Mojzeƺá y Aároná / á trapiłem Egipt / á gdym to ucżynił w pośrzód niego / potymem was wywiódł.
Y wywiodłem Ojce wáƺe z Egiptu / á przyƺliśćie <i>áż</i> do morzá : y gonili Egipcżánie Ojce wáƺe z wozmi / y z jezdnemi <i>áż</i> do morzá cżerwonego.
Tedy wołáli do PANA / á on położył ćiemnośći miedzy wámi / y miedzy Egipcżány / y przywiódł ná nie morze / á okryło je : y widźiáły ocży wáƺe / com uczynił w Egipćie / y mieƺkáliśćie ná puƺczy przez długi cżás.
Potymem przywiódł was do źiemie Amorrejcżyká / mieƺkájącego zá Jordanem / y walcżyli przećiwko wam : Alem je podał w rękę wáƺę / y pośiedliśćie źiemię ich / á wygłádźiłem je przed wámi.
Powſtał też Bálák / Syn Seforów / Król Moábſki / áby walcżył przećiw Izráelowi : á poſławƺy / przyzwał Báláámá Syná Beorowego / áby was przeklinał.
Y nie chćiałem ſłucháć Báláámá : przetoż błogoſłáwiąc błogoſłáwił wam / á ták wybáwiłem was z rąk jego.
Przepráwiliśćie śię potym przez Jordan / y przyƺliśćie do Jerichá / y walcżyli przećiwko wam mężowie z Jerichá / Amorrejcżyk / y Ferezejcżyk / y Chánánejcżyk / y Hettejcżyk / y Gergezejcżyk / y Hewejcżyk / y Jebuzejcżyk : álem je podał w ręce wáƺe.
Y poſłałem przed wámi śierƺenie / którzy je wypędźili przed obliczem wáƺym / dwu Królów Amorrejſkich / nie miecżem twojm / áni łukiem twoijm.
Y dałem wam źiemię / w któreyśćie nie robili / y miáſtá którycheśćie nie budowáli / w których mieƺkaćie : á winnic y Oliwnic którycheśćie nie ſádźili / pożywaćie.
Przetoż teraz / bójćież śię PAná / á ſłużćie mu w doſkonáłośći / y w prawdźie : á znieśćie Bogi / którym ſłużyli Ojcowie wáƺy zá rzeką / y w Egipćie : á ſłużćie PANU.
A jeſli śię wam zda źle ſłużyć PAnu / obierzćież ſobie dźiś / komubyśćie ſłużyli : choćiaż Bogi / którym ſłużyli Ojcowie wáƺy / co <i>byli</i> zá rzeką : choćiáż Bogi Amorrejſkie / w których wy źiemi mieƺkaćie : Aleć ja y dom mój będźiemy ſłużyli PAnu.
Y odpowiedźiał lud mówiąc : Nie day Boże / ábyſmy mieli odſtąpić PANA / á ſłużyć Bogom cudzym :
Abowiem PAN Bóg náƺ / on jeſt / który nas wywiódł / y Ojce náƺe z źiemie Egipſkiey / z domu niewoli / á który uczynił przed ocżymá náƺymi te znáki wielkie y ſtrzegł nas we wƺyſtkiey drodze którąſmy ƺli / y miedzy wƺyſtkimi narody / przez któreſmy przeƺli :
Y wypędźił PAN wƺyſtkie narody / y Amorrejcżyká mieƺkájącego w źiemi / przed twarzą náƺą : A ták my będźiemy ſłużyli PANU : bo on <i>jeſt</i> Bóg náƺ.
Tedy rzekł Jozue do ludu : Nie możećie <i>wy</i> ſłużyć PAnu : bo Bóg święty jeſt / Bóg zápálcżywy jeſt / nie przepuśći złośćiam wáƺym / áni grzechom wáƺym.
Jeſliż opuśćićie PAná / á będźiećie ſłużyli Bogom cudzym / obróći śię / y utrapi was / y zniƺczy was / choć wam przed tym dobrze cżynił.
Y odpowiedźiał lud Jozuemu : Nie ták : ále PAnu ſłużyć będźiemy.
Tedy rzekł Jozue do ludu : świádkámi będźiećie ſámi przećiwko ſobie / iżeśćie ſobie obráli PANA / ábyśćie mu ſłużyli : á oni rzekli / Świádkámi jeſteſmy.
<i>Y rzekł :</i> Terazże znieśćie Bogi cudze / którzy ſą w pośrzodku was / á nákłońćie ſercá wáƺe ku PAnu Bogu Izráelſkiemu.
Y odpowiedźiał lud Jozuemu : PAnu Bogu náƺemu ſłużyć będźiemy / y głoſowi jego poſłuƺni być chcemy.
A ták ucżynił Jozue przymierze z ludem dniá onego / y przełożył im rozkazánie y ſąd w Sychem.
Y nápiſał Jozue ſłowá te w kśięgi Zakonu Bożego : wźiął też kámień wielki / y poſtáwił go tám pod dębem / który był u świątnice PAńſkiey.
Tedy rzekł Jozue do wƺyſtkiego ludu : Oto kámień ten będźie nam świádectwem : ábowiem on ſłyƺał wƺyſtkie ſłowá PAńſkie / które mówił z námi / y będźie przećiwko wam ná świádectwo byśćie ſnadź nie ſkłamáli przećiwko Bogu wáƺemu.
Zátym roſpuśćił Jozue lud / káżdego do dźiedźictwá ſwego.
Y ſtáło śię potym / że umárł Jozue Syn Nunów / Sługá PAńſki / we ſtu y w dźieśiąći lat :
Y pogrzebli go ná gránicy dźiedźictwá jego / w Támnát Sare / które <i>jeſt</i> ná górze Efrájim / ku pułnocy góry Gáás.
Y ſłużył Izráel PAnu / po wƺyſtkie dni Jozuego / y po wƺyſtkie dni Stárƺych : którzy długo żyli po Jozuem / á którzy wiedźieli o wƺyſtkich ſpráwách PAńſkich / które cżynił Izráelowi.
Kośći też Józefowe / które byli przenieśli Synowie Izráelſcy z Egiptu / pogrzebli w Sychem / ná częśći polá / które był kupił Jákób od Synów Hemorá Ojcá Sychemowego / zá ſto jágniąt : y były u Synów Józefowych w dźiedźictwie <i>ich</i>.
Eleázár tákże Syn Aáronów umárł : y pogrzebli go ná págórku Fineeſá Syná jego który mu był dány ná górze Efrájim.
