Tedy przyzwał Jozue Rubenitów / y Gáditów / y połowice pokolenia Mánáſſeſſowego /
Y rzekł do nich : Wyśćie ſtrzegli wƺyſtkiego / co wam rozkazał Mojzeƺ ſługá PAńſki / y byliśćie poſłuƺni głoſowi memu we wƺyſtkim com wam rozkazał :
Nie opuśćiliśćie Bráćij wáƺey przez długi cżás / áż do dniá tego : áleśćie ſtrzegli pilnie rozkazánia PAná Bogá wáƺego.
A teraz / ponieważ odpocżynek dał PAN Bóg wáƺ bráćij wáƺy / jáko im był obiecał / przetoż teraz wróććie śię / á idźćie do przybytków wáƺych / y do źiemie ośiádłośći wáƺey / którą wam dał Mojzeƺ ſługá PAńſki przed Jordanem :
Tylko ſtrzeżćie pilnie / ábyśćie záchowáli przykazánie / y Zakon który wam rozkazał Mojzeƺ ſługá PAńſki : ábyśćie miłowáli PANA Bogá wáƺego / á chodźili wƺyſtkiemi drogámi jego / chowájąc rozkazánia jego / dźierżąc śię go / y ſłużąc mu / ze wƺyſtkiego ſercá wáƺego / y ze wƺyſtkiey duƺe wáƺey.
Y błogoſłáwił im Jozue / á rozpuśćił je : y odeƺli do przybytków ſwoich.
Ale połowicy pokolenia Mánáſſeſſowego / dał był Mojzeƺ <i>ośiadłość</i> w Báſán / <i>á drugiey</i> połowicy jego dał Jozue <i>dźiał</i> z bráćią ich / z tey ſtrony Jordanu / ná zachód ſłońcá. A gdy je rozpuƺczał Jozue do przybytku ich / błogoſłáwił im.
Y rzekł do nich / mówiąc : z wielkimi bogáctwy wracaćie śię do przybytków wáƺych / y z májętnośćią bárzo wielką / z śrebrem y z złotem / y z miedźią y z żelázem / y ƺat bárzo wielą : dźielćież śię łupem nieprzyjaćiół wáƺych z bráćią ſwoją.
Tedy wrácájąc śię odeƺli Synowie Rubenowi / y Synowie Gádowi / y połowicá pokolenia Mánáſſeſſowego / od Synów Izráelſkich z Sylo / które jeſt w źiemi Chánánejſkiey / áby ƺli do źiemie Gáláád / do źiemie ośiádłośći ſwojey / którą dźiedźicznie otrzymáli według ſłowá PAńſkiego przez Mojzeƺá.
Y przyƺli do gránic Jordanu / które <i>były</i> w źiemi Chánánejſkiey / y zbudowáli tám Synowie Rubenowi / y Synowie Gádowi / y połowicá pokolenia Mánáſſeſſowego Ołtarż nád Jordanem : Ołtarż wielki ná podźiw.
Y uſłyƺeli Synowie Izráelſcy / iż powiádano : Oto zbudowáli Synowie Rubenowi / y Synowie Gádowi / y połowicá pokolenia Mánáſſeſſowego Ołtarz przećiw źiemi Chánánejſkiey / ná gránicách / nád Jordanem / kędy przeƺli Synowie Izráelſcy.
To gdy uſłyƺeli Synowie Izráelſcy / zeƺło śię wƺyſtko zgromádzenie ich do Sylo / áby śię ruƺyli przećiwko nim ná wojnę.
Y poſłáli Synowie Izráelſcy do Synów Rubenowych / y do Synów Gádowych / y do połowice pokolenia Mánáſſeſſowego / do źiemie Gáláád / Fineeſá Syná Eleázárá Kápłaná :
A z nim dźieśięć Kśiążąt / po jednym Kśiążęćiu z káżdego domu Ojcowſkiego : ze wƺyſtkich pokoleń Izráelſkich / á káżde Kśiążę z tych było przedniejƺym w domu Ojców ſwojich / w tyśiącách Izráelſkich.
Tedy <i>ći</i> przyƺli do Synów Rubenowych / y do Synów Gádowych / y do połowice pokolenia Mánáſſeſſowego / do źiemie Gáláád / y rzekli do nich / mówiąc :
Ták mówi wƺyſtko zgromádzenie PAńſkie : cóż to <i>jeſt</i> zá przeſtępſtwo / którymeśćie wyſtąpili przećiwko Bogu Izráelſkiemu? Żeśćie śię dźiś odwróćili od PANA budując ſobie Ołtarz / ábyśćie dźiś byli przećiwnemi PANu?
Azaż nam máło ná złośći Fegorowey? odktórey nie jeſteſmy oczyśćieni y po dźiś dźień? zkąd byłá pomſtá w zgromádzeniu PAńſkim?
Żeśćie śię dźiś odwróćili / <i>żebyśćie nie ƺli</i> zá PAnem : Zácżym ſtánie śię / ponieważeśćie wy dźiś odpornymi PAnu / że śię on jutro ná wƺyſtko zgromádzenie Izráelſkie rozgniewa.
A jeſliż jeſt niecżyſta źiemiá ośiádłośći wáƺey / przeprowádźćie śię do źiemie dźiedźictwá PAńſkiego / w którey przebywa przybytek PAńſki : y weźmiećie ośiádłośći w pośrzodku nas : tylko PAnu nie bądźćie odpornymi / á nie odpadajćie od nas / budując ſobie Ołtarz / oprócż Ołtarzá PAná Bogá náƺego.
Azaż przez Acháná Syná Zárego / gdy śię dopuśćił prżeſtępſtwá w rżecży przeklętey / ná wƺyſtko zgromádzenie Izráelſkie nie przypadł gniew? á nie on ſam jedē umárł dla niepráwośći ſwojey.
Tedy odpowiedźieli Synowie Rubenowi / y Synowie Gádowi / y połowicá pokolenia Mánáſſeſſowego / á mówili do Kśiążąt tyśiąców Izráelſkich :
BÓG nád BOGI / PAN / Bóg nád Bogi / PAN / on <i>to</i> wie / y Izráel ſam pozna / jeſli śię <i>to ſtáło</i> z uporu / álbo jeſli z przeſtępſtwá przećiw PANU / niechże nas nie żywi dniá tego.
Jesliżeſmy ſobie zbudowáli Ołtarz / ábyſmy śię odwróćili od PAná : á jeſliż ku ofiárowániu ná nim Cáłopalenia / y ofiar śniednych / álbo ku ſpráwowániu ná nim ofiar ſpokojnych : PAN niech to rozezna :
Jeſliżeſmy nie <i>ráczey</i> obawiájąc śię tey rzecży uczynili to / mówiąc : Nápotym rzeką Synowie wáƺy Synom náƺym / mówiąc : cóż wam do PANA Bogá Izráelſkiego?
Oto gránicę położył PAN miedzy námi y miedzy wámi / Synowie Rubenowi / y Synowie Gádowi / Jordan : nie maćie wy dźiału w PANU / y odwrócą Synowie wáƺy Syny náƺe od bojáźni PAńſkiey.
Przetoſmy rzekli : ucżyńmy ták / á zbudujmy ſobie Ołtarz / nie dla cáłopalenia / áni <i>innych</i> ofiar /
Ale iżby był świádkiem / miedzy námi y miedzy wámi / y miedzy potomſtwy náƺymi po nas / y ábyſmy ſłużyli PAnu przed oblicżem jego w cáłopaleniách náƺych / y w śniednych ofiárách náƺych / y w ſpokojnych ofiárách náƺych / á iżby nie rzekli Synowie wáƺy nápotym Synom náƺym : nie maćie cżąſtki w PAnu.
Nádtoſmy rzekli : Gdyby nápotym rzekli nam / álbo potomſtwu náƺemu / tedy rzecżemy : Pátrzajćie ná podobieńſtwo Ołtarzá PAńſkiego / który uczynili Ojcowie náƺy / nie dla cáłopalenia / áni <i>innych</i> Ofiar / ále żeby on był świádkiem miedzy námi y miedzy wámi.
Boże nas uchowaj / żebyſmy mieli przećiwnymi być PAnu / á odſtąpić dźiś od PAná / zbudowawƺy Ołtarz dla cáłopalonych / dla śniednych / y dla <i>innych</i> Ofiar / oprócż Ołtarzá PAná Bogá náƺego / który jeſt przed przybytkiem jego.
A uſłyƺawƺy Finees Kápłan / y Kśiążętá zgromádzenia / y przełożeni nád tyśiącmi Izráelſkimi / którzy z nim byli / ſłowá które mówili Synowie Rubenowi / y Synowie Gádowi / y Synowie Mánáſſeſſowi / podobáło śię im <i>to</i>.
Y rzekł Finees / Syn Eleázárá Kápłaná do Synów Rubenowych / y do Synów Gádowych / y do Synów Mánáſſeſſowych : Dźiśiaſmy poználi / iż w pośrzodku nas jeſt PAN / iżeśćie śię nie dopuśćili przećiw PAnu przeſtępſtwá tego / y wyſwobodźiliśćie Syny Izráelſkie z ręki PAńſkiey.
A ták wróćili śię Finees Syn Eleázárá Kápłaná / <i>z onymi</i> Kśiążęty od Synów Rubenowych / y do Synów Gádowych z źiemie Gáláád / do źiemie Chánánejſkiey / do Synów Izráelſkich / y odnieśli im <i>tę</i> rzecż.
Y podobáło śię to Synom Izráelſkim / á błogoſłáwili Bogá Synowie Izráelſcy : y nie mówili <i>więcey</i> żeby mieli iść przećiwko nim ná wojnę / y wytráćić źiemię / w którey Synowie Rubenowi y Synowie Gádowi mieƺkáli.
Przezwáli tedy Synowie Rubenowi y Synowie Gádowi Ołtarz on Ed / <i>mówiąc:</i> świádkiem będźie miedzy námi / że PAN <i>jeſt</i> Bogiem.
