A gdy uſłyƺał Adoniſedek / Król Jerozolimſki / iż wźiął Jozue Háj / y zburzył je : ( bo jáko ucżynił Jerychowi / y Królowi jego / ták ucżynił Hájowi y Królowi jego / ) á iż ucżynili pokój obywátele Gábáon z Izráelem / y mieƺkáją w pośrzodku ich :
Tedy śię ulękł bárzo / przeto że miáſto wielkie <i>było</i> Gábáon / jáko jedno z miaſt Królewſkich / á że było więkƺe niż Háj / á wƺyſcy mężowie jego walecżni.
Przetoż poſłał Adoniſedek Król Jerozolimſki / do Hohámá Królá Hebron / y do Fárán Królá Jerimotá / y do Jáfiá / Królá Láchis / y do Dábir / Królá Eglon / mówiąc :
Przyjedźćie do mnie / á dajćie mi pomoc / ábyſmy pobili Gábáonity / którzy ucżynili pokój z Jozuem y z Syny Izráelſkimi.
Zebráło śię tedy / á wyćiągnęło pięć Królów Amorrejſkich / Król Jerozolimſki / Król Hebron / Król Jerimoth / Król Láchis / Król Eglon / ſámi y wƺyſtkie wojſká ich / y położyli śię obozem u Gábáon / y dobywáli go.
Tedy poſłáli obywátele Gábáon do Jozuego / y do Obozu w Gálgál / mówiąc : Nie záwśćiągaj ręki ſwey / od ſług twojich : przyćiągni do nas rychło / á wybaw nas / y pomóż nam / boć śię zebráli przećiwko nam / wƺyſcy Królowie Amorrejſcy którzy mieƺkáją po górách.
Ruƺył śię tedy Jozue z Gálgál ſam y wƺyſtek lud wojenny z nim / y wƺyſcy mężowie wálecżni :
( Bo był rzekł PAN do Jozuego : Nie bój śię ich / ábowiem w ręce twoje podałem je / á nie oſtoji śię żaden z nich przed tobą. )
Y przypadł ná nie Jozue nagle / <i>bo</i> cáłą noc ćiągnął z Gálgál.
Y potrwożył je PAN przed obliczem Izráelá / który je poráźił porażką wielką w Gábáon / y gonił je drogą / którą chodzą ku Betoron / á bił je áż do Aſeká y áż do Mácedá.
Y ſtáło śię / gdy ućiekáli przed Izráelem / bieżąc z góry do Betoron / że PAN ſpuśćił ná nie kámienie wielkie z Niebá / áż do Aſeká / y umieráli : więcey ich pomárło od kámienia grádowego / niż ich pobili Synowie Izráelſcy miecżem.
Tedy mówił Jozue do PAná / dniá którego podał PAN Amorrejczyká w ręce Synom Izráelſkim : y rzekł przed ocżymá Izráelá : Słońce w Gábáon záſtánów śię / á mieśiącu w dolinie Ajálon.
Y záſtánowiło śię Słońce / á Mieśiąc ſtánął / áż śię lud pomśćił nád nieprzyjaćioły ſwymi : Izali to nie jeſt nápiſano w Kśięgách ſpráwiedliwego? Tedy ſtánęło Słońce w pośrzód niebá / á nie poſpieƺało śię záchodźić / jákoby <i>przez</i> cáły dźień :
Y nie był tákowy dźień przed tym / áni potym / w któryby uſłucháć miał PAN głoſu cżłowiecżego / bo PAN walcżył zá Izráelem.
Potym śię wróćił Jozue / y wƺyſtek Izráel z nim do obozu do Gálgál :
A ućiekło było onych pięć Królów / y ſkryli śię w jáſkinią przy Mácedá.
Y dano znáć Jozuemu / mówiąc : Náleźiono pięć Królów / którzy śię pokryli w jáſkini w Mácedá.
Y rzekł Jozue : przywalćie kámienie wielkie do dźiury jáſkinie á poſtawćie u niey męże / áby ich ſtrzegli.
A wy nie ſtójćie / gońćie nieprzyjaćioły wáƺe / á bijćie oſtátek ich / áni im dajćie uchodźić do miaſt ich : boć je podał PAN Bóg wáƺ w rękę wáƺę.
A gdy przeſtał Jozue z Syny Izráelſkimi bić ich porażką bárzo wielką / áż je do ƺczętu wytráćili / á którzy żywo zoſtáli z nich / uƺli do miaſt obronnych.
Tedy wróćił śię wƺyſtek lud zdrowo do Obozu / do Jozuego w Mácedá / <i>á</i> nie ruƺył przećiwko Synom Izráelſkim nikt językiem ſwojim.
Potym rzekł Jozue : Otwórzćie tę dźiurę jáſkinie / á wywiedźćie do mnie tych piąći Królów z jáſkinie.
Y ucżynili ták / y wywiedli do niego piąći Królów onych z jáſkinie / Królá Jerozolimſkiego / Królá Hebron / Królá Jerimot / Królá Láchis / królá Eglon.
A gdy wywiedli one Króle do Jozuego / tedy przyzwał Jozue wƺyſtkich mężów Izráelſkich / y rzekł do Rotmiſtrzów / mężów walecznych / którzy z nim chodźili : przyſtąpćie <i>ſám,</i> á náſtąpćie nogámi wáƺymi ná ƺyje tych Królów : którzy przyſtąpiwƺy náſtąpili nogámi ſwymi ná ƺyje jich.
Zátym rzekł do nich Jozue : Nie bójćie śię / áni śię lękajćie : zmacniajćie śię / y mężnie ſobie poczynajćie : boć ták ucżyni PAN wƺyſtkim nieprzyjaćiołom wáƺym / przećiw którym walczyćie.
Potym pobił je Jozue / y pomordował je / y záwieśił je ná piąći drzewách / á wiśieli ná drzewách áż do wieczorá.
A gdy záƺło Słońce / rozkazał Jozue / że je złożono z drzewá / y wrzucono je do jáſkinie / w którey śię byli ſkryli / á záwálono kámieńmi wielkimi dźiurę <i>u</i> jáſkinie / <i>które tám</i> ſą jeƺcze y do dniá tego.
Tegoż dniá wźiął Jozue Mácedá / y wyśiekł je oſtrzem miecżá / y Królá ich zámordował weſpółek z nimi / y wƺelką duƺę / która byłá w nim / nie zoſtáwił żadnego żywo / y ucżynił Królowi Mácedá / jáko ucżynił Królowi Jerychá.
Potym ćiągnął Jozue / y wƺyſtek Izráel z nim z Mácedá do Lebny / y dobywał Lebny.
A podał PAN y ono w ręce Izráelá / y Królá jego / y wyśiekł je oſtrzem miecżá / y wƺelką duƺę która byłá w nim / nie zoſtáwił w nim żadnego żywo / y ucżynił Królowi jego / jáko ucżynił Królowi Jerychá.
Potym ćiągnął Jozue / y wƺyſtek Izráel z nim / z Lebny do Láchis / á położywƺy śię przy nim Obozem / dobywał go.
Y podał PAN Láchis w ręce Izráelá / y wźiął je dniá drugiego / y wyśiekli je oſtrzem miecżá / y wƺelką duƺę która byłá w nim ták właśnie jáko ucżynił Lebnie.
Tedy przyƺedł Horám Król Gázer / ná rátunek Láchiſowi / ále go poráźił Jozue / y lud jego / ták iż nie zoſtáwił mu żadnego żywo.
Potym ćiągnął Jozue / y wƺyſtek Izráel z nim z Láchis / do Eglon / y położyli śię obozem przećiwko niemu / y dobywáli go :
Które wźiąwƺy onegoż dniá / wyśiekli je oſtrzem miecżá / y wƺelką duƺę która byłá w nim onegoż dniá zábił / ták właśnie jáko ucżynił Láchis.
Potym śię ruƺył Jozue / y wƺyſtek Izráel z nim / z Eglonu do Hebronu / y dobywał go :
Y wźięli je / á wyśiekli je oſtrzem miecżá / y Królá jego / y wƺyſtkie miáſtá jego / y wƺelką duƺę / która byłá w nim : nie zoſtáwił żadnego żywo / ták właśnie / jáko ucżynił Eglonowi / y wytráćił je / y wƺelką duƺę która w nim <i>byłá</i>.
Z támtąd obróćił śię Jozue / y wƺyſtek Izráel z nim / do Dábir / y dobywał go.
Y wźiął je / y Królá jego / y wƺyſtkie miáſtá jego / y wyśiekli je oſtrzem miecżá / y pomordował wƺyſtkie duƺe które w nim były : nie zoſtáwił żadnego żywo / jáko ucżynił Hebronowi / ták ucżynił Dábirowi / y Królowi jego : y jáko ucżynił Lebnie y Królowi jego.
A ták pobił Jozue wƺyſtkę źiemię górną / y południową / y polną / y podgórną / y wƺyſtkie Króle ich / nie zoſtáwił żadnego żywo / ále wƺyſtkie duƺe wytráćił / jáko mu był przykazał PAN Bóg Izráelſki.
Y poráźił je Jozue / od Kádes Bárny áż do Gázy / y wƺyſtkę źiemię Goſen / y áż do Gábáon.
A wƺyſtkie te Króle / y źiemię ich / wźiął Jozue jednym rázem : ábowiem PAN Bóg Izráelſki walcżył zá Izráelem.
Zátym śię wróćił Jozue / y wƺyſtek Izráel z nim / do Obozu do Gálgál.
