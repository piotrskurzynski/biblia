Y zſtárzał śię Jozue / á był zeƺły w lećiech / y rzekł PAN do niego : Tyś śię zſtárzał / á zƺedłeś w lećiech / á źiemie zoſtawa bárzo wiele ku pośiadániu.
Táć jeſt źiemiá która pozoſtawa / wƺyſtkie gránice Filiſtynów / y wƺyſcy Geſſurytowie.
Od Nylu który oblewa Egipt / áż do gránice Akáronu ná pułnocy / przynależy Chánánejcżykowi / pięćioro Kśięſtw Filiſtyńſkich : Aſáckie / y Asdodźiejſkie / Aſkálońſkie / Getejſkie / y Akáronickie : y Háwejcżycy.
Od południá wƺyſtká źiemiá Chánánejſka / y Márá / które jeſt Sydońcżyków áż do Afeká / <i>y</i> áż do gránice Amorrejcżyká.
Y źiemiá Giblitów ze wƺyſtkim Libánem ná wſchód ſłońcá / od Báálgád / pod górę Hermon / áż gdźie chodzą do Emáth.
Wƺyſtkie mieƺkájące ná górách / od Libánu áż do wód gorących / wƺyſtkie Sydońcżyki ja wypędzę / przed Syny Izráelſkimi / tylko ją podźiel Izráelitom w dźiedźictwo / jákomći rozkazał.
Przetoż teraz / rozdźiel tę źiemię w dźiedźictwo / dźiewięćiorgu pokoleniu / y połowicy pokolenia Mánáſſeſſowego.
Gdyż z drugą <i>połowicą</i> Rubenitowie y Gáditowie / wźięli dźiedźictwo ſwoje / które im dał Mojzeƺ zá Jordanem / ná wſchód ſłońcá / jáko im dał Mojzeƺ ſługá PAńſki :
Od Aroer / które jeſt nád brzegiem potoku Arnon / y miáſto które jeſt w pośrzód potoká / y wƺyſtkę równinę Medebá / áż do Dibon.
Y wƺyſtkie miáſtá Sehoná / Królá Amorrejſkiego / który królował w Heſebon / áż do gránice Synów Ammonowych.
Tákże Gáláád / y gránice Geſſurytów / y Máchátytów / y wƺyſtkę górę Hermon / y wƺyſtko Báſán / áż do Sálechá.
Wƺyſtko króleſtwo Ogá w Báſán / który królował w Aſtárot / y w Edrey / ten był pozoſtał z Refáimitów / á pobił je Mojzeƺ / y wygłádźił je.
Ale nie wygnáli Synowie Izráelſcy Geſſurytów y Máchátytów : przetoż mieƺkał Geſſur / y Máchát w pośrzód Izráelcżyków áż do dniá tego.
Tylko pokoleniu Lewi nie dał dźiedźictwá : Ofiáry Ogniſte PANá Bogá Izráelſkiego ſą dźiedźictwem jego / jáko mu powiedźiał <i>Bóg</i>.
A ták oddał Mojzeƺ pokoleniu Synów Rubenowych <i>dźiedźictwo,</i> według fámilij ich.
Y byłá gránicá ich od Aroer / które jeſt nád brzegiem potoku Arnon / y miáſto które jeſt w pośrzód potoku / y wƺyſtkę równinę ku Medebá.
Heſebon / y wƺyſtkie miáſtá jego / które były w równinie : Dibon y Bámot Báál / y Bet Báál-Meon :
Y Jáſſá y Cedimot y Mefáát :
Y Káriátháim / y Sebámá / y Sárátáſár / ná górze w dolinie :
Y Betfegor / y Asdot / Fáſgá / y Betieſimot.
Wƺyſtkie też miáſtá w równinie / y wƺyſtko króleſtwo Sehoná Królá Amorrejſkiego / który królował w Heſebon / którego zábił Mojzeƺ / y kśiążętá Mádiáńſkie / Ewi / y Recem / y Sur / y Hur / y Rebá : Kśiążętá Sehonowe / Obywátele źiemie :
Y Báláámá Syná Beorowego / wieƺczká / zábili Synowie Izráelſcy miecżem / z innymi pobitymi.
Byłá tedy gránicá Synów Rubenowych / Jordan / z gránicámi <i>ſwymi</i>. Toć <i>jeſt</i> dźiedźictwo Synów Rubenowych / według domów ich / miaſt y wśi ich.
Dał też Mojzeƺ pokoleniu Gád / Synom Gádowym / według domów ich / <i>dźiedźictwo.</i>
A były ich gránice / Jázer / y wƺyſtkie miáſtá Gáláád / y połowicá źiemie Synów Ammonowych / áż do Aroer / które <i>jeſt</i> przećiw Rábbá.
Y od Heſebon áż do Rámát Máſſá / y Betonim / á od Máhánáim / áż do gránice Dábir.
W dolinie też Betárám / y Betnimrá / y Sochot / y Sáfon oſtátek króleſtwá Sehoná Królá Heſebońſkiego / Jordan / y pogránicże <i>jego,</i> áż do końcá morzá Cinneret zá Jordanem ná wſchód Słońcá.
Toć <i>jeſt</i> dźiedźictwo Synów Gád / według domów ich / miaſt y wśi ich.
Nád to dał Mojzeƺ <i>ośiádłość,</i> połowicy pokolenia Mánáſſeſſowego : y byłá <i>tá</i> połowicá pokolenia Synów Mánáſſeſſowych według domów ich.
Byłá gránicá ich od Máhánáim / wƺyſtko Báſán / <i>y</i> wƺyſtko króleſtwo Ogá Królá Báſáńſkiego / y wƺyſtkie wśi Jáirowe / które <i>ſą</i> w Báſán / ƺeśćdźieśiąt miaſt.
Y połowicę Gáláád / y Aſtárot / y Erdey / miáſtá króleſtwá Ogá w Báſán / <i>dał</i> Synom Máchirá Syná Mánáſſeſſowego / połowicy Synów Máchirowych / według domów ich.
Teć ſą <i>ośiádłośći,</i> które podźielił Mojzeƺ / w polách Moábſkich / zá Jordanem <i>przećiw</i> Jerichu ná wſchód ſłońcá.
Ale pokoleniu Lewi / nie dał Mojzeƺ dźiedźictwá / bo PAN Bóg Izráelſki ſam <i>jeſt</i> dźiedźictwem ich / jáko im powiedźiał.
