Potym rzekł PAN do Jozuego / nie bój śię áni śię lękaj : weźmi z ſobą wƺyſtek lud wojenny / á wſtawƺy ćiągni do Háj / otom dał w ręce twoje Królá Háj / y lud jego / y źiemię jego.
A ucżyniƺ Hájowi y Królowi jego / jákoś ucżynił Jerychu y Królowi jego / wƺákże łupy jego / y bydłá jego rozbierzećie miedzy śię : ucżyńże záſadzkę ná miáſto z tyłu jego.
A ták wſtał Jozue / y wƺyſtek lud walecżny / áby ćiągnęli ku Háj / y przebrał Jozue trzydźieśći tyśięcy mężów bárzo mocnych / y poſłał je nocą.
Y rozkazał im mówiąc : Pátrzajćie wy / ábyśćie ucżynili záſadzkę zá miáſtem : nie oddalajćie śię od miáſtá dáleko bárzo / á bądźćie wƺyſcy pogotowiu.
A ja / y wƺyſtek lud który ze mną <i>jeſt,</i> przyćiągniemy pod miáſto : á gdy oni wynidą przećiwko nam / jáko pierwey / ućieczemy przed nimi.
A oni pójdą zá námi / áż je uwiedźiemy od miáſtá : bo rzeką : ućiekáją przed námi jáko y pierwey / gdyż ućiekáć będźiemy przed nimi.
Tedy wy wſtániećie z záſadzki / y wyprzećie oſtátek ludu z miáſtá / y da je PAN Bóg wáƺ w rękę wáƺę.
A wźiąwƺy miáſto zápalićie je ogniem : według ſłowá PAńſkiego uczynićie : pátrzajćież / rozkazałem wam.
Poſłał je tedy Jozue / y ƺli ná záſadzkę : á zoſtáli miedzy Bethel / y miedzy Háj / ná Zachód Hájowi : á Jozue przez onę noc zoſtał w pośrzodku ludu.
Potym wſtawƺy Jozue bárzo ráno / obliczył lud / á ƺedł ſam y ſtárƺy z Izráelá przed ludem przećiw Háj.
Wƺyſtek też lud wojenny / który z nim był / ruƺyli śię / y przyćiągnąwƺy / przyƺli pod miáſto / y położyli śię obozem ná ſtronie pułnocney / ku Háj : á byłá doliná miedzy nim / y miedzy Háj.
Nád to wźiął około piąći tyśięcy mężów / które poſtáwił ná záſadzce miedzy Bethel / y miedzy Háj / od ſtrony zachodniey miáſtá.
Y przybliżył śię lud / <i>to jeſt,</i> wƺyſtko wojſko / które było od pułnocy miáſtá / y którzy byli ná záſadzce jego / od zachodu miáſtá / y przyćiągnął Jozue oney nocy w pośrodek doliny.
Y ſtáło śię / gdy je ujrzał Król Háj / poſpieƺyli śię y wſtáli ráno / y wyƺli ludźie z miáſtá przećiw Izráelowi ku bitwie / ſam Król / y wƺyſtek lud jego ná cżás náznácżony przed równinę / niewiedząc że záſadzká byłá ucżyniona nań zá miáſtem.
Tedy Jozue y wƺyſtek Izráel / <i>jákoby</i> od nich poráżeni / ućiekáli drogą ku puƺczy.
Y zwołány jeſt wƺyſtek lud / który był w mieśćie / áby je gonili / y gonili Jozuego : y ták uwiedźieni byli od miáſtá.
Y nie zoſtał nikt w Háj y w Bethel / któryby nie wyƺedł zá Izráelem / y zoſtáwili miáſto otworzone / á gonili Izráelá.
Tedy rzekł PAN do Jozuego : Podnieś Chorągiew którą maƺ w ręce ſwey / przećiwko Háj : bo je w ręce twoje dam : y podnióſł Jozue chorągiew / którą <i>miał</i> w ręce ſwey / przećiwko miáſtu.
A oni / co byli ná záſadzce / wſtawƺy prętko z miejſcá ſwego / bieżeli / gdy <i>on</i> podnióſł rękę ſwą : á ubieżawƺy miáſto / wźięli je / y záraz je zápalili ogniem.
A obejrzawƺy śię mężowie miáſtá Háj / ujrzeli / á oto wſtępował dym miáſtá ku niebu / y nie mieli miejſcá do ućiekánia / áni tám áni ſám : bo lud który ućiekał ku puƺczy / obróćił śię ná one co je gonili.
Tedy Jozue y wƺyſtek <i>lud</i> Izráelſki / widząc iż oni / co byli ná záſadzce / wźięli miáſto / á iż wychodźił dym z miáſtá / obróćili śię / y pobili męże miáſtá Háj.
Oni też <i>drudzy</i> wyƺli z miáſtá przećiwko nim / y obtoczyli je Izráelcżycy / jedni z tąd / á drudzy z owąd : y poráźili je ták / yż z nich żaden nie zoſtał / áni uƺedł.
Támże Królá Háj pojmáli żywo / y przywiedli go przed Jozuego.
Gdy tedy Izráelcżycy pobili wƺyſtkie obywátele Háj ná polu przy puƺczy / tedy zá nimi ƺli w pogonią / á polegli oni wƺyſcy od miecżá / áż wygłádzeni ſą : obróćili śię wƺyſcy Izráelcżycy do Háj / y wyśiekli je oſtrzem miecżá.
Y było wƺyſtkich / którzy polegli dniá onego / od mężá áż do niewiáſty / dwánaśćie tyśięcy / wƺyſtkich obywátelów Háj.
A Jozue nie ſpuśćił ręki ſwey / którą był podnióſł z Chorągwią / áż pobił wƺyſtkie obywátele Háj.
Tylko bydło / y łupy miáſtá onego / rozebráli miedzy śię Izráelcżycy według ſłowá PAńſkiego / które rozkazał Jozuemu.
Tedy zápalił Jozue Háj / y ucżynił je mogiłą wiecżną / <i>y</i> puſtynią áż do dniá tego.
A Królá Háj obieśił ná drzewie áż do wiecżorá : á gdy ſłońce záƺło / rozkazał Jozue / áby zjęto trupá jego z drzewá / á porzucono go <i>w ſámym</i> weśćiu bramy miejſkiey / y námiotáli nań kupę wielką kámienia / <i>która jeſt</i> áż do dniá tego.
Tedy Jozue zbudował Ołtarz PANU Bogu Izráelſkiemu / ná górze Hebál :
Jáko był rozkazał Mojzeƺ ſługá PAńſki Synom Izráelſkim / <i>á</i> jáko nápiſano w kśięgách Zakonu Mojzeƺowego : Ołtarz z cáłego kámienia / ná którym <i>żadne</i> żelázo nie poſtáło / y ſpráwowáli ná nim cáłopalenia PAnu / ofiárowáli też ſpokojne ofiáry.
Támże nápiſał ná kámieniách powtórzenie Zakonu Mojzeƺowego / który nápiſał przed oblicżnośćią Synów Izráelſkich.
A wƺyſtek Izráel / y Stárƺy jego / y Przełożeni / y Sędźiowie jego / ſtali po obu ſtron ſkrzynie przed kápłanámi Lewitámi / którzy nośili Skrzynię przymierza PAńſkiego / ták przychodźień / jáko domá zrodzony : połowicá ich / przećiw górze Gáryzim / á połowicá ich przećiw górze Hebál : jáko był przed tym rozkazał Mojzeƺ ſługá PAńſki / áby błogoſłáwiono ludowi Izráelſkiemu.
A potym cżytał wƺyſtkie ſłowá Zakonu : błogoſłáwieńſtwo / y przeklęctwo / według wƺyſtkiego / co nápiſano w kśięgách Zakonu.
Nie było y ſłowá / ze wƺyſtkiego co rozkazał Mojzeƺ / czegoby nie czytał Jozue / przed wƺyſtkim zgromádzeniem Izráelſkim / przed niewiáſtámi / y przed dźiatkámi / y przed przychodniámi którzy mieƺkáli miedzy nimi.
