Tedy Jozue wſtał bárzo ráno / y ruƺyli śię z Syttym / á przyƺli áż do Jordanu / on / y wƺyſcy Synowie Izráelſcy / y támże przenocowáli / niżli śię przepráwili.
A po trzećim dniu przeƺli przełożeni przez pośrzodek Obozu /
Y rozkazáli ludowi / mówiąc : Gdy ujrzyćie ſkrzynię przymierza PANA Bogá wáƺego / y Kápłany Lewity nioſące ją / wy też ruƺyćie śię z miejſcá ſwego / á pójdźiećie zá nią :
Wƺákże plác miedzy wámi / y miedzy nią będźie / ná dwá tyśiącá łokći miáry <i>zwycżájney,</i> nie przyſtępujćie bliſko do niey / ábyśćie wiedźieli drogę / którą iść maćie : ábowiem nie chodźiliśćie tą drogą przed tym.
Tedy rzekł Jozue do ludu : poświęććie śię : ábowiem jutro ucżyni PAN miedzy wámi / dźiwne rzeczy.
Przytym rzekł Jozue do Kápłanów / mówiąc : Wezmićie Skrzynię przymierza / á idźćie przed ludem : y wźięli Skrzynię przymierza / y ƺli przed ludem.
Y rzekł PAN do Jozuego : Dźiś ćię pocznę wywyƺáć / przed ocżymá wƺyſtkiego Izráelá / áby poználi / iż jákom był z Mojzeƺem / <i>ták</i> będę <i>y</i> z tobą.
Rozkażże ty Kápłanom / nioſącym Skrzynię przymierza / á rzecz im : Gdy wnidźiećie w brzeg wód Jordańſkich / w Jordanie ſtániećie.
Rzekł też Jozue do Synów Izráelſkich / przyſtąpćie ſám / á ſłuchajćie ſłów PAná Bogá wáƺego.
Y rzekł Jozue : W tym poznaćie że Bóg żywiący / jeſt w pośrzodku was / á iż koniecżnie wypędźi przed twarzą wáƺą Chánánejcżyká / y Hettejcżyká / y Hewejcżyká / y Ferezejcżyká / y Gergezejcżyká / y Amorrejcżyká / y Jebuzejcżyká.
Oto Skrzyniá przymierza / pánującego nád wƺyſtką źiemią / pójdźie przed wámi przez Jordan.
Przetoż teraz / obierzćie ſobie dwánaśćie mężów / z pokoleń Izráelſkich / po jednym mężu z káżdego pokolenia :
A gdy śię záſtánowią ſtopy nóg Kápłanów / nioſących Skrzynię PANA pánującego nád wƺyſtką źiemią / w wodźie Jordańſkiey / <i>tedy</i> śię wody Jordańſkie rozſtąpią / <i>ták</i> iż wodá płynąca z góry / ſtánie w jedney kupie.
Y ſtáło śię / gdy śię ruƺył lud z namiotów ſwych / áby śię przepráwili przez Jordan / á Kápłani nioſący Skrzynię przymierza / <i>ƺli</i> przed ludem.
A gdy przyƺli nioſący Skrzynię áż do Jordanu / á nogi Kápłanów / którzy nieśli ſkrzynię / omoczyły śię w brzegu wód : ( bo Jordan wzbiera y wylewa ná wƺyſtkie brzegi ſwoje ná káżdy cżás żniwá )
Tedy śię záſtánowiły wody płynące z gór / á ſtánęły w jedney kupie / bárzo dáleko od Adámá miáſtá / które <i>jeſt</i> ku ſtronie Sártán : á które płynęły ná dół do morzá puſtego / morzá ſłonego / zginęły y uſtáły : á ták lud przepráwiał śię przećiwko Jerychu.
A Kápłani którzy nieśli Skrzynię przymierza PAńſkiego ſtali ná ſuƺy w pośrzód Jordanu porządnie / á wƺyſcy Izráelcżycy ƺli po ſuƺy / áż śię lud wƺyſtek przepráwił przez Jordan.
