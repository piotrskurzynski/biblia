Ná łożu mojim w nocy ƺukáłám tego / którego miłuje duƺá mojá ; ƺukáłám go / álem go nie ználázłá.
Już tedy wſtánę / á obieżę miáſto ; po rynkách y po ulicách będę ƺukáć tego / którego miłuje duƺá mojá ; ƺukáłám go / álem go nie ználázłá.
Nátráfili mię ſtróżowie / którzy chodźili po mieśćie ; y ſpytáłám : Widźieliżeśćie tego / którego miłuje duƺá mojá?
A gdym málucżko odeƺłá od nich / zárázem ználázłá tego / którego miłuje duƺá mojá. Uchwyćiłám śię go / á nie puƺcżę go / áż go wprowádzę do domu mátki mojey / y do pokoju rodźićielki mojey.
Poprzyśięgám was / córki Jeruzálemſkie! przez ſárny y łánie polne / ábyśćie nie budźiły áni przerywáły ſnu miłego mojego / dokądby nie zechćiał.
Któráż to jeſt / co wyſtępuje z puƺcży jáko ſłupy dymu / okurzoná będąc myrrą y kádźidłem drożƺem nád wƺeláki proƺek áptekárſki?
Oto łoże Sálomonowe / około którego ſtoi ƺeśćdźieśiąt mocárzów z mocárzów Izráelſkich.
Wƺyſcy ći trzymáją miecż / będąc wyćwicżeni do bitwy ; káżdy z nich ma ſwój miecż przy boku ſwym dla ſtráchu nocnego.
Páłác ſobie król Sálomon wyſtáwił z drzewá Libáńſkiego.
Słupy jego pocżynił ſrebrne / á pokłád jego złoty / podniebienie ƺárłátne / á wewnątrz uſłány jeſt miłośćią córek Jeruzalemſkich.
Wynijdźćie / córki Syońſkie! á oglądájćie królá Sálomoná w koronie / którą go ukoronowáłá mátká jego w dźień zrękowin jego / y w dźień weſelá ſercá jego.
