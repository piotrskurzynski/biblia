A ták wƺyſcy Izráelcżycy oblicżeni ſą : á oto zápiſáni ſą w Kśięgách Królów Izráelſkich y Judſkich ; <i>á</i> przenieśieni ſą do Bábilonu / dla przeſtępſtwá ſwego.
Lecż którzy pierwƺy mieƺkáli w ośiádłośćiách ſwych / y w mieśćiech ſwojich / Izráelcżycy / Kápłani / Lewitowie / y Netynejcżycy.
W Jeruzalem mieƺkáli z Synów Judowych / y z Synów Benjáminowych / y z Synów Efrájimowych / y Mánáſſeſſowych.
Uttáj Syn Ammiudá Syná Amri / Syná Imri / Syná Bonny / z Synów Fáreſá Syná Judowego.
A z Siloná Aſájaƺ pierworodny / y Synowie jego.
A z Synów Zeráchowych / Jehuel / y bráći ich ƺeść ſet y dźiewięćdźieśiąt.
A z Synów Benjáminowych / Sálu / Syn Meſullámá / Syná Hodáwiego / Syná Aſenuowego.
A Ibniaƺ Syn Jerochámów / y Elá Syn Uzego / Syná Michri : y Meſullám Syn Sefátiaƺá / Syná Rehuelowego / Syná Ibniaƺowego.
Tákże bráći ich według narodów ich <i>było</i> dźiewięć ſet y pięćdźieśiąt y ƺeść : Ci wƺyſcy mężowie <i>byli</i> Kśiążętámi rodzájów / według domów Ojców ſwojich.
Z Kápłanów záśię / Jedájaƺ / y Jechojáryb / y Jáchin.
Y Azáriaƺ Syn Helkiaƺá / Syná Meſullámowego / Syná Sádokowego / Syná Merájátowego / Syná Achitobowego / był Kśiążęćiem domu Bożego.
Y Adájaƺ Syn Jerohámá / Syná Fáſſurowego / Syná Melchiaƺowego / y Mááſái Syn Adielá / Syná Jáchſerowego / Syná Meſullámowego / Syná Meſullemitowego / Syná Immerowego.
A bráći ich Kśiążąt według domów Ojców jich / <i>było</i> tyśiąc y śiedm ſet y ƺeśćdźieśiąt : mężów dużych ku ſpráwowániu poſługi w domu Bożym.
A z Lewitów Semejaƺ Syn Háſſubá / Syná Aſrykámowego / Syná Háſábiaƺowego z Synów Merárego.
Y Bákbákár / Cheres / y Gálál : y Mátániaƺ Syn Michy / Syná Zychrego / Syná Aſáfowego.
Obádiaƺ też Syn Semáhájaƺá / Syná Gálálowego / Syná Jedytunowego : y Báráchiaƺ Syn Aſy / Syná Elkánowego / który mieƺkał we wśiách Netofátyckich.
A odźwierni : Sállum / y Akkub / y Tálmon / y Ahimán / y bráćia ich : Sállum Kśiążę <i>miedzy nimi</i>.
<i>Który</i> áż do tąd w bramie Królewſkiey <i>ſtawał</i> ná wſchód ſłońcá : ći byli odźwiernymi według pocżtów Synów Lewi.
Ale Sállum Syn Korego / Syná Abiázáfowego / Syná Korego / y bráćia jego z domu Ojcá jego Korytcżycy / <i>byli</i> ku odpráwowániu poſługi / ſtróżámi progów Namiotu : á Ojcowie ich byli nád Obozem PAńſkim ſtróżámi weśćia.
A Finees Syn Eleázárów był Kśiążęćiem nád nimi / á PAN był z nim.
Zácháriaƺ <i>záśię</i> Syn Meſellemiaƺów / <i>był</i> odźwiernym drzwi u Namiotu zgromádzenia.
Cić wƺyſcy <i>ſą</i> obráni zá odźwierne do drzwi / dwie śćie <i>oſób</i> y dwánaśćie : ći we wśiách ſwych policżeni ſą / które poſtánowił Dawid y Sámuel Widzący / dla wiernośći ich.
Aby oni / y Synowie ich <i>byli</i> we drzwiách domu PAńſkiego / w domu Namiotu ná ſtraży.
Y byli odźwierni po cżterech ſtronách / ná wſchód / ná Zachód / ná północy / y ná południe.
Bráćia záśię ich <i>byli</i> we wśiách ſwych / przychodząc káżdego śiódmego dniá / od cżáſu áż do cżáſu / <i>odmieniájąc śię</i> z nimi.
Abowiem pod ſpráwą tych cżterech przedniejƺych odźwiernych <i>byli</i> Lewitowie : á byli <i>przełożeni</i> nád gmáchámi / y nád ſkárbámi domu Bożego.
A około domu Bożego nocowáli : gdyż im należáłá ſtraż <i>jego</i> ; á oni <i>go</i> ná káżdy poránek otwieráli.
A z nich <i>niektórzy</i> byli nád nacżyniem ku poſługowániu : ábowiem pod licżbą wnośili je / y pod licżbą wynośili je.
<i>Niektórzy</i> záśię z nich byli poſtánowieni nád <i>innym</i> nacżyniem / y nád wƺyſtkim nacżyniem świątnice / y nád mąką pƺenną y winem / y oliwą / y kádźidłem / y nád rzecżámi wonnymi.
A niektórzy z Synów kápłáńſkich ſpráwowáli máśći z rzecży wonnych.
Mátátiaƺ też z Lewitów / pierworodny Sállumá Korytcżyká / był przełożony nád rzecżámi które w panwiách <i>ſmáżono</i>.
A z Synów Káátowych z bráći ich / <i>byli niektórzy</i> przełożonymi nád chleby pokłádnymi / áby <i>je</i> gotowáli ná káżdy Sábbát.
A z tych <i>byli</i> śpiewacy przedniejƺy / z domów Ojcowſkich miedzy Lewitámi / mieƺkájący w gmáchách od inƺych prac wolni : bo we dnie / y w nocy powinnośći ſwey pilnowáć muśieli.
Ci przedniejƺy z <i>domów</i> Ojcowſkich miedzy Lewitámi / według narodów ſwych przedniejƺy : ći mieƺkáli w Jeruzalem.
A w Gábáonie mieƺkáli / Oćiec Gábáońcżyków / Jehiel / á imię żony jego / Mááchá.
A Syn jego pierworodny / Abdon / po nim Sur / y Cys / y Báál / y Ner / y Nádáb.
Y Gedor / y Achio / y Zácháriaƺ / y Michlot :
( A Michlot zpłodźił Symmámá ) á ći tákże przećiw bráći ſwey mieƺkáli w Jeruzalem / z bráćią ſwoją.
A Ner zpłodźił Cyſſá / á Cyſs zpłodźił Saulá / á Saul zpłodźił Jonátáná / y Melchiſuego / y Abinádábá / y Eſbáálá.
A Syn Jonátánów Merybbáál : á Merybbáál zpłodźił Micháſá.
Synowie záś Micháſowi / Fiton / y Melech / y Táreá <i>/ y Acház</i>.
A Acház zpłodźił Járę : á Járá zpłodźił Alemetá / y Aſmáwetá / y Zymrego : á Zymri zpłodźił Moſę.
A Moſá zpłodźił Binę : á Refájaƺ Syn jego / Eláſá Syn jego / Aſel Syn jego.
A Aſel miał ƺeśći Synów ; á teć imioná ich : Aſrykám / y Bochru / y Iſmáel / y Seáryjaƺ / y Obádiaƺ / y Hánán : Cić ſą Synowie Aſelowi.
