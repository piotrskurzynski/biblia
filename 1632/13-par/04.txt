Synowie Judowi : Fáres / Hesron / y Chármi / y Hur / y Sobál.
A Rejaƺ Syn Sobálów zpłodźił Jáhátá / á Jáhát zpłodźił Achumá / y Láádá : teć ſą rodzáje Zárátego.
Ci też ſą z Ojcá Etámá ; Jezreel / y Jeſemá / y Idbás : á imię śioſtry ich Selelfuni.
A Fánuel Oćiec Gedory : y Ezer / Oćiec Hoſy. Cić ſą Synowie Hurá pierworodnego Efráty / Ojcá Betlehemcżyków.
A Aſſur Oćiec Tekui / miał dwie żenie / Chelę y Náárę.
Y urodźiłá mu Náárá / Achuſámá / y Heferá / y Temáná / y Acháſtárá. Cić <i>ſą</i> Synowie Nááry.
Synowie záśię Chele / Seret / Jeſochár / y Etnán.
A Kos zpłodźił Anubá / y Háſobobá / y dom Achárchelá / Syná Hárumy.
A Jábes był zacniejƺy nád bráćią ſwą / któremu mátká jego dáłá imię Jábes / mówiąc ; Bom go w boleśći urodźiłá.
Y wzywał Jábes Bogá Izráelſkiego / mówiąc : Jeſli błogoſłáwiąc błogoſłáwić mi będźieƺ / á rozƺyrzyƺ gránice moje / á będźie ręká twojá ze mną / á wybáwiƺ <i>mię</i> od złego / ábym utrapiony nie był. Y wypełnił to Bóg / o co go prośił.
A Chelub brát Suchá zpłodźił Mechirá : ten jeſt Ojcem Eſtoná.
A Eſton zpłodźił Betráfá / y Páſeáchá / y Techinná / Ojcá miáſtá Náhás. Cić <i>ſą</i> mężowie Rechy.
A Synowie Kenezowi / Otoniel / y Serájáh : á Synowie Otonielowi : Hátát <i>y Meánátáj</i>.
A Meánátáj zpłodźił Ofrá : á Serájaƺ zpłodźił Joábá / Ojcá w dolinie mieƺkájących rzemieśników : bo rzemieśnicy byli.
Synowie záśię Kálebá Syná Jefuni / Hiru / Elá / y Náháin : á Syn Elá / jeſt Kenáz.
A Synowie Jeháleleelá / Zyf / y Zyfá / Tiriá / y Azárel.
A Synowie Ezry / Jeter / y Mered / y Efer / y Jálon : á żoná <i>Merodowá</i> urodźiłá Miriámá / y Sámájá / y Isbáchá / Ojcá Eſtemoá.
Żoná też jego Judájá / urodźiłá Jeredá / Ojcá Gedor : y Heberá / Ojcá Socho : y Jekutielá Ojcá Zánoách. Cić <i>ſą</i> Synowie Betie córki Fáráonowey którą był pojął Mered.
A Synowie żony Hodiaƺowey / śioſtry Náhámá / Ojcá Ceyle ; Gármi / y Eſtemoá Máhátcżyk.
A Synowie Simonowi / Amnon / y Rinná / Benchánán y Tilon. A Synowie Iſy / Zochet y Benſochet.
Synowie Sele / Syná Judowego / Her Oćiec Lechá ; y Láádá / Oćiec Máráá ; y rodzáje domów tych którzy robili około biśioru w domu Asbeát.
Y Jokim y mężowie Chozeby / y Joás y Sáráf / ( którzy pánowáli w Moáb) y Jáſubi Lechem : ále te rzecży <i>ſą</i> dawne.
Cić ſą Gárncarze którzy mieƺkáli w ſádziech y miedzy płoty / którzy tám przy królu dla robót jego mieƺkáli.
Synowie Symeonowi / Námuel / y Jámin / Járyb / Zerá / Saul.
Sállum / Syn jego : Mábſám Syn jego : Miſmá Syn jego.
A Synowie Miſmy / Hámuel Syn jego : Záchur Syn jego : Semei Syn jego.
Ten Semei miał Synów ƺeſnaśćie / y Córek ƺeść : ále bráćia jego nie mieli wiele Synów : ták / że wƺyſtkiey rodźiney ich nie było ták wiele jáko Synów Judowych.
Y mieƺkáli w Beerſebá y w Moládá / y w Háſerſuál.
Y w Bálá / y w Aſem / y w Etolád.
Y w Betul / y w Hormá / y w Sycelegu.
Y w Bet Márchábot / y w Háſerſuſá / y w Betbirze / y w Sááráim. Te miáſtá ich <i>były</i> póki królował Dawid.
A wśi ich <i>były</i> Etám / y Háin / Remnon / y Tochen y Aſán <i>przy tych</i> piąći miáſtách ;
Y wƺyſtkie wśi ich / które <i>były</i> około tych miaſt / áż do Báál ; teć były mieƺkániem ich według narodu ich.
A Moſobáb / y Jámlech / y Joſá / Syn Amáſyaƺów :
Y Joel / y Jehu Syn Joſábiaƺá / Syná Serájaƺowego / Syná Aſielowego :
Y Elichenáj / y Jákóbá / y Jeſochájá / y Aſájá / y Adiel / y Jeſymiel / y Bánájás :
Y Syſá Syn Syfy / Syná Allonowego / Syná Jedájaƺowego / Syná Simry / y Syná Semájaƺowego :
Ci miánowáni / poſtánowieni ſą zá Kśiążętá w národźiech ſwych / á domy Ojców ich rozmnożyły śię bárzo.
Przetoż ruƺyli śię / áby ƺli do Gádor / áż ná wſchód ſłońcá <i>oney</i> doliny / áby ƺukáli paƺe bydłu ſwemu.
Y ználeźli obfite y wyborne paƺe / á źiemię przeſtronną y ſpokojną / y rodzájną ; bo tám <i>naród</i> Chámów mieƺkał przed tym :
Przetoż przyƺedƺy ći z imienia opiſáni / zá dni Ezechiaƺá Królá Judſkiego / poburzyli namioty ich y przybytki ich / które tám były ználeźione ; á wymordowáli je / <i>y nie maƺ ich</i> áż do dniá tego / y ośiedli miejſce ich : bo tám mieli paƺe dla bydeł ſwojich.
A z onych Synów Symeonowych <i>niektórzy</i> ƺli ná górę Seir / pięć ſet mężów : á Fáltiaƺ / y Necháriaƺ / y Ráfájaƺ / y Huſiel / Synowie Iſy <i>byli</i> wodzámi ich.
Y wymordowáli oſtátki / które były uƺły z Amálekitów : á mieƺkáli tám áż po dźiś dźień.
