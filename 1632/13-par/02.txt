Cić ſą Synowie Izráelowi / Ruben / Symeon / Lewi / y Judá / Iſáſchár y Zábulon.
Dán / Józef / y Benjámin / Neftáli / Gád y Aſer.
Synowie Judy / Her / y Onán / y Selá. Ci trzej urodźili mu śię z córki Suj Chánánejſkiey. Ale Her pierworodny Judy / był złym przed ocżymá PAńſkimi / przetoż go zábił.
Támár zaśię niewiaſtká jego / urodźiłá mu Fáresá y Zárę. Wƺyſtkich Synów Judowych pięć.
Synowie Fáreſowi / Heſron / y Hámuel.
Synowie záś Záry / Zámry / y Etán / y Hemán / y Chálkol / y Dárdá : wƺyſtkich tych <i>było</i> pięć.
A Synowie <i>Zámrego</i> / Chármi / <i>wnuk</i> Achárów / który zámięƺánie ucżynił w Izráelu / zgrzeƺywƺy <i>kradźieżą</i> rzecży przeklętych.
A Synowie Etánowi / Azáryaƺ.
A Synowie Eſronowi / którzy mu śię urodźili / Jeráhmeel / y Rám / y Chálubáj.
Ale Rám zpłodźił Aminádábá : á Aminádáb zpłodźił Náháſoná / Kśiążę Synów Judſkich.
A Náháſon zpłodźił Sálmoná : á Sálmon zpłodźił Boozá.
A Booz zpłodźił Obedá : á Obed zpłodźił Iſájego.
A Iſáji zpłodźił pierworodnego ſwego Eliábá / y Abinádábá wtórego / y Sámmá trzećiego.
Nátánáelá cżwártego : Ráddájá piątego.
Ozemá ƺóſtego : Dawidá śiódmego.
A śioſtry ich Sárwiá / y Abigáil. A Synowie Sárwiey / Abisái y Joáb / y Aſáel / trzej.
A Abigáil urodźiłá Amázę. A Oćiec Amázy <i>był</i> Jeter Iſmáelcżyk.
A Káleb Syn Heſronów zpłodźił z Azubą małżonką <i>ſwoją</i> y z Jeriotą <i>Syny</i>. A ći byli Synowie jego ; Jeſer / y Sobáb / y Ardon.
A gdy umárłá Azubá / pojął ſobie Káleb Eſrátę / która mu urodźiłá Hurá.
A Hur zpłodźił Ury : á Ury zpłodźił Beſáleelá.
Potym wƺedł Hesron do Córki Máchyrá Ojcá Gáláádowego / á pojął ją / będąc w ƺeśćidźieśiąt lat : która mu urodźiłá Segubá.
A Segub zpłodźił Jáirá / który miał trzy á dwádźieśćiá miaſt / w źiemi Gáláádſkiey.
Bo wźiął Geſſurytom / y Aſſyryjcżykom wśi Jáirowe / y Kánát z miáſtecżkámi jego / ƺeśćdźieśiąt miaſt. To wƺyſtko <i>pobráli</i> Synowie Máchyrá Ojcá Gáláádowego.
A gdy umárł Hesron w Káleb Efrátá / tedy żoná Hesronowá Abijá / porodźiłá mu Aſſurá Ojcá Tekui.
Byli też Synowie Jeráhmeelowi / pierworodnego Hesronowego / pierworodny Rám : po nim Buná / y Oren / y Osem / z Abiey.
Miał tákże drugą żonę Jeráhmeel / imieniem Atárá : tá <i>jeſt</i> mátká Onámowá.
Ale Synowie Rámowi / pierworodnego Jeráhmeelowego <i>byli</i> Máás / y Jámin / y Achár.
Byli też Synowie Onámowi / Semáji / y Jádá : á Synowie Semejego / Nádáb y Abisur.
A imię żony Abisurowey / Abiháil / która mu urodźiłá Achobbáná y Molidá.
Synowie Nádábowi Sáled / y Affáim : lecż Sáled umárł bez potomſtwá.
A Synowie Affájimowi Jeſſy : á Synowie Jeſſy Seſán : á Synowie Seſán Achiáláj.
A Synowie Jády brátá Semejego / Jeter / y Jonátán : ále Jeter umárł bez potomſtwá.
A Synowie Jonátánowi / Fálet / y Zyzá. Cić byli Synowie Jeráhmeelowi.
Lecż nie miał Seſán Synów / jedno córki : miał też Seſán ſługę Egipcżániná imieniem Jeráchá.
Y dał Seſán córkę Jeráchowi ſłudze ſwemu zá żonę ; która mu urodźiłá Etájá.
Etáj zpłodźił Nátáná : á Nátán zpłodźił Zábádá.
A Zábád zpłodźił Eflálá : á Eflál zpłodźił Obedá.
A Obed zpłodźił Jehu : á Jehu zpłodźił Azáryaƺá.
A Azáryaƺ zpłodźił Helesá : á Heles zpłodźił Elásá.
A Elás zpłodźił Syſmájá : á Syſmáj zpłodźił Sellumá.
A Sellum zpłodźił Ikámiaƺá : á Ikámiaƺ zpłodźił Eliſámá.
A Synowie Kálebá brátá Jeráhmeelowego / Mesá pierworodny jego / który był Ojcem Zyfejcżyków / y Synów Máresy / Ojcá Hebronowego.
A Synowie Hebronowi / Kore / y Táfuá / y Rechem / y Semmá.
A Semmá zpłodźił Ráhámá / Ojca Jerkáámowego : á Rechem zpłodźił Sámmájego.
A Sámmáj <i>był</i> Synem Máonowym : á Máon <i>był</i> Ojcem Betſurcżyków.
Efá też założnicá Kálebowá / urodźiłá Háráná / y Moze / y Gezezá : á Hárán zpłodźił Gezezá.
A Synowie Jáchdájowi / Regiem / y Jotám / y Geſán / y Fálet / y Efá / y Sááf.
Założnicá <i>zaś drugá</i> Kálebowá / Mááchá / urodźiłá Sáberá / y Tyrcháná.
A <i>żoná</i> Sááfowá urodźiłá Pjcá Mádmeńcżyków / y Sewá Ojcá Máchbeńcżyków / y Ojcá Gábáońcżyków : á córká Kálebowá <i>byłá</i> Achſá.
Cić byli Synowie Kálebá / Syná Hurowego / pierworodnego Efráty : Sobál Oćiec Káryátiárymcżyków.
Sálmá Oćiec Betlehemcżyków : Cháref Oćiec Betgádercżyków.
Miał też Synów Sobál Oćiec Káryátiárymcżyków / który doglądał połowice Menuchoty.
A domy Káryátiárymſkie / <i>były</i> Jetrejcżycy / y Futejcżycy / y Semátejcżycy / y Máſerejcżycy : z których też poƺli Sárájtowie / y Eſtáolitowie.
A Synowie Sálmy / Betlehemcżycy / y Netofátcżycy / ozdoby domu Joábowego : y połowica Mánácháty / <i>Ojcá</i> Sorygo.
A domy piſárzów mieƺkájących w Jábes : Tyriátejcżycy / Symátejcżycy / Suchátejcżycy. Cić ſą Cynejcżycy / którzy poƺli z Hemátá / Ojcá domu Rechábowego.
