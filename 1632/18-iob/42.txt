Tedy odpowiedźiał Ijob Pánu / y rzekł :
Wiem / że wƺyſtko możeƺ / y nie może być záhámowány zámyſł twój.
Któż jeſt ten / pytáƺ / który záćiemniá rádę Bożą nieumiejętnie? Dla tego przyznáję / żem nie zrozumiał ; dźiwniejƺe ſą te rzecży / niżbym je mógł pojąć y zrozumieć.
Wyſłuchájże / proƺę / gdybym mówił ; á gdy śię będę pytał / oznájmijże mi.
Przed tym tylko ucho ſłyƺáło o tobie ; ále teraz oko moje widźi ćię.
Przetoż żáłuję y pokutuję w prochu y w popiele.
A gdy odmówił Pán te ſłowá do Ijobá / rzekł Pán do Elifáſá Temáńcżyká : Rozpálił śię gniew mój przećiw tobie / y przećiw dwom przyjáćiołom twoim / żeśćie o mnie nie mówili ták przyſtojnie / jáko Ijob / ſługá mój.
Przetoż teraz / weźmijćie ſobie śiedm ćielców / y śiedm báránów / á idźćie do ſługi mego Ijobá / y ofiárujćie cáłopálenie zá śię ; á Ijob / ſługá mój / niech śię modli zá wámi ; bo oblicże jego przyjmę / ábym nie ucżynił z wámi według głupſtwá wáƺego ; bośćie nie mówili ták przyſtojnie o mnie / jáko Ijob / ſługá mój.
A ták odeƺli Elifás Temáńcżyk / y Bildád Suhitcżyk / y Sofár Náámátcżyk / y ucżynili / jáko im rozkázał Pán ; y przyjął Pán oblicże Ijobowe.
Zátem Pán przywróćił to / co było pobráne Ijobowi / gdy śię modlił zá przyjáćiół ſwojich ; y rozmnożył Pán wƺyſtko / cokolwiek miał Ijob / w dwójnáſób.
Zeƺli śię tedy do niego wƺyſcy bráćia jego / y wƺyſtkie śioſtry jego / y inni wƺyſcy / którzy go przed tym ználi / y jedli z nim chleb w domu jego / á żáłując go ćieƺyli go z ſtrony wƺyſtkiego złego / które był Pán nań przywiódł ; y dał mu káżdy z nich upominek jeden / y káżdy náuƺnicę złotą jednę.
A ták Pán błogoſłáwił oſtátnim cżáſom Ijobowym / więcey niż pocżątkom jego. Bo miał cżternaśćie tyśięcy owiec / y ƺeść tyśięcy wielbłądów / y tyśiąc járzm wołów / y tyśiąc oślic.
Miał też śiedm ſynów / y trzy córki.
Y dał imię pierwƺej Jeminá / á imię drugiej Kietzyjá / á imię trzećiej Kierenháppuch.
A nie znájdowáły śię niewiáſty ták piękne / jáko córki Ijobowe / we wƺyſtkiej oney źiemi ; y dał im oćiec ich dźiedźictwo miedzy bráćmi ich.
Potem Ijob żył ſto y cżterdźieśći lat / y oglądał ſynów ſwych / y ſynów ſynów ſwojich / áż do cżwártego pokolenia.
A umárł Ijob / będąc ſtárym y dni ſytym.
