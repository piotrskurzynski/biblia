Przetoż złożywƺy wƺeláką złość / y wƺeláką zdrádę / y obłudę / y zazdrość / y wƺelákie obmowiſká :
Jáko dopiero národzone niemowiątká / ƺcżerego mleká ſłowá <i>Bożego</i> pożądajćie / ábyśćie przez nie urośli.
Jeſliśćie tylko ſkoƺtowáli że dobrotliwy jeſt PAN.
Do którego przyſtępując / do kámieniá żywego ; ácż od ludźi odrzuconego / ále od Bogá wybránego / y koƺtownego :
Y wy jáko żywe kámienie budujćie śię / w dom duchowny / w kápłáńſtwo święte / ku ofiárowániu duchownych ofiar / przyjemnych Bogu przez JEzuſá CHryſtuſá.
A przetoż mowi Piſmo : Oto kłádę ná Syonie kámień narożny węgielny / wybrány / koƺtowny ; á kto weń uwierzy / nie będźie záwſtydzony.
Wam tedy wierzącym <i>jest</i> ucżćiwośćią : ále niepoſłuƺnym / kámień który odrzućili budujący / ten śię ſtał głową węgielną / y kámieniem obráżenia / y opoką zgorƺenia :
Tym którzy śię obrażáją o ſłowo / niepoſłuƺni będąc / ná co też wyſtáwieni ſą.
Ale wy <i>jeſteśćie</i> rodzájem wybránym / królewſkim kápłáńſtwem / narodem świętym / ludem nábytym / ábyśćie opowiádáli cnoty tego / który was powołał z ćiemnośći / ku dźiwney ſwojey świátłośći.
Którzyśćie niekiedy <i>byli</i> nie ludem / áleśćie teraz ludem Bożym : którzyśćie <i>niekiedy</i> nie doſtąpili byli miłośierdźia / áleśćie teraz miłośierdźia doſtąpili.
Namilƺy / proƺę <i>was</i> ábyśćie śię jáko przychodniowie y gośćie wſtrzymawáli od ćieleſnych pożądliwośći / które walcżą przećiwko duƺey.
Obcowánie wáƺe májąc pocżćiwe miedzy Pogány : áby miáſto tego w cżym was pomawiáją jáko złocżyńce / dobrym śię ucżynkom <i>waƺym</i> przypátrując / chwalili Bogá w dźień náwiedzenia.
Bądźćież tedy poddáni wƺelkiemu ludzkiemu Urzędowi dla PAná / bądź Królowi / jáko nawyżƺemu :
Bądź Przełożonym / jáko od niego poſłánym / ku pomśćie źle cżyniących ; á ku chwale dobrze cżyniących.
Abowiem ták jeſt wola Boża / ábyśćie dobrze cżyniąc uſtá zátkáli nieumiejętnośći głupich ludźi.
Jáko wolni / á nie jáko ći którzy wolność zaſłoną złośći máją ; ále jáko ſłudzy Boży.
Wƺyſtkie cżćićie. Bráterſtwo miłujćie. Bogá śię bojćie. Królá w ucżćiwośći miejćie.
Słudzy / bądźćie poddáni pánom we wƺelákiey bojáźni ; nie tylko dobrym y bácżnym / ále y dźiwnym.
Boć to <i>jeſt</i> łáſká / jeſli kto dla ſumnienia Bożego ponośi fráſunki ćierpiąc bezwinnie.
Bo coż jeſt zá chwałá / jeſli grzeƺąc y policżkowáni będąc ćierpićie ? Ale jeſli dobrze cżyniąc y ćierpiąc znośićie / to jeſt łáſká u Bogá.
Abowiem ná to też powołáni jeſteśćie : Ponieważ y CHryſtus ćierpiał zá nas / zoſtáwiwƺy nam przykład / ábyśćie náśládowáli ſtop jego.
Który grzechu nie ucżynił / áni ználeźiona jeſt zdrádá w uśćiech jego.
Któremu gdy złorzecżono / nie odzłorzecżył : gdy ćierpiał / nie groźił : ále porucżył <i>krzywdę</i> temu który ſpráwiedliwie ſądźi.
Który grzechy náƺe ná ćiele ſwojim zánioſł ná drzewo ; ábyſmy obumárƺy grzechom / ſpráwiedliwośći żyli : Którego śiniáłośćią uzdrowieni jeſteśćie.
Abowiemeśćie byli jáko owce błądzące : ále teraz jeſteśćie náwroceni do Páſterzá y Biſkupá duƺ wáƺych.
