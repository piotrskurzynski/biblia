Piotr Apoſtoł JEzuſá CHryſtuſá / przychodniom roſproƺonym w Ponćie / w Gálácyey / w Kápádocyey / w Azyey / y w Bityniey :
Wybránym według przejrzenia Bogá Ojcá / przez poświęcenie Duchá / ku poſłuƺeńſtwu y pokropieniu krwie JEzuſá CHryſtuſá. Łáſká wam y pokój niech śię rozmnoży.
Błogoſłáwiony <i>niech będźie</i> Bóg y Oćiec PAná náƺego JEzuſá CHryſtuſá / który według wielkiego miłośierdźia ſwego odrodźił nas ku nádźiejey żywey przez zmartwychwſtánie JEzuſá CHryſtuſá od umárłych :
Ku dźiedźictwu nieſkáźitelnemu y niepokálánemu / y niezwiędłemu / w niebieśiech dla was záchowánemu :
Którzy mocą Bożą ſtrzeżeni bywaćie przez wiárę ku zbáwieniu / które zgotowáne jeſt / áby było objáwione cżáſu oſtátecżnego.
W cżym weſelićie śię / teraz málucżko ( jeſliże potrzebá ) záſmuceni w rozmájitych pokuſách.
Aby doświadcżenie wiáry wáƺey dáleko drożƺe niż złoto które ginie / którego jednák przez ogień doświadcżáją / ználeźione było / <i>wam</i> ku chwale / y ku cżći / y ku ſławie / w objáwienie JEzuſá CHryſtuſá.
Którego nie widźiawƺy miłujećie : którego teraz niewidząc / wƺákże weń wierząc / weſelićie śię rádośćią niewymowną y chwalebną :
Odnoƺąc koniec wiáry wáƺey / zbáwienie duƺ.
O którym zbáwieniu wywiádowáli śię / y bádáli śię Prorocy ; którzy o tey łáſce / która ná was przyść miáłá / prorokowáli.
Badájąc śię / ná który / álbo ná jáki cżás objawiał Duch CHryſtuſów który w nich <i>był</i> : świadcżąc pierwey o utrapieniách / które <i>miały przyść</i> ná CHryſtuſá / y o wielkiey zátym chwale.
Którym objáwiono jeſt / iż nie ſámym ſobie / ále nam tym uſługowáli / co wam teraz zwiáſtowano przez te którzy wam kazáli Ewángelią przez Duchá Świętego z niebá zeſłánego / ná które rzecży prágną pátrzyć Anjołowie.
Przetoż przepáſawƺy biodrá myśli wáƺey / y trzeźwimi będąc / doſkonáłą miejćie nádźieję ku tey łáſce / która wam dána będźie w objáwienie JEzuſá CHryſtuſá.
Jáko ſynowie poſłuƺni / którzy śię nie przypodobywaćie przeƺłym w nieumiejętnośći wáƺey pożądliwośćiom.
Ale jáko ten który was powołał święty jeſt ; y wy bądźćie świętymi we wƺelkim obcowániu :
Dla tego że nápiſano : Świętymi bądźćie / iżem Ja jeſt Święty.
A ponieważ Ojcem názywaćie tego / który bez bráku oſob káżdego ſądźi według ucżynku / <i>pátrzćież</i> ábyśćie w bojáźni cżás pielgrzymowánia wáƺego trawili :
Wiedząc iż nie ſkáźitelnymi <i>rzecżámi,</i> śrebrem álbo złotem wykupieni jeſteśćie / od márnego obcowánia wáƺego od ojców podánego :
Ale drogą krwią / jáko Báránká niewinnego y niepokalánego / CHryſtuſá :
Przejrzánego przed záłożeniem świátá / á objáwionego cżáſów oſtátecżnych dla was :
Którzy przezeń wierzyćie w Bogá / który go wzbudźił od umárłych / y dał mu chwałę / áby wiárá y nádźiejá wáƺá byłá w Bogu.
Ponieważeśćie duƺe wáƺe ocżyśćili poſłuƺeńſtwem prawdy przez Duchá <i>Świętego,</i> ku nieobłudney bráterſkiey miłośći / z cżyſtego ſercá jedni drugie miłujćie uprzyjmie :
Odrodzeni będąc nie z naśienia ſkáźitelnego / ále z nieſkáźitelnego przez ſłowo Boże żywe y trwájące ná wieki.
Ponieważ wƺelkie ćiáło jeſt jáko trawá / y wƺelka chwałá cżłowieká jáko kwiát trawy : uwiędłá trawá / y kwiát jey opadł.
Ale ſłowo Boże trwa ná wieki. A toć jeſt ſłowo / które wam jeſt zwiáſtowáne.
