A gdy śię on rozruch ućiƺył / zwoławƺy Páweł ucżniów / y <i>z nimi</i> śię pożegnawƺy / wyƺedł <i>z támtąd</i> áby ƺedł do Mácedoniey.
A przeƺedƺy one ſtrony / y nápomniawƺy je ƺerokimi ſłowy / przyƺedł do Grecyey.
A tám zmieƺkawƺy przez trzy mieśiące / gdźie nań Żydowie záſadzkę ucżynili / gdy miał płynąć do Syryey / umyślił śię náwróćić przez Mácedonią.
Y puśćił śię z nim áż do Azyey Sopáter Bereeńcżyk : A z Teſſálońcżyków Aryſtárchus y Sekundus y Gájus Derbejcżyk / y Tymoteuƺ :
A z Azyáńcżyków / Tychikus y Trofimus : którzy w przód poƺedƺy / cżekáli nas w Troádźie.
A my po dniách Przáſników odpłynęliſmy z Filippowá / y przyƺliſmy do nich do Troády zá pięć dni ; gdźieſmy zmieƺkáli śiedm dni.
Tedy w pierwƺy dźień po Sábbáćie / gdy śię ucżniowie zgromádźili ná łamánie chlebá / Páweł rozmáwiał z nimi / májąc iść precż názájutrz / y przedłużył mowę áż do północy.
A było wiele lámp ná oney ſali / gdźie byli zgromádzeni.
Tám śiedząc niektóry młodźieniec / imieniem Eutychus / w oknie ; będąc ćiężkim ſnem zjęty / gdy <i>ták</i> Páweł długo mówił / ſnem zmorzony ſpadł ná dół z trzećiego piętrá : y podnieśiony jeſt umárły.
A Páweł zſtąpiwƺy ná dół / przypadł nań ; á obłápiwƺy go / rzekł : Nie trwożćie śię ; boć w nim jeſt duƺá jego.
A wſtąpiwƺy záśię / łamał chleb / y jadł / y kazał im długo áż do świtánia ; potym odƺedł precż.
Y przywiedli one° młodźieńcá żywego / y byli náder ućieƺeni.
A my przyƺedƺy wprzód do okrętu / puśćiliſmy śię do Aſſonu / ábyſmy z támtąd wźięli Páwłá : ábowiem ták był poſtánowił / májąc ſam pieƺo iść.
A gdy śię z námi zƺedł w Aſſonie / wźiąwƺy go / przyjácháliſmy do Mityleny.
A z támtąd odpłynąwƺy / drugiego dniá przyƺliſmy przećiw Chiu : á trzećiego dniá przypłynęliſmy do Sámu : á pomieƺkawƺy w Trogillu / názájutrz przyƺliſmy do Miletu.
Abowiem Páweł umyślił był minąć Efes / áby mu nie przyƺło cżáſu trawić w Azyey : bo śię kwápił / jeſliby mu można / áby ná dźień Świątecżny był w Jeruzalem.
Tedy z Miletu poſławƺy do Efezu / przyzwał do śiebie Stárƺych Zborowych.
Którzy gdy do niego przyƺli / rzekł im : Wy wiećie od pierwƺego dniá / któregom przyƺedł do Azyey / jákom z wámi po wƺyſtek cżás był.
Służąc PAnu ze wƺelką uniżonośćią / y z wielą łez y pokus / które ná mię przychadzáły z záſadzek Żydowſkich.
Jákom śię nie zchraniał nicżego / coby było pożytecżno / ábym wam nie oznájmił y nie ucżył was jáwnie / y po domiech.
Świádectwo wydawájąc y Żydom y Grekom / o pokućie ku Bogu / y o wierze w PAná náƺego JEzuſá CHryſtuſá.
A oto teraz ja będąc związány Duchem / idę do Jeruzalem / nie wiedząc co tám ná mię przyść ma.
Tylko że Duch Święty po mieśćiech świadcży / powiádájąc że mię więźienie y ućiſki cżekáją.
Wƺákże ja ná nic nie dbam / y nie jeſt mi ták droga duƺá mojá : bym tylko bieg mój z rádośćią wykonał / y poſługę którąm wźiął od PAná JEzuſá / ná oświadcżenie Ewángeliey łáſki Bożey.
A teraz oto ja wiem / że już więcey nie oglądaćie oblicża mojego wy wƺyſcy / miedzy którymim chodźił każąc króleſtwo Boże.
Przetoż oświadcżam śię wam dniá dźiśiejƺego / żem ja jeſt cżyſty od krwie wƺyſtkich.
Abowiem nie chroniłem śię żebym wam nie miał oznájmić wƺelkiey rády Bożey.
Pilnujćież tedy ſámych śiebie / y wƺyſtkiey trzody / w którey was Duch Święty poſtánowił Biſkupámi ; ábyśćie páśli Zbór Boży / którego nábył przez właſną krew.
Boć ja to wiem / że po odeśćiu mojim wnidą miedzy was wilcy okrutni / którzy trzodźie folgowáć nie będą.
A z was ſámych powſtáną mężowie / mówiący <i>rzecży</i> przewrotne / áby zá ſobą poćiągnęli ucżnie.
Przetoż cżujćie / pomniąc żem przez trzy látá w nocy y we dnie nie przeſtawał nápomináć ze łzámi káżdego <i>z was</i>.
A teraz bráćia porucżam was Bogu / y ſłowu łáſki jego / który może pobudowáć y dáć wam dźiedźictwo miedzy wƺyſtkimi poświęconymi.
Śrebrá / álbo złotá / álbo ƺáty nie pożądałem od nikogo.
Owƺem ſámi wiećie / że mojim potrzebam / y tych którzy ſą zemną / ſłużyły te ręce.
Wƺyſtkomći wam okazał / iż ták prácując muśimy podejmowáć ſłábe / á pámiętáć ná ſłowá PAná JEzuſowe / że on rzekł : Szcżęśliwƺa jeſt rzecż dawáć / niżeli bráć.
A to powiedźiawƺy / klęknął ná koláná ſwoje / y modlił śię z nimi wƺyſtkimi.
Y ſtał śię wielki płácż wƺyſtkich : á upadájąc ná ƺyję Páwłowę / cáłowáli go.
Smęcąc śię bárzo / nawięcey z tych ſłów które im rzekł : Że już więcey nie mieli oglądáć oblicża jego. Y prowádźili go do okrętu.
