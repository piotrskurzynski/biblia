A byli w Antyochiey we Zborze który <i>tám</i> był / niektórzy Prorocy y náucżyćiele / jáko Bárnábaƺ y Symeon / którego zwano Niger / y Lucius Cyrenejcżyk / y Mánáhen / który był wychowány z Herodem Tetrárchą / y Saul.
A gdy oni ſłużbę PAńſką jáwnie odpráwowáli y pośćili / rzekł <i>im</i> Duch Święty : Odłącżćie mi Bárnábaƺá y Saulá do tey ſpráwy / do któreym ich powołał.
Tedy poƺcżąc y modląc śię / y wkładájąc ná nie ręce / odpráwili je.
Oni tedy wyſłáni będąc od Duchá Świętego / przyƺli do Seleuciey / á z támtąd płynęli do Cypru.
A gdy byli w Sáláminie / opowiádáli ſłowo Boże w bóżnicách Żydowſkich / á mieli <i>z ſobą</i> y Janá do uſługi.
A przeƺedƺy on wyſep / áż do Páfu / ználeſli tám jákiegoś cżárnokśiężniká / fałƺywego Proroká Żydá / któremu imię <i>było</i> Báriezus.
Który był przy zacnym Stárośćie Sergiuƺu Páwle / mężu roſtropnym. Ten przyzwawƺy Bárnábaƺá y Saulá / prágnął ſłucháć ſłowá Bożego.
Lecż śię im zprzećiwił Elimás on Cżárnokśiężnik / ( ábowiem śię ták wykłada imię jego / ) ſtárájąc śię jákoby Stároſtę od wiáry odwróćił.
Tedy Saul / ( którego <i>zowią</i> y Páweł ) nápełniony będąc Duchá Świętego / á pilnie nań pátrząc /
Rzekł : O pełny wƺelkiey zdrády y wƺelkiey przewrotnośći / ſynu Dyabelſki / nieprzyjaćielu wƺelkiey ſpráwiedliwośći / nie przeſtánieƺże podwracáć proſtych dróg PAńſkich?
A oto teraz ręká PAńſka nád tobą : y będźieƺ ślepym / nie widząc ſłońcá áż do cżáſu. A zárázem przypádłá nań chmurá y ćiemność : á błąkájąc śię / ƺukał ktoby go wiódł zá rękę.
Tedy widząc Stároſtá co śię ſtáło / uwierzył / zdumiewájąc śię nád náuką PAńſką.
A puśćiwƺy śię z Páfu / <i>Páweł</i> y ći którzy z nim byli / przyƺli do Pergi Pámfiliejſkiey. A Jan odƺedƺy od nich / wróćił śię do Jeruzalem.
A oni odƺedƺy z Pergi / przyƺli do Antyochiey Piſydejſkiey : á wƺedƺy do bóżnice w dźień Sobotni / uśiedli.
A po przecżytániu zakonu y Proroków / poſłáli do nich przełożeni bóżnice / mówiąc : Mężowie bráćia / maćieli <i>wolą</i> jákie nápominánie ucżynić do ludu / mówćie.
Tedy powſtawƺy Páweł / á ręką ſkinąwƺy rzekł : Mężowie Izráelſcy / y którzy śię bojićie Bogá / ſłuchajćie.
Bóg ludu tego Izráelſkiego / wybrał Ojce náƺe / y wywyżƺył lud gdy byli przychodniámi w źiemi Egipſkiey / y w rámieniu wyćiągnionym wywiódł je z niey.
Y przez cżás cżterdźieſtu lat / znośił obycżáje ich ná puƺcży.
A wygłádźiwƺy śiedm narodów w źiemi Chánánejſkiey / loſem rozdźielił miedzy nie onę źiemię ich.
A potym około cżterech ſet y piąćidźieśiąt lat / dawał <i>im</i> Sędźie / áż do Sámuelá Proroká.
A od onego cżáſu prośili o Królá. Y dał im Bóg Saulá / ſyná Cyſowego / mężá z pokolenia Benjáminowego / przez lat cżterdźieśći.
A gdy go odrzućił / wzbudźił im Dawidá zá Królá ; któremu też świádectwo wydawájąc / powiedźiał : Ználazłem Dawidá ſyná Jeſſego / mężá według ſercá mego / który będźie cżynił wƺyſtkę wolą moję.
Z jegoż naśienia Bóg według obietnice wzbudźił Izráelowi zbáwićielá JEzuſá.
Przed którego przyśćiem kazał Jan chrzeſt pokuty wƺyſtkiemu ludowi Izráelſkiemu.
A gdy Jan dokonał biegu <i>ſwego,</i> rzekł : Kim mię być mniemaćie / nie jeſtem ja : ále oto idźie zá mną / u którego nóg obuwia nie jeſtem godźien rozwiązáć.
Mężowie bráćia / ſynowie narodu Abráhámowego / y którzy śię miedzy wámi Bogá boją / wamći ſłowo zbáwienia tego poſłáne jeſt.
Abowiem ći co mieƺkáją w Jeruzalem y Przełożeni ich / nieznájąc tego <i>JEzuſá</i> y głoſów Prorockich / które przez káżdy Sábbát bywáją cżytáne ; wypełnili je oſądźiwƺy go.
A żadney przycżyny śmierći <i>w nim</i> nie ználazƺy / prośili Piłatá / áby był zábity.
A gdy wykonáli wƺyſtko co o nim było nápiſano / zdjąwƺy go z drzewá / włożyli go do grobu.
Ale go Bóg wzbudźił od umárłych.
Który widźiány jeſt przez wiele dni / od tych którzy z nim poſpołu przyƺli z Gálilejey do Jeruzalem / którzy ſą świádkámi jego do ludu.
Y my wam opowiádamy tę obietnicę / która śię Ojcom ſtáłá : iż ją Bóg wypełnił / nam dźiatkom ich / wzbudźiwƺy JEzuſá.
Jáko też w Pſálmie wtórym nápiſano jeſt : Syn mój jeſteś ty / Jam ćiebie dźiś zpłodźił.
A iż go wzbudźił od umárłych / áby śię więcey nie wróćił do ſkáżenia / ták powiedźiał : Dam wam święte dobrodźiejſtwá Dawidowe wierne.
Przeto y indźiey powiáda : Nie daƺ Świętemu twemu widźieć ſkáżenia.
Abowiemći Dawid <i>zá</i> wieku ſwego uſłużywƺy woli Bożey / záſnął y przyłącżony jeſt do Ojców ſwojich / á widźiał ſkáżenie.
Lecż ten którego Bóg wzbudźił / nie widźiał ſkáżenia.
Niechże wam tedy będźie wiádomo mężowie bráćia / iż śię wam przez tego opowiáda odpuƺcżenie grzechów :
Y od wƺyſtkiego / od cżegośćie nie mogli być przez Zakon Mojzeƺów uſpráwiedliwieni / przez tego káżdy wierzący uſpráwiedliwiony bywa.
A przetoż pátrzćie / áby ná was nie przyƺło to co powiedźiano w Prorocech :
Obacżćie wy wzgárdźićiele / y dźiwujćie śię / á wniwecż śię obróććie : Bo Ja ſpráwuję ſpráwę zá dni wáƺych / ſpráwę którey nie wierzyćie / choćby wam kto <i>o niey</i> powiádał.
A gdy oni wychodźili z bóżnice Żydowſkiej / prośili ich Pogánie / áby y w drugi Sábbát mówili do nich też ſłowá.
A po rozpuƺcżeniu zgromádzenia / poƺło wiele Żydów / y nabożnych Nowowierników zá Páwłem y Bárnábaƺem : którzy mówiąc do nich / rádźili im áby trwáli w łáſce Bożey.
A w drugi Sábbát / niemal wƺyſtko miáſto śię zgromádźiło ná ſłuchánie ſłowá Bożego.
Tedy Żydowie widząc lud / nápełnieni ſą zázdrośći ; y zprzećiwiáli śię temu co Páweł powiádał / mówiąc przećiwko temu y bluźniąc.
A Páweł y Bárnábaƺ / beſpiecżnie mówiąc / rzekli : Wamći napierwey miáło być opowiádáne ſłowo Boże ; ále ponieważ je odrzucaćie / á oſądzaćie śię być niegodnymi żywotá wiecżnego / otóż śię obracamy do Pogan.
Abowiem nam ták rozkazał PAN / <i>mówiąc</i> : Położyłem ćię świátłośćią Pogánom / ábyś był zbáwieniem áż do krájów źiemie.
A ſłyƺąc to Pogánie / rádowáli śię / y wielbili ſłowo PAńſkie : y uwierzyli ile ich kolwiek było zporządzonych do żywotá wiecżnego.
Y roznośiło śię ſłowo PAńſkie po wƺyſtkiey oney krájinie.
A Żydowie poduƺcżáli niewiáſty nabożne y ucżćiwe / y przedniejƺe w mieśćie : á wzbudźili prześládowánie przećiwko Páwłowi y przećiwko Bárnábaƺowi / y wygnáli je z gránic ſwojich.
A oni otrząſnąwƺy proch z nóg ſwojich ná nie / przyƺli do Ikoniey.
A ucżniowie byli nápełnieni rádośći y Duchá Świętego.
