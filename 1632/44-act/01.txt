Pierwƺe wprawdźie Kśięgi nápiſałem / <i>ô</i> Teofile! o wƺyſtkim co pocżął JEzus y cży nić y ucżyć :
áż do dniá onego / które° dawƺy rozkazánie Apoſtołom które był przez Duchá Ś. obrał / wźięty jeſt wzgórę.
Którym też ſámego śiebie po męce ſwojey żywym ſtáwił / w wielu niewątpliwych dowodách / przez cżterdźieśći dni ukázując śię im / y mówiąc o króleſtwie Bożym.
A zgromádźiwƺy je / przykazał im áby nie odchodźili z Jeruzalem : ále iżby cżekáli obietnice Ojcowſkiey : o któreyśćie <i>mówi,</i> ſłyƺeli ode mnie.
Abowiemći Jan chrżćił wodą : ále wy będźiećie ochrzcżeni Duchem Świętym po nie wielu tych dni.
A ták oni zƺedƺy śię / pytáli go mówiąc : PAnie / izali w tym cżáśie nápráwiƺ króleſtwo Izráelſkie?
Lecż on rzekł do nich : Nie wáƺá <i>rzecż</i> jeſt znáć cżáſy y chwile / które Oćiec w ſwojey mocy położył.
Ale przyjmiećie moc Duchá Świętego / który przydźie ná was : y będźiećie mi świádkámi y w Jeruzalem / y we wƺyſtkiey Judſkiey źiemi / y w Sámáryjey áż do oſtátniego <i>kráju</i> źiemie.
A to rzekƺy / gdy oni pátrzáli / wzgórę podnieśiony jeſt : á obłok wźiął go od ocżu ich.
A gdy zá nim do niebá idącym pilnie pátrzáli / oto dwá mężowie ſtánęli przy nich w białym odźieniu /
Y rzekli : Mężowie Gálilejſcy / przecż ſtojićie pátrząc w niebo? Ten JEzus który wzgórę wźięty jeſt od was do niebá / ták przydźie / jákośćie go widźieli idącego do niebá.
Tedy śię wróćili do Jeruzalem od góry / którą zowią Oliwną / która jeſt bliſko Jeruzalem / májąc drogi <i>przez jeden</i> Sábbát.
A gdy weƺli / wſtąpili ná ſalę / gdźie mieƺkáli ; Piotr / y Jákób / y Jan / y Andrzyj / y Filip / y Tomaƺ / Bártłomiey / y Mátteuƺ / Jákób Alfeuƺów / y Symon Zelotes / y Judás Jákubów.
Ci wƺyſcy trwáli jednomyślnie ná modlitwie y proźbách / z żonámi / y z Máryą mátką JEzuſową / y z bráćią jego.
A woneż dni / powſtawƺy Piotr w pośrzodku ucżniów / rzekł ; ( á był pocżet oſób weſpół zgromádzonych około ſtá y dwudźieſtu )
Mężowie bráćia / muśiáło śię wypełnić ono piſmo / które opowiedźiał Duch Święty przez uſtá Dawidowe o Judaƺu / który był wodzem tych co pojimáli JEzuſá.
Bo był policżony z námi / y doſtał był cżąſtki tego uſługowánia.
Tenći wprawdźie otrzymał rolą z zapłáty nieſpráwiedliwośći : á obieśiwƺy śię / rozpukł śię ná poły / y wypłynęły wƺyſtkie wnętrznośći jego.
Y było to jáwno wƺyſtkim mieƺkájącym w Jeruzalem / ták iż názwano onę rolą właſnym ich językiem / Akeldámá ; to jeſt / rola krwie.
Abowiem nápiſano w Kśięgách Pſálmów ; Niechaj będźie mieƺkánie jego puſte / á niech nie będźie ktoby w nim mieƺkał / á biſkupſtwo jego niech weźmie inny.
Potrzebá tedy / áby jeden z tych mężów którzy z námi bywáli po wƺyſtek cżás / którego PAn JEzus przebywał miedzy námi /
Pocżąwƺy od chrztu Janowego / áż do tego dniá / które° jeſt wźięty wzgórę od nas / był z námi świádkiem zmartwychwſtánia jego.
Y poſtáwili dwu : Józefá / którego zwano Bárſábaƺem / którego <i>też</i> názywano Juſtem ; y Máćiejá.
A modląc śię mówili : Ty PAnie który znaƺ ſercá wƺyſtkich / okaż z tych dwu jednego / któregoś obrał /
Aby przyjął cżąſtkę uſługowánia tego / y Apoſtolſtwá / z którego wypadł Judaƺ / áby odƺedł ná miejſce ſwoje.
Y rzućili loſy ich. Y padł los ná Máćiejá : á przyłącżony jeſt ſpólnym zdániem do jedenaſtu Apoſtołów.
