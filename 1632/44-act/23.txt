A Páweł pilnie pátrząc ná onę Rádę / rzekł : Mężowie bráćia / ja ze wƺyſtkiego ſumnienia dobrego chodźiłem przed Bogiem / áż do dniá tego.
Tedy Anániaƺ Nawyżƺy Kápłan / rozkazał go tym którzy przy nim ſtali bić w gębę.
Tedy rzekł Páweł do niego : Uderzy ćię Bóg śćiáno pobielána ; y ty śiędźiƺ ſądząc mię według Zakonu / á rozkázujeƺ mię bić przećiwko Zakonowi?
Zátym ći którzy tám ſtali / rzekli : Nawyżƺemu kápłanowi Bożemu złorzecżyƺ?
A Páweł rzekł : Nie wiedźiałem bráćia / żeby był Nawyżƺym Kápłanem. Bo nápiſano : Kśiążęćiu ludu twego złorzecżyć nie będźieƺ.
A poznawƺy Páweł / że ich jedná cżęść byłá Sáduceuƺów / á drugá Fáryzeuƺów / záwołał w oney Rádźie : Mężowie bráćia / jam jeſt Fáryzeuƺ ſyn Fáryzeuƺów : o nádźieję y <i>o</i> powſtánie umárłych mnie tu <i>dźiś</i> ſądzą.
A gdy on to mówił / wƺcżął śię rozruch miedzy Fáryzeuƺámi y Sáduceuƺámi : y rozerwáło śię ono mnóſtwo.
Abowiem Sáduceuƺowie mówią / iż niemáƺ <i>zmartwych</i>wſtánia / áni Anjołá / áni Duchá : ále Fáryzeuƺowie to oboje wyznawáją.
Y wƺcżęło śię wołánie wielkie. A powſtawƺy Náucżeni w piśmie z ſtrony Fáryzeuƺów / ſpieráli śię / mówiąc : Niceſmy złego nie ználeźli w tym cżłowieku : y jeſli mu co powiedźiał Duch / álbo Anjoł / nie walcżmyƺ z Bogiem.
A gdy śię wƺcżął wielki rozruch / obawiájąc śię Hetman / áby Páwłá miedzy ſobą nie rozƺárpáli / rozkazał iść żołnierzom ná dół / á wydrzeć go z pośrzodku ich / y odwieść do obozu.
A drugiey nocy ſtánąwƺy przy nim PAN / rzekł : Bądź dobrego ſercá Páwle. Abowiem jákoś o mnie świadcżył w Jeruzalem / ták muśiƺ świadcżyć y w Rzymie.
A gdy był dźień / zƺedƺy śię niektórzy z Żydów / záwiązáli śię klątwą / mówiąc : Że nie mieli jeść áni pić / áżby Páwłá zábili.
A było ich więcey niż cżterdźieśći / którzy to zprzyśiężenie ucżynili.
Którzy przyƺedƺy do Przedniejƺych kápłanów / y do Stárƺych / rzekli : Klątwąſmy śię záwiązáli / że nic nie ukuśimy áżbyſmy Páwłá zábili.
Przetoż wy teraz dájćie znáć Hetmanowi z pozwoleniem <i>wƺyſtkiej</i> Rády / áby go jutro do was wywiódł / jákobyśćie śię chćieli doſtátecżniey wywiedźieć o ſpráwách jego ; á my pierwey niż tu przydźie / gotowiſmy go zábić.
A gdy uſłyƺał śieſtrzeniec Páwłów o tey záſadzce / przyƺedł ; á wƺedƺy do obozu / oznájmił <i>to</i> Páwłowi.
Tedy Páweł záwoławƺy jednego z Setników / rzekł : Záprowadź tego młodźieńcá do Hetmáná bo mu coś ma powiedźieć.
A ták on wźiąwƺy go / wiódł go do Hetmáná / y rzekł : Páweł więźień / záwoławƺy mię prośił / ábym tego młodźieńcá przywiódł do ćiebie / któryć ma coś powiedźieć.
Tedy Hetman wźiąwƺy go zá rękę / y uſtąpiwƺy ná ſtronę / wywiádował śię ; Cóż to jeſt co mi maƺ powiedźieć?
A <i>on</i> rzekł : Poſtánowili Żydowie prośić ćię / ábyś jutro wywiódł Páwłá przed Rádę / jákoby śię chćieli co doſtátecżniejƺego wywiedźieć o nim.
Ale ty nie pozwalaj im <i>tego</i> : bo śię nań náſádźiło z nich więcey niż cżterdźieśći mężów / którzy śię klątwą záwiązáli / iż nie máją áni jeść áni pić / áżby go zábili. Y ſą już pogotowiu / cżekájąc od ćiebie odpowiedźi.
Tedy Hetman odpráwił onego młodźieńcá / przykazawƺy mu / áby tego przed nikim nie powiádał iż mu to oznájmił.
A záwoławƺy dwu niektórych z Setników / rzekł : Nágotujćie dwie śćie żołnierzów / áby ƺli áż do Cezáryey ; do tego śiedmdźieśiąt jezdnych y dwie śćie drábántów / ná trzećią godźinę w noc.
Nágotowáć też bydlętá ; áby wſádźiwƺy Páwłá <i>ná nie,</i> zdrowo go záprowádzono do Felixá Stároſty :
Nápiſawƺy liſt / w ten ſpoſób ;
Klaudiuƺ Lizyaƺ Namożniejƺemu Stárośćie Felixowi zdrowia życży.
Tego mężá pojimánego od Żydów / gdy już od nich miał być zábity / przypadƺy z rotą odjąłem go / dowiedźiawƺy śię iż jeſt Rzymiáninem.
A chcąc wiedźieć przycżynę / dla któreyby nań ſkárżyli / wywiodłem go przed ich Rádę.
<i>Y</i> ználazłem że nań ſkárżą o <i>jákieś</i> gadki z ſtrony Zakonu ich ; á że nie ma żadney winy / dla któreyby był godźien śmierći álbo więźienia.
A gdy mi powiedźiano o záſadzce którą mieli ucżynić Żydźi ná tego mężá / zárazem <i>go</i> poſłał do ćiebie ; opowiedźiawƺy też tym co nań ſkárżyli / áby przed tobą mówili to / coby przećiwko niemu <i>mieli</i>. Miej śię dobrze.
Żołnierze tedy ták jáko im było rozkazano / wźiąwƺy Páwłá / prowádźili go nocą do Antipátridy.
A názájutrz / zoſtáwiwƺy jezdne / áby z nim jácháli / wróćili śię do obozu.
Którzy przyjáchawƺy do Cezáryey / á oddawƺy liſt Stárośćie / ſtáwili przed nim y Páwłá.
A Stároſtá <i>liſt</i> przecżytawƺy / zpytał <i>go</i> z któreyby był krájiny : á zrozumiawƺy / że był z Cylicyey /
Rzekł : Będę ćię ſłuchał / gdy <i>też</i> przybędą ći którzy ná ćię ſkárżą. Y rozkazał go ſtrzedz ná Ratuƺu Herodowym.
