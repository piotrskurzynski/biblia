Y ſtáło śię po śmierći Saulowey / gdy śię Dawid wróćił od poráżki Amálekitów / że zmieƺkał Dawid w Sycelegu przez dwá dni.
Tedy dniá trzećiego / przybieżał mąż z obozu Saulowego / á ƺáty jego rozdárte były / y proch ná głowie jego : Który przyƺedƺy do Dawidá / upadł ná źiemię / y pokłonił śię.
Y rzekł do niego Dawid ; z kąd idźieƺ : Y odpowiedźiał mu : Z obozum Izráelſkiego uƺedł.
Rzekł mu z nowu Dawid : Cóż śię ſtáło / proƺę / powiedz mi? Y odpowiedźiał : <i>To,</i> że ućiekł lud z bitwy / á do tego wiele poległo z ludu / y pomárło : Tákże y Saul / y Jonátán Syn jego polegli.
Zátym rzekł Dawid do młodźieńcá który mu <i>to</i> powiedźiał : Jákoż wieƺ / iż umárł Saul y Jonátán Syn jego?
Odpowiedźiał mu młodźieniec który mu to oznájmił : przyƺedłem z trefunku ná górę Gelboe / á oto Saul tkwiał ná włócżni ſwojey / á wozy y jezdni doganiáli go.
Tedy obejrzawƺy śię obacżył mię / y záwołał ná mię / y rzekłem ; Owom ja :
Y rzekł mi / Coś ty <i>zácż</i>? A jam mu odpowiedźiał / Jeſtem Amálekitá.
Y rzekł mi : Stań / proƺę / náde mną / á zábij mię : bo mię zjęły ćięƺkośći / gdyż jeƺcże wƺyſtká duƺá mojá we mnie <i>jeſt</i>.
Przetoż ſtánąwƺy nád nim zábiłem go : bom wiedźiał że nie będźie żyw po upadku ſwojim / y wźiąłem koronę / która <i>była</i> ná głowie jego / y záwieƺenie które <i>było</i> ná rámieniu jego / á przynioſłem je tu do Páná mego :
Tedy Dawid pochwyćiwƺy ƺáty ſwoje / rozdárł je : tákże y wƺyſcy mężowie którzy z nim byli.
A żáłując płákáli / y pośćili áż do wiecżorá / dla Saulá / y dla Jonátáná Syná jego / y dla ludu PAńſkiego / y dla domu Izráelſkiego / że polegli od miecżá.
Y rzekł Dawid do młodźieńcá który mu <i>to</i> oznájmił : Zkądeś ty? y odpowiedźiał ; Jeſtem Synem mężá przychodniá Amálekity.
Zátym rzekł do niego Dawid : Jákożeś śię nie bał zćiągnąć ręki twey / ábyś zábił pomázáńcá PAńſkiego?
Záwołał tedy Dawid jednego z ſług / y rzekł : Przyſtąp á zábij go : A on go uderzył że umárł.
Y rzekł do niego Dawid : Krew twojá ná głowę twoję : bo uſtá twoje świadcżyły przećiw tobie / mówiąc : Jam zábił pomázáńcá PAńſkiego.
Lámentował tedy Dawid lámentem tákowym nád Saulem / y nád Jonátánem Synem jego.
( Rozkazawƺy jednák / áby ucżono Synów Judſkich z łuku <i>ſtrzeláć,</i> jáko nápiſano w Kśięgách Jáſár. )
O ozdobo Izráelſka / ná górách twojich zránionyś jeſt : Jákoż polegli mocarze twoji!
Nie powiádajćieƺ w Get / áni rozgłaƺajćie po ulicách w Aƺkálonie : áby śię ſnadź nie weſeliły córki Filiſtyńſkie / by śię ſnadź nie rádowáły córki nieobrzezáńców.
O Góry Gelboe / áni roſá / áni deƺcż niech <i>nie upada</i> ná was / y niech <i>tám nie będą</i> polá urodzájne : ábowiem tám porzucona jeſt tarcż mocárzów / tarcż Saulowá / <i>jakoby</i> nie był pomázány olejem.
Od krwie zábitych / y od ſádłá mocárzów <i>ſtrzałá</i> łuku Jonátánowego nie wracáłá śię ná wſtecż / á miecż Saulów nie wracał śię prózno.
Saul y Jonátán / miłośni y przyjemni zá żywotá ſwego / y w śmierći ſwojey nie ſą rozłącżeni / nád Orły lekczejƺy / nád lwy mocnieyƺy byli.
Córki Izráelſkie płácżćie nád Saulem / który was przyodźiewał ƺárłatem roſkoƺnym / á który was ubierał w klenoty złote ná ƺáty wáƺe.
Jákoż polegli mocarze w pośrzód bitwy! Jonátán ná górách twojich zábity jeſt!
Bárzo mi ćię żal bráćie mój Jonátánie / byłeś mi bárzo wdźięcżnym : więkƺa u mnie byłá miłość twojá niż miłość niewieśćia.
Jákoż polegli mocarze / á poginęłá broń wojenna?
