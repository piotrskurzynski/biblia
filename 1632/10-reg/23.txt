A teć ſą oſtátecżne ſłowá Dawidowe : Rzekł Dawid Syn Iſájego : rzekł mówię mąż / który był zacnie wywyżƺony / pomázániec Bogá Jákóbowego / y wdźięcżny w pieśniách Izráelſkich.
Duch PAńſki mówił przez mię / á ſłowá jego <i>pochodźiły</i> przez język mój.
Mówił Bóg Izráelſki do mnie / mówiłá ſkáłá Izráelſka : ten który pánowáć będźie nád ludem / będźie ſpráwiedliwy / pánowáć będźie w bojáźni Bożey.
<i>Będźie</i> jáko bywa świátłość poránna / gdy ſłońce ráno bez obłoków wſchodźi : <i>á jáko</i> od jáſnośći po deƺcżu wyraſta źiele z źiemie.
A choć nie táki <i>jeſt</i> dom mój przed Bogiem : jednak przymierze wiecżne poſtánowił ze mną utwirdzone we wƺyſtkim y obwárowáne : A <i>w tymći</i> jeſt wƺyſtko zbáwienie moje / y wƺyſtká ućiechá <i>mojá,</i> ácżkolwiek temu jeƺcże wzroſtu nie dawa.
Ale niepobożni / wƺyſcy będą jáko ćiernie wyrwáni / którego rękomá nie biorą.
Lecż kto śię go jedno chce dotknąć / obwáruje śię żelázem y drzewem włócżniey / álbo ogniem wypala je do ƺcżętu / ná miejſcu <i>jego</i>.
Teć ſą imioná mocárzów / które miał Dawid : Jozeb Báſebet Táchmojicżyk / naprzedniejƺy miedzy trzemá : który śię z ućiechą rzućił ná ośm ſet <i>ludu</i> z włócżnią / áby je zábił w jedney potrzebie.
A po nim był Eleázár Syn Dodonów / Syná Ahohowego / miedzy trzemá mocárzámi / którzy byli z Dawidem ; á ſromotnie lżyli Filiſtyny / którzy śię byli zebráli ku bitwie / gdy byli odćiągnęli mężowie Izráelſcy.
Ten powſtawƺy / bił Filiſtyny / ták / iż uſtáłá ręká jego / y zdrętwiáłá ręká jego przy miecżu : Tedy ſpráwił PAN wielkie wybáwienie dniá onego / ták że śię lud wróćił zá nim / tylko áby korzyśći zbierał.
A po nim <i>był</i> Semmá / Syn Age / Hárárcżyk : ábowiem gdy śię byli Filiſtynowie zebráli do kupy / kędy byłá cżęść polá pełnego ſocżewice / á lud <i>inny</i> był ućiekł przed Filiſtynámi :
Tedy ſtánąwƺy w pośrzód oney cżęśći <i>polá</i> obronił go / y pobił Filiſtyny : á ták ſpráwił PAN wielkie wybáwienie.
Wyƺli też oni trzej z trzydźieſtu przedniejƺych / á przyƺli we żniwá do Dawidá / do jáſkini Odollám / gdy śię wojſko Filiſtyńſkie było obozem położyło w dolinie Refáim.
A Dawid ná ten cżás <i>był</i> ná miejſcu obronnym : ſtraż też Filiſtyńſka ná ten cżás <i>byłá</i> w Betlehem :
Tedy prágnął Dawid / y rzekł ; O by mi śię kto dał nápić wody z ſtudni Betlehemſkiey / która jeſt u bramy!
Przetoż wpádli ći trzey mocárze do obozu Filiſtyńſkiego / y nácżerpáli wody z ſtudnie Betlehemſkiey / która <i>byłá</i> u bramy : którą nieśli / y przynieśli do Dawidá. Ale jey on nie chćiał pić / lecż ją wylał przed PANEM.
Y rzekł : Nie daj mi tego PAnie / ábym to miał ucżynić : izali to nie krew mężów / którzy ƺli z niebeſpiecżeńſtwem duƺ ſwojich? y nie chćiał jey pić. Toć ucżynili oni trzej mocárze.
Tákże Abiſáj brát Joábów / Syn Sárwijey / był przedniejƺym miedzy trzemá. Ten podnióſł włócżnią ſwą przećiwko trzemá ſtom / y zábił je : y był ſławnym miedzy trzemá.
Z tych trzech będąc naſławniejƺym / był ich Hetmánem / wƺákże onych trzech <i>pierwƺych</i> nie doƺedł.
Bánájás też Syn Jojády / Syn mężá rycerſkiego / zacny w ſwych ſpráwách z Kábſeel : ten zábił dwu mocárzów Moábſkich / tenże ƺedƺy zábił lwá w pośrzodku ſtudnie we dni śnieżne.
Tenże zábił mężá Egipcżániná / mężá ná podźiw <i>wielkiego.</i> który Egipcżánin <i>miał</i> w ręku włócżnią : á on ƺedł ku niemu z kijem / á wydárƺy włócżnią z ręki Egipcżániná / zábił go włócżnią jego.
Toć ucżynił Bánájás Syn Jojády / który też <i>był</i> ſławny miedzy onymi trzemá mocárzámi.
Z tych trzydźieſtu był naſławniejƺym : Wƺákże onych trzech niedoƺedł / y poſtáwił go Dawid nád drábántámi ſwojimi.
