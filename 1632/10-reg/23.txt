A teć ſą oſtátecżne ſłowá Dawidowe. Rzekł Dawid / ſyn Iſájego / rzekł mówię mąż / który był zácnie wywyżƺony / pomázániec Bogá Jákóbowego / y wdźięcżny w pieśniách Izráelſkich :
Duch Páńſki mówił przez mię / á ſłowá jego pochodźiły przez język mój.
Mówił Bóg Izráelſki do mnie / mówiłá ſkáłá Izráelſká : Ten / który pánowáć będźie nád ludem / będźie ſpráwiedliwy / pánowáć będźie w bojáźni Bożey.
Będźie jáko bywa świátłość poránná / gdy ſłońce ráno bez obłoków wſchodźi / á jáko od jáſnośći po deƺcżu wyráſtá źiele z źiemi.
A choć nie táki jeſt dom mój przed Bogiem / jednák przymierze wiecżne poſtánowił ze mną / utwierdzone we wƺyſtkim y obwárowáne. A w temći jeſt wƺyſtko zbáwienie moje / y wƺyſtká ućiechá mojá / ácżkolwiek temu jeƺcże wzroſtu nie dáwá.
Ale niepobożni wƺyſcy będą jáko ćierń wyrwáni / którego rękomá nie biorą.
Lecż kto śię go jedno chce dotknąć / obwáruje śię żelázem y drzewem włócżni / álbo ogniem wypálá go do ƺcżętu ná miejſcu jego.
Teć ſą imioná mocárzów / które miał Dawid : Jozeb Báſebet Táchmojcżyk / nájprzedniejƺy miedzy trzemá ; który śię z ućiechą rzućił ná ośm ſet ludu z włócżnią / áby je zábił w jedney potrzebie.
A po nim był Eleázár / ſyn Dodoná / ſyná Ahohowego / miedzy trzemá mocárzámi / którzy byli z Dawidem ; á ſromotnie lżyli Filiſtyny / którzy śię byli zebráli ku bitwie / gdy byli odćiągnęli mężowie Izráelſcy.
Ten powſtáwƺy bił Filiſtyny / ták iż uſtáłá ręká jego / y zdrętwiáłá ręká jego przy miecżu. Tedy ſpráwił Pán wielkie wybáwienie dniá onego / ták / że śię lud wróćił zá nim / tylko áby korzyśći zebrał.
A po nim był Semmá / ſyn Agi / Hárárcżyk ; ábowiem gdy śię byli Filiſtynowie zebráli do kupy / kędy byłá cżęść polá pełnego ſocżewicy / á lud inny był ućiekł przed Filiſtynámi :
Tedy ſtánąwƺy w pośród oney cżęśći polá / obronił go / y pobił Filiſtyny. A ták ſpráwił Pán wielkie wybáwienie.
Wyƺli też oni trzej z trzydźieſtu przedniejƺych / á przyƺli we żniwá do Dawidá / do jáſkini Odollám / gdy śię wojſko Filiſtyńſkie było obozem położyło w dolinie Refáim.
A Dawid ná ten cżás był ná miejſcu obronnem : ſtráż też Filiſtyńſká ná ten cżás byłá w Betlehem.
Tedy prágnął Dawid / y rzekł : O by mi śię kto dał nápić wody z ſtudni Betlehemſkiej / która jeſt u bramy!
Przetoż wpádli ći trzej mocárze do obozu Filiſtyńſkiego / y nácżerpáli wody z ſtudni Betlehemſkiej / która byłá u bramy ; którą nieśli / y przynieśli do Dawidá. Ale jey on nie chćiał pić / lecż ją wylał przed Pánem.
Y rzekł : Nie dáj mi tego Pánie / ábym to miał ucżynić. Izali to nie krew mężów / którzy ƺli z niebezpiecżeńſtwem duƺ ſwojich? Y nie chćiał jey pić. Toć ucżynili oni trzej mocárze.
Tákże Abiſáj brát Joábá / ſyn Sárwii / był przedniejƺym miedzy trzemá. Ten podnióſł włócżnią ſwą przećiwko trzemá ſtom / y zábił je / y był ſłáwnym miedzy trzemá.
Z tych trzech będąc nájſłáwniejƺym / był ich hetmánem ; wƺákże onych trzech pierwƺych nie doƺedł.
Bánájás też ſyn Jojády / ſyn mężá rycerſkiego / zácny w ſwych ſpráwách / z Kábſeel ; ten zábił dwu mocárzów Moábſkich / tenże ƺedƺy zábił lwá w pośrzodku ſtudni / we dni śnieżne.
Tenże zábił mężá Egipcżániná / mężá ná podźiw wielkiego / który Egipcżánin miał w ręku włócżnią ; á on ƺedł ku niemu z kijem / á wydárƺy włócżnią z ręki Egipcżániná / zábił go włócżnią jego.
Toć ucżynił Bánájás / ſyn Jojády / który też był ſłáwny miedzy onymi trzemá mocárzámi.
Z tych trzydźieſtu był nájſłáwniejƺym ; wƺákże onych trzech nie doƺedł ; y poſtáwił go Dawid nád drábántámi ſwoimi.
Był też Aſáel / brát Joábá / miedzy trzydźieſtomá. A ći ſą : Elkánán / ſyn Dodoná / Betlehemcżyk ;
Sámmá Hárodcżyk ; Eliká Hárodcżyk.

Abijezer Anátotcżyk ; Mobonáj Huſátcżyk ;
Selmon Ahohytcżyk ; Máháráj Netofátcżyk ;
Heleb / ſyn Báány / Netofátcżyk ; Itáj / ſyn Rybájego / z Gábáát ſynów Benjáminowych ;
Bánájás Fárátońcżyk ; Háddáj od potoku Gáás ;
Abijálbon Arbátcżyk ; Aſmáwet Bárchomcżyk ;
Elijáchbá Sálábońcżyk ; z ſynów Jáſſonowych Jonátán ;
Semmá Hororcżyk ; Ahijám / ſyn Sárárá / Arárytcżyk ;
Elifelet / ſyn Acháſbájá / ſyná Máchátego ; Elijám / ſyn Achitofelá Gilońcżyká.
Hezráj Kármelcżyk ; Fáráj Arbitcżyk.
Igál / ſyn Nátáná z Soby ; Báni Gádcżyk.
Selek Ammonitcżyk ; Nácháráj Berotcżyk / który nośił broń Joábá / ſyná Sárwii ;

Uryjáƺ Hetejcżyk. Owá wƺyſtkich trzydźieśći y śiedm.
