Y był głód zá dni Dawidowych przez trzy látá / jednego roku po drugim : tedy ƺukał Dawid oblicża PAńſkiego / któremu rzekł PAN : Dla Saulá / y dla domu jego krwáwego : przeto iż pomordował Gábáonity.
Przyzwał tedy Król Gábáonitów / y rzekł do nich : ( A ći Gábáonitowie nie byli z Synów Izráelſkich / ále z oſtátków Amorrejcżyków : którym ácż byli Synowie Izráelſcy przyśięgli / wƺákże je uśiłował Saul wyplenić z gorliwośći ſwey dla Synów Izráelſkich y Judſkich. )
Y rzekł Dawid do Gábáonitów : Cóż wam mam ucżynić? á cżym was ubłagáć / ábyśćie błogoſłáwili dźiedźictwu PAńſkimu?
Y odpowiedźieli mu Gábáonitowie : Nie idźie nam o śrebro / áni o złoto z Saulem / y z domem jego : áni o to żebyſmy zábili kogo w Izráelu? A <i>on</i> rzekł : Cokolwiek rzecżećie ucżynię wam.
Którzy rzekli do Królá : Mężá który nas wygubił / y ná tym był áby nas do ƺcżętu wytráćił / żeby nas nic nie zoſtáło we wƺyſtkich gránicách Izráelſkich :
Wydajćie nam śiedmi mężów z Synów jego / á obieśiemy je PAnu w Gábáá / Saulá <i>niekiedy</i> wybránego PAńſkiego : Tedy rzekł ; Wydam.
Lecż zfolgował Król Mefiboſetowi Synowi Jonátáná / Syná Saulowego / dla przyśięgi PAńſkiey / która <i>byłá</i> miedzy nimi : miedzy Dawidem / y miedzy Jonátánem Synem Saulowym.
Ale wźiął Król dwu Synów Reſfy / Córki Aje / które porodźiłá Saulowi / Armoniego / y Mefiboſetá / y piąći Synów <i>śioſtry</i> Michole / Córki Saulowey / które porodźiłá Adrielowi / Synowi Bárſylájá Meholátyckiego.
Y wydał je w ręce Gábáonitów : á obieśili je ná górze przed PAnem : y umárli oni śiedm poſpołu / á pobići ſą w pierwƺe dni żniwá / ná pocżątku żniwá jęcżmiennego.
A wźiąwƺy Resfá / córká Aje / wór / rozpoſtárłá go ná ſkále / ná pocżątku żniwá / áżby ná nie kropił deƺcż z niebá : y nie dopuƺcżáłá ptáctwu powietrznemu pádáć ná nie we dnie / áni zwierzowi polnemu w nocy.
Tedy oznájmiono Dawidowi / co ucżyniłá Resfá córká Aje / założnicá Saulowá.
Przetoż ƺedƺy Dawid wźiął kośći Saulowe / y kośći Jonátáná Syná jego / od ſtárƺych Jábes Gáláádſkiego / którzy je byli ukrádli z ulice Betſáńſkiey / kędy je byli záwieśili Filiſtynowie onegoż dniá / gdy poráźili Filiſtynowie Saulá w Gelboe.
A ták wźiął z támtąd kośći Saulowe / y kośći Jonátáná Syná jego : Zebráno też kośći obieƺonych.
Y pogrzebli kośći Saulowe / y Jonátáná Syná jego / w źiemi Benjáminá w Selá / w grobie Cyſá Ojcá jego : á ucżynili wƺyſtko co był rozkazał Król : á ták potym ubłagány był Bóg źiemi.
Y byłá záśię wojná miedzy Filiſtynámi y Izráelem : y ćiągnął Dawid y ſłudzy jego z nim / á walcżyli przećiwko Filiſtynom / ták że uſtał Dawid.
Tedy Jeſbibenob / który był z Synów jednego Obrzymá / ( á grot drzewcá jego ważył trzy ſtá ſyklów miedźi / á miał przypáſány <i>miecż</i> nowy ) umyślił był zábić Dawidá.
Ale go rátował Abiſáj Syn Sárwijey / á rániwƺy Filiſtyná / zábił go : Przetoż przyśięgli mężowie Dawidowi / mówiąc mu ; Nie pójdźieƺ więcey z námi ná wojnę / ábyś nie zgáśił pochodnie Izráelſkiey.
Y ſtáło śię potym / że byłá znowu wojná w Gob z Filiſtynámi / y zábił Sobocháj Huſátycki Sáfá / który był z Synów tegoż Obrzymá.
Byłá też jeƺcże inna wojná w Gob z Filiſtynámi / kędy zábił Elháná Syn Jááre Oregim Betlehemcżyk / <i>brátá</i> Goliatowego z Get / którego drzewce u włócżnie było jáko nawój tkácki.
Nád to jeƺcże byłá wojná w Get / kędy był mąż <i>wielkiego</i> wzroſtu májąc po ƺeśći pálców u rąk ſwojich / y po ƺeśći pálców u nóg ſwojich / wƺyſtkich dwádźieśćiá y cżtery : á ten też był Synem tegoż Obrzymá.
Ten gdy urągał Izráelowi / zábił go Jonátán / Syn Sámáá / brátá Dawidowego.
Ci cżterej byli Synowie <i>jednego</i> Obrzymá z Get : á ći polegli od ręki Dawidowey / y od ręki ſług jego.
