Y byłá długa Wojná miedzy domem Saulowym / y miedzy domem Dawidowym : wƺákże Dawid poſtępował / y zmacniał śię : ále dom Saulów zchodźił y niƺcżał.
Y národźiło śię Dawidowi w Hebronie Synów : á był pierworodny jego Amnon / z Achinoámey Jezreelitki.
Wtóry po nim był Heliáb / z Abigájley / żony <i>przed tym</i> Nábálowey z Kármelu : á trzeći Abſálom Syn z Mááchy / córki Tolmájá / królá Geſſur.
A cżwarty Adoniaƺ Syn Hágity : á piąty Sefátiaƺ / Syn Abitáli.
A ƺóſty Jetráám / z Egle / żony Dawidowey. Cić śię urodźili Dawidowi w Hebronie.
Y ſtáło śię / gdy byłá wojná / miedzy domem Saulowym / y miedzy domem Dawidowym ; á Abner śię mężnie záſtawiał o dom Saulów /
( A Saul miał założnicę / którey imię <i>było</i> Resfá / córká Aje ) że rzekł <i>Isboſet</i> do Abnerá : Cżemuś wƺedł do założnice Ojcá mojego?
Y rozgniewał śię Abner bárzo / o one ſłowá Isboſetowe / y rzekł : Izalim ja pśia głowá / którym przećiw Judźie dźiś ucżynił miłośierdźie / nád domem Saulá Ojcá twego / y nád bráćią jego / y nád przyjaćiołmi jego : y nie wydałem ćię w rękę Dawidowę ; á ty dźiś ƺukaƺ ná mnie niepráwośći tey niewiáſty?
To niech ucżyni Bóg Abnerowi / y to niech mu przycżyni / jeſliże / jáko przyśiągł PAN Dawidowi / nie pomogę do tego :
Aby przenieśione było króleſtwo od domu Saulowego / á wyſtáwiona ſtolicá Dawidowá nád Izráelem / y nád Judą / od Dán áż do Berſábee.
Y nie mógł nic więcey odpowiedźieć Abnerowi / przeto że śię go bał.
A ták wypráwił Abner poſły do Dawidá od śiebie / mówiąc : Cżyjáż <i>jeſt</i> źiemiá? Y żeby mówili : Ucżyń przymierze twoje ze mną : á oto ręká mojá będźie z tobą : áby obrócon był do ćiebie wƺyſtek Izráel.
Któremu odpowiedźiał : Dobrze : Ucżynię z tobą przymierze : A wƺákże o jedno ćię proƺę / miánowićie : ábyś nie przychodźił przed oblicże moje / áż mi pierwey przywiedźieƺ Michol córkę Saulowę / gdy będźieƺ chćiał przyść / ábyś widźiał twarz moję.
Y wypráwił Dawid poſły do Isboſetá Syná Saulowego / mówiąc : Wydaj mi żonę moję Michol / którąm ſobie poślubił ſtem nieobrzezek Filiſtyńſkich.
Przetoż poſłał Isboſet / y wźiął ją od mężá / od Fáltielá Syná Láiſowego.
Tedy ƺedł z nią mąż jey / á idąc zá nią / płákał jey / áż do Báchurym : y rzekł do niego Abner : Idź / á wróć śię : y wróćił śię.
Ucżynił potym Abner rzecż do ſtárƺych Izráelſkich / mówiąc : Przeƺłych cżáſów ƺukáliśćie Dawidá / áby był Królem nád wámi :
Przetoż teraz ucżyńćie <i>ták</i> / bo PAN rzekł o Dawidźie / mówiąc : Przez rękę Dawidá ſługi mego / wybáwię lud mój Izráelſki z ręki Filiſtyńſkiey / y z ręki wƺyſtkich nieprzyjaćiół jego.
Toż też mówił Abner y do Benjámcżyków. Potym odƺedł Abner / áby mówił z Dawidem w Hebronie / wƺyſtko co dobrego było w ocżách Izráelá / y w ocżách wƺyſtkiego domu Benjáminowego.
Gdy tedy przyƺedł Abner do Dawidá / do Hebronu / á z nim dwádźieśćiá mężów : ſpráwił Dawid ná Abnerá / y ná męże którzy z nim byli / ucżtę.
Y rzekł Abner do Dawidá : Wſtánę / á pójdę / ábym zebrał do Królá Páná mego wƺyſtkiego Izráelá / którzy z tobą ucżynią przymierze : á będźieƺ królował nád wƺyſtkimi / jáko żąda duƺá twojá. A ták odpráwił Dawid Abnerá / który odƺedł w pokoju.
A oto ſłudzy Dawidowi / y Joáb wracáli śię z wojny / korzyśći wielkie z ſobą prowádząc : ále Abnerá już nie było u Dawidá w Hebronie : bo go był odpráwił / y odƺedł był w pokoju.
Joáb tedy y wƺyſtko wojſko które z nim było / przyƺli tám : y dano znáć Joábowi / mówiąc : Był tu Abner Syn Nerów u Królá / ále go odpráwił / y odƺedł w pokoju.
Przetoż wƺedƺy Joáb do Królá / rzekł : Cóżeś ucżynił? oto przyƺedł był Abner do ćiebie : Przecżżeś go puśćił / áby záś odƺedł?
Znaƺ Abnerá Syná Nerowego? gdyż przyƺedł áby ćię zdrádźił / y żeby wiedźiał wyśćie twoje / y weśćie twoje / áby śię wywiedźiał o wƺyſtkim co ty cżyniƺ.
Tedy odƺedƺy Joáb od Dawidá / wypráwił poſły zá Abnerem : którzy go wróćili od ſtudni Syrá / o cżym Dawid nie wiedźiał.
A gdy śię wróćił Abner do Hebronu / odwiódł go Joáb w pośrzód bramy / áby z nim po ćichu mówił / y przebił go tám pod piąte żebro / że umárł / dla krwie Aſáelá brátá ſwego.
Co gdy potym uſłyƺał Dawid / rzekł : Nie jeſtem winien / áni króleſtwo moje przed PAnem áż ná wieki krwie Abnerá Syná Nerowego :
Niechaj przydźie ná głowę Joábowę / y ná wƺyſtek dom Ojcá jego : y niech nie uſtáje z domu Joábowego płynienie <i>naśienia</i> ćierpiący / y trędowáty / y o kiju chodzący / y od miecżá upadájący / y nie májący chlebá.
A ták Joáb / y Abiſáj brát jego / zábili Abnerá / przeto yż <i>on też</i> był zábił Aſáelá brátá ich w bitwie u Gábáonu.
Rzekł potym Dawid do Joábá / y do wƺyſtkiego ludu / który był z nim : Porozdźierajćie odźienia wáƺe / á opáƺćie śię w wory / y płácżćie nád Abnerem : A Król Dawid ƺedł zá márámi.
A gdy pogrzebli Abnerá w Hebronie / podnióſł Król głos ſwój / y płákał nád grobem Abnerowym : płákał też wƺyſtek lud.
A ták lámentując Król nád Abnerem / rzekł : Izali <i>ták</i> miał umrzeć Abner / jáko umierá nikcżemnik?
Ręce twoje nie były związáne : á nogi twoje nie były pętámi obćiążone! Poległeś jáko ten który pada przed Synámi niezbożnymi! Tedy tym więcey wƺyſtek lud płákał nád nim.
Potym przyƺedł wƺyſtek lud <i>prośić</i> Dawidá áby jadł chleb / gdy jeƺcże był <i>jáſny</i> dźień : ále przyśiągł Dawid / mówiąc : To mi niechaj ucżyni Bóg / y to niech przycżyni / jeſli przed zaśćiem ſłońcá ſkoƺtuję chlebá / álbo cżego innego.
Co gdy wƺyſtek lud obacżył / podobáło śię im to : A wƺyſtko cokolwiek cżynił Król / podobáło śię w ocżách wƺyſtkiego ludu.
Y poznał wƺyſtek lud / y wƺyſtek Izráel dniá onego / że nie z napráwy Królewſkiey zábity był Abner / Syn Nerów.
Y rzekł Król do ſług ſwojich : Aza nie wiećie że Hetmán / á <i>bárzo</i> wielki / poległ dźiś w Izráelu?
A jam dźiś <i>jáko</i> nowy / y dopiero pomázány Król : Ći záśię mężowie / Synowie Sárwijey / ſrożƺy ſą niżli ja : niechże odda PAN cżyniącemu złe według złośći jego.
