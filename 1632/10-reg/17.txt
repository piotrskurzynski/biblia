Nád to rzekł Achitofel do Abſálomá : Niech proƺę wybiorę dwánaśćie tyśięcy mężów / á wſtawƺy będę gonił Dawidá tey nocy :
Y przypádnę nań / pokąd jeſt ſprácowány y zemdlonych rąk / á ztrwożę go / y ućiecże wƺyſtek lud który <i>jeſt</i> z nim / á zábiję Królá ſámego.
A ták przywrócę wƺyſtek lud do ćiebie : <i>bo</i> jákoby śię wƺyſcy ku tobie náwróćili <i>gdy zábiję</i> tego mężá którego ty ƺukaƺ / á wƺyſtek śię lud uſpokoji.
Y ſpodobáło śię to Abſálomowi / y wƺyſtkim ſtárƺym Izráelſkim.
Jednák rzekł Abſálom : Záwołaj rychło y Chuſájá Aráchity / ábyſmy uſłyƺeli / co on też powie.
A gdy przyƺedł Chuſáj do Abſálomá / rzekł Abſálom do niego / mówiąc : Ták powiedźiał Achitofel : mamyli ucżynić według rády jego / cżyli nie / <i>y</i> ty powiedz?
Tedy odpowiedźiał Chuſáj Abſálomowi : nie dobra jeſt rádá którą teraz dał Achitofel.
Nád to rzekł Chuſáj : Świádomeś Ojcá twego / y mężów jego / iż ſą mężni / y ſercá zájuƺonego / jáko niedźwiedźicá ośieroćiáła w polu : Do tego Oćiec twój jeſt mąż walecżny / y nie będźie nocował z ludem.
A podobno y teraz śię kryje w jákiey jáſkini / álbo ná którymkolwiek miejſcu. Y ſtáłoby śię / jeſliżeby kto z twojich poległ ná tym pocżątku / żeby káżdy ktoby o tym uſłyƺał / rzekł : Stáłá śię porażká w ludu / który <i>ƺedł</i> zá Abſálomem.
Tedy y namężniejƺy / którego ſerce jáko ſerce lwie / bárzo oſłábieje / bo wie wƺyſtek Izráel / że mężnym jeſt Oćiec twój / y mężni <i>wƺyſcy</i> którzy ſą z nim.
Aleć rádzę / áby śię do ćiebie cále zebrał wƺyſtek Izráel / od Dán / áż do Beerſebá / jáko piaſek / który jeſt przy morzu w mnóſtwie : á ty żebyś oſobą ſwą ƺedł ná wojnę.
A ták poćiągniemy przećiwko niemu / ná którymkolwiek miejſcu náleźiony będźie : y przypádniemy nań / jáko páda roſá ná źiemię : y nie zoſtánie z niego / to jeſt / z tych wƺyſtkich mężów którzy ſą z nim / áni jeden.
A jeſliżby <i>do którego</i> miáſtá uƺedł / tedy znieśie wƺyſtek Izráel do onego miáſtá powrozy / á poćiągniemy je áż do potoká / ták yż tám nie będźie náleźion / áni kámyk.
Tedy rzekł Abſálom / y wƺyſcy mężowie Izráelſcy ; Lepƺa jeſt rádá Chuſájego Aráchity / niż rádá Achitofelowá : Abowiem PAN był poſtánowił / áby rozerwána byłá rádá Achitofelowá / która <i>byłá</i> dobra ; á ták áby przywiódł PAN złe ná Abſálomá.
Y oznájmił Chuſáj Sádokowi / y Abiátárowi Kápłanom ; Ták á ták rádźił Achitofel Abſálomowi / y ſtárƺym Izráelſkim / álem ja ták á ták rádźił.
Teraz tedy poślićie co rychley / á oznájmićie Dawidowi / mówiąc : Nie zoſtawaj tey nocy w równinách puƺcży / ále bez odwłoki przejdźi : by ſnadź nie był pożárty Król / y wƺyſtek lud który jeſt z nim.
A Jonátán y Achimáás ſtali u ſtudnie Rogel : y poƺłá dźiewká / á oznájmiłá im / áby poƺli / y donieśli to Królowi Dawidowi : bo śię nie śmieli ukazáć / áni wniść do miáſtá.
Wƺákże obacżył je <i>niektóry</i> ſługá / y powiedźiał Abſálomowi : Przetoż poƺedƺy obádwá ſpieƺno / weƺli w dom <i>niektórego</i> mężá w Báhurym / który miał ſtudnią ná dworze ſwym / y ſpuśćili śię do niey.
A wźiąwƺy niewiáſtá płáchtę / rozćiągnęłá ją ná wierzchu ſtudnie / y náſypáłá ná niey krup ; á ták śię tego nie dowiedźiano.
Bo gdy przyƺli ſłudzy Abſálomowi do oney niewiáſty w dom / rzekli : Gdźie <i>jeſt</i> Achimáás y Jonátán? Odpowiedźiáłá im niewiáſtá : Przeƺli przez rzekę : á poƺukawƺy ich / y nie ználazƺy / wróćili śię do Jeruzalem.
A gdy oni odeƺli / tedy owi wyſtąpiwƺy z ſtudnie / poƺli / y oznájmili Królowi Dawidowi / mówiąc do niego : Wſtańćie / przeprawćie śię co rychliey przez wodę : ábowiem ták rádźił przećiwko wam Achitofel.
Przetoż wſtawƺy Dawid / y wƺyſtek lud który był z nim / przepráwili śię przez Jordan / pierwey niż śię rozedniáło : á nie zoſtał y jeden któryby śię nie przepráwił przez Jordan.
Tedy Achitofel widząc iż śię nie ſtáło podług rády jego / ośiodłał oſłá ; á wſtawƺy jáchał do domu ſwego / do miáſtá ſwego / á rozpráwiwƺy dom ſwój / obieśił śię y umárł / á pogrzebion jeſt w grobie Ojcá ſwego.
A Dawid już był przyƺedł do Máhánáim / gdy śię Abſálom przepráwił przez Jordan / on y wƺyſcy mężowie Izráelſcy z nim.
Y przełożył Abſálom Amázę / miáſto Joábá nád wojſkiem / (á ten Amázá był Synem mężá / którego imię było Itrá / Izráelcżyk / który był wƺedł do Abigáili córki Náháſowey / śioſtry Sárwijey / mátki Joábowey ).
Y położył śię obozem Izráel z Abſálomem w źiemi Gáláád.
Y ſtáło śię gdy przyƺedł Dawid do Máhánáim / że Soby / Syn Náháſów z Rábby / Synów Ammonowych / y Máchyr Syn Ammielów / z Lodebáru / y Bárſyláj Gáláádcżyk z Rogelim :
Pośćiel / y miednice y nacżynia zduńſkie / y pƺenicę y jęcżmiony / y mąki / y krupy / y boby / y ſocżewice / y práżmá :
Y miodu / y máſłá / y owiec / y ſerów krowich / przynieśli Dawidowi / y ludowi / który był z nim / áby jedli : Bo mówili ; Lud ten głodny jeſt / y zprácowány / y prágnieniem zmorzony ná puƺcży.
