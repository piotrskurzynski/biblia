Y ſtáło śię potym / że pytał Dawid PAná mówiąc : Mamże iść do któregokolwiek miáſtá Judſkiego? Któremu PAN odpowiedźiał ; Idź. Y rzekł Dawid : dokądże pójdę? Y odpowiedźiał : do Hebronu.
Tedy tám jáchał Dawid / tákże y dwie żonie jego / Achinoám Jezreelitká / y Abigáil żoná <i>przed tym</i> Nábálowá z Kármelu.
Tákże męże ſwe którzy z nim byli / zábrał Dawid / káżdego z domem jego : y mieƺkáli w miáſtách Hebrońſkich.
Przyƺli potym mężowie Judá / y pomázáli tám Dawidá zá Królá nád domem Judá : tedy opowiedźiáno Dawidowi / mówiąc ; Mężowie z Jábes Gáláád / ći pogrzebli Saulá.
Tedy wypráwił Dawid poſły do mężów z Jábes Gáláád / y rzekł do nich : Błogoſłáwieniśćie wy od PAná / którzyśćie ucżynili to miłośierdźie nád Pánem wáƺym Saulem / żeśćie go pogrzebli.
Przetoż teraz niech ucżyni PAN z wámi miłośierdźie / y prawdę / á ja też oddam wam to dobrodźiejſtwo / żeśćie ucżynili tę rzecż.
Teraz tedy niech śię zmacniáją ręce wáƺe / á bądźćie mężnymi / bo choć umárł Saul Pán wáƺ / wƺákże mnie pomázał dom Judá / zá Królá nád ſobą.
A Abner Syn Nerów / Hetman nád Wojſkiem Saulowym / wźiął Izbozetá Syná Saulowego / y przyprowádźił go do Máchánáim.
Y ucżynił go Królem nád Gáláádem / y nád Aſſury / y nád Jezreelem / y nád Efrájimem / y nád Benjáminem / y nád wƺyſtkim Izráelem.
Cżterdźieśći lat miał Izbozet Syn Saulów / gdy pocżął królowáć nád Izráelem / á dwie lećie królował / ( tylko dom Judá ſtał przy Dawidźie.
Y byłá licżbá dni / których był Dawid Królem w Hebronie / nád domem Judſkim / śiedm lat y ƺeść mieśięcy. )
Potym wyƺedł Abner Syn Nerów / y ſłudzy Izbozetá Syná Saulowego / z Máchánáim do Gábáonu.
Joáb tákże Syn Sárwijey / z ſługámi Dawidowymi wyƺli / y potkáli śię z ſobą práwie u ſtáwu Gábáońſkiego : y zoſtáli jedni ná jedney ſtronie ſtáwu / á drudzy ná drugiey ſtronie ſtáwu.
Tedy rzekł Abner do Joábá : Niech teraz wſtáną młodźieńcy / á pojigráją przed námi : y rzekł Joáb ; Niech wſtáną.
Wſtáli tedy : y wyƺło w licżbie dwánaśćie z Benjámcżyków / ze ſtrony Izbozetá / Syná Saulowego : á dwánaśćie z ſług Dawidowych /
Którzy uchwyćiwƺy śię / káżdy zá głowę przećiwniká ſwego / <i>utopił</i> miecż ſwój w boku jeden drugiego / y polegli poſpołu : przetoż názwano miejſce ono Helkáthaſſurim / które jeſt w Gábáonie.
Y byłá bitwá bárzo ſroga dniá onego / á poráżon jeſt Abner / y mężowie Izráelſcy / od ſług Dawidowych.
Byli też tám trzej Synowie Serwijey : Joáb / Abiſáj / y Aſáel : ále Aſáel <i>był</i> prętkich nóg / jáko dźika kozá.
Y gonił Aſáel Abnerá / á nie uſtąpił idąc / áni ná práwo / áni ná lewo / śćigájąc Abnerá.
Ale obejrzawƺy śię Abner ná zad / rzekł : Tyżeś jeſt Aſáel? á on mu odpowiedźiał ; Ja.
Tedy mu rzekł Abner : Uchyl śię ná práwą ſtronę twoję álbo ná lewą <i>ſtronę</i> twoję / á pojimaj ſobie jednego z młodźieńców / y weźmi ſobie łupy z niego : ále Aſáel niechćiał od niego uſtąpić.
Tedy powtóre rzekł Abner do Aſáelá : Idź precż ode mnie ; bym ćię ſnadź nie przebił ku źiemi : Bo jákożbym śmiał podnieść twarz moję ná Joábá brátá twego?
A gdy niechćiał uſtąpić / uderzył go Abner końcem włócżnie / pod piąte <i>żebro</i> ták iż wyƺłá włócżnia ná drugą ſtronę : támże padł / y umárł ná onymże miejſcu : á wƺyſcy którzy przychodźili do onego miejſcá / gdźie poległ Aſáel y umárł / záſtánawiáli śię.
Wƺákże gonili / Joáb y Abiſáj Abnerá : y záchodźiło ſłońce / gdy przyƺli do págórku Ammá / który jeſt przećiw Giá / ná drodze puſtynie Gábáońſkiey.
Tedy śię zebráli Synowie Benjáminowi do Abnerá / ſkupiwƺy śię w jeden huf / y ſtánęli ná wierzchu jednego págórká.
Y záwołał Abner ná Joábá y rzekł : Yzali śię ná wieki będźie ſrożył <i>ten</i> miecż? ázaż nie wieƺ że ná oſtátku bywa gorzkość? y dokądże nie rzecżeƺ ludowi / áby śię wróćił od pogoni bráćij ſwojey?
Tedy rzekł Joáb : Żywie Bóg / byś ty był nie wyzywał / záraz by śię był z poránku lud wróćił / káżdy od pogoni bráćij ſwojey.
A ták zátrąbił Joáb w trąbę / y záſtánowił śię wƺyſtek lud / á nie gonili dáley Izráelá / áni śię więcey potykáli.
Ale Abner y mężowie jego / ƺli polem cáłą onę noc : á przepráwiwƺy śię przez Jordan / przeƺli przez wƺyſtko Betoron / áż przyƺli do Máhánáim.
A Joáb wróćiwƺy śię z pogoni zá Abnerem / zebrał wƺyſtek lud : y nie doſtawáło mu z ſług Dawidowych dźiewiętnaſtu mężów / y Aſáelá.
Ale ſłudzy Dawidowi pobili z Benjámcżyków / y z mężów Abnerowych / trzy ſtá y ƺeśćdźieśiąt mężów / którzy tám pomárli.
A wźiąwƺy Aſáelá / pogrzebli go w grobie Ojcá jego / który był w Betlehem : potym ƺli cáłą noc Joáb y mężowie jego ; á ná świtániu <i>przyƺli</i> do Hebronu.
