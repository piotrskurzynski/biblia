Y ſtáło śię potem / że poráźił Dawid Filiſtyny / y poniżył je ; á wźiął Dawid Meteg Ammá z rąk Filiſtyńſkich.
Poráźił też y Moábity / które pomierzył ƺnurem / zrównáwƺy je z źiemią / y wymierzył ich dwá ƺnury ná zábićie / á cáły ƺnur ná záchowánie przy żywoćie ; y byli Moábitowie ſługámi Dawidowymi / przynoƺąc mu podátki.
Poráźił też Dawid Hádádezerá / ſyná Rochobowego / królá Soby / gdy wyjechał / áby rozprzeſtrzenił gránice ſwe nád rzeką Eufrátes.
Y pojmał z nich Dawid tyśiąc y śiedm ſet jezdnych / á dwádźieśćiá tyśięcy ludu pieƺego. Y poderznął Dawid żyły wƺyſtkim woźnikom / záchowáwƺy koni do ſtá wozów.
Przyćiągnął tedy Syryjcżyk z Dámáƺku ná pomoc Hádádezerowi / królowi Soby / y poráźił Dawid Syryjcżyków dwádźieśćiá y dwá tyśiące mężów.
Y oſádźił Dawid żołnierzem Syryją Dámáſką. A ták Syryjcżycy byli ſługámi Dawidowymi / przynoƺąc mu podátki ; y bronił Pán Dawidá wƺędźie / gdźiekolwiek śię obróćił.
Pobrał też Dawid tárcże złote / które mieli ſłudzy Hádádezerowi / y wnióſł je do Jeruzálemu.
Przytem z Betáchu y z Berotu / miaſt Hádádezerowych / przynióſł król Dawid bárzo wiele miedźi.
To uſłyƺáwƺy Tohy / król Emát / iż poráźił Dawid wƺyſtko wojſko Hádádezerowe /
Poſłał Tohy Jorámá / ſyná ſwego / do królá Dawidá / áby go pozdrowił w pokoju / y winƺował mu / przeto że zwálcżył Hádádezerá / y poráźił go / ( ábowiem wálcżył z Tohym Hádádezer / ) y przynióſł z ſobą nacżynia ſrebrne / y nacżynia złote / y nacżynia miedźiáne.
Które też rzecży poświęćił król Dawid Pánu z innem ſrebrem y złotem / które był poświęćił / pobráwƺy od wƺyſtkich narodów / które ſobie podbił.
Jáko od Syryjcżyków / y od Moábcżyków / y od ſynów Ammonowych / y od Filiſtynów / y od Amálekitów / y z łupów Hádádezerá / ſyná Rochobowego / królá Soby.
A ták ucżynił ſobie Dawid imię / gdy śię wróćił / poráźiwƺy Syryjcżyki w dolinie ſolney / gdźie pobił ośmnaśćie tyśięcy ludu.
Poſtánowił też ſtráż w Edom / wƺyſtkę źiemię Edomſką oſádźiwƺy żołnierzámi ; y byli wƺyſcy Edomcżycy ſługámi Dawidowymi / á bronił Pán Dawidá wƺędźie / gdźie śię obróćił.
Y królował Dawid nád wƺyſtkim Izráelem / y cżynił ſąd y ſpráwiedliwość wƺyſtkiemu ludowi ſwemu.
A Joáb / ſyn Sárwii / był nád wojſkiem / á Jozáfát ſyn Ahiludá / kánclerzem.
A Sádok / ſyn Achitobá / y Achimelech / ſyn Abijátárá / byli kápłánámi / á Sárájá piſárzem.
Bánájás też / ſyn Jojádá / nád Cheretcżykámi y Feletcżykámi / á ſynowie Dawidowi byli kśiążętámi.
