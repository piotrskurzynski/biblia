A uſłyƺawƺy <i>Isboſet</i> Syn Saulów / że poległ Abner w Hebronie / zemdláły ręce jego / y wƺyſtek Izráel był przeſtráƺony.
Miał też Syn Saulów dwu mężów Hetmánów nád hufcámi : imię jednego Bááná / á imię drugiego Recháb : Synowie Remmoná Berotcżyká / z Synów Benjáminowych : Bo też Berod policżon był w Benjáminie.
Ućiekli tedy Berotcżykowie do Getájm / y byli tám przychodniámi / áż do onego dniá.
A Jonátán Syn Saulów / miał jednego Syná chromego ná nogi : <i>bo</i> gdy miał pięć lat / á przyƺłá wieść <i>o śmierći</i> Saulowey / y Jonátánowey z Jezreel / wźiąwƺy go mámká jego / ućiekáłá : á gdy prętko ućiekáłá / upadł / y ochramiał : á imię jego Mefiboſet.
Poƺli tedy Synowie Remmoná Berotcżyká / Recháb / y Bááná ; y weƺli gdy był nagorętƺy dźień / do domu Isboſetá ; Który ſpał ná łóƺku w południe.
Ci tedy weƺli w dom jego / jákoby kupowáć zbożá ; támże go przebili pod piąte żebro / Recháb y Bááná brát jego / y ućiekli.
Bo gdy byli weƺli w dom / á on ſpał ná łóƺku ſwym w pokoju / kędy legał / tedy go przebili / y zábili go / á ućiąwƺy głowę jego / wźięli ją / y poƺli drogą puƺcży przez cáłą onę noc.
Y przynieśli głowę Isboſetowę do Dawidá do Hebronu / y rzekli do Królá : Oto głowá Isboſetá Syná Saulowego / nieprzyjaćielá twego ; który ƺukał duƺe twojey : á dał PAN Królowi Pánu memu pomſtę dźiśia nád Saulem / y nád naśieniem jego.
Tedy odpowiedźiał Dawid Rechábowi y Báánie brátu jego / Synom Remmoná Berotcżyká / y rzekł do nich : Żywie PAN który wybáwił duƺę moję ze wƺego ućiſku :
Jeſlim onego który mi oznájmił / mówiąc : Oto umárł Saul : ( choć mu śię zdáło że weſołą nowinę przynióſł / ) pojimawƺy zábił w Sycelegu ; który <i>rozumiał</i> żem mu miał dáć zapłátę / zá poſelſtwo <i>jego</i> ;
Jáko dáleko więcey ludźie niepobożne / gdyż zábili mężá ſpráwiedliwego / w domu jego / ná łożu jego? A teraz / izali nie mam ƺukáć krwie jego z ręki wáƺey / y wygłádźić was z źiemie?
A ták rozkazał Dawid ſługom / y zábili je : á obćiąwƺy ręce ich / y nogi ich / záwieśili je nád ſtáwem w Hebronie : ále głowę Isboſetowę wźiąwƺy / pogrzebli w grobie Abnerowym w Hebronie.
