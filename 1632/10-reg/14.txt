A porozumiáwƺy Joáb / ſyn Sárwii / że śię ſerce królewſkie obróćiło ku Abſálomowi /
Poſłał Joáb do Tekuj / y wźiął z támtąd niewiáſtę mądrą / y rzekł do niej : Proƺę zmyśl / jákobyś w żáłobie byłá / á oblecż śię proƺę w ƺáty żáłobne / y nie námázuj śię olejkiem / ále bądź jáko niewiáſtá / która przez wiele dni w żáłobie chodźi po umárłym.
Y wnijdźieƺ do królá / á będźieƺ mówiłá do niego w ten ſpoſób ; y náucżył jey Joáb / jáko miáłá mówić.
Przetoż mówiłá oná niewiáſtá Tekuitſká do królá / upádƺy oblicżem ſwem ná źiemię / á pokłoniwƺy śię rzekłá : Rátuj królu!
Y rzekł jey król : Cóż ći? A oná odpowiedźiáłá : Záiſtem ja niewiáſtá wdowá ; bo mi mąż mój umárł :
A ſłużebnicá twojá miáłá dwu ſynów / którzy śię powádźili z ſobą ná polu ; á gdy nie był / ktoby je rozwádźił / ránił jeden drugiego / y zábił go.
A oto / powſtáwƺy wƺyſtká rodźiná przećiw ſłużebnicy twojey / mówią : Wydáj tego / który zábił brátá ſwego / że go zábijemy zá duƺę brátá jego / którego zámordował / owƺem zgłádźimy y dźiedźicá ; á ták zágáƺą iſkierkę moję / która pozoſtáłá / áby nie zoſtáwili mężowi memu imienia y ƺcżątku ná źiemi.
Tedy rzekł król do niewiáſty : Idź do domu twego / á ja ſkáżę zá tobą.
Y odpowiedźiáłá niewiáſtá Tekuitſká królowi : Królu / pánie mój / niech będźie ná mnie tá niepráwość / y ná domu ojcá mego ; ále król y ſtolicá jego niech będźie niewinná.
Y rzekł król : Będźieli kto mówił przećiwko tobie / przywiedź go do mnie / á potem nie tknie śię ćiebie więcey.
Tedy oná rzekłá : Wſpomnij proƺę / królu / ná Páná Bogá twego / áby śię nie mnożyli mśćićiele krwi ná zgubę / á nie zgłádźili ſyná mego. Y odpowiedźiał : Jáko żywy Pán / że nie ſpádnie y nájmniejƺy włos ſyná twego ná źiemię.
Zátem rzekłá niewiáſtá : Niech przemówi proƺę ſłużebnicá twojá do królá / páná mego / ſłowo. A on rzekł : Mów.
Rzekłá tedy niewiáſtá : Y cżemużeś umyślił podobną rzecż przećiw ludowi Bożemu? ábowiem król mówi to ſłowo / jákoby był winny / ponieważ nie chceƺ przywróćić / królu / wygnáńcá ſwego.
Wƺyſcy umierámy / á jeſteśmy jáko wody rozláne po źiemi / które nie mogą być zebráne ; lecż jemu Bóg nie odjął żywotá / ále pewnie umyślił / áby nie wygániał od śiebie wygnáńcá.
A teraz / żem przyƺłá mówić do królá / páná mego / te ſłowá / przycżyną jeſt / że mię przeſtráƺył lud ; przetoż rzekłá ſłużebnicá twojá : Będę teraz mówiłá do królá / ſnadź co ucżyni król ná prośbę ſłużebnicy ſwojey.
Abowiem uſłyƺy to król / y wybáwi ſłużebnicę ſwoję z rąk mężá / który wygłádźić chce mnie / y ſyná mego ſpołem / z dźiedźictwá Bożego.
Rzekłá też ſłużebnicá twojá : Wżdyć mi będźie ſłowo królá / páná mego / ku poćieƺe ; ábowiem jáko Anjoł Boży / ták jeſt król / pán mój / ſłuchájąc dobrego y złego / á Pán Bóg twój niech będźie z tobą.
A odpowiádájąc król rzekł do niewiáſty : Proƺę nie táj przedemną tego / o co śię ſpytám. Y rzekłá niewiáſtá : Mów proƺę / królu / pánie mój.
Tedy rzekł król : Izali ty tego wƺyſtkiego nie cżyniƺ z nápráwy Joábowej? Y odpowiedźiáłá niewiáſtá / y rzekłá : Jáko żyje duƺá twojá / królu / pánie mój / że nie można uchylić śię áni ná práwo / áni ná lewo od wƺyſtkiego / co mówił król / pán mój ; ábowiem ſługá twój Joáb / on mi to rozkázał / y on náucżył ſłużebnicy twojey tych wƺyſtkich ſłów.
Żem odmieniłá ſpoſób tey mowy / ſpráwił to ſługá twój Joáb ; lecż pán mój mądry jeſt / jáko jeſt mądry Anjoł Boży / wiedząc wƺyſtko / co śię dźieje ná źiemi.
Przetoż rzekł król do Joábá : Otom teraz to ucżynił. Idźże á przywróć dźiećię me Abſálomá.
Y upádł Joáb oblicżem ſwojim ná źiemię / á pokłoniwƺy śię błogoſłáwił królowi / y rzekł Joáb : Dźiś poznał ſługá twój / żem ználázł łáſkę w ocżách twojich / królu / pánie mój / ponieważ ucżynił król zádoſyć prośbie ſługi ſwego.
Wſtał tedy Joáb / á ƺedł do Gieſſur / y przywiódł Abſálomá do Jeruzálemu.
Y rzekł król : Niech śię wróći do domu ſwego / ále oblicża mego niech nie widźi. A ták wróćił śię Abſálom do domu ſwego / ále oblicża królewſkiego nie widźiał.
A nie było mężá ták kráſnego / jáko Abſálom we wƺyſtkim Izráelu / coby miał ták wielką chwałę ; od ſtopy nogi jego áż do wierzchu głowy jego nie było ná nim zmázy.
A gdy ſtrzygł głowę ſwoję / ( á zwykł ją ná káżdy rok ſtrzydz ; bo mu ćiążyłá / przetoż ją ſtrzygł / ) wáżyły włoſy głowy jego dwieśćie ſyklów wági królewſkiej.
Y urodźili śię Abſálomowi trzej ſynowie / y córká jedná / którey imię było Támár / która niewiáſtá byłá piękná ná wejrzeniu.
Y mieƺkał Abſálom w Jeruzálemie dwá látá / á twárzy królewſkiej nie widźiał.
Przetoż poſłał Abſálom do Joábá / chcąc go poſłáć do królá / ále on nie chćiał przyść do niego ; poſłał potem powtóre / y nie chćiał przyść.
Tedy rzekł do ſług ſwojich : Przepátrzćie rolą Joábowę podle roli mojey / gdźie ma jęcżmień ; idźćież / á ſpálćie go ogniem. Y zápálili ſłudzy Abſálomowi rolą onę ogniem.
Zátem wſtáwƺy Joáb / przyƺedł do Abſálomá w dom / y rzekł do niego : Cżemuż ſłudzy twoi ſpálili rolą moję ogniem?
Y odpowiedźiał Abſálom Joábowi : Otom poſłał do ćiebie / mówiąc : Przyjdź ſám / á poślę ćię do królá / ábyś mówił : Ná cóżem przyƺedł z Gieſſur? Lepiej mi było tám jeƺcże zoſtáć ; przetoż teraz niech oglądám oblicże królewſkie ; wƺák jeſli jeſt przy mnie niepráwość / niech mię rozkáże zábić.
Tedy przyƺedł Joáb do królá / y oznájmił mu. Y przyzwał Abſálomá / który przyƺedł do królá / y ukłonił śię twárzą ſwą ku źiemi przed królem ; y pocáłował król Abſálomá.
