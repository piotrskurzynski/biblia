A porozumiawƺy Joáb Syn Sárwijey / że śię ſerce królewſkie <i>obróćiło</i> ku Abſálomowi :
Poſłał Joáb do Tekuj / y wźiął z támtąd niewiáſtę mądrą / y rzekł do niey ; Proƺę zmyśl jákobyś w żałobie byłá : á oblecż śię proƺę / w ƺáty żałobne / y nie námázuj śię olejkiem / ále bądź jáko niewiáſtá / która przez wiele dni w żałobie chodźi po umárłym.
Y wnidźieƺ do Królá / á będźieƺ mówiłá do niego w ten ſpoſób : y náucżył jey Joáb jáko miáłá mówić.
Przetoż mówiłá oná niewiáſtá Tekuitſka do Królá / upadƺy oblicżem ſwym ná źiemię / á pokłoniwƺy śię rzekłá ; Rátuj Królu!
Y rzekł jey Król ; Cóż ći? A oná odpowiedźiáłá ; Záiſtem ja niewiáſtá wdowá / bo mi mąż mój umárł :
A ſłużebnicá twojá miáłá dwu Synów / którzy śię powádźili z ſobą ná polu ; á gdy nie był ktoby je rozwádźił / ránił jeden drugiego / y zábił go.
A oto powſtawƺy wƺyſtká rodźiná / przećiw ſłużebnicy twojey / mówią ; Wydaj tego / który zábił brátá ſwego / że go zábijemy zá duƺę brátá jego / którego zámordował / owƺem zgłádźimy y dźiedźicá : á ták zágáƺą iſkierkę moję / która pozoſtáłá / áby niezoſtáwili mężowi memu imienia y ƺcżątku ná źiemi.
Tedy rzekł Król do niewiáſty ; Idź do domu twego / á ja ſkażę zá tobą.
Y odpowiedźiáłá niewiáſtá Tekuitſka Królowi : Królu Pánie mój ; niech będźie ná mnie tá niepráwość / y ná domu Ojcá mego : ále Król y ſtolicá jego niech będźie niewinna.
Y rzekł Król : Będźieli kto mówił przećiwko tobie / przywiedź go do mnie / á potym nie tknie śię ćiebie więcey.
Tedy oná rzekłá : Wspomni / proƺę / Królu ná PAná Bogá twego / áby śię nie mnożyli mśćićiele krwie ná zgubę / á nie zgłádźili Syná mego. Y odpowiedźiał : Żywie PAN / że nie ſpádnie <i>y namniejƺy</i> włos Syná twego ná źiemię.
Zátym rzekłá niewiáſtá : Niech przemówi proƺę ſłużebnicá twojá / do Królá Páná mego ſłowo : A on rzekł ; Mów.
Rzekłá tedy niewiáſtá : Y cżemużeś umyślił podobną rzecż przećiw ludowi Bożemu? ábowiem Król mówi to ſłowo jákoby był winny : ponieważ nie chceƺ przywróćić Królu wygnáńcá ſwego.
Wƺyſcy umieramy / á <i>jeſteſmy</i> jáko wody rozlane po źiemi / które nie mogą być zebráne : Lecż <i>jemu</i> Bóg nie odjął żywotá / ále pewnie umyślił / áby niewyganiał od śiebie wygnáńcá.
A teraz / żem przyƺłá mówić do Królá Páná mego te ſłowá : <i>przycżyną jeſt,</i> że mię przeſtráƺył lud : Przetoż rzekłá ſłużebnicá twojá : Będę teraz mówiłá do Królá / ſnadź co ucżyni Król ná proźbę ſłużebnice ſwojey :
Abowiem uſłyƺy <i>to</i> Król / y wybáwi ſłużebnicę ſwoję z rąk mężá / który wygłádźić chce mnie / y Syná mego ſpołem / z dźiedźictwá Bożego.
Rzekłá też ſłużebnicá twojá : Wżdyć mi będźie ſłowo Królá Páná mego ku poćieƺe : ( ábowiem jáko Anjoł Boży / ták jeſt Król Pán mój / ſłuchájąc dobrego y złego : ) á PAN Bóg twój niech będźie z tobą.
A odpowiádájąc Król / rzekł do niewiáſty : Proƺę nie taj przede mną tego o co śię ſpytam : Y rzekłá niewiáſtá : mów proƺę / Królu Pánie mój.
Tedy rzekł Król : Izali ty tego wƺyſtkiego niecżyniƺ z napráwy Joábowey? Y odpowiedźiáłá niewiáſtá / y rzekłá : Jáko żywie duƺá twojá / Królu Pánie mój / że nie można uchylić śię / áni ná práwo / áni ná lewo / od wƺyſtkiego co mówił Król / Pán mój : ábowiem ſługá twój Joáb / on mi to rozkazał / y on náucżył ſłużebnice twojey / tych wƺyſtkich ſłów.
Żem odmieniłá ſpoſób tey mowy / ſpráwił to ſługá twój Joáb : lecż Pán mój mądry jeſt / jáko jeſt mądry Anjoł Boży / wiedząc wƺyſtko co śię dźieje ná źiemi.
Przetoż rzekł Król do Joábá : Otom teraz to ucżynił. Idźże á przywróć dźiećię <i>me</i> Abſálomá.
Y upadł Joáb oblicżem ſwojim ná źiemię / á pokłoniwƺy śię błogoſłáwił Królowi / y rzekł Joáb ; Dźiś poznał ſługá twój / żem ználazł łáſkę w ocżách twojich Królu Pánie mój : ponieważ ucżynił Król doſyć proźbie ſługi ſwego.
Wſtał tedy Joáb / á ƺedł do Geſſur / y przywiódł Abſálomá do Jeruzalem.
Y rzekł Król : Niech śię wróći do domu ſwego / ále oblicża mego niech nie widźi. A ták wróćił śię Abſálom do domu ſwego / ále oblicża królewſkiego nie widźiał.
A nie było mężá ták kraſnego / jáko Abſálom we wƺyſtkim Izráelu / coby miał ták wielką chwałę : Od ſtopy nogi jego / áż do wierzchu głowy jego / nie było ná nim zmázy.
A gdy ſtrzygł głowę ſwoję / ( á zwykł ją ná káżdy rok ſtrzydz : Bo mu ćiężáłá / przetoż ją ſtrzygł / ) ważyły włoſy głowy jego / dwieśćie Syklów wagi królewſkiey.
Y urodźili śię Abſálomowi trzej Synowie / y córká jedná / którey imię było Támár : która niewiáſtá byłá piękná ná wejrzeniu.
Y mieƺkał Abſálom w Jeruzalem dwie lećie / á twarzy królewſkiey nie widźiał :
Przetoż poſłał Abſálom do Joábá / chcąc go poſłáć do Królá / ále on nie chćiał przyść do niego : poſłał potym powtóre / y nie chćiał przyść.
Tedy rzekł ſługom ſwojim : Przepátrzćie rolą Joábowę / podle <i>roli</i> mojey / gdźie ma jęcżmień : idźćież á ſpalćie go ogniem : y zápálili ſłudzy Abſálomowi rolą onę ogniem.
Zátym wſtawƺy Joáb / przyƺedł do Abſálomá w dom / y rzekł do niego : Cżemuż ſłudzy twoji ſpalili rolą moję ogniem?
Y odpowiedźiał Abſálom Joábowi : Otom poſłał do ćiebie / mówiąc : Przydź ſám / á poślę ćię do Królá ábyś mówił : Ná cóżem przyƺedł z Geſſur? Lepiey mi było tám jeƺcże <i>zoſtáć,</i> przetoż teraz niech oglądam oblicże królewſkie : wƺák jesli jeſt przy mnie niepráwość / niech mię rozkaże zábić.
Tedy przyƺedł Joáb do Królá / y oznájmił mu : Y przyzwał Abſálomá / który przyƺedł do Królá / y ukłonił śię twarzą ſwą ku źiemi przed Królem : y pocáłował Król Abſálomá.
