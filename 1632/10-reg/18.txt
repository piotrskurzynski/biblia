Tedy oblicżył Dawid lud który <i>miał</i> z ſobą / á poſtánowił nád nimi Hetmány / y Rotmiſtrze.
Y porucżył Dawid ludu trzećią cżęść / pod rękę Joábowę / á trzećią cżęść pod rękę Abiſájá / Syná Sárwijey brátá Joábowego / á trzećią cżęść pod rękę Itájá Gettejcżyká : y rzekł Król do ludu : Wynidę y ja tákże z wámi.
Ale lud rzekł : Nie wynidźieƺ : Bo jeſlibyſmy my tył podáli / oni máło dbáć o nas będą : choćby też nas poległá połowicá / máło dbáć o nas będą : Abowiemeś ty ſam / jáko nas dźieśięć tyśięcy. Przetoż teraz lepiey ábyś nam był w mieśćie ná pomocy.
Y rzekł do nich Król ; Co śię wam zda dobrego / to ucżynię. Tedy ſtał Król przy bramie : á wƺyſtek lud wychodźił po ſtu y po tyśiącu.
Y rozkazał Król Joábowi / y Abiſájowi / y Itájowi / mówiąc ; Łáſkáwie mi śię <i>obchodźćie</i> z Synem mojim Abſálomem. A wƺyſtek lud ſłyƺał / gdy przykázywał Król wƺyſtkim Hetmánom o Abſálomie.
A ták wyćiągnął lud w pole przećiw Izráelowi / y zwiedli bitwę w leśie Efrájim.
Támże poráżon jeſt lud Izráelſki od ſług Dawidowych : y ſtáłá śię tám porażká wielka dniá onego ; <i>á poległo ich</i> dwádźieśćiá tyśięcy.
Bo gdy byłá bitwá rozproƺona po wƺyſtkiey źiemi / więcey lás pogubił ludu / niż ich miecż pożárł dniá onego.
Y nápadł Abſálom ná ſługi Dawidowe : á Abſálom jáchał ná mule : y wbieżał z nim muł pod gęſty á wielki dąb / y uwięzłá głowá jego ná dębie / y záwiſł miedzy niebem y miedzy źiemią : ále muł który był pod nim wybieżał.
Co ujrzawƺy mąż niektóry / oznájmił Joábowi / mówiąc ; Otom widźiał Abſálomá wiƺącego ná dębie.
Tedy rzekł Joáb mężowi / który mu to oznájmił ; Jeſliś widźiał / á cżemużeś go tám nie zábił / <i>y nie zrzućił</i> ná źiemię? A jabym ći był powinien dáć dźieśięć śrebrników / y jeden pás rycerſki.
Y odpowiedźiał on mąż Joábowi : A ja choćbym miał odważonych ná ręku mych tyśiąc śrebrników / nie podnióſłbym ręki mojey ná Syná Królewſkiego : boſmy ſłyƺeli gdy przykazał Król tobie y Abiſájowi / y Itájowi / mówiąc : Ochraniajćie wƺyſcy Syná <i>mego</i> Abſálomá :
Chybá żebym chćiał wdáć duƺę moję w niebeſpiecżeńſtwo. Bo nie bywa nic zátájono przed Królem ; y ty ſam byłbyś przećiwko mnie.
Tedy rzekł Joáb : Nie będęć śię ja tu báwił z tobą : przetoż wźiąwƺy trzy drzewcá w rękę ſwoję / wráźił je w ſerce Abſálomowe / gdy jeƺcże żyw był ná dębie.
A obſkocżywƺy Abſálomá dźieśięć ſług / którzy nośili broń Joábowę / bili / y zábili go.
W tym zátrąbił Joáb w trąbę / y wróćił śię lud z pogoni zá Izráelem : bo Joáb zátrzymał lud.
A wźiąwƺy Abſálomá / wrzućili go <i>w tymże</i> leśie w dół wielki / y nánośili nań bárzo wielką kupę kámienia. Ale wƺyſtek Izráel ućiekł / káżdy do namiotów ſwojich.
( A Abſálom wźiął był / y wyſtáwił ſobie zá żywotá ſwego ſłup / który jeſt w dolinie królewſkiey : bo mówił ; Nie mam Syná : Jednák zoſtáwię pámiątkę imienia mego. Przetoż názwał on ſłup imieniem ſwojim / który zowią / miejſce Abſálomowe / áż do dźiśiejƺego dniá. )
Tedy Achimáás Syn Sádoków / rzekł ; Proƺę niech idę / á oznajmię Królowi ućieƺną nowinę / yż go wybáwił PAN z ręki nieprzyjaćiół jego.
Ale mu rzekł Joáb : Nie byłbyś wdźięcżnym poſłem dźiśiá ; lecż to opowieƺ dniá drugiego / á dźiś niedawaj o tym znáć / przeto yż Syn królewſki zginął.
Potym Joáb rzekł do Chuſego : Idź / oznájmi Królowi coś widźiał : á ták ukłoniwƺy śię Chuſi Joábowi / bieżał.
Y mówił powtóre Achimáás Syn Sádoków / y rzekł do Joábá : Bądź co bądź / proƺę niech y ja bieżę zá Chuſim. Y rzekł Joáb : Przecżbyś ty miał bieżeć Synu mój? gdyż niemáƺ cobyś dobrego zwiáſtował?
Y <i>rzekł</i> : Bądź co bądź / pobieżę : y rzekł mu Joáb ; Bieżże : á ták bieżał Achimáás proſtƺą drogą / y uprzedźił Chuſego.
A Dawid śiedźiał miedzy dwiemá bramámi : y wyƺedł ſtróż ná dách bramy ná mur : á podnióżƺy ocży ſwe / ujrzał mężá jednego bieżącego.
Tedy záwoławƺy ſtróż / opowiedźiał <i>to</i> Królowi : Y rzekł Król : Jeſliżeć ſam jeſt / dobre poſelſtwo w uśćiech jego : á <i>gdy</i> ten ſpieƺnie ƺedł / y przybliżał śię :
Ujrzał ſtróż y drugiego mężá bieżącego / y záwołał ſtróż ná wrotnego / mówiąc ; Oto <i>y drugi</i> mąż bieży ſam. Y rzekł Król : y ten dobre poſelſtwo nieśie.
Nád to rzekł ſtróż ; Zda mi śię bieg pierwƺego / jáko bieg Achimááſá / Syná Sádokowego : Y rzekł Król : Mąż to dobry / y z dobrym poſelſtwem idźie.
Tedy záwołał Achimáás / y rzekł do Królá : Pokój : y ukłonił śię Królowi twarzą ſwoją ku źiemi / y rzekł ; Błogoſłáwiony PAN Bóg twój / któryć podał te męże / co podnieśli ręce ſwe przećiw Królowi Pánu memu.
Y rzekł Król : Jákoli śię ma Syn <i>mój</i> Abſálom? tedy Achimáás odpowiedźiał : Widźiałem zámięƺánie wielkie / gdy poſyłał ſługę Królewſkiego Joáb / y <i>mnie</i> ſługę twego / ále nie wiem co <i>było</i>.
Potym rzekł Król ; Odſtąp / á ſtań tám : á on odſtąpiwƺy / ſtánął.
A wtym Chuſy przyƺedł / y rzekł : Opowiáda śię Królowi Pánu memu / że ćię wybáwił PAN dźiśia z ręki wƺyſtkich / którzy powſtáli przećiwko tobie.
Y rzekł Król do Chuſego : A jákoli śię ma Syn <i>mój</i> Abſálom? odpowiedźiał Chuſy : Bodaj ták byli nieprzyjaćiele Królá Páná mego / y wƺyſcy którzy powſtawáją przećiw tobie ná złe / jáko Syn <i>twój</i>.
Tedy śię záſmućił Król / y wſtąpił ná ſalę oney bramy / á płákał ; y ták mówił idąc ; Synu mój Abſálomie / Synu mój! Synu mój Abſálomie! o bym ja był umárł miáſto ćiebie! Abſálomie Synu mój / Synu mój!
