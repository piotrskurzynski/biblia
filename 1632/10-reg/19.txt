Y oznájmiono Joábowi : Oto Król płácżę y żáłuje Abſálomá.
Przetoż śię ono zwyćięſtwo dniá onego obróćiło w płácż wƺyſtkiemu ludowi : ábowiem uſłyƺał lud dniá onego / że mówiono / żáłośny jeſt Król dla Syná ſwego :
Zácżym wkradał śię lud onego dniá / wchodząc do miáſtá / jáko śię więc wkrada lud / który śię wſtydźi ućiekájąc z bitwy.
A Król nákrywƺy oblicże ſwoje / wołał głoſem wielkim : Synu mój Abſálomie / Abſálomie Synu mój / Synu mój!
Tedy wƺedł Joáb do Królá w dom / y rzekł : Zháńbiłeś dźiś oblicże wƺyſtkich ſług twojich / którzy wybáwili duƺę twoję dźiśia / y duƺę Synów twojich / y córek twojich / y duƺę żon twojich / y duƺę założnic twojich /
Miłując te którzy ćię máją w nienawiśći / á nienawidząc tych / którzy ćię miłują : Abowiem pokazałeś dźiś / że ſobie nie poważaƺ Hetmánów / y ſług <i>twojich</i> / bom doznał <i>tego</i> dźiś / że gdyby Abſálom był żyw / choćbyſmy my wƺyſcy dźiś byli pobići / tedyćby śię to bárzo podobáło.
Przetoż teraz wſtań / wynidź / á mów łágodnie do ſług twojich : Boć przez PANA przyśięgam / jeſli ty nie wynidźieƺ / że nie zoſtánie żaden z tobą tey nocy : á będźieć to gorzey / niżli wƺyſtko złe / którekolwiek ná ćię przychodźiło od młodośći twojey / áż do tąd.
Wſtał tedy Król / y śiadł w bramie : y opowiedźiano <i>to</i> wƺyſtkiemu ludowi / mówiąc : Oto Król śiedźi w bramie : y przyƺedł wƺyſtek lud przed oblicże królewſkie : Ale Izráelcżycy ućiekli byli káżdy do namiotu ſwego.
Y ſtáło śię / <i>że</i> śię wƺyſtek lud ſprzecżał <i>z ſobą,</i> we wƺyſtkich pokoleniách Izráelſkich / mówiąc : Król wyrwał nas z rąk nieprzyjaćiół náƺych / tenże nas też wyrwał z rąk Filiſtyńſkich : á teraz ućiekł z źiemie przed Abſálomem.
Lecż Abſálom / któregoſmy byli pomázáli nád ſobą / zginął w bitwie : á teraz przecżże wy zániedbywaćie przyprowádźić záśię Królá?
Przetoż Król Dawid poſłał do Sádoká y do Abiátárá Kápłanów / z tymi ſłowy : Powiedzćie Stárƺym Judſkim / mówiąc ; Przecżże maćie bydź pośledniejƺymi w przyprowádzeniu záśię Królá do domu jego? ( Abowiem ſłowá wƺyſtkiego Izráelá dochodźiły Królá do domu jego ).
Bráćiaśćie moji / kość mojá / y ćiáłośćie moje : przecżże tedy maćie bydź pośledniejƺymi w przywróceniu Królá?
Amáźie <i>tákże</i> powiedzćie : Yzaliś ty nie jeſt kość mojá / y ćiáło moje? to niech mi ucżyni Bóg / y to niech przycżyni / jeſli Hetmánem Wojſká nie będźieƺ przede mną po wƺyſtkie dni / miáſto Joábá.
A ták nákłonił ſerce wƺyſtkich mężów Judſkich / jáko mężá jednego / że poſłáli do Królá mówiąc : Náwróć śię ty / y wƺyſcy ſłudzy twoji.
Wróćił śię tedy Król / y przyƺedł áż do Jordanu : á lud Judſki wyƺedł był do Gálgál / áby záƺedł w drodze Królowi / á przeprowádźił Królá przez Jordan.
Poſpieƺył śię tákże Semej Syn Gery / Syná Jemini / który był z Báhurym : y wyƺedł z mężámi Judſkimi przećiwko Królowi Dawidowi.
A było tyśiąc mężów z nim z Benjámitów : Sybá tákże / ſługá domu Saulowego / y piętnaśćie Synów jego / y dwádźieśćiá ſług jego z nim : y ƺcżęśliwie śię przepráwili zá Jordan do Królá.
Przepráwili też prum / áby przewieźiono cżeladź królewſką / á yżby ucżyniono coby mu śię nalepiey podobáło : á Semej Syn Gery / upadł przed Królem / gdy śię przepráwił przez Jordan.
Y rzekł do Królá : Nie przycżytaj mi / Pánie mój niepráwośći / áni wſpominaj co lekkomyślnie ucżynił ſługá twój onegoż dniá / gdy wyƺedł Król Pán mój z Jeruzalem : áby to miał przypuƺcżáć Król do ſercá ſwego :
Abowiem zna ſługá twój / żem zgrżeƺył : á otom dźiś przyƺedł pierwey niż kto / ze wƺyſtkiego domu Józefowego / ábym zájáchał drogę Królowi Pánu memu.
Tedy odpowiedźiał Abiſáj Syn Sárwiey : y rzekł ; Izaż dla tego nie ma bydź zábity Semej / że złorzecżył pomázáńcowi PAńſkimu?
Ale mu rzekł Dawid : Cóż wam do tego Synowie Sárwiey / żeśćie mi dźiś przećiwnymi? izali dźiś ma być zábity kto w Izráelu. Bo ázaż nie wiem / żem ja dźiś <i>zoſtał</i> Królem nád Izráelem?
Y rzekł Król do Semejá : Nie umrzeƺ : Y przyśiągł mu Król.
Mefiboſet tákże wnuk Saulów / wyjáchał przećiw Królowi ; który áni chędożył nóg ſwojich / áni cżoſał brody ſwojey / áni prał ƺat ſwojich / ode dniá / którego był wyƺedł Król / áż do dniá / którego śię wróćił w pokoju.
Y ſtáło śię / gdy zábieżał w Jeruzalem Królowi / rzekł mu Król ; Przecżżeś nie ƺedł ze mną Mefiboſećie?
A <i>on</i> mu odpowiedźiał : Królu Pánie mój / zdrádźił mię ſługá mój ; bo rzekł był ſługá twój / ośiodłam ſobie oſłá żebym wśiadƺy nań / jáchał z Królem : gdyż jeſt chromy ſługá twój.
Y oſkárżył ſługę twego przed Królem Pánem mojim : ále Król Pán mój / jeſt jáko Anjoł Boży : przetoż ucżyń co dobrego jeſt w ocżách twojich :
Abowiem wƺyſcy z domu Ojcá mego byliſmy godni śmierći przed Królem Pánem mojim / á <i>przećięś</i> ty poſádźił ſługę twego miedzy tymi / którzy jadáją u ſtołu twego : y cóż jeƺcże zá ſpráwiedliwość mojá / ábym śię miał więcey uſkarzáć ná Królá :
Rzekł mu tedy Król : Cóż maƺ więcey mówić w ſpráwie twojey? Jużem rzekł / ty / y Sybá podźielćie śię májętnośćią.
A Mefiboſet rzekł Królowi : Y wƺyſtko niech weźmie / gdy śię tylko wróćił Król Pán mój w pokoju do domu ſwego.
Bárſyláj też Gáláádcżyk wyƺedƺy z Rogelim / przepráwił śię z Królem przez Jordan / áby go prowádźił zá Jordan.
A Bárſyláj był bárzo ſtáry / májąc ośmdźieśiąt lat / który podejmował Królá / póki mieƺkał w Máhánáim / bo był cżłowiekiem bogátym bárzo.
Y rzekł Król do Bárſylájego ; Pódź ze mną / á będę ćię chował przy ſobie w Jeruzalem.
Ale Bárſyláj odpowiedźiał Królowi : Wieleż jeſt dni lat żywotá mego / żebym miał iść z Królem do Jeruzalem?
Ośmdźieśiąt lat mi dźiśia / izali mogę rozeznáć miedzy dobrym á złym? izali pocżuje ſmák ſługá twój w tym cobym jadł / álbo cobym pił? izali ſłucháć mogę więcey głoſu śpiewaków y śpiewacżek? á przecżżeby miał bydź ſługá twój jeƺcże ćiężarem Królowi Pánu memu?
Jeƺcże trochę pójdźie ſługá twój zá Jordan z Królem : Bo cżemużby mi miał dawáć Król tákową nagrodę?
Niech śię wróći proƺę ſługá twój / ábym umárł w mieśćie mojim / przy grobie Ojcá mego / y mátki mojey : ále oto ſługá twój Chimhám pójdźie z Królem Pánem mojim : Ucżyńże mu co dobrego jeſt w ocżách twojich.
Y rzekł Król : Niechże ze mną idźie Chimhám / á ja mu ucżynię co dobrego będźie w ocżách twojich : nád to cokolwiek żądáć będźieƺ ode mnie / toć ucżynię.
A gdy śię przepráwił wƺyſtek lud przez Jordan / Król śię też przepráwił : tedy pocáłował Król Bárſylájego / y błogoſłáwił mu : który śię wróćił do miejſcá ſwego.
Potym przyƺedł Król do Gálgál / przyƺedł też z nim Chimhám : Wƺyſtek też lud Judſki prowádźił Królá / tákże y połowicá ludu Izráelſkiego.
A oto wƺyſcy mężowie Izráelſcy zƺedƺy śię do Królá / mówili do niego : Cżemuż ćię wykrádli / bráćia náƺyj mężowie Judſcy / y przeprowádźili Królá y dom jego przez Jordan / y wƺyſtkie męże Dawidowe z nim?
Y odpowiedźieli wƺyſcy mężowie Judſcy mężom Izráelſkim : <i>Przeto,</i> yż nam powinny jeſt Król. A przecżże śię gniewáć maćie o to? izaliż nam <i>zá to</i> jeść Król dawa / álbo nam <i>jákie</i> dáry rozdał?
Tedy odpowiedźieli mężowie Izráelſcy mężom Judſkim / y rzekli : Dźieśięć kroć więcey mamy do Królá : przetoż y Dawid więcey do nas należy niż do was : przecżżeśćie nas lekce poważyli? Azażeſmy my o to pierwey niemówili / ábyſmy przywróćili Królá ſwego? ále ſrożƺa byłá mowá mężów Judſkich / niż mowá mężów Izráelſkich.
