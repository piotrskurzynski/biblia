A gdy śię JEzus národźił w Betlehemie Judſkim zá dni Herodá Królá / oto Mędrcy ze Wſchodu ſłońcá przybyli do Jeruzalem /
Mówiąc ; Gdźież jeſt ten który śię národźił Król Żydowſki? Boſmy widźieli gwiazdę jego ná Wſchód ſłońcá : y przyjácháliſmy / ábyſmy mu śię pokłonili.
Co gdy Król Herod uſłyƺał / zátrwożył śię y wƺyſtko Jeruzalem z nim.
Przetoż zebrawƺy wƺyſtkie przedniejƺe Kápłany y Náucżyćiele ludu / dowiádował śię od nich / gdźieby śię miał CHryſtus národźić?
A oni mu rzekli ; w Betlehemie Judſkim : Bo ták nápiſano przez Proroká :
Y ty Betlehemie źiemio Judſka / żadną miárą nie jeſteś namniejƺe miedzy Kśiążęty Judſkimi : ábowiem z ćiebie wynidźie Wódz / który rządźić będźie lud mój Izráelſki.
Tedy Herod wezwawƺy potájemnie <i>onych</i> Mędrców / pilnie śię wywiádował od nich o cżáśie / którego śię gwiazdá ukazáłá.
A poſławƺy je do Betlehem / rzekł : Jáchawƺy / pilnie śię wywiádujćie o tym dźiećiątku : á gdy znajdźiećie / oznájmićie mi ; ábym y ja przyjáchawƺy / pokłonił mu śię.
Oni tedy wyſłuchawƺy Królá / poƺli : A oto oná gwiazdá / którą widźieli ná Wſchód <i>słońcá,</i> prowádźiłá je / áż przyƺedƺy ſtánęłá nád <i>miejſcem</i> gdźie było dźiećiątko.
A gdy ujrzeli onę gwiazdę / urádowáli śię rádośćią bárzo wielką.
Y wƺedƺy w dom / ználeźli dźiećiątko z Máryą mátką jego ; á upadƺy / pokłonili mu śię : y otworzywƺy ſkárby ſwoje / ofiárowáli mu dáry / złoto y kádźidło y myrrę.
Lecż będąc upomnieni od Bogá we ſnie / áby śię nie wracáli do Herodá / inƺą drogą wróćili śię do krájiny ſwojey.
A gdy oni odeƺli / oto Anjoł PAńſki ukazał śię we ſnie Józefowi / mówiąc : Wſtawƺy / weźmi dźiećiątko y mátkę jego / á ućiecż do Egiptu / á bądź tám áż ći powiem. Abowiem Herod będźie ƺukał dźiećiątká / áby je zátráćił.
Który wſtawƺy w nocy / wźiął dźiećiątko y mátkę jego ; y uƺedł do Egiptu.
Y był tám áż do śmierći Herodowey : áby śię wypełniło / co powiedźiano od PAná przez Proroká mówiącego : Z Egiptum wezwał Syná mego.
Tedy Herod ujrzawƺy że był oƺukány od Mędrców / rozgniewał śię bárzo / á poſławƺy / pobił wƺyſtkie dźiatki / które były w Betlehemie y po wƺyſtkich gránicách jego / ode dwu lat y niżey / według cżáſu / o którym śię był pilnie wywiedźiał od Mędrców.
Tedy śię wypełniło co powiedźiano przez Jeremiaƺá Proroká / mówiącego :
Głos w Rámie ſłyƺány jeſt / láment y płácż / y nárzekánie wielkie : Ráchel płácżąca ſynów ſwojich / y nie dáłá śię poćieƺyć ; przeto / że ich niemáƺ.
A gdy umárł Herod / oto Anjoł PAńſki ukazał śię we ſnie Józefowi w Egipćie /
Mówiąc : Wſtawƺy / weźmi dźiećiątko y mátkę jego / á idź do źiemie Izráelſkiey : Abowiem pomárli ći / którzy ƺukáli duƺe dźiećięcey.
A on wſtawƺy wźiął do śiebie dźiećiątko y mátkę jego / y przyƺedł do źiemie Izráelſkiey.
Lecż gdy uſłyƺał iż Archeláus królował w Judſkiey źiemi / ná miejſcu Herodá ojcá ſwego / bał śię tám iść : ále nápomniony będąc od Bogá we ſnie / uſtąpił w ſtrony Gálilejſkie.
A przyƺedƺy mieƺkał w mieśćie / które zowią Názáret : áby śię wypełniło co powiedźiano przez Proroki : Yż Názárejcżykiem názwány będźie.
