Strzeżćie śię ábyśćie jáłmużny wáƺey nie cżynili przed ludźmi / <i>dla tego</i> ábyśćie byli widźiáni od nich : inácżey nie będźiećie mieli zapłáty u Ojcá wáƺego / który jeſt w niebieśiech.
Przetoż / gdy cżyniƺ jáłmużnę / nie trąb przed ſobą jáko obłudnicy cżynią w bóżnicách y ná ulicách / áby byli chwaleni od ludźi : Záprawdę powiádam wam / odbieráją zapłátę ſwoję.
Ale ty gdy cżyniƺ jáłmużnę ; niechaj nie wie lewicá twojá / co cżyni práwicá twojá :
Aby jáłmużná twojá byłá w ſkrytośći ; á Oćiec twój / który widźi w ſkrytośći / tenći jáwnie odda.
A gdy śię modliƺ / nie bądź jáko obłudnicy : ábowiem śię oni rádźi w bożnicách y ná rogách ulic ſtojąc modlą / áby byli widźiáni od ludźi : Záprawdę powiádam wam / iż odbieráją zapłátę ſwoję.
Ale ty gdy śię modliƺ / wnidź do komory ſwojey : á záwárƺy drzwi ſwoje / módl śię Ojcu twemu który jeſt w ſkrytośći : á Oćiec twój który widźi w ſkrytośći / oddać jáwnie.
A modląc śię / nie bądźćie wielomówni jáko Pogánie : ábowiem oni mniemáją / że dla ſwojey wielomównośći wyſłucháni będą.
Nie bądźćież tedy im podobni : gdyż wie Oćiec wáƺ cżego potrzebujećie / pierwey niżbyśćie wy go prośili.
Wy tedy ták śię módlćie : Ojcże náƺ / któryś jeſt w niebieśiech. Święć śię Imię twoje.
Przydź króleſtwo twoje. Bądź wola twa jáko w niebie / ták y ná źiemi.
Chlebá náƺego powƺedniego daj nam dźiśia.
Y odpuść nam náƺe winy / jáko y my odpuƺcżamy náƺym winowájcom.
Y nie w wódź nas ná pokuƺenie ; Ale nas zbaw ode złego. Abowiem twoje jeſt króleſtwo / y moc / y chwałá ná wieki / Amen.
Bo jeſli odpuśćićie ludźiom upadki ich / odpuśći y wam on Oćiec wáƺ niebieſki.
Ale jeſli nie odpuśćićie ludźiom upadków ich / y Oćiec wáƺ nie odpuśći wam upadków wáƺych.
A gdy pośćićie / nie bądźćież ſmętney twarzy / jáko obłudnicy : ƺpecą bowiem twarzy ſwoje / áby byli widźiáni od ludźi że poƺcżą. Záprawdę powiádam wam / odbieráją zapłátę ſwoję.
Ale ty gdy pośćiƺ / námáż głowę twoję / y umyj twarz twoję :
Abyś nie był widźiány od ludźi że pośćiƺ / ále od onego Ojcá twojego który jeſt w ſkrytośći : á Oćiec twój który widźi w ſkrytośći / oddać jáwnie.
Nie ſkarbćie ſobie ſkárbów ná źiemi / gdźie mól y rdzá pſuje / y gdźie złodźieje podkopywáją y krádną.
Ale ſobie ſkarbćie ſkárby w niebie : gdźie áni mól áni rdzá pſuje ; y gdźie złodźieje nie podkopywáją áni krádną.
Abowiem gdźie jeſt ſkarb wáƺ / tám jeſt y ſerce wáƺe.
Oko twoje / jeſtći świecą ćiáłá twego : Jeſliby tedy oko twoje było ƺcżere / wƺyſtko ćiáło twoje jáſne będźie.
Jeſliby záś oko twoje złe było / wƺyſtko ćiáło twoje ćiemne będźie. Jeſli tedy świátłość / która jeſt w tobie / ćiemnośćią jeſt ; <i>ſamá</i> ćiemność jákaż <i>będźie</i>?
Nikt niemoże dwiemá Pánom ſłużyć : gdyż álbo jednego będźie miał w nienawiśći / á drugiego będźie miłował : álbo jednego trzymáć śię będźie / á drugim pogárdźi. Nie możećie Bogu ſłużyć y Mámmonie.
Dla tego powiádam wam : Nie troƺcżćie śię o żywot wáƺ / cobyśćie jedli álbo cobyśćie pili ; áni o ćiáło wáƺe / cżymbyśćie śię odźiewáli : Azaż żywot nie jeſt zacniejƺy niż pokarm / y ćiáło niż odźienie?
Pojrzyćie ná ptaki niebieſkie / iż nie śieją / áni żną / áni zbieráją do gumien / á wżdy Oćiec wáƺ niebieſki żywi je. Izali wy nie jeſteśćie dáleko zacniejƺy nád nie.
Y któż z was troſkliwie myśląc może przydáć do wzroſtu ſwe° łokieć jeden?
A o odźienie przecżże śię troƺcżećie? Przypátrzćie śię liliam polnym / jáko roſtą ; nie prácują áni przędą?
A Ja wam powiádam / iż áni Sálomon we wƺyſtkiey ſławie ſwojey nie był <i>ták</i> przyodźiány / jáko jedná z tych.
Jeſli tedy trawę polną / która dźiś jeſt / á jutro bywa w piec wrzucona / Bóg ták przyodźiewa / ázaż nie dáleko więcey was / o máłowierni?
Nie troƺcżćie śię tedy / mówiąc ; Cóż będźiemy jeść? álbo co będźiemy pić? álbo cżym śię będźiemy przyodźiewáć.
Boć tego wƺyſtkiego Pogánie ƺukáją : wieć bowiem Oćiec wáƺ niebieſki / że tego wƺyſtkiego potrzebujećie.
Ale ƺukajćie naprzód króleſtwá Bożego / y ſpráwiedliwośći jego / á to wƺyſtko będźie wam przydano.
Przetoż / nie troƺcżćie śię o jutrzejƺy dźień : ábowiem jutrzejƺy dźień troſkáć śię będźie o ſwoje <i>potrzeby</i>. Doſyćći ma dźień ná ſwojim utrapieniu.
