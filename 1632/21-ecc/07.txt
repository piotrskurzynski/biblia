Lepƺe jeſt imię dobre / niżeli máść wyborná ; á dźień śmierći / niż dźień národzeniá.
Lepiej iść do domu żáłoby / niż iść do domu bieśiády / przeto / iż tám widźimy koniec káżdego cżłowieká / á żyjący ſkłádá to do ſercá ſwego.
Lepƺy jeſt ſmutek / niżeli śmiech ; bo przez ſmutek twárzy nápráwiá śię ſerce.
Serce mądrych w domu żáłoby ; ále ſerce głupich w domu weſelá.
Lepiej jeſt ſłucháć gromieniá mądrego / niżeli ſłucháć pieśni głupich.
Bo jáki jeſt trzáſk ćierniá pod gárncem / ták jeſt śmiech głupiego ; y toć jeſt márność.
Zájiſte ućiſk przywodźi mądrego do ƺáleńſtwá / á dar záślepiá ſerce.
Lepƺe jeſt dokońcżenie rzecży / niżeli pocżątek jey ; lepƺy jeſt cżłowiek ćierpliwego duchá / niż duchá wynioſłego.
Nie bądź porywcży w duchu twym do gniewu ; bo gniew w zánádrzu głupich odpocżywá.
Nie mów : Cóż to jeſt / że dni pierwƺe lepƺe były / niż teráźniejƺe? Bobyś śię o tym nie mądrze pytał.
Dobrá jeſt mądrość przy májętnośći / y jeſt pożytecżná tym / którzy widzą ſłońce.
Abowiem pod ćieniem mądrośći / y pod ćieniem ſrebrá odpocżywá cżłowiek / á wƺákże przedniejƺá jeſt umiejętność mądrośći ; bo przynośi żywot tym / którzy ją máją.
Przypátrz śię ſpráwie Bożey ; bo któż może wyproſtowáć / co on ſkrzywi?
W dźień dobry záżywáj dobrá / á w dźień zły miej śię ná piecży : boć ten ucżynił Bóg przećiwko owemu / dla tego / áby nie doƺedł cżłowiek tego / co náſtánie po nim.
Tom wƺyſtko widźiał zá dni márnośći mojey : Bywa ſpráwiedliwy / który ginie z ſpráwiedliwośćią ſwoją ; tákże bywa niezbożnik / który długo żyje we złośći ſwojey.
Nie bądź názbyt ſpráwiedliwym / áni názbyt mądrym ; przecżżebyś miał do zguby przychodźić?
Nie bądź náder niepobożnym / áni názbyt głupim ; przecżżebyś miał umrzeć przed cżáſem ſwoim?
Dobrá jeſt / ábyś śię owego trzymał / á tego śię nie puƺcżał ; kto śię boi Bogá / uchodźi tego wƺyſtkiego.
Mądrość umácniá mądrego więcey / niżeli dźieśięć kśiążąt / którzy ſą w mieśćie.
Zájiſte niemáƺ cżłowieká ſpráwiedliwego ná źiemi / któryby cżynił dobrze / á nie grzeƺył.
Nie do wƺyſtkich też ſłów / które mówią ludźie przykłádáj ſercá twego ; y niech ćię to nie obchodźi / choććiby y ſługá twój złorzecżył.
Boć wie ſerce twoje / żeś y ty cżęſtokroć drugim złorzecżył.
Wƺyſtkiegom tego doświádcżył mądrośćią / y rzekłem : Będę mądrym ; áleć śię mądrość oddáliłá odemnie.
A co dálekiego / y co bárzo głębokiego jeſt / któż to znájdźie?
Wƺyſtkom ja przeƺedł myślą ſwoją / ábym poznał y wybádał śię / y wynálázł mądrość y rozum / á żebym poznał niezbożność / głupſtwo / y błąd / y ƺáleńſtwo.
Y ználázłem rzecż gorzćiejƺą nád śmierć / to jeſt / táką niewiáſtę / którey ſerce jeſt jáko śieći y śidło / á ręce jey jáko pętá. Kto śię Bogu podobá / wolny będźie od niej ; ále grzeƺnik będźie od niej pojmány.
Otom to ználázł / ( mówi káznodźiejá / ) ſtoſując jedno z drugiem / ábym doƺedł umiejętnośći.
Cżego záś nád to ƺukáłá duƺá mojá / tedym nie ználázł. Mężá jednego z tyśiącá ználázłem ; álem niewiáſty miedzy temi wƺyſtkiemi nie ználázł.
To tylko obácż / com ználázł / że ſtworzył Bóg cżłowieká dobrego ; ále oni událi śię zá rozmáitemi myślámi.
Któż może z mądrym porównáć? á kto może wyłożyć káżdą rzecż?
