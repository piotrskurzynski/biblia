Rzekłem ja do ſercá mego : Nuże teraz doświádcżę ćię w weſelu / używájże dobrych rzecży ; ále y toć márność.
Śmiechowi rzekłem : Szálejeƺ / á weſelu : Cóż to cżyniƺ?
Przemyśláłem w ſercu ſwem / ábym pozwolił winá ćiáłu memu / ( ſerce jednák ſwoje ſpráwując mądrośćią ) y ábym śię trzymał głupſtwá dotąd / áżbym obácżył / coby lepƺego było ſynom ludzkim cżynić pod niebem / przez wƺyſtkie dni żywotá ich.
Wielkiem ſpráwy wykonał ; pobudowáłem ſobie domy / náſádźiłem ſobie winnic ;
Nácżyniłem ſobie ogrodów / y ſádów / y náƺcżepiłem w nich drzew wƺelákiego owocu ;
Pobudowáłem ſobie ſtáwy ku odwilżániu przez nie láſu / w którym rośćie drzewo ;
Nábyłem ſobie ſług y dźiewek / y miáłem cżeládź w domu moim ; do tego y ſtádá wołów / y wielkie trzody owiec miáłem nád wƺyſtkich / którzy byli przedemną w Jeruzálemie.
Zgromádźiłem też ſobie ſrebro y złoto / y klejnoty od królów y kráin. Sporządźiłem też ſobie śpiewáków y śpiewácżki / y inne rozkoƺe ſynów ludzkich / y muzycżne nacżynia rozlicżne.
A ták ſtałem śię wielkim y możniejƺym nád wƺyſtkich / którzy byli przedemną w Jeruzálemie ; nádto mądrość mojá zoſtáwáłá przy mnie.
A wƺyſtkiego / cżego pożądáły ocży moje / nie zábrániáłem im / ánim odmáwiał ſercu memu żádnego weſelá ; ále ſerce moje weſeliło śię ze wƺyſtkiej prácy mojey. A toć był dźiał mój ze wƺyſtkiej prácy mojey.
Lecż gdym śię obejrzał ná wƺyſtkie ſpráwy ſwoje / które cżyniły ręce moje / y ná práce / którem podejmował prácując : oto wƺyſtko márność / y utrápienie duchá / y niemáƺ nic pożytecżnego pod ſłońcem.
Przetoż obróćiłem śię do tego / ábym śię przypátrywał mądrośći / y ƺáleńſtwu / y głupſtwu ; ( bo cóżby cżłowiek cżynił ten / który náſtánie po królu? to / co już inni cżynili. )
Y obácżyłem / iż jeſt pożytecżniejƺá mądrość niżeli głupſtwo / ták jáko jeſt pożytecżniejƺá świátłość / niżeli ćiemność.
Mądry ma ocży w głowie ſwej / ále głupi w ćiemnośćiách chodźi ; á wƺákżem poznał / że jednákie przygody ná wƺyſtkich przychodzą.
Dlátegom rzekł w ſercu mojim : Máli mi śię ták dźiáć / jáko śię głupiemu dźieje / przecżżem go ja tedy mądrośćią przeƺedł? Przetożem rzekł w ſercu mojim : Y toć jeſt márność.
Abowiem nie ná wieki będźie pámiątki mądrego y głupiego / dla tego / iż to / co teraz jeſt / we dni przyƺłe wƺyſtkiego zápomną ; á jáko umierá mądry / ták y głupi.
Przetoż mi żywot omierzł ; bo mi śię nie podobá żádná rzecż / która śię dźieje pod ſłońcem ; ábowiem wƺyſtkie ſą márnośćią / y utrapieniem duchá.
Náwet omierzłá mi y wƺyſtká prácá mojá / którąm podejmował pod ſłońcem / przeto / że ją zoſtáwić muƺę cżłowiekowi / który náſtánie po mnie.
A kto wie / będźieli mądrym / cżyli głupim? á wƺákże będźie pánował nád wƺyſtką prácą moją / którąm prowádźił / y w którejm był mądry pod ſłońcem. Aleć y to márność.
Y przypádłem ná to / ábym zwątpił w ſercu mojim o wƺyſtkiej prácy / którąm śię mądrze báwił pod ſłońcem.
Nie jeden zájiſte cżłowiek prácuje mądrze / y umiejętnie / y ſpráwiedliwie ; á wƺákże to innemu / który nie robił ná to / zá dźiał zoſtáwi. Y toć márność y wielka biedá.
Bo cóż ma cżłowiek ze wƺyſtkiej prácy ſwej / y z uśiłowánia ſercá ſwego / które podejmuje pod ſłońcem?
Ponieważ wƺyſtkie dni jego ſą boleſne / á zábáwá jego jeſt fráſunek / ták iż y w nocy nie odpocżywá ſerce jego. Y toć jeſt márność.
Izali nie lepƺá cżłowiekowi / áby jádł y pił / y dobrze ucżynił duƺy ſwojey z prácy ſwojey? álemći widźiał / że y to z ręki Bożey pochodźi.
Abowiem któżby ſłuƺniej miał jeść / y pożywáć tego nád mię?
Bo cżłowiekowi / który mu śię podobá / dáje mądrość / umiejętność / y weſele ; ále grzeƺnikowi dáje fráſunek / áby zbierał y zgromádzał / coby zoſtáwił temu / który śię podobá Bogu. Y toć jeſt márność / á utrápienie duchá.
