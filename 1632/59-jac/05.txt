Nuż teraz bogacże / płácżćie nárzekájąc nád nędzámi wáƺymi które przydą.
Bogáctwo wáƺe zgniło : á ƺáty wáƺe mole zgryzły.
Złoto wáƺe y śrebro wáƺe pordzewiáło : á rdzá ich będźie ná świádectwo przećiwko wam / y pożrże ćiáłá wáƺe jáko ogień. Zgromádźieliśćie ſkarb ná oſtátnie dni.
Oto zapłátá robotników którzy żęli krájiny wáƺe od was zátrzymána woła / á wołánia żeńców weƺły do uƺu PAná Zaſtępów.
Żyliśćie w roſkoƺách ná źiemi / y bujáliśćie : wytucżyliśćie ſercá wáƺe / jáko ná dźień zábijánia <i>ofiar</i>.
Potępiliśćie / zámordowáliśćie ſpráwiedliwego ; á nie zprzećiwia śię wam.
Przetoż bráćia bądźćie ćierpliwymi / áż do przyśćia PAńſkiego. Oto oracż ocżekawa drogiego pożytku źiemie / ćierpliwie go ocżekawájąc / áżby otrzymał deƺcż ránny y wiecżorny.
Bądźćież y wy ćierpliwymi / <i>á</i> utwirdzajćie ſercá wáƺe : ábowiem śię przybliża przyśćie PAńſkie.
Nie wzdychajćie jedni przećiwko drugim / bráćia : ábyśćie nie byli oſądzeni : Oto Sędźia już przede drzwiámi ſtoji.
Bierżćie ná przykład bráćia moji / utrapienia y ćierpliwośći / Proroki : którzy mówili w Imieniu PAńſkim.
Oto zá błogoſłáwione mamy te którzy ćierpieli. O ćierpliwośći Jobowey ſłycháliśćie / y koniec PAńſki widźieliśćie / iż wielce miłośierny jeſt PAn y litośćiwy.
A przed wƺyſtkimi <i>rzecżámi</i> bráćia moji / nie przyśięgajćie áni przez niebo / áni przez źiemię / áni żadną inƺą przyśięgą. Ale niech będźie <i>mowá</i> wáƺá ; Ták / ták ; y Nie / nie : ábyśćie w ſąd nie wpádli.
Jeſt kto utrapiony miedzy wámi? Niechże śię modli. Jeſt kto dobrey myśli? Niechajże śpiewa.
Choruje kto miedzy wámi : Niechże záwoła Stárƺych Zborowych / á niech śię modlą zá nim / pomázując go olejkiem w Imieniu PAńſkim :
A modlitwá wiáry uzdrowi chorego / y podnieśie go PAn : á jeſliby śię grzechu dopuśćił / będźie mu odpuƺcżono.
Wyznawajćie jedni przed drugimi upadki : á módlćie śię jedni zá drugimi / ábyśćie byli uzdrowieni ; Wiele może uprzyjma modlitwá ſpráwiedliwego.
Eliaƺ był cżłowiek tymże biedam poddány jáko y my / á modlitwą modlił śię żeby deƺcż nie pádał : y nie pádał deƺcż ná źiemię trzy látá y ƺeść mieśięcy.
Y záś śię modlił á wydáło niebo deƺcż / y źiemiá zrodźiłá owoce ſwoje.
Bráćia / jeſliby śię kto z was obłądźił od prawdy / á náwróćiłby go kto :
Niechże wie / że ktoby odwróćił grzeƺniká od błędney drogi jego / záchowa duƺę od śmierći / y zákryje mnóſtwo grzechów.
