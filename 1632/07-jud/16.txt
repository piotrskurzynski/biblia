Potym ƺedł Sámſon do Gázy / á ujrzawƺy tám niewiáſtę nierządną wƺedł do niey.
Y powiedźiano mieƺcżánom w Gázie / Przyƺedł tu Sámſon : Którzy obſtąpiwƺy go / ſtrzegli nań cáłą noc w bramie miejſkiey : á ſpráwując śię ćicho przez onę cáłą noc / mówili : Gdy śię pocżnie rozedńiwáć / zábijemy go.
Ale Sámſon ſpał áż do pułnocy : á wſtawƺy o pułnocy / ujął wrotá bramy miejſkiey ze dwiemá podwojámi / y wyrwał je z zaworą / y włożył ná rámioná ſwoje / á zánióſł je ná wierzch góry / która byłá przećiw Hebronowi.
Y ſtáło śię potym / że śię rozmiłował niewiáſty w dolinie Sorek / którey imię Dálilá.
Y przyƺli do niey Kśiążętá Filiſtyńſkie / y mówili jey : Oƺukaj go / á wywiedz śię w cżym <i>jeſt</i> moc jego wielka / á jákobyſmy go przemóc y związawƺy utrapić mogli / á dać káżdy z nas tyśiąc y ſto śrebrników.
Tedy rzekłá Dálilá do Sámſoná : powiedz mi proƺę / w cżym <i>jeſt</i> moc twojá wielka / á cżymbyś związány y utrapiony bydź mógł?
Y odpowiedźiał jey Sámſon : Jeſliby mię związano śiedmią wići ſurowych / które jeƺcże nie uſchły / tedy oſłábieję / y będę jáko inny cżłowiek.
Y przynioſły jey Kśiążętá Filiſtyńſkie śiedm wići ſurowych : które jeƺcże nie były uſchły / y związáłá go nimi.
A oni śię byli nań záſádźili w komorze / y rzekłá mu : Filiſtynowie nád tobą Sámſonie : Ale <i>on</i> zerwał wići / jákoby kto zerwał nić zgrzebną ogniem nápaloną : y nie poznano <i>w cżym byłá</i> moc jego.
Rzekłá potym Dálilá do Sámſoná : Otoś mię oƺukał y zkłamałeś mną / teraz powiedz mi proƺę / cżymby ćię związáć.
A on jey odpowiedźiał : Jeſliby mię związano powrozámi nowymi / których <i>jeƺcże</i> nie używano / tedy oſłábieję / y będę jáko inny cżłowiek.
A ták wźięłá Dálilá powrozy nowe / y związáłá go nimi / y rzekłá do niego : Filiſtynowie nád tobą Sámſonie / ( á oni śię byli nań záſádźili w komorze / ) ále porwał je ná rámionách ſwych jáko nići.
Rzekłá zátym Dálilá do Sámſoná : Pókiż ze mnie ƺydźić będźieƺ / y kłamáć mną? powiedzże mi / cżymbyś mógł być związány : Y powiedźiał jey : gdybyś przywiłá śiedm kędźierzy głowy mojey do wáłu tkáckiego.
Oná tedy przybiwƺy goźdźiem <i>do wáłu tkáckiego</i> / rzekłá do niego : Filiſtynowie nád tobą Sámſonie : ále <i>on</i> ocućiwƺy ze ſnu ſwego wyrwał góźdź z oſnową y z wáłem.
Znowu rzekłá do niego : Jákoż mówiƺ / miłuję ćię / á ſerce twoje nie jeſt ze mną / jużeś mię po trzy kroć oƺukał / y nie powiedźiałeś mi / w cżym jeſt twojá moc wielka.
A gdy mu śię uprzykrzáłá ſłowy ſwymi ná káżdy dźień y trapiłá go / áż zemdláłá duƺá jego ná śmierć.
Tedy jey otworzył cále ſerce ſwoje / y powiedźiał jey : Brzytwá nigdy nie poſtałá ná głowie mojey / gdyżem jeſt Názárejcżykiem Bożym <i>záraz</i> z żywotá mátki mojey : gdyby mię ogolono / odejdźie ode mnie moc mojá / y oſłábieję / y będę jáko inny cżłowiek.
Widząc tedy Dálilá że jey otworzył cále ſerce ſwoje / poſłáłá y wezwáłá Kśiążąt Filiſtyńſkich / mówiąc : pódźćież <i>jeƺcże</i> raz / boć mi otworzył cále ſerce ſwoje / y przyƺły do niey Kśiążętá Filiſtyńſkie / nioſąc śrebro w ręku ſwych.
Tedy go uſpiłá ná łonie ſwojim / á przyzwawƺy <i>niektórego</i> cżłowieká / dáłá ogolić śiedm kędźierzy głowy jego / potym go jęłá draźńić / gdy odeƺłá moc jego od niego.
Y rzekłá : Filiſtynowie nád tobą Sámſonie : A ocućiwƺy śię ze ſnu ſwego / rzekł : Wynidę jáko y pierwey : á wybiję śię. A nie wiedźiał że PAN odſtąpił od niego.
Tedy pojimawƺy go Filiſtynowie / wyłupili mu ocży / y wiedli go do Gázy / związawƺy go dwiemá miedźiánymi łáńcuchámi / y muśiał mleć w domu więźniów.
Potym pocżęły włoſy ná głowie jego odraſtáć po onym ogoleniu.
A Kśiążętá Filiſtyńſkie zebráli śię ſpráwowáć Ofiáry wielkie Dágonowi Bogu ſwemu / weſelili śię / y mówili / Podał Bóg náƺ w ręce náƺe Sámſoná nieprzyjaćielá náƺego.
Którego też ujrzawƺy lud / chwalili Bogá ſwego / bo mówili : Podał Bóg náƺ w ręce náƺe nieprzyjaćielá náƺego / á tego który puſtoƺył źiemię náƺę / y który wiele z náƺych pozábijał.
Y ſtáło śię gdy byli dobrey myśli / że rzekli : Záwołajćie Sámſoná / áby błaznował <i>przed</i> námi. A ták záwołano Sámſoná z domu więźniów / áby błaznował przed nimi : y poſtáwili go miedzy dwiemá ſłupámi.
Zátym rzekł Sámſon do chłopcá który go trzymał zá rękę jego : Przywiedź mię / ábym pomácał ſłupów / ná których dom ſtoij / y podpárł śię ná nich.
A dom pełen był mężów y niewiaſt / támże <i>były</i> wƺyſtkie Kśiążętá Filiſtyńſkie / á ná dáchu około trzech tyśięcy mężów y niewiaſt / którzy śię przypátrowáli gdy błaznował Sámſon.
Wzywał tedy Sámſon PAná y rzekł : PANIE Boże wſpomni ná mię proƺę / á zmocni mię proƺę / tylko ten raz : Boże ábym śię raz pomśćił obu ocżu mojich nád Filiſtynámi.
A ująwƺy Sámſon obá ſłupy pośrzednie ná których dom ſtał / wſpárł śię o nie / o jeden práwą ręką ſwoją / á o drugi lewą ręką ſwoją.
Zátym rzekł Sámſon : Niech umrze duƺá mojá z Filiſtynámi : á gdy śię o nie mocno opárł / upadł dom ná Kśiążętá / y ná wƺyſtek lud który w nim był. Y było umárłych które on pobił umierájąc więcey niż onych które pobił zá żywotá ſwego.
A przyƺedƺy bráćia jego / y wƺyſtek dom Ojcá jego / wźięli go / á wróćiwƺy śię pogrzebli go miedzy Sáráá / y miedzy Eſtáol / w grobie Mánue / Ojcá jego : á on ſądźił Izráelá przez dwádźieśćiá lat.
