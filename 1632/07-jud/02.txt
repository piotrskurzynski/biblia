Y przyƺedł Anjoł PAńſki z Gálgál do Bochim / mówiąc : wywiodłem was z Egiptu / y w wiodłem was do źiemie / o którąm przyśiągł Ojcom wáƺym / y mówiłem : Nie wzruƺę przymierza mego z wámi ná wieki :
Jedno wy nie wchodźćie w przymierze z mieƺkájącymi w tey źiemi / <i>owƺem</i> Ołtarze ich porozwalajćie : áleśćie nie ſłucháli głoſu mojego. Przecżżeśćie to ucżynili?
Przetożem ták rzekł : Nie wypędzę ich od oblicżnośći wáƺey / ále będą wam <i>jáko ćiernie</i> ná boki / á Bogowie ich będą wam jáko śidło.
Y ſtáło śię / gdy mówił Anjoł PAńſki te ſłowá do wƺyſtkich Synów Izráelſkich / że lud podnióſł głos ſwój / y płákáli.
Y názwáli imię miejſcá onego Bochim / á támże Ofiárowáli PAnu :
A Jozue rozpuśćił był lud / y rozeƺli śię Synowie Izráelſcy / káżdy do dźiedźictwá ſwego / áby pośiedli źiemię.
Tedy ſłużył lud PAnu po wƺyſtkie dni Jozuego / y po wƺyſtkie dni Stárƺych / którzy długo żyli po <i>śmierći</i> Jozuego / <i>á</i> którzy widźieli wƺyſtkie ſpráwy PAńſkie wielkie / które uczynił Izráelowi.
Ale gdy umárł Jozue Syn Nunów / ſługá PAńſki / będąc we ſtu y w dźieśiąći lat :
Y <i>gdy</i> go pogrzebli ná gránicy dźiedźictwá jego / w Támnátheres / ná górze Efrájm / od pułnocy góry Gáás :
Tákże <i>gdy</i> wƺyſtek on rodzaj przyłącżony jeſt do Ojców ſwojich / y powſtał po nich inƺy naród / który nie znał PANA / áni też ſpraw które ucżynił Izráelowi :
Tedy cżynili Synowie Izráelſcy złe przed oczymá PAńſkiemi / á ſłużyli Báálom.
Y opuśćili PANA Bogá Ojców ſwojich / który je wywiódł z źiemi Egipſkiey / y ƺli zá Bogámi cudzymi / którzy byli z Bogów onych narodów okolicżnych / y kłaniáli śię im / á ták rozdraznili PANA.
Bo opuśćili PANA / á ſłużyli Báálowi y Aſtárotowi.
Y rozpalił śię gniew PAńſki przećiw Izráelowi / y podał je w ręce łupieżcom / którzy je łupili / á záprzedał je / w ręce nieprzyjaćiół ich okolicżnych : ták iż śię nie mogli dáliey oſtać / przed nieprzyjaćioły ſwymi.
A gdźie śię kolwiek ruƺyli / ręká PAńſka byłá przećiwko nim ku złemu / jáko powiedźiał PAN / y jáko im przyśiągł PAN : y byli śćiśnieni bárzo.
Potym PAN wzbudźił Sędźie / którzy je wyzwaláli z rąk łupieżców ich :
Ale y Sędźiów ſwych nie ſłucháli / owƺem śię zcudzołożyli z Bogámi obcymi / kłaniájąc śię im / <i>y</i> uſtępowáli prętko z drogi / którą chodźili Ojcowie ich : á ſłuchájąc przykazáń PAńſkich / nie cżynili ták.
A gdy im wzbudzał PAN Sędźie / bywał PAN z káżdym Sędźią / y wybáwiał je z rąk nieprzyjaćiół ich / po wƺyſtkie dni <i>onego</i> Sędźiego : ( bo śię użalił PAN nárzekánia ich / do którego je przywodźili <i>ći</i> którzy je ućiſkáli y trápili. )
Wƺákże / po śmierći Sędźiego / odwracáli śię / y pſowáli śię bárdźiey / niż Ojcowie ich / chodząc zá Bogámi cudzymi / á ſłużąc im / y kłaniájąc śię im / <i>nic</i> nie opuƺcżáli z ſpraw ſwojich / y z drogi ſwojey uporney.
Przetoż wzruƺyłá śię popędliwość PAńſka przećiw Izráelowi / y rzekł : dla tego / że przeſtąpił ten naród przymierze moje / którem przykazał Ojcom ich / á nie byli poſłuƺni głoſowi mojemu :
Ja też ná potym nie wypędzę żadnego od twarzy ich z tych narodów / które pozoſtáwił Jozue / kiedy umárł.
Abym przez nie doświadczał Izráelá / będąli ſtrzedz drogi PAńſkiey / chodząc po niey / jáko <i>jey</i> ſtrzegli Ojcowie ich / cżyli nie?
Y zoſtáwił PAN one narody nie wyganiájąc ich rychło / áni ich podał / w rękę Jozuego.
