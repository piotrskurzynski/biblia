Y ſtáło śię w one dni / gdy Królá nie było w Izráelu / że mąż <i>niektóry</i> Lewitá / mieƺkájący przy ſtronie góry Efrájim / pojął ſobie żonę założnicę z Betlehem Judá.
A báwiłá śię nierządem przy nim założnicá jego : potym odeƺłá od niego do domu Ojcá ſwego / do Betlehem Judá : y byłá tám u niego przez cżtery mieśiące.
Wſtawƺy tedy mąż jey / ƺedł zá nią / áby ją ubłagawƺy záśię ją przywiódł / májąc z ſobą ſługę ſwego / y parę oſłów : Tedy <i>oná</i> w wiodłá go w dom Ojcá ſwego / którego gdy ujrzał Oćiec oney dźiewki rádował śię z przyśćia jego.
Y przyjął go wdźięcżnie Świekier jego / Oćiec dźiewki <i>oney,</i> á mieƺkał u niego przez trzy dni / y jedli y pili y nocowáli tám.
A dniá cżwartego / gdy wſtáli bárzo ráno / wſtał y on áby odƺedł. Ale rzekł Oćiec oney dźiewki do źięćiá ſwego : pośilń ſerce twoje trochą chlebá / á potym pójdźiećie.
Tedy śiedli / y jedli oboje weſpół / y nápili śię : Zátym rzekł Oćiec oney dźiewki do mężá <i>jey</i> : zoſtań proƺę / á przenocuj tu y bądź dobrey myśli.
A gdy wſtał on mąż chcąc przećię iść w drogę / gwałtem przymuśił go Świekier jego / iż śię wróćiwƺy zoſtał tám ná noc.
Wſtał potym bárzo ráno dniá piątego chcąc iść : ále mówił Oćiec oney dźiewki / pośilń proƺę ſerce twoje : y zábáwili śię áż śię dźień náchylił / á jedli obá ſpołu.
Wſtał tedy on mąż áby ƺedł ſam y założnicá jego / y ſługá jego : któremu rzekł Świekier jego / Oćiec oney dźiewki : Oto śię już dźień náchylił ku wiecżorowi / przenocujćieƺ tu proƺę / oto zchodźi dźień / przenocujże tu / á bądź dobrey myśli : á jutro ráno wypráwićie śię w drogę ſwą / y pójdźieƺ do przybytku twego.
Tedy on mąż niechćiał zoſtáć ná noc / ále wſtał y odƺedł / á przyƺedł áż ku Jebus / ( które jeſt Jeruzalem ) májąc z ſobą dwu Oſłów z brzemiony / y założnicę ſwoję.
A gdy byli bliſko Jebus / á dźień śię już bárzo náchylił / tedy rzekł ſługá Pánu ſwemu : pódź proƺę / á wſtąpmy do tego miáſtá Jebuzejcżyków / y przenocujmy w nim.
Któremu odpowiedźiał Pán jego : Nie wſtępujmy do miáſtá cudzoźiemców / które nie <i>jeſt</i> z Synów Izráelſkich / ále idźmy áż do Gábáá.
Nád to rzekł do ſługi ſwego : Pódź / ábyſmy przyƺli ná jedno z tych miejſc / y przenocowáli álbo w Gábáá álbo w Rámá.
A minąwƺy poƺli : y záƺło im ſłońce u Gábáá / które jeſt <i>pokolenia</i> Benjáminowego.
Y událi śię tám / áby wƺedƺy przenocowáli w Gábáá : á <i>gdy</i> wƺedł / uśiadł ná ulicy w mieśćie / przeto że nie był ktoby je przyjął w dom / y przenocował.
A oto mąż ſtáry ƺedł od roboty ſwojey z polá wiecżór : ( A ten mąż <i>był</i> z góry Efrájim / będąc przychodniem w Gábáá : ále ludźie miejſcá onego byli Synowie Jemini. )
Ten podnióżƺy ocży ſwych / ujrzał mężá onego podróżnego ná ulicy miáſtá / y rzekł do niego ſtárzec : dokąd idźieƺ / y z kądeś przyƺedł?
Któremu on odpowiedźiał : Idźiemy z Betlehem Judá áż ku ſtronie góry Efrájimowey z kądem jeſt : bom chodźił do Betlehem Judſkiego : á <i>teraz</i> idę do domu PAńſkiego : ále niemáƺ nikogo coby mię przyjął w dom :
Choć y plewy y śiáno mam dla oſłów náƺych / tákże chleb y wino <i>mam</i> dla śiebie / y dla ſłużebnice twey / y dla ſługi który jeſt ze <i>mną</i> ſługą twojim : niemam niedoſtátku z żadney rzecży.
Tedy mu rzekł on mąż ſtáry : Nie fráſuj śię / cżegoćkolwiek nie doſtánie / to ja opátrzę : tylko ná ulicy nie zoſtawaj przez noc.
W wiódł go tedy do domu ſwego / y dał oſłom obrok : potym umywƺy nogi ſwoje / jedli y pili.
A gdy rozweſelili ſerce ſwoje / oto mężowie miáſtá tego / mężowie niepobożni / obſtąpili dom kołácąc we drzwi / y rzekli do goſpodarzá domu onego / do mężá ſtárego / mówiąc : wywiedź mężá który wƺedł w dom twój / ábyſmy go poználi.
A wyƺedƺy do nich on mąż / goſpodarż domu / rzekł im : nie ták bráćia moji : Nie cżyńćie proƺę tey złośći : gdyż wƺedł ten mąż do domu mego / nie cżyńćież tey ſproſnośći.
Oto córká mojá pánná / y założnicá jego / wywiodę je záraz / że je obelżyćie / á ucżynićie z nimi / co śię wam będźie dobrego zdáło : tylko mężowi temu nie cżyńćie tey zelżywośći.
Ale nie chćieli oni mężowie ſłucháć głoſu jego : przetoż wźiąwƺy on mąż założnicę ſwoję wywiódł ją do nich ná dwór : y poználi ją / á cżynili jey gwałt przez cáłą noc / áż do záránku : á potym puśćili ją gdy wſchodźiłá zorza.
A przyƺedƺy oná niewiáſtá ná świtániu / upádłá u drzwi domu onegoż mężá gdźie był Pán jey / áż śię rozedniáło.
Potym wſtawƺy Pán jey ráno / otworzył drzwi u domu / y wyƺedł chcąc iść w drogę ſwoję : á oto oná niewiáſtá założnicá jego / leżáłá u drzwi domu / á ręce jey <i>były</i> ná progu.
Y rzekł do niey : wſtań á pódźmy : ále nic nie odpowiedźiáłá : wźiąwƺy ją tedy ná Oſłá / wſtał on mąż / y ƺedł do miejſcá ſwego.
Tám przyƺedƺy w dom ſwój / porwał miecż / á zdjąwƺy założnicę ſwoję / rozrąbał ją y z kosćiámi jey ná dwánaśćie ƺtuk / y rozeſłał ją po wƺyſtkich gránicách Izráelſkich.
A ktokolwiek to widźiał / mówił : Nigdy śię to nie ſtáło / áni co tákowego widźiano od onego dniá / jáko wyƺli Synowie Izráelſcy z źiemie Egipſkiey / áż do tego dniá. Uważajćież to z pilnośćią / á radźćie / y mówćie o tym.
