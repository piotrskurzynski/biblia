Y śpiewáłá Deborá y Bárák / Syn Abinoemów dniá onego / mówiąc :
Dla pomſty ucżynioney w Izráelu / á iż śię <i>ná to</i> dobrowolnie lud ofiárował / błogoſławćie PAná.
Słuchájćie Królowie / bierzćie w uƺy Kśiążętá / ja / ja PANU záśpiewam / śpiewáć będę PANU Bogu Izráelſkiemu.
PAnie! gdyś wyƺedł z Seir / <i>á</i> przechodźiłeś przez pole Edom / źiemiá śię wzruƺyłá / niebá też kropiły / á obłoki wydawáły wody.
Góry śię rozpłynęły od oblicża PAńſkiego / á <i>góra</i> Synáj od oblicża PANA Bogá Izráelſkiego.
Zá dni Sámgárá Syná Anátowego / <i>y</i> zá dni Jáele záginęły śćieƺki / á którzy ƺli w drogę / chodźili śćieƺkámi krzywymi.
Zpuſtoƺáły wśi w Izráelu / zpuſtoƺáły / áżem powſtáłá <i>ja</i> Deborá / áżem powſtáłá mátką w Izráelu.
Gdy <i>Izráel</i> obierał ſobie Bogi nowe / tedy bywáłá wojná w bramách : tarcży <i>jednák</i> nie było widáć / áni drzewcá miedzy cżterdźieſtą tyśięcy w Izráelu.
Serce moje <i>nákłonione</i> do Kśiążąt Izráelſkich : Ochotni z ludu / błogoſławćież PAná.
Którzy jeźdźićie ná Oślicách białych / y záśiadaćie ná ſądách / y którzy chodźićie po drogách / rozmawiajćie z ſobą.
Że <i>ućichł</i> trzaſk ſtrzelców / miedzy miejſcámi gdźie czerpáją wodę : tám niech opowiádáją ſpráwiedliwośći PAńſkie / ſpráwiedliwośći we wśiách jego w Izráelu : tedy zſtąpi do bram lud PAńſki.
Powſtań / powſtań Deboro : powſtań / powſtań / á záśpiewaj pieśń : powſtań Báráku / á pojimaj więźnie twoje Synu Abinoemów.
Teraz pánowáć będźie potłoczony nád możnymi z ludu : PAN dopomógł mi pánowáć nád mocárżámi.
Z Efrájimá <i>wyƺedł</i> korzeń ich przećiw Amálekowi / zá tobą Benjámin miedzy ludem twojim / z Máchiru wyƺli zakonodawcy / á z Zábuloná piſárze.
Kśiążętá też Iſáſchár <i>były</i> z Deborą : Iſáſchár też jáko y Bárák w dolinę poſłan jeſt pieƺo : <i>Ale</i> w dźiale Rubenitów <i>byli</i> ludźie wyſokich myśli :
Cżemuś śiedźiał miedzy dwiemá oborámi ſłuchájąc wrzaſku trzód? w dźiale Rubenitów <i>byli ludźie</i> wyſokich myśli.
Gáláád zá Jordanem odpocżynął : á Dán przecż śię báwił okrętámi? Aſer <i>cżemu</i> śiedźiał ná brzegu morſkim / á w ſkáłách ſwojich mieƺkał?
Zábulon <i>jeſt</i> lud który wydał duƺę ſwą ná śmierć : tákże y Neftálim : <i>á to</i> ná wyſokich polách.
Przyƺli Królowie / walcżyli : ná ten cżás walcżyli Królowie Chánánejſcy w Tánách / u wód Mágeddo : <i>jednák</i> korzyśći śrebrá nie odnieśli.
Z niebá walcżono : gwiazdy z miejsc ſwojich walcżyły z Siſárą.
Potok Ciſon porwał je / potok Kedumim / potok Ciſon : Podeptáłáś o duƺo mojá możne.
Tedy śię popádáły kopytá końſkie od wielkiego tąpánia mocárzów jego :
Przeklinajćie Meroz : rzekł Anjoł PAńſki : przeklinajćie przeklinájąc obywátele jego : ábowiem nie przyƺli ná rátunek PANU / ná rátunek PANU z mocárzámi.
Błogoſłáwiona miedzy niewiáſtámi Jáel / żoná Heberá Cynejcżyká : nád niewiáſty w namiećie <i>mieƺkájące</i> błogoſłáwiona będźie.
Prośił o wodę / <i>á oná</i> mleká dáłá : á ná przyſtáwce Kśiążęcey przynioſłá máſłá.
<i>Lewą</i> rękę ſwą do goźdźiá zćiągnęłá / á práwicę ſwoję do młotá kowálſkiego : y uderzyłá Siſárę / przebiłá głowę jego / y przeráźiłá / y przekłółá ſkronie jego.
U nóg jey ſkurcżył śię / padł / leżał : u nóg jey ſkurcżył śię / padł : kędy śię ſkurcżył / tám upadł zábity.
Oknem wyglądáłá / á wołáłá mátká Siſáry przez kratę : przecżże omięƺkiwa wróćić śię wóz jego? przecż śię nie ſpieƺą nogi woźników jego?
Przedniejƺe y mędrƺe <i>niewiáſty</i> odpowiedźiáły / jáko też y ſámá ſobie odpowiádáłá :
Snadź tráfili ná łup / y dźielą go : Pánienká jedná álbo dwie doſtáną śię mężowi jednemu : łupy rozlicżnych barw oddawáją Siſárze : á łupy pſtro háftowáne / y łupy pſtro z obu ſtron tkáne / doſtáwáją śię ná ƺyję łupy biorących.
Ták niechaj zginą wƺyſcy nieprzyjaćiele twoji PAnie : A ći którzy ćiebie miłują / niech będą jáko ſłońce / gdy wſchodźi w mocy ſwojey. Y byłá w pokoju źiemiá przez czterdźieśći lat.
