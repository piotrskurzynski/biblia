Tedy odƺedł Abimelech Syn Jerobáálów do Sychem / do bráćij mátki ſwey / y mówił do nich / y do wƺyſtkiego narodu domu Ojcá mátki ſwey / á rzekł :
Mówćie proƺę / gdźieby ſłyƺeli wƺyſcy przełożeni Sychem : Co wam lepƺego / áby nád wámi pánowáło śiedmdźieśiąt mężów / wƺyſcy Synowie Jerobáálowi : cżyli żeby pánował nád wámi mąż jeden? Wżdy pámiętajćie żem ja kość wáƺá / y ćiáło wáƺe.
Tedy mówili bráćia mátki jego o nim / gdźie ſłyƺeli wƺyſcy przełożeni Sychem / wƺyſtkie te ſłowá / y nákłoniło śię ſerce ich zá Abimelechem / bo rzekli : brát náƺ jeſt.
Y dáli mu śiedmdźieśiąt śrebrników z domu Báálberit / y náprzyjmował zá nie Abimelech ludźi lekkomyślnych / y tułáczów / którzy chodźili zá nim.
Y przyƺedł w dom Ojcá ſwego do Efrá / á pobił bráćią ſwą / Syny Jerobáálowe / śiedmdźieśiąt mężów ná jednym kámieniu : tylko zoſtał Jotám / Syn Jerobáálów namniejƺy / iż śię był ſkrył.
Y zebráli śię wƺyſcy mężowie Sychem / y wƺyſtek dom Mello / á ƺedƺy obráli Abimelechá Królem / ná równinie / kędy ſtał ſłup w Sychem.
Co gdy powiedźiano Jotámowi / ƺedƺy ſtánął ná wierzchu góry Gáryzim / á podnióżƺy głos ſwój wołał / y rzekł im : poſłuchajćie mię mężowie Sychem / á was <i>też</i> Bóg uſłyƺy.
Zeƺły śię drzewá / áby pomázáły nád ſobą Królá / y rzekły do Oliwnego drzewá : króluj nád námi.
Którym odpowiedźiáło Oliwne drzewo : Izali opuƺczę tłuſtość moję / przez którą ucżcżony bywa Bóg y ludźie / á pójdę ábym wyſtáwione było nád drzewy?
Rzekły potym drzewá do figowego drzewá : pódź ty / króluj nád námi.
Którym odpowiedźiáło figowe drzewo : Yzali opuƺczę ſłodkość moję / y owoc mój wyborny / á pójdę / ábym wyſtáwione było nád drzewy?
Potym rzekły drzewá do máćice winney : pódź ty / króluj nád námi.
Tedy im odpowiedźiáłá máćicá : Izali opuƺczę moƺcż mój który uweſelá Bogá y ludźie / á pójdę ábym wyſtáwioną byłá nád drzewá.
Y rzekły wƺyſtkie drzewá do Oſtu : pódź ty / króluj nád námi.
Tedy odpowiedźiał Oſet drzewom : Jeſli wy prawdźiwie chcećie pomázáć mię zá królá nád ſobą / pódźćież á odpoczywajćie pod ćieniem mojim : á jeſliż nie / niech wynidźie ogień z Oſtu / á ſpali Cedry Libáńſkie.
A ták teraz jeſliśćie prawdźiwie á ƺczyrze ucżynili / obrawƺy ſobie Królem Abimelechá : á jeſliśćie śię dobrze obeƺli z Jerobáálem / y z domem jego / á jeſliśćie podług dobrodźiejſtw ręki jego ucżynili z nim?
( Abowiem / walcżył Oćiec mój zá was / y podał duƺę ſwą w niebeſpiecżeńſtwo / áby was wyrwał z ręki Mádiáńcżyków :
Aleśćie wy powſtáli przećiw domowi Ojcá mego dźiś / y pobiliśćie Syny jego / Śiedmdźieśiąt mężów ná jednym kámieniu / y obráliśćie Królem Abimelechá / Syná ſłużebnice jego / nád mężámi Sychem / iż brátem wáƺym jeſt. )
Jeſliśćie prawdźiwie á ƺczyrze obeƺli śię z Jerobáálem / y z domem jego / dniá tego / weſelćież śię z Abimelechá / á on niech śię też weſeli z was.
Ale jeſliż nie / niechajże wynidźie ogień z Abimelechá / á pożrze męże Sychem / y dom Mello : niechajże też wynidźie ogień od mężów Sychem / y z domu Mello / á pożrze Abimelechá.
Tedy ućiekł Jotám : á ućiekƺy poƺedł do Beer / y mieƺkał tám / bojąc śię Abimelechá brátá ſwego.
A ták pánował Abimelech nád Izráelem przez trzy látá.
Y posłał Bóg Duchá złego / miedzy Abimelechá / y miedzy męże Sychemſkie / á złamáli wiárę mężowie Sychem Abimelechowi.
Aby śię zemƺczono krzywdy śiedmdźieśiąći Synów Jerobáálowych / áby krew ich przyƺłá ná Abimelechá brátá ich / który je pobił : y ná męże Sychemſkie / którzy zmocnili ręce jego / áby pobił bráćią ſwoję.
Y záſádźili śię nań mężowie Sychemſcy ná wierzchu gór : á rozbijáli káżdego / który jedno ƺedł tám tą drogą. Y powiedźiano to Abimelechowi.
Nád to przyƺedł Gáál Syn Obedów / y bráćia jego / y przyƺedł do Sychem / á poufáli mu mężowie Sychem.
A wyƺedƺy ná polá zbieráli winá ſwoje / y tłocżyli weſeląc śię : á wƺedƺy w dom Bogów ſwojich / jedli y pili / á złorzecżyli Abimelechowi.
Tedy rzekł Gáál Syn Obedów : Cóż jeſt Abimelech? y co jeſt Sychem / żebyſmy mu ſłużyli? ázaż nie <i>jeſt</i> Syn Jerobáálów / á Zebul urzędnikiem jego? <i>rácżey</i> ſłużćie mężom Hemorá Ojcá Sychemowego : Bo czemużbyſmy onemu ſłużyć mieli?
O by kto podał ten lud w ręce moje / ábym zprzątnął Abimelechá. Y rzekł Abimelechowi : zbierz ſwoje wojſko á wynidź.
A uſłyƺawƺy Zebul / Przełożony miáſtá onego / ſłowá Gáálá Syná Obedowego / zápalił śię gniew jego.
Y wypráwił poſły do Abimelechá potájemnie / mówiąc : Oto Gáál Syn Obedów / y bráćia jego przyƺli do Sychem / á oto chcą walczyć z miáſtem przećiwko tobie.
A ták teraz wſtań nocą / ty / y lud który jeſt z tobą / á ucżyń záſadzkę w polu.
A ráno gdy ſłońce wznidźie / wſtawƺy / uderzyƺ ná miáſto : á gdy on / y lud który <i>jeſt</i> z nim wynidźie przećiw tobie / ucżyniƺ z nim co będźie chćiáłá ręká twojá.
Tedy wſtawƺy Abimelech / y wƺyſtek lud który z nim był w nocy : záſádźili śię przećiw Sychem ná czterech miejſcách.
A wyƺedƺy Gáál Syn Obedów / ſtánął w ſámey bramie miáſtá : wſtał też y Abimelech / y lud który z nim był z záſadzki.
A widząc Gáál lud / rzekł do Zebulá : Oto lud idźie z wierzchu gór : Któremu odpowiedźiał Zebul : Cień góry ty widźiƺ jákoby ludźie.
Tedy powtóre rzekł Gáál / mówiąc : Oto lud zſtępuje z góry : á jeden huf idźie drogą równiny Meonenim.
Tedy rzekł do niego Zebul : gdźież teraz uſtá twoje / które mówiły / Co jeſt Abimelech / ábyſmy mu ſłużyć mieli? izali to nie ten lud któryś wzgárdźił? wynidź teraz / á walcż przećiw niemu.
A ták wyƺedł Gáál przed mężámi Sychem / á walcżył przećiw Abimelechowi.
Y gonił go Abimelech / gdy przed nim ućiekał / á poległo wiele ránnych / áż do ſámey bramy.
Y zoſtał Abimelech w Arumá : á Zebul wygnał Gáálá z bráćią jego / áby nie mieƺkáli w Sychem.
Ale názájutrz wyƺedł lud w pole y powiedźiano to Abimelechowi :
Wźiąwƺy tedy lud / rozdźielił go ná trzy hufce / y záſádźił śię w polu : á widząc / á oto lud wychodźił z miáſtá / wypadł ná nie / y pobił je.
Bo Abimelech / y hufy które z nim były / uderzyły <i>ná nie,</i> y ſtánęli u ſámey bramy miáſtá / á inne dwá hufy uderzyły ná wƺyſtkie którzy <i>byli</i> w polu / y pobili je.
A Abimelech dobywał miáſtá przez on wƺyſtek dźień / y wźiął je : á lud który w nim <i>był</i> pomordował / á zburzywƺy miáſto / pośiał je ſolą.
A uſłyƺawƺy wƺyſcy mężowie którzy byli ná wieży Sychem / przyƺli do twierdze domu Bogá Berit.
Y opowiedźiano Abimelechowi / że śię tám zgromádźili / wƺyſcy mężowie wieże Sychem :
Tedy wƺedł Abimelech ná górę Sálmon / on y wƺyſtek lud który był z nim / á nábrawƺy z ſobą śiekier / ućiął gáłąź z drzewá / á wźiąwƺy ją włożył ná rámioná ſwoje / y rzekł do ludu który z nim był : Co widźićie żem ucżynił / prętko ucżyńćie <i>ták</i> jáko ja.
A ták / ućiąwƺy z onego wƺyſtkiego ludu káżdy gáłąź ſwoję / ƺli zá Abimelechem / á ſkłádli je około twierdze / y ſpalili nimi twierdzą ogniem : y poginęło tám wƺyſtkich mężów wieże Sychemſkiey około tyśiącá mężów y niewiaſt.
Potym ƺedł Abimelech do Tebes á położywƺy śię przećiwko Tebes / dobył go.
Ale wieżá byłá mocna w pośrzodku miáſtá / ná którą ućiekli wƺyſcy mężowie / y niewiáſty / y wƺyſcy przedniejƺy miáſtá : á zámknąwƺy ją zá ſobą / weƺli ná dách wieże.
Tedy przyƺedł Abimelech áż do ſámey wieże / y dobywał jey / á ſtánąwƺy u ſámych drzwi wieże / chćiał ją ſpalić ogniem.
Miedzy tym zrzućiłá niewiáſtá niektóra ƺtukę kámieniá od żarn ná głowę Abimelechowę / y rozbiłá wierzch głowy jego :
Który ná tych miaſt záwoławƺy pácholiká co nośił broń jego / rzekł do niego : Dobądź miecżá twego / á zábij mię : by ſnadź nie rzecżono o mnie : Niewiáſtá go zábiłá : á ták przebił go pácholik jego / y umárł.
A widząc mężowie Izráelſcy iż umárł Abimelech / rozeƺli śię káżdy do miejſcá ſwego.
Y oddał Bóg ono złe Abimelechowi / które ucżynił nád Ojcem ſwojim / zábiwƺy śiedmdźieśiąt bráćij ſwey :
Y wƺyſtko złe mężów Sychem obróćił Bóg ná głowę ich : á przyƺło ná nie przeklęctwo Jotámá Syná Jerobáálowego.
