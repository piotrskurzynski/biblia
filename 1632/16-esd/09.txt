Potem dniá dwudźieſtego y cżwártego tegoż mieśiącá zgromádźili śię ſynowie Izráelſcy / y pośćili w worzech / y poſypáli śię prochem.
A odłącżyło śię naśienie Izráelſkie od wƺyſtkich cudzoźiemców / á ſtánąwƺy wyznáwáli grzechy ſwe y niepráwośći ojców ſwych.
Y powſtáli ná miejſcách ſwych / á cżytáli kśięgi zakonu Páná / Bogá ſwego / cżtery kroć przez dźień / y cżtery kroć wyznáwáli á kłániáli śię Pánu Bogu ſwemu.
Zátem ſtánęli ná ſtopniách Lewitów Jeſuá y Báni / Kádmiel / Sebánijáƺ / Bunni / Serebijáƺ / Báni / Chenáni / á wołáli głoſem wielkim do Páná Bogá ſwego.
Y mówili Lewitowie Jeſuá / y Kádmiel / Báni / Háſábnejáƺ / Serebijáƺ / Odyjáƺ / Sebánijáƺ / Petáchyjáƺ : Wſtáńćie / błogoſłáwćie Pánu / Bogu wáƺemu / od wieku áż ná wieki / á niech błogoſłáwią imieniowi twojimu chwálebnemu y wywyżƺonemu nád wƺelkie błogoſłáwieńſtwo y chwałę.
Ty / Pánie! ſám / ty ſám / tyś ucżynił niebo / niebá niebios / y wƺyſtko wojſko ich / źiemię / y wƺyſtko co jeſt ná niej / morzá / y wƺyſtko / co w nich jeſt / á ty ożywiáƺ to wƺyſtko ; á wojſká niebieſkie tobie śię kłániáją.
Tyś jeſt / Pánie Boże! któryś wybrał Abrámá / á wywiodłeś go z Ur Cháldejſkiego / y dáłeś mu imię Abráhám.
Y ználázłeś ſerce jego wierne przed oblicżem twojim / y ucżyniłeś z nim przymierze / że dáƺ źiemię Chánánejcżyká / Hetejcżyká / Amorejcżyká / y Ferezejcżyká / y Jebuzejcżyká / y Giergiezejcżyká / że ją dáƺ náśieniu jego / y źiśćiłeś ſłowá twoje ; boś ty ſpráwiedliwy.
Wejrzáłeś zájiſte ná utrápienie ojców náƺych w Egipćie / á wołánie ich wyſłucháłeś nád morzem Cżerwonem.
A pokázywáłeś znáki y cudá ná Fáráonie / y ná wƺyſtkich ſługách jego / y ná wƺyſtkim ludu źiemi jego ; boś poznał / że ſobie hárdźie poſtępowáli przećiwko nim / y ucżyniłeś ſobie imię / jáko śię to dźiś pokázuje.
Y rozdźieliłeś morze przed nimi / á przeƺli przez pośrodek morzá po ſuƺy ; á tych / którzy ich gonili / wrzućiłeś w głębokośći / jáko kámień w wody gwáłtowne.
A w ſłupie obłokowym prowádźiłeś ich we dnie / á w ſłupie ogniſtym w nocy / ábyś im oświecał drogę / którą iść mieli.
Potemeś ná górę Synáj zſtąpił / y mówiłeś do nich z niebá / á dáłeś im ſądy práwe / y zakony práwdźiwe / uſtáwy y rozkázánia dobre.
Y ſábát twój święty oznájmiłeś im / á przykázánia y uſtáwy / y zakon wydáłeś im przez Mojżeƺá / ſługę twego.
Dáłeś im też chleb w głodźie ich z niebá / y wodęś im z ſkáły wywiódł w prágnieniu ich / á rozkázáłeś im / áby ƺli / y pośiedli źiemię / o którąś podnióſł rękę ſwą / że im ją dáƺ.
Ale oni y ojcowie náśi hárdźie ſobie pocżynáli / y zátwárdźili kárk ſwój / y nie ſłucháli rozkázánia twego.
Owƺem nie chćieli ſłucháć / áni wſpomnieli ná cudá twoje / któreś cżynił / przy nich ; ále zátwárdźiwƺy kárk ſwój poſtánowili ſobie wodzá / chcąc śię wróćić w niewolę ſwoję w uporze ſwoim. Lecż ty / o Boże miłośćiwy / łáſkáwy / y miłośierny / nieſkwápliwy / y wielkiego miłośierdźiá! nie opuśćiłeś ich.
Náwet gdy ſobie ucżynili ćielcá ulánego / á mówili : Ten jeſt Bóg twój / który ćię wywiódł z źiemi Egipſkiej / y dopuśćili śię bluźnierſtw wielkich /
Ty jednák dla litośći twojich wielkich nie opuśćiłeś ich ná puƺcży ; ſłup obłokowy nie odſtąpił od nich we dnie / prowádząc ich w drodze / áni ſłup ogniſty w nocy / oświecájąc ich / y drogę / którą iść mieli.
Nádto duchá twojego dobrego dáłeś im / áby ich ucżył / y mánny twojey nie odjąłeś od uſt ich / y wodę dáłeś im w prágnieniu ich.
A ták przez cżterdźieśći lat żywiłeś ich ná puƺcży ; ná nicżem im nie ſchodźiło / ƺáty ich nie zwiotƺáły / y nogi ich nie nápuchły.
Y podáłeś im króleſtwá y narody / któreś rozegnał po kątách / ták / że pośiedli źiemię Sehonową / y źiemię królá Heſebońſkiego / y źiemię Ogá / królá Báſáńſkiego.
A ſynów ich rozmnożyłeś jáko gwiázdy niebieſkie / y wwiodłeś ich do źiemi / o którejś máwiał ojcom ich / że wnijdą y pośiędą ją.
Bo przyƺedƺy ſynowie ich pośiedli tę źiemię / gdyś poniżył przed nimi obywáteli oney źiemi / Chánánejcżyków / y podáłeś ich w ręce ich / y królów ich / y narody oney źiemi / áby śię z nimi obchodźili według woli ſwojey.
Pobráli tedy miáſtá obronne / y źiemię tłuſtą / y pośiedli domy pełne wƺelkich dóbr / ſtudnie wykopáne / winnice / oliwnice / y drzew rodzájnych bárzo wiele ; á jedli / y byli náſyceni / y otyli / y opływáli w rozkoƺy z dobroći twojey wielkiej.
Ale gdy ćię rozdráźnili / y ſtálić śię odpornymi / zárzućiwƺy zakon twój w tył ſwój / á proroków twojich pobili / którzy śię oświádcżáli przed nimi / áby ich náwróćili do ćiebie / y dopuƺcżáli śię bluźnierſtwá wielkiego :
Podáłeś ich w ręce nieprzyjáćiołom ich / którzy ich trápili. A gdy cżáſu utrapienia ſwego wołáli do ćiebie / tyś ich z niebá wyſłuchał / á według litośći twojich wielkich dáwáłeś im wybáwićieli / którzy ich wybáwiáli z rąk nieprzyjáćiół ich.
Wtem / gdy trochę odpocżęli / znowu cżynili złość przed twárzą twoją ; przetoż opuśćiłeś ich w ręce nieprzyjáćiół ich / áby pánowáli nád nimi. Lecż gdy śię znowu náwróćili / á wołáli do ćiebie / tyś ich z niebá wyſłuchał / y wybáwiłeś ich według litośći twojich przez wiele cżáſów.
Y oświádcżáłeś śię przed nimi / ábyś ich náwróćił do zakonu ſwego ; ále oni ſobie hárdźie pocżynáli / á nieſłucháli przykázáń twojich / owƺem przećiw ſądom twoim grzeƺyli / które gdyby cżłowiek cżynił / żyłby przez nie ; ále uchylájąc rámion ſwych / kárk ſwój zátwárdźili / y nie ſłucháli.
Wƺákżeś ty im folgował przez wiele lat / oświádcżájąc śię przed nimi Duchem twym przez proroków twojich ; á gdy nie ſłucháli / podáłeś ich w ręce narodów onych źiem.
Ale dla litośći twojich wielkich nie dáłeś ich ná wytrácenie / ániś ich opuśćił ; boś ty Bóg łáſkáwy y miłośierny.
Teraz tedy o Boże náƺ / Boże wielki / możny / y ſtráƺny! który ſtrzeżeƺ przymierza y miłośierdźiá / niech nie będźie máłe przed tobą káżde utrápienie / które przyƺło ná nas / ná królów náƺych / ná kśiążąt náƺych / y ná kápłanów náƺych / y ná proroków náƺych / y ná ojców náƺych / y ná wƺyſtek lud twój / ode dni królów Aſſyryjſkich áż do dniá tego ;
Acżeś ty jeſt ſpráwiedliwy we wƺyſtkim tym / co przyƺło ná nas ; boś ſpráwiedliwie ucżynił / á myśmy niezbożnie cżynili.
Y królowie náśi / kśiążętá náśi / kápłáni náśi / y ojcowie náśi nie pełnili zakonu twego / y nie przeſtrzegáli przykázáń twojich / y świádectw twych / któremiś śię oświádcżał przed nimi.
Bo oni w króleſtwie ſwem y w dobroći twojey wielkiej / którąś im pokázał / y w źiemi przeſtronney y tłuſtey / którąś im był dał / nie ſłużyli tobie / áni śię odwróćili od ſpráw złych ſwojich.
Oto myśmy dźiś niewolnikámi / y źiemiá / którąś dał ojcom náƺym / áby jedli owoc jey / y dobrá jey / otośmy niewolnikámi w niej.
Już urodzáje ſwoje obfite wydáje królom / któreś poſtánowił nád námi dla grzechów náƺych ; pánują nád ćiáły náƺemi / y nád bydłem náƺem według woli ſwej / ták / żeśmy w wielkiem ućiśnieniu.
Wƺákże w tym wƺyſtkim cżynimy mocne przymierze / y zápiſujemy je / które piecżętują kśiążętá náśi / Lewitowie náśi / y kápłáni náśi.
