Potym wyƺedł Dawid z támtąd y uƺedł do jáſkini Adullám : co gdy uſłyƺeli bráćia jego / y wƺyſtek dom Ojcá jego / przyƺli tám do niego.
Y zebráli śię do niego wƺyſcy którzy byli utrapieni / y wƺyſcy którzy byli dłużni / y wƺyſcy którzykolwiek byli w gorzkośći ſercá / y był nád nimi Kśiążęćiem ; á było ich z nim około cżterech ſet mężów.
Y poƺedł Dawid z támtąd do Másfá Moábſkiego / y rzekł do Królá Moábſkiego ; Niech śię przyprowádźi / proƺę / Oćiec mój / y mátká mojá / <i>áby mieƺkáli</i> z wámi / áż śię dowiem co ucżyni Bóg ze mną.
A ták przywiódł je przed Królá Moábſkiego / y mieƺkáli z nim po wƺyſtkie dni których był Dawid ná onym Zamku.
Rzekł potym Gád Prorok do Dawidá : Nie mieƺkaj <i>więcey</i> ná tym Zamku ; idź á wróć śię do źiemie Judſkiey. Tedy poƺedł Dawid / á przyƺedł do láſá Háret.
A uſłyƺawƺy Saul / że śię pojáwił Dawid / y mężowie którzy byli z nim / ( bo Saul mieƺkał w Gábáá pod gájem w Rámáćie / májąc włócżnią ſwoję w rękách ſwych : á wƺyſcy ſłudzy jego ſtali przed nim. )
Rzekł tedy Saul do ſług ſwych / którzy ſtali przed nim ; Słuchajćie proƺę Synowie Jemini : Izaż wam wƺyſtkim da Syn Iſájego role y winnice / á wƺyſtkich was pocżyni Półkownikámi y Rotmiſtrzámi?
Żeśćie śię zprzyśięgli wy wƺyſcy przećiwko mnie / á niemáƺ ktoby mi objáwił? gdyż śię zbuntował y Syn mój / z Synem Iſájego / á niemáƺ ktoby śię mnie użalił miedzy wámi / á oznájmił mi / iż podburzył Syn mój ſługę mego przećiwko mnie / áby cżyhał ná mię jáko śię to dźiś <i>okázuje</i>.
Tedy odpowiedźiał Doeg Edomcżyk / który też ſtał z ſługámi Saulowymi / y rzekł ; Widźiałem Syná Iſájego / gdy przyƺedł do Noby do Achimelechá Syná Achitobowego :
Który śię oń rádźił PAná : y dał mu żywnośći / dał mu też y miecż Goliatá Filiſtyńcżyká.
A ták poſłał Król áby przyzwano Achimelechá / Syná Achitobowego / Kápłaná / y wƺyſtkiego domu Ojcá jego / Kápłanów którzy <i>byli</i> w Nobe : Y przyƺli oni wƺyſcy do Królá.
Tedy rzekł Saul : Słuchaj teraz Synu Achitobów : A on rzekł : Owom ja Pánie mój.
Y rzekł do niego Saul : Cżemuśćie śię zprzyśięgli przećiwko mnie / ty / y Syn Iſájego / gdyś mu dał chleb y miecż / á rádźiłeś śię oń Bogá / áby powſtał przećiwko mnie / cżyhájąc ná mię / jáko śię to dźiś <i>okázuje</i> :
Y odpowiedźiał Achimelech Królowi / á rzekł ; Y któż jeſt ták wierny / miedzy wƺyſtkimi ſługámi twojimi / jáko Dawid ; <i>który jeſt</i> y źięćiem królewſkim / y idźie zá rozkazániem twojim / á <i>jeſt</i> zacnym w domu twojim?
Azaż dźiś pocżąłem śię oń rádźić Bogá? Uchowaj mię Boże! Niech nie wkłada Król ná ſługę twego nic tákiego / áni ná wƺyſtek dom Ojcá mego : bo nie wiedźiał ſługá twój o tym wƺyſtkim namniejƺey rzecży.
Y rzekł Król : Śmierćią umrzeƺ Achimelechu / ty / y wƺyſtek dom Ojcá twego.
Przytym rzekł Król ſługom którzy ſtali przed nim : Obróććie śię / á pobijćie Kápłany PAńſkie ; bo też ręká ich <i>jeſt</i> z Dawidem : gdyż wiedząc że on ućiekał / nie oznájmili mi. Ale ſłudzy królewſcy nie chćieli podnieść ręki ſwey áni śię rzućić ná Kápłany PAńſkie.
Przetoż rzekł Król do Doegá : Obróć śię ty / á rzuć śię ná Kápłany. A ták obróćiwƺy śię Doeg Edomcżyk / rzućił śię ná Kápłany : y zábił onegoż dniá ośmdźieśiąt y pięć mężów / którzy nośili Efod lniáne.
Nobe też miáſto Kápłańſkie wyśiekł oſtrzem miecżá / od mężá áż do niewiáſty / od máłego áż do ſącego : y Woły / Oſły / y Owce <i>wyśiekł</i> oſtrzem miecżá.
Uƺedł tylko Syn jeden Achimelechá / Syná Achitobowego ; á imię jego Abiátár / y ućiekł do Dawidá.
Tedy oznájmił Abiátár Dawidowi / że pobił Saul Kápłany PAńſkie.
Y rzekł Dawid Abiátárowi : Wiedźiałem onegoż dniá / gdyż tám <i>był</i> Doeg Edomcżyk / że pewnie oznájmić miał Saulowi : Jam jeſt przycżyną <i>śmierći</i> wƺyſtkich duƺ domu Ojcá twego.
Zoſtańże przy mnie / nie bój śię. Bo ktoby ƺukał duƺe mojey / będźie ƺukał duƺe twojey / ále ty będźieƺ ochroniony przy mnie.
