Y ſtáło śię / gdy przeſtał mówić do Saulá / że duƺá Jonátánowá ſpoiłá śię z duƺą Dawidową / y umiłował go Jonátán / jáko duƺę ſwoję.
Y wźiął go Saul dniá onego / áni mu dopuśćił / żeby śię wrácał do domu ojcá ſwego.
A ták ucżynił Jonátán z Dawidem przymierze / bo go miłował ják duƺę ſwoję.
A zdjąwƺy z śiebie Jonátán płáƺcż / który miał ná ſobie / dał go Dawidowi / y ƺáty ſwe / áż do miecżá ſwego / y áż do łuku ſwego / y áż do páſá ſwego rycerſkiego.
Y wychádzał Dawid do wƺyſtkiego / do cżego go kolwiek poſyłał Saul / á roztropnie śię ſpráwował ; y przełożył go Saul nád rycerſtwem / y był wdźięcżnym w ocżách wƺyſtkiego ludu / tákże y w ocżách ſług Saulowych.
Y ſtáło śię / gdy śię wrácáli / á Dawid śię też wrácał od poráżki Filiſtynów / że wyƺły niewiáſty ze wƺyſtkich miaſt Izráelſkich / śpiewájąc y grájąc przećiwko Saulowi królowi z bębnámi / z weſelem / y z gęślámi.
A śpiewájąc ná przemiány one niewiáſty / gráły y mówiły : Poráźił Saul ſwój tyśiąc / ále Dawid ſwojich dźieśięć tyśięcy.
Y rozgniewał śię Saul bárzo / bo śię nie podobáły w ocżách jego te ſłowá ; y rzekł : Przywłáƺcżyli Dawidowi dźieśięć tyśięcy / á mnie przywłáƺcżyli tyśiąc : á cżegoż mu niedoſtáje / jedno króleſtwá?
Przetoż Saul krzywo pátrzył ná Dawidá od onegoż dniá y ná potem.
Y ſtáło śię drugiego dniá / że przypádł Duch Boży zły ná Saulá / y prorokował w pośrzodku domu / á Dawid grał ręką ſwoją dniá onego / jáko y przed tym / á Saul miał włócżnią w ręce ſwej.
Y ćiſnął Saul włócżnią / mówiąc : Przebiję Dawidá áż ku śćiánie ; ále śię uchylił Dawid przed nim po dwá kroć.
Y bał śię Saul Dawidá / przeto że Pán był z nim / á od Saulá odſtąpił.
Y odpráwił go Saul od śiebie / á ucżynił go hetmánem nád tyśiącem / y wychádzał á wchádzał przed ludem.
Owá Dawid we wƺyſtkich drogách ſwych roztropnie śię ſpráwował ; bo Pán był z nim.
Co gdy widźiał Saul / iż ták bárzo roztropnie ſobie pocżynał / bał śię go.
Ale wƺyſtek Izráel y Judá miłował Dawidá ; bo on wychádzał y wchádzał przed nimi.
Y rzekł Saul do Dawidá : Oto / córkę moję ſtárƺą Merob dam ći zá żonę / jedno bądź mężem mocnym / y odpráwuj wojny Páńſkie ; bo ták Saul ſobie mówił : Niech nie będźie ręká mojá ná nim / ále niech będźie ná nim ręká Filiſtynów.
Tedy rzekł Dawid do Saulá : Któżem ja? álbo co zá ſtán mój? álbo co zá dom ojcá mego w Izráelu / żebym był źięćiem królewſkim?
Y ſtáło śię / gdy przyƺedł cżás / którego miáłá być dáná Merob / córká Saulowá / Dawidowi / że oná dáná jeſt Adryjelowi Meholátyckiemu zá żonę.
Ale śię rozmiłowáłá Michol / córká Saulowá / Dawidá ; co gdy powiedźiáno Saulowi / miło mu to było.
Y rzekł Saul : Dam mu ją / żeby mu byłá śidłem / á żeby byłá ná nim ręká Filiſtynów. Przetoż rzekł Saul do Dawidá : Po drugie będźieƺ źięćiem moim dźiśiáj.
Tedy rozkázał Saul ſługom ſwoim : Rzecżćie do Dawidá potájemnie / mówiąc : Oto / upodobał ćię ſobie król / y wƺyſcy ſłudzy jego miłują ćię / á ták teraz bądź źięćiem królewſkim.
A gdy mówili ſłudzy Saulowi w uƺy Dawidowe te ſłowá / odpowiedźiał Dawid : Cży śię wam máłá rzecż widźi / być źięćiem królewſkim / gdyżem ja jeſt mężem ubogim y podłym?
Tedy ſłudzy Saulowi oznájmili mu / mówiąc : Ták powiedźiał Dawid.
Y rzekł Saul : Ták powiedzćie Dawidowi : Nie dbáć król o wiáno / tylko chce mieć ſto nieobrzezek Filiſtyńſkich / áby śię ſtáłá pomſtá nád nieprzyjáćiołmi królewſkimi ; bo Saul myślił / jákoby Dawidá podáć w ręce Filiſtynom.
Tedy ſłudzy jego powiedźieli te ſłowá Dawidowi ; y ſpodobáło śię to Dawidowi / áby zoſtał źięćiem królewſkim ; á jeƺcże śię nie były wypełniły dni one.
Wſtał tedy Dawid / y poƺedł / on y mężowie jego / y zábił z Filiſtynów dwieśćie mężów / y przynióſł Dawid nieobrzeſki ich / y oddáno je ſpełná królowi / áby był źięćiem królewſkim. A ták dał mu Saul Michol / córkę ſwą zá żonę.
A widząc Saul / y bácżąc / że Pán był z Dawidem / á iż Michol / córká jego / miłowáłá go /
Tym więcey Saul obáwiał śię Dawidá / y ſtał śię Saul nieprzyjáćielem Dawidowi po wƺyſtkie dni.
Y wpádáły kśiążętá Filiſtyńſkie do źiemi. A kiedykolwiek wpádáły / roztropniej ſobie pocżynał Dawid nád wƺyſtkie ſługi Saulowe ; przetoż ſłáwne było imię jego bárzo.
