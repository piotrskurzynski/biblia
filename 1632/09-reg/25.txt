W tym umárł Sámuel : A zebrawƺy śię wƺyſcy Izráelcżycy / płákáli go / y pogrzebli go w domu jego / w Rámáćie : Tedy wſtawƺy Dawid / poƺedł ná puƺcżą Fárán.
A <i>był niektóry</i> mąż w Máon / który miał májętność ná Kármelu / á on mąż był możny bárzo / májąc owiec trzy tyśiące / á tyśiąc kóz : Y tráfiło śię że ſtrzygł owce ſwoje ná Kármelu.
A <i>było</i> imię mężá onego Nábál / á imię żony jego Abigáil : która niewiáſtá byłá mądra / y piękna : ále mąż jey był nieużyty / y złych poſtępków / á był <i>z narodu</i> Kálebowego.
A uſłyƺawƺy Dawid ná puƺcży / iż Nábál ſtrzygł owce ſwoje :
Poſłał dźieśiąćiu ſług / y rzekł im : Idźćie do Kármelu / á przyƺedƺy do Nábálá / pozdrówćie go imieniem mojim ſpokojnie :
A mówćie ták : Żyj : á niech będźie tobie pokój / y domowi twemu pokój / y wƺyſtkiemu co maƺ pokój.
A teraz ſłyƺałem / że maƺ te coć Owce ſtrzygą : ( á páſterze twoji bywáli z námi : niebyliſmy im przykrymi / y nic im nie zginęło po wƺyſtkie dni / których byli w Kármelu :
Spytaj ſług twojich / á powiedząć : ) Przetoż / niech najdą ſłudzy łáſkę w ocżu twojich / gdyżeſmy w dobry dźień przyƺli : Daj proƺę cokolwiek znajdźie ręká twojá ſługom twojim / y Synowi twemu Dawidowi.
A ták przyƺli ſłudzy Dawidowi / y mówili do Nábálá wƺyſtkie one ſłowá imieniem Dawidowym / y przeſtáli.
A odpowiádájąc Nábál ſługom Dawidowym / rzekł : Cóż jeſt Dawid? á co zacż Syn Iſájego? dźiś śię námnożyło ſług / którzy ućiekáją od pánów ſwojich.
Y wezmęż ja chleb mój / y wodę moję / y mięſo <i>bydłá</i> mego / którem pobił dla tych którzy ſtrzygą owce moje / á dam mężom / których nie znam z kąd ſą?
A obróćiwƺy śię ſłudzy Dawidowi w drogę ſwoję / wróćili śię / y przyƺli / á powiedźieli mu wƺyſtkie te ſłowá.
Tedy rzekł Dawid mężom ſwym ; Przypáƺćie káżdy miecż ſwój : Y przypáſáli káżdy miecż ſwój : przypáſał też y Dawid miecż ſwój : y ƺło zá Dawidem około cżterech ſet mężów / á dwie śćie zoſtáło przy rzecżách.
Ale Abigáili / żenie Nábálowey / oznájmił <i>to</i> jeden cżeládnik z ſług <i>Nábálowych</i> / mówiąc : Oto przyſłał Dawid poſły z puƺcży / áby błogoſłáwili Pánu náƺemu / lecż je on zfukał.
A mężowie ći / dobrzy nam byli bárzo / y nie przykrzyli śię nam : nic nam nie zginęło po wƺyſtkie dni / pókiſmy z nimi chodźili / będąc ná polu :
Miáſto muru byli nam / ták w nocy jáko we dnie / po wƺyſtkie dni pókiſmy przy nich trzody páśli.
Przetoż teraz obacż / á rozmyśl śię co maƺ cżynić? boć już gotowe nieƺcżęśćie ná Páná náƺego / y ná wƺyſtek dom jego : lecż on jeſt cżłowiekiem niezbożnym / że z nim trudno mówić.
Poſpieƺyłá śię tedy Abigáil / y wźięłá dwie śćie chlebá / y dwie łagwi winá / y pięć owiec opráwnych / y pięć miar práżmá / y ſto wiązanek rozynków / y dwie śćie funtów fig / á włożyłá to ná oſły :
Y rzekłá ſługom ſwojim ; Idźćie przede mną á ja pojádę zá wámi : ále mężowi ſwemu Nábálowi nie oznájmiłá.
Y ſtáło śię że jádąc ná ośle / y zjeżdżájąc głębią góry : oto też Dawid y mężowie jego zjeżdżáli przećiwko niey / y potkáłá śię z nimi.
( A Dawid był rzekł : Záprawdę dármom ſtrzegł wƺyſtkiego / co ten miał ná puƺcży / że nic nie zginęło ze wƺyſtkiego co ma ; bo mi oddał złym zá dobre :
To niech ucżyni Bóg nieprzyjaćiołom Dawidowym / y to niech przycżyni / jeſli co do záránia zoſtáwię ze wƺyſtkiego co ma / áż do namniejƺego ƺcżenięćiá. )
Tedy ujrzawƺy Abigáil Dawidá / poſpieƺyłá śię / y zśiádłá z Oſłá ; y upádłá przed Dawidem ná oblicże ſwoje / y ukłoniłá śię áż do źieme.
A padƺy do nóg jego / mówiłá : Niech będźie ná mnie Pánie mój tá niepráwość / á niech mówi / proƺę / ſłużebnicá twojá / do uƺu twojich / y poſłuchaj ſłów ſłużebnice twojey.
Niech śię / proƺę / nieobraża Pan mój ná mężá tego bezbożnego Nábálá / gdyż według imienia ſwego / tákim jeſt : Nábál <i>jeſt</i> imię jego / y głupſtwo jeſt przy nim : álemći ja ſłużebnicá twojá nie widźiáłá ſług Páná mego / któreś był poſłał.
Przetoż teraz Pánie mój / żywie PAN / żywie y duƺá twojá / że ćię záwśćiągnął PAN / ábyś nie ƺedł ná <i>rozlanie</i> krwie / y żeby śię nie mśćiłá ręká twojá : á teraz niech będą jáko Nábál nieprzyjaćiele twoji / y którzy ƺukáją złego Pánu memu.
Teraz tedy to błogoſłáwieńſtwo / które przynioſłá ſłużebnicá twojá Pánu ſwemu / niech będźie dáne ſługom / którzy chodzą zá Pánem mojim.
Przepuść proƺę wyſtępek ſłużebnicy twojey ; gdyż zápewne ucżyni PAN Pánu memu dom trwáły : ponieważ walki PAńſkie Pán mój odpráwuje / á złość nie jeſt ználeźioná w tobie áż do tąd.
A choćby powſtał cżłowiek coby ćię prześládował / y ƺukał duƺe twojey ; tedy będźie duƺá Páná mego / záchowáná w wiązance żywiących / u PANA Bogá twego : lecż duƺę nieprzyjaćiół twojich <i>Bóg</i> jáko z proce wyrzući.
A gdy ucżyni PAN Pánu memu wƺyſtko co mówił dobrego o tobie / á zlećić ábyś był wodzem nád Izráelem :
Tedy nie będźieć to ku záchwianiu / áni ku uráźie ſercá Páná mego ; jáko gdyby rozlał krew niewinną / y gdyby śię ſam pomśćił Pán mój : Gdy tedy dobrze ucżyni PAN Pánu memu / wſpomniƺ ná ſłużebnicę twoję.
Y rzekł Dawid do Abigáile ; Błogoſłáwiony PAN Bóg Izráelſki / który ćię dźiś poſłał przećiwko mnie :
Błogoſłáwiona wymowá twojá ; y błogoſłáwionaś ty / któraś mię záwśćiągnęłá dźiśia / żem nie ƺedł ná rozlanie krwie / á żem śię ſam nie mśćił krzywdy ſwojey.
A záprawdę / żywie PAN Bóg Izráelſki / który mię záwśćiągnął / ábymći nic złego nie ucżynił : Bo gdybyś śię byłá nie poſpieƺyłá / á nie zájácháłá mi drogi / nie zoſtáłoby było Nábálowi áż do świtánia / y namniejƺego ƺcżenięćiá.
A ták przyjął Dawid z ręki jey co mu byłá przynioſłá / y rzekł do niey : Idź w pokoju do domu twego / otom uſłuchał głoſu twego / y przyjąłem ćię łáſkáwie.
Tedy śię wróćiłá Abigáil do Nábálá ; á on miał ucżtę w domu ſwojim / jáko ucżtę Królewſką : á ſerce Nábálowe było weſołe w nim ; á był pijány bárzo : y nie oznájmiłá mu namniejƺey rzecży / áż do poránku.
Ale názájutrz gdy wytrzeźwiał Nábál z winá / oznájmiłá mu żoná jego te rzecży : y zmartwiáło w nim ſerce jego / y ſtał śię <i>jáko</i> kámień.
A gdy wyƺło jákoby dźieśięć dni / uderzył PAN Nábálá / y umárł.
A uſłyƺawƺy Dawid / iż umárł Nábál / rzekł : Błogoſłáwiony PAN / który śię pomśćił poháńbienia mego nád Nábálem / á ſługę ſwego zátrzymał od złego / á złość Nábálowę obróćił PAN ná głowę jego : Tedy poſłał Dawid wſkázując do Abigájile / że ją ſobie chce wźiąć zá żonę.
Y przyƺli ſłudzy Dawidowi do Abigájle / do Kármelu / y rzekli do niey / mówiąc ; Dawid poſłał nas do ćiebie / áby ćię ſobie wźiął zá żonę.
Która wſtawƺy pokłoniłá śię oblicżem do źiemie / y rzekłá : Oto ſłużebnicá twojá / niech będźie ſługą / áby umywáłá nogi ſług Páná mego.
Przetoż poſpieƺywƺy śię wſtáłá Abigáil y wśiádłá ná Oſłá / z piąćią pánienek ſwymi / które zá nią chodźiły : y ták jácháłá zá poſłámi Dawidowymi : á byłá mu zá żonę.
Ale y Achinoámę wźiął Dawid z Jezreel / y były mu te dwie zá żony.
Abowiem Saul dał był Michol córkę ſwoję / żonę Dawidowę / Fáltemu / Synowi Láiſowemu / który był z Gállim.
