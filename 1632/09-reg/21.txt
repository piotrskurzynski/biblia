Potym przyƺedł Dawid do Noby / do Achimelechá Kápłaná : á zlękƺy śię Achimelech / wyƺedł przećiwko Dawidowi : y rzekł mu ; Przecżżeś ty ſam / á niemáƺ nikogo z tobą?
Y odpowiedźiał Dawid Achimelechowi Kápłanowi : Rozkazał mi Król nieco / y rzekł do mnie : Niech nikt nie wie tego / po co ćię poſyłam / y comći zlećił ; Przetożem ſługi zoſtáwił ná pewnym miejſcu.
A ták teraz / maƺli co przy ręku twojich / <i>áby</i> z pięćioro chlebá / daj do ręki mojey / álbo cokolwiek najdźieƺ.
Y odpowiedźiał Kápłan Dawidowi / y rzekł : niemam chlebá poſpolitego przy ręce mojey / tylko chleb poświęcony : jeſli śię tylko wſtrzymáli ſłudzy od niewiaſt.
Tedy odpowiedźiał Dawid Kápłanowi / y rzekł mu : Zájiſte niewiáſty oddalone były od nas od wcżorájƺego / y dźiś trzećiego dniá / gdym wyƺedł : Przetoż były nacżynia ſług święte : Ale jeſli tá drogá zmázána jeſt / wƺákże y tá dźiśia poświęconá będźie w nacżyniách.
A ták dał mu Kápłan <i>chleby</i> poświęcone : ábowiem nie było tám chlebá / tylko chleby pokłádne / które były odjęte od oblicżnośći PAńſkiey / áby położono chleby ćiepłe onegoż dniá / którego one odjęte były.
A był tám mąż z ſług Saulowych onego dniá / zábáwiony przed PAnem : którego imię Doeg Edomcżyk / námożniejƺy z páſterzów / które miał Saul.
Y rzekł Dawid do Achimelechá : A niemaƺże tu przy ręce ſwey włócżnie / álbo miecżá? bom áni miecżá mego / áni żadney broni mojey nie wźiął w rękę moję / gdyż ſłowo Królewſkie przynagláło.
Tedy rzekł Kápłan ; miecż Goliatá Filiſtyńcżyká / któregoś zábił w dolinie Elá / oto jeſt uwiniony w ſukno / zá Efodem : jeſli ten chceƺ ſobie wźiąć / weźmi : bo tu inƺego niemáƺ oprócż tego. Y rzekł Dawid : Niemáƺ podobnego temu ; daj mi go.
A ták wſtał Dawid / y ućiekł dniá onego przed Saulem / y przyƺedł do Achiſá Królá Getſkiego.
Tedy rzekli ſłudzy Achiſowi do niego ; Izali nie ten jeſt Dawid / Król źiemie? izali nie temu śpiewano w hufcách / mówiąc : Poráźił Saul ſwój tyśiąc / á Dawid ſwojich dźieśięć tyśięcy.
Y złożył Dawid ſłowá te do ſercá ſwego / á bał śię bárzo Achiſá Królá Getſkiego ;
Przetoż zmienił obycżáje ſwoje przed ocżymá ich / á cżynił śię ƺalonym w ręku ich / y kryśleł ná drzwiách bramy / y puƺcżał śliny ná brodę ſwoję.
Tedy rzekł Achis do ſług ſwojich ; Otośćie widźieli cżłowieká ƺalonego / cżemużeśćie go przywiedli do mnie?
Niedoſtawa mi ƺalonych / żeśćie przywiedli tego / áby ƺálał przede mną? tenże ma wniść do domu mego?
