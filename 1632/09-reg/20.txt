Ale Dawid ućiekƺy z Nájotu / który jeſt w Rámáćie / przyƺedł / y mówił przed Jonátánem : Cóżem ucżynił? co zá niepráwość mojá? y co zá grzech mój przećiw ojcu twemu / że ƺuká duƺy mojey?
Który mu odpowiedźiał : Boże uchowáj! nie umrzeƺ ; oto nie cżyni oćiec mój nic wielkiego álbo máłego / áż mi pierwej oznájmi ; ázáżby táić miał oćiec mój przedemną y tego? Nie ucżyni tego.
A nádto przyśiągł Dawid / rzekƺy : Wie zájiſte oćiec twój / żem ználázł łáſkę w ocżách twojich / y myśli : Niech o tym niewie Jonátán / by śię ſnadź nie fráſował ; y owƺem jáko żywy Pán / żywá y duƺá twojá / że tylko krok jeden jeſt miedzy mną / y miedzy śmierćią.
Y odpowiedźiał Jonátán Dawidowi : Co mi kolwiek rzecże duƺá twojá / ucżynięć.
Tedy rzekł Dawid do Jonátáná : Oto / nów mieśiącá jutro / á jam zwykł śiádáć z królem przy ſtole ; puść mię tedy / że śię ſkryję ná polu áż do wiecżorá trzećiego dniá.
A jeſliby śię pilnie pytał o mnie oćiec twój / rzecżeƺ : Prośił mię bárzo Dawid / áby ƺedł do Betlehem / miáſtá ſwego ; bo tám ofiárę urocżyſtą ma ſpráwowáć wƺyſtká rodźiná jego.
Jeſli ták rzecże : Dobrze / pokój będźie ſłudze twemu ; ále jeſli śię rozgniewá / wiedz / iż śię dopełniłá złość jego.
Przetoż ucżyń miłośierdźie nád ſługą twoim / gdyżeś w przymierze Páńſkie przywiódł z ſobą ſługę twego ; á jeſli we mnie jeſt niepráwość / ty mię zábij ; á do ojcá twego przecżbyś mię miał wodźić?
Y rzekł Jonátán : Boże ćię tego uchowáj ; bo jeſli śię pewnie dowiem / że śię dopełniłá złość ojcá mego / áby przyƺłá przećiw tobie / izálibym ći tego nie oznájmił?
Y rzekł Dawid do Jonátáná : Któż mi oznájmi / jeſliżeć co odpowie oćiec twój przykrego?
Odpowiedźiał Jonátán Dawidowi : Pójdź / á wynijdźmy ná pole. Y wyƺli obáj ná pole.
Tedy rzekł Jonátán do Dawidá : Pán / Bóg Izráelſki / ( ſkoro śię wywiem o woli ojcá mego o tym cżáśie jutro / álbo dniá trzećiego / á będźie co dobrego o Dawidźie / á jeſli zárázem nie poślę do ćiebie / y nie oznájmięć / )
To niech ucżyni Pán / Bóg Izráelſki / mówię / Jonátánowi / y to niech przycżyni. A jeſliże będźie chćiał oćiec mój przywieść złe ná ćię / y toć objáwię / y puƺcżę ćię / ábyś ƺedł w pokoju / á niech będźie Pán z tobą / jáko był z ojcem moim.
Tákże y ty / będęli żyw / y ty mówię ucżyniƺ ze mną miłośierdźie Páńſkie ; á choćbym y umárł /
Przećię nie oddáliƺ miłośierdźiá twego od domu mego áż ná wieki / áni gdy wykorzeni Pán nieprzyjáćioły Dawidowe wƺyſtkie z źiemi.
Y ucżynił Jonátán przymierze z domem Dawidowym / mówiąc : Niech tego ƺuká Pán z ręki nieprzyjáćiół Dawidowych.
Nádto jeƺcże Jonátán przyśiągł Dawidowi przez miłość / którą go miłował ; bo jáko miłował duƺę ſwoję / ták go też miłował.
Y rzekł do niego Jonátán : Jutro nów mieśiącá / á będą śię pytáć o tobie / ponieważ próżne będźie miejſce twoje.
Przetoż przez trzy dni będźieƺ śię ukrywał / y zſtąpiƺ prędko / á przyjdźieƺ ná miejſce / gdźieś śię był ukrył / gdy byłá ſpráwá o tobie / á będźieƺ śiedźiał u kámieniá Ezel.
A ja wyſtrzelę trzy ſtrzáły po bok jego / zmierzájąc ſobie do celu.
A potem poślę chłopcá / mówiąc mu : Idź / nájdźij ſtrzáły. A jeſli rzekę chłopcu : Owo ſtrzáły zá tobą ſám bliżey / przynieś je / tedy przyjdź ; bo maƺ pokój / y nie ſtánieć śię nic złego / jáko żywy Pán.
Ale jeſliż rzekę chłopcu : Oto ſtrzáły przed tobą tám dálej ; idź / bo ćię wypuśćił Pán.
A tego / o cżemeśmy mówili ja y ty / tego Pán świádkiem będźie miedzy mną á miedzy tobą áż ná wieki.
A ták ſkrył śię Dawid w polu. A gdy przyƺedł nów mieśiącá / śiádł król do ſtołu / áby jádł.
A gdy uśiádł król ná ſtolicy ſwojey według zwycżáju / ná ſtolicy przy śćiánie / powſtał Jonátán ; y śiádł Abner podle Saulá / á zoſtáło próżne miejſce Dawidowe.
Lecż nie rzekł Saul nic onego dniá / bo myślał : Przydáło mu śię podobno coś / lub jeſt cżyſtym lub niecżyſtym.
A gdy było názájutrz dniá wtórego po nowiu mieśiącá / było záś próżne miejſce Dawidowe. Y rzekł Saul do Jonátáná / ſyná ſwego : Cżemuż nie przyƺedł ſyn Iſájego / áni wcżoráj / áni dźiś do ſtołu?
Odpowiedźiał Jonátán Saulowi : Uśilnie mię prośił Dawid / áby ƺedł do Betlehem ;
Y mówił : Puść mię proƺę / bo ſpráwuje ofiárę rodźiná náƺá w mieśćie ; támże mię wezwał brát mój. A ták teraz jeſlim ználázł łáſkę w ocżách twojich / pójdę proƺę / y oglądám bráćią moję ; dla tegoć nie przyƺedł do ſtołu królewſkiego.
Y zápálił śię gniewem Saul ná Jonátáná / y rzekł mu : Synu złośliwy / á uporney mátki / ázaż nie wiem / iżeś ſobie obrał ſyná Iſájego ku zelżywośći twojey / y ku poháńbieniu y ſromoćie mátki twojey?
Bo po wƺyſtkie dni / których ſyn Iſájego będźie żył ná źiemi / nie będźieƺ umocniony / ty y króleſtwo twoje ; á ták teraz poślij / á przywiedź go do mnie / bo jeſt godźien śmierći.
Tedy odpowiedźiał Jonátán Saulowi / ojcu ſwemu / y rzekł do niego : Przecż ma umrzeć? cóż ucżynił?
Y ćiſnął Saul włócżnią ná niego / áby go przebił. Tedy poznał Jonátán / że koniecżnie oćiec jego umyślił zábić Dawidá.
Y wſtał Jonátán od ſtołu z wielkim gniewem / y nie jádł dniá wtórego po nowiu mieśiącá chlebá ; bo śię záfráſował o Dawidá / á iż go zelżył oćiec jego.
A ráno wyƺedł Jonátán ná pole według cżáſu poſtánowionego z Dawidem / y chłopiec máły z nim.
Y rzekł do chłopcá ſwego : Bież / ƺukáj prędko ſtrzał / które ja wyſtrzelę. Tedy chłopiec bieżał ; á on wyſtrzelił ſtrzáły dálej przedeń.
A gdy przyƺedł chłopiec áż ná miejſce ſtrzáły / którą był wyſtrzelił Jonátán / záwołał Jonátán zá chłopcem / y rzekł : Azaż ſtrzáłá nie jeſt zá tobą tám dálej?
Y wołał Jonátán zá chłopcem : Spieƺ śię co nájrychlej / nie ſtój. Tedy zebráwƺy chłopiec Jonátáná ſtrzáły / przyƺedł do páná ſwego.
( Ale chłopiec nic nie wiedźiał / tylko Jonátán y Dawid wiedźieli / co śię dźiáło. )
Y dał Jonátán oręż ſwój chłopcu / który z nim był / y rzekł mu : Idź / odnieś do miáſtá.
A gdy odƺedł chłopiec / Dawid wſtał od ſtrony południowej / y upádƺy twárzą ſwoją ná źiemię / ukłonił śię po trzy kroć / y pocáłowáwƺy jeden drugiego / płákáli poſpołu ; ále Dawid obfićiej.
Y rzekł Jonátán do Dawidá : Idź w pokoju ; á to / cośmy ſobie obáj przyśięgli przez imię Páńſkie / mówiąc : Pán niech będźie miedzy mną y miedzy tobą / y miedzy náśieniem mojim / y miedzy náśieniem twojim świádkiem áż ná wieki / trzymáć będźiemy.
A ták wſtáwƺy Dawid odƺedł / á Jonátán wƺedł do miáſtá.
