Ale Dawid ućiekƺy z Nájotu <i>który jeſt</i> w Rámáćie / przyƺedł / y mówił przed Jonátánem : Cóżem ucżynił? co zá niepráwość mojá? y co zá grzech mój przećiw Ojcu twemu / że ƺuka duƺe mojey?
Który mu odpowiedźiał : Boże uchowaj! nie umrzeƺ : oto nie cżyni Oćiec mój nic wielkiego álbo máłego / áż mi pierwey oznájmi : ázażby tájić miał Oćiec mój przede mną y tego? nie ucżyni tego!
A nád to przyśiągł Dawid ( rzekƺy : Wie zájiſte Oćiec twój / żem ználazł łáſkę w ocżách twojich : Y myśli : Niech o tym nie wie Jonátán / by śię ſnadź niefráſował : ) y owƺem żywie PAN żywie y duƺá twojá / że tylko krok jeden jeſt miedzy mną / y miedzy śmierćią.
Y odpowiedźiał Jonátán Dawidowi ; Co mi kolwiek rzecże duƺá twojá / ucżynięć.
Tedy rzekł Dawid do Jonátáná : Oto <i>nów</i> mieśiącá jutro / á jam zwykł śiadáć z Królem przy ſtole : puść mię tedy / że śię ſkryję ná polu / áż do wiecżorá trzećiego <i>dniá</i>.
A jeſliby śię pilnie pytał o mnie Oćiec twój rzecżeƺ : prośił mię bárzo Dawid / áby ƺedł do Betlehem / miáſtá ſwego : bo tám ofiárę urocżyſtą <i>ma ſpráwowáć</i> wƺyſtká rodźiná jego.
Jeſli ták rzecże / Dobrze : pokój <i>będźie</i> ſłudze twemu : ále jeſli śię rozgniewa / wiedz yż śię dopełniłá złość jego.
Przetoż ucżyń miłośierdźie nád ſługą twojim / gdyżeś w przymierze PAńſkie przywiódł z ſobą ſługę twego : á jeſli we mnie jeſt niepráwość / ty mię zábij. A do Ojcá twego przecżbyś mię miał wodźić?
Y rzekł Jonátán : Boże ćię tego uchowaj : bo jeſli śię pewnie dowiem / że śię dopełniłá złość Ojcá mego / áby przyƺłá przećiw tobie / yzalibymći tego nie oznájmił?
Y rzekł Dawid do Jonátáná : Któż mi oznájmi / jeſliżeć co odpowie Oćiec twój przykrego?
Odpowiedźiał Jonátán Dawidowi : Pódź á wynidźmy ná pole : y wyƺli obá ná pole.
Tedy rzekł Jonátán do Dawidá : PAN Bóg Izráelſki / ( ſkoro śię wywiem o woli Ojcá mego o tym cżáśie jutro / <i>álbo</i> dniá trzećiego / á będźie co dobrego o Dawidźie / á jeſli zárázem nie poślę do ćiebie / y nie oznájmięć )
To niech ucżyni PAN / <i>Bóg Izráelſki mówię,</i> Jonátánowi / y to niech przycżyni : á jeſliże będźie chćiał Oćiec mój przywieść złe ná ćię : y toć objáwię / y puƺcżę ćię / ábyś ƺedł w pokoju / á niech będźie PAN z tobą / jáko był z Ojcem mojim.
Tákże y ty / będęli żyw / y ty <i>mówię</i> ucżyniƺ ze mną miłośierdźie PAńſkie : á choćbym <i>y</i> umárł /
Przećię nie oddaliƺ miłośierdźia twego od domu mego / áż ná wieki : áni / gdy wykorzeni PAN nieprzyjaćioły Dawidowe wƺyſtkie z źiemie.
Y ucżynił Jonátán przymierze z domem Dawidowym <i>mówiąc</i> : Niech <i>tego</i> ƺuka PAN z ręki nieprzyjaćiół Dawidowych.
Nád to jeƺcże Jonátán przyśiągł Dawidowi przez miłość / którą go miłował : bo <i>jáko</i> miłował duƺę ſwoję / <i>ták</i> go też miłował.
Y rzekł do niego Jonátán : jutro nów mieśiącá / á będą śię pytáć o tobie / ponieważ próżne będźie miejſce twoje.
Przetoż przez trzy dni będźieƺ śię ukrywał / y zſtąpiƺ prętko / á przydźieƺ ná miejſce / gdźieś śię był ukrył gdy byłá ſpráwá o tobie : á będźieƺ śiedźiał u kámieniá Ezel :
A ja wyſtrzelę trzy ſtrzały pobok jego / zmierzájąc ſobie do celu.
A potym poślę chłopcá / <i>mówiąc mu :</i> Idź najdźi ſtrzały : A jeſli rzekę chłopcu ; owo ſtrzały zá tobą ſám bliżey / przynieś je : tedy przydź / bo maƺ pokój / y <i>nie ſtánieć śię</i> nic złego / żywie PAN.
Ale jeſliż rzekę chłopcu / oto ſtrzały przed tobą tám dáliey : idź / bo ćię wypuśćił PAN.
A tego / o cżemeſmy mówili ja y ty / tego PAN <i>świádkiem będźie</i> miedzy mną á miedzy tobą / áż ná wieki.
A ták ſkrył śię Dawid w polu. A gdy przyƺedł nów mieśiącá / śiadł Król do ſtołu áby jadł.
A gdy uśiadł Król ná ſtolicy ſwojey według zwycżáju / ná ſtolicy przy śćienie : powſtał Jonátán : y śiadł Abner podle Saulá : á zoſtáło próżne miejſce Dawidowe.
Lecż nie rzekł Saul nic onego dniá : bo myślił : przydáło mu śię podobno coś / lub jeſt cżyſtym / lub niecżyſtym.
A gdy było názájutrz / dniá wtórego / po nowiu mieśiącá : było záś próżne miejſce Dawidowe : y rzekł Saul do Jonátáná Syná ſwego : Cżemuż nieprzyƺedł Syn Iſájego / áni wcżorá / áni dźiś / do ſtołu?
Odpowiedźiał Jonátán Saulowi : Uśilnie mię prośił Dawid / áby <i>ƺedł</i> do Betlehem.
Y mówił : Puść mię proƺę / bo ſpráwuje Ofiárę rodźiná náƺá w mieśćie : támże mię wezwał brát mój : á ták teraz jeſlim ználazł łáſkę w ocżách twojich / pójdę proƺę y oglądam bráćią moję : dla tegoć nieprzyƺedł do ſtołu Królewſkiego.
Y zápalił śię gniewem Saul ná Jonátáná / y rzekł mu : Synu złośliwy / á uporney <i>mátki,</i> áza niewiem yżeś ſobie obrał Syná Iſájego ku zelżywośći twojey / y ku poháńbieniu y ſromoćie mátki twojey :
Bo po wƺyſtkie dni / których Syn Iſájego będźie żył ná źiemi / nie będźieƺ umocniony / ty / y Króleſtwo twoje : á ták teraz pośli / á przywiedź go do mnie / bo jeſt godźien śmierći.
Tedy odpowiedźiał Jonátán Saulowi Ojcu ſwemu : Y rzekł do niego : Przecż ma umrzeć? cóż ucżynił?
Y ćiſnął Saul włócżnią ná niego / áby go przebił. Tedy poznał Jonátán / że koniecżnie Oćiec jego umyślił zábić Dawidá.
Y wſtał Jonátán od ſtołu z wielkim gniewem / y nie jadł dniá wtórego po nowiu mieśiącá chlebá ; bo śię záfráſował o Dawidá : á yż go zelżył Oćiec jego.
A ráno wyƺedł Jonátán ná pole według cżáſu poſtánowionego z Dawidem / y chłopiec máły z nim.
Y rzekł do chłopcá ſwego ; Bież ƺukaj prętko ſtrzał / które ja wyſtrzelę : tedy chłopiec bieżał ; á on wyſtrzelił ſtrzały dáliey przedeń.
A gdy przyƺedł chłopiec áż ná miejſce ſtrzały / którą był wyſtrzelił Jonátán ; záwołał Jonátán zá chłopcem / y rzekł : ázaż ſtrzałá nie jeſt zá tobą tám dáliey?
Y wołał Jonátán zá chłopcem ; Spieƺ śię co narychliey / nie ſtój : tedy zebrawƺy chłopiec Jonátáná ſtrzały / przyƺedł do Páná ſwego.
( Ale chłopiec nic nie wiedźiał / tylko Jonátán y Dawid wiedźieli co śię dźiało. )
Y dał Jonátán oręże ſwe chłopcu który z nim był / y rzekł mu ; Idź odnieś do miáſtá.
A gdy odƺedł chłopiec / Dawid wſtał od ſtrony południowey : y upadƺy twarzą ſwoją ná źiemię / ukłonił śię po trzy kroć / y pocáłowawƺy jeden drugiego / płákáli poſpołu : ále Dawid obfićiey :
Y rzekł Jonátán do Dawidá : Idź w pokoju : <i>á to,</i> coſmy ſobie obá przyśięgli / przez imię PAńſkie / mówiąc / PAN niech będźie miedzy mną / y miedzy tobą / y miedzy naśieniem mojim / y miedzy naśieniem twojim <i>świadkiem</i> áż ná wieki / <i>trzymáć będźiemy.</i> 
A ták wſtawƺy <i>Dawid</i> odƺedł : á Jonátán wƺedł do miáſtá.
