Saul tedy pierwƺego roku królowánia ſwego ( bo tylko dwá látá królował nád Izráelem / )
Wybrał ſobie trzy tyśiące z Izráelá ; y byli przy Saulu dwá tyśiące w Máchmás / y ná górze Betel / á tyśiąc był z Jonátánem w Gábáá Benjámin / á oſtátek ludu rozpuśćił / káżdego do przybytku ſwego.
Tedy Jonátán pobił ſtráż Filiſtyńſką / która byłá w Gábáá / y uſłyƺeli Filiſtynowie. Zátem Saul zátrąbił w trąbę po wƺyſtkiej źiemi / mówiąc : Niech uſłyƺą Hebrejcżycy.
A ták uſłyƺał wƺyſtek Izráel / że powiádáno : Pobił Saul ſtráż Filiſtyńſką / dla cżego też obrzydłym był Izráel miedzy Filiſtyny. Y zwołáno ludu zá Saulem do Gálgál.
Filiſtynowie też zebráli śię / áby wálcżyli z Izráelem / májąc trzydźieśći tyśięcy wozów / y ƺeść tyśięcy jezdnych / á ludu bárzo wiele jáko piáſku / który jeſt ná brzegu morſkim / y ćiągnęli á położyli śię obozem w Máchmás / ná wſchód ſłońcá od Betáwen.
Ale mężowie Izráelſcy widząc / iż byli śćiśnieni / ( bo był ućiśniony lud / ) pokryli śię w jáſkini / y w obronne miejſcá / y w ſkáły / y w wieże / y w jámy.
Niektórzy też Hebrejcżykowie przepráwili śię zá Jordán / do źiemi Gád y Gáláád ; ále Saul jeƺcże pozoſtał był w Gálgál / á wƺyſtek lud potrwożony ƺedł zá nim.
Y cżekał przez śiedm dni według cżáſu zámierzonego od Sámuelá / á gdy nie przyƺedł Sámuel do Gálgál / rozbieżał śię lud od niego.
Tedy rzekł Saul : Przynieśćie do mnie ofiárę cáłopalenia / y ofiáry ſpokojne ; támże ofiárował cáłopálenie.
A gdy dokońcżył ofiáry cáłopalenia / oto Sámuel przyƺedł / y wyƺedł Saul przećiwko niemu / żeby go przywitał.
Y rzekł Sámuel : Cóżeś ucżynił? Odpowiedźiał Saul : Iżem widźiał / że śię rozchodźi lud odemnie / á tyś nie przyƺedł ná cżás náznácżony / Filiſtynowie śię też zebráli do Máchmás /
Tedym rzekł : Oto przypádną Filiſtynowie ná mię w Gálgál / á jam jeƺcże nie ubłágał twárzy Páńſkiej / y ták powáżyłem śię / y ofiárowáłem cáłopálenie.
Y rzekł Sámuel do Saulá : Głupieś ucżynił / nie záchowáłeś przykázánia Páná Bogá twego / któreć rozkázał ; ábowiem terazby był utwierdźił Pán króleſtwo twoje nád Izráelem áż ná wieki.
Ale teraz króleſtwo twoje nie oſtoi śię ; Pán ſobie ználázł mężá według ſercá ſwego / któremu rozkázał Pán / áby był wodzem nád ludem jego / gdyżeś nie záchował / coć przykázał Pán.
Wſtáwƺy tedy Sámuel / poƺedł z Gálgál do Gábáá w Benjámin / y policżył Saul lud / którego śię ználázło przy nim około ƺeśćiu ſet mężów.
Przetoż Saul / y Jonátán / ſyn jego / y lud / który śię ználázł przy nim / zoſtáli w Gábáá w Benjámin / á Filiſtynowie leżeli obozem w Máchmás.
Y wyƺły dla zdobycży z obozu Filiſtyńſkiego trzy hufce : hufiec jeden obróćił śię drogą ku Ofrá do źiemi Saul ;
A drugi hufiec obróćił śię drogą ku Betoron ; trzeći záś hufiec udał śię drogą ku gránicy przyległej dolinie Soboim ku puƺcży.
Ale kowál nie znájdował śię we wƺyſtkiej źiemi Izráelſkiej ; bo byli zábieżeli temu Filiſtynowie / żeby ſnadź Hebrejcżycy nie robili miecżów áni oƺcżepów.
Przetoż chádzał wƺyſtek Izráel do Filiſtynów / oſtrzyć ſobie káżdy lemieƺ ſwój / y motykę ſwoję / y śiekierę ſwoję / y rydel ſwój.
Bo ſtępiáły były oſtrzá lemieƺów / y motyk / y wideł / y śiekier áż do ośćieniá / które było oſtrzyć potrzebá.
Y było pod cżás wojny / że śię nie znájdował miecż / áni oƺcżep w ręku wƺyſtkiego ludu / który był z Saulem / y z Jonátánem ; tylko śię znájdował u Saulá y Jonátáná / ſyná jego.
A ſtráż Filiſtyńſká wyƺłá ná drogę ku Máchmás.
