Y przyƺli Zyfejcżycy do Saulá do Gábáá / á mówili ; Yzali śię nie kryje Dawid ná págórku Háchilá / przećiw Jeſimon.
Ruƺył śię tedy Saul / y ćiągnął ná puƺcżą Zyf / á z nim trzy tyśiące mężów przebránych z Izráelá / áby ƺukał Dawidá ná puƺcży Zyf.
Y położył śię Saul obozem / ná págórku Háchilá / które <i>jeſt</i> przećiw Jeſimon / podle drogi : á Dawid mieƺkał ná puƺcży : y dowiedźiał śię / że przyćiągnął Saul zá nim ná puƺcżą :
Bo poſławƺy Dawid ƺpiegi / dowiedźiał śię / że przyćiągnął Saul zápewne.
Przetoż wſtał Dawid / y przyƺedł áż ku miejſcu / gdźie śię położył obozem Saul : y upátrzył Dawid miejſce gdźie ſpał Saul / y Abner Syn Nerów / Hetman wojſká jego : bo Saul ſpał w oboźie / á lud leżał około niego.
Y odpowiedźiał Dawid / á rzekł do Achimelechá Hettejcżyká / y do Abiſájego Syná Sárwie / brátá Joábowego / mówiąc ; Któż pójdźie ze mną do Saulá / do obozu. Odpowiedźiał Abiſái : Ja z tobą pójdę.
A ták przyƺedł Dawid y Abiſái do ludu w nocy / á oto Saul leżąc ſpał w oboźie / á włócżnia jego byłá utkniona w źiemi u głowy jego : Abner też z ludem leżeli około niego.
Tedy rzekł Abiſái do Dawidá : Zámknął dźiś Bóg nieprzyjaćielá twego w ręce twojey / á teraz niech go przebiję proƺę włócżnią ku źiemi raz / á więcey nie powtórzę.
Ale rzekł Dawid do Abiſájego : Nie zábijaj go : Bo któż zśćiągnąwƺy rękę ſwą ná pomázáńcá PAńſkiego / niewinnym będźie?
Nád to rzekł Dawid : Żyje PAN! że / jeſli go PAN niezábije / álbo dźień jego nie przydźie / áby umárł ; álbo ná wojnę wyjáchawƺy / nie zginie?
<i>Tedy</i> uchowaj mię PAnie / ábym miał zćiągnąć rękę moję ná pomázáńcá PAńſkiego : Ale weźmi / proƺę / włócżnią / która jeſt u głów jego / y kubek od wody / á odejdźmy.
Tedy wźiął Dawid włócżnią / y kubek od wody / który był u głów Saulowych / y odeƺli á nie był ktoby widźiał / áni ktoby wiedźiał / áni ktoby śię ocućił : ále wƺyſcy ſpáli / bo ſen twárdy od PAná przypadł był ná nie.
Y przyƺedł Dawid ná drugą ſtronę / y ſtánął ná wierzchu góry z dáleká : á był plác wielki miedzy nimi.
Y záwołał Dawid ná lud / y ná Abnerá Syná Nerowego / mówiąc ; Nie ozwieƺże śię Abnerze? Y odpowiádájąc Abner rzekł : Któżeś ty / co wołaƺ ná Królá?
Y rzekł Dawid do Abnerá : Azaż ty nie mąż : á któż jákoś ty w Izráelu? przecżżeś tedy nie ſtrzegł Królá Páná twego? bo przyƺedł jeden z ludu chcąc zábić Królá Páná twego.
Nie dobrá to coś ucżynił. Żywie PAN żeśćie winni śmierći / którzyśćie nie ſtrzegli Páná wáƺego / pomázáńcá PAńſkiego : á teraz pátrz kędy jeſt włócżnia Królewſka / y kubek od wody / co był w głowách jego?
Poznał tedy Saul głos Dawidów / y rzekł ; Twójże to głos Synu mój Dawidźie? Odpowiedźiał Dawid : Głos <i>to</i> mój / Królu Pánie mój.
Nád to rzekł : Cżemuż Pán mój prześláduje ſługę ſwego : bo / cóżem ucżynił? á co <i>jeſt</i> złego w ręce mojey?
Przetoż teraz niech poſłucha proƺę Król Pan mój ſłów ſługi ſwego : jeſli ćię PAN pobudźił przećiwko mnie / niech powonia Ofiáry : ále jeſli Synowie ludzcy / przeklęći ſą przed Pánem / którzy mię dźiś wygnáli / ábym nie mieƺkał w dźiedźictwie PAńſkim / <i>jákoby</i> rzekli ; Idź ſłuż Bogom cudzym.
A teraz niech nie będźie wylana krew mojá ná źiemię przed oblicżem PAńſkim : Bo wyƺedł król Izráelſki ƺukáć pchły jedney / jákoby też <i>kto</i> gonił kuropátwę po górách.
Tedy rzekł Saul : Zgrzeƺyłem : wróćże śię Synu mój Dawidźie / boć już nic złego nie ucżynię więcey / ponieważ droga byłá duƺá mojá w ocżách twojich dniá tego : Otom głupie ucżynił / y zbłądźiłem náder bárzo.
A odpowiádájąc Dawid rzekł : Oto włócżnia Królewſka / niech <i>ſám</i> przydźie kto z ſług / á weźmie ją.
A PAN niech odda káżdemu ſpráwiedliwość jego / y wiárę jego : ábowiem podał ćię był PAN dźiś w ręce moje / álem niechćiał zćiągnąć ręki mojey ná pomázáńcá PAńſkiego.
Przetoż / jáko dźiś poważona byłá duƺá twojá w ocżách mojich : ták niech będźie poważona duƺá mojá w ocżách PAńſkich / á niech mię wyrwie <i>PAN</i> ze wƺego ućiſku.
Y rzekł Saul do Dawidá : Błogoſłáwionyś ty Synu mój Dawidźie / ták cżyniąc dokażeƺ / á ták śię zmacniájąc / mocnym będźieƺ : Odƺedł potym Dawid w drogę ſwą / á Saul śię wróćił ná miejſce ſwoje.
