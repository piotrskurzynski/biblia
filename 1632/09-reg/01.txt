Był niektóry mąż z Rámátájm Sofim / z góry Efráim / któremu było imię Elkáná / Syn Jerohámá / Syná Elihu / Syná Tohu / Syná Suf Efrátejcżyká.
A ten miał dwie żenie / imię jedney Anná / á imię drugiey Fenenná / y miáłá Fenenná dźiatki / ále Anná nie miáłá dźiatek.
Y chadzał on mąż z miáſtá ſwego ná káżdy rok / áby chwałę dawał y ofiárował PAnu Zaſtępów w Sylo : gdźie byli dwá Synowie Heli / Ofni y Finees / Kápłani PAńſcy.
A gdy przyƺedł dźień którego ſpráwował ofiáry Elkáná / dał Fenennie żenie ſwey / y wƺyſtkim Synom jey / y córkam jey cżąſtki :
Ale Annie dał jednę cżęść wyborną : Bo Annę miłował / choćiaż był PAN żywot jey zámknął.
Y drażniłá ją bárzo przećiwnicá jey / áby ją tylko rozgniewáłá / dla tego iż zámknął był PAN żywot jey.
To gdy cżynił <i>Elkáná</i> ná káżdy rok / á <i>Anná</i> też chodźiłá do Domu PAńſkiego / ták ją drażniłá <i>przećiwnicá</i> że płákiwáłá / y nie jadáłá.
Rzekł jey tedy Elkáná mąż jey : Anno cżemu płácżeƺ / y cżemu nie jeƺ? á przecż śię ták trapi ſerce twe? izalim ja tobie nie jeſt lepƺy niż dźieśięć ſynów?
Wſtáłá tedy Anná gdy śię nájedli y nápili w Sylo : á Heli Kápłan śiedźiał ná ſtołku u podwojá kośćiołá PAńſkiego.
A oná będąc w gorzkośći ſercá / modliłá śię PAnu / y wielce płákáłá.
Y ucżyniłá ślub / mówiąc : PAnie Zaſtępów / jeſliż wejrzawƺy wejrzyƺ ná utrapienie ſłużebnice twojey / y wſpomniƺ ná mię / á nie zábacżyƺ ſłużebnice twojey / y daƺ ſłużebnicy twey potomſtwo męſkiey płći / tedy je dam PAnu po wƺyſtkie dni żywotá jego / á brzytwá nie poſtoji ná głowie jego.
Y ſtáło śię / gdy przedłużáłá modlitwy przed PAnem / że Heli przypátrował śię uſtam jey.
Ale Anná mówiłá w ſercu ſwym : tylko wárgi jey rucháły śię / ále głoſu jey ſłycháć nie było : y miał ją Heli zá pijáną.
Przetoż rzekł do niej Heli : Długoż będźieƺ pijáną? wytrzeźwi śię z winá twego.
Ale odpowiedźiáłá Anná y rzekłá : nie ták Pánie mój : niewiáſtá utrapionego Duchá jeſtem : ánim winá áni nápoju mocnego nie piłá : álem wylałá duƺę moję przed oblicżem PAńſkim.
Nie rozumiejże o ſłużebnicy twojey jáko o niewieśćie niepobożney / gdyż z wielkiego myślenia y fráſunku mego / mówiłám áż do tąd :
Tedy odpowiedźiał Heli / y rzekł : Idźże w pokoju / á Bóg Izráelſki niechći da proźbę twoję / któreyś żądáłá od niego.
Y rzekłá : Niech znajdźie ſłużebnicá twojá łáſkę przed ocżymá twymi : y odeƺłá niewiáſtá w drogę ſwą / y jádłá : á twarz jey nie byłá więcey <i>ſmętna</i>.
Y wſtáli bárzo ráno / á pokłoniwƺy śię przed PAnem / wróćili śię / y przyƺli do domu ſwego do Rámátá : Tedy poznał Elkáná Annę żonę ſwą / á wſpomniał ná nię PAN.
Y ſtáło śię po wypełnieniu dni / jáko pocżęłá Anná / że porodźiłá Syná / y názwáłá imię jego Sámuel : bo <i>rzekłá,</i> u PAnám go uprośiłá.
Potym ƺedł on mąż Elkáná / y wƺyſtek dom jego / áby oddał PAnu Ofiárę urocżyſtą y ślub ſwój.
Ale Anná nie ƺłá : bo mówiłá mężowi ſwemu. <i>Nie pójdę</i> áż zoſtáwię dźiećiątko / potym odwiodę je / że śię ukaże przed PAnem / y zoſtánie tám záwƺe.
Y rzekł jey Elkáná mąż jey : Ucżyń co <i>jeſt</i> dobrego w ocżách twojich / zoſtań áż go zoſtáwiƺ : tylko niech utwierdźi PAN ſłowo ſwoje. Zoſtáłá tedy niewiáſtá / y kármiłá pierśiámi Syná ſwego / áż go zoſtáwiłá.
A gdy go zoſtáwiłá / przywiodłá go z ſobą ze trzemá ćielcámi / y z jednym Efá mąki / y z łagwią winá / y przywiodłá go do domu PAńſkiego w Sylo : á dźiećię <i>było</i> máłe.
Y zábiwƺy ćielcá przywiedli dźiećię do Heli.
A oná rzekłá : Słuchaj Pánie mój / żywie duƺá twojá Pánie mój : Jam jeſt oná niewiáſtá którám tu ſtałá przy tobie / modląc śię PAnu.
Prośiłám o to dźiećiątko / y dał mi PAN proźbę moję któreym żądáłá od niego.
Przetoż je też ja oddawam PAnu : ná wƺyſtkie dni / których będźie żyło / jeſt oddáne PAnu. Y pokłonili śię tám PAnu.
