Y był mąż <i>z pokolenia</i> Benjámin którego imię było Cyſs / Syn Abielów / Syná Seror / Syná Bechorát / Syná Afiá / Syná mężá Jemini / duży w śiłę.
Ten miał Syná imieniem Saulá / młodźieńcá urodźiwego : á nie <i>było</i> nikogo z Synów Izráelſkich urodźiwƺego nádeń : głową był wyżƺy nád wƺyſtek <i>inny</i> lud.
A zginęły były Oślice Cyſſowi Ojcu Saulowemu : y rzekł Cyſs do Saulá Syná ſwego : Weźmi teraz z ſobą jednego z ſług / á wſtawƺy idź y ƺukaj Oślic.
Tedy on ƺedł przez górę Efráijm / y przeƺedł źiemię Sáliſá / lecż nie ználeźli : Przeƺli tákże źiemię Sálim / á nie <i>ználeźli</i> : Nád to przeƺli y źiemię Jemini / á nie ználeźli.
A przyƺedƺy do źiemie Suf / rzekł Saul ſłudze ſwemu który był z nim : pódź / á wróćmy śię / by ſnadź zániechawƺy Oćiec mój Oślic nie fráſował śię o nas.
Który mu odpowiedźiał : Oto teraz <i>jeſt</i> mąż Boży w tym mieśćie / á mąż <i>to</i> zacny : Cokolwiek powie / wƺyſtko śię ſtawa : przetoż pódźmy tám / ſnadź nam powie o drodze náƺey / którą iść mamy.
Tedy odpowiedźiał Saul ſłudze ſwemu : Więc pójdźiemy : Ale cóż przynieśiemy onemu mężowi? Bo chlebá nieſtáło w ſumkách náƺych / á podarku niemáƺ / którybyſmy przynieśli mężowi Bożemu : cóż mamy?
Tedy ſługá znowu odpowiedźiał Saulowi / y rzekł : Otom ználazł u śiebie cżwartą cżęść ſyklá śrebrnego / którą damy mężowi Bożemu / áby nam oznájmił drogę náƺę.
( Przed tym w Izráelu ták mawiał káżdy / gdy śię ƺedł rádźić Bogá : chodźćie / á pójdźiemy áż do Widzącego : Bo którego dźiś zowią Prorokiem / tego przed tym názywano Widzącym. )
Tedy rzekł Saul do ſługi ſwego : Dobre <i>jeſt</i> ſłowo twoje : chodź / pódźmy : y ƺli do miáſtá w którym był mąż Boży.
A gdy wſtępowáli ná górę miáſtá / á potkáli dźiewecżki wychodzące cżerpáć wodę / rzekli im : A jeſtże tu Widzący?
Które odpowiádájąc im / rzekły / Jeſt : Oto przed tobą / ſpieƺ śię tedy? dźiś bowiem przyƺedł do miáſtá / gdyż dźiś Ofiáry <i>ſpráwuje</i> lud ná górze.
Skoro wnidźiećie do miáſtá / znajdźiećie go / pierwey niż pójdźie ná górę áby jadł : ábowiem lud nie będźie jadł / áż <i>on</i> przydźie : bo on będźie błogoſłáwił ofierze : potym będą jeść wezwáni : á przetoż idźćie / bo go o tey godźinie znajdźiećie.
Weƺli tedy do miáſtá : á gdy przyƺli w pośrzodek miáſtá / Oto Sámuel wychodźił przećiwko nim / áby ƺedł ná górę :
A PAN objáwił był Sámuelowi dźień przed tym / niźli Saul przyƺedł / mówiąc :
O tym cżáśie jutro poślę do ćiebie mężá / z źiemie Benjámin : którego pomáżeƺ zá wodzá nád ludem mojim Izráelſkim : á on wybáwi lud mój z rąk Filiſtyńſkich : Bom wejrzał ná lud mój / gdyż przyƺło wołánie jego do mnie.
A gdy Sámuel wejrzał ná Saulá / rzekł mu PAN : Otóż mąż / o którymemći powiedźiał : tenći będźie pánował nád ludem mojim.
A ták przyſtąpił Saul do Sámuelá / w pośrzodku bramy / y rzekł : Proƺę powiedz mi gdźie tu jeſt dom Widzącego?
Y odpowiedźiał Sámuel Saulowi / mówiąc : Jam <i>jeſt</i> Widzący / wſtąp przede mną ná górę / á będźiećie dźiś jedli ze mną / potym ćię odpráwię ráno / á cokolwiek jeſt w ſercu twym / oznájmię tobie.
A o Oślice / któreć zginęły dźiś trzeći dźień / nie fráſuj śię / boć śię ználázły : y cżyjeƺ wƺyſtko co nalepƺego w Izráelu? yzali nie twoje? y nie wƺyſtkiego domu Ojcá twego?
A odpowiádájąc Saul rzekł : yzalim ja nie Syn Jemini / z namniejƺego pokolenia Izráelſkiego? á dom mój ázaż nie napodlejƺy miedzy wƺyſtkimi domy pokolenia Benjáminowego? Przecżżeś tedy mówił do mnie tákowe ſłowá?
A ták wźiąwƺy Sámuel Saulá y ſługę jego / w wiódł je ná Sálę / y dał im miejſce przedniejƺe miedzy wezwánymi / których było około trzydźieſtu mężów.
Y rzekł Sámuel kuchárzowi : Daj <i>ſám</i> tę cżąſtkę którąmći dał / y o którymći rzekł / Zchowaj ją u śiebie.
A gdy przynióſł kucharz łopátkę / y to co było ná niey / położył <i>Sámuel</i> przed Saulá / y rzekł : Oto co zoſtáło weźmi przed śię / á jedz / bo ná ten cżás zchowano <i>to</i> dla ćiebie / <i>gdym</i> rzekł : Wezwałem ludu. Y jadł Saul z Sámuelem dniá onego.
A gdy zſtąpili z góry do miáſtá / rozmawiał z Saulem ná dáchu.
Potym wſtáli bárzo ráno : Y ſtáło śię / gdy śię pocżęło rozedniewáć : záwołał Sámuel Saulá ná dách / mówiąc : Wſtań / á odpráwię ćię : wſtawƺy tedy Saul / wyƺli obá z domu / on y Sámuel.
A gdy zchodźili ku końcowi miáſtá / rzekł Sámuel do Saulá : rzecż ſłudze áby ƺedł przed námi / y ƺedł : á ty pozoſtań trochę żeć opowiem ſłowo Boże.
