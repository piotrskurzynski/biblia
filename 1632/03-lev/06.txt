Potym rzekł PAN do Mojzeƺá mówiąc :
Gdyby cżłowiek zgrzeƺył / á wyſtępkiem wyſtąpił przećiwko PANU / á záprzałby rzeczy ſobie zwierzoney / y do ſchowánia dáney / álboby co wydárł / álboby gwałtem wźiął / od bliźniego ſwego.
Tákże / jeſliby rzecż zgubioną nálazł / á záprzałby jey / álboby też przyśiągł fałƺywie / o którąkolwiek rzecz z tych które czyni cżłowiek / grzeƺąc przez nię :
Gdyby tedy zgrzeƺył / á winien był / wróći rzecż którą wydárł / álbo którą gwałtem wźiął / álbo też rzecż ſobie powierzoną / álbo rzecż zgubioną którą ználazł :
Albo też o cobykolwiek fałƺywie przyśiągł : tedy wróći to cále / y piątą część <i>do tego</i> przyda / temu czyje było / to wróći w dźień Ofiáry zá grzech ſwój.
A Ofiárę zá wyſtępek ſwój przywiedźie PANU / báráná zupełnego / z drobnego bydłá / według oƺácowánia twego / ná Ofiárę zá grzech / do Kápłáná.
Y oczyśći go Kápłán przed PAnem / á będźie mu odpuƺczona káżda z tych rzecży którą uczynił / y był jey winien.
Y rzekł PAN do Mojzeƺá mówiąc :
Rozkaż Aáronowi y ſynom jego / y rzecż : Táć <i>będźie</i> uſtáwá Ofiáry cáłopalenia ( Ofiárá cáłopalenia jeſt od palenia ná Ołtarzu / przez cáłą noc / áż do poránku : bo ogień ná Ołtarzu uſtáwicznie gorzeć będźie ).
Y oblecże śię Kápłán w odźienie ſwoje lniáne / y Ubiory lniáne oblecże ná ćiáło ſwoje / y zbierze popiół / gdy ſpali ogień Ofiárę cáłopalenia ná Ołtarzu / á wyſypie go podle Ołtarzá.
Potym zewlecże ƺáty ſwe / y oblecże śię w ƺáty inne / á wynieśie popiół on zá Obóz ná miejſce czyſte.
A ogień ná Ołtarzu <i>uſtáwicżnie</i> gorzeć będźie / nie będźie gáƺony : á będźie zápalał ná nim Kápłán drwá ná káżdy poránek / y włoży nań Ofiárę cáłopalenia / á pálić będźie ná nim tłuſtość Ofiár ſpokojnych.
Ogień uſtáwicznie będźie gorzał ná Ołtarzu / nie będźie gáƺony.
A táć też jeſt uſtáwá Ofiáry śniedney / którą Ofiárowáć będą Synowie Aáronowi przed oblicżem PAńſkim u Ołtarzá.
Weźmie garść ſwoję pƺenney mąki / z tey Ofiáry śniedney / y z Oliwy jey / ze wƺyſtkim kádźidłem / które będźie ná ofierze śniedney / y to ſpali ná Ołtarzu ku wdźięcżney wonnośći / ná Pámiątkę jey / PANU.
A co zoſtánie z niey / jeść będą Aáron y Synowie jego / bez kwáſu jedźione będźie / ná miejſcu świętym / w śieni Namiotu zgromádzenia / jeść to będą.
Nie będą <i>tego</i> wárzyć z kwáſem / bo zá dźiał ich dałem im to / z ofiár mojich ogniſtych : rzecz Naświętƺa to jeſt / jáko y ofiárá zá grzech / y jáko ofiárá zá wyſtępek.
Káżdy mężczyzná z Synów Aáronowych / jeść to będźie : uſtáwá to wieczna / w narodźiech wáƺych / o Palonych ofiárách PAńſkich / wƺyſtko co śię ich dotknie poświęcone będźie.
Potym rzekł PAN do Mojzeƺá / mówiąc :
Táć jeſt ofiárá Aároná y Synów jego / którą ofiárowáć będą PANU / w dźień pomázánia ſwego : dźieśiątą część Efy mąki pƺenney / zá ofiárę śniedną uſtáwiczną : połowicę jey ráno / á połowicę jey wiecżór.
W panwi z Oliwą będźie gotowána / ſmáżoną przynieśieƺ ją / uſmáżoną ofiárę śniedną / w ƺtukách ofiárowáć będźieƺ / ku wdźięcżney wonnośći PANU.
A Kápłán pomázány z Synów jego po nim / ofiárowáć ją będźie : uſtáwá to wiecżná PAnu / wƺyſtká ſpalona będźie.
Y káżda śniedná Ofiárá kápłáńſka wƺyſtká ſpalona będźie : nie będą jey jeść.
Zátym rzekł PAN do Mojzeƺá / mówiąc :
Mów do Aároná / y do Synów jego / á rzecż : Tá będźie uſtáwá ofiáry zá grzech : Ná miejſcu / gdźie biją Ofiáry cáłopalenia / będźie zábita ofiárá zá grzech / przed PAnem : rzecz Naświętƺa jeſt.
Kápłán któryby ją Ofiárował zá grzech / jeść ją będźie : ná miejſcu świętym jedźiona będźie / w śieni Namiotu zgromádzenia.
Wƺyſtko co śię dotknie mięſá jey / będźie poświęcone / á jeſliby krwią jey ƺátá pokropiona byłá / co śię pokropiło omyjeƺ ná miejſcu świętym :
Naczynie też gliniáne / w którymby ją wárzono / ſtłuczone będźie : á jeſliby w naczyniu miedźiánym wárzona byłá / wytrą je / y wymyją wodą.
Wƺelki mężcżyzná z Kápłanów / jeść ją będźie : Naświętƺa to rzecż jeſt.
A żadna ofiárá zá grzech / którey krew wnaƺána bywa do Namiotu zgromádzenia / dla ocżyśćienia w świątnicy / nie będźie jedźioná / <i>ále</i> ogniem ſpalona będźie.
