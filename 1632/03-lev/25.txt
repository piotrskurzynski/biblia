Rzekł nád to PAN do Mojzeƺá ná górze Synáj / mówiąc :
Powiedz Synom Izráelſkim / á mów do nich : Gdy wnidźiećie do źiemie którą ja wam dáję / tedy święćić będźie źiemiá Sábbát PAnu.
Przez ƺeść lat ośiewáć będźieƺ pole twoje / y przez ƺeść lat winnicę twoje obrzynáć będźieƺ / zbierájąc urodzáje z niey.
Ale roku śiódmego Sábbát odpoczynienia mieć będźie źiemiá / Sábbát PAńſki : Polá twego nie będźieƺ ośiewał / áni winnice twojey obrzynał.
Co śię ſámo przez śię zrodźi zbożá twego / nie będźieƺ żął / y jágód zániecháney winnice twojey nie będźieƺ zbierał : rok odpoczynienia będźie miáłá źiemiá.
Y będźie co śię urodźi w onym odpoczynieniu źiemie / tobie ná pokarm / y ſłudze twemu / y ſłużebnicy twey / y najemnikowi twemu / y przychodniowi twemu / który mieƺka z tobą.
Tákże bydłu twemu / y zwierzowi który jeſt w źiemi twojey / będźie wƺyſtek urodzaj jey ná pokarm.
Naliczyƺ też ſobie śiedm tegodniów lat / <i>to jeſt</i> śiedm kroć śiedm lat : Y uczyniąć dni śiedmi tegodniów / lat czterdźieśći y dźiewięć lat.
Tedy każeƺ zátrąbić w trąbę huczną / mieśiącá śiódmego <i>dniá</i> dźieśiątego <i>tegoż</i> mieśiącá / w dźień oczyśćiánia każećie zátrąbić w trąbę / po wƺyſtkiey źiemi wáƺey.
Y święćić będźiećie rok pięćdźieśiąty / á obwołaćie wolność w źiemi wƺyſtkim obywátelom jey : Láto miłośćiwe mieć będźiećie : Y wróći śię káżdy do ośiádłośći ſwojey / y káżdy do rodźiny ſwojey wróći śię.
To miłośćiwe láto pięćdźieśiątego roku miewáć będźiećie : nie będźiećie śiać / y nie będźiećie żąć tego / co śię ſámo przez śię zrodźi áni zbieráć będźiećie gron z winnic zániechánych.
Bo miłośćiwy rok jeſt / święty wam będźie : co śię ná polu przed tym zrodźiło / to jeść będźiećie.
W ten miłośćiwy rok / wróći śię káżdy do ośiádłośći ſwojey.
Jesli co przedaƺ bliźniemu twemu / álbo co kupiƺ od bliźniego twego niech nie oƺukywa jeden drugiego.
Według liczby lat po miłośćiwym roku / kupiƺ od bliźniego twego y według liczby lat / dochody przeda tobie.
Jeſli więcey będźie lat / tym drożey oƺácujeƺ kupno ono / á jeſli mniey będźie lat / tedy też tániey oƺácujeƺ kupno ono : ponieważ <i>tylko</i> liczbá dochodów przedawa śię tobie.
A ták nie oƺukawajćie żaden bliźniego ſwego / ále śię bój <i>káżdy</i> Bogá ſwego : bom <i>JA PAN</i> Bóg wáƺ.
Przeſtrzegajćie uſtaw mojich / y ſądy moje záchowywajćie / y czyńćie je / ábyśćie mieƺkáć mogli w źiemi oney beſpiecznie.
Tedy wyda źiemiá owoc ſwój / á będźiećie jeść áż do ſytośći / y będźiećie mieƺkáć beſpiecznie w niey.
A jeſlibyśćie rzekli : Cóż będźiemy jeść roku śiódmego / jeſli nie będźiem śiać áni zbieráć urodzájów náƺych?
Tedy rozkażę błogoſłáwieńſtwu memu <i>przyść</i> ná was roku ƺóſtego / y przynieśie urodzaj ná trzy látá.
Y będźiećie śiać roku óſmego / á będźiećie jeść urodzaj ſtáry / áż do roku dźiewiątego / póki nie nádeydą pożytki jego / ſtáre jeść będźiećie.
Ziemiá tedy nie będźie przedawáná ná wieczność : bo mojá jeſt źiemiá / á wyśćie gośćmi y przychodniámi u mnie.
A po wƺyſtkiey źiemi ośiádłośći wáƺey / <i>pozwolićie</i> wykupowáć źiemię.
Gdyby zubożał brát twój / á przedałby <i>nieco</i> z májętnośći ſwojey / y przyƺedłby májący práwo odkupienia / powinny jego / niech wykupi co przedał brát jego.
A jeſliby kto nie miał tego / coby odkupić mógł / á ſamby przemógł / y ználazł doſtátek ná to wykupno :
Tedy obráchowawƺy látá od przedánia ſwego / wróći co zbywa temu / któremu przedał : <i>á</i> wróći śię do májętnośći ſwojey.
A jeſliby nie miał doſtátku áby wróćił / tedy zoſtánie májętność przedáná / w ręku tego który ją kupił / áż do Roku miłośćiwego : y uſtąpi mu jey w rok miłośćiwy / á <i>on</i> wróći śię do májętnośći ſwojey.
Jeſliby <i>też</i> kto przedał dom mieƺkánia w mieśćie murowánym / będźie miał wolność wykupić go / pókiey nie wynidźie rok przedánia jego : cáły rok będźie miał práwo do wykupienia jego.
A jeſli go nie wykupi póki nie wynidźie rok cáły / tedy zoſtánie on dom w mieśćie murowánym temu który go kupił dźiedźicżnie / y potomkom jego : Y nie uſtąpi w miłośćiwe láto.
Ale domy we wśiách które nie ſą murem obtocżone / te práwem / jáko pole źiemie / ƺácowáne będą : będą mogły być odkupowáne / y w miłośćiwe láto <i>z rąk obcych</i> wynidą.
Ale miáſtá Lewitów / y domy w dźiedźicżnych mieśćiech ich / káżdego czáſu wykupowáne być mogą / przez Lewity.
Lecz temu co kupuje od Lewitów / wynidźie kupno domu / y miejſkiey ośiádłośći jego w rok miłośćiwy : gdyż domy miaſt Lewitſkich ſą dźiedźiczne ich / w pośrzodku Synów Izráelſkich.
Ale pole ná przedmieśćiu ich / nie będźie przedawáne / bo dźiedźictwem ich jeſt wiecznym.
Gdyby też zubożał brát twój / á oſłábiáłáby ręká jego przy tobie / podeprzeƺ go / á jáko y przychodźień / niech śię żywi przy tobie.
Nie bierz od niego lichwy / áni płátu / ále śię bój Bogá ſwego / áby śię żywił brát twój przy tobie.
Pieniędzy twojich nie dawaj mu ná lichwę / áni mu z zyſku pożycżaj żywnośći twojey.
<i>JAM PAN</i> / Bóg wáƺ / którym was wywiódł z źiemie Egipſkiey / ábym wam dał źiemię Chánánejſką / á był wam zá BOgá.
Jesliby też zubożał brát twój przy tobie / ták żeby śię tobie záprzedał / nie będźieƺ go dręczył ſłużbą niewolniczą :
Jáko najemnik / jáko przychodźień będźie u ćiebie / áż do roku miłośćiwego ſłużyć ći będźie.
Potym wynidźie od ćiebie on y dźieći jego z nim / á wróći śię do rodźiny ſwojey / y do dźiedźictwá przodków ſwych / wróći śię.
Słudzy bowiem moji ſą / którem ja wywiódł z źiemie Egipſkiey / niechże nie będą przedawáni jáko niewolnicy.
Nie będźieƺ pánował nád nimi ſurowie / ále śię będźieƺ bał PAná Bogá twego.
Niewolnik też twój / y niewolnicá twojá / które mieć będźieƺ / <i>będą</i> z Narodów tych / które <i>ſą</i> około was / z nich kupowáć będźiećie niewolniká y niewolnicę.
Tákże też Syny przychodniów mieƺkájących miedzy wámi / kupowáć będźiećie / y z potomſtwá tych którzy ſą z wámi / które zpłodźili w źiemi wáƺey / á ći będą wam zá dźiedźictwo.
Práwem dźiedźicznym trzymać je będźiećie y Synowie wáƺy po was / ábyśćie je dźiedźicznie odźierzeli : ná wieki ſłużby ich używáć będźiećie : lecz nád bráćią ſwą Syny Izráelſkimi / żaden nád brátem ſwojim nie będźie pánował ſurowie.
Jeſliby śię też gość / álbo przychodźień zbogáćił / który mieƺka z tobą / á zubożałby brát twój przy nim / ták żeby śię záprzedał gośćiowi / álbo przychodniowi / który jeſt z tobą / álbo potomſtwu z domu cudzoźiemców :
Gdyby śię záprzedał / może być wykupiony : ktokolwiek z bráći jego odkupi go :
Albo ſtryj jego / álbo Syn ſtryjá jego / odkupi go / álbo z blizkich pokrewnych jego / z rodźiny jego odkupi go / álbo jeſliby przemógł / wykupi śię ſam.
Y poráchuje śię z onym co go kupił / od roku / którego mu śię przedał / áż do miłośćiwego látá / áby pieniądze zá które śię przedał / odłożone były według liczby lat : jáko z najemnikiem / z nim ſobie poſtąpi.
Jeſliby jeƺcze nie máło lat zoſtawáło / wedle nich wróći okup ſwój z pieniędzy / zá które kupiony jeſt.
A jeſliby nie wiele lat zoſtawáło do miłośćiwego látá / tedy poráchuje śię z nim / <i>á</i> według onych lat wróći okup ſwój.
Jáko najemnik doroczny niech będźie u niego / nie będźie nád nim ſurowie pánował / przed oczymá twemi.
A jeſliby śię tym obyczájem nie wykupił / tedy wynidźie w miłośćiwe láto / on y dźieći jego z nim :
Abowiem Synowie Izráelſcy ſą ſługámi mojimi / ſługámi mojimi ſą / którem wywiódł z źiemie Egipſkiey : <i>JA PAN,</i> Bóg wáƺ.
