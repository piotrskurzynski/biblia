Tedy Synowie Aáronowi Nádáb y Abiu / wźiąwƺy káżdy kádźielnicę ſwoję / włożyli w nię ogniá / y włożywƺy nań kádźidłá / ofiárowáli przed obliczem PAńſkim ogiē obcy / cżego im był nie rozkazał.
Przetoż wyƺedƺy ogień od twarzy PAńſkiey / poráźił je / y pomárli przed PAnem.
Zátym rzekł Mojzeƺ do Aároná : Toć to jeſt co opowiedźiał PAN / mówiąc : w tych którzy przyſtępują do mnie / poświęcony będę / y przed oblicżnośćią wƺyſtkiego ludu / uwielbiony będę : y zámilkł Aáron.
Tedy wezwał Mojzeƺ Mizáelá y Elizáfáná Synów Uzielá / Stryjá Aáronowego / y rzekł do nich : Przyſtąpćie / á wynieśćie bráćią wáƺę z świątnice / precz zá Obóz.
Przyƺli tedy / á wynieśli je / y z ƺátámi ich / precz zá obóz / jáko był rozkazał Mojzeƺ /
Rzekł potym Mojzeƺ do Aároná / y do Eleázárá / y do Itámárá Synów jego : Głów wáƺych nie obnażájćie / áni ƺát ſwych rozdźierajćie / byśćie nie pomárli / á <i>Bóg</i> nie rozgniewał śię ná wƺyſtko zgromádzenie. Ale bráćia wáƺy / wƺyſtek dom Synów Izráelſkich / niech płáczą tego ſpalenia / które uczynił PAN.
Ze drzwi też Namiotu zgromádzenia nie wychodźćie / byśćie ſnadź nie pomárli / ábowiem olejek pomázánia PAńſkiego jeſt ná was : y uczynili według rozkazánia Mojzeƺowego.
Zátym rzekł PAN do Aároná / mówiąc :
Winá y napoju mocnego / nie będźieƺ pił / ty / y ſynowie twoji z tobą / gdy będźiećie mieli wchodźić do Namiotu zgromádzenia / ábyśćie nie pomárli : uſtáwá to wieczná <i>będźie</i> w narodźiech wáƺych.
Abyśćie rozeznawáć mogli miedzy rzeczą świętą / y miedzy rzeczą poſpolitą / y miedzy rzeczą nieczyſtą / y miedzy rzeczą czyſtą.
A żebyśćie náuczáli ſynów Izráelſkich / wƺyſtkich uſtaw / które im rozkazał PAN przez Mojzeƺá.
Mówił potym Mojzeƺ do Aároná / y do Eleázárá / y Itámárá ſynów jego / którzy byli pozoſtáli : Weźmićie ofiárę śniedną / która zoſtáłá od ogniſtych Ofiar PAńſkich / á jedzćie ją z przáſnikámi przy Ołtarzu : bo rzecz naświętƺa jeſt :
Przetoż jeść ją będźiećie ná miejſcu świętym : bo to Práwo twoje / y práwo ſynów twojich / z ogniſtych Ofiar PAńſkich : bo mi ták rozkazano.
Tákże moſtek obracánia / y łopátkę podnoƺenia będźiećie jedli ná miejſcu czyſtym / ty y ſynowie twoji / y córki twoje z tobą : ábowiem <i>to</i> práwem tobie / y práwem ſynom twojim dano / z ofiar ſpokojnych ſynów Izráelſkich.
Łopátkę podnoƺenia / y moſtek obracánia / z ofiárámi ogniſtymi / <i>y</i> tłuſtośći przyniosą / áby je tám y ſám obracano przed obliczem PAńſkim : á <i>to</i> będźie tobie y ſynom twojim z tobą / práwem wiecżnym / jáko rozkazał PAN.
Potym Mojzeƺ ƺukał pilnie kozłá ofiárowánego zá grzech / á oto <i>już</i> ſpalony był : y dla tego rozgniewawƺy śię ná Eleázárá y Itámárá ſyny Aáronowe / którzy byli pozoſtáli / mówił :
Przeczżeśćie nie jedli Ofiáry zá grzech ná miejſcu świętym? ábowiem to jeſt rzecz naświętƺa : ponieważ ją wam dano / ábyśćie nośili niepráwość <i>wƺyſtkiego</i> ludu / ná oczyśćienie ich przed obliczem PAńſkim :
A oto nie jeſt wnieśiona krew jego we wnątrz do świątnice : mieliśćie go jeść w świątnicy jákom rozkazał.
Tedy Aáron odpowiedźiał Mojzeƺowi : Oto dźiś ofiárowáli ofiárę ſwoję zá grzech / y ofiárę cáłopalenia ſwego przed obliczem PAńſkim á oto mię podkáło. A gdybym był jadł dźiś ofiárę zá grzech / yzaliby śię to było podobáło PAnu?
To gdy uſłyƺał Mojzeƺ przeſtał ná tym.
