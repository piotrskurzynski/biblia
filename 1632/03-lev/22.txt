Zátym rzekł PAN do Mojzeƺá / mówiąc :
Powiedz Aáronowi y Synom jego / áby śię wſtrzymawáli od rzecży które ſą poświęcone od Synów Izráelſkich / á nie plugáwili świętego imienia mojego <i>w tym</i> co mi oni poświącáją : <i>JA PAN.</i>
A <i>ták</i> rzecż do nich : w narodźiech wáƺych / ktobykolwiek przyſtąpił / ze wƺyſtkiego potomſtwá wáƺego / do poświęconych <i>rzeczy,</i> któreby poświęćili Synowie Izráelſcy PAnu / gdy nieczyſtość jego ná nim jeſt / wytrácony będźie ten od oblicznośći mojey : <i>JA PAN.</i>
Ktobykolwiek z naśienia Aáronowego był trędowátym / álbo płynienie naśienia ćierpiącym / rzecży poświęconych jeść nie będźie / pókiby śię nie oczyśćił : tákże ktoby śię dotknął jákiey niecżyſtośći ćiáłá zmárłego / álbo tego z któregoby płynęło naśienie złącżenia.
Tákże / ktoby śię dotknął czego co śię czołga po źiemi / przez coby śię nieczyſtym ſtał / álbo cżłowieká przez któregoby śię ſplugáwił / według wƺelákiey nieczyſtośći jego :
Ten ktoby śię <i>cżego</i> z tych rzeczy dotknął / nieczyſtym będźie áż do wiecżorá / y nie będźie jadł rzeczy poświęconych / áżby umył ćiáło ſwoje wodą.
Y áż po zachodźie ſłońcá czyſtym będźie / á potym będźie jeść z rzeczy poświęconych / bo to jeſt pokarm jego.
Śćierwu też y rozƺárpánego <i>od zwierza</i> jeść nie będźie : áby śię tym nie ſplugáwił : <i>JA PAN.</i>
A ták przeſtrzegáć będą rozkazánia mego / áby nie podlegli grzechowi / y nie pomárli w nim / gdyby śię ſplugáwili / <i>JA PAN /</i> który je poświącam.
Żaden obcy nie będźie jadł z rzeczy poświęconych / komornik kápłáńſki / áni nájemnik nie będźie jadł rzeczy poświęconych.
A jeſliby Kápłan cżłowieká kupił zá pieniądze ſwoje / ten jeść będźie z rzeczy tych : tákże zrodzony w domu jego / ći będą jadáć z pokármów jego.
Lecz córká kápłáńſká któraby ƺłá zá mężá obcego / tá z <i>ofiar</i> podnoƺenia / rzeczy świętych / jeść nie będźie.
Gdyby záś córká kápłáńſká wdową zoſtáłá / álbo odrzuconą byłá <i>od mężá</i> y dźiatek nie miáłá / á wróćiłáby śię w dom Ojcá ſwego / ták jáko w dźiećińſtwie ſwym / Chleb Ojcá ſwego jeść będźie ále żaden obcy jeść z niego nie będźie.
A jeſliby kto jadł z niewiádomośći rzeczy poświęcone / nádda piątą część do tego / y odda Kápłanowi rzecz poświęconą.
Aby nie plugáwili rzeczy poświęconych / które Synowie Izráelſcy ofiárują PANU /
Y nie przywodźili ná śię karánia zá wyſtępek / gdyby jedli poświęcone rzeczy ich : bom <i>JA PAN</i> który je poświącam.
Potym rzekł PAN do Mojzeƺá / mówiąc :
Powiedz Aáronowi y Synom jego / y wƺyſtkim Synom Izráelſkim / á mów do nich : ktoby kolwiek z domu Izráelſkiego álbo z przychodniów w Izráelu ofiárował ofiárę ſwoję według wƺyſtkich ślubów ſwojich y według wƺyſtkich dárów dobrowolnych ſwojich / któreby ofiárowáli PAnu ná ofiárę cáłopalenia :
Z dobrey woli ſwey <i>ofiárowáć będźie</i> zupełnego ſámcá / z bydłá rogátego / z Owiec y z kóz.
Coby miáło ná ſobie wádę / ofiárowáć nie będźiećie / bo nie będźie przyjemne od was.
Jeſliby kto ofiárował ofiárę ſpokojną PAnu / pełniąc ślub / álbo dobrowolny dar oddájąc z rogátego bydłá / álbo z drobnego bydłá / bez wády będźie áby przyjemne było : żadney wády nie będźie ná nim.
Ślepego álbo ułomnego / álbo ná czym ochromionego / álbo guzowátego / álbo kroſtáwego / álbo párƺywego / nie ofiárujćie PAnu / áni ná ofiárę ogniſtą dawajćie ich ná Ołtarz PAnu.
Wołu też álbo owcę / zbytnich / álbo nie zupełnych cżłonków / zá dobrowolny dar ofiárowáć je możeƺ : ále ślub z nich przyjemny nie będźie.
Zgniećionego y ſtłuczonego / y przerwánego / y rzezánego / nie będźiećie ofiárowáć PAnu / w źiemi wáƺey nie uczynićie tego.
Ani z ręki cudzoźiemcá nie będźiećie ofiárowáć chlebá Bogu wáƺemu z tych wƺyſtkich rzeczy : bo ułomek jeſt w nich / wádę máją / nie będą przyjemne od was.
Nád to rzekł PAN do Mojzeƺá : mówiąc :
Wół álbo Owcá / álbo kozá / gdy śię urodźi / niech będźie śiedm dni przy mátce ſwojey / á dniá óſmego / y potym / będźie przyjemne ku paloney ofierze PAnu.
Krowy też áni owce z płodem ich nie zábijećie dniá jednego.
A gdybyśćie ofiárowáli ofiárę dźiękczynienia PAnu / z dobrey woli ſwey ofiárowáć będźiećie.
Onegoż dniá jedźiona będźie / nie zoſtáwićie z niey nic áż do jutrá / <i>JA PAN.</i>
Przetoż ſtrzeżćie przykazań mojich / á czyńćie je <i>JA PAN</i>.
Y nie plugawćie imienia mego świętego / ábym był poświęcony w pośrzodku ſynów Izráelſkich : <i>JA PAN,</i> który was poświącam.
Którym was wywiódł z źiemie Egipſkiey / ábym wam był zá Bogá : <i>JA PAN.</i>
