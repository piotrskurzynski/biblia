Potym rzekł PAN do Mojzeƺá / mówiąc :
Powiedz Synom Izráelſkim : ktobykolwiek z Synów Izráelſkich / álbo z przychodniów mieƺkájących w Izráelu / ofiárował potomſtwo ſwoje Molochowi / śmierćią niech umrze : lud źiemie / niechaj go ukámionuje :
Bo ja poſtáwię twarz moję <i>rozgniewáną</i> przećiwko temu mężowi / y wytrácę go z pośrzodku ludu jego : przeto yż potomſtwo ſwoje ofiárował Molochowi / y ſplugáwił świątnicę moję / á zmázał imię świątobliwośći mojey.
A jesliby lud źiemie nie dbájąc przeglądał mężowi tákiemu / któryby ofiárował Molochowi potomſtwo ſwe / y nie zábiłby go :
Tedy ja poſtáwię twarz moję <i>zágniewáną</i> przećiw temu mężowi / y przećiw domowi jego / y wytrácę go ( y wƺyſtkie którzy cudzołożąc ƺliby zá nim / áby cudzołożyli náśládując Molochá / ) z pośrzodku ludu jego.
Cżłowiek któryby śię udał do czárowników / y do wieƺczków áby cudzołożył idąc zá nimi : poſtáwię twarz ſwoję <i>rozgniewáną</i> przećiwko niemu / y wytrácę go / z pośrzodku ludu jego.
Przetoż poświęcajćie śię á bądźćie świętymi : bom <i>JA PAN /</i> Bóg wáƺ.
A ſtrzeżćie uſtaw mojich / y czyńćie je : <i>JA PAN</i> poświącájący was.
Ktobykolwiek złorzecżył Ojcu ſwemu / álbo mátce ſwey śmierćią umrze. Ojcu ſwemu y mátce ſwey złorzeczył : krew jego <i>będźie</i> ná nim.
Ktoby śię kolwiek cudzołóſtwá dopuśćił z cżyją żoną / ponieważ cudzołożył z żoną bliźniego ſwego / śmierćią umrze / cudzołożnik on y cudzołożnicá.
Ktobykolwiek też ſpał z żoną Ojcá ſwego / ſromotę Ojcá ſwego odkrył / śmierćią umrą oboje : krew ich <i>będźie</i> ná nich.
Jesliby też kto ſpał z ſynową ſwoją : śmierćią umrą oboje / obrzydliwośći śię dopuśćili / krew ich <i>będźie</i> ná nich.
Mąż tákże któryby z mężczyzną obcował / ſpoſobem niewieśćiem / obrzydliwość ucżynili obá / śmierćią umrą : krew ich będźie ná nich.
Ktoby też pojął córkę z mátką jey / ſproſna rzecz jeſt / ogniem ſpalą onego / y onę / áby nie byłá <i>tá</i> ſprosność miedzy wámi.
Tákże ktoby śię złącżył z bydlęćiem / śmierćią umrze : bydlę też zábijećie.
Niewiáſtá któraby przyſtąpiłá do jákiego bydlęćiá áby z nim obcowáłá / zábijeƺ niewiáſtę y bydlę / śmierćią umrą : krew ich <i>będźie</i> ná nich.
Ktoby też pojął śioſtrę ſwoję / córkę Ojcá ſwego / álbo córkę Mátki ſwey / y widźiałby ſromotę jey / y onáby widźiáłá ſromotę jego / rzecż hániebna jeſt / przetoż wytráceni będą przed ocżymá ſynów ludu ſwego / ſromotę śioſtry ſwey odkrył / niepráwość ſwoję ponieśie :
Ktoby ſpał z niewiáſtą cżáſu przyrodzoney choroby jey / y odkryłby ſromotę jey / y obnáżyłby płynienie jey / y onáby też odkrywáłá płynienie krwie ſwojey / wytráceni będą oboje / z pośrzodku ludu ſwego.
Sromoty śioſtry mátki twey / y śioſtry Ojcá twego nie odkryjeƺ / bo <i>ktoby</i> pokrewną ſwoję obnáżył niepráwość ſwoję ponieśie.
Ktoby też ſpał z żoną ſtryjá ſwego / ſromotę ſtryjá ſwego odkrył / grzech ſwój ponioſą / bez dźieći pomrą.
Tákże ktoby pojął żonę brátá ſwego / ſprosność jeſt : ſromotę brátá ſwego odkrył bez dźieći będą.
Strzeżćież tedy wƺyſtkich uſtaw mojich / y wƺyſtkich ſądów mojich / á czyńćie je / áby was nie wyrzućiłá źiemiá / do którey ja was wprowádzę / ábyśćie w niey mieƺkáli.
A nie chodźćie w uſtáwách tego narodu który ja wypędzam od oblicza wáƺego / bo to wƺyſtko czynili / y obrzydźiłem je ſobie.
Wam záś powiedźiałem : wy pośiądźiećie źiemię ich / á ja dam ją wam w dźiedźictwo : źiemię opływájącą mlekiem y miodem. <i>JAM PAN</i> Bóg wáƺ / którym was wyłącżył <i>od innych</i> Narodów.
A ták wy rozeznawajćie miedzy bydlęćiem czyſtym y nieczyſtym / y miedzy ptakiem nieczyſtym y czyſtym / á nie plugawćie duƺ wáƺych / bydłem y ptáſtwem / y wƺyſtkim co śię cżołga po źiemi / którem wam odłączył zá nieczyſte.
Y będźiećie mi świętymi / bom <i>JA</i> święty <i>PAN,</i> y odłącżyłem was od <i>innych</i> narodów / ábyśćie byli mojemi.
Mąż álbo Niewiáſtá w którychby był Duch Cżárnokśięſki / álbo wieƺcży / śmierćią umrą : kámieniem ukámionują ich : krew ich <i>będźie</i> ná nich.
