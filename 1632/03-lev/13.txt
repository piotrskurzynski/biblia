Rzekł záśię PAN do Mojzeƺá y do Aároná / mówiąc :
Cżłowiek któryby miał ná ſkórze ćiáłá ſwego ſadzel / álbo liƺaj / álbo białe blizny / ták yżby śię ná ſkórze ćiáłá jego okazáłá plagá trądu : przywiedźiony będźie do Aároná Kápłaná / álbo do którego z ſynów jego Kápłanów.
Tedy ogląda Kápłan on ſadzel ná ſkórze ćiáłá <i>jego</i> : jeſliby włos ná onym ſadzelu pobielał / á on ſadzel ná pojrzeniu byłby głębƺy / niż <i>inƺa</i> ſkórá ćiáłá jego / plagá trądu jeſt : przetoż oglądawƺy go Kápłan / oſądźi go być nieczyſtym.
A jesliby biała <i>tylko</i> blizná byłá ná ſkórze ćiáłá jego / á nie byłáby głębƺa ná pojrzeniu niż <i>inna</i> ſkórá / y włoſy w niey nie pobieláłyby / tedy zámknie Kápłan májącego táką zárázę / przez śiedm dni.
Potym obejrzy go Kápłan / dniá śiódmego : á jesliby oná blizná ták zoſtáłá w oczách jego / á nie ƺerzyłá śię oná blizná po ſkórze / tedy go zámknie Kápłan przez śiedm dni powtóre.
Y obejrzy go Kápłan dniá śiódmego powtóre : á jeſliby tá zárázá poczerniáłá á nie ƺerzyłáby śię tá zárázá po ſkórze / tedy go zá cżyſtego oſądźi Kápłan / bo świerzb jeſt : á <i>on</i> upierze ƺáty ſwe / á będźie czyſtym.
Ale jeſliby śię bárźiey rozƺerzał po ſkórze świerzb on / po oglądániu Kápłanowym / á po oczyśćieniu jego / pokaże śię z nowu Kápłanowi.
Tedy obejrzy go Kápłan : á jeſliby śię tá zárázá rozƺerzáłá po ſkórze <i>jego,</i> oſądźi go zá nieczyſtego Kápłan : <i>bo</i> trąd jeſt.
Zárázá trądu gdy będźie ná cżłowieku / przywiedźion będźie do Kápłaná :
Którego obejrzy Kápłan / á będźieli ſadzel biały ná ſkórze / żeby śię uczyniły włoſy białe / choćby y zdrowe ćiáło było ná tym ſadzelu.
Trąd záſtárzáły jeſt / ná ſkórze ćiáłá jego : y oſądźi go zá nieczyſtego Kápłan / á nie będźie go záwierał / gdyż nieczyſtym jeſt.
A jesliby śię trąd rozƺerzał ná ſkórze / y okryłby trąd wƺyſtkę ſkórę záráżonego / od głowy jego áż do nóg jego / wƺędy gdźie oczymá Kápłan dojrzeć może /
Y obejrzy Kápłan : á jeſli okrył trąd wƺyſtko ćiáło jego / zá czyſtą oſądźi zárázę jego bo yż wƺyſtká pobieláłá / <i>dla tego</i> czyſty jeſt :
Ale którego dniá ukazáłoby śię ná tákowym mięſo dźiwie / nieczyſtym będźie.
A ták ogląda Kápłan mięſo dźiwie / á oſądźi go być zá nieczyſtego / bo ono mięſo dźiwie nieczyſte jeſt : trąd <i>to</i> jeſt.
Ale jeſliby záś zginęło mięſo dźiwie / y obróćiłoby śię w białość / tedy przydźie do Kápłaná.
A widząc go Kápłan / yż śię obróćiłá zárázá w białość : zá czyſtego oſądźi Kápłan záráżonego / bo cżyſtym jeſt.
Jesliby záś był ná ſkórze ćiáłá jego wrzód / á zágoiłby śię :
A ná miejſcu wrzodu onego / uczyniłby śię ſadzel biały / álbo blizná biała záczerwieniáła / tedy okazáná będźie Kápłanowi.
A widząc Kápłan / że ná wejrzeniu głębƺa jeſt niż inna ſkórá / y włoſyby jey pobieláły / zá nieczyſtego oſądźi go Kápłan : zárázá trądu jeſt / z wrzodu wyroſłá.
Ale jesliby ją obaczył Kápłan / że w niey włos nie bieleje / y nie jeſt głębƺa nád <i>inƺą</i> ſkórę / ále tylko náczerniáłá : tedy zámknie Kápłan tákowego przez śiedm dni.
A jesliby śię ƺerzyłá po ſkórze / zá nieczyſtego oſądźi go Kápłan / zárázá to <i>trądu</i>.
Wƺákże jeſliby blizná oná biała ná ſwym miejſcu zoſtawáłá / y nie ƺerzyłáby śię / zápalenie wrzodu jeſt : przetoż zá czyſtego oſądźi go Kápłan.
Tákże ćiáło ná którego ſkórze byłáby ſpárzeliná <i>od</i> ogniá / á po zgojeniu oney ſpárzeliny / byłáby blizná biała / záczerwieniáła / álbo <i>biała</i> tylko /
Ogląda to Kápłan : á jesliby włos w bliźnie pobielał y lſnił śię / á ná pojrzeniu byłáby głębƺa <i>oná blizná</i> niż ſkórá / trąd jeſt z ſpárzeliny wyroſły : przetoż zá nieczyſtego oſądźi go Kápłan / <i>bo</i> zárázá trądu jeſt.
A jeſli Kápłan obaczy / że ná oney bliźnie białey włos nie pobielał / á yż nie jeſt głębƺa nád <i>inƺą</i> ſkórę / ále yż nie co náczerniáłá / zámknie Kápłan tákowego przez śiedm dni.
Y obejrzy go Kápłan dniá śiódmego : jeſliby śię bárziey ƺerzyłá po ſkórze / oſądźi go zá nieczyſtego Kápłan / zárázá to trądu.
A jeſliż tá blizná biała zoſtawa ná ſwym miejſcu / á nie ƺerzy śię po ſkórze / ále śię záczerniwa / pryƺczel z ſpárzenia jeſt : y oſądźi go zá czyſtego Kápłan / bo blizná ſpárzeliny jeſt.
Gdyby mąż álbo niewiáſtá mieli jáką plámę ná głowie álbo ná brodźie /
Tedy obejrzy Kápłan onę plámę : á będźieli ná pojrzeniu głębƺa niż <i>inƺa</i> ſkórá / y byłby ná niey włos pożółkły / y ſubtelny : tedy tákowego zá nieczyſtego oſądźi Kápłan / zmázá jeſt : trąd ná głowie álbo ná brodźie jeſt.
Ale jeſliby obaczył Kápłan zárázę oney plámy / á oto ná wejrzeniu jeſt głębƺa nád <i>inƺą</i> ſkórę / á nie byłby ná niey włos czarny / zámknie Kápłan zárázę plámy <i>májącego</i> / przez śiedm dni.
Potym obejrzy Kápłan tę zárázę dniá śiódmego : á jesli śię nie ƺerzy zmázá / y niemá ná niey pożółkłego włoſá / y ná pojrzeniu tá zmázá nie byłáby głębƺa nád <i>inƺą</i> ſkórę :
Tedy ogolony będźie <i>ten cżłowiek</i> / ále zmázy oney golić nie będźie : y zámknie Kápłan májącego zmázę przez śiedm dni powtóre :
Y oglądá Kápłan onę zmázę dniá śiódmego / á jesli śię nie rozƺerzyłá zmázá po ſkórze / á ná pojrzeniu nie jeſt głębƺa nád <i>inƺą</i> ſkórę / oſądźi go zá czyſtego Kápłan / á <i>on</i> uprawƺy odźienie ſwoje / czyſtym będźie.
A jesliby śię poczęłá ƺerzyć oná zmázá ná ſkórze po oczyśćieniu jego :
Tedy obejrzy go Kápłan / á jeſliż śię ƺerzy oná zmázá po ſkórze / nie będźie więcey upátrował Kápłan włoſu żółtego : nieczyſtym jeſt.
Wƺákże jesli przed ocżymá jego ták zoſtawa oná zmázá / y włos czarny wyróſłby ná niey / zgoiłá śię oná zmázá : czyſty jeſt / y zá czyſtego oſądźi go Kápłan.
A gdyby też ná ſkórze ćiáłá mężczyzny / álbo Niewiáſty były blizny białe :
Tedy obejrzy <i>je</i> Kápłan : á jesliżby śię blizny one białe ná ſkórze ćiáłá jego záczerniwáły / blizná biała jeſt / wyroſła ná ſkórze : czyſtym jeſt.
Mąż tákże któremuby opádáły włoſy z głowy jego / łyſy jeſt / y czyſty jeſt.
A jeſliżby przećiwko jedney ſtronie twarzy / opádły włoſy głowy jego / przełyśiáły jeſt / czyſty jeſt.
Wƺákże jesliby ná łyśinie / álbo ná <i>tym</i> przełyśieniu okazáłá śię blizná biała / <i>á</i> zczerwieniáłáby : trąd wyróſł z łyśiny jego / álbo z przełyśienia jego.
Y obejrzy go Kápłan : á jesliżby ſadzel zárázy jego był biały / álbo zczerwieniáły / ná łyśinie jego / álbo ná obłyśieniu jego / ná kƺtałt trądu / ná ſkórze ćiáłá /
<i>Tákowy</i> cżłowiek trędowáty jeſt / nieczyſty jeſt / y oſądźi go bezpiecznie Kápłan zá nieczyſtego : <i>bo</i> ná głowie jego / jeſt trąd jego.
Trędowáty záś / któryby miał ná ſobie tę zárázę / ƺáty jego będą rozdárte / y głowá jego będźie odkryta / y uſtá ſobie zákryje / á wołáć będźie : Nieczyſty / nieczyſty <i>jeſtem</i>.
Po wƺyſtkie dni póki jeſt zárázá ná nim / nieczyſtym będźie / <i>bo</i> nieczyſtym jeſt : ſam będźie mieƺkał / precz zá obozem będźie mieƺkánie jego.
Jesliby też ná ƺáćie byłá zárázá trądu / ná ƺáćie ſukniáney álbo ná ƺáćie lniáney /
Albo ná oſnowie / álbo ná wątku / ze lnu álbo z wełny / álbo ná ſkórze / álbo ná jákieykolwiek rzeczy ſkórzáney :
A byłáby tá zárázá źielona / álbo czerwona ná ƺáćie / álbo ná ſkórze / álbo ná oſnowie / álbo ná wątku / álbo ná jákimkolwiek naczyniu ſkórzánym / zárázá trądu jeſt : będźie ukazáná Kápłanowi.
A oglądawƺy Kápłan zárázę onę / zámknie onę rzecz záráżoną / przez śiedm dni.
Potym obejrzy zárázę onę dniá śiódmego / jesliby śię ƺerzyłá zmázá oná ná ƺáćie / álbo ná oſnowie / álbo ná wątku / álbo ná ſkórze / y ná káżdey rzeczy z ſkóry urobioney / trąd jeſt jádowity / zárázá nieczyſta jeſt.
Tedy ſpali tę ƺátę álbo oſnowę / álbo wątek z wełny / álbo ze lnu / álbo jákiekolwiek naczynie ſkórzáne / ná którymby byłá zárázá : ábowiem jeſt trąd jádowity : ogniem ſpalono będźie.
Ale gdyby obaczył Kápłan / iż oná zmázá nie ƺerzy śię ná ƺáćie / álbo ná oſnowie / álbo ná wątku / álbo ná jákimkolwiek naczyniu ſkórzánym :
Rozkaże Kápłan / áby upráno to / ná czym jeſt zárázá / y zámknie to przez śiedm dni / powtóre.
Y obejrzy Kápłan po uprániu / onę zárázę / á jeſli nie odmieniłá oná zárázá bárwy ſwojey : choćby śię oná zárázá nie rozƺerzyłá / <i>rzecz</i> nieczyſta jeſt : Ogniem ją ſpaliƺ / záráźliwa rzecz jeſt / bądź ná zwierzchniey bądź ná ſpodniey ſtronie jey.
Wƺákże jeſliby Kápłan obaczył yż przyczernieyƺym będźie zárázá / po wyprániu ſwym : odedrze ją od ƺáty álbo od ſkóry / álbo od oſnowy / álbo od wątku.
A jesliby śię jeƺcze ukazáłá ná ƺáćie / álbo ná oſnowie / álbo ná wątku / álbo ná jákim naczyniu ſkórzánym / trąd jeſt ƺerzący śię : ogniem to ſpaliƺ / ná czymby byłá tákowa zárázá.
Szátę záś álbo oſnowę / álbo wątek / álbo káżde naczynie ſkórzáne / którebyś uprał / á odeƺłáby od niego zárázá / upierzeƺ je powtóre / á czyſte będźie.
Táć jeſt uſtáwá o záráźie trądu / ná ƺáćie ſukniáney / álbo lniáney / álbo ná oſnowie / álbo ná wątku / álbo ná jákimkolwiek naczyniu ſkórzánym / jáko to ma być rozeznano / yż jeſt czyſte álbo nieczyſte.
