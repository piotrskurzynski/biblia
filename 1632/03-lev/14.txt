Potym rzekł PAN do Mojzeƺá / mówiąc :
Táć jeſt uſtáwá około trędowátego / w dźień oczyśćienia jego : przywiedźiony będźie do Kápłaná :
A wynidźie Kápłan precż zá Obóz / á obaczyli Kápłan / że oto ulecżona jeſt zárázá trądu / trędem záráżonego /
Tedy rozkaże Kápłan temu który śię oczyśćia / áby wźiął dwá wróble żywe y zdrowe / y drzewo Cedrowe / y jedwabiu Kármázynowego / y Hyſopu :
Y rozkaże Kápłan zábić jednego wróblá nád naczyniem gliniánym / nád wodą żywą.
Wróblá <i>tedy</i> żywego weźmie / y drzewo Cedrowe / y jedwab kármázynowy / y Hyſop / á omoczy to <i>wƺyſtko</i> z wróblem żywym / we krwi wróblá zábitego / nád wodą żywą.
Y pokropi tego który śię oczyśćia od trądu śiedm kroć / y ogłośi go być czyſtym / á puśći wróblá żywego w pole.
A ten który śię oczyśćia / upierze ƺáty ſwoje / y ogoli wƺyſtkie włosy ſwoje / á umyje śię wodą / y czyſtym będźie :
Potym wnidźie do obozu á będźie mieƺkał przed namiotem ſwojim / przez śiedm dni. Potym dniá śiódmego / ogoli wƺyſtkie włosy ſwe / głowę ſwą / y brodę ſwą / y brwi nád oczymá ſwymi / y wƺyſtkie inne włosy ſwe ogoli : przytym upierze ƺáty ſwe / y ćiáło ſwe omyje wodą / á ták oczyśćion będźie.
A dniá óſmego / weźmie dwu báránków zupełnych / y owcę jednę roczną / zupełną / y trzy dźieśiąte częśći <i>Efy</i> mąki pƺenney / ná ofiárę śniedną / zmieƺáną z oliwą / y miárkę oliwy.
Tedy Kápłan który oczyścia cżłowieká który ma być oczyściony / poſtáwi z tymi <i>rzecżami</i> przed PAnem u drzwi Namiotu zgromádzenia.
Potym weźmie Kápłan báráná jednego / y będźie go ofiárował ná ofiárę zá wyſtępek / <i>z oną</i> miarką Oliwy / y będźie to tám y ſám obracał ná Ofiárę obracánia / przed obliczem PAńſkim.
Zábije też báránká <i>onego</i> ná miejſcu gdźie biją ofiáry zá grzech / y ofiárę cáłopalenia / ná miejſcu świętym / bo jáko ofiárá zá grzech / <i>ták</i> ofiárá zá wyſtępek należy Kápłanowi : rzecż naświętƺa jeſt.
Y weźmie Kápłan krwie z ofiáry zá wyſtępek / y pomáże Kápłan koniec uchá práwego / onemu który śię oczyśćia : tákże pálec wielki u práwey ręki jego : y pálec wielki <i>u</i> práwey nogi jego.
Weźmie też Kápłan z oney miarki Oliwy / á náleje ná dłoń ſwoję lewą.
A omoczy pálec ſwój práwy w Oliwie / która jeſt ná lewey dłoni jego / y pokropi Oliwą z pálcá ſwego śiedm kroć przed obliczem PAńſkim.
A z oſtátku oliwy / która jeſt ná dłoni jego / pomáże Kápłan koniec uchá práwego / onemu który śię oczyśćia / y wielki pálec práwey ręki jego / tákże wielki pálec práwey nogi jego / z oneyże krwie <i>która jeſt</i> ofiárą zá wyſtępek.
A coby zoſtáło Oliwy która jeſt ná dłoni Kápłanowey / pomáże tym głowę onego który śię oczyśćia / y ták go oczyśći Kápłan przed obliczem PAńſkim.
Ucżyni tákże Kápłan ofiárę zá grzech / y oczyśći tego / który śię oczyśćia / od nieczyſtośći jego / á potym zábije Ofiárę cáłopalenia.
Y ofiárowáć będźie Kápłan ofiárę cáłopalenia / y ofiárę śniedną / ná Ołtárzu : ták oczyśći go Kápłan / y czyſtym będźie.
A jeſliżby <i>kto</i> był ták ubogi / yżby tego nie przemógł : tedy weźmie báránká jednego ná ofiárę zá wyſtępek / ná podnoƺenie / dla oczyśćienia ſwego / y jednę dźieśiątą część Efy mąki pƺenney / zágniećioney z Oliwą / ná Ofiárę śniedną / y miarkę Oliwy.
Nád to dwie śynogárlice / álbo dwoje gołąbiąt / czego doſtáć może : z których jedno / będźie ná ofiárę zá grzech / á drugie ná ofiárę cáłopalenia.
Y przynieśie je w óſmy dźień oczyśćienia ſwego do Kápłaná / do drzwi Namiotu zgromádzenia / przed obliczność PAńſką.
Weźmie tedy Kápłan báránká Ofiáry zá wyſtępek / y miarkę Oliwy : y będźie to obracał tám y ſám Kápłan <i>ná ofiárę</i> obracánia przed PAnem.
A zábije báránká ná ofiárę zá wyſtępek : á wźiąwƺy Kápłan ze krwie Ofiáry zá wyſtępek / pomáże koniec uchá práwego / temu który śię oczyśćia / y pálec wielki práwey ręki jego / y pálec wielki práwey nogi jego.
Oliwy tákże náleje Kápłan ná lewą dłoń ſwoję.
Y kropić będźie Kápłan pálcem ſwoim práwym / z Oliwy która jeſt ná lewey ręce jego / śiedm kroć przed obliczem PAńſkim.
Pomáże też Kápłan oną Oliwą która jeſt ná dłoni jego koniec uchá práwego / temu który śię oczyśćia : tákże wielki pálec práwey ręki jego / y wielki pálec práwey nogi jego / ná miejſcu krwie / z Ofiáry zá wyſtępek.
A oſtátkiem Oliwy / która jeſt ná dłoni Kápłaná / pomáże głowę onego / który śię oczyśćia / áby go oczyśćił przed PAnem.
Tákże uczyni z jedną śynogárlicą / álbo z jednym gołębięćiem / cżegokolwiek z tych doſtáć może.
Czego doſtáć mógł / jedno z tych / będźie ofiárą zá grzech / á drugie ná ofiárę cáłopalenia / z ofiárą śniedną : á ták oczyśći Kápłan tego / który śię oczyśćia przed obliczem PAńſkim.
A táć jeſt uſtáwá o tym / ná którymby byłá zárázá trądu / który <i>wƺyſtkiego</i> mieć nie może / ku oczyśćieniu ſwemu.
Rzekł potym PAN do Mojzeƺá / y do Aároná mówiąc :
Gdy wnidźiećie do źiemie Chánánejſkiey / którą ja wam dawam w ośiádłość / á dopuśćiłbym zárázę trądu ná który dom ośiádłośći wáƺey /
Tedy on / którego dom jeſt / przydźie / y opowie to Kápłanowi / mówiąc : Jákoby zárázá <i>trądu</i> zda mi śię być w domu mojim.
Rozkaże tedy Kápłan wyprzątnąć dom / pierwey niż ſam wnidźie / áby oglądał zárázę onę / yżby śię nic nie ſplugáwiło / coby było w domu : á potym Kápłan wnidźie / áby oglądał on dom.
A oględując onę zárázę / ujrzyli zárázę ná śćienie domu / <i>jákoby</i> dołki czarne / przyźieleńƺym / álbo przyczerwieńƺym / á ná pojrzeniu byłoby głębƺe niż śćiáná.
Tedy wynidźie Kápłan z domu onego przede drzwi / y zámknie on dom / przez śiedm dni.
Wróći śię potym Kápłan dniá śiódmego / y obejrzy : á jeſli śię rozƺerzyłá zárázá ná śćiánách domu onego /
Rozkaże Kápłan wyłamáć ono kámienie / ná którymby byłá zárázá / y wyrzućić je precz zá miáſto / ná miejſce nieczyſte :
A dom rozkaże we wnątrz oſkrobáć / wƺędy w około : y wyrzucą on proch który oſkrobáli precż zá miáſto / ná miejſce nieczyſte.
Y wezmą kámienie inƺe / y wpráwią ná miejſce innych kámieni / y wapná też inƺego wezmą / á potynkują dom.
A jeſliby śię odnowiłá oná zárázá / y rozƺerzyłá śię po domu / po wyrzuceniu kámienia / y po wyſkrobániu domu / y po tynkowániu <i>jego :</i>
Tedy wnidźie Kápłan / á ujrzyli / że śię rozƺerzyłá oná zárázá po domu / trąd jeſt jádowity w domu <i>onym,</i> nie cżyſty jeſt.
Zátym rozwálą on dom / kámienie jego / y drzewo jego / y wƺyſtko wapno domu onego / á wynioſą precz zá miáſto ná miejſce nieczyſte.
A ten ktoby wƺedł do domu onego / po wƺyſtkie dni póki był záwárty / nieczyſtym będźie / áż do wiecżorá.
A ktoby ſpał w onym domu / upierze ƺáty ſwoje : tákże ktoby jadł w tymże domu / upierze ƺáty ſwoje.
Lecz jesliby wyƺedƺy Kápłan obacżył / yż śię nie ƺerzy zárázá po domu / po tynkowániu jego : tedy oſądźi Kápłan / że dom on jeſt cżyſty / bo ulecżona jeſt zárázá oná :
A weźmie ná oczyśćienie onego domu / dwu wróblów / y drzewo Cedrowe / y jedwabiu kármázynu / y Hyſopu :
Y zábije wróblá jednego nád naczyniem gliniánym / nád wodą żywą.
A wźiąwƺy drzewo Cedrowe / y Hyſop / y jedwab kármázynowy / y wróblá żywego / omocży to <i>wƺyſtko</i> we krwi wróblá zábitego / y w wodźie żywey / á pokropi ten dom śiedm kroć.
Y ták oczyśći on dom krwią wróblá onego / y wodą żywą y wróblem żywym / y drzewem Cedrowym / y Hyſopem / y jedwabiem czerwonym.
Potym puśći wróblá żywego precż zá miáſto w pole : ták oczyśći <i>on</i> dom / y czyſtym będźie.
Táć jeſt Uſtáwá o káżdey záráźie trądu y plámy czarney /
Y o trądźie ná ƺáćie / y ná domu /
Y o ſadzelu / y o świerżbie / y o białey plámie :
Żeby poznáć / gdy kto jeſt nieczyſtym / y gdy kto czyſtym. Táć jeſt Uſtáwá <i>około</i> trądu.
