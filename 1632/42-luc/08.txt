Y ſtáło śię potym / że on chodźił po miáſtách y po miáſtecżkách / każąc y opowiádájąc króleſtwo Boże. A oni dwánaście <i>byli</i> z nim.
Y niektóre niewiáſty które był uzdrowił od duchów złych y od niemocy ich : <i>jáko</i> Márya którą zwano Mágdáleną / z którey było śiedm dyabłów wyƺło :
Y Joánná / żoná Chuzego / Urzędniká Herodowego : y Zuzánná : y inƺych wiele / które mu ſłużyły z májętnośći ſwojich.
A gdy śię zchodźił wielki lud / y z różnych miaſt gárnęli śię do niego / rzekł przez podobieńſtwo ;
Wyƺedł rozśiewcá / áby rozśiewał naśienie ſwoje. A gdy on rozśiewał / tedy jedno pádło podle drogi / y podeptáne jeſt / á ptacy niebieſcy pozobáli je.
A drugie pádło ná opokę : á gdy weƺło / uſchło : przeto iż nie miáło wilgotnośći.
A drugie pádło miedzy ćiernie : ále ćiernie weſpół z nim wzroſło / y záduśiło je.
A drugie pádło ná źiemię dobrą : á gdy weƺło / przynioſło pożytek ſtokrotny. To mówiąc wołał : Kto ma uƺy ku ſłuchániu / niechaj ſłucha!
Y pytáli go ucżniowie jego / mówiąc : Coby to było zá podobieńſtwo?
A on im rzekł : Wam dano wiedźieć tájemnice króleſtwá Bożego / ále innym w podobieńſtwách : áby widząc nie widźieli / á ſłyƺąc nie rozumieli.
A to podobieńſtwo tákie <i>jeſt</i> : Naśienie jeſt ſłowo Boże.
A którzy podle drogi / ći ſą którzy ſłucháją : zátym przychodźi dyabeł / y wybiera ſłowo z ſercá ich / áby uwierzywƺy / nie byli zbáwieni.
A którzy ná opoce / <i>ći ſą</i> którzy gdy ſłucháją / z rádośćią ſłowo przyjmują : ále ći korzenia nie máją : ći do cżáſu wierzą / á cżáſu pokuſy odſtępują.
A które pádło miedzy ćiernie / ći ſą którzy ſłucháją ſłowá : ále odƺedƺy / od piecżołowánia / y bogactw / y roſkoƺy żywotá bywáją záduƺeni / y nie przynoƺą pożytku.
Ale które <i>pádło</i> ná źiemię dobrą / ći ſą którzy w ſercu uprzyjmym y dobrym ſłyƺáne ſłowo záchowują / y owoc przynoƺą w ćierpliwośći.
A żaden zápaliwƺy świecę / nie nákrywa jey nacżyniem / áni jey kłádźie pod łoże / ále ją ſtawia ná świecżniku ; áby ći którzy wchodzą widźieli świátło.
Bo niemáƺ nic tájemnego / coby nie miáło być objáwiono : y niemáƺ nic ſkrytego / cżegoby śię nie dowiedźiano / y coby ná jáwią nie wyƺło.
Przetoż pátrzćie jáko ſłuchaćie : Abowiem kto ma / temu będźie dano ; á kto nie ma / y to co mniema że ma / będźie odjęto od niego.
Tedy przyƺli do niego mátká y bráćia jego : ále do niego przyſtąpić nie mogli dla ludu.
Y dano mu znáć / mówiąc : Mátká twojá y bráćia twoji ſtoją przed domem / chcąc ćię widźieć.
A on odpowiádájąc / rzekł do nich ; Mátká mojá y bráćia moji ſą ći / którzy ſłowá Bożego ſłucháją / y cżynią je.
Y ſtáło śię dniá jednego / że on wſtąpił w łódź y ucżniowie jego / y rzekł do nich ; Przeprawmy śię ná drugą ſtronę jeźiorá. Y puśćili śię.
A gdy płynęli / uſnął. Y przypádłá náwáłność wiátru ná jeźioro / <i>y łódź śię</i> nápełniáłá / ták / że byli w niebeſpiecżeńſtwie.
A przyſtąpiwƺy / obudźili go / mówiąc ; Miſtrzu / miſtrzu / giniemy! A on ocknąwƺy śię / zgromił wiátr y wáły wodne / y uśmierzyły śię : y ſtáło śię ućiƺenie.
Tedy im rzekł : Gdźież jeſt wiárá wáƺá? A bojąc śię / dźiwowáli śię / mówiąc jedni do drugich ; Któż wżdy jeſt ten / że y wiátrom rozkázuje y wodam / á ſą mu poſłuƺne.
Y przewieźli śię do krájiny Gádáreńcżyków / która jeſt przećiw Gálilejey <i>ná drugi ſtronie</i>.
A gdy wſtąpił ná źiemię / zábieżał mu mąż niektóry z onego miáſtá / co miał dyabły od niemáłego cżáſu ; á nie obłócżył śię w ƺáty / y nie mieƺkał w domu / tylko w grobách.
Ten ujrzawƺy JEzuſá / zákrzyknął y upadł przed nim / á głoſem wielkim rzekł : Cóż ja mam z tobą JEzuśie Synu Bogá Nawyżƺego? Proƺę ćię nie dręcż mię.
Abowiem rozkázował onemu duchowi niecżyſtemu / áby wyƺedł z onego cżłowieká. Bo od wiela cżáſów porywał go ; á choćia go wiązano łáńcuchámi y w pętách ſtrzeżono / jednák on porwawƺy okowy / bywał od dyabłá ná puſtynie pędzony.
Y pytał go JEzus / mówiąc : Co maƺ zá imię? A on rzekł : Wojſko ; ábowiem wiele dyabłów wſtąpiło było weń.
Tedy go prośili / áby im nie rozkázował <i>z támtąd</i> odejść w przepáść.
A byłá tám trzodá wielka świni / która śię páſłá ná górze / y prośili go / áby im dopuśćił wſtąpić w nie. Y dopuśćił im.
A wyƺedƺy dyabli z onego cżłowieká / weƺli w świnie : y porwáłá śię oná trzodá pędem z przykrá do jeźiorá / y utonęłá.
A widząc páſterze co śię ſtáło / ućiekli : á poƺedƺy oznájmili to w mieśćie y we wśiách.
Y wyƺli / áby oglądáli to co śię ſtáło : á przyƺedƺy do JEzuſá / ználeźli cżłowieká onego z którego wyƺli dyabli / oblecżonego / przy dobrym bacżeniu śiedzącego u nóg JEzuſowych ; y bali śię.
Opowiedźieli im tedy ći którzy widźieli jáko uzdrowiono tego który był opętány.
Y prośiło go wƺyſtko mnóſtwo oney okolicżney krájiny Gádáreńcżyków / áby odƺedł od nich : ábowiem ich był wielki ſtrách ogárnął. A on wſtąpiwƺy w łódź / wróćił śię.
Y prośił go on mąż z którego wyƺli dyabli / áby był przy nim : ále go JEzus odpráwił / mówiąc ;
Wróć śię do domu twego / á opowiádaj jákoć wielkie rzecży Bóg ucżynił. Y odƺedł / po wƺyſtkim mieśćie opowiádájąc / jáko mu wielkie rzecży JEzus ucżynił.
Y ſtáło śię gdy śię wróćił JEzus / że go przyjął lud : ábowiem nań wƺyſcy ocżekawáli.
A oto przyƺedł mąż imieniem Jájirus ; á ten był przełożonym bóżnice : á przypadƺy do nóg JEzuſowych / prośił go áby wƺedł w dom jego.
Abowiem miał córkę jedyną około dwunaści lat / która już konáłá. ( A gdy on ƺedł / ćiſnął go lud. )
A niewiáſtá która płynienie krwie ćierpiáłá od lat dwunaści / y wynáłożyłá byłá ná lekárze wƺyſtko <i>ſwoje</i> pożywienie / á nie mogłá być od nikogo ulecżona :
Przyſtąpiwƺy z tyłu / dotknęłá śię podołká ƺáty jego : á zárázem śię záſtánowiło płynienie krwie jey.
Y rzekł JEzus ; Któż jeſt co śię mnie dotknął? A gdy śię wƺyſcy zápieráli / rzekł Piotr y ći którzy z nim <i>byli</i> : Miſtrzu / lud ćię ćiśnie y tłocży ; á ty mówiƺ ; Kto śię mnie dotknął?
Y rzekł JEzus ; Dotknął śię mnie ktoś ; bom poznał że moc ode mnie wyƺłá.
A widząc oná niewiáſtá że śię nie utájiłá / ze drżeniem przyſtąpiłá / y upádłá przed nim ; <i>y</i> dla cżego śię go dotknęłá / powiedźiáłá mu przed wƺyſtkim ludem / y jáko záraz uzdrowiona byłá.
A on jey rzekł ; Ufaj córko / wiárá twojá ćiebie uzdrowiłá : Idźże w pokoju.
A gdy on to jeƺcże mówił / przyƺedł niektóry od Przełożonego bóżnice / powiádájąc mu ; Iż umárłá córká twojá / nie trudź Náucżyćielá.
Ale JEzus uſłyƺawƺy <i>to,</i> odpowiedźiał mu / mówiąc : Nie bój śię / tylko wierż ; á będźie uzdrowiona.
A wƺedƺy w dom / nie dopuśćił z ſobą wniść nikomu / tylko Piotrowi / y Jákubowi / y Janowi / y Ojcu y mátce oney dźiewecżki.
A płákáli wƺyſcy / y nárzekáli nád nią. Ale on rzekł : Nie płácżćie ; nie umárłáć / ále śpi.
Y náśmiewáli śię z niego / wiedząc iż byłá umárłá.
A on wygnawƺy precż wƺyſtkie / Y ująwƺy ją zá rękę / záwołał mówiąc ; Dźiewecżko / wſtań.
Y wróćił śię duch jey ; y wſtáłá záraz. Y rozkazał / áby jey jeść dano.
Y zdumieli śię rodźicy jey. A on im zákazał / áby nikomu nie powiádáli <i>tego,</i> co śię było ſtáło.
