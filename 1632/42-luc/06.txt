Y ſtáło śię w drugi Sábbát / że ƺedł JEzus przez zbożá : y rwáli ucżniowie jego kłoſy / á rękámi wyćierájąc / jedli.
Ale niektórzy z Fáryzeuƺów rzekli do nich : Przecżże cżynićie to / cżego śię nie godźi cżynić w Sábbát.
A odpowiádájąc JEzus / rzekł do nich ; Azaśćie tego nie cżytáli co ucżynił Dawid / gdy łáknął ſam / y ći którzy z nim byli?
Jáko wƺedł do domu Bożego / á wźiął chleby pokłádne / y jadł : á dał y tym / którzy z nim byli ; których śię nie godźiło jeść tylko ſámym Kápłanom.
Y rzekł im : Syn cżłowiecży jeſtći PAnem y Sábbátu.
Stáło śię tákże y w inƺy Sábbát / że JEzus wƺedł do bóżnice / y náucżał. Y był tám cżłowiek / którego ręká práwa byłá uſchłá.
Y podſtrzegáli go Náucżeni w piśmie y Fáryzeuƺowie / jeſliby w Sábbát uzdrawiał : áby ználeźli o coby nań ſkárzyli.
Ale on wiedźiał myśli ich : Y rzekł cżłowiekowi / który miał rękę uſchłą ; Wſtań / á ſtań pośrzodku. A on wſtawƺy / ſtánął.
Rzekł tedy do nich JEzus : Spytam was o jednę rzecż ; Godźili śię w Sábbáty dobrze cżynić / cżyli źle cżynić? Cżłowieká záchowáć / cżyli zátráćić?
A pojrzawƺy w koło po wƺyſtkich / rzekł onemu cżłowiekowi : Wyćiągni rękę twoję. A on ták ucżynił. Y przywrócona jeſt do zdrowia ręká jego jáko y druga.
Ale oni nápełnieni będąc ƺaleńſtwem / rozmawiáli miedzy ſobą coby ucżynić mieli JEzuſowi.
Y ſtáło śię w oneż dni / odƺedł ná górę áby śię modlił : y był <i>tám</i> przez noc ná modlitwie Bożey.
A gdy był dźień / zwołał ucżniów ſwych / y wybrał <i>z nich</i> dwánaśćie ; które też názwał Apoſtołmi.
( Symoná / którego też názwał Piotrem / y Andrzejá brátá jego / Jákubá / y Janá / Filipá / y Bártłomiejá.
Mátteuƺá / y Tomaƺá / Jákubá <i>ſyná</i> Alfeuƺowego / y Symoná którego zowią Zelotes.
Judáſſá <i>brátá</i> Jákubowego / y Judaƺá Iƺkáryotę / który potym był zdrajcą. )
A zſtąpiwƺy z nimi / ſtánął ná miejſcu polá równego ; y gromádá ucżniów jego / y wielkie mnóſtwo ludu ze wƺyſtkiey Judſkiej źiemie / y z Jeruzalem / y z kráju pomorſkiego / leżącego przy Tyrze y Sydonie / którzy byli przyƺli / áby go ſłucháli ; y byli uzdrowieni od chorób ſwojich :
Y ći którzy byli trapieni od duchów niecżyſtych / byli uzdrowieni.
A wƺyſtek lud ƺukał jákoby śię go dotknąć : ábowiem moc wychodźiłá z niego / y uzdrawiáłá wƺyſtkie.
A on podniózƺy ocży ſwoje ná ucżnie / mówił ; Błogoſłáwieni <i>jeſteśćie wy</i> ubodzy : bo wáƺe jeſt króleſtwo Boże.
Błogoſłáwieni jeſteśćie którzy teraz łákniećie : bo będźiećie náſyceni. Błogoſłáwieni jeſteśćie którzy teraz płácżećie : bo śię śmiać będźiećie.
Błogoſłáwieni będźiećie gdy was ludźie nienawidźieć będą / y gdy wyłącżą / y będą was ſromoćić / y imię wáƺe wyrzucą jáko złe / dla Syná cżłowiecżego.
Rádujćie śię dniá tego / y weſelćie śię : ábowiem oto zapłátá wáƺá jeſt obfita w niebieśiech. Boć ták właśnie Prorokom cżynili Ojcowie ich.
Ale biádá wam bogacżom : bo <i>już</i> maćie poćiechę wáƺę.
Biádá wam którzyśćie náſyceny : ábowiem łáknąć będźiećie. Biádá wam którzy śię teraz śmiejećie : bo śię ſmęćić y płákáć będźiećie.
Biádá wam gdyby dobrze o was mówili wƺyſcy ludźie : bo ták cżynili fałƺywym Prorokom Ojcowie ich.
Ale wam powiádam którzy ſłuchaćie ; Miłujćie nieprzyjaćioły wáƺe / cżyńćie dobrze tym którzy was máją w nienawiśći.
Błogoſławćie tym którzy was przeklináją / y módlćie śię zá tymi którzy wam złość wyrządzáją.
Temu któryby ćię uderzył w policżek / nádſtaw mu y drugiego : á temu któryćby brał płaƺcż / y ſuknie nie zbraniaj.
Y káżdemu któryby ćię prośił / daj ; á temu co twoje bierze / nie upominaj śię.
Y cobyśćie chćieli áby wam ludźie cżynili / ták y wy im cżyńćie.
Abowiem jeſli miłujećie te którzy was miłują / jákąż łáſkę maćie? ábowiem y grzeƺnicy toż właśnie cżynią.
A jeſli dobrze cżynićie tym którzy wam dobrze cżynią / jákąż łáſkę maćie? ábowiem toż y grzeƺnicy cżynią.
A jeſli pożycżaćie tym od których śię ſpodźiewaćie odebráć / jákąż łáſkę maćie? ábowiem y grzeƺnicy grzeƺnikom pożycżáją / áby záśię tyle odebráli.
Owƺem miłujćie nieprzyjaćioły wáƺe / y cżyńćie im dobrze / y pożycżajćie / nic śię z tąd nie ſpodźiewájąc ; á będźie wielka zapłátá wáƺá / y będźiećie ſynámi Nawyżƺego : ábowiem on dobrotliwy jeſt przećiw niewdźięcżnym y złym.
Przetoż / bądźćie miłośierni / jáko y Oćiec wáƺ miłośierny jeſt.
Nie ſądźćie / á nie będźiećie ſądzeni : Nie potępiajćie / á nie będźiećie potępieni : odpuƺcżajćie / á będźie wam odpuƺcżono.
Dawajćie / á będźie wam dano : miárę dobrą / nátłocżoną / y potrzęśioną / y opływájącą dádzą ná łono wáƺe. Abowiem tąż miárą którą mierzyćie / będźie wam záś odmierzono.
Y powiedźiał im podobieńſtwo ; Izali może ślepy ślepego prowádźić? Azaż nie obádwá w dół wpádną?
Nie jeſtći ucżeń nád miſtrzá ſwego ; lecż doſkonáły będźie káżdy / <i>będźieli</i> jáko miſtrz jego.
A cżemuż widźiƺ źdźbło w oku brátá twego : á bálki która jeſt w oku twojim nie bacżyƺ?
Albo jákoż możeƺ rzec brátu twemu? Bráćie / dopuść yż wyjmę źdźbło które jeſt w oku twojim ; á ſam bálki która jeſt w oku twojim nie widźiƺ? Obłudniku / wyjmij pierwey bálkę z oká twego : á tedy przejrzyƺ / ábyś wyjął źdźbło które jeſt w oku brátá twego.
Nie jeſt bowiem drzewo dobre / które przynośi owoc zły : áni jeſt drzewo złe / które przynośi owoc dobry.
Gdyż káżde drzewo / z owocu właſnego poznáne bywa : boć nie zbieráją z ćiernia figów / áni z głogu zbieráją winnych gron.
Cżłowiek dobry z dobrego ſkárbu ſercá ſwego / wynośi rzecży dobre ; á zły cżłowiek ze złego ſkárbu ſercá ſwego / wynośi rzecży złe : ábowiem z obfitośći ſercá mówią uſtá jego.
Przecżże mię tedy zowiećie PAnie / PAnie ; á nie cżynićie tego co mówię?
Káżdy który przychodźi do mnie / á ſłucha ſłów mojich / y cżyni je / pokażę wam komu jeſt podobnym.
Podobny jeſt cżłowiekowi dom budującemu ; który kopał / y wykopał głęboko / á záłożył grunty ná opoce : á gdy przyƺłá powódź / otrąćiłá śię rzeká o on dom / ále nie mogłá go poruƺyć ; bo był záłożony ná opoce.
Ale który ſłucha á nie cżyni / podobny jeſt cżłowiekowi który zbudował dom ſwój ná źiemi bez gruntu : o który śię otrąćiłá rzeká / y zárázem upadł á był upadek domu onego wielki.
