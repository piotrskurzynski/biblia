A przybliżáło śię Święto Przáſników / które zowią Wielkanoc.
Y ƺukáli przedniejƺy Kápłani y Náucżeni w piśmie / jákoby go zábili : ále śię bali ludu.
Y wſtąpił Szátan w Judaƺá którego zwano Iƺkáryotem / który był z licżby dwunaści.
<i>Ten</i> tedy odƺedƺy / z mówił śię z Przedniejƺymi Kápłanámi / y z Przełożonymi <i>Kośćiołá</i> jákoby go im wydał.
Y urádowáli śię / y umówili śię z nim / że mu chcą dáć pieniądze.
Y obiecał. Y ƺukał ſpoſobnego cżáſu áby go im wydał bez rozruchu.
Tedy przyƺedł dźień Przáſników / którego miał Báránek być zábity.
Y poſłał Piotrá y Janá mówiąc ; Poƺedƺy / nágotujćie nam Báránká / ábyſmy jedli.
Ale oni mu rzekli ; Gdźież chceƺ ábyſmy <i>go</i> nágotowáli?
A on rzekł do nich ; Oto gdy do miáſtá wchodźić będźiećie / potka śię z wámi cżłowiek nioſący dzban wody ; idźćież zá nim do domu / do którego wnidźie.
A rzecżćie goſpodarzowi domu onego : Kazałći powiedźieć Náucżyćiel ; Gdźie jeſt goſpodá / kędybym jadł Báránká z ucżniámi mojimi?
A on wam ukaże ſalę wielką uſłáną : támże nágotujćie.
Tedy odƺedƺy / ználeźli jáko im był powiedźiał : y nágotowáli Báránká.
A gdy przyƺłá godźiná / uśiadł zá ſtół y dwánaśćie Apoſtołów z nim.
Y rzekł do nich : Żądájąc żądałem tego Báránká jeść z wámi / pierwey niżbym ćierpiał.
Boć wam powiádam / że go więcey jeść nie będę / áż śię wypełni w króleſtwie Bożym.
A wźiąwƺy kielich / y podźiękowawƺy / rzekł ; Weźmićie to / á podźielćie miedzy śię.
Abowiem powiádam wam / że nie będę pił z rodzáju winney máćice / áż przydźie króleſtwo Boże.
A wźiąwƺy chleb / y podźiękowawƺy / łamał y dał im mówiąc : To jeſt ćiáło moje / które śię zá was dawa ; to cżyńćie ná pámiątkę moję.
Tákże y kielich gdy było po wiecżerzy / mówiąc : Ten kielich jeſt nowy Teſtáment we krwi mojey / która śię zá was wylewa.
Ale oto ręká tego / co mię wydawa / ze mną jeſt zá ſtołem.
Synći zájiſte cżłowiecży idźie / ták jáko jeſt poſtánowiono ; ále biádá cżłowiekowi temu / który go wydawa.
Tedy śię oni pocżęli miedzy ſobą pytáć o tym / ktoby wżdy z nich był / coby to ucżynić miał.
A wƺcżął śię też ſpór miedzy nimi o tym / ktoby śię z nich zdał być więkƺy?
Ale on im rzekł : Królowie narodów pánują nád nimi / á którzy nád nimi moc máją / dobrodźiejámi názywáni bywáją.
Lecż wy nie ták. Owƺem kto nawiękƺy jeſt miedzy wámi / niech będźie jáko namniejƺy ; á kto jeſt wodzem / <i>będźie</i> jáko ten co ſłuży.
Bo któryż więkƺy jeſt? Ten co śiedźi / cżyli ten co ſłuży? Izali nie ten co śiedźi? Alem Ja jeſt w pośrzodku was jáko ten co ſłuży.
A wy jeſteśćie / którzyśćie wytrwáli przy mnie w pokuſách mojich.
Y Jać wam ſporządzam / jáko mi ſporządźił Oćiec mój / Króleſtwo.
Abyśćie jedli y pili zá ſtołem mojim w króleſtwie mojim / y śiedźieli ná ſtolicách / ſądząc dwojenaśćie pokolenie Izráelſkie.
Y rzekł PAN ; Symonie Symonie / oto ƺátan wyprośił was / áby was odwiewał jáko pƺenicę.
Alem Ja prośił zá tobą / áby nie uſtáłá wiárá twojá. A ty niekiedy náwróćiwƺy śię / utwirdzaj bráćią twoję.
A on mu rzekł ; PAnie / Gotowem z tobą iść y do więźienia / y ná śmierć.
A on rzekł : Powiádamći Pietrze / Nie zápieje dźiś kur / áż śię pierwey trzy kroć záprzyƺ / że mię nie znaƺ.
Y rzekł im ; Gdym was poſyłał bez mieƺká / y bez tájſtry / y bez bótów / izali wam cżego niedoſtawáło? A oni rzekli ; Nicżego.
Tedy im rzekł : Ale teraz kto ma mieƺek / niech <i>go</i> weźmie / tákże y tájſtrę : á kto nie ma miecżá / niech przeda ſuknia ſwoję / á kupi <i>go</i>.
Abowiem powiádam wam / Yż śię jeƺcże muśi to co nápiſano wypełnić ná mnie / miánowićie ; Y z złocżyńcámi policżony jeſt Boć te rzecży które <i>świadcżą</i> o mnie / koniec biorą.
Ale oni rzekli ; PAnie / oto tu dwá miecże. A on im rzekł ; Doſyć jeſt.
Y wyƺedƺy / ƺedł według zwycżáju ná górę Oliwną / á ƺli zá nim y ucżniowie jego.
A gdy przyƺedł ná miejſce / rzekł im : Módlćie śię / ábyśćie nie weƺli w pokuƺenie.
A ſam oddalił śię od nich / jákoby ná ćiśnienie kámieniem : á klęknąwƺy ná koláná / modlił śię /
Mówiąc ; Ojcże / jeſli chceƺ przenieś ten kielich ode mnie : Wƺákże nie mojá wola / lecż twojá niech śię ſtánie.
Y ukazał mu śię Anjoł z niebá pośilájący go.
Ale będąc w boju / gorliwiey śię modlił. A był pot jego jáko krople krwie / zćiekájące ná źiemię.
A wſtawƺy od modlitwy / przyƺedł do ucżniów / <i>y</i> ználazł je ſpiące od ſmętku.
Y rzekł im : Cóż ſpićie? Wſtáńćie á módlćie śię / byśćie nie weƺli w pokuƺenie.
A gdy on jeƺcże mówił / oto zgrájá / y ten którego zwano Judaƺem jeden ze dwunaści ƺedł przed nimi / y przybliżył śię do JEzuſá áby go pocáłował.
A JEzus mu rzekł : Judaƺu / pocáłowániem wydawaƺ Syná cżłowiecżego?
A widząc ći którzy przy nim byli / co śię dźiać miáło / rzekli mu ; PAnie / mamyli bić miecżem?
Y uderzył jeden z nich ſługę Nawyżƺego kápłaná / y ućiął mu ucho práwe.
Ale JEzus odpowiádájąc / rzekł : Zániechajćie áż póty. A dotknąwƺy śię uchá jego / uzdrowił go.
Y rzekł JEzus do onych którzy byli przyƺli przeciwko niemu / do Przedniejƺych kápłanów / y do Przełożonych Kośćiołá / y do Stárƺych ; Wyƺliśćie jáko na zbójcę z miecżmi y z kijmi.
Gdym ná káżdy dźień bywał z wámi w Kośćiele / nie śćiągnęliśćie rąk ná mię : Ale táć jeſt oná godźiná wáƺá / y moc ćiemnośći.
Pojimawƺy go tedy / prowádźili go / y przyprowádźili go w dom Nawyżƺego Kápłaná. A Piotr ƺedł zá nim z dáleká.
A gdy oni rozniećili ogień w pośrzodku dworu / y weſpół uśiedli ; uśiadł y Piotr miedzy nimi.
A ujrzawƺy go niektóra dźiewká u ogniá śiedzącego / y pilnie mu śię przypátrzywƺy / rzekłá ; Y ten z nim był.
A on śię go záprzał / mówiąc ; Niewiáſto / nie znam go.
A po máłey chwili ujrzawƺy go drugi / rzekł ; Y tyś jeſt z nich. Ale Piotr rzekł ; Cżłowiecże nie jeſtem.
A gdy wyƺłá jákoby jedná godźiná / ktoś inƺy twirdźił / mówiąc : Prawdźiwie y ten z nim był ; bo też jeſt Gálilejcżyk.
A Piotr rzekł ; Cżłowiecże / nie wiem co mówiƺ. A zátym záraz gdy on jeƺcże mówił / kur zápiał.
A PAn obróćiwƺy śię / pojrzał ná Piotrá. Y wſpomniał Piotr ná ſłowo PAńſkie / jáko mu był powiedźiał ; Iż pierwey niż kur zápieje / trzy kroć śię mnie záprzyƺ.
A Piotr wyƺedƺy precż / gorzko płákał.
Lecż mężowie którzy weſpół trzymáli JEzuſá / náśmiewáli śię z niego / bijąc go.
A zákrywƺy go / bili twarz jego ; y pytáli go mówiąc : Prorokuj kto jeſt co ćię uderzył.
Y wiele innych rzecży bluźniąc mówili przećiwko niemu.
A gdy był dźień / zeƺli śię Stárƺy z ludu y Nawyżƺy kápłani / y Náucżeni w piśmie / á przywiedli go do Rády ſwojey /
Mówiąc ; Jeſliś ty jeſt CHryſtus / powiedz nam? Y rzekł im : Choćbym wam powiedźiał / nie uwierzyćie.
A jeſlibym też <i>o co</i> pytał / nie odpowiećie mi ; áni mię wypuśćićie.
Od tego cżáſu będźie Syn cżłowiecży śiedźiał ná práwicy mocy Bożey.
Y rzekli wƺyſcy : Tyś tedy jeſt on Syn Boży? A on rzekł do nich : Wy powiádaćie żem Ja jeſt.
A oni rzekli : Cóż jeƺcże potrzebujemy świádectwá? Wƺákieſmy ſámi ſłyƺeli z uſt jego.
