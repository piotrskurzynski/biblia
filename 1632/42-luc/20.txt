Y ſtáło śię z onych dni <i>dniá</i> jednego / gdy ucżył lud w Kośćiele y kazał Ewángelią / że nádeƺli Przedniejƺy kápłani / y Náucżeni w piśmie / z Stárƺymi.
Y rzekli do niego / mówiąc : Powiedz nam którą mocą to cżyniƺ? Albo kto jeſt <i>ten</i> coć dał tę moc?
A on odpowiádájąc rzekł do nich : Zpytam y Ja was o jednę rzecż / á powiedzćie mi :
Chrzeſt Janów byłli z niebá / cżyli z ludźi?
A oni myślili ſámi w ſobie / mówiąc : Jeſli powiemy / Z niebá / rzecże : Cżemużeśćie mu tedy nie wierzyli?
Jeſliż záśię rzecżemy / Z ludźi : Wƺyſtek lud ukámionuje nas / ponieważ zápewną máją / że Jan jeſt Prorokiem.
Y odpowiedźieli ; że nie wiedzą z kądby <i>był</i>.
A JEzus im rzekł : Y Ja wam nie powiem / którą mocą to cżynię.
Y pocżął do ludu mówić to podobieńſtwo : Cżłowiek niektóry náſádźił winnicę / y nájął ją winiárzom : y odiáchał precż ná cżás niemáły.
A cżáſu <i>ſwego</i> poſłał ſługę do onych winiárzów / áby mu dáli z pożytku oney winnice. Ale oni winiárze ubiwƺy go / odeſłáli próznego.
Y poſłał záśię drugiego ſługę ; ále oni y tego ubiwƺy y zelżywƺy / odeſłáli próżnego.
Y poſłał záśię trzećiego. Ale oni y tego zrániwƺy / wyrzućili precż.
A ták rzekł PAn oney winnice : Cóż ucżynię? poślę ſyná mego miłego : ſnadź gdy tego ujrzą / záwſtydzą śię.
Ale winiárze ujrzawƺy go / rzekli miedzy ſobą / mówiąc : Tenći jeſt dźiedźic : pódźćie / zábijmy go / áby náƺe było dźiedźictwo.
Y wypchnąwƺy go precż z winnice / zábili. Cóż im tedy ucżyni PAn oney winnice?
Przydźie / á potráći one winiárze / á winnicę odda innym. Co oni uſłyƺawƺy rzekli : Nie daj tego Boże!
Lecż on pojrzawƺy ná nie rzekł : Cóż tedy jeſt ono co nápiſano? Kámień który odrzućili budujący / ten śię ſtał głową węgielną.
Wƺelki który upádnie ná ten kámień / roztrąći śię : á ná kogoby on upadł / zetrze go.
Y ſtáráli śię Przedniejƺy kápłani y Náucżeni w piśmie / jákoby nań ręce wrzućili onejże godźiny ; ále śię ludu bali : ábowiem poználi iż przećiwko nim wyrzekł to podobieńſtwo.
A ták podſtrzegájąc go / poſłáli ƺpiegi / którzy zmyśláli jákoby byli ſpráwiedliwymi / áby go podchwyćili w mowie je° : á potym / áby go podáli Zwierżchnośći / y w moc Stárośćinę.
Y pytáli go / mówiąc : Náucżyćielu / wiemy że dobrze mówiƺ y ucżyƺ / áni przyjmujeƺ oſób : ále drogi Bożey w prawdźie ucżyƺ.
Godźili śię nam dáć cżynƺ Ceſárzowi / cżyli nie?
Ale on obacżywƺy chytrość ich / rzekł do nich : Cżemuż mię kuśićie?
Ukażćie mi groƺ : cżyj ma obraz y napis? A odpowiádájąc rzekli / Ceſárzów.
Zátym on im rzekł ; Oddawajćież tedy co jeſt Ceſárſkiego Ceſárzowi / á co jeſt Bożego Bogu.
Y nie mogli <i>go</i> podchwyćić w mowie jego przed ludem : á zádźiwiwƺy śię odpowiedźi jego / umilknęli.
A przyƺedƺy niektórzy z Sáduceuƺów ( którzy przecżą <i>y</i> mówią / iż niemáƺ <i>zmartwych</i>wſtánia ) pytáli go /
Mówiąc : Náucżyćielu / Mojzeƺ nam nápiſał : Jeſliby komu brát umárł / májąc żonę / á umárłby bez dźiatek / áby brát jego pojął onę żonę / á wzbudźił naśienie brátu ſwemu.
Było tedy śiedm bráći : Z których pierwƺy pojąwƺy żonę / umárł bez dźiatek.
Y pojął wtóry onę żonę : á umárł y ten bez dźiatek.
Potym ją pojął <i>y</i> trzeći ; tákże y oni wƺyſcy śiedm : á nie zoſtáwiwƺy dźiatek / pomárli.
Po wƺyſtkich też umárłá y oná niewiáſtá.
Przetoż przy <i>zmartwych</i>wſtániu któregoż z nich oná będźie żoną? Ponieważ śiedm ich mieli ją zá żonę.
Tedy odpowiádájąc rzekł im JEzus : Synowie tego wieku żenią śię / y zá mąż dawáją.
Ale ći którzy godni ſą áby doſtąpili onego wieku / y powſtánia od umárłych / áni śię żenić / áni zá mąż dawáć będą.
Abowiem umieráć więcey nie będą mogli : bo będą równi Anjołom / będąc Synámi Bożymi / gdyż ſą ſynámi <i>zmartwych</i>wſtánia.
A iż umárli zmartwychwſtáną / y Mojzeƺ pokazał przy onym krżu / gdy zowie PAná Bogá / Bogiem Abráhámowym / y Bogiem Izáákowym / y Bogiem Jákóbowym.
A Bógći nie jeſt <i>Bogiem</i> umárłych / ále żywych : bo jemu wƺyſcy żyją.
Tedy odpowiádájąc niektórzy z Náucżonych w piśmie / rzekli : Náucżyćielu / dobrześ powiedźiał.
Y nie śmieli go więcey o nic pytáć.
Y rzekł do nich : Jákoż powiádáją że CHryſtus jeſt Synem Dawidowym?
A ſam Dawid mówi w Kśięgách Pſálmów : Rzekł PAn PAnu memu / siądź po práwicy mojey /
Aż położę nieprzyjaćioły twoje podnóżkiem nóg twojich.
Ponieważ go tedy Dawid názywa PAnem : y jákoż jeſt Synem jego?
A gdy ſłuchał wƺyſtek lud / rzekł ucżniom ſwojim ;
Strzeżćie śię Náucżonych w piśmie / którzy chcą chodźić w ƺátách długich / y miłują pozdrawiánia ná rynkách / y pierwƺe ſtołki w bóżnicách / y pierwƺe miejſcá ná wiecżerzách :
Którzy pożeráją domy wdów / á <i>to</i> pod pokrywką długich modlitew. Ćić odnioſą ćiężƺy ſąd.
