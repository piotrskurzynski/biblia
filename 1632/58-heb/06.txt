Przetoż zániechawƺy pocżątecżnych náuk o CHryſtuśie / miejmy śię ku doſkonáłośći : nie znowu zákładájąc grunty pokuty od ucżynków martwych / y wiáry w Bogá.
Náuki o chrzćie / y o wkładániu rąk / y o powſtániu umárłych / y o ſądźie wiecżnym.
A to ucżyniemy / jeſli tylko Bóg dopuśći.
Abowiem nie można jeſt / áby ći którzy ſą raz oświeceni / y ſkoƺtowáli dáru niebieſkiego / y ucżeſnikámi śię ſtáli Duchá Świętego :
Skoƺtowáli też dobrego ſłowá Bożego / y mocy przyƺłego wieku ;
Gdyby odpádli : <i>áby</i> śię záś odnowili ku pokućie / jáko ći którzy ſobie znowu krzyżują Syná Bożego / y jáwnie go ſromocą.
Abowiem źiemiá / która cżęſto ná śię przychodzący deƺcż pije / y rodźi źiele przygodne tym / którzy ją ſpráwują / bierze błogoſłáwieńſtwo od Bogá :
Lecż która przynośi ćiernie y oſty / odrzucona jeſt / y bliſka przeklęctwá / która ná koniec <i>bywa</i> ſpalona.
A wƺákże namilejƺy / pewniſmy o was coś lepƺego y zbáwienia bliżƺego / choćiaż ták mówimy.
Abowiem nie jeſt Bóg nieſpráwiedliwy / áby zápámiętał prace wáƺey y prácowitey miłośći / którąśćie okazáli ku Imieniu jego ; gdyśćie ſłużyli świętym / y <i>jeƺcże</i> ſłużyćie.
A żądamy / áby káżdy z was toż ſtáránie pokázował / ku <i>nábyćiu</i> zupełney nádźieje áż do końcá :
Abyśćie nie byli gnuśnymi / ále náśládowcámi tych / którzy przez wiárę y ćierpliwość odźiedźicżyli obietnicę.
Abowiem Bóg obietnicę cżyniąc Abráhámowi / gdy nie miał przez kogo więkƺego przyśiąc / przyśiągł przez śiebie ſámego /
Mówiąc : Zájiſte błogoſłáwiąc błogoſłáwić ćię będę ; y rozmnażájąc / rozmnożę ćię.
A ták długo cżekájąc / doſtąpił obietnice.
Ludźieć wprawdźie przez więkƺego przyśięgáją : á przyśięgá która śię dźieje ku potwierdzeniu / jeſt miedzy nimi końcem wƺyſtkich ſporów.
Dla tego też Bóg chcąc doſtátecżnie okazáć dźiedźicom obietnice nieodmienność rády ſwojey / ucżynił ná to przyśięgę.
Abyſmy przez dwie rzecży nieodmienne / ( w których niemożna áby Bóg kłamał ) wárowną poćiechę mieli ; <i>my,</i> którzyſmy śię ućiekli ku otrzymániu wyſtáwioney nádźieje :
Którą mamy jáko kotwicę duƺe / y beſpiecżną / y pewną / y wchodzącą áż wewnątrz <i>zá</i> zaſłonę :
Gdźie Przewodnik dla nas wƺedł JEzus / ſtawƺy śię według porządku Melchiſedekowego Nawyżƺym Kápłanem ná wieki.
