Abowiem ten Melchiſedek <i>był</i> Król Sálem / Kápłan BOgá Nawyżƺego / który záƺedł drogę Abráhámowi / gdy śię wracał od porażki Królów ; y błogoſłáwił mu.
Któremu y dźieśięćiny <i>ze</i> wƺyſtkiego udźielił Abráhám : <i>który</i> naprzód wykłada śię Król ſpráwiedliwośći : potym też / Król Sálem ; co jeſt / Król pokoju.
Bez ojcá / bez mátki / bez rodu / áni pocżątku dni / áni końcá żywotá nie májąc : ále przypodobány będąc Synowi Bożemu / zoſtawa Kápłanem ná wieki.
Obacżćież tedy jáko wielki ten <i>był,</i> któremu też dźieśięćinę z łupów dał Abráhám Pátryárchá.
A ći którzy ſą z ſynów Lewiego urząd Kápłáńſki przyjmujący / rozkazánie máją / áby bráli dźieśięćinę od ludu według zakonu ; to jeſt / od bráći ſwojey / choć wyƺli z biódr Abráhámowych.
Ale ten którego ród nie jeſt pocżytány miedzy nimi / dźieśięćinę wźiął od Abráhámá / y temu który miał obietnicę błogoſłáwił.
A bez wƺelkiego ſporu mniejƺy od więkƺego błogoſłáwieńſtwo bierze.
A tuć dźieśięćiny biorą / ludźie którzy umieráją : tám záśię <i>on</i> o którym świadcżono iż żyje.
A iż ták rzekę / y ſam Lewi który dźieśięćiny bierze / dał w Abráhámie dźieśięćinę.
Abowiē jeƺcże był w biodrách ojcowſkich / gdy wyƺedł przećiwko niemu Melchiſedek.
A przetoż byłáli doſkonáłość przez kápłáńſtwo Lewitſkie / ( gdyż zá niego wydány jeſt zakon ludowi ) jákaż tego jeƺcże byłá potrzebá / áby inƺy kápłan według porządku Melchiſedekowego powſtał / á nie był według porządku Aáronowego miánowány?
A ponieważ kápłáńſtwo jeſt przenieśione / muśi też y Zakon przenieśiony bydź.
Bo ten o którym śię to mówi / inƺego jeſt pokolenia / z którego żaden nieſłużył Ołtarzowi.
Abowiem jáwna jeſt / iż z pokolenia Judowego poƺedł PAN náƺ : o którem pokoleniu nic z zſtrony kápłáńſtwá nie mówił Mojzeƺ.
Owƺem obfićie to jeƺcże <i>y z tego</i> jáwna jeſt / iż powſtał inƺy kápłan według porządku Melchiſedekowego :
Który śię ſtał nie według Zakonu przykazánia ćieleſnego / ále według mocy żywotá nieſkáźitelnego.
Abowiem <i>ták</i> świadcży : Tyś jeſt kápłanem ná wieki / według porządku Melchiſedekowego.
Bo śię ſtáło znieśienie onego przeƺłego przykazánia dla ſłábośći jego / y niepożytku :
Bo nicżego do doſkonáłośći nie przywiódł Zakon / ále <i>ná miejſce jego</i> wprowádzona jeſt lepƺa nádźiejá / przez którą śię przybliżámy <i>do</i> Bogá.
A to y względem tego / że nie bez przyśięgi <i>jeſt wprowádzona</i>.
Boć śię oni bez przyśięgi kápłany ſtawáli : á ten z przyśięgą przez tego / który rzekł do niego ; Przyśiągł PAn / á nie będźie tego żáłował / Tyś jeſt Kápłanem ná wieki / według porządku Melchiſedekowego.
Ták dálece lepƺego przymierza ſtał śię rękojmią JEzus.
Więc też onych wiele bywáło kápłany / dla tego iż im śmierć nie dopuśćiłá záwżdy trwáć :
Ale ten iż ná wieki zoſtawa / wiecżne ma kápłáńſtwo.
Przetoż y doſkonále zbáwić może te / którzy przezeń przyſtępują do Bogá : záwƺe żyjąc / áby orędował zá nimi.
Tákiegoć zájiſte przyſtało nam mieć Nawyżƺego Kápłaná / świętego / niewinnego / niepokalánego / odłącżonego od grzeƺników / y któryby śię ſtał wyżƺy nád niebioſá :
Któryby nie potrzebował ná káżdy dźień jáko oni Nawyżƺi Kápłani / pierwey zá ſwoje grzechy właſne / ofiar ſpráwowáć / á potym zá ludzkie : bo to ucżynił raz / ſámego śiebie ofiárowawƺy.
Abowiem zakon / ludźie podległe krewkośći / ſtánowił zá Nawyżƺe kápłany : ále ſłowo przyśięgi / które śię ſtáło po zakonie / <i>poſtanowiło</i> Syná <i>Bożego</i> doſkonáłego ná wieki.
