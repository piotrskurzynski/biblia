A Elizeuƺ Prorok záwołał jednego z Synów Prorockich / y rzekł mu : Przepaƺ biodrá twoje / á weźmi tę báńkę Olejku w rękę twą / á idź do Rámot Gáláádſkiego.
A gdy tám przydźieƺ / ujrzyƺ tám Jehu Syná Jozáfátowego / Syná Námſy / á wƺedƺy <i>tam</i> odwiedźieƺ go z pośrzodku bráći jego / y wprowádźiƺ go do gmáchu naſkrytƺego.
A wźiąwƺy báńkę Olejku / wylejeƺ ná głowę jego / y rzecżeƺ : Ták mówi PAN : Pomázałem ćię zá Królá nád Izráelem. A otworzywƺy drzwi / ućiecżeƺ / y nie zábáwiƺ śię tám.
Tedy odƺedł on młodźieniec ſługá Prorocki / do Rámot Gáláádſkiego.
A gdy przyƺedł / oto Hetmáni wojſk śiedźieli. Y rzekł ; Hetmánie / mam nieco z tobą mówić. Y rzekł Jehu : Z którymże ze wƺyſtkich nas? Y odpowiedźiał : Z tobą Hetmánie.
Tedy wſtawƺy / wƺedł do gmáchu / á on wylał Olejek ná głowę jego ; y rzekł mu : Ták mówi PAN Bóg Izráelſki ; Pomázałem ćię zá Królá nád ludem Páńſkim / nád Izráelem.
Y wytráćiƺ dom Achábá Páná twego : ábowiem pomƺcżę śię krwie ſług mojich proroków / y krwie wƺyſtkich ſług Páńſkich / z ręki Jezábele.
A ták zginie wƺyſtek dom Achábów / y wykorzenię z <i>domu</i> Achábowego / áż do namniejƺego ƺcżęnięćiá / y więźniá / y opuƺcżonego w Izráelu.
Y ucżynię domowi Achábowemu / jáko domowi Jeroboámá / Syná Nábátowego / y jáko domowi Báázy Syná Achiaƺowego.
Jezábelę też zjedzą pśi ná polu Jezreelſkim / á nie będźie ktoby ją pogrzebł. <i>To rzekƺy</i> otworzył drzwi / y ućiekł.
A gdy Jehu wyƺedł do ſług Páná ſwego / rzekł mu jeden : A dobrzeƺ wƺyſtko? Po cóż przychodźił ten ƺalony do ćiebie? A <i>on</i> jm odpowiedźiał : Wy znaćie tego mężá / y mowę jego.
Tedy rzekli : Nie prawdá to : proƺę powiedz nam. A on rzekł : Ták á ták rzekł do mnie mówiąc : Ták mówi PAn : Pomázałem ćię zá Królá nád Izráelem.
Poſpieƺyli śię tedy / á wźiąwƺy káżdy ƺátę ſwą kłádli je podeń ná nawyżƺym ſtopniu : Y zátrąbiwƺy w trąbę / mówili : Króluje Jehu!
Tedy śię ſprzyśiągł Jehu Syn Jozáfátá / Syná Námſy przećiw Jorámowi : ( A ná ten cżás Jorám ſtrzegł Rámot Gáláádſkiego / on / y wƺyſtek Izráel / przed Házáelem Królem Syryjſkim.
Ale śię był wróćił Król Jorám / áby śię lecżył w Jezreel ná rány które mu byli zádáli Syryjcżycy / gdy walcżył z Házáelem Królem Syryjſkim. ) Y rzekł Jehu : Jeſli śię wam zda niech nie wychodźi nikt z miáſtá / żeby ƺedł co oznájmić w Jezreelu.
Y wśiadł ná wóz Jehu / y jáchał do Jezreelá / bo tám Jorám leżał : Ochozyaƺ tákże Król Judſki / przyjáchał był áby náwiedźił Jorámá.
W tym ſtróż / który ſtał ná wieży w Jezreelu / ujrzawƺy pocżet Jehu przyjeżdżájący / rzekł : Pocżet jákiś widzę. Y rzekł Jorám : Weźmi jezdnego / á wyśli przećiwko nim / áby śię ſpytał / jeſli pokój :
A ták bieżał jezdny przećiwko niemu / y rzekł : Ták mówi Król / A pokój? y odpowiedźiał Jehu : Co tobie do pokoju? Obróć śię / <i>jedź</i> zá mną. Przetoż oznájmił ſtróż / mówiąc. Dojáchałći poſeł do nich ; ále śię nie wraca.
Zátym poſłał drugiego jezdnego / który przyjáchawƺy do nich / rzekł : Ták mówi Król : A pokój? odpowiedźiał Jehu : Co tobie do pokoju? Obróć śię á jedź zá mną.
Znowu oznájmił <i>to</i> ſtróż / mówiąc : Przyjáchałći do nich / ále śię nie wraca : A przyjazd jego / jákoby przyjazd Jehu Syná Námſy : bo ƺalenie jedźie.
Tedy rzekł Jorám : Záprzągaj. Y záprzężono w wóz jego. Y wyjáchał Jorám Król Izráelſki / y Ochozyaƺ Król Judſki / káżdy ná woźie ſwym. A wyjáchawƺy przećiw Jehu / tráfili go ná polu Nábotá Jezreelſkiego.
A gdy ujrzał Jorám Jehu / rzekł : Jeſtże pokój Jehu? Y odpowiedźiał : Co zá pokój? <i>ponieważ</i> jeƺcże cudzołóſtwá Jezábele mátki twojey / y cżáry jey wielkie <i>ſą</i>?
Przetoż obróćiwƺy śię Jorám ućiekał / mówiąc do Ochozyaƺá ; Zdrádá Ochozyaƺu!
Tedy Jehu wźiąwƺy w ręce ſwoje łuk poſtrzelił Jorámá miedzy rámioná jego / áż przeƺłá ſtrzałá przez ſerce jego / ták / że padł ná woźie ſwojim.
Potym rzekł <i>Jehu</i> do Bádákerá Hetmáná ſwego : Weźmi go / á porzuć ná polu Nábotá Jezreelſkiego : Abowiem pámiętaƺ / gdyſmy ja y ty jácháli ſpołu zá Achábem Ojcem jego / że PAN wydał był przećiwko niemu tę pogróƺkę.
Zájiſte krwie Nábotá y krwie Synów jego / którąm widźiał wcżorá rzekł PAN / pomƺcżę śię nád tobą ná tym polu. PAN <i>to</i> rzekł. Przetoż teraz weźmi go / á porzuć go ná polu / według ſłowá Páńſkiego.
Co Ochozyaƺ Król Judſki ujrzawƺy / ućiekał drogą do domu ogrodowego. Ale go gonił Jehu / y rzekł : Y tego zábjićie ná woźie jego. <i>Y zranili go</i> ná wſtępie Guru / który jeſt podle Jebláám : A ućiekł do Mágeddy / y tám umárł.
Y kazáli go záwieść ſłudzy jego do Jeruzalem / á pogrzebli go w grobie jego z Ojcámi jego w mieśćie Dawidowym.
A roku jedenaſtego Jorámá Syná Achábowego / królował Ochozyaƺ nád Judą.
Zátym przyƺedł Jehu do Jezreel. Co gdy Jezábelá uſłyƺáłá / ufárbowáłá twarz ſwoję / y ochędożyłá głowę ſwą / á pátrzáłá z okná.
A gdy Jehu wjeżdżał w bramę / rzekłá : Jeſtże pokój / o Zimry morderzu Páná ſwego?
A on podnióżƺy twarz ſwoję ku oknu / rzekł : Któż ze mną <i>trzyma,</i> kto? Tedy wejrzeli nań dwá <i>álbo</i> trzej komornicy <i>jey</i>.
Którym rzekł ; Zrzuććie ją. Y zrzućili ją : Y popryſkáłá śię śćiáná y konie krwią jey : y podeptał ją.
A gdy <i>tám</i> wƺedł / jadł y pił / y rzekł : Obacżćie / proƺę / onę przeklętą / á pogrzebćie ją : boć córką królewſką jeſt.
Tedy ƺedƺy áby ją pogrzebli / nie ználeźli z niey jedno cżáƺkę z głowy / y nogi / y dłoni rąk.
A wróćiwƺy śię / oznájmili mu to : który rzekł : <i>Wypełniło śię</i> ſłowo Páńſkie które powiedźiał przez ſługę ſwego Eliaƺá Tesbitę / mówiąc : Ná polu Jezreel zjedzą pśi ćiáło Jezábeli.
Niech będźie trup Jezábeli / jáko gnój ná roli / ná polu Jezreel ; ták / żeby nie mówiono ; Táć jeſt Jezábelá.
