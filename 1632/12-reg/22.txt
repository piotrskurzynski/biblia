Ośm lat było Jozyaƺowi / gdy pocżął królowáć : á trzydźieśći y jeden rok królował w Jeruzalem : á imię mátki jego było Jedydá Córká Adájá z Beſekátu.
Y cżynił co było dobrego przed ocżymá Páńſkimi / chodząc wƺyſtkimi drogámi Dawidá Ojcá ſwego / á nie uchylał śię / áni ná práwo / áni ná lewo.
A ośmnaſtego roku Królá Jozyaƺá / poſłał Król Sáfáná / Syná Azáliaƺowego / Syná Meſſulámá piſárzá do domu Páńſkiego / mówiąc :
Idź do Helkiaƺá / Kápłaná nawyżƺego / áby zebrał pieniądze / które wnoƺono do domu Páńſkiego / które wybieráli ſtróżowie progu od ludu.
A niech je dawáją w ręce rzemieśników przełożonych nád robotą domu Páńſkiego / áby je dawáli robotnikom którzy robili w domu PAŃſkim / náprawiájąc ſkázę domu :
<i>To jeſt,</i> budownikom y ćieślom y murárzom / y ná zkupowánie drzewá / y kámienia ćioſánego ku nápráwie domu.
Wƺákże / niech nie cżynią licżby z pieniędzy / które dawáją do ręku ich / bo oni wiernie nimi ƺáfowáć będą.
Y rzekł Helkiaƺ / Kápłan nawyżƺy / do Sáfáná piſárzá : Kśięgi zakonu ználazłem w domu Páńſkim. Y dał Helkiaƺ one Kśięgi Sáfánowi : y cżytał je.
Przyƺedƺy tedy Sáfán piſarz do Królá / odnióſł to Królowi / y rzekł : Zebráli ſłudzy twoji pieniądze / które śię ználázły w domu <i>PAŃſkim,</i> y oddáli je w ręce rzemieśników / przełożonych nád robotą w domu Páńſkim.
Oznájmił też Sáfán piſarz Królowi / mówiąc ; Dał mi Kśięgę Helkiaƺ Kápłan : y cżytał ją Sáfán przed Królem.
A gdy uſłyƺał Król ſłowá Kśiąg Zakonu / rozdárł ƺáty ſwe.
Y rozkazał Król Helkiaƺowi Kápłanowi / y Ahikámowi Synowi Sáfánowemu / y Achborowi Synowi Micheaƺowemu / y Sáfánowi piſárzowi / y Aſájaƺowi ſłudze ſwemu / mówiąc ;
Idźćie poradźćie śię PANA o mię / y o lud / y o wƺyſtkiego Judę z ſtrony ſłów tych Kśiąg / które ſą ználeźione / bo wielki jeſt gniew Páńſki który śię zápalił przećiwko nam ; przeto iż nie poſłucháli Ojcowie náƺy ſłów tych Kśiąg żeby cżynili według wƺyſtkiego / co nam jeſt nápiſano.
A ták poƺedł Helkiaƺ Kápłan / y Ahikám / y Achbor / y Sáfán / y Azájaƺ / do Huldy Prorokini żony Sellumá Syná Tekui / Syná Arááƺowego / który był ſtróżem ƺat / á oná mieƺkáłá w Jeruzalem ná drugiey ſtronie <i>miáſtá,</i> y mówili z nią.
Która rzekłá do nich ; Ták mówi PAN Bóg Izráelſki / Powiedzćie mężowi który was poſłał do mnie.
Ták mówi PAN : Oto ja przywiodę złe ná to miejſce / y ná obywátele jego / <i>według</i> wƺyſtkich ſłów Kśiąg tych / które cżytał Król Judſki :
Przeto że mię opuśćili / y kádźili Bogom cudzym / áby mię draźnili wƺyſtkimi ſpráwámi rąk ſwojich : dla cżego rozpaliłá śię popędliwość mojá przećiwko miejſcu temu / y nie będźie ugáƺona.
A Królowi Judſkiemu / który was poſłał o rádę do PANA / ták powiedzćie : Ták mówi PAN Bóg Izráelſki <i>o</i> ſłowách któreś ſłyƺał :
Ponieważ zmiękcżone jeſt ſerce twoje / á upokorzyłeś śię przed oblicżem PAŃſkim / ſłyƺąc com powiedźiał przećiwko temu miejſcu / y przećiwko obywátelom jego / iż ma przyść w ſpuſtoƺenie / y w przeklęctwo : y rozdárłeś ƺáty twe / á płákałeś przedemną : y jam <i>ćię</i> też wyſłuchał / mówi PAN.
Przetoż oto ja ćię zbiorę do Ojców twojich / á będźieƺ zebrány do grobu twego w pokoju / áby nie oglądáły ocży twoje wƺyſtkiego złego które ja przywiodę ná to miejſce. Y odnieśiono tę rzecż Królowi.
