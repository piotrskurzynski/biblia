Jeſli tedy <i>maćie</i> jáką poćiechę w CHryſtuśie / jeſli jáką ućiechę miłośći / jeſli jáką ſpołecżność duchá / jeźli ſą jákie wnętrznośći zlitowánia <i>w was</i> :
Dopełnćie weſela mojego / ábyśćie jednoż rozumieli / jednoſtájną miłość májąc / <i>będąc</i> jednomyślni y jednoż rozumiejący :
Nic nie cżyniąc ſpornie álbo <i>przez</i> prózną chwałę / ále w pokorze jedni drugie májąc zá wyżƺe nád śię.
Nie upátrujćie káżdy <i>tylko</i> co jeſt jego / ále káżdy też co jeſt drugich.
Tego tedy bądźćie o ſobie rozumienia / które <i>było</i> y w CHryſtuśie JEzuśie.
Który będąc w kƺtałćie Bożym / nie pocżytał ſobie tego zá drapieſtwo równym być Bogu.
Ale wyniƺcżył ſámego śiebie / przyjąwƺy kƺtałt niewolniká / ſtawƺy śię podobny ludźiom :
Y poſtáwą ználeźiony jáko cżłowiek / ſam śię poniżył / będąc poſłuƺnym áż do śmierći / á to śmierći krzyżowey.
Dla tegoteż Bóg náder go wywyżƺył / y dárował mu Imię / które jeſt nád wƺelkie imię.
Aby w Imieniu JEzuſowym / wƺelkie śię koláno ſkłaniáło / tych którzy ſą ná niebieśiech / y tych którzy ſą ná źiemi / y tych którzy ſą pod źiemią.
A wƺelki język <i>áby</i> wyznawał / że JEzus CHryſtus jeſt PAnem ku chwale Bogá Ojcá.
Przetoż moji mili / jákośćie záwƺe poſłuƺni byli / nie tylko w przytomnośći mojey / ále teraz dáleko więcey w niebytnośći mojey / z bojáźnią y ze drżeniem zbáwienie ſwoje ſpráwujćie.
Abowiem Bóg jeſt który ſpráwuje w was y chcenie y ſkutecżne wykonánie / według upodobánia <i>ſwego</i>.
Wƺyſtko cżyńćie bez ƺemránia y poſwarków.
Abyśćie byli bez nágány / y ƺcżerymi / dźiatkámi Bożymi nienágánionymi w pośrzodku narodu złego y przewrotnego ; między którymi świećićie / jáko świátłá ná świećie.
Záchowywájąc ſłowá żywotá / ku chlubie mojey w dźień CHryſtuſów / żem dármo nie bieżał y dármo nie prácował.
Ale choćbym ofiárowány był dla ofiáry y uſługi wiáry wáƺey / weſelę śię / y ſpół-weſelę śię ze wƺyſtkimi wámi.
Z tegoż tedy y wy weſelćie śię / y ſpół-weſelćie śię ze mną.
A mam nádźieję w PAnu JEzuśie / iż Tymoteuƺá w rychle poślę do was ; ábym śię y ja ućieƺył / dowiedźiawƺy śię co śię z wámi dźieje.
Abowiem nie mam nikogo w umyśle <i>jemu</i> równego / któryby śię uprzyjmie o rzecży wáƺe ſtáráć chćiał.
Bo wƺyſcy ſwojich rzecży ƺukáją ; á nie tych które ſą CHryſtuſá JEzuſá.
Ale wiećie iż on jeſt doświadcżonym / á iż jáko ſyn z ojcem / ze mną ſłużył w Ewángeliey.
Mam tedy nádźieję / że tego do was poślę ; ſkoro obacżę co śię ze mną dáliey dźiáć będźie.
A mam dufność w PAnu / że y ſam w rychle do was przydę.
Alem rozumiał rzecżą potrzebną Epáfrodytá / brátá y pomocniká y ſpół-bojowniká mego / á wáƺego Apoſtołá y ſługę w potrzebie mojey / poſłáć do was.
Ponieważ prágnął was wƺyſtkich / y bárzo śię fráſował / żeśćie ſłyƺeli iż záchorzał.
Bo wprawdźie chorował / máło nie ná śmierć : Ale śię Bóg zmiłował nád nim ; á nie tylko nád nim / ále y náde mną ; ábym nie miał ſmutku ná ſmutek.
Przetoż tym ochotniey poſłałem go / ábyśćie záśię ujrzawƺy go / uweſelili śię ; á ja ábym miał mniey ſmutku.
Przyjmićież go tedy w PAnu ze wƺelákim weſelem / á tákie w pocżćiwośći miejćie.
Boć dla dźiełá CHryſtuſowego bliſkim był śmierći / odważywƺy zdrowie ſwoje / áby dopełnił tego cżego nie doſtawáło w uſłudze wáƺey przećiwko mnie.
