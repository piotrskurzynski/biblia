Stáło śię záś ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! zádáj zágádkę / á mów w podobieńſtwie o domu Izráelſkim /
Y rzecż : Ták mówi pánujący Pán : Orzeł wielki z wielkiemi ſkrzydłámi y z długiemi pióry / pełen pierzá pſtrego / przylećiał ná Libán / y wźiął wierzch cedru.
Wierzch młodoćiánych látorośli jego ułámał / y zánióſł go do źiemi kupieckiej / á w mieśćie kupieckiem położył go.
Wźiął też náśieniá oney źiemi / á wſádźił je ná polu urodzájnem / á wſádźił je bárzo oſtrożnie przy wodách wielkich ;
Które weƺłoby było / y byłoby winną máćicą bujną / choć niſkiego wzroſtu ; y byłyby látorośli jey ku niemu / á korzenie jey byłyby mu poddáne. A ták byłoby było máćicą winną / któráby byłá wydáłá látorośli / y wypuśćiłá gáłązki.
Ale był orzeł jeden wielki z wielkiemi ſkrzydłámi y z gęſtem pierzem / á oto oná winná máćicá przypoiłá korzenie ſwoje ku niemu / y gáłązki ſwe rozćiągnęłá do niego / áby ją odwilżał z brózd ſádu ſwego.
Choć ná polu dobrem przy wodách wielkich wſádzoná byłá / áby wypuśćiłá látorośli / y przynioſłá owoce / y byłá máćicą winną wſpániáłą.
Rzecżże tedy : Ták mówi pánujący Pán : Izali śię jey poƺcżęśći? Izali orzeł korzeniá jey nie wyrwie / y owocu jey nie oberwie / y nie poſuƺy? Izali wƺyſtkich látorośli wyroſłych z niej nie uſuƺy? Izali z wielką mocą á z obfitym ludem jey nie wygłádźi z korzeniá jey?
Oto jákożkolwiek wſádzoná jeſt / izali śię jey poƺcżęśći? Izali do ƺcżętu nie uſchnie / ſkoro śię jey dotknie wiátr wſchodni? Izali przy brózdách / przy których śię przyjęłá / nie uſchnie?
Zátem ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Mów teraz do domu odpornego : Azaż nie wiećie / co to jeſt? Zátem mów : Oto przyćiągnął król Bábilońſki do Jeruzálemu / y zábrał królá jego / y kśiążąt jego / y záwiódł ich z ſobą do Bábilonu.
Wźiął też z náśieniá królewſkiego / á ucżyniwƺy z nim przymierze / záwiązał go przyśięgą / y mocnych z oney źiemi zábrał /
Aby było króleſtwo zniżone / przeto / żeby śię nie wynioſło / żeby ták ſtrzegąc przymierza jego / ſtáło.
Ale mu śię ſprzećiwił / poſłáwƺy poſłów ſwych do Egiptu / áby mu dał koni y wiele ludu. Izali śię to poƺcżęśći? Izali pomſty ujdźie ten / co ták ucżynił? Ten / który wzruƺá przymierze / izali pomſty ujdźie?
Jáko żyję Ja / mówi pánujący Pán / że ná miejſcu królá tego / który go królem ucżynił / którego przyśięgą wzgárdźił / á którego przymierze wzruƺył / u niego w Bábilonie umrze.
Ani mu Fáráo z wielkim wojſkiem y z ludem gęſtym co pomoże ná wojnie / gdy uſypie wał / y porobi ƺáńce / áby wykorzenił mnóſtwo ludu /
Ponieważ pogárdźił przyśięgą / zrzućiwƺy przymierze ; bo oto dał ná to rękę ſwą / á przećie to wƺyſtko ucżyni ; nie ujdźie pomſty.
Dla tego ták mówi pánujący Pán : Jáko żyję Ja / że przyśięgę ſwoję / którą wzgárdźił / y przymierze moje / które wzruƺył / pewnie obrócę ná głowę jego.
Bo rozćiągnę nań śieć moję / á będźie niewodem moim pojmány / y záwiodę go do Bábilonu / á tám śię z nim rozſądzę o wyſtępek jego / którym wyſtąpił przećiwko mnie.
A wƺyſcy też / którzy ućiekli od niego ze wƺyſtkiemi hufámi jego / od miecżá polegną / á pozoſtáli ná wƺyſtkie ſtrony rozproƺeni będą / á dowiećie śię / że Ja Pán mówiłem to.
Ták mówi pánujący Pán : Wƺákże wezmę z wierzchu tego wyſokiego cedru / y wſádzę ; z wierzchu młodoćiánych látorośli jego młodą látoroſłkę ułámię / á wƺcżepię ją ná górze wyſokiej y wynioſłej ;
Ná wyſokiej górze Izráelſkiej wƺcżepię ją / y wypuśći gáłązki / y wydá owoc / y będźie cedrem zácnym ; y będźie pod nim mieƺkáć wƺelkie ptáſtwo / y wƺyſtko / co ma ſkrzydłá / pod ćieniem gáłęźiá jego mieƺkáć będźie.
A ták poznáją wƺyſtkie drzewá polne / żem Ja Pán poniżył drzewo wyſokie / á wywyżƺyłem drzewo niſkie / uſuƺyłem drzewo źielone / á ucżyniłem / że zákwitło drzewo ſuche. Ja Pán rzekłem to / y ucżynię.
