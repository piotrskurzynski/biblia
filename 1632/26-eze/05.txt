Potem ty / ſynu cżłowiecży! weźmij ſobie nóż oſtry / to jeſt brzytwę bárwierſką / weźmij ją ſobie / á ogól nią głowę ſwoję y brodę ſwoję ; potem weźmij ſobie wágę / y rozdźiel je.
Jednę trzećią cżęść ogniem ſpál w pośród miáſtá / gdy śię wypełnią dni oblężeniá ; potem wźiąwƺy drugą trzećią cżęść / pośiekáj miecżem około niego / á oſtátnią trzećią cżęść roztrząśnij ná wiátr ; bo miecżá dobędę zá nimi.
A wƺákże weźmij z nich jáką trochę / y záwiąż w kráje ƺáty twojey.
A y z tych jeƺcże wźiąwƺy wrzuć je w pośród ogniá / y ſpál je w ogniu / ſkąd wynijdźie ogień ná wƺyſtek dom Izráelſki.
Ták mówi pánujący Pán : Toć jeſt Jeruzálem / którem poſtáwił w pośród pogán / á zewƺąd otocżył kráinámi ;
Ale odmieniło ſądy moje w niezbożność / bárdźiej niż pogánie / á uſtáwy moje bárdźiej niż inne kráiny / które ſą około niego ; bo ſądámi mojemi pogárdźili / y w uſtáwách mojich nie chodźili.
Przetoż ták mówi pánujący Pán : Dla tego / żeśćie pogáńſtwo grzechámi przewyżƺyli / które jeſt około was / á w uſtáwách mojich nie chodźiliśćie / y ſądów mojich nie záchowáliśćie / náwet áni ták jáko pogánie / którzy ſą około was / ſąduśćie nie cżynili :
Przetoż ták mówi pánujący Pán : Otom Ja przećiwko tobie / otom Ja / y wykonam w pośrzodku ćiebie ſądy przed ocżymá pogán ;
Bo ucżynię przy tobie to / cżegom pierwej nie ucżynił / y cżego nápotem nie ucżynię dla wƺyſtkich obrzydliwośći twojich /
Ták / że ojcowie jeść będą ſynów w pośrzodku ćiebie / á ſynowie jeść będą ojców ſwojich ; y wykonam ſądy przećiwko tobie / á rozproƺę wƺyſtkie oſtátki twoje ná wƺyſtkie ſtrony.
Przetoż jáko żyję Ja / mówi pánujący Pán : Poniewáżeś ty świątnicę moję ſplugáwiło wƺelákiemi niecżyſtośćiámi twojimi / y wƺelákiemi obrzydliwośćiámi twemi / y Ja ćię też podám w lekkość / á nie ſfolgujeć oko moje / áni śię zlituję.
Trzećiá cżęść z ćiebie morem pomrze y głodem wyginie w pośrzodku ćiebie / á drugá trzećiá cżęść od miecżá pádnie około ćiebie / á trzećią oſtátnią cżęść ná wƺyſtkie ſtrony rozproƺę / y miecżá dobędę zá nimi.
A ták śię do końcá wyleje zápálcżywość mojá / y nátrę popędliwośćią ſwoją ná nich / y ućieƺę śię / y doznáją / żem Ja Pán / którym to wyrzekł w zápálcżywośći mojey / gdy wypełnię gniew ſwój nád nimi.
A podám ćię w ſpuſtoƺenie y w poháńbienie národom / które ſą około ćiebie / przed ocżymá káżdego mijájącego.
A ták będźieƺ ná háńbę / ná pośmiech / ná ſrogi przykłád y ná zdumienie národom / które ſą około ćiebie / gdy wykonam przećiwko tobie ſądy w popędliwośći y w gniewie y w ſrogiem zágniewániu. Ja Pán mówiłem.
Gdy wypuƺcżę ſrogie ſtrzáły głodu ná zgubę wáƺę / które wypuƺcżę / ábym was wygubił / á głód zgromádzę przećiwko wam / y złámię wam láſkę chlebá.
Poślę zájiſte ná was głód / y zwierzętá okrutne / które ćię ośierocą ; y mór y krew przydźie ná ćię / gdy ná ćię miecż przywiodę. Ja Pán mówiłem.
