Wiódł mię potem ku bramie / która bramá pátrzyłá ku drodze ná wſchód ſłońcá.
A oto chwałá Bogá Izráelſkiego przychodźiłá drogą od wſchodu / á ƺum jey był jáko ƺum wód wielkich / á źiemiá śię lśniłá od chwały jego.
A podobne było ono widzenie / którem widźiał / cále onemu widzeniu / którem widźiał / gdym przychodźił / ábym pſuł miáſto ; widzenie / mówię / podobne onemu widzeniu / którem widźiał u rzeki Chebár / y upádłem ná twárz moję.
A gdy chwałá Páńſka wchodźiłá do domu drogą bramy / która pátrzáłá ku drodze ná wſchód ſłońcá /
Tedy mię podnióſł duch / y wwiódł mię do śieni wewnętrzney / á oto dom pełen był chwały Páńſkiej.
Y uſłyƺáłem / á oto mówiono do mnie z domu / á on mąż ſtał podle mnie /
Y mówił do mnie : Synu cżłowiecży! miejſce ſtolicy mojey / y miejſce ſtóp nóg mojich / ná którem mieƺkáć będę w pośród ſynów Izráelſkich ná wieki ; nie ſplugáwi więcey dom Izráelſki imienia świętobliwośći mojey / áni oni / áni królowie ich wƺetecżeńſtwy ſwemi y trupámi królów ſwych / áni wyżynámi ſwemi.
Gdy kłádli próg ſwój podle progu mego / á podwoje ſwoje podle podwoi mojich / á śćiánę miedzy mną y miedzy ſobą ; á ták ſplugáwiáli imię świętobliwośći mojey obrzydliwośćiámi ſwemi / które cżynili / przetożem ich zniƺcżył w popędliwośći mojey.
Ale teraz niech odrzucą wƺetecżeńſtwo ſwoje / y trupy królów ſwojich odemnie / á będę mieƺkał w pośrzodku ich ná wieki.
Ty / ſynu cżłowiecży! powiedz domowi Izráelſkiemu o tym domu / á niech śię wſtydzą zá niepráwośći ſwoje / y niech ſobie rozmierzą wizerunek jego.
A jeſliby śię záwſtydźili zá wƺyſtko / co cżynili / tedy im oznájmij wizerunek domu tego / y wymiár jego / wyjśćie jego / y wejśćie jego / y wƺyſtkie kƺtáłty jego / y wƺyſtkie uſtáwy jego / wƺyſtkie / mówię / kƺtáłty / y wƺyſtkie práwá jego / á nápiƺ przed ocżymá ich / áby przeſtrzegáli wƺyſtkiego kƺtáłtu jego / y wƺyſtkich uſtáw jego / y cżynili je.
A tenći jeſt zakon domu tego : Ná wierzchu góry wƺyſtko ogrodzenie jego wƺędźie w około nájświętƺe jeſt ; oto tenći jeſt zakon domu tego.
A teć ſą pomiáry ołtárzá według tychże łokći / á miárá łokćiá ná łokieć y ná dłoń ; podſtáwek jego ná łokieć wzwyż / á wƺerz tákże ná łokieć / á krániec jego áż do kráju jego w około był ná piędź jedną ; á táć byłá wyſtáwá ołtárzá ;
To jeſt od podſtáwku / który był przy źiemi / áż do przepáſánia niżƺego / dwá łokćie / á ƺerokość ná jeden łokieć ; á od mniejƺego przepáſánia áż do przepáſánia więkƺego cżtery łokćie / á ƺerokość ná łokieć ;
Ale ſám ołtárz niech będźie ná cżtery łokćie / á z ołtárzá w górę cżtery rogi.
A ołtárz ná dwánaśćie łokći wdłuż / á ná dwánaśćie wƺerz cżworográniáſty po cżterech ſtronách ſwojich ;
A przepáſánie jego ná cżternaśćie łokći wdłuż / á ná cżternaśćie wƺerz / po cżterech ſtronách jego / á krániec około niego ná pół łokćiá / á podſtáwek jego ná łokieć w około / á wſchód jego ná wſchód ſłońcá.
Y rzekł do mnie : Synu cżłowiecży! ták mówi pánujący Pán : Teć ſą uſtáwy około ołtárzá w dźień / w który zbudowány będźie / áby ná nim ofiárowano cáłopalenia / y krwią ná nim kropiono.
A kápłánom Lewitom / którzy ſą z náśieniá Sádokowego / á przyſtępują do mnie / mówi pánujący Pán / áby mi ſłużyli / dáƺ ćielcá młodego ná ofiárę zá grzech ;
Weźmieƺ tedy ze krwi jego / á włożyƺ ná cżtery rogi jego / y ná cżtery węgły przepáſánia / y ná krániec w około ; á ták go ocżyśćiƺ / y poświęćiƺ go.
Potem weźmieƺ onego ćielcá zá grzech / á ſpáliƺ go ná miejſcu poſtánowionem w onym domu zewnątrz przed świątnicą.
A wtórego dniá będźieƺ ofiárował kozłá z kóz bez wády zá grzech / y ocżyƺcżą ołtárz / ták jáko go ćielcem ocżyśćili.
A gdy dokońcżyƺ ocżyƺcżánia / będźieƺ ofiárował ćielcá młodego bez wády / y báráná z trzody bez wády.
Które gdy ofiárowáć będźieƺ przed Pánem / wrzucą kápłáni ná nie ſoli / y ucżynią z nich ofiárę cáłopalenia Pánu.
Przez śiedm dni będźieƺ ofiárował kozłá zá grzech ná káżdy dźień / tákże y ćielcá młodego / y báráná z trzody bez wády ofiárowáć będą /
Przez śiedm dni ocżyƺcżáć będą ołtárz / y ocżyƺcżą go / á poświęcą ręce ſwoje.
A wypełniwƺy te dni / óſmego dniá nápotem ſpráwowáć będą kápłáni ná ołtárzu cáłopalenia wáƺe / y ſpokojne ofiáry wáƺe / y przyjmę was łáſkáwie / mówi pánujący Pán.
