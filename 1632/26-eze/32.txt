A dwunáſtego roku / mieśiącá dwunáſtego / pierwƺego dniá tegoż mieśiącá / ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! podnieś láment nád Fáráonem / królem Egipſkim / á powiedz mu : Podobnyś ty lwowi młodemu miedzy národámi / tyś jeſt jáko wieloryb w morzu / gdyż bujájąc po rzekách twojich mąćiƺ wodę nogámi twemi / y mięƺáƺ rzeki jego.
Ták mówi pánujący Pán : Rozćiągnę też ná ćię śieć moję przez zebránie wielu narodów / którzy ćię wyćiągną niewodem moim ;
Y zoſtáwię ćię ná źiemi / ná polu porzucę ćię / y ſpráwię / że mieƺkáć będźie ná tobie wƺelkie ptáſtwo niebieſkie / y nákármię tobą zwierzá wƺyſtkiej źiemi ;
Y rozrzucę mięſo twoje po górách / y nápełnię doliny wyſokośćią twoją /
Y nápoję źiemię twoję / w którey pływáƺ / krwią twoją áż do gór / ták / że y rzeki będą nápełnione tobą.
A gdy ćię zgáƺę / zákryję niebioſá / y ćiemne ucżynię gwiázdy ich / ſłońce obłokiem záſłonię / á kśiężyc nie da świátłá ſwego.
Wƺyſtkie świátłá jáſne ná niebioſách záćmię dla ćiebie / y przywiodę ćiemność ná źiemię twoję / mówi pánujący Pán.
Nádto záſmucę ſerce wielu narodów / gdy zá ſpráwą moją przydźie wieść o ſtárćiu twojim miedzy narody / do źiem / którycheś nie znał.
Ucżynię / mówię / że śię zdumieje nád tobą wiele narodów / á królowie ich zátrwożą śię bárzo dla ćiebie / gdy ƺermowáć będę miecżem ſwoim przed twárzą ich ; będą śię zájiſte lękáć co chwilá káżdy o duƺę ſwoję w dźień upádku twego.
Bo ták mówi pánujący Pán : Miecż królá Bábilońſkiego przydźie ná ćię.
Miecżámi mocárzów poráżę zgráję twoję ; nájokrutniejśi ze wƺyſtkich narodów / ći ſkáżą pychę Egipſką / y będźie zgłádzone wƺyſtko mnóſtwo jego.
Zágłádzę y wƺyſtko bydło jego / które jeſt przy wodách wielkich / ták / że ich więcey nie zmąći nogá cżłowiecżá / áni kopytá bydlęce mąćić ich będą.
Tedy ucżynię / że śię wody ich uſtoją / á rzeki ich jáko oliwá pójdą / mówi pánujący Pán.
Gdyż obrócę źiemię Egipſką w ſpuſtoƺenie / á źiemiá będźie wypróżnioná z pełnośći ſwojey / gdy pobiję wƺyſtkich mieƺkájących w niej ; y dowiedzą śię / żem Ja Pán.
Toć jeſt láment / którym nád nią lámentowáć będą ; córki narodów nárzekáć będą nád nią / nád Egiptem y nád wƺyſtkim mnóſtwem jego nárzekáć będą / mówi pánujący Pán.
Potem dwunáſtego roku / piętnáſtego dniá tegoż mieśiącá / ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! nárzekáj nád mnóſtwem Egipſkiem / á zepchnij je / y córki tych narodów ſłáwnych áż do nájniżƺych miejſc źiemi / do tych / co zſtępują do dołu /
Y mów : Nád kogożeś wdźięcżniejƺy? Zſtąp / á połóż śię z nieobrzezáńcámi.
W pośród pobitych miecżem upádną ; pod miecż podány jeſt / wywlecżćież go ze wƺyſtką zgráją jego.
Mówić do niego będą nájmocniejśi z mocárzów z pośrzodku grobu z pomocnikámi jego / którzy tám zſtąpiwƺy / leżą z nieobrzezáńcámi pobitymi miecżem.
Tám jeſt Aſſur / y wƺyſtká zgrájá jego / w około niego ſą groby jego ; wƺyſcy ći pobići upádli od miecżá.
Którego groby położone ſą przy ſtronách dołu / áby byłá zgrájá jego w około grobu jego ; ći wƺyſcy pobići polegli od miecżá / którzy puƺcżáli ſtrách w źiemi żyjących.
Tám Elám y wƺyſtká zgrájá jego około grobu jego / ći wƺyſcy pobići upádli od miecżá / którzy zſtąpili w nieobrzezce do niſkośći źiemi / którzy puƺcżáli ſtrách ſwój w źiemi żyjących ; jużći odnoƺą háńbę ſwoję z tymi / którzy zſtępują do dołu.
W pośrzodku pobitych poſtáwili mu łoże / y wƺyſtkiej zgrái jego / w około niego ſą groby jego ; wƺyſcy ći nieobrzezáńcy pobići ſą miecżem / przychodźił ſtrách ich ná źiemię żyjących / jużći odnoƺą háńbę ſwoję z tymi / którzy zſtąpili do grobu / á w pośrzodku pobitych położeni ſą.
Tám Meſech / Tubál y wƺyſtká zgrájá jego / y w około niego groby jego / ći wƺyſcy nieobrzezáńcy pobići miecżem / choć puƺcżáli ſtrách ſwój w źiemi żyjących.
Acżkolwiekći jeƺcże nie polegli z mocárzámi / którzy upádli z nieobrzezáńców / co zſtąpili do grobu z wojennym orężem ſwoim / y położyli miecże ſwe pod głowy ſwe : á wƺákże przydźie niepráwość ich ná kośći ich / choćiáż ſtrách tych mocárzów był w źiemi żyjących.
Y ty w pośrzodku nieobrzezáńców ſtárty będźieƺ / y będźieƺ leżał miedzy pobitymi miecżem.
Tám Edom / y królowie jego / y wƺyſcy kśiążętá jego / którzy położeni ſą / z mocą ſwoją y z pobitymi miecżem ; y ći z nieobrzezáńcámi leżeć będą / y z tymi / którzy zſtępują do dołu.
Tám wƺyſcy zgołá kśiążętá północney ſtrony / y wƺyſcy Sydońcżycy / którzy zſtępują do pobitych z ſtráchem ſwoim / zá moc ſwoję wſtydźić śię będą / y leżeć będą ći nieobrzezáńcy z pobitymi miecżem / á odnioſą háńbę ſwoję z tymi / którzy zſtępują do dołu.
Tych ujrzáwƺy Fáráo ućieƺy śię nád wƺyſtką zgráją ſwoją / która jeſt miecżem pobitá / Fáráo y wƺyſtko wojſko jego / mówi pánujący Pán.
Bo puƺcżę ſtrách mój w źiemi żyjących / y położony będźie miedzy nieobrzezáńcámi z pobitymi miecżem Fáráo y wƺyſtká zgrájá jego / mówi pánujący Pán.
