Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! obróć twárz ſwoję ku Jeruzálemowi / á krop jáko roſą przećiwko miejſcom świętym / y prorokuj przećiwko źiemi Izráelſkiej /
A powiedz źiemi Izráelſkiej : Ták mówi pánujący Pán : Otom Ja przećiwko tobie / y dobędę miecżá mego z pochew jego / y wytnę z ćiebie ſpráwiedliwego y niezbożnego.
Abym tedy wyćiął z ćiebie ſpráwiedliwego / y niezbożnego / przeto wynijdźie miecż mój z pochew ſwojich ná wƺelkie ćiáło od południá áż ku północy /
Y dowie śię wƺelkie ćiáło / żem Ja Pán dobył miecżá mego z pochew jego ; nie náwróći śię więcey.
A ty / ſynu cżłowiecży! wzdycháj / jákobyś miał biodro złámáne / á w gorzkośći wzdycháj przed ocżymá ich.
A gdyć rzeką : Dlácżego wzdycháƺ? Tedy odpowieƺ : Dla wieśći / która idźie / ná którą śię rozpłynie wƺelkie ſerce / y oſłábieją wƺelkie ręce / y śćiśniony będźie wƺelki duch / y wƺelkie koláno rozpłynie śię jáko wodá ; oto idźie / y ſtánie śię / mówi pánujący Pán.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! prorokuj / á rzecż : Ták mówi Pán : Powiedz / miecż / miecż náoſtrzony jeſt y wypolerowány ;
Wyoſtrzony jeſt ná pomordowánie ku zábićiu náznácżonych / wypolerowány jeſt / áby śię lśnił. Izali śię weſelić mamy? Gdyż rózgá ſyná mego káżde drzewo lekce wáży.
Dáłći go ná wypolerowánie / áby mógł być ujęty ręką / jeſtći wyoſtrzony ten miecż / jeſt y wypolerowány / áby dány był do ręki zábijájącego.
Wołáj á kwil / ſynu cżłowiecży! przeto / że ten miecż będźie przećiwko ludowi memu / ten też przećiwko wƺyſtkim kśiążętom Izráelſkim ; ſtráchy miecżá przyjdą ná lud mój ; przetoż śię w biodrę uderz.
Gdym ich kárał / cóż to pomogło? Y nie mámże rózgi wƺyſtko lekce powáżájącey ná nich wyćiągnąć? mówi pánujący Pán.
Ty tedy / ſynu cżłowiecży! prorokuj / á bij ręką w rękę ; bo powtóre y potrzećie miecż przydźie / miecż mordujących / ten miecż mordujących bez litośći / przenikájący áż do pokojów ich.
We wƺyſtkich bramách ich dáłem ſtrách miecżá / áby śię rozpłynęło ſerce / y upádków śię námnożyło. Ach! wypolerowány jeſt / áby śię błyƺcżał / á wyoſtrzony / áby zábijał.
Zbierz śię miecżu / udáj śię ná práwo y ná lewo / gdźiekolwiek jeſt chęć twárzy twojey.
Boć y Ja uderzę ręką moją w rękę moję / y uſpokoję rozgniewánie moje. Ja Pán mówiłem to.
Wtem śię ſtáło ſłowo Páńſkie do mnie / mówiąc :
A ty / ſynu cżłowiecży! połóż przed ſobą dwie drogi / którędyby iść miał miecż królá Bábilońſkiego ; z jedney źiemi niech wychodzą obiedwie / á ná rozdrożu obierz tę ku miáſtu / tę obierz.
Pokáż drogę / którąby miecż iść miał / cżyli ku Rábbáćie ſynów Ammonowych / cżyli ku źiemi Judzkiej ná Jeruzálemſkie twierdze.
Abowiem ſtánie król Bábilońſki ná rozdrożu / ná pocżątku dwu dróg / pytájąc śię wieƺcżby ; będźie polerował ſtrzáły / będźie śię rádźił báłwánów / będźie pátrzył ná wątrobę.
Po práwej ręce jego wieƺcżbá ukáże Jeruzálem / áby ƺykował hetmánów / którzyby pobudzáli do mordowánia / á podnośili głos z okrzykiem / áby záſádźili tárány przećiwko bramom / áby uſypáli wał / á urobili ƺáńce.
Y będą to mieli zá próżną wieƺcżbę przed ocżymá ſwemi ći / co śię obowiązáli przyśięgámi ; á to przywiedźie ná pámięć niepráwość ich / áby pojmáni byli.
Przetoż ták mówi pánujący Pán : Dla tego / że ná pámięć przywodźićie niepráwość ſwoję / á odkrywá śię przeſtępſtwo wáƺe / ták / że jáwne ſą grzechy wáƺe we wƺyſtkich ſpráwách wáƺych / przeto / mówię / że ná pámięć przychodźićie / tą ręką pojmáni będźiećie.
A ty / niecżyſty bezbożniku / kśięćiu Izráelſki! którego dźień przychodźi / gdy niepráwość ſkońcżoná będźie :
Ták mówi pánujący Pán : Zdejm tę cżápkę / á zrzuć tę koronę / która już nigdy tákowá nie będźie ; tego / który w poniżenie przyƺedł / wywyżƺę / á wywyżƺonego poniżę.
W niwecż / w niwecż / w niwecż ją obrócę / cżego pierwej nie bywáło / áż przydźie ten / co do niej ma práwo / którem mu dał.
Ale ty / ſynu cżłowiecży! prorokuj / á mów : Ták mówi pánujący Pán o ſynách Ammonowych y o háńbie ich ; rzecż mówię : Miecż / miecż dobyty jeſt / ku zábijániu wypolerowány jeſt / áby wytrácał wƺyſtko / y áby śię błyſkał.
A choćiáż ći opowiádáją márność / y wróżąć kłámſtwo / áby ćię przyłożyli do ƺyi niezbożników pobitych / których dźień przychodźi / gdy niepráwość / ſkońcżoná będźie /
Schowáj jednák miecż do pochew jego ; ná miejſcu / ná któremeś ſpłodzoná / w źiemi mieƺkánia twego / ſądźić ćię będę.
Y wyleję ná ćię rozgniewánie moje ; ogniem popędliwośći mojey ná ćię dmucháć będę / y podám ćię w ręce ludźi zuchwáłych y przemyślnych ná wytrácenie.
Ogniowi potráwą będźieƺ / krew twojá będźie w pośrzodku źiemi / nie będźieƺ więcey wſpomináná ; bo Ja Pán mówiłem to.
