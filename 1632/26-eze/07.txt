Potem ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Słucháj ty / ſynu cżłowiecży : Ták mówi pánujący Pán o źiemi Izráelſkiej : Koniec / koniec przyƺedł ná wƺyſtkie cżtery ſtrony źiemi.
Teraz przydźie koniec ná ćię : poślę ná ćię popędliwość moję / y będę ćię ſądźił według dróg twojich / y zwálę ná ćię wƺyſtkie obrzydliwośći twoje.
A nie ſfolgujeć oko moje / y nie zmiłuję śię / ále drogi twoje zwálę ná ćię / á obrzydliwośći twoje będą w pośrzodku ćiebie / y poznáćie / żem Ja Pán.
Ták mówi pánujący Pán : Utrápienie jedno / oto utrápienie ſtráƺne przychodźi ;
Koniec przychodźi / przychodźi koniec / ocknął śię przećiwko tobie / oto przychodźi.
Przychodźi prędki poránek ná ćię / o obywátelu źiemi! przychodźi ten cżás / przybliżá śię ten dźień grzmotu / á nie głoſu rozlegájącego śię po górách.
Już prędko / już wyleję gniew mój ná ćię / á wykonam zápálcżywość moję nád tobą / á oſądzę ćię według dróg twojich / y włożę ná ćię wƺyſtkie obrzydliwośći twoje.
Nie ſfolgujeć zájiſte oko moje / áni śię zlituję / ále według dróg twojich nágrodzęć / y obrzydliwośći twoje w pośrzodku ćiebie będą ; á ták poznáćie / żem Ja Pán / który biję.
Oto ten dźień / oto przyƺedł ; przyƺedł prędki poránek / zákwitnęłá rózgá / wybijá śię pychá.
Okrućieńſtwo wyroſło w rózgę niezbożnośći ; nie zoſtánie z nich nic / áni z mnóſtwá ich / áni z huku ich / y nie będźie żádnego nárzekánia nád nimi.
Idźie cżás / przybliżá śię dźień. Kto kupi / nie będźie śię weſelił / á kto ſprzedá / nie będźie żáłował ; bo popędliwość przydźie ná wƺyſtko mnóſtwo jey.
Bo kto ſprzedał / nie wróći śię do rzecży ſprzedáney choćby jeƺcże miedzy żyjącymi był żywot ich ; ponieważ widzenie ná wƺyſtko mnóſtwo jey nie wróći śię / á żáden w niepráwośći żywotá ſwego nie zmocni śię.
Trąbić będą w trąbę / y wƺyſtko przygotują / jednák nie będźie kto miał iść ná wojnę ; bo popędliwość mojá obórzy śię ná wƺyſtko mnóſtwo jey.
Miecż zewnątrz / á mór y głód będźie wewnątrz ; kto będźie ná polu / od miecżá umrze ; á kto w mieśćie / głód y mór go pożre.
A którzy z nich ućiekną / ći będą ná górách jáko gołębice w dolinie ; wƺyſcy będą nárzekáli / káżdy nád niepráwośćią ſwoją.
Wƺyſtkie ręce oſłábieją / y wƺyſtkie śię koláná rozpłyną jáko wodá.
Y obloką śię w wory / y okryje ich ſtrách / y ná wƺelkiej twárzy będźie wſtyd / y ná wƺyſtkich głowách ich łyśiná.
Srebro ſwoje po ulicách rozrzucą / á złoto ich będźie jáko niecżyſtość ; ſrebro ich y złoto ich nie będźie ich mogło wybáwić w dźień popędliwośći Páńſkiej ; nie náſycą duƺy ſwojey / y wnętrznośći ſwych nie nápełnią / przeto / że im jeſt ku obráżeniu niepráwość ich ;
A iż w ſłáwie ozdoby ſwojey / którą ná chwałę ſwoję Bóg wyſtáwił / obrázów obrzydliwośći ſwojich y ſproſnośći ſwojich nácżynili / przetożem im je w niecżyſtość obróćił ;
Y podám je w ręce cudzoźiemców ná rozchwycenie / y niezbożnym w źiemi ná łup / którzy ją ſplugáwią ;
Odwrócę też twárz moję od nich / á zgwáłcą świątnicę moję / á wnijdą do niej rozbójnicy / y ſplugáwią ją.
Ucżyń łáńcuch ; bo źiemiá pełna jeſt krwáwych ſądów / á miáſto pełne jeſt krzywd.
Przetoż nájgorƺych z pogán przywiodę / áby pośiedli domy ich ; y ucżynię wſtręt pyƺe mocárzów / á ſplugáwieni będą / którzy je poświęcáją.
Zginienie przyƺło ; przetoż ƺukáć będą pokoju / ále go nie będźie.
Ućiſk zá ućiſkiem przydźie / á wieść zá wieśćią przypádnie ; y będą ƺukáć widzeniá od proroká! ále zakon zginie od kápłáná / á rádá od ſtárców.
Król płákáć będźie / á kśiążę w ſmutek oblecżony będźie / á ręce ludu w źiemi przeſtráƺone będą. Według drogi ich ucżynię im / y według ſądów ich oſądzę ich / y poznáją / żem Ja Pán.
