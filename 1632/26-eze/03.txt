Y rzekł do mnie : Synu cżłowiecży / Co przed tobą jeſt / zjedz : zjedz tę kśięgę / á idź / y mów do domu Izráelſkiego.
Otworzyłem tedy uſtá ſwe / y dał mi zjeść one kśięgi :
A mówił do mnie ; Synu cżłowiecży / nákarm brzuch twój / á wnętrznośći twoje nápełni tymi kśięgámi któreć dawam : y zjadłem <i>je,</i> y były w uśćiech mojich ſłodkie jáko miód.
Zátym rzekł do mnie : Synu cżłowiecży / idź / <i>á</i> wnidź do domu Izráelſkiego / y mów ſłowy mojimi do nich.
Bo ćię nie do ludu nieznájomey mowy / álbo trudnego języká poſyłam / <i>ále</i> do domu Izráelſkiego :
Nie do wiela narodów nieznájomey mowy / y trudnego języká / którychbyś ſłów nie zrozumiał : Którzy jednák / gdybym ćię do nich poſłał / uſłucháliby ćię :
Lecż dom Izráelſki nie będźie ćię chćiał uſłucháć / ponieważ mnie <i>ſámego</i> uſłucháć nie chcą : bo wƺyſtek dom Izráelſki jeſt twárdego cżołá y zátwárdzonego ſercá.
<i>Ale,</i> otom ucżynił twarz twoję twárdą przećiwko twarzy ich : á cżoło twe twárde przećiwko cżołu ich.
Ucżyniłem cżoło twe jáko Diáment / <i>y</i> twárdƺe nád ſkáłę : Nie bójże śię ich / áni śię lękaj twarzy ich / przeto / że ſą domem odpornym.
Y rzekł do mnie : Synu cżłowiecży / wƺyſtkie ſłowá moje / które mówić będę do ćiebie / przyjmi do ſercá twego / á ſłuchaj uƺymá twymi.
Idź / á wnidź do pojimánych / do ſynów ludu twego / y mów do nich / á powiedz im : Ták mówi Pánujący PAn : Niech oni ſłucháją / álbo nie.
Tedy mię Duch podnióſł / y ſłyƺałem zá ſobą głos grzmotu wielkiego ; Błogoſłáwiona <i>niech będźie</i> chwałá PAńſka z miejſcá ſwego :
Y ƺum ſkrzydeł onych zwierząt które śię náſpół dotykáły / y głos kół ná przećiwko nim / y głos grzmotu wielkiego.
A Duch podnióſł mię y wźiął mię. Y odƺedłem z gorzkośćią w rozgniewániu duchá mego : ále ręká PAńſka nádemną mocna byłá.
Y przyƺedłem do pojimánych do Telábib / którzy mieƺkáli przy rzece Chebár / y śiadłem gdźie oni mieƺkáli / á śiedźiałem tám śiedm dni / w pośrzodku ich zdumiawƺy śię.
A gdy minęło śiedm dni / było ſłowo PAńſkie do mnie mówiące :
Synu cżłowiecży / Dałem ćię ſtróżem domowi Izráelſkiemu / ábyś ſłyƺąc ſłowo z uſt mojich / nápominał ich odemnie.
Gdybym Ja rzekł niepobożnemu / Śmierćią umrzeƺ ; á nie nápomniałbyś go / <i>y</i> nie mówiłbyś / ábyś go odwiódł od niezbożney drogi jego / ták / żebyś go przy żywoćie záchował : Tedy / onći niezbożny w niepráwośći ſwojey umrze / ále krwie jego z ręki twojey ƺukáć będę.
Lecż jeſlibyś ty nápomniał niezbożnego / á nie odwróćiłby śię od niezbożnośći ſwey / y od drogi ſwey niezbożney : Tedy / onći w niepráwośći ſwojey umrze / ále ty duƺę ſwoję wybáwiƺ.
<i>Także, jesliby</i> śię odwróćił ſpráwiedliwy od ſpráwiedliwośći ſwojey / á cżyniłby niepráwość / á Jabym położył záwádę przed nim / y tákby umárł / á tybyś go nie nápomniał : w grzechu ſwym umrze / á nie przydą ná pámięć ſpráwiedliwośći jego które cżynił : Lecż krwie jego z ręki twojey ƺukáć będę.
Ale jeſlibyś ty nápomniał ſpráwiedliwego / áby nie grzeƺył <i>ten</i> ſpráwiedliwy / y nie grzeƺyłby / zájiſte żyć będźie / bo nápomnienie przyjął : á ty duƺę ſwoję wybáwiƺ.
Y byłá tám nádemną ręká PAńſka / y rzekł do mnie : Wſtawƺy wynidź w pole / á tám śię z tobą rozmówię.
A ták wſtawƺy ƺedłem w pole / á oto chwałá PAńſka ſtałá tám / jáko chwałá którąm widźiał u rzeki Chebár : y upadłem na oblicże moje.
Tedy Duch wſtąpił w mię : á poſtáwiwƺy mię ná nogi moje / mówił do mnie / y rzekł mi ; Wnidź / zámkni śię w domu ſwym.
Bo oto ná ćię / Synu cżłowiecży / włożą powrozy / y zwiążą ćię nimi / á nie będźieƺ mógł wyniść miedzy nie.
A Ja ucżynię / że język twój przylgnie do podniebienia twego : y będźieƺ niemy / á nie będźieƺ im mężem ſtrofującym / przeto / że ſą domem odpornym.
Ale gdy z tobą mówić będę / otworzę uſtá twoje / tedy rzecżeƺ do nich ; Ták mówi Pánujący PAN : Kto <i>chce</i> ſłucháć / niech ſłucha : á kto nie chce / niech nie ſłucha : gdyż domem odpornym ſą.
