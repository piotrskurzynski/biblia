Potem przyƺedƺy do mnie mężowie z ſtárƺych Izráelſkich / uśiedli przedemną.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! ći mężowie złożyli plugáwe báłwány ſwoje do ſercá ſwego / á niepráwość / która im jeſt ku obráżeniu / położyli przed twárzą ſwoją ; mniemáƺże / iż mię uprzejmie pytáją o rádę?
Dlátegoż powiedz im / y mów do nich : Ták mówi pánujący Pán : Ktokolwiek z domu Izráelſkiego położył plugáwe báłwány ſwoje w ſercu ſwojim / á niepráwość / która mu jeſt ku obráżeniu / położył przed twárzą ſwoją / á przyƺedłby do proroká / Ja Pán odpowiem temu / który przydźie / o mnóſtwie plugáwych báłwánów jego.
Abym ułápał dom Izráelſki w ſercu ich / że śię odwróćili odemnie do plugáwych báłwánów ſwojich wƺyſcy zgołá.
Przetoż rzecż do domu Izráelſkiego : Ták mówi pánujący Pán : Náwróććie śię / á cofnijćie śię od plugáwych báłwánów wáƺych / y od wƺyſtkich obrzydliwośći wáƺych odwróććie twárz ſwoję.
Bo ktobykolwiek z domu Izráelſkiego y z przychodniów / którzy mieƺkáją w Izráelu / odwróćił śię od náśládowánia mnie / á położyłby plugáwe báłwány ſwoje w ſercu ſwem / y niepráwość / która mu jeſt ku obráżeniu / położyłby przed twárzą ſwoją / y przyƺedłby do proroká / áby mię przezeń pytał / Ja Pán odpowiem mu ſám przez śię /
Y poſtáwię oblicże moje przećiw temu mężowi / y dam go ná znák / y ná przyſłowie / á wytrácę go z pośrzodku ludu mojego ; y dowiećie śię / żem Ja Pán.
A gdyby śię prorok dał zwieść / áby mówił ſłowo / Ja Pán zwiodłem proroká onego / y wyćiągnę nań rękę ſwoję / y wygłádzę go z pośrzodku ludu mego Izráelſkiego.
Y ták ponioſą niepráwość ſwoję ; jáká jeſt káźń ná tego / któryby śię pytał / táká też káźń ná proroká będźie /
Aby więcey nie błądźił dom Izráelſki odemnie / á nie mázáli śię więcey żádnemi przeſtępſtwámi ſwojimi / áby byli ludem moim / á Ja ábym był Bogiem ich / mówi pánujący Pán.
Znowu ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! gdyby źiemiá zgrzeƺyłá przećiwko mnie / dopuƺcżájąc śię przeſtępſtwá / tedy jeſlibym wyćiągnął rękę moję ná nią / á złámáłbym jey láſkę chlebá / y poſłáłbym ná nią głód / y wytráćiłbym z niej ludźi y zwierzętá ;
Choćby byli w pośrzodku niej ći trzej mężowie / Noe / Dánijel y Ijob / oni w ſpráwiedliwośći ſwojey wybáwiliby duƺe ſwe / mówi pánujący Pán.
Tákże jeſlibym zły zwierz przepuśćił ná źiemię / á ośieroćiłby ją / y byłáby ſpuſtoƺoná / żeby jey nikt przechodźić nie mógł dla zwierzá /
Jáko żyję Ja / mówi pánujący Pán / że choćby ći trzej mężowie byli w pośrzodku jey / żádną miárąby nie wybáwili ſynów áni córek : oniby tylko ſámi wybáwieni byli / lecżby źiemiá ſpuſtoƺoná byłá.
Albo jeſlibym przywiódł miecż ná tę źiemię / á rzekłbym miecżowi : Przejdź przez tę źiemię / ábym wytráćił z niej ludźi y zwierzętá /
Jáko żyję Ja / mówi pánujący Pán / że choćby ći trzej mężowie byli w pośrzodku jey / żádną miárąby nie wybáwili ſynów áni córek / áleby oni tylko ſámi byli wybáwieni.
Albo poſłáłlibym mór ná tę źiemię / y wyláłbym popędliwość ſwoję ná nią ku wytráceniu / áby z niej ludźie y zwierzętá byli wytráceni.
Że choćby Noe / Dánijel y Ijob byli w pośrzodku jey / jáko żyję Ja / mówi pánujący Pán / żádną miárąby áni ſyná áni córki nie wybáwili / oniby tylko w ſpráwiedliwośći ſwojey wybáwili duƺe ſwe.
Owƺem / ták mówi pánujący Pán : Choćbym cżtery káźni moje ćiężkie / miecż / y głód / y zły zwierz / y mór poſłał ná Jeruzálem / ábym wytráćił z niego ludźi y zwierzętá /
A oto jeſliby zoſtáli w nim / którzyby tego uƺli / á wywiedźieni byli ſynowie álbo córki / oto y oni muƺą iść do was ; y oglądáćie drogę ich / y ſpráwy ich / á ućieƺyćie śię nád tym złem / które przywiodę ná Jeruzálem / nád wƺyſtkim / co przywiodę ná nie.
A ták ućieƺą was / gdy ujrzyćie drogę ich y ſpráwy ich ; y zrozumiećie / żem to wƺyſtko nie dármo ucżynił / com ucżynił w nim / mówi pánujący Pán.
