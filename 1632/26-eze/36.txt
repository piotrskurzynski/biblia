A ty ſynu cżłowiecży! prorokuj o górách Izráelſkich / á mów : Góry Izráelſkie! ſłuchájćie ſłowá Páńſkiego ;
Ták mówi pánujący Pán : Przeto / iż nieprzyjáćiel rzekł o was : Hej / hej! y wyſokośći wiecżne doſtáły śię nam w dźiedźictwo ;
Przetoż prorokuj á mów : Ták mówi pánujący Pán : Dla tego / dla tego / mówię / iż was zburzyli á pochłonęli zewƺąd / y ſtáliśćie śię dźiedźictwem pozoſtáłym národom / y przyƺliśćie ná język y ná obmowiſko ludzkie ;
Przetoż / góry Izráelſkie! ſłuchájćie ſłowá pánującego Páná. Ták mówi pánujący Pán górom y págórkom / ſtrumieniom y dolinom / puſtyniom / obálinom / y miáſtom opuƺcżonym / które ſą ná ſplundrowánie / y ná pośmiewiſko oſtátkowi narodów okolicżnych.
Dla tego / ták mówi pánujący Pán / zápráwdę w ogniu zápálcżywośći mojey mówić będę przećiw oſtátkom tych narodów / y przećiwko wƺyſtkiej źiemi Edomſkiej / którzy ſobie przywłáƺcżyli źiemię moję zá dźiedźictwo z weſelem cáłego ſercá / y z ochotnem puſtoƺeniem / áby śiedliſko wygnánych jego było ná rozƺárpánie.
Przetoż prorokuj o źiemi Izráelſkiej / á mów górom / y págórkom / ſtrumieniom y dolinom : Ták mówi pánujący Pán : Oto Ja w zápálcżywośći mojey / y w popędliwośći mojey mówię / dla tego / iż háńbę od narodów ponośićie.
Przetoż ták mówi pánujący Pán : Jam podnióſł rękę moję / iż te narody / które ſą zewƺąd około was / ſámi háńbę ſwoję ponioſą.
A wy góry Izráelſkie / wypuśćićie gáłązki ſwe / y owoc ſwój przynieśćie ludowi memu Izráelſkiemu / gdy śię przybliżą á przyjdą.
Bo oto Ja idę do was / y náwrócę śię do was / á upráwiáne y pośiewáne będźiećie ;
Y rozmnożę ná was ludźi / wƺyſtek zgołá dom Izráelſki ; y będą mieƺkáć w miáſtách / á miejſcá zburzone pobudowáne będą.
Rozmnożę / mówię / ná was ludźi y dobytek / á rozmnożą śię y uroſną ; y ſpráwię / że mieƺkáć będźiećie jáko zá dáwnych cżáſów wáƺych / y lepiej wam cżynić będę niż przed tym / y dowiećie śię / żem Ja Pán.
Bo przyprowádzę ná was ludźi / lud mój Izráelſki / y pośiądą ćię / y będźieƺ im dźiedźictwem / á więcey ich nie ośieroćiƺ.
Ták mówi pánujący Pán : Dla tego / że o was powiádáją : Tyś jeſt tá źiemiá / która pożeráƺ ludźi / y ośierácáƺ narody twoje ;
Przetoż nie będźieƺ więcey ludźi pożeráłá / áni narodów twojich więcey ośierácáłá / mówi pánujący Pán.
Y nie dopuƺcżę w tobie więcey ſłyƺeć háńby narodów / áni zelżywośći ludzkiej nie ponieśieƺ więcey / y narodów twojich nie przywiedźieƺ więcey do upádku / mówi pánujący Pán.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! dom Izráelſki mieƺkájąc w źiemi ſwej zplugawiał ją drogámi ſwemi y ſpráwámi ſwemi / ták / że drogá ich przed oblicżem mojim byłá jáko niecżyſtość niewiáſty odłącżoney.
Przetoż wyláłem gniew mój ná nich dla krwi / którą wyláli ná źiemię / y dla plugáwych báłwánów ich / któremi ją ſplugáwili.
Y rozproƺyłem ich miedzy narody / á rozwiáni ſą po źiemiách ; według dróg ich y według ſpráw ich ſądźiłem ich.
A gdy weƺli do narodów / do których przyƺli / pomázáli tám imię świętobliwośći mojey / gdy o nich mówiono : Lud to Páńſki / á z źiemi jego wyƺli.
Alem im ſfolgował dla imienia świętobliwośći mojey / które ſplugáwił dom Izráelſki miedzy národámi / do których przyƺli.
Przetoż mów do domu Izráelſkiego : Ták mówi pánujący Pán : Nie dla was Ja to cżynię / o domie Izráelſki! ále dla imienia świętobliwośći mojey / któreśćie ſplugáwili miedzy národámi / do którycheśćie przyƺli ;
Abym poświęćił wielkie imię moje / które było ſplugáwione miedzy národámi / któreśćie wy zmázáli w pośrzodku ich ; y dowiedzą śię narody / żem Ja Pán / mówi pánujący Pán / gdy poświęcony będę w was przed ocżymá ich ;
Bo was zbiorę z narodów / y zgromádzę was ze wƺyſtkich źiem / y przywiodę was do źiemi wáƺej ;
Y pokropię was wodą cżyſtą / á ocżyƺcżeni będźiećie od wƺyſtkich niecżyſtót wáƺych / y od wƺyſtkich plugáwych báłwánów wáƺych ocżyƺcżę was ;
Y dam wam ſerce nowe / á duchá nowego dam do wnętrznośći wáƺych / y odejmę ſerce kámienne z ćiáłá wáƺego / á dam wam ſerce mięśiſte.
Duchá mego / mówię / dam do wnętrznośći wáƺej / á ucżynię / że w uſtáwách mojich chodźić / y ſądów mojich przeſtrzegáć / y cżynić je będźiećie.
A będźiećie mieƺkáć w źiemi / którąm dał ojcom wáƺym / y będźiećie ludem moim / á Ja będę Bogiem wáƺym.
Bo was wyzwolę od wƺyſtkich niecżyſtót wáƺych / y przywołám zbożá / y rozmnożę je / á nie dopuƺcżę ná was głodu.
Rozmnożę też owoc drzew / y urodzáje polne / żebyśćie więcey nie nośili háńby głodu miedzy národámi.
Y wſpomnićie ná złe drogi wáƺe / y ná ſpráwy wáƺe / które nie były dobre / y omierzniećie ſámi ſobie w ocżách ſwojich dla niepráwośći wáƺych y dla obrzydliwośći wáƺych.
Nie dla wáſći Ja to cżynię / mówi pánujący Pán / niech wam to jáwno będźie ; ſromájćie śię / á wſtydźćie śię zá drogi wáƺe / domie Izráelſki!
Ták mówi pánujący Pán : Którego was dniá ocżyƺcżę od wƺyſtkich niepráwośći wáƺych / oſádzę miáſtá / á miejſcá zburzone będą pobudowáne.
A ták źiemiá ſpuſtoƺáłá ſpráwowáná będźie / która przed tym byłá ſpuſtoƺoná przed ocżymá wƺyſtkich przechodzących.
Y rzeką : Ziemiá tá ſpuſtoƺáłá ſtáłá śię jáko ogród Eden ; tákże miáſtá puſte y opuƺcżone y rozwálone / obronne ſą y oſádzone.
Y dowiedzą śię narody / którekolwiek zoſtáną około was / żem Ja Pán pobudował rozwáliny / á náſádźił miejſcá ſpuſtoƺone. Ja Pán mówiłem / y ucżynię.
Ták mówi pánujący Pán : Jeƺcże tego będźie u mnie ƺukał dom Izráelſki / ábym to im ucżynił / ábym ich rozmnożył w ludźi jáko trzodę.
Jáko trzodę ná ofiáry / jáko trzodę Jeruzálemſką ná świętá jego urocżyſte / ták ſpuſtoƺone miáſtá będą nápełnione trzodámi ludźi ; y dowiedzą śię / żem Ja Pán.
