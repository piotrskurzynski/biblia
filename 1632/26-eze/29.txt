Roku dźieśiątego / dźieśiątego mieśiącá / dwunáſtego dniá tegoż mieśiącá ſtáło śię ſłowo Páńſkie do mnie mówiąc :
Synu cżłowiecży! obróć twárz ſwoję przećiwko Fáráonowi / królowi Egipſkiemu / á prorokuj przećiw niemu y przećiwko wƺyſtkiemu Egiptowi ;
Mów / á rzecż : Ták mówi pánujący Pán : Otom Ja przećiwko tobie o Fáráonie / królu Egipſki / wielorybie wielki! który leżyƺ w pośrzodku rzek twojich / y mówiƺ : Mojáć jeſt rzeká / y jam ją ſobie ucżynił ;
Przetoż włożę wędę w cżeluśći twoje / y ucżynię / że powięzną ryby rzek twojich ná łuſkách twych / y wywlekę ćię z pośrzodku rzek twojich y wƺyſtkie ryby rzek twojich / które ná łuſkách twojich powięzną ;
Y zoſtáwię ćię ná puƺcży / ćiebie y wƺyſtkie ryby rzek twojich : polężeƺ ná polu / y nie będźieƺ zebrány áni zgromádzony ; dam ćię beſtyjom źiemſkim y ptáſtwu niebieſkiemu ku pożárćiu ;
Y dowiedzą śię wƺyſcy mieƺkájący w Egipćie / żem Ja Pán / przeto żeśćie láſką trzćiniáną domowi Izráelſkiemu.
Gdy śię ćiebie ręką chwytáją / łámieƺ śię / y rozćináƺ im wƺyſtko rámię ; á gdy śię podpieráją tobą / kruƺyƺ śię / choć im nádſtáwiáƺ wƺyſtkich biódr.
Dlátegoż ták mówi pánujący Pán : Oto Ja przywiodę ná ćię miecż / y wygłádzę z ćiebie cżłowieká y bydlę ;
A źiemiá Egipſká będźie puſtynią y ſpuſtoƺeniem / y dowiedzą śię / żem Ja Pán / dla tego / żeś mówił : Rzeká mojá / y jam ją ucżynił ;
Przetoż oto Ja będę przećiwko tobie y przećiwko rzece twojey / y podám źiemię Egipſką w ſpuſtoƺenie / y we ſrogie poburzenie / od wieży Sewene áż do gránic Murzyńſkich.
Nie przejdźie przez nią nogá cżłowiecżá / y nogá bydlęcá nie przejdźie przez nią / áni w niej będą mieƺkáć przez cżterdźieśći lat.
A ták ucżynię źiemię Egipſką puſtynią nád inne źiemie ſpuſtoƺone / á miáſtá jey nád inne miáſtá ſpuſtoƺone / będą ſpuſtoƺone przez cżterdźieśći lat : gdyż rozproƺę Egipcżán miedzy narody / y rozwieję ich po źiemiách.
A wƺákże ták mówi pánujący Pán : Gdy śię ſkońcży cżterdźieśći lat / zgromádzę Egipcżán z narodów / do których rozproƺeni będą.
Y przywrócę záśię więźniów Egipſkich / y przywiodę ich do źiemi Pátros / do źiemi mieƺkánia ich / y będą tám króleſtwem podłem.
Miedzy innemi króleſtwámi będźie nájpodlejƺem / á nie wynieśie śię więcey nád inne narody / y umniejƺę ich / áby nie pánowáli nád národámi.
Y nie będźie więcey domowi Izráelſkiemu ufnośćią / któráby mi ná pámięć przywodźiłá niepráwość / gdyby śię oglądáli ná nie ; y dowiedzą śię / żem Ja pánujący Pán.
Potem ſtáło śię dwudźieſtego y śiódmego roku / pierwƺego mieśiącá / pierwƺego dniá tegoż mieśiącá / ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! Nábuchodonozor / król Bábilońſki / przyniewolił gwáłtem wojſko ſwe do ſłużby ćiężkiej przećiwko Tyrowi ; káżda głowá obłyśiáłá / y káżde rámię obnáżone / á przećie nie ma zápłáty on / áni wojſko jego z Tyru zá onę ſłużbę / którą podejmował / wálcżąc przećiwko jemu.
Przetoż ták mówi pánujący Pán : Oto Ja dáję Nábuchodonozorowi / królowi Bábilońſkiemu / źiemię Egipſką / áby zábrał doſtátki jey / y rozƺárpał łupy jey / y rozchwyćił korzyśći jey / áby miáło zápłátę wojſko jego.
Zá prácę ich / którą dla mnie podjęli / dam im źiemię Egipſką / przeto / że mnie prácowáli / mówi pánujący Pán.
Dniá onego ucżynię / że wyrośnie róg domu Izráelſkiego / tobie też uſtá twoje otworzę w pośrzodku ich ; y dowiedzą śię / żem Ja Pán.
