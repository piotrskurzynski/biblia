Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! były dwie niewiáſty / córki jedney mátki ;
Te nierząd płodźiły w Egipćie / w młodośći ſwojey nierząd płodźiły ; tám ſą omácáne pierśi ich / y tám ſą zgniećione pierśi pánieńſtwá ich.
Imioná ich te ſą : Więkƺej Aholá / á śioſtry jey Aholibá. Teć były moje / y zrodźiły ſynów y córki ; imioná / mówię / ich ſą : Sámáryjá Aholá / á Jeruzálem Aholibá.
Ale Aholá płodźiłá wƺetecżeńſtwo przy mnie / á nierządu pilnowáłá z miłośnikámi ſwojimi / z Aſſyryjcżykámi bliſkimi /
Którzy byli oblecżeni w hijácynt / z kśiążętámi / y pánámi / y ze wƺyſtkimi młodźieńcámi udátnymi / y z jezdnymi jeżdżącymi ná koniách ;
Udáłá śię / mówię / ná wƺetecżeńſtwo ſwoje z nimi / ze wƺyſtkimi nájprzedniejƺymi ſynámi Aſſyryjſkimi / y ze wƺyſtkimi / zá którymi miłośćią páłáłá / á ſplugáwiáłá śię wƺyſtkiemi plugáwemi báłwánámi ich.
A ták wƺetecżeńſtw ſwojich egipſkich nie zániecháłá ; bo z nią ſypiáli w młodośći jey / á oni omácáli pierśi pánieńſtwá jey / y wyláli wƺetecżeńſtwo ſwe ná nią.
Dla tego podáłem ją w rękę zálotników jey / w rękę ſynów Aſſyryjſkich / zá którymi miłośćią páłáłá.
Oni odkryli nágość jey / ſynów jey y córki jey zábráli / á ſámę miecżem zábili ; y ſtáłá śię oſłáwioną miedzy niewiáſtámi / gdy ſądy wykonáli przy niej.
To widząc śioſtrá jey Aholibá / bárdźiej śię niż oná zápáliłá miłośćią / á wƺetecżeńſtwo jey więkƺe było / niżeli wƺetecżeńſtwo śioſtry jey.
Zá ſynámi Aſſyryjſkimi páłáłá miłośćią / zá kśiążętámi / y pánámi bliſkimi / ubránymi w ƺáty koƺtowne / zá jezdnymi jeżdżącymi ná koniách / y zá wƺyſtkimi młodźieńcámi urodźiwymi.
Y widźiałem / że śię ſplugáwiłá / á iż jednáká drogá obu byłá.
Ale tá jeƺcże to przydáłá do wƺetecżeńſtw ſwojich / że widząc mężów wymálowánych ná śćiánie / obrázy Cháldejcżyków málowáne fárbámi /
Opáſáne páſámi ná biodrách ich / y kołpáki fárbowáne ná głowách ich / á że wƺyſcy ná wejrzeniu byli jáko hetmáni / podobni ſynom Bábilońſkim / w źiemi Cháldejſkiej zrodzonym ;
Y páłáłá miłośćią ku nim / ujrzáwƺy ich ocżymá ſwemi / á wypráwiłá poſłów do nich do źiemi Cháldejſkiej.
Tedy weƺli do niej ſynowie Bábilońſcy do komory nierządu / y zmázáli ją wƺetecżeńſtwem ſwojim ; á gdy śię ſplugáwiłá z nimi / odſtąpiłá duƺá jey od nich.
Y gdy odkryłá wƺetecżeńſtwá ſwe / odkryłá y nágość ſwoję ; odſtąpiłá duƺá mojá od niej / jáko byłá odſtąpiłá duƺá mojá od śioſtry jey.
Bo rozmnożyłá wƺetecżeńſtwá ſwoje / wſpominájąc ná dni młodośći ſwojey / których nierządu pátrzyłá w źiemi Egipſkiej.
Y páłáłá miłośćią przećiwko nierządnikom ich / których ćiáłá ſą jáko ćiáłá oſłów / á przyrodzenie ich jáko przyrodzenie końſkie.
A tákeś śię záś náwróćiłá do ſproſnośći młodośći twojey / gdy mácáli Egipcżánie pierśi twoje dla pierśi młodośći twojey.
Przetoż / o Aholibo! ták mówi pánujący Pán : Oto ja pobudzę zálotników twojich przećiwko tobie / tych / od których odſtąpiłá duƺá twojá / y przywiodę ich ná ćię zewƺąd ;
Synów Bábilońſkich / y wƺyſtkich Cháldejcżyków z Pekot / y z Sohy / y z Kohy / y wƺyſtkich ſynów Aſſyryjſkich z nimi / młodźieńców udátnych / kśiążąt y pánów wƺyſtkich / hetmánów y ludźi zácnych / wƺyſtkich jeżdżących ná koniách ;
Y przyjádą przećiwko tobie ná wozách żeláznych / y ná rydwánách / y ná kárách / á to z zgráją narodów / z tárcżámi / y z páncerzámi / y z przyłbicámi / położą  śię przećiwko tobie zewƺąd ; y dam im práwo / áby ćię ſądźili według práw ſwojich.
Y wyleję gorliwość moję ná ćię / y obejdą śię z tobą zápálcżywie / nos twój y uƺy twoje obrzeżą / á oſtátek twój od miecżá polegnie ; oni ſynów twojich y córki twoje pojmáją / á oſtátek twój ogniem pożárty będźie ;
Y zewloką ćię z ƺát twojich / á pobiorą ſtrój ozdoby twojey.
A ták ucżynię koniec ſproſnośći twojey przy tobie / y wƺetecżeńſtwu twemu / któreś przynioſłá z źiemi Egipſkiej / á nie podnieśieƺ ocżów twych do nich / áni ná Egipt nie wſpomniƺ więcey.
Bo ták mówi pánujący Pán : Oto ćię Ja podáję w ręce tych / których maƺ w nienáwiśći / w ręce tych / od których odſtąpiłá duƺá twojá ;
Y obejdą śię z tobą według nienáwiśći / y zábiorą wƺyſtkę prácę twoję / á zoſtáwią ćię nágą y obnáżoną / y będźie jáwná nágość wƺetecżeńſtw twojich / y ſproſnośći twojey / y nierządów twojich.
Toć śię ſtánie przeto / żeś nierząd płodźiłá náśládując pogán / przeto / żeś śię zmázáłá plugáwemi báłwánámi ich ;
Chodźiłáś drogą śioſtry ſwej / dla tego dam kubek jey w rękę twoję.
Ták mówi pánujący Pán : Kubek śioſtry twojey głęboki y ƺeroki pić będźieƺ / ſpory będźie ; pośmiech tákże y igrzyſko będą mieli z ćiebie.
Pijáńſtwem y boleśćią nápełnioná będźieƺ / kubkiem ſpuſtoƺeniá y ſmutku / kubkiem śioſtry ſwej Sámáryi!
Y wypijeƺ go y wyśiącżyƺ / potem go pokruƺyƺ / á pierśi ſwe poobrywáƺ ; bom Ja rzekł / mówi pánujący Pán.
Dla tego ták mówi pánujący Pán : Poniewáżeś zápomniáłá ná mię / á zárzućiłáś mię zá tył ſwój / y ty też ponoś niecnotę ſwoję / y wƺetecżeńſtwá ſwoje.
Y rzekł Pán do mnie : Synu cżłowiecży! izali śię będźieƺ záſtáwiał zá Aholę álbo zá Aholibę? Oznájmij im rácżey obrzydliwośći ich :
Że cudzołożyły / á krew jeſt ná rękách ich / y z plugáwemi báłwánámi ſwemi cudzołożyły ; nádto y ſynów ſwych / których mi národźiły / przez ogień przeprowádzáły im ná pożárćie.
Więc y to mi ucżyniły / że świątnicę moję ſplugáwiły dniá onego / á ſábáty moje pogwáłćiły.
Bo gdy ofiárowáły ſynów ſwych plugáwym báłwánom ſwojim / wchodźiły do świątnicy mojey onegoż dniá / áby ją zmázáły ; oto tákći cżyniły w pośrzodku domu mego.
Nádto poſyłáły też do mężów / áby przyƺli z dáleká ; którzy záráz przychodźili / kiedy poſeł do nich wyſłány był. Tymeś kwoli śię ty umywáłá / fárbowáłáś twárz ſwoję / y zdobiłáś śię ochędóſtwem ſwojim.
Siádáłáś ná łożu zácnem / przed którem był ſtół przygotowány / ná któremeś y kádzenie moje y olejek mój pokłádáłá.
A gdy głos onego mnóſtwá ućichł / tedy y do mężów ludu poſpolitego poſyłáły / których przywodzono ożártych z puƺcży / y kłádli mánele ná ręce ich / y korony ozdobne ná głowy ich.
A choćiáżem przymáwiał oney cudzołożnicy záſtárzáłej / á iż oni raz z jedną / raz z drugą nierząd płodzą /
Y że káżdy z nich wchodźi do niej / jáko wchodzą do niewiáſty wƺetecżney : przećie jednák wchodźili do Aholi y do Aholiby / niewiáſt niecnotliwych.
Przetoż ſpráwiedliwi mężowie / ći je oſądzą ſądem cudzołożnic / y ſądem wylewájących krew / przeto / że cudzołożyły / á krew jeſt ná rękách ich.
Bo ták mówi pánujący Pán : Przywiodę ná nie wojſko / á poddám je ná potłukánie y ná łup ;
Y ukámionuje je ono zgromádzenie kámieniem / y rozśieką je miecżámi ſwemi / ſynów ich y córki ich pobiją / á domy ich ogniem ſpálą.
A ták uprzątnę ſproſność z tey źiemi / y będą śię tym káráć wƺyſtkie niewiáſty / á nie ucżynią według niecnoty wáƺej.
Bo włożoná będźie ná was niecnotá wáƺá / á grzechy plugáwych báłwánów wáƺych ponieśiećie / y poznáćie / żem Ja pánujący Pán.
